## ¿Por Qué Elegir GroupDocs.Conversion para Java?

* **Broad format support** – Handles Word, Excel, CSV, ZIP, email, and many more, making it a one‑stop solution for all your conversion needs.  
* **High fidelity** – Preserves layout, fonts, and complex elements (tables, charts, annotations) so the output PDF looks exactly like the source.  
* **Performance‑oriented** – Streamlined APIs let you convert large batches or single files with minimal memory footprint—ideal for cloud services.  
* **Security‑ready** – Built‑in options for handling password‑protected PDFs, removing embedded files, and extracting metadata securely.

## Comienza Rápido

1. **Añade la dependencia Maven** (o equivalente Gradle) para GroupDocs.Conversion.  
2. **Instancia el `Converter`** con tu clave de licencia.  
3. **Elige el formato de origen** (p.ej., `.docx`, `.xlsx`, `.csv`).  
4. **Configura `PdfConvertOptions`** – establece el tamaño de página, sustitución de fuentes o manejo de contraseñas.  
5. **Llama a `convert`** y escribe el flujo de salida a un archivo o respuesta.

> **Consejo profesional:** Cuando necesites **convert excel to pdf java** con fuentes personalizadas, establece `PdfConvertOptions.setFontSubstitution(true)` y proporciona un `FontMapper` para garantizar la consistencia visual.
