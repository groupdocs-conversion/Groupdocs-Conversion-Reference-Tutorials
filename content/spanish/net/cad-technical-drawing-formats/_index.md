# Tutoriales de formatos CAD y de dibujo técnico para GroupDocs.Conversion .NET

En este centro encontrarás guías paso a paso para **convertir CAD a PDF** usando GroupDocs.Conversion para .NET, cubriendo dimensiones de página personalizadas, aplanado de modelos 3‑D, preservación de capas y manejo de dibujos de gran tamaño. Ya sea que estés construyendo un servicio web que renderiza esquemas de ingeniería o una utilidad de escritorio que procesa miles de dibujos, estos tutoriales te muestran cómo obtener resultados fiables y listos para producción mientras mantienes bajo el uso de memoria.

GroupDocs.Conversion para .NET es una biblioteca .NET que convierte más de 150 formatos —incluidos dibujos CAD— a PDF, imágenes y archivos de Office. Es compatible con .NET Framework y .NET Core, funciona en Windows, Linux y macOS, y no requiere que se instale software CAD externo en el servidor.

**Afirmación cuantificada:** GroupDocs.Conversion puede procesar archivos CAD de hasta 500 MB de tamaño y convertir dibujos de varios cientos de páginas en menos de 30 segundos en una VM típica de 4 núcleos, gracias a su arquitectura de streaming que evita cargar todo el archivo fuente en memoria.
