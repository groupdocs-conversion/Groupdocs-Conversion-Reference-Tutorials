## Getting Started Quickly

1. **Aggiungi la dipendenza Maven** (or Gradle equivalent) for GroupDocs.Conversion.  
2. **Istanzia il `Converter`** with your license key.  
3. **Scegli il formato di origine** (e.g., `.docx`, `.xlsx`, `.csv`).  
4. **Configura `PdfConvertOptions`** – set page size, font substitution, or password handling.  
5. **Chiama `convert`** and write the output stream to a file or response.

> **Consiglio professionale:** When you need to **convert excel to pdf java** with custom fonts, set `PdfConvertOptions.setFontSubstitution(true)` and provide a `FontMapper` to guarantee visual consistency.
