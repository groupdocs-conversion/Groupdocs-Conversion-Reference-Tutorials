## คู่มือการใช้งาน
ทำตามขั้นตอนต่อไปนี้เพื่อแปลงไฟล์ DOCX เป็น PDF โดยใช้ GroupDocs.Conversion

### คุณสมบัติการแปลงเอกสารเป็น PDF
มุ่งเน้นที่การแปลงเอกสาร Word ให้เป็นไฟล์ PDF ที่สามารถเข้าถึงได้ทั่วโลก นี่คือวิธีการใช้งาน:

#### ขั้นตอนที่ 1: กำหนดเส้นทางอินพุตและเอาต์พุต
ระบุตำแหน่งเอกสารอินพุตและเส้นทางการบันทึก PDF เอาท์พุต:

```java
String inputFilePath = "YOUR_DOCUMENT_DIRECTORY/sample.docx";
String outputFilePath = "YOUR_OUTPUT_DIRECTORY/HelloWorld.pdf";
```

#### ขั้นตอนที่ 2: สร้างวัตถุตัวแปลง
เริ่มต้นการใช้งาน `Converter` คลาสที่มีเส้นทางไฟล์ DOCX ของคุณ อ็อบเจ็กต์นี้จัดการกระบวนการแปลง:

```java
Converter converter = new Converter(inputFilePath);
```

#### ขั้นตอนที่ 3: เริ่มต้นตัวเลือกการแปลง PDF
ตั้งค่าตัวเลือกสำหรับการแปลงเป็นรูปแบบ PDF โดยใช้ `PdfConvertOptions`-

```java
class PdfConvertOptions {
    // กำหนดค่าการตั้งค่าการแปลงของคุณที่นี่
}

PdfConvertOptions options = new PdfConvertOptions();
```

#### ขั้นตอนที่ 4: ดำเนินการแปลง
ดำเนินการแปลงโดยเรียกใช้ `convert` วิธีการแปลงไฟล์ DOCX ของคุณเป็น PDF:

```java
converter.convert(outputFilePath, options);
```

- **ประเด็นเรื่องใบอนุญาต**ตรวจสอบการตั้งค่าใบอนุญาตของคุณหากคุณพบข้อจำกัดในการใช้งาน
