# วิธีการแปลงเอกสารที่ป้องกันด้วยรหัสผ่านโดยใช้ GroupDocs.Conversion สำหรับ Java

## การแนะนำ

คุณกำลังประสบปัญหาในการแปลงเอกสารที่ป้องกันด้วยรหัสผ่านเป็นรูปแบบสเปรดชีตใน Java หรือไม่ ด้วย GroupDocs.Conversion API การแปลงเอกสาร Word ที่ได้รับการป้องกันเป็นสเปรดชีต Excel จะทำได้อย่างราบรื่นและมีประสิทธิภาพ บทช่วยสอนนี้จะแนะนำคุณเกี่ยวกับการโหลดและการแปลงไฟล์ Word ที่ป้องกันด้วยรหัสผ่านโดยใช้ตัวเลือกขั้นสูง

**สิ่งที่คุณจะได้เรียนรู้:**
- การตั้งค่า GroupDocs.Conversion สำหรับ Java
- การโหลดเอกสารที่ป้องกันด้วยรหัสผ่านด้วย API
- การแปลงหน้าเฉพาะเป็นรูปแบบ XLS
- การกำหนดค่าการตั้งค่าการแปลงขั้นสูงเช่นระดับการซูมและช่วงหน้า

ก่อนที่เราจะเริ่ม ให้แน่ใจว่าคุณมีสิ่งที่จำเป็นเบื้องต้นพร้อมแล้ว

## คู่มือการใช้งาน

เราจะครอบคลุมการโหลดเอกสารที่ป้องกันด้วยรหัสผ่านและการแปลงเป็นรูปแบบสเปรดชีตด้วยตัวเลือกขั้นสูง

### การโหลดเอกสารที่ป้องกันด้วยรหัสผ่าน

**ภาพรวม:**
การโหลดเกี่ยวข้องกับการตั้งค่าตัวเลือกการโหลดซึ่งรวมถึงรหัสผ่านของเอกสาร ขั้นตอนนี้มีความสำคัญต่อการแปลง

#### ขั้นตอนที่ 1: ตั้งค่าตัวเลือกการโหลด
```java
import com.groupdocs.conversion.options.load.WordProcessingLoadOptions;

WordProcessingLoadOptions loadOptions = new WordProcessingLoadOptions();
loadOptions.setPassword("12345"); // แทนที่ด้วยรหัสผ่านเอกสารของคุณ
```
- **คำอธิบาย:** การ `WordProcessingLoadOptions` คลาสนี้จัดการเอกสารที่ป้องกันด้วยรหัสผ่าน โปรดระบุรหัสผ่านที่ถูกต้องเพื่อเข้าถึงเนื้อหา

### การแปลงเป็นสเปรดชีตด้วยตัวเลือกขั้นสูง

**ภาพรวม:**
หลังจากโหลดแล้ว ให้แปลงเอกสารเป็นสเปรดชีต Excel โดยเน้นที่หน้าเฉพาะและการตั้งค่าแบบกำหนดเอง เช่น ระดับการซูม

#### ขั้นตอนที่ 2: กำหนดค่าตัวเลือกการแปลง
```java
import com.groupdocs.conversion.Converter;
import com.groupdocs.conversion.filetypes.SpreadsheetFileType;
import com.groupdocs.conversion.options.convert.SpreadsheetConvertOptions;

SpreadsheetConvertOptions options = new SpreadsheetConvertOptions();
options.setPageNumber(2); // เริ่มตั้งแต่หน้า 2.
options.setPagesCount(1); // แปลงเพียงหน้าเดียว
options.setFormat(SpreadsheetFileType.Xls); // รูปแบบเอาท์พุตเป็น XLS
options.setZoom(150); // กำหนดระดับการซูมเพื่อการแปลง (ปัจจัยมาตราส่วน)
```
- **คำอธิบาย:** ระบุหน้าที่ต้องการแปลง (`setPageNumber` และ `setPagesCount`) และรูปแบบเอาท์พุต `setZoom` ตัวเลือกนี้จะปรับขนาดเนื้อหาในระหว่างการแปลง

#### ขั้นตอนที่ 3: ดำเนินการแปลง
```java
String convertedFile = Constants.getConvertedPath("ConvertToSpreadsheetWithAdvancedOptions.xls");
Converter converter = new Converter(Constants.SAMPLE_DOCX_WITH_PASSWORD, () -> loadOptions);
converter.convert(convertedFile, options);
```
- **คำอธิบาย:** การ `convert` วิธีการนี้จะดำเนินการแปลงโดยใช้การกำหนดค่าที่กำหนดไว้ก่อนหน้านี้ ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ถูกต้องสำหรับอินพุตและเอาต์พุต

หากมีปัญหาเกิดขึ้น:
