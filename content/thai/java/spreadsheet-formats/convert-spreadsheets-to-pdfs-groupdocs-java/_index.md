## การแนะนำ

ต้องการแปลงสเปรดชีตของคุณเป็นรูปแบบ PDF ที่สามารถเข้าถึงได้ทั่วไปพร้อมๆ กับการจัดการความซับซ้อน เช่น ความคิดเห็นที่ซ่อนอยู่หรือแผ่นงานหลายหน้าหรือไม่ คู่มือนี้สาธิตวิธีการโหลดและแปลงไฟล์สเปรดชีตอย่างมีประสิทธิภาพโดยใช้ไลบรารี GroupDocs.Conversion ที่ทรงพลังใน Java ด้วยตัวเลือกขั้นสูงที่คุณสามารถใช้ได้ ปรับแต่งกระบวนการแปลงของคุณเพื่อซ่อนรายละเอียดที่ไม่จำเป็น เช่น ความคิดเห็น และตรวจสอบให้แน่ใจว่าแผ่นงานแต่ละแผ่นถูกส่งออกเป็นหน้าเดียวในรูปแบบ PDF

- การแปลงสเปรดชีตเป็น PDF ในขณะที่ใช้การตั้งค่าขั้นสูง

## คู่มือการใช้งาน

### โหลดสเปรดชีตด้วยตัวเลือกขั้นสูง

#### ภาพรวม:
การโหลดสเปรดชีตของคุณด้วยตัวเลือกเฉพาะช่วยให้คุณปรับแต่งวิธีการจัดการเอกสารก่อนการแปลงได้ ซึ่งจะช่วยปรับปรุงเวิร์กโฟลว์ของคุณโดยการซ่อนความคิดเห็นและทำให้แน่ใจว่าแต่ละชีตได้รับการปฏิบัติเหมือนเป็นหน้าเดียว

**ทีละขั้นตอน:**

##### ขั้นตอนที่ 1: ตั้งค่าตัวเลือกการโหลด
สร้างอินสแตนซ์ของ `SpreadsheetLoadOptions` และกำหนดค่าตามความต้องการของคุณ:

```java
import com.groupdocs.conversion.options.load.SpreadsheetLoadOptions;

public class LoadSpreadsheetWithOptions {
    public static void main(String[] args) {
        // สร้างอินสแตนซ์ของ SpreadsheetLoadOptions
        SpreadsheetLoadOptions loadOptions = new SpreadsheetLoadOptions();
        
        // ตั้งค่าตัวเลือกเพื่อซ่อนความคิดเห็นและตั้งค่าหนึ่งหน้าต่อแผ่นงาน
        loadOptions.setHideComments(true);
        loadOptions.setOnePagePerSheet(true);

        System.out.println("Loading options configured!");
    }
}
```

##### คำอธิบาย:
- `setHideComments(true)`:ซ่อนความคิดเห็นทั้งหมดในสเปรดชีต ทำให้มองไม่เห็นในผลลัพธ์การแปลง
- `setOnePagePerSheet(true)`:ช่วยให้แน่ใจว่าแต่ละแผ่นจะถูกแปลงเป็นหน้า PDF เดียว ป้องกันการใช้สเปรดชีตหลายหน้า

### แปลงสเปรดชีตเป็น PDF

#### ภาพรวม:
เมื่อคุณกำหนดค่าตัวเลือกการโหลดเรียบร้อยแล้ว ก็ถึงเวลาแปลงสเปรดชีตของคุณเป็นไฟล์ PDF ขั้นตอนนี้เกี่ยวข้องกับการเริ่มต้นคลาส Converter ด้วยตัวเลือกการโหลดที่ระบุและใช้การตั้งค่าการแปลง

**ทีละขั้นตอน:**

##### ขั้นตอนที่ 2: กำหนดเส้นทางไฟล์
ตั้งค่าเส้นทางอินพุตและเอาต์พุตสำหรับไฟล์ของคุณ:

```java
String inputFilePath = "YOUR_DOCUMENT_DIRECTORY/SAMPLE_XLSX";
String outputFilePath = "YOUR_OUTPUT_DIRECTORY/ConvertSpreadsheetAndHideComments.pdf";
```

##### ขั้นตอนที่ 3: เริ่มต้นตัวแปลงด้วยตัวเลือกโหลด

ใช้ฟังก์ชั่นแลมบ์ดาเพื่อส่งตัวเลือกการโหลดเมื่อสร้าง `Converter` ตัวอย่าง:

```java
import com.groupdocs.conversion.Converter;
import com.groupdocs.conversion.options.convert.PdfConvertOptions;

public class ConvertSpreadsheetToPdf {
    public static void main(String[] args) {
        SpreadsheetLoadOptions loadOptions = new SpreadsheetLoadOptions();
        loadOptions.setHideComments(true);
        loadOptions.setOnePagePerSheet(true);

        // สร้างอินสแตนซ์ของตัวแปลงพร้อมตัวเลือกการโหลด
        Converter converter = new Converter(inputFilePath, () -> loadOptions);

        System.out.println("Converter ready for conversion!");
    }
}
```

##### ขั้นตอนที่ 4: แปลงเป็น PDF

สุดท้ายให้ใช้การตั้งค่าการแปลงและดำเนินการตามกระบวนการ:

```java
PdfConvertOptions options = new PdfConvertOptions();
converter.convert(outputFilePath, options);
System.out.println("Conversion completed successfully!");
```

**คำอธิบาย:**
- `PdfConvertOptions`: ระบุการตั้งค่าเพิ่มเติมสำหรับผลลัพธ์ PDF การตั้งค่าเริ่มต้นจะใช้ที่นี่ แต่คุณสามารถปรับแต่งเพิ่มเติมได้ตามต้องการ

### เคล็ดลับการแก้ไขปัญหา:
- **ปัญหาเส้นทางไฟล์**:ให้แน่ใจว่าเส้นทางได้รับการกำหนดอย่างถูกต้องและสามารถเข้าถึงได้
- **ข้อผิดพลาดเกี่ยวกับการอ้างอิง**ตรวจสอบการกำหนดค่า Maven และลิงก์ที่เก็บข้อมูลของคุณอีกครั้ง
