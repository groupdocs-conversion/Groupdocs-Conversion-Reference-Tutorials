## คู่มือการใช้งาน

### แปลงเอกสาร Word เป็น Excel
ฟีเจอร์นี้สาธิตวิธีการแปลงเอกสาร Microsoft Word เป็นสเปรดชีต Excel โดยใช้ GroupDocs.Conversion ทำตามขั้นตอนเหล่านี้:

#### ขั้นตอนที่ 1: กำหนดเส้นทางไฟล์เอาท์พุต
ขั้นแรกระบุตำแหน่งสำหรับไฟล์ที่คุณแปลง:

```java
String outputFilePath = "YOUR_OUTPUT_DIRECTORY/ConvertToSpreadsheet.xlsx";
```
ควรแทนที่เส้นทางนี้ด้วยเส้นทางไดเร็กทอรีจริงระหว่างการใช้งาน

#### ขั้นตอนที่ 2: เริ่มต้นตัวแปลง
ตามที่แสดงในส่วนการตั้งค่า ให้เริ่มต้นการทำงาน `Converter` วัตถุกับเอกสาร Word ของคุณ:

```java
import com.groupdocs.conversion.Converter;
// ระบุเส้นทางเอกสารอินพุต
Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SampleDocument.docx");
```

#### ขั้นตอนที่ 3: กำหนดค่าตัวเลือกการแปลง
ถัดไปสร้างอินสแตนซ์ของ `SpreadsheetConvertOptions` เพื่อระบุการตั้งค่าสำหรับรูปแบบ Excel:

```java
import com.groupdocs.conversion.options.convert.SpreadsheetConvertOptions;
// กำหนดค่าตัวเลือกที่เฉพาะเจาะจงสำหรับการแปลงสเปรดชีต
SpreadsheetConvertOptions options = new SpreadsheetConvertOptions();
```

#### ขั้นตอนที่ 4: ดำเนินการแปลง
สุดท้ายให้โทรหา `convert` วิธีการของคุณ `Converter` อินสแตนซ์ที่จะดำเนินการแปลงและบันทึกผลลัพธ์:

```java
// ดำเนินการแปลงและบันทึกผลลัพธ์ในเส้นทางที่ระบุ
converter.convert(outputFilePath, options);

System.out.println("Document converted successfully.");
```

**เหตุใดจึงใช้แนวทางนี้?**
ใช้คลาสเฉพาะเช่น `SpreadsheetConvertOptions` ช่วยให้คุณปรับแต่งการแปลงได้อย่างแม่นยำ ช่วยให้มั่นใจว่าผลลัพธ์ของคุณตรงตามข้อกำหนดที่แน่นอน
