## คู่มือการใช้งาน

ตอนนี้เรามาแบ่งการใช้งานออกเป็นสองคุณสมบัติหลัก: การโหลดไฟล์ PPT ต้นฉบับและการแปลงเป็นรูปแบบ HTML

### คุณสมบัติ 1: โหลดไฟล์ PPT ต้นฉบับ

การโหลดไฟล์ PPT นั้นทำได้ง่ายด้วย GroupDocs.Conversion ดังต่อไปนี้:

**ขั้นตอนที่ 1: กำหนดเส้นทางเอกสาร**
```csharp
string documentPath = "YOUR_DOCUMENT_DIRECTORY/SamplePresentation.ppt";
```

บรรทัดนี้ระบุตำแหน่งไฟล์ PowerPoint ของคุณ

**ขั้นตอนที่ 2: เริ่มต้นวัตถุตัวแปลง**
```csharp
using (var converter = new Converter(documentPath))
{
    // ตอนนี้วัตถุตัวแปลงจะถือไฟล์ PPT ที่โหลดไว้
}
```
การ `Converter` คลาสนี้มีหน้าที่รับผิดชอบในการโหลดและเตรียมเอกสารเพื่อการแปลง และช่วยให้มั่นใจว่าสามารถเข้าถึงข้อมูลที่จำเป็นทั้งหมดจากไฟล์ PPT ของคุณได้

### คุณสมบัติ 2: แปลง PPT เป็นรูปแบบ HTML

เมื่อโหลดไฟล์ต้นฉบับแล้ว เราสามารถแปลงเป็นรูปแบบ HTML ได้:

**ขั้นตอนที่ 1: กำหนดเส้นทางเอาต์พุต**
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "ppt-converted-to.html");
```

บรรทัดเหล่านี้จะกำหนดว่าไฟล์ HTML ที่คุณแปลงแล้วจะถูกบันทึกที่ไหน

**ขั้นตอนที่ 2: ตั้งค่าตัวเลือกการแปลงและดำเนินการแปลง**
```csharp
using (var converter = new Converter(documentPath))
{
    var options = new WebConvertOptions();
    
    // ดำเนินการแปลงจากรูปแบบ PPT เป็นรูปแบบ HTML
    converter.Convert(outputFile, options);
}
```
การ `WebConvertOptions` คลาสนี้ช่วยให้คุณระบุการตั้งค่าต่างๆ สำหรับผลลัพธ์ HTML ได้ สไนปเป็ตนี้จะดำเนินการแปลงและบันทึกไฟล์ผลลัพธ์
