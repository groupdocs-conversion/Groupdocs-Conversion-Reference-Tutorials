## คู่มือการใช้งาน

### ภาพรวมคุณลักษณะการแปลง PCL เป็น HTML
คุณสมบัตินี้ช่วยให้คุณสามารถแปลงเอกสาร PCL ให้เป็นรูปแบบ HTML เพื่อให้สามารถดูแลและแก้ไขในเว็บเบราว์เซอร์ได้อย่างง่ายดาย 

#### ขั้นตอนที่ 1: เตรียมสภาพแวดล้อมของคุณ
ตรวจสอบให้แน่ใจว่าโครงการของคุณอ้างอิง GroupDocs.Conversion โดยทำตามคำแนะนำในการติดตั้งด้านบน

#### ขั้นตอนที่ 2: โหลดเอกสาร PCL
```csharp
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

string inputFilePath = "path/to/your/file.pcl";

// ใช้อินสแตนซ์ตัวแปลงเพื่อโหลดเอกสาร PCL ของคุณ
using (Converter converter = new Converter(inputFilePath))
{
    // ขั้นตอนการแปลงจะตามมาที่นี่
}
```

#### ขั้นตอนที่ 3: ตั้งค่าตัวเลือกการแปลง HTML
```csharp
var options = new MarkupConvertOptions();

// ปรับแต่งพารามิเตอร์การแปลงหากจำเป็น
options.FixedLayout = true; // รักษาเค้าโครงของเอกสารต้นฉบับ
```

#### ขั้นตอนที่ 4: ดำเนินการกระบวนการแปลง
```csharp
string outputFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "output.html");

converter.Convert(outputFilePath, options);
```
- **คำอธิบายพารามิเตอร์:** `MarkupConvertOptions` ช่วยให้คุณระบุการตั้งค่าเฉพาะ HTML ได้ เช่น การรักษาเค้าโครง
- **ค่าส่งคืน:** กระบวนการแปลงจะเขียนไฟล์ HTML ที่เส้นทางเอาต์พุตที่ระบุ

**เคล็ดลับการแก้ไขปัญหา:**
หากไฟล์ PCL ของคุณไม่แปลงตามที่คาดหวัง โปรดตรวจสอบว่าไฟล์นั้นสามารถเข้าถึงได้และไม่เสียหาย ตรวจสอบข้อยกเว้นใดๆ ที่เกิดขึ้นระหว่างขั้นตอนการโหลด
