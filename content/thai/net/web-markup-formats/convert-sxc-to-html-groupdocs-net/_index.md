## คู่มือการใช้งาน

ตอนนี้เราได้ติดตั้งและเริ่มต้นใช้งาน GroupDocs.Conversion แล้ว มาแบ่งการใช้งานออกเป็นขั้นตอนที่จัดการได้:

### ขั้นตอนที่ 1: กำหนดเส้นทางไดเรกทอรีเอาต์พุต

เริ่มต้นด้วยการระบุตำแหน่งที่จะจัดเก็บไฟล์ที่แปลงแล้ว วิธีนี้จะช่วยให้เข้าถึงผลลัพธ์ HTML ได้ง่ายหลังจากการแปลง

```csharp
private static string GetOutputDirectoryPath()
{
    return "YOUR_OUTPUT_DIRECTORY";
}
```

แทนที่ `YOUR_OUTPUT_DIRECTORY` พร้อมด้วยเส้นทางไดเร็กทอรีที่ต้องการในระบบของคุณ

### ขั้นตอนที่ 2: โหลดและแปลงไฟล์ SXC

ขั้นตอนต่อไปคือโหลดไฟล์ .sxc และแปลงเป็น HTML โค้ดต่อไปนี้จะสาธิตกระบวนการนี้:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

namespace GroupDocsConversionExample
{
    internal static class ConvertSxcToHtml
    {
        public static void Run()
        {
            string outputFolder = GetOutputDirectoryPath();
            string outputFile = Path.Combine(outputFolder, "sxc-converted-to.html");

            // โหลดไฟล์ SXC ต้นฉบับ แทนที่ด้วยไดเร็กทอรีเอกสารของคุณ
            string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "your-file.sxc");
            
            using (Converter converter = new Converter(inputFilePath))
            {
                var options = new HtmlConvertOptions();
                converter.Convert(outputFile, options);
                
                Console.WriteLine("Conversion completed successfully!");
            }
        }
    }
}
```

### คำอธิบายชิ้นส่วนโค้ด

- **`GetOutputDirectoryPath()`**: กำหนดเส้นทางในการจัดเก็บไฟล์เอาท์พุต
- **`Converter` วัตถุ**:โหลดไฟล์ SXC ของคุณเพื่อการแปลง
- **`HtmlConvertOptions`**: กำหนดค่าการตั้งค่าการแปลงให้เหมาะสมกับเอาท์พุต HTML

#### เคล็ดลับการแก้ไขปัญหา:

- ตรวจสอบให้แน่ใจว่าเส้นทางไดเร็กทอรีได้รับการตั้งค่าอย่างถูกต้องและสามารถเข้าถึงได้
- ตรวจสอบสิทธิ์การเข้าถึงไฟล์หากคุณพบปัญหาในการเข้าถึงระหว่างการดำเนินการไฟล์
- ใช้การจัดการข้อยกเว้นเพื่อจัดการข้อผิดพลาดรันไทม์อย่างเหมาะสม
