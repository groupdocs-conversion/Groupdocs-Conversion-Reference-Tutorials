## คู่มือการใช้งาน

### โหลดไฟล์ HTM ต้นฉบับ

เรียนรู้วิธีโหลดและเตรียมเอกสาร HTML เพื่อการแปลงโดยใช้ GroupDocs.Conversion

#### ขั้นตอนที่ 1: กำหนดไดเรกทอรีเอกสาร
ก่อนอื่น ระบุไดเร็กทอรีที่เอกสารของคุณอยู่:

```csharp
string documentDirectory = "/path/to/your/documents";
```

#### ขั้นตอนที่ 2: เริ่มต้นวัตถุตัวแปลง
สร้าง `Converter` วัตถุที่จะจัดการกระบวนการโหลดไฟล์:

```csharp
using (Converter converter = new Converter(Path.Combine(documentDirectory, "sample.htm")))
{
    // การกำหนดค่าการแปลงและการดำเนินการจะเกิดขึ้นที่นี่
}
```

**คำอธิบายพารามิเตอร์:**
- `documentDirectory`: เส้นทางที่ไฟล์ต้นฉบับของคุณตั้งอยู่
- `Path.Combine(...)`: รวมเส้นทางไดเร็กทอรีกับชื่อไฟล์เพื่อสร้างเส้นทางแบบเต็ม

#### ขั้นตอนที่ 3: กำหนดค่าตัวเลือกการแปลง
กำหนดค่าการตั้งค่าการแปลงตามความต้องการของคุณ (เช่น รูปแบบเป้าหมาย):

```csharp
var options = new PdfConvertOptions(); // ตัวอย่างการแปลงเป็น PDF
```

**ตัวเลือกการกำหนดค่าคีย์:**
- `PdfConvertOptions`: ระบุการตั้งค่าสำหรับการแปลง PDF
