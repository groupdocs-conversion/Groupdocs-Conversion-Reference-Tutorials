## คู่มือการใช้งาน

### คุณสมบัติ: โหลดไฟล์ JLS

#### ภาพรวม
การโหลดไฟล์ JLS ลงในแอปพลิเคชันของคุณเป็นขั้นตอนแรกในการแปลงไฟล์ คุณลักษณะนี้สาธิตวิธีการใช้ GroupDocs.Conversion สำหรับ .NET เพื่อโหลดไฟล์ JLS

#### ขั้นตอนที่ 1: เริ่มต้นวัตถุตัวแปลง
ในการเริ่มต้น ให้สร้างอินสแตนซ์ของ `Converter` คลาสที่มีเส้นทางไปยังไฟล์ JLS ของคุณ:
```csharp
string sourceFilePath = "YOUR_DOCUMENT_DIRECTORY/sample.jls";
var converter = new Converter(sourceFilePath);
```
**คำอธิบาย:** การ `Converter` วัตถุจะถูกเริ่มต้นด้วยเส้นทางของไฟล์เพื่อเตรียมพร้อมสำหรับการดำเนินการแปลง ตรวจสอบให้แน่ใจว่าไดเร็กทอรีและชื่อไฟล์ของคุณได้รับการระบุอย่างถูกต้อง

### คุณสมบัติ: แปลง JLS เป็น HTML

#### ภาพรวม
ฟีเจอร์นี้จะแปลงไฟล์ JLS ที่โหลดไว้เป็นรูปแบบ HTML โดยใช้ความสามารถอันแข็งแกร่งของ GroupDocs.Conversion

#### ขั้นตอนที่ 2: กำหนดพารามิเตอร์เอาต์พุต
ระบุไดเรกทอรีเอาท์พุตและชื่อไฟล์เอาท์พุตที่ต้องการ:
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = System.IO.Path.Combine(outputFolder, "jls-converted-to.html");
```
**คำอธิบาย:** การ `Path.Combine` วิธีนี้ใช้เพื่อสร้างเส้นทางไฟล์ที่ถูกต้องสำหรับเอาท์พุต HTML ของคุณ เพื่อให้แน่ใจว่าสามารถใช้งานร่วมกับระบบปฏิบัติการต่างๆ ได้

#### ขั้นตอนที่ 3: กำหนดค่าตัวเลือกการแปลง
กำหนดตัวเลือกการแปลงสำหรับรูปแบบ HTML โดยใช้ `WebConvertOptions`-
```csharp
var options = new WebConvertOptions();
```
**คำอธิบาย:** การ `WebConvertOptions` คลาสนี้จัดเตรียมการตั้งค่าเฉพาะสำหรับรูปแบบเว็บ เช่น HTML ซึ่งช่วยให้สามารถปรับแต่งคุณภาพเอาต์พุตและพารามิเตอร์อื่นๆ ได้

#### ขั้นตอนที่ 4: ดำเนินการแปลง
ดำเนินการแปลงและบันทึกไฟล์ผลลัพธ์:
```csharp
converter.Convert(outputFile, options);
```
**คำอธิบาย:** การ `Convert` วิธีนี้จะนำเส้นทางปลายทางและตัวเลือกเป็นอาร์กิวเมนต์ เพื่อแปลงไฟล์ JLS ของคุณเป็นรูปแบบ HTML อย่างมีประสิทธิภาพ
