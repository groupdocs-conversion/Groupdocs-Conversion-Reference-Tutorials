## คู่มือการใช้งาน

หัวข้อนี้สาธิตวิธีโหลดไฟล์ PPSX โดยใช้ GroupDocs.Conversion

#### ขั้นตอนที่ 1: กำหนดเส้นทางเอกสาร

ให้แน่ใจว่าคุณได้กำหนดไว้ `documentDirectory` ด้วยเส้นทางเอกสารของคุณ:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
```

#### ขั้นตอนที่ 2: เริ่มต้นคลาสตัวแปลง

สร้างอินสแตนซ์ของ `Converter` คลาสโดยการส่งเส้นทางไฟล์ PPSX

```csharp
var converter = new Converter(Path.Combine(documentDirectory, "sample.ppsx"));
```

### แปลง PPSX เป็น HTML

ตอนนี้มาแปลงไฟล์ PPSX ที่คุณโหลดไว้เป็นรูปแบบ HTML กัน

#### ขั้นตอนที่ 1: กำหนดเส้นทางไดเร็กทอรีเอาท์พุต

ตั้งค่าไดเรกทอรีเอาท์พุตที่จะบันทึกไฟล์ที่แปลงแล้ว:

```csharp
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
```

#### ขั้นตอนที่ 2: กำหนดค่าตัวเลือกการแปลง

สร้างอินสแตนซ์ของ `WebConvertOptions` เพื่อกำหนดค่าการตั้งค่าการแปลงของคุณ:

```csharp
var options = new WebConvertOptions();
```

#### ขั้นตอนที่ 3: ดำเนินการแปลง

ดำเนินการแปลงและบันทึกไฟล์ HTML ลงในไดเร็กทอรีที่คุณระบุ

```csharp
converter.Convert(Path.Combine(outputDirectory, "ppsx-converted-to.html"), options);
```
