## คู่มือการใช้งาน

### แปลงไฟล์ LaTeX (.tex) เป็นรูปแบบ CSV

#### ภาพรวม

ฟีเจอร์นี้จะแสดงวิธีการแปลงไฟล์ .tex เป็นรูปแบบ CSV ที่จัดการได้ในระดับสากล การแปลงนี้มีประโยชน์อย่างยิ่งสำหรับการวิเคราะห์ข้อมูลและการบูรณาการกับแอปพลิเคชันต่างๆ ที่รองรับไฟล์ CSV

##### ขั้นตอนที่ 1: กำหนดเส้นทางไฟล์ของคุณ

เริ่มต้นโดยระบุเส้นทางไดเร็กทอรีสำหรับไฟล์ LaTeX ต้นทางและไฟล์ CSV เอาท์พุต:

```csharp
string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY";

string texFilePath = Path.Combine(documentDirectory, "sample.tex");
string csvOutputFile = Path.Combine(outputDirectory, "converted.csv");
```

##### ขั้นตอนที่ 2: โหลดและแปลงไฟล์

การใช้งาน API ของ GroupDocs.Conversion โหลดไฟล์ .tex ของคุณและระบุตัวเลือกการแปลงสำหรับ CSV:

```csharp
using (var converter = new Converter(texFilePath))
{
    // ตั้งค่าตัวเลือกการแปลงเพื่อกำหนดเป้าหมายรูปแบบ CSV
    SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Csv };

    // ดำเนินการแปลง
    converter.Convert(csvOutputFile, options);
}
```

ตัวอย่างนี้สาธิตการโหลดไฟล์ .tex และการใช้การตั้งค่าการแปลงที่ปรับแต่งสำหรับเอาท์พุต CSV

#### เคล็ดลับการแก้ไขปัญหา

- **ปัญหาทั่วไป:** ไม่พบเส้นทางไฟล์ โปรดตรวจสอบให้แน่ใจว่าเส้นทางไดเร็กทอรีของคุณถูกต้อง
- **สารละลาย:** ตรวจสอบอีกครั้งว่า `YOUR_DOCUMENT_DIRECTORY` และ `YOUR_OUTPUT_DIRECTORY` ถูกกำหนดไว้อย่างแม่นยำตามโครงสร้างโครงการของคุณ
