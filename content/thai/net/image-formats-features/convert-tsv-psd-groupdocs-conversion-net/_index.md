## คู่มือการใช้งาน
ตอนนี้ มาแยกการใช้งานออกเป็นคุณสมบัติที่แตกต่างกันเพื่อความชัดเจน

### คุณสมบัติ 1: โหลดไฟล์ TSV
การโหลดไฟล์ TSV เป็นขั้นตอนแรกก่อนการแปลง คุณลักษณะนี้จะช่วยให้มั่นใจว่าข้อมูลต้นฉบับของคุณพร้อมสำหรับการแปลง

#### ภาพรวม
การ `Converter` คลาสจาก GroupDocs.Conversion ช่วยให้คุณโหลดไฟล์ TSV ได้อย่างง่ายดาย ช่วยกำหนดขั้นตอนสำหรับการประมวลผลเพิ่มเติม

#### ขั้นตอนการดำเนินการ
##### 1. เริ่มต้นวัตถุตัวแปลง
สร้างอินสแตนซ์ของ `Converter` คลาสที่มีเส้นทางของไฟล์ TSV ของคุณ

```csharp
using System.IO;
using GroupDocs.Conversion;

string tsvFilePath = "YOUR_DOCUMENT_DIRECTORY/sample.tsv";
using (Converter converter = new Converter(tsvFilePath))
{
    // ไฟล์ TSV โหลดเสร็จแล้ว
}
```
- **พารามิเตอร์:** `tsvFilePath` - เส้นทางไปยังไฟล์ TSV ของคุณ
- **วัตถุประสงค์:** นี่เป็นการเริ่มกระบวนการแปลงโดยโหลดไฟล์ต้นฉบับ

### คุณสมบัติ 2: แปลงเป็นรูปแบบ PSD
เมื่อโหลดแล้ว คุณสามารถแปลงข้อมูล TSV เป็นรูปแบบ PSD ได้โดยใช้ตัวเลือกเฉพาะที่ให้ไว้โดย GroupDocs.Conversion

#### ภาพรวม
การแปลงจาก TSV เป็น PSD เกี่ยวข้องกับการตั้งค่าตัวเลือกการแปลงและดำเนินการแปลง

#### ขั้นตอนการดำเนินการ
##### 1. กำหนดไดเรกทอรีและเทมเพลตเอาต์พุต
ตั้งค่าเส้นทางเอาต์พุตสำหรับไฟล์ที่แปลงของคุณ

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");
```

##### 2. สร้างฟังก์ชันสตรีมสำหรับหน้าต่างๆ
กำหนดวิธีการบันทึกแต่ละหน้าในระหว่างการแปลง

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```
- **วัตถุประสงค์:** ฟังก์ชันนี้จะสร้างสตรีมไฟล์สำหรับแต่ละหน้าที่แปลงแล้วเพื่อให้แน่ใจว่าได้รับการบันทึกอย่างถูกต้อง

##### 3. ตั้งค่าตัวเลือกการแปลง
กำหนดค่าการตั้งค่าการแปลงให้เป็นเอาท์พุตรูปแบบ PSD

```csharp
using GroupDocs.Conversion.Options.Convert;

ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd };
```
- **การกำหนดค่าคีย์:** `Format` ระบุประเภทไฟล์เป้าหมาย ในกรณีนี้คือ PSD

##### 4. ดำเนินการแปลง
ดำเนินการแปลงโดยใช้ตัวแปลงวัตถุที่เริ่มต้นและการตั้งค่าที่กำหนดไว้

```csharp
converter.Convert(getPageStream, options);
```
- **วัตถุประสงค์:** ขั้นตอนนี้จะแปลงข้อมูล TSV ให้เป็นไฟล์ PSD ตามการกำหนดค่าของคุณ

#### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่าเส้นทางได้รับการตั้งค่าอย่างถูกต้องเพื่อหลีกเลี่ยงข้อผิดพลาดไม่พบไฟล์
- ตรวจสอบว่า GroupDocs.Conversion ได้รับการติดตั้งและอ้างอิงอย่างถูกต้องในโครงการของคุณ
