## คู่มือการใช้งาน

หัวข้อนี้จะแนะนำคุณเกี่ยวกับการใช้งานคุณลักษณะการแปลงโดยใช้ GroupDocs.Conversion สำหรับ .NET

### โหลดและแปลงไฟล์ GIF

#### ภาพรวม
ฟังก์ชันหลักคือการโหลดไฟล์ GIF และกำหนดค่าให้แปลงเป็นรูปแบบ PSD มาแบ่งขั้นตอนออกเป็นแต่ละขั้นตอนกัน:

**1. กำหนดเส้นทาง**
ตั้งค่าไดเร็กทอรีอินพุตและเอาต์พุตของคุณ:
```csharp
string inputGifPath = "YOUR_DOCUMENT_DIRECTORY/sample.gif"; // แทนที่ด้วยเส้นทางจริงของคุณ
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
```

**2. สร้างเทมเพลตผลลัพธ์**
กำหนดค่าเทมเพลตการตั้งชื่อสำหรับไฟล์ที่แปลง:
```csharp
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");
Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

**3. เริ่มต้นตัวแปลง**
ใช้ `Converter` คลาสสำหรับโหลดไฟล์ GIF ของคุณ:
```csharp
using (Converter converter = new GroupDocs.Conversion.Converter(inputGifPath))
{
    // กำหนดค่าตัวเลือกการแปลงสำหรับรูปแบบ PSD
    ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd };
    
    // ดำเนินการแปลงจาก GIF เป็น PSD
    converter.Convert(getPageStream, options);
}
```

#### คำอธิบาย
- **`Converter Class`**: เริ่มต้นด้วยเส้นทาง GIF ต้นทาง
- **`ImageConvertOptions`**: ระบุว่ารูปแบบเอาต์พุตควรเป็น PSD สามารถตั้งค่าอื่นๆ ได้ที่นี่ตามความต้องการ
- **`converter.Convert()`**: ดำเนินการกระบวนการแปลงโดยใช้ตัวเลือกที่ระบุและตรรกะการจัดการสตรีม

#### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่าเส้นทาง GIF อินพุตถูกต้องและสามารถเข้าถึงได้
- ตรวจสอบสิทธิ์การเขียนสำหรับไดเร็กทอรีเอาท์พุต
- ตรวจสอบว่าคุณได้ติดตั้ง GroupDocs.Conversion เวอร์ชันที่ถูกต้องสำหรับ .NET หรือไม่

2. **สินทรัพย์ทางการตลาดดิจิตอล**:แปลงกราฟิกการตลาดให้เป็นรูปแบบคุณภาพสูงที่เหมาะสำหรับแคมเปญสื่อสิ่งพิมพ์และดิจิทัล
