## คู่มือการใช้งาน
หัวข้อนี้จะแนะนำคุณเกี่ยวกับการแปลงไฟล์ VTX เป็นรูปแบบ PSD โดยใช้ GroupDocs.Conversion
### การโหลดและการแปลงไฟล์
#### ภาพรวม
เรียนรู้วิธีโหลดไฟล์ .vtx และแปลงเป็นไฟล์ .psd หลายไฟล์ โดยแต่ละไฟล์จะสอดคล้องกับหน้าในเอกสารต้นฉบับ ซึ่งมีประโยชน์ในการเตรียมเทมเพลต Visio สำหรับงานออกแบบกราฟิกใน Photoshop
#### การดำเนินการแบบทีละขั้นตอน
**1. ตั้งค่าเส้นทาง**
```csharp
string documentPath = Path.Combine(@"YOUR_DOCUMENT_DIRECTORY", "sample.vtx");
string outputFolder = Path.Combine(@"YOUR_OUTPUT_DIRECTORY");
```
**2. กำหนดฟังก์ชั่นการสร้างสตรีม**
ฟังก์ชั่นนี้จะสร้างสตรีมใหม่สำหรับแต่ละเพจที่จะถูกแปลง:
```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext =>
{
    return new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
};
```
**3. โหลดและแปลงไฟล์ VTX**
โหลดไฟล์ VTX และระบุตัวเลือกการแปลง:
```csharp
using (Converter converter = new Converter(documentPath))
{
    ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Psd };
    converter.Convert(getPageStream, options);
}
```
**คำอธิบาย:**
- `SavePageContext`: ให้บริบทเกี่ยวกับหน้าที่ถูกแปลง
- `ImageConvertOptions`กำหนดค่าการตั้งค่าการแปลง โดยระบุ PSD เป็นรูปแบบเป้าหมาย
