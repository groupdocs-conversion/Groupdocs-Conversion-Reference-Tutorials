# แปลง DWF เป็น PSD ด้วย GroupDocs.Conversion สำหรับ .NET

## การแนะนำ

การแปลงไฟล์ DWF เป็นรูปแบบ PSD อเนกประสงค์เป็นความต้องการทั่วไปของสถาปนิกและนักออกแบบที่ต้องการรักษาการออกแบบที่มีคุณภาพสูงในขณะที่ใช้ซอฟต์แวร์ออกแบบกราฟิกเช่น Adobe Photoshop คำแนะนำที่ครอบคลุมนี้จะแนะนำคุณเกี่ยวกับการใช้ GroupDocs.Conversion สำหรับ .NET เพื่อแปลงไฟล์ DWF เป็น PSD อย่างมีประสิทธิภาพ

**สิ่งที่คุณจะได้เรียนรู้:**
- การตั้งค่าสภาพแวดล้อมของคุณด้วย GroupDocs.Conversion สำหรับ .NET
- คำแนะนำทีละขั้นตอนในการแปลงไฟล์ DWF เป็นรูปแบบ PSD
- เคล็ดลับในการระบุไดเรกทอรีเอาท์พุตระหว่างการแปลง

เริ่มต้นด้วยการครอบคลุมข้อกำหนดเบื้องต้นที่จำเป็นสำหรับกระบวนการนี้

- **ไลบรารีและเวอร์ชัน:** GroupDocs.Conversion สำหรับ .NET เวอร์ชัน 25.3.0 ขึ้นไป

## คู่มือการใช้งาน

มาสำรวจวิธีนำคุณลักษณะแต่ละอย่างไปใช้กัน

### โหลดและแปลง DWF เป็น PSD

#### ภาพรวม
ฟีเจอร์นี้ช่วยให้คุณโหลดไฟล์ DWF และแปลงเป็นรูปแบบ PSD ซึ่งใช้กันอย่างแพร่หลายในแอพพลิเคชันการออกแบบกราฟิก เช่น Adobe Photoshop

**ขั้นตอนที่ 1: กำหนดเส้นทางไฟล์**

ประการแรก ตั้งค่าเส้นทางเอกสารต้นฉบับและโฟลเดอร์ผลลัพธ์:

```csharp
string documentPath = @"YOUR_DOCUMENT_DIRECTORY\sample.dwf";
string outputFolder = @"YOUR_OUTPUT_DIRECTORY\";
```

**ขั้นตอนที่ 2: สร้างเทมเพลตไฟล์เอาท์พุต**

กำหนดเทมเพลตสำหรับการตั้งชื่อไฟล์ที่แปลง:

```csharp
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");
```

**ขั้นตอนที่ 3: จัดการสตรีมหน้า**

สร้างฟังก์ชั่นสำหรับจัดการสตรีมไฟล์ในระหว่างการแปลง:

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

**ขั้นตอนที่ 4: ตั้งค่าตัวเลือกการแปลงและดำเนินการ**

กำหนดค่าการตั้งค่าการแปลงสำหรับรูปแบบ PSD และดำเนินการแปลง:

```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd };

// ดำเนินการแปลงเป็น PSD
converter.Convert(getPageStream, options);
```

### บันทึกผลลัพธ์การแปลงไปยังไดเร็กทอรีที่ระบุ

#### ภาพรวม
คุณลักษณะนี้ช่วยให้คุณระบุไดเร็กทอรีเอาท์พุตที่จะบันทึกไฟล์ที่แปลงแล้ว

**ขั้นตอนที่ 1: กำหนดไดเรกทอรี**

ระบุเอกสารและไดเร็กทอรีผลลัพธ์ของคุณ:

```csharp
string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY\";
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY\";
```

**ขั้นตอนที่ 2: ใช้เทมเพลตไฟล์เอาท์พุต**

สร้างเส้นทางสำหรับเทมเพลตไฟล์เอาท์พุตเพื่อให้แน่ใจว่าไฟล์ได้รับการบันทึกไว้ในตำแหน่งที่กำหนด:

```csharp
string outputFileTemplate = Path.Combine(outputDirectory, "converted-page-{0}.psd");
```
