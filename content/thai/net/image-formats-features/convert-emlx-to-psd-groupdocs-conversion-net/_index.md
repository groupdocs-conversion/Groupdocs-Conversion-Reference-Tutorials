## คู่มือการใช้งาน

ตอนนี้เรามาแบ่งการใช้งานออกเป็นส่วนๆ ตามตรรกะกัน

### โหลดไฟล์ EMLX แหล่งที่มา

#### ภาพรวม
การโหลดไฟล์ EMLX เป็นขั้นตอนแรกในการเตรียมพร้อมสำหรับการแปลง ไลบรารี GroupDocs.Conversion มอบวิธีตรงไปตรงมาในการจัดการสิ่งนี้โดยใช้ `Converter` ระดับ.

#### การดำเนินการแบบทีละขั้นตอน

1. **เริ่มต้นตัวแปลง:**
   เริ่มต้นด้วยการสร้างอินสแตนซ์ของ `Converter` วัตถุ ส่งผ่านเส้นทางไปยังไฟล์ EMLX ของคุณ

   ```csharp
   string sourceFilePath = "path/to/your/sample.emlx";
   
   using (Converter converter = new Converter(sourceFilePath))
   {
       // ขั้นตอนการแปลงเพิ่มเติมจะตามมา
   }
   ```

2. **การทำความเข้าใจพารามิเตอร์:**
   - `sourceFilePath`: เส้นทางไปยังไฟล์ EMLX ของคุณ ตรวจสอบให้แน่ใจว่าได้ระบุอย่างถูกต้องเพื่อหลีกเลี่ยงข้อผิดพลาดในการโหลด

### ตั้งค่าตัวเลือกการแปลงสำหรับรูปแบบ PSD

#### ภาพรวม
หากต้องการแปลงไฟล์เป็นรูปแบบ PSD ที่ต้องการ ให้ระบุตัวเลือกการแปลงที่ปรับแต่งเอาต์พุตตามความต้องการของคุณ

#### การดำเนินการแบบทีละขั้นตอน

1. **กำหนดไดเรกทอรีเอาท์พุตและเทมเพลตการตั้งชื่อ:**

   ```csharp
   string outputFolder = "path/to/output/directory";
   string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");
   ```

2. **สร้างฟังก์ชั่นตัวจัดการสตรีมหน้า:**
   ฟังก์ชันนี้จัดการวิธีการแปลงไฟล์ EMLX แต่ละหน้าเป็นไฟล์ PSD แยกกัน

   ```csharp
   Func<SavePageContext, Stream> getPageStream = savePageContext =>
       new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
   ```

3. **กำหนดค่าตัวเลือกการแปลงภาพ:**
   ตั้งค่ารูปแบบการแปลงเป็น PSD โดยใช้ `ImageConvertOptions`-

   ```csharp
   ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd };
   ```

### แปลง EMLX เป็นรูปแบบ PSD

#### ภาพรวม
เมื่อตั้งค่าทุกอย่างเสร็จเรียบร้อยแล้ว คุณสามารถดำเนินการแปลงจริงจาก EMLX เป็น PSD ได้

#### การดำเนินการแบบทีละขั้นตอน

1. **ดำเนินการแปลง:**
   ใช้ `Convert` วิธีการของ `Converter` วัตถุที่ส่งผ่านตัวจัดการสตรีมและตัวเลือกของคุณ

   ```csharp
   using (Converter converter = new Converter(sourceFilePath))
   {
       // ดำเนินการแปลง
       converter.Convert(getPageStream, options);
   }
   ```

2. **การทำความเข้าใจพารามิเตอร์:**
   - `getPageStream`: ฟังก์ชั่นที่กำหนดวิธีการบันทึกไฟล์เอาท์พุต
   - `options`: การตั้งค่าสำหรับการแปลงเป็น PSD
