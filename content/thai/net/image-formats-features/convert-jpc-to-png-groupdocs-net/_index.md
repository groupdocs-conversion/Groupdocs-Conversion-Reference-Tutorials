## คู่มือการใช้งาน

### คุณสมบัติ 1: โหลดไฟล์ JPC
**ภาพรวม:** การโหลดไฟล์ JPC ต้นฉบับจะตั้งค่าเอกสารสำหรับการแปลง PNG

#### การดำเนินการทีละขั้นตอน:
1. **เริ่มต้นคลาสตัวแปลง**
   สร้างอินสแตนซ์ของ `Converter` คลาสที่จะโหลดไฟล์ JPC ของคุณ
   ```csharp
   string jpcFilePath = "YOUR_DOCUMENT_DIRECTORY/SAMPLE_JPC";
   using (Converter converter = new Converter(jpcFilePath))
   {
       // ไฟล์ JPC พร้อมสำหรับการแปลงแล้ว
   }
   ```
2. **การกำจัดไฟล์**
   ใช้ `using` คำสั่งให้จัดการการกำจัดทรัพยากรโดยอัตโนมัติ

### คุณสมบัติที่ 2: ตั้งค่าตัวเลือกการแปลง PNG
**ภาพรวม:** การตั้งค่าตัวเลือกที่ถูกต้องจะช่วยให้การแปลงเป็นรูปแบบ PNG ถูกต้อง

#### การดำเนินการทีละขั้นตอน:
1. **สร้าง ImageConvertOptions**
   กำหนดค่าการตั้งค่าการแปลงด้วย `ImageConvertOptions`-
   ```csharp
   using GroupDocs.Conversion.Options.Convert;

   ImageConvertOptions options = new ImageConvertOptions { Format = FileTypes.ImageFileType.Png };
   ```
2. **ตัวเลือกการกำหนดค่า**
   ตั้งค่ารูปแบบเป็น PNG เพื่อการแปลงไฟล์ JPC ทั้งหมดอย่างแม่นยำ

### คุณสมบัติที่ 3: แปลง JPC เป็น PNG
**ภาพรวม:** ดำเนินการแปลงจริงโดยใช้ตัวเลือกที่ตั้งไว้ก่อนหน้านี้

#### การดำเนินการทีละขั้นตอน:
1. **กำหนดเทมเพลตไฟล์เอาท์พุต**
   ใช้หลักการตั้งชื่อสำหรับไฟล์เอาต์พุต
   ```csharp
   string outputFolder = "YOUR_OUTPUT_DIRECTORY";
   string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.png");
   ```
2. **ฟังก์ชั่นการสร้างสตรีม**
   นำฟังก์ชันแลมบ์ดาไปใช้เพื่อสร้างสตรีมไฟล์สำหรับแต่ละหน้า
   ```csharp
   Func<SavePageContext, Stream> getPageStream = savePageContext => 
       new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
   ```
3. **ดำเนินการแปลง**
   ดำเนินการแปลงด้วย `Converter` อินสแตนซ์และตัวเลือก
   ```csharp
   using (Converter converter = new Converter(jpcFilePath))
   {
       converter.Convert(getPageStream, options);
   }
   ```

**เคล็ดลับการแก้ไขปัญหา:**
- ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ถูกต้องเพื่อหลีกเลี่ยง `FileNotFoundException`-
- ตรวจสอบสิทธิ์การเขียนในไดเร็กทอรีเอาต์พุต
