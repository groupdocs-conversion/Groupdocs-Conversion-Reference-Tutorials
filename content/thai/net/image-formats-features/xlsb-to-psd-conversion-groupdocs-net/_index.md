## คู่มือการใช้งาน

### โหลดไฟล์ XLSB

**ภาพรวม:**
เริ่มต้นด้วยการโหลดไฟล์ XLSB ของคุณโดยใช้ `Converter` คลาสสำหรับการเข้าถึงและแปลงเนื้อหา

#### ทีละขั้นตอน:
1. **ตัวแปลงการเริ่มต้น**: สร้างอินสแตนซ์ใหม่ของ `Converter` ด้วยเส้นทางไฟล์ของคุณ
    ```csharp
    string documentPath = "YOUR_DOCUMENT_DIRECTORY/SAMPLE_XLSB";
    using (Converter converter = new Converter(documentPath))
    {
        // โค้ดการแปลงจะอยู่ที่นี่
    }
    ```
2. **กำจัดทรัพยากร**:ให้แน่ใจว่าทรัพยากรจะได้รับการปลดปล่อยหลังการใช้งาน

### ตั้งค่าตัวเลือกการแปลงสำหรับรูปแบบ PSD

**ภาพรวม:**
กำหนดค่าการตั้งค่าการแปลงที่เฉพาะเจาะจงกับรูปแบบ PSD โดยใช้ `ImageConvertOptions`-

#### ทีละขั้นตอน:
1. **สร้าง ImageConvertOptions**: ระบุเอาท์พุตเป็น PSD
    ```csharp
    using GroupDocs.Conversion.Options.Convert;

    ImageConvertOptions options = new ImageConvertOptions 
    {
        Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd
    };
    ```
2. **คำอธิบายพารามิเตอร์**- `Format` กำหนดรูปแบบการแปลงเป้าหมายตั้งค่าเป็น 'Psd'

### แปลง XLSB เป็น PSD

**ภาพรวม:**
ดำเนินการแปลงไฟล์โดยให้แน่ใจว่าแต่ละหน้าได้รับการบันทึกเป็น PSD แยกกันในไดเร็กทอรีเอาต์พุตของคุณ

#### ทีละขั้นตอน:
1. **ตั้งค่าไดเร็กทอรีและเทมเพลตเอาท์พุต**: กำหนดว่าไฟล์ที่แปลงจะถูกจัดเก็บไว้ที่ไหน
    ```csharp
    string outputFolder = "YOUR_OUTPUT_DIRECTORY";
    string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");
    ```
2. **สร้างฟังก์ชั่นสตรีม**: สร้างเส้นทางไฟล์แบบไดนามิกสำหรับแต่ละหน้า
    ```csharp
    Func<SavePageContext, Stream> getPageStream = savePageContext => 
        new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
    ```
3. **ดำเนินการแปลง**: ใช้ `Converter` อินสแตนซ์และตัวเลือกในการแปลงไฟล์ของคุณ
    ```csharp
    using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_XLSB"))
    {
        ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd };
        converter.Convert(getPageStream, options);
    }
    ```

### เคล็ดลับการแก้ไขปัญหา
- **ปัญหาเส้นทางไฟล์**:ให้แน่ใจว่าเส้นทางสำหรับเอกสารถูกต้อง
- **การจัดการหน่วยความจำ**:กำจัดทรัพยากรเพื่อป้องกันการรั่วไหลของหน่วยความจำ
