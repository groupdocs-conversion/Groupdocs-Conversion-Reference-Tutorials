### โหลดและแปลงไฟล์ POTM

#### ภาพรวม
เราจะโหลดไฟล์ POTM และแปลงเป็นรูปแบบ DOC โดยใช้ GroupDocs.Conversion กระบวนการนี้เกี่ยวข้องกับการระบุตัวเลือกการแปลงและการดำเนินการวิธีการแปลง

#### การดำเนินการแบบทีละขั้นตอน

**1. กำหนดเส้นทางเอกสาร**
เริ่มต้นด้วยการกำหนดเส้นทางสำหรับไฟล์ POTM อินพุตและไฟล์ DOC เอาท์พุต:

```csharp
const string DocumentDirectory = "YOUR_DOCUMENT_DIRECTORY";
const string OutputDirectory = "YOUR_OUTPUT_DIRECTORY/";
```

**2. เริ่มต้นตัวแปลง**
โหลดไฟล์ POTM ลงในอ็อบเจ็กต์ GroupDocs.Conversion:

```csharp
using (var converter = new GroupDocs.Conversion.Converter(Path.Combine(DocumentDirectory, "sample.potm")))
{
    // ตรรกะการแปลงจะอยู่ที่นี่
}
```

**3. ตั้งค่าตัวเลือกการแปลง**
ระบุตัวเลือกในการแปลงเอกสารเป็นรูปแบบ DOC:

```csharp
var options = new WordProcessingConvertOptions
{
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc
};
```

**4. ดำเนินการแปลง**
ดำเนินการแปลงและบันทึกไฟล์เอาท์พุต:

```csharp
string outputFile = Path.Combine(OutputDirectory, "potm-converted-to.doc");
converter.Convert(outputFile, options);
```
