## คู่มือการใช้งาน
ตอนนี้เรามาดูวิธีการใช้งานการแปลงไฟล์ HTML เป็นรูปแบบ DOCX กัน

### การโหลดและการแปลงไฟล์ HTM เป็นรูปแบบ DOCX
#### ภาพรวม
ฟีเจอร์นี้ช่วยให้คุณโหลดไฟล์ HTM และแปลงเป็นเอกสาร Word โดยใช้ GroupDocs.Conversion กระบวนการนี้มีประโยชน์สำหรับวัตถุประสงค์ด้านเอกสารหรือการรวมเนื้อหาเว็บกับแอปพลิเคชัน Office
#### การดำเนินการแบบทีละขั้นตอน
##### 1. ตั้งค่าเส้นทางไดเรกทอรี
กำหนดเส้นทางสำหรับไฟล์ HTML อินพุตและไฟล์ DOCX เอาท์พุต:
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY"; // แทนที่ด้วยเส้นทางไดเร็กทอรีจริง
string outputDirectory = "YOUR_OUTPUT_DIRECTORY"; // แทนที่ด้วยเส้นทางไดเร็กทอรีจริง
```
**คำอธิบาย**:ตัวแปรเหล่านี้จะจัดเก็บเส้นทางไปยังไฟล์ HTML ต้นฉบับของคุณ และตำแหน่งที่คุณต้องการบันทึกเอกสาร Word ที่แปลงแล้ว
##### 2. กำหนดเส้นทางไฟล์ต้นทางและปลายทางเอาท์พุต
```csharp
// กำหนดเส้นทางไฟล์ HTM ต้นทาง
cstring sourceFilePath = Path.Combine(documentDirectory, "sample.htm"); // แทนที่ 'sample.htm' ด้วยชื่อไฟล์จริงของคุณ

// กำหนดเส้นทางไฟล์ DOCX เอาท์พุต
cstring outputFile = Path.Combine(outputDirectory, "htm-converted-to.docx");
```
**คำอธิบาย**: ใช้ `Path.Combine` เพื่อเชื่อมโยงไดเรกทอรีและเส้นทางชื่อไฟล์อย่างน่าเชื่อถือระหว่างระบบปฏิบัติการที่แตกต่างกัน
##### 3. แปลง HTM เป็น DOCX
โหลดไฟล์ต้นฉบับและดำเนินการแปลง:
```csharp
using (var converter = new Converter(sourceFilePath))
{
    // เริ่มต้นตัวเลือกการแปลงสำหรับรูปแบบ WordProcessing (DOCX)
    var options = new GroupDocs.Conversion.Options.Convert.WordProcessingConvertOptions();
    
    // ดำเนินการแปลงและบันทึกไฟล์ DOCX
    converter.Convert(outputFile, options);
}
```
**คำอธิบาย**: เดอะ `Converter` คลาสจัดการการโหลดไฟล์ HTML โดยระบุ `WordProcessingConvertOptions`คุณกำหนดว่ารูปแบบผลลัพธ์ควรจะเป็นเอกสาร Word
##### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่าเส้นทางไดเร็กทอรีของคุณถูกต้องและสามารถเข้าถึงได้
- ตรวจสอบว่าไม่มีสิทธิ์ใด ๆ บนไดเร็กทอรีหรือไฟล์หรือไม่
- ตรวจสอบว่าเวอร์ชันของ GroupDocs.Conversion เข้ากันได้กับการตั้งค่าโครงการของคุณ
