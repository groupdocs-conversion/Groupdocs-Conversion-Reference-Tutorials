## คู่มือการใช้งาน

ในส่วนนี้ เราจะอธิบายกระบวนการแปลงไฟล์ MPX เป็นเอกสาร DOC โดยใช้ GroupDocs.Conversion สำหรับ .NET

### ภาพรวมคุณลักษณะ: การแปลง MPX เป็น DOC

ฟีเจอร์นี้ใช้ประโยชน์จากความสามารถอันแข็งแกร่งของ GroupDocs.Conversion เพื่อแปลงไฟล์ข้อมูลโครงการจากรูปแบบ MPX ให้เป็นเอกสาร Word ที่แก้ไขได้ มาดูขั้นตอนการใช้งานแต่ละขั้นตอนกัน:

#### ขั้นตอนที่ 1: กำหนดไดเรกทอรีอินพุตและเอาต์พุต

เริ่มต้นด้วยการระบุตำแหน่งไฟล์ MPX ต้นฉบับของคุณ และตำแหน่งที่คุณต้องการบันทึกไฟล์ DOC ที่แปลงแล้ว

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY"; // แทนที่ด้วยเส้นทางไดเร็กทอรีจริง
string outputFileDirectory = "YOUR_OUTPUT_DIRECTORY"; // แทนที่ด้วยเส้นทางไดเร็กทอรีจริง

string sourceMpxFilePath = Path.Combine(documentDirectory, "sample.mpx"); 
string outputFile = Path.Combine(outputFileDirectory, "mpx-converted-to.doc");
```

#### ขั้นตอนที่ 2: เริ่มต้นตัวแปลง

สร้างอินสแตนซ์ของ `Converter` คลาสโดยใช้ไฟล์ MPX ของคุณ

```csharp
using (var converter = new GroupDocs.Conversion.Converter(sourceMpxFilePath))
{
    // ตรรกะการแปลงจะถูกเพิ่มที่นี่
}
```

#### ขั้นตอนที่ 3: ตั้งค่าตัวเลือกการแปลงสำหรับรูปแบบ DOC

กำหนดตัวเลือกการแปลงเพื่อระบุรูปแบบเป้าหมายเป็น DOC

```csharp
WordProcessingConvertOptions options = new WordProcessingConvertOptions 
{ 
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc 
};
```

#### ขั้นตอนที่ 4: ดำเนินการแปลง

ใช้ `Convert` วิธีการของอินสแตนซ์ตัวแปลง โดยส่งผ่านเส้นทางไฟล์เอาต์พุตและตัวเลือกการแปลง

```csharp
converter.Convert(outputFile, options);
```
