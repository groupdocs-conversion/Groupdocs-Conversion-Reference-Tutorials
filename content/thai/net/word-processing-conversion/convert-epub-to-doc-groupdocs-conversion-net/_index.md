## คู่มือการใช้งาน

### โหลดและแปลงไฟล์ EPUB เป็นรูปแบบ DOC

#### ภาพรวม

การแปลงไฟล์ EPUB เป็นรูปแบบเอกสาร Microsoft Word (.doc) ช่วยให้แก้ไขและแบ่งปันเนื้อหาข้อความได้อย่างง่ายดาย ดังต่อไปนี้:

##### 1. กำหนดไดเรกทอรี

ระบุว่าไฟล์ EPUB อินพุตและไฟล์ DOC เอาต์พุตของคุณจะอยู่ที่ไหน

```csharp
string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY";

// ตรวจสอบให้แน่ใจว่ามีไดเร็กทอรีเอาต์พุตอยู่หรือสร้างขึ้นใหม่
if (!Directory.Exists(outputDirectory))
{
    Directory.CreateDirectory(outputDirectory);
}
```

##### 2. ระบุเส้นทางไฟล์

ระบุไฟล์ EPUB ต้นทางของคุณและเส้นทางเอาต์พุต DOC ที่ต้องการ

```csharp
string inputFile = Path.Combine(documentDirectory, "sample.epub");
string outputFile = Path.Combine(outputDirectory, "epub-converted-to.doc");
```

##### 3. เริ่มต้นตัวแปลง

ตั้งค่ากระบวนการแปลงโดยใช้ GroupDocs.Conversion

```csharp
using (var converter = new Converter(inputFile))
{
    var options = new WordProcessingConvertOptions { Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc };
    converter.Convert(outputFile, options);
}
```

**คำอธิบาย**- 
- `Converter`: เริ่มต้นด้วยไฟล์อินพุต
- `WordProcessingConvertOptions`: ระบุการตั้งค่าการแปลง
- `Format`กำหนดเอาท์พุตเป็น DOC
