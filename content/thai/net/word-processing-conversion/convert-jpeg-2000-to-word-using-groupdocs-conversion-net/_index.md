## คู่มือการใช้งาน
### โหลดไฟล์ J2C ต้นฉบับ
**ภาพรวม:** ฟีเจอร์นี้ช่วยให้คุณโหลดไฟล์ภาพ JPEG 2000 เพื่อเตรียมการแปลง

#### ทีละขั้นตอน:
1. **ระบุเส้นทางไฟล์ J2C:**
   ตั้งค่าเส้นทางที่ไฟล์ J2C ต้นทางของคุณตั้งอยู่:

   ```csharp
   string j2cFilePath = "YOUR_DOCUMENT_DIRECTORY/sample.j2c";
   ```

2. **เริ่มต้นวัตถุตัวแปลง:**
   สร้าง `Converter` อินสแตนซ์สำหรับจัดการกระบวนการแปลง:

   ```csharp
   using (var converter = new Converter(j2cFilePath))
   {
       // ตรรกะการแปลงจะถูกดำเนินการที่นี่
   }
   ```

3. **อธิบายพารามิเตอร์และวิธีการ:**
ผู้สร้างของ `Converter` คลาสใช้เส้นทางไฟล์เพื่อเริ่มต้นวัตถุสำหรับการดำเนินการครั้งต่อไป

### แปลง J2C เป็นรูปแบบ DOCX
**ภาพรวม:** คุณลักษณะนี้จะแปลงไฟล์ J2C ที่คุณโหลดไว้เป็นรูปแบบ Microsoft Word Open XML Document (.docx)

#### ทีละขั้นตอน:
1. **ตั้งค่าไดเรกทอรีผลลัพธ์และชื่อไฟล์:**
   กำหนดว่าคุณต้องการบันทึกเอกสารที่แปลงแล้วที่ไหน:

   ```csharp
   string outputFolder = "YOUR_OUTPUT_DIRECTORY";
   string outputFile = Path.Combine(outputFolder, "j2c-converted-to.docx");
   ```

2. **ระบุตัวเลือกการแปลง:**
   ใช้ `WordProcessingConvertOptions` สำหรับการแปลง DOCX:

   ```csharp
   var options = new WordProcessingConvertOptions();
   ```

3. **ดำเนินการแปลง:**
   แปลงและบันทึกไฟล์เอาท์พุตไปยังเส้นทางที่คุณระบุ:

   ```csharp
   using (var converter = new Converter(j2cFilePath))
   {
       converter.Convert(outputFile, options);
   }
   ```

4. **เคล็ดลับการแก้ไขปัญหา:**
   - ตรวจสอบให้แน่ใจว่าเส้นทางได้รับการตั้งค่าอย่างถูกต้องเพื่อหลีกเลี่ยง `FileNotFoundException`-
   - ตรวจสอบว่าไลบรารี GroupDocs.Conversion ได้รับการติดตั้งอย่างถูกต้องหรือไม่
