## คู่มือการใช้งาน

### โหลดไฟล์ XLT แหล่งที่มา

**ภาพรวม:**
การโหลดไฟล์ XLT เป็นขั้นตอนแรกในการเตรียมไฟล์สำหรับการแปลง ฟีเจอร์นี้ใช้ GroupDocs.Conversion `Converter` คลาสสำหรับโหลดไฟล์อย่างมีประสิทธิภาพ

#### การดำเนินการทีละขั้นตอน:

**1. กำหนดเส้นทางไฟล์และเริ่มต้นตัวแปลง:**

```csharp
using System;
using GroupDocs.Conversion;

namespace FeatureLoadSourceXltFile
{
    public static class LoadXltFile
    {
        public static void Execute()
        {
            // แทนที่ด้วยเส้นทางไดเร็กทอรีเอกสารจริงของคุณ
            string sampleXltPath = "YOUR_DOCUMENT_DIRECTORY\\sample.xlt";

            // เริ่มต้นวัตถุ Converter ด้วยเส้นทางไฟล์ XLT
            using (var converter = new GroupDocs.Conversion.Converter(sampleXltPath))
            {
                Console.WriteLine("Source XLT file loaded successfully.");
            }
        }
    }
}
```

**คำอธิบาย:** การ `Converter` คลาสนี้ใช้สำหรับโหลดไฟล์ XLT ซึ่งจะตั้งค่าสภาพแวดล้อมของเราสำหรับขั้นตอนการแปลงในลำดับถัดไป

### แปลง XLT เป็นรูปแบบ DOC

**ภาพรวม:**
การแปลงไฟล์ XLT เป็นรูปแบบ DOC ช่วยให้ผู้ใช้สามารถใช้ประโยชน์จากคุณลักษณะข้อความที่หลากหลายของ Word ได้ ทำให้แก้ไขและแบ่งปันเอกสารได้ง่ายยิ่งขึ้น

#### การดำเนินการทีละขั้นตอน:

**1. กำหนดเส้นทางผลลัพธ์และตัวเลือกการแปลง:**

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

namespace FeatureConvertXltToDocFormat
{
    public static class ConvertXltToDoc
    {
        public static void Execute()
        {
            // กำหนดเส้นทางไดเรกทอรีเอาท์พุต
            string outputFolder = "YOUR_OUTPUT_DIRECTORY";
            string outputFile = Path.Combine(outputFolder, "xlt-converted-to.doc");

            // แทนที่ด้วยเส้นทางไดเร็กทอรีเอกสารจริงของคุณสำหรับไฟล์ XLT ต้นฉบับ
            string sampleXltPath = "YOUR_DOCUMENT_DIRECTORY\\sample.xlt";

            // เริ่มต้นวัตถุ Converter ด้วยเส้นทางไฟล์ XLT
            using (var converter = new GroupDocs.Conversion.Converter(sampleXltPath))
            {
                // ตั้งค่าตัวเลือกการแปลงสำหรับการประมวลผลคำรูปแบบ DOC
                WordProcessingConvertOptions options = new WordProcessingConvertOptions 
                { 
                    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc 
                };

                // ดำเนินการแปลงและบันทึกไฟล์ DOC เอาท์พุต
                converter.Convert(outputFile, options);

                Console.WriteLine("Conversion to DOC completed successfully.");
            }
        }
    }
}
```

**คำอธิบาย:** เราตั้งไว้ตรงนี้ `WordProcessingConvertOptions` เพื่อระบุรูปแบบเป้าหมายเป็น DOC จากนั้นกระบวนการแปลงจะดำเนินการโดยใช้การตั้งค่าเหล่านี้
