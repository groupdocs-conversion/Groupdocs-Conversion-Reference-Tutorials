## คู่มือการใช้งาน

### แปลงไฟล์ JPEG 2000 เป็นรูปแบบ DOCX

#### ภาพรวม
เรียนรู้วิธีการแปลงรูปภาพ JPEG 2000 (JP2) เป็นเอกสาร Word ในรูปแบบ DOCX ได้อย่างราบรื่น เหมาะสำหรับการแก้ไขหรือบูรณาการข้อมูลภาพกับเนื้อหาข้อความ

##### ขั้นตอนที่ 1: ตั้งค่าเส้นทางไฟล์
กำหนดเส้นทางสำหรับไฟล์ JP2 และเอาต์พุต DOCX ของคุณ:
```csharp
string sourceFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.jp2");
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "jp2-converted-to.docx");
```

##### ขั้นตอนที่ 2: โหลดไฟล์ต้นฉบับ
เริ่มต้นการใช้งาน `Converter` วัตถุ:
```csharp
using (var converter = new Converter(sourceFilePath))
{
    // ดำเนินการตั้งค่าตัวเลือกการแปลง
}
```

##### ขั้นตอนที่ 3: กำหนดค่าตัวเลือกการแปลง
ตั้งค่าตัวเลือกการแปลงการประมวลผลคำสำหรับรูปแบบ DOCX:
```csharp
var options = new WordProcessingConvertOptions();
```

##### ขั้นตอนที่ 4: ดำเนินการแปลง
ดำเนินการและบันทึกการแปลง:
```csharp
converter.Convert(outputFile, options);
```
