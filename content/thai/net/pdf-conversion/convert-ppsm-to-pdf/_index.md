## นำเข้าเนมสเปซ
ก่อนที่จะเจาะลึกกระบวนการแปลง ตรวจสอบให้แน่ใจว่าคุณได้นำเข้าเนมสเปซที่จำเป็นเพื่อเข้าถึงฟังก์ชัน GroupDocs.Conversion

```csharp
using System;
using System.IO;
using GroupDocs.Conversion.Options.Convert;
```
ตอนนี้เราได้กำหนดขั้นตอนแล้ว เรามาเจาะลึกกระบวนการทีละขั้นตอนในการแปลงไฟล์ PPSM (PowerPoint Slide Show) เป็น PDF โดยใช้ GroupDocs.Conversion สำหรับ .NET
## 1. กำหนดโฟลเดอร์เอาท์พุตและเส้นทางไฟล์
```csharp
string outputFolder = "Your Document Directory";
string outputFile = Path.Combine(outputFolder, "ppsm-converted-to.pdf");
```
## 2. โหลดไฟล์ PPSM ต้นทาง
```csharp
using (var converter = new GroupDocs.Conversion.Converter(Constants.SAMPLE_PPSM))
{
    var options = new PdfConvertOptions();
```
## 3. กำหนดค่าตัวเลือกการแปลง (ไม่บังคับ)
คุณสามารถกำหนดค่าตัวเลือกการแปลงต่างๆ ได้ ขึ้นอยู่กับความต้องการของคุณ ในตัวอย่างนี้ เรากำลังแปลงเป็น PDF โดยไม่มีตัวเลือกเฉพาะใดๆ
## 4. ทำการแปลง
```csharp
// บันทึกไฟล์ PDF ที่แปลงแล้ว
converter.Convert(outputFile, options);
```
## 5. จัดการการแปลงให้เสร็จสิ้น
```csharp
Console.WriteLine("\nConversion to pdf completed successfully. \nCheck output in {0}", outputFolder);
```
