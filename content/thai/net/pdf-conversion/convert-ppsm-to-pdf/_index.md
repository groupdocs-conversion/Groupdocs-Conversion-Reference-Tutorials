## นำเข้าเนมสเปซ
ก่อนจะเจาะลึกลงไปในกระบวนการแปลง ให้แน่ใจว่าคุณได้นำเข้าเนมสเปซที่จำเป็นเพื่อเข้าถึงฟังก์ชันการทำงานของ GroupDocs.Conversion

```csharp
using System;
using System.IO;
using GroupDocs.Conversion.Options.Convert;
```

## 1. กำหนดโฟลเดอร์ผลลัพธ์และเส้นทางไฟล์
```csharp
string outputFolder = "Your Document Directory";
string outputFile = Path.Combine(outputFolder, "ppsm-converted-to.pdf");
```
## 2. โหลดไฟล์ PPSM ต้นฉบับ
```csharp
using (var converter = new GroupDocs.Conversion.Converter(Constants.SAMPLE_PPSM))
{
    var options = new PdfConvertOptions();
```
## 3. กำหนดค่าตัวเลือกการแปลง (ทางเลือก)
คุณสามารถกำหนดค่าตัวเลือกการแปลงต่างๆ ได้ตามความต้องการของคุณ ในตัวอย่างนี้ เราจะแปลงเป็น PDF โดยไม่ระบุตัวเลือกเฉพาะใดๆ
## 4. ดำเนินการแปลง
```csharp
// บันทึกไฟล์ PDF ที่แปลงแล้ว
converter.Convert(outputFile, options);
```
## 5. จัดการการแปลงให้เสร็จสมบูรณ์
```csharp
Console.WriteLine("\nConversion to pdf completed successfully. \nCheck output in {0}", outputFolder);
```
