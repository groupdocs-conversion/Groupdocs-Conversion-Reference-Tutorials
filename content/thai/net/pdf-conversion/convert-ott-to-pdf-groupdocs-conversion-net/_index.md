## คู่มือการใช้งาน

### โหลดและแปลง OTT เป็น PDF

#### ภาพรวม
หัวข้อนี้จะกล่าวถึงวิธีการโหลดไฟล์ Open Document Template (OTT) และแปลงเป็น Portable Document Format (PDF)

**ขั้นตอนที่ 1: กำหนดไดเรกทอรีผลลัพธ์**
ระบุตำแหน่งที่จะจัดเก็บไฟล์ที่แปลงแล้ว:

```csharp
using System.IO;

string outputFolder = "YOUR_OUTPUT_DIRECTORY";

// ตรวจสอบให้แน่ใจว่ามีไดเร็กทอรีอยู่หรือสร้างขึ้นใหม่ถ้าไม่มี
if (!Directory.Exists(outputFolder))
{
    Directory.CreateDirectory(outputFolder);
}
```

**ขั้นตอนที่ 2: เริ่มต้นตัวแปลงและแปลง**
ใช้ GroupDocs.Conversion เพื่อโหลดไฟล์ OTT ของคุณและแปลงมัน:

```csharp
string outputFile = Path.Combine(outputFolder, "ott-converted-to.pdf");

using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_OTT"))
{
    var options = new PdfConvertOptions();
    converter.Convert(outputFile, options);
}
```

**คำอธิบาย:**
- **คลาสตัวแปลง**:เริ่มต้นด้วยเส้นทางไปยังไฟล์ OTT ของคุณ
- **ตัวเลือกการแปลง PDF**: กำหนดค่าการตั้งค่าเฉพาะ PDF สำหรับการแปลง ปรับแต่งตามต้องการ

### กำหนดค่าไดเรกทอรีเอาท์พุต
การตั้งค่าไดเร็กทอรีเอาต์พุตจะทำให้แน่ใจว่าไฟล์ที่แปลงแล้วจะถูกเก็บไว้ในตำแหน่งที่เฉพาะเจาะจง หลีกเลี่ยงความยุ่งวุ่นวายและการเขียนทับ

**ขั้นตอนที่ 1: ตรวจสอบและสร้างไดเรกทอรี**
ตรวจสอบให้แน่ใจว่ามีไดเร็กทอรีอยู่:

```csharp
if (!Directory.Exists(outputFolder))
{
    Directory.CreateDirectory(outputFolder);
}
```

### ใช้ตัวเลือกการแปลง
การปรับแต่งตัวเลือกการแปลงทำให้คุณปรับแต่งผลลัพธ์ PDF ตามข้อกำหนดเฉพาะเช่นระยะขอบหรือขนาดหน้า

**ขั้นตอนที่ 1: ปรับแต่ง PdfConvertOptions**
ปรับเปลี่ยนการตั้งค่าโดยใช้ `PdfConvertOptions`-

```csharp
var options = new PdfConvertOptions()
{
    MarginTop = 10,
    MarginBottom = 10,
    PageSize = PageSize.A4 // ตั้งค่าขนาดหน้ากระดาษที่ต้องการ
};
```
