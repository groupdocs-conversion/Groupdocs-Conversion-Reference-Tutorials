## การตั้งค่า GroupDocs.Conversion สำหรับ .NET

### การเริ่มต้นและการตั้งค่าเบื้องต้น

เริ่มต้นด้วยการนำเข้าเนมสเปซที่จำเป็นลงในแอปพลิเคชัน C# ของคุณ:
```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;
```
ขั้นตอนต่อไป ให้เริ่มกระบวนการแปลงของคุณด้วยการตั้งค่านี้:
```csharp
string sourceFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.png");
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "png-converted-to.pdf");

using (var converter = new GroupDocs.Conversion.Converter(sourceFilePath))
{
    var options = new PdfConvertOptions();
    converter.Convert(outputFile, options);
}
```
ตัวอย่างโค้ดนี้สาธิตการโหลดไฟล์ PNG และแปลงเป็น PDF ที่เส้นทางเอาต์พุตที่ระบุ มาสำรวจฟีเจอร์นี้โดยละเอียดกัน

การแปลงไฟล์รูปภาพ (PNG) ให้เป็นเอกสาร PDF ที่มีประโยชน์นั้นมีความสำคัญต่อการแบ่งปันและการพิมพ์ โดยมีขั้นตอนดังนี้:

#### ขั้นตอนที่ 1: กำหนดเส้นทาง
ตั้งค่าเส้นทางไฟล์ต้นทางและไดเร็กทอรีเอาต์พุตดังนี้:
```csharp
string sourceFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.png");
string outputFile = Path.Combine("YOUR_OUTPUT_DIRECTORY", "png-converted-to.pdf");
```
#### ขั้นตอนที่ 2: โหลดไฟล์ต้นฉบับ
ใช้ GroupDocs.Conversion เพื่อโหลดไฟล์ PNG ของคุณ:
```csharp
using (var converter = new GroupDocs.Conversion.Converter(sourceFilePath))
{
    // ตัวเลือกการแปลงจะถูกตั้งค่าที่นี่
}
```
#### ขั้นตอนที่ 3: ตั้งค่าตัวเลือกการแปลง
สร้างอินสแตนซ์ของ `PdfConvertOptions` สำหรับการตั้งค่า PDF ในระหว่างการแปลง
```csharp
var options = new PdfConvertOptions();
```
#### ขั้นตอนที่ 4: ดำเนินการแปลง
ดำเนินการแปลงจริงด้วยตัวเลือกที่กำหนด:
```csharp
converter.Convert(outputFile, options);
```
