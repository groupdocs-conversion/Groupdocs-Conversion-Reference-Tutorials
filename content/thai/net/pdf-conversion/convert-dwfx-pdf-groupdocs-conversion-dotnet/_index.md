## คู่มือการใช้งาน

ตอนนี้คุณพร้อมแล้ว มาเริ่มกระบวนการแปลงกันเลย

### การแปลง DWFX เป็น PDF

หัวข้อนี้จะแนะนำคุณเกี่ยวกับการแปลงไฟล์ Design Web Format XPS (.dwfx) เป็น Portable Document Format (.pdf)

#### ขั้นตอนที่ 1: โหลดไฟล์ DWFX ของคุณ

เริ่มต้นด้วยการโหลดไฟล์ DWFX ของคุณโดยใช้ `Converter` คลาส นี่คือที่ที่เราระบุเอกสารอินพุต
```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main()
    {
        // เริ่มต้นตัวจัดการการแปลงด้วยเส้นทางไฟล์ DWFX
        Converter converter = new Converter("path/to/your/file.dwfx");
        
        Console.WriteLine("DWFX file loaded successfully!");
    }
}
```
#### ขั้นตอนที่ 2: ตั้งค่าตัวเลือกการแปลง PDF

ถัดไป ให้กำหนดรูปแบบผลลัพธ์ของคุณโดยระบุ `PdfConvertOptions`วิธีนี้ช่วยให้คุณกำหนดค่าพารามิเตอร์ต่างๆ ให้กับ PDF ที่ได้
```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main()
    {
        // เริ่มต้นตัวจัดการการแปลงด้วยเส้นทางไฟล์ DWFX
        Converter converter = new Converter("path/to/your/file.dwfx");

        // ตั้งค่าตัวเลือกการแปลง PDF
        PdfConvertOptions options = new PdfConvertOptions();

        Console.WriteLine("PDF conversion options set successfully!");
    }
}
```
#### ขั้นตอนที่ 3: แปลงและบันทึก PDF

สุดท้ายดำเนินการแปลงโดยใช้ `Convert` วิธีการโดยระบุทั้งไฟล์ต้นฉบับและรูปแบบเอาต์พุตที่ต้องการ
```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main()
    {
        // เริ่มต้นตัวจัดการการแปลงด้วยเส้นทางไฟล์ DWFX
        Converter converter = new Converter("path/to/your/file.dwfx");

        // ตั้งค่าตัวเลือกการแปลง PDF
        PdfConvertOptions options = new PdfConvertOptions();

        // แปลงและบันทึกผลลัพธ์เป็นเอกสาร PDF
        converter.Convert("output/path/for/convertedFile.pdf", options);

        Console.WriteLine("Conversion to PDF completed successfully!");
    }
}
```
ด้วยรหัสนี้ ไฟล์ DWFX ของคุณจะถูกแปลงเป็น PDF และบันทึกในเส้นทางที่ระบุ คุณสามารถปรับเปลี่ยนได้ `PdfConvertOptions` สำหรับการตั้งค่าขั้นสูงเพิ่มเติมหากจำเป็น
