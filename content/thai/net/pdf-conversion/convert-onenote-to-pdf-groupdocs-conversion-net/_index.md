# แปลงไฟล์ OneNote เป็น PDF ด้วย GroupDocs.Conversion สำหรับ .NET

## การแนะนำ

คุณกำลังมองหาวิธีแปลงไฟล์ Microsoft OneNote ของคุณเป็นรูปแบบที่สามารถเข้าถึงได้ทั่วไป เช่น PDF หรือไม่ ไม่ว่าคุณจะกำลังเตรียมเอกสารสำหรับการแบ่งปัน การจัดเก็บถาวร หรือเพียงแค่ต้องการรูปแบบที่พกพาสะดวกกว่า การแปลง `.one` การแปลงไฟล์ OneNote เป็น PDF เป็นงานที่สำคัญ ในบทช่วยสอนนี้ เราจะแนะนำคุณเกี่ยวกับการใช้ไลบรารี GroupDocs.Conversion ใน .NET เพื่อแปลงไฟล์ OneNote ของคุณเป็น PDF ได้อย่างราบรื่น

**สิ่งที่คุณจะได้เรียนรู้:**
- การตั้งค่า GroupDocs.Conversion สำหรับ .NET
- การโหลดและการแปลง `.one` ไฟล์เป็น PDF
- เพิ่มประสิทธิภาพการทำงานและแก้ไขปัญหาทั่วไป

## การตั้งค่า GroupDocs.Conversion สำหรับ .NET

หากต้องการเริ่มต้นการแปลงไฟล์ OneNote เป็น PDF ก่อนอื่นคุณต้องติดตั้งไลบรารี GroupDocs.Conversion ดังต่อไปนี้:

### การใช้คอนโซลตัวจัดการแพ็คเกจ NuGet
```bash
Install-Package GroupDocs.Conversion -Version 25.3.0
```

### การใช้ .NET CLI
```bash
dotnet add package GroupDocs.Conversion --version 25.3.0
```

#### ขั้นตอนการรับใบอนุญาต
- **ทดลองใช้งานฟรี**คุณสามารถเริ่มต้นด้วยการทดลองใช้ฟรีเพื่อสำรวจคุณสมบัติต่างๆ
- **ใบอนุญาตชั่วคราว**:สำหรับการทดสอบแบบขยายเวลา ให้ขอใบอนุญาตชั่วคราวจาก [เอกสารกลุ่ม](https://purchase-groupdocs.com/temporary-license/).
- **ซื้อ**หากต้องการใช้ในการผลิต คุณจะต้องซื้อใบอนุญาตเต็มรูปแบบ

#### การเริ่มต้นและการตั้งค่าเบื้องต้น

เมื่อติดตั้งแล้ว ให้เริ่มต้น GroupDocs.Conversion เช่นนี้:

```csharp

class Program
```

## คู่มือการใช้งาน

มาแบ่งกระบวนการออกเป็นขั้นตอนที่ชัดเจนกัน

### การโหลดและการแปลงไฟล์ .one เป็น PDF

#### ภาพรวม
หัวข้อนี้เน้นที่วิธีโหลดไฟล์ OneNote และแปลงเป็นรูปแบบ PDF โดยใช้ GroupDocs.Conversion สำหรับ .NET

##### ขั้นตอนที่ 1: กำหนดเส้นทาง

เริ่มต้นด้วยการกำหนดเส้นทางสำหรับแหล่งที่มาของคุณ `.one` ไฟล์และผลลัพธ์ PDF เป้าหมาย:

```csharp
using System;
using System.IO;

string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// ระบุเส้นทางไปยังไฟล์ OneNote ของคุณและ PDF ที่ได้
string oneFilePath = Path.Combine(documentDirectory, "sample.one");
string pdfOutputFile = Path.Combine(outputDirectory, "one-converted-to.pdf");
```

##### ขั้นตอนที่ 2: โหลดไฟล์ต้นฉบับหนึ่งไฟล์

โหลดของคุณ `.one` ไฟล์ที่ใช้ GroupDocs.Conversion:

```csharp
using (var converter = new Converter(oneFilePath))
{
    // ดำเนินการระบุตัวเลือกการแปลง
}
```

##### ขั้นตอนที่ 3: ระบุตัวเลือกการแปลงสำหรับรูปแบบ PDF

ตั้งค่าตัวเลือกการแปลง PDF:

```csharp
var pdfOptions = new PdfConvertOptions();
```

##### ขั้นตอนที่ 4: แปลงและบันทึกไฟล์ .one เป็นเอกสาร PDF

ดำเนินการแปลงและบันทึกไฟล์เอาท์พุต:

```csharp
converter.Convert(pdfOutputFile, pdfOptions);
Console.WriteLine("Conversion completed successfully.");
```

#### ตัวเลือกการกำหนดค่าคีย์
- **ตัวเลือกการแปลง PDF**ปรับแต่งช่วงหน้า การหมุน และการตั้งค่าอื่น ๆ เพื่อปรับแต่งผลลัพธ์ของคุณ

#### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่าเส้นทางได้รับการตั้งค่าอย่างถูกต้อง
- ตรวจสอบว่า `.one` ไฟล์สามารถเข้าถึงได้และไม่เสียหาย
