## คู่มือการใช้งาน

มาแบ่งกระบวนการใช้งานออกเป็นขั้นตอนเชิงตรรกะเพื่อแปลงไฟล์ OTP เป็น PDF กัน

### โหลดและกำหนดค่าเอกสารของคุณ
ขั้นแรก โหลดไฟล์ OTP ของคุณโดยใช้ `Converter` คลาสนี้ทำหน้าที่เป็นรากฐานสำหรับการดำเนินการแปลงทั้งหมด:

```csharp
// กำหนดเส้นทาง
string sourceFilePath = "YOUR_DOCUMENT_DIRECTORY\\sample.otp";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// เริ่มต้นตัวแปลงด้วยเส้นทางไฟล์ OTP
using (var converter = new Converter(sourceFilePath))
{
    // กำหนดค่าตัวเลือกการแปลง PDF
    var pdfOptions = new PdfConvertOptions();

    // ตั้งค่าเส้นทางไฟล์เอาท์พุต
    string outputFile = Path.Combine(outputDirectory, "otp-converted-to.pdf");

    // แปลงและบันทึกเอกสารเป็น PDF
    converter.Convert(outputFile, pdfOptions);
}
```

#### คำอธิบายพารามิเตอร์หลัก
- **คลาสตัวแปลง**: แสดงถึงไฟล์ต้นฉบับ จำเป็นต้องมีเส้นทางไฟล์ OTP ที่ถูกต้อง
- **ตัวเลือกการแปลง PDF**: กำหนดค่าวิธีการดำเนินการแปลง คุณสามารถปรับการตั้งค่าต่างๆ เช่น หมายเลขหน้าและระดับการซูมได้
- **ตัวแปลง.แปลง()**: วิธีการที่ใช้ในการแปลงไฟล์จริง
