### การเริ่มต้นและการตั้งค่าเบื้องต้น

```csharp
using System;
using System.IO;
using GroupDocs.Conversion.Options.Convert;

// เริ่มต้นตัวแปลงด้วยเส้นทางเอกสาร DWG
string documentPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.dwg");
using (var converter = new GroupDocs.Conversion.Converter(documentPath))
{
    // ตั้งค่าตัวเลือกสำหรับการแปลง PDF และแปลงไฟล์
    var options = new PdfConvertOptions();
    string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
    string outputFile = Path.Combine(outputFolder, "dwg-converted-to.pdf");
    
    converter.Convert(outputFile, options);
}
```

### คุณสมบัติ: แปลง DWG เป็น PDF

การแปลงไฟล์ DWG เป็นเอกสาร PDF เป็นเรื่องง่ายด้วย GroupDocs.Conversion ทำตามขั้นตอนเหล่านี้:

#### ขั้นตอนที่ 1: โหลดไฟล์ DWG ต้นฉบับ
โหลดไฟล์ DWG ของคุณโดยใช้ `Converter` คลาส โดยระบุเส้นทางไปยังไฟล์ต้นฉบับของคุณ

```csharp
string documentPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.dwg");
using (var converter = new GroupDocs.Conversion.Converter(documentPath))
{
    // ดำเนินการต่อด้วยขั้นตอนการแปลง...
}
```

*ทำไม* การโหลดไฟล์ DWG จะเริ่มต้นการทำงาน `Converter` วัตถุที่กำลังเตรียมจะประมวลผล

#### ขั้นตอนที่ 2: ตั้งค่าตัวเลือกการแปลง PDF
สร้างอินสแตนซ์ของ `PdfConvertOptions`ซึ่งเก็บการกำหนดค่าสำหรับไฟล์ PDF ที่คุณส่งออก

```csharp
var options = new PdfConvertOptions();
```

*ทำไม* การปรับแต่งตัวเลือกเหล่านี้ช่วยให้คุณสามารถควบคุมเอาต์พุต เช่น ขนาดหน้าและการวางแนว

#### ขั้นตอนที่ 3: แปลงและบันทึกผลลัพธ์
ใช้ `Convert` วิธีการดำเนินการแปลงและบันทึกผลลัพธ์เป็นรูปแบบ PDF

```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "dwg-converted-to.pdf");

converter.Convert(outputFile, options);
```

*ทำไม* ขั้นตอนนี้จะดำเนินการแปลงโดยใช้การตั้งค่าที่ระบุ เพื่อสร้างไฟล์ PDF

#### เคล็ดลับการแก้ไขปัญหา

- **ปัญหาทั่วไป:** ข้อผิดพลาดไม่พบไฟล์ ตรวจสอบให้แน่ใจว่าเส้นทางไปยังไดเร็กทอรีต้นทางและปลายทางของคุณถูกต้อง
- **สารละลาย:** ตรวจสอบชื่อไดเร็กทอรีอีกครั้งและตรวจสอบให้แน่ใจว่ามีอยู่ในระบบไฟล์ของคุณ
