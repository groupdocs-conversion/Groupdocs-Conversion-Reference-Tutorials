## คู่มือการใช้งาน

ในส่วนนี้เราจะแบ่งกระบวนการออกเป็นขั้นตอนที่จัดการได้เพื่อใช้ในการแปลง TXT เป็น PDF พร้อมตัวเลือกช่องว่างท้ายขั้นสูง

### ภาพรวม: การควบคุมช่องว่างท้ายในการแปลง

คุณลักษณะนี้ช่วยให้คุณกำหนดวิธีจัดการช่องว่างท้ายในระหว่างการแปลงเพื่อให้แน่ใจว่าเอกสารของคุณยังคงการจัดรูปแบบตามที่ต้องการ

#### ขั้นตอนที่ 1: โหลดไฟล์ TXT
ใช้ `TxtLoadOptions` เพื่อระบุการจัดการพิเศษใด ๆ สำหรับช่องว่างต่อท้าย

```csharp
// เริ่มต้น TxtLoadOptions ด้วยตัวเลือกการตัดช่องว่าง
var loadOptions = new TxtLoadOptions();
loadOptions.TrimTrailingSpaces = true; // ควบคุมพฤติกรรมของช่องว่างท้าย
```

**คำอธิบาย**: การตั้งค่า `TrimTrailingSpaces` ถึง `true` ช่วยให้แน่ใจว่าช่องว่างพิเศษใดๆ ที่ตอนท้ายบรรทัดจะถูกลบออกในระหว่างการแปลง

#### ขั้นตอนที่ 2: กำหนดค่าตัวเลือกการแปลง PDF

```csharp
// ตั้งค่า PdfConvertOptions หากจำเป็น แม้ว่าตามค่าเริ่มต้น การตั้งค่าส่วนใหญ่จะทำงานได้ดีสำหรับการแปลงแบบง่าย
var convertOptions = new PdfConvertOptions();
```

**คำอธิบาย**- `PdfConvertOptions` สามารถใช้ปรับแต่งเอาต์พุต PDF เพิ่มเติมได้ ถึงแม้ว่าค่าเริ่มต้นจะเพียงพอต่อความต้องการพื้นฐานก็ตาม

#### ขั้นตอนที่ 3: ดำเนินการแปลง

```csharp
using (Converter converter = new Converter("input.txt", () => loadOptions))
{
    // แปลง TXT เป็น PDF ด้วยตัวเลือกที่ระบุ
    converter.Convert("output.pdf", convertOptions);
}
```

**คำอธิบาย**: เดอะ `Convert` วิธีการนี้จะดำเนินการแปลงจริงโดยใช้การตั้งค่าที่คุณกำหนดไว้ 

#### เคล็ดลับการแก้ไขปัญหา
- **ไม่พบไฟล์**: ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์อินพุตถูกต้องและสามารถเข้าถึงได้
- **การแปลงล้มเหลว**: ตรวจสอบว่าสิ่งที่ต้องติดตั้งทั้งหมดถูกต้อง
