## คู่มือการใช้งาน
เราจะแบ่งกระบวนการแปลงออกเป็นรายละเอียดเพื่อความชัดเจนและง่ายต่อการเข้าใจ

### โหลดและแปลงไฟล์ DNG เป็น PDF
#### ภาพรวม
หัวข้อนี้จะอธิบายวิธีการโหลดไฟล์ DNG และแปลงเป็น PDF โดยใช้ GroupDocs.Conversion

#### การดำเนินการแบบทีละขั้นตอน
##### กำหนดเส้นทางและเริ่มต้นตัวแปลง
```csharp
// กำหนดเส้นทางสำหรับไดเร็กทอรีเอกสารและเอาต์พุต\string sourceDirectory = "YOUR_DOCUMENT_DIRECTORY";\string outputDirectory = "YOUR_OUTPUT_DIRECTORY/";

// เส้นทางแบบเต็มไปยังไฟล์ DNG อินพุต\สตริง dngFilePath = Path.Combine(sourceDirectory, "sample.dng");

// เส้นทางแบบเต็มไปยังไฟล์ PDF เอาท์พุต\สตริง pdfOutputPath = Path.Combine(outputDirectory, "dng-converted-to.pdf");
```
ที่นี่ เราตั้งค่าเส้นทางที่จำเป็นและเริ่มต้นสภาพแวดล้อมการแปลง

##### เริ่มต้นการแปลงด้วยไฟล์ DNG ต้นฉบับ
```csharp
using (var converter = new Converter(dngFilePath))
{
    // กำหนดตัวเลือกการแปลงสำหรับรูปแบบ PDF
    var convertOptions = new PdfConvertOptions();

    // ดำเนินการแปลงและบันทึกไฟล์ PDF เอาท์พุต
    converter.Convert(pdfOutputPath, convertOptions);
}
```
- **การเริ่มต้นตัวแปลง**: เดอะ `Converter` วัตถุจะถูกเริ่มต้นด้วยเส้นทางไฟล์ DNG ต้นทาง
- **ตัวเลือกการแปลง**: เราให้คำจำกัดความ `PdfConvertOptions`, ระบุการตั้งค่ารูปแบบเป้าหมาย
- **ดำเนินการแปลง**: เดอะ `Convert` วิธีการดำเนินการแปลงและบันทึก PDF ไปยังเส้นทางเอาต์พุตที่ระบุ

#### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่าระบุเส้นทางไฟล์อย่างถูกต้อง
- ตรวจสอบสิทธิ์ที่เพียงพอในไดเร็กทอรีที่ระบุ
- ตรวจสอบความเข้ากันได้กับเวอร์ชัน GroupDocs.Conversion
