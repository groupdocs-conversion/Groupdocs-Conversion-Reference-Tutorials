## คู่มือการใช้งาน

### การแยกและการแปลงไฟล์จาก RAR เป็น PDF

#### ภาพรวม
ฟีเจอร์นี้จะแสดงวิธีการแยกไฟล์จากไฟล์ RAR และแปลงไฟล์เป็นรูปแบบ PDF มาดูขั้นตอนต่างๆ กัน

#### ขั้นตอนที่ 1: โหลดไฟล์ RAR Archive
```csharp
using GroupDocs.Conversion;
using GroupDocs.Conversion.Contracts;
using GroupDocs.Conversion.Options.Convert;

string rarPath = "path/to/your/archive.rar";

// เริ่มต้นการแปลงด้วยไฟล์ RAR
using (var converter = new Converter(rarPath))
{
    // ประมวลผลแต่ละไฟล์ในไฟล์เก็บถาวร
}
```

#### ขั้นตอนที่ 2: กำหนดค่าตัวเลือกการแปลง
```csharp
PdfConvertOptions options = new PdfConvertOptions();

// ปรับแต่งการตั้งค่าการแปลงตามความต้องการ
options.MarginTop = 10;
options.MarginBottom = 10;

// แปลงและบันทึก PDF
converter.Convert("output.pdf", options);
```

### คำอธิบาย:
- **โหลดไฟล์ RAR Archive**: เริ่มต้นการ `Converter` วัตถุที่มีเส้นทางไฟล์ RAR ของคุณ
- **กำหนดค่าตัวเลือกการแปลง**: สร้างอินสแตนซ์ของ `PdfConvertOptions` เพื่อปรับแต่งผลลัพธ์ PDF ของคุณ

#### เคล็ดลับการแก้ไขปัญหา:
- ตรวจสอบให้แน่ใจว่าไฟล์ RAR ของคุณไม่เสียหายก่อนที่จะพยายามแปลง
- ตรวจสอบเส้นทางไฟล์และการอนุญาตที่ถูกต้องในการตั้งค่าสภาพแวดล้อมของคุณ
