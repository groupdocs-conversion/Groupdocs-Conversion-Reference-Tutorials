## คู่มือการใช้งาน

ในส่วนนี้ เราจะแนะนำคุณเกี่ยวกับการแปลงไฟล์ LOG เป็น PDF โดยใช้ GroupDocs.Conversion

### ขั้นตอนที่ 1: กำหนดไดเรกทอรีแหล่งที่มาและเอาต์พุต

ขั้นแรก ระบุไดเร็กทอรีสำหรับไฟล์ LOG อินพุตและไฟล์ PDF เอาท์พุต:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
```

### ขั้นตอนที่ 2: โหลดไฟล์ LOG ต้นฉบับ

โหลดไฟล์ LOG ของคุณโดยใช้ `Converter` คลาส ตรวจสอบให้แน่ใจว่าคุณใช้เส้นทางที่ถูกต้องเพื่อหลีกเลี่ยงข้อผิดพลาด:

```csharp
using (var converter = new Converter(Path.Combine(documentDirectory, "sample.log")))
{
    // ดำเนินการตามขั้นตอนการแปลง
}
```

### ขั้นตอนที่ 3: เริ่มต้นตัวเลือกการแปลง PDF

กำหนดค่าตัวเลือกการแปลงของคุณโดยใช้ `PdfConvertOptions` คลาสสำหรับการปรับแต่งเอาท์พุต:

```csharp
var options = new PdfConvertOptions();
```

### ขั้นตอนที่ 4: แปลงและบันทึกไฟล์ PDF

สุดท้ายแปลงไฟล์ LOG เป็น PDF และบันทึกลงในไดเร็กทอรีที่คุณกำหนด:

```csharp
string outputFile = Path.Combine(outputDirectory, "log-converted-to.pdf");
converter.Convert(outputFile, options);
```
