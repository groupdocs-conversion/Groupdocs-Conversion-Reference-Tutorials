### คุณสมบัติ: แปลง GIF เป็น PDF
ฟีเจอร์นี้จะแปลงภาพ GIF ให้เป็นเอกสาร PDF ที่สามารถเข้าถึงได้ทั่วโลก คุณสามารถนำไปใช้งานฟีเจอร์นี้ได้ดังนี้:

#### ขั้นตอนที่ 1: โหลดไฟล์ต้นฉบับของคุณ
ขั้นแรก โหลดไฟล์ GIF ต้นฉบับของคุณ คุณจะต้องมีเส้นทางไปยัง GIF และไดเร็กทอรีเอาต์พุตสำหรับ PDF
```csharp
using System.IO;
using GroupDocs.Conversion;

namespace ConversionFeatures {
    public static class GifToPdfConverter {
        public static void ConvertGifToPdf() {
            string outputFolder = "YOUR_OUTPUT_DIRECTORY";
            string outputFile = Path.Combine(outputFolder, "gif-converted-to.pdf");
            
            // โหลดไฟล์ GIF ต้นฉบับ
            using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY/Sample.gif")) {
                var options = new PdfConvertOptions();
                converter.Convert(outputFile, options);
            }
        }
    }
}
```

#### พารามิเตอร์และวิธีการ
- `Converter`: เริ่มกระบวนการแปลง ส่งเส้นทางไปยังไฟล์ GIF ของคุณ
- `PdfConvertOptions`: กำหนดค่าวิธีการสร้าง PDF
- `converter.Convert()`: ดำเนินการแปลงจาก GIF เป็น PDF
