### การเริ่มต้นและการตั้งค่าเบื้องต้น

นี่คือวิธีเริ่มต้น GroupDocs.Conversion ในแอปพลิเคชัน C# ของคุณ:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

// กำหนดไดเรกทอรีเอาท์พุตและเส้นทางไฟล์
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "tif-converted-to.pdf");

// โหลดไฟล์ TIF ต้นฉบับ
string tifFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.tif");

using (var converter = new Converter(tifFilePath))
{
    var options = new PdfConvertOptions(); // กำหนดค่าการแปลงเป็นรูปแบบ PDF
    
    // บันทึกไฟล์ PDF ที่แปลงแล้ว
    converter.Convert(outputFile, options);
}
```

## คู่มือการใช้งาน

### การโหลดและการแปลงไฟล์ TIFF

#### ภาพรวม

การแปลงไฟล์ TIFF เป็น PDF เกี่ยวข้องกับการโหลดไฟล์ต้นฉบับโดยใช้ GroupDocs.Conversion และระบุรูปแบบเอาต์พุตที่คุณต้องการ ซึ่งในกรณีนี้คือ PDF วิธีการทำงานมีดังนี้:

1. **โหลดไฟล์ TIFF ต้นฉบับ**
   
   สร้างอินสแตนซ์ของ `Converter` ด้วยเส้นทางไฟล์ TIF ของคุณ

2. **กำหนดค่าตัวเลือกการแปลง**
   
   ใช้ `PdfConvertOptions` เพื่อตั้งค่าพารามิเตอร์สำหรับการแปลง PDF เช่น การตั้งค่าเค้าโครงและคุณภาพ

3. **บันทึกไฟล์ที่แปลงแล้ว**
   
   โทรหา `Convert` วิธีการประมวลผลไฟล์และบันทึกไว้ในไดเร็กทอรีเอาต์พุตที่ระบุ

นี่คือการแยกรายละเอียดพร้อมตัวอย่างโค้ด:

```csharp
// โหลดไฟล์ TIF ต้นฉบับ
string tifFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.tif");
using (var converter = new Converter(tifFilePath))
{
    var options = new PdfConvertOptions(); // กำหนดค่าการแปลงเป็นรูปแบบ PDF
    
    // บันทึกไฟล์ PDF ที่แปลงแล้ว
    converter.Convert(outputFile, options);
}
```

- **ตัวเลือกการแปลง PDF**:วัตถุการกำหนดค่าที่ช่วยให้คุณระบุการตั้งค่าต่าง ๆ สำหรับเอาท์พุต PDF
