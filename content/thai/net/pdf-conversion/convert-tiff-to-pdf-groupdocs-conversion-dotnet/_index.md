## คู่มือการใช้งาน

ตอนนี้มาแบ่งกระบวนการแปลงออกเป็นขั้นตอนที่เป็นตรรกะกัน

### แปลง TIFF เป็น PDF

ฟีเจอร์นี้ช่วยให้คุณแปลงไฟล์ภาพ TIFF เป็นเอกสาร PDF ได้อย่างมีประสิทธิภาพ โดยมีวิธีการทำงานดังนี้:

#### ขั้นตอนที่ 1: โหลดไฟล์ TIFF ต้นฉบับ

ขั้นแรก ระบุเส้นทางไปยังไฟล์ TIFF ต้นทางของคุณ:
```csharp
string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.tiff");
```
ขั้นตอนนี้จะช่วยให้แน่ใจว่า GroupDocs.Conversion ทราบว่าคุณต้องการแปลงไฟล์ใด

#### ขั้นตอนที่ 2: ตั้งค่าตัวเลือกการแปลง

ขั้นตอนต่อไปคือการกำหนดตัวเลือกการแปลงสำหรับรูปแบบ PDF ซึ่งคุณสามารถกำหนดค่าว่า TIFF จะถูกแปลงเป็น PDF ได้อย่างไร:
```csharp
var options = new PdfConvertOptions();
```
ตัวเลือกเหล่านี้ช่วยให้คุณปรับแต่งเอาต์พุต PDF ตามความต้องการของคุณได้

#### ขั้นตอนที่ 3: ดำเนินการแปลง

สุดท้ายให้ดำเนินการแปลงและบันทึก PDF ในตำแหน่งที่คุณต้องการ:
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "tiff-converted-to.pdf");

using (var converter = new Converter(inputFilePath))
{
    // แปลงและบันทึกไฟล์ TIFF เป็น PDF
    converter.Convert(outputFile, options);
}
```
ขั้นตอนนี้จะแปลง TIFF ของคุณเป็น PDF และบันทึกลงในไดเร็กทอรีที่ระบุ

1. **การจัดเก็บถาวร**:แปลงภาพ TIFF ที่มีความละเอียดสูงเป็น PDF ที่สามารถเข้าถึงได้ง่ายเพื่อวัตถุประสงค์ในการเก็บถาวร
2. **การแบ่งปัน**:แบ่งปันเอกสารในรูปแบบที่ได้รับการยอมรับทั่วโลกเช่น PDF เพื่อให้มั่นใจว่าเข้ากันได้กับแพลตฟอร์มต่างๆ
3. **การพิมพ์**:เตรียมไฟล์ TIFF สำหรับการพิมพ์โดยแปลงเป็น PDF ซึ่งได้รับการสนับสนุนจากเครื่องพิมพ์อย่างกว้างขวาง
