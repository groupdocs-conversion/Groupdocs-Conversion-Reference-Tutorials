### การโหลดไฟล์ต้นฉบับ

#### ภาพรวม

โหลดไฟล์ภาพ JPEG 2000 (.j2c) ของคุณโดยใช้ `Converter` คลาสที่จัดทำโดย GroupDocs.Conversion คลาสนี้จัดการรูปแบบอินพุตต่างๆ สำหรับการแปลง

**โหลดไฟล์ J2C**

```csharp
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY/sample.j2c"))
{
    // ตรรกะการแปลงจะอยู่ที่นี่
}
```

- **พารามิเตอร์**: แทนที่ `"YOUR_DOCUMENT_DIRECTORY/sample.j2c"` ด้วยเส้นทางจริงไปยังไฟล์ของคุณ

### การแปลงเป็น PDF

#### ภาพรวม

แปลงไฟล์ J2C ของคุณเป็น PDF โดยระบุตัวเลือกการแปลงเฉพาะ PDF ขั้นตอนนี้จะแปลงรูปภาพของคุณเป็นรูปแบบที่เข้าถึงได้ทั่วไป

#### การดำเนินการแบบทีละขั้นตอน

**ตั้งค่าตัวเลือกการแปลง**

```csharp
var options = new PdfConvertOptions();
```

- **วัตถุประสงค์**- `PdfConvertOptions` ระบุว่าเอาท์พุตควรอยู่ในรูปแบบ PDF

**ดำเนินการแปลง**

```csharp
string outputFile = "YOUR_OUTPUT_DIRECTORY/j2c-converted-to.pdf";
converter.Convert(outputFile, options);
```

- **พารามิเตอร์**-
  - `outputFile`: เส้นทางที่จะบันทึกไฟล์ PDF ที่แปลงแล้ว ตรวจสอบให้แน่ใจว่ามีไดเร็กทอรีนี้อยู่หรือสร้างไว้ล่วงหน้า
