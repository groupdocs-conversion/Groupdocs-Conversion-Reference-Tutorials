## นำเข้าเนมสเปซ
ก่อนที่คุณจะเริ่มแปลงไฟล์ คุณต้องนำเข้าเนมสเปซที่จำเป็นลงในโค้ด C# ของคุณก่อน เนมสเปซเหล่านี้ให้สิทธิ์ในการเข้าถึงคลาสและวิธีการที่จำเป็นสำหรับการแปลงไฟล์โดยใช้ GroupDocs.Conversion

```csharp
using System;
using System.IO;
using GroupDocs.Conversion.Options.Convert;
```

เนมสเปซ System.IO มอบคลาสสำหรับการทำงานกับไฟล์และไดเร็กทอรี ซึ่งจำเป็นสำหรับการจัดการไฟล์อินพุตและเอาต์พุตในระหว่างกระบวนการแปลง

ตอนนี้ คุณได้ตั้งค่าข้อกำหนดเบื้องต้นและนำเข้าเนมสเปซที่จำเป็นแล้ว มาลงรายละเอียดกระบวนการทีละขั้นตอนในการแปลงไฟล์ SXC (OpenOffice Spreadsheet) เป็นรูปแบบ PDF โดยใช้ GroupDocs.Conversion สำหรับ .NET กัน
## ขั้นตอนที่ 1: กำหนดโฟลเดอร์ผลลัพธ์และชื่อไฟล์
```csharp
string outputFolder = "Your Document Directory";
string outputFile = Path.Combine(outputFolder, "sxc-converted-to.pdf");
```
ในขั้นตอนนี้ คุณกำหนดโฟลเดอร์เอาต์พุตที่จะบันทึกไฟล์ PDF ที่แปลงแล้ว พร้อมด้วยชื่อไฟล์ที่ต้องการ
## ขั้นตอนที่ 2: โหลดไฟล์ SXC ต้นฉบับ
```csharp
using (var converter = new GroupDocs.Conversion.Converter(Constants.SAMPLE_SXC))
{
    // โค้ดสำหรับการแปลงอยู่ที่นี่
}
```
ที่นี่ คุณจะเริ่มต้นอินสแตนซ์ใหม่ของคลาส GroupDocs.Conversion.Converter โดยส่งเส้นทางไปยังไฟล์ SXC ต้นฉบับเป็นพารามิเตอร์
## ขั้นตอนที่ 3: กำหนดค่าตัวเลือกการแปลง
```csharp
var options = new PdfConvertOptions();
```
คุณสร้างอินสแตนซ์ของคลาส PdfConvertOptions เพื่อระบุตัวเลือกเพิ่มเติมสำหรับการแปลง PDF ขั้นตอนนี้เป็นทางเลือก แต่คุณสามารถปรับแต่งการตั้งค่าการแปลงตามความต้องการของคุณได้
## ขั้นตอนที่ 4: ดำเนินการแปลง
```csharp
converter.Convert(outputFile, options);
```
โดยการใช้เมธอด Convert ของคลาส Converter คุณจะเริ่มกระบวนการแปลงโดยระบุเส้นทางไฟล์เอาต์พุตและตัวเลือกการแปลง
## ขั้นตอนที่ 5: แสดงสถานะการแปลง
```csharp
Console.WriteLine("\nConversion to pdf completed successfully. \nCheck output in {0}", outputFolder);
```
ในที่สุด คุณจะให้ข้อเสนอแนะแก่ผู้ใช้เพื่อยืนยันว่ากระบวนการแปลงเสร็จสมบูรณ์ และระบุตำแหน่งที่สามารถพบไฟล์ PDF ที่แปลงแล้วได้
