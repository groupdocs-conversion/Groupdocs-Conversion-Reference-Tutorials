# วิธีแปลงไฟล์ CF2 เป็น TXT โดยใช้ GroupDocs.Conversion .NET: คำแนะนำทีละขั้นตอน

## คู่มือการใช้งาน

### คุณสมบัติ: แปลงไฟล์ CF2 เป็นรูปแบบ TXT

คุณสมบัตินี้มุ่งเน้นที่การแปลงไฟล์ Common File Format (CF2) เป็น Plain Text (TXT) โดยทำดังนี้:

#### ขั้นตอนที่ 1: กำหนดไดเรกทอรีเอาต์พุตและเส้นทางไฟล์

ตั้งค่าเส้นทางไดเรกทอรีเอกสารของคุณและกำหนดว่าจะบันทึกไฟล์ที่แปลงแล้วไว้ที่ใด:
```csharp
string YOUR_DOCUMENT_DIRECTORY = "YOUR_DOCUMENT_DIRECTORY"; // ตัวแทนสำหรับเส้นทางไดเรกทอรีเอกสาร
string YOUR_OUTPUT_DIRECTORY = "YOUR_OUTPUT_DIRECTORY"; // ตัวแทนสำหรับเส้นทางไดเรกทอรีเอาท์พุต

string cf2FilePath = Path.Combine(YOUR_DOCUMENT_DIRECTORY, "sample.cf2"); // ไฟล์ CF2 ที่จะแปลง
string outputFile = Path.Combine(YOUR_OUTPUT_DIRECTORY, "cf2-converted-to.txt"); // เส้นทางไฟล์ TXT เอาท์พุต
```

#### ขั้นตอนที่ 2: โหลดไฟล์ CF2

ใช้ GroupDocs.Conversion `Converter` คลาสสำหรับโหลดไฟล์ CF2 ของคุณ:
```csharp
using (var converter = new Converter(cf2FilePath))
{
    // ขั้นตอนต่อไปจะกล่าวถึงที่นี่...
}
```

#### ขั้นตอนที่ 3: ตั้งค่าตัวเลือกการแปลง

ระบุการตั้งค่าการแปลงโดยใช้ `WordProcessingConvertOptions`, ตั้งค่ารูปแบบเป็น TXT
```csharp
var options = new WordProcessingConvertOptions { Format = FileType.Txt };
```

#### ขั้นตอนที่ 4: แปลงและบันทึกไฟล์

ดำเนินการแปลงและบันทึกไฟล์เอาท์พุต:
```csharp
converter.Convert(outputFile, options);
```
