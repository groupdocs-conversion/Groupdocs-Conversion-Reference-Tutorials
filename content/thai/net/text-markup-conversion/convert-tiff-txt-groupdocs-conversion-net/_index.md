## คู่มือการใช้งาน

### การแปลง TIFF เป็น TXT

#### ภาพรวม
แปลงไฟล์ TIFF เป็นรูปแบบข้อความอย่างมีประสิทธิภาพด้วยไลบรารี GroupDocs ซึ่งเป็นประโยชน์ต่อการแยกข้อมูลข้อความจากรูปภาพหรือเอกสารที่สแกน

#### ขั้นตอนการดำเนินการแปลง
##### 1. กำหนดค่าเส้นทางไฟล์
กำหนดไดเรกทอรีอินพุตและเอาต์พุตของคุณ:
```csharp
string sourceTiffPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.tiff");
string outputPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "output.txt");
```

##### 2. เริ่มต้นตัวแปลง
สร้างอินสแตนซ์ของ `Converter` คลาสส่งผ่านเส้นทางไปยังไฟล์ TIFF ของคุณ
```csharp
using (Converter converter = new Converter(sourceTiffPath))
{
    // ตรรกะการแปลงจะไปที่นี่
}
```

##### 3. ตั้งค่าตัวเลือกการแปลง
ระบุตัวเลือกการแปลงสำหรับรูปแบบ TXT โดยใช้ `TextConvertOptions`-
```csharp
var convertOptions = new TextConvertOptions();
```

##### 4. ดำเนินการแปลง
ดำเนินการแปลงและบันทึกผลลัพธ์ไปยังเส้นทางที่ระบุ
```csharp
converter.Convert(outputPath, convertOptions);
```

- **ข้อผิดพลาดในการแปลง**: ตรวจสอบว่าไฟล์ TIFF สามารถเข้าถึงได้และมีรูปแบบถูกต้องหรือไม่
- **ปัญหาด้านความจำ**:ตรวจสอบการใช้ทรัพยากรในระหว่างการแปลงไฟล์ขนาดใหญ่
