#### ภาพรวม

#### การดำเนินการแบบทีละขั้นตอน
**1. กำหนดเส้นทางและเริ่มต้นตัวแปลง**
ตั้งค่าเส้นทางอินพุตและเอาต์พุตของคุณ:
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "emz-converted-to.txt");
string emzFilePath = "YOUR_DOCUMENT_DIRECTORY/SAMPLE_EMZ";

using (var converter = new GroupDocs.Conversion.Converter(emzFilePath))
{
    // ตรรกะการแปลงจะตามมาที่นี่
}
```
**2. กำหนดค่าตัวเลือกการแปลง**
ระบุการตั้งค่าการแปลงสำหรับเอาท์พุต TXT:
```csharp
var options = new WordProcessingConvertOptions { Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt };
```
**3. ดำเนินการและบันทึกการแปลง**
ดำเนินการแปลงและบันทึกผลลัพธ์ของคุณ:
```csharp
converter.Convert(outputFile, options);
```
