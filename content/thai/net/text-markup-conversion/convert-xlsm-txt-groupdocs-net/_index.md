## คู่มือการใช้งาน
เราจะแนะนำขั้นตอนการแปลงไฟล์ XLSM เป็นรูปแบบ TXT ทีละขั้นตอน

### แปลง XLSM เป็น TXT
คุณลักษณะนี้ช่วยให้คุณแปลงข้อมูลสเปรดชีตของคุณให้เป็นข้อความธรรมดาเพื่อการแยกวิเคราะห์และวิเคราะห์ที่ง่ายขึ้น

#### ขั้นตอนที่ 1: กำหนดเส้นทางไฟล์
เริ่มต้นด้วยการกำหนดเส้นทางอินพุตและเอาต์พุตสำหรับไฟล์ของคุณ:

```csharp
using System.IO;

// ค่าคงที่สำหรับไดเร็กทอรี
cnst string inputFilePath = "YOUR_DOCUMENT_DIRECTORY\\\\sample.xlsm";
cnst string outputDirectoryPath = "YOUR_OUTPUT_DIRECTORY";

// เส้นทางไฟล์เอาท์พุต
string outputFile = Path.Combine(outputDirectoryPath, "xlsm-converted-to.txt");
```

#### ขั้นตอนที่ 2: โหลดไฟล์ต้นฉบับ
ใช้ `Converter` คลาสสำหรับโหลดไฟล์ XLSM ของคุณ:

```csharp
using (var converter = new Converter(inputFilePath))
{
    // ตรรกะการแปลงจะไปที่นี่
}
```
**คำอธิบาย**: เดอะ `Converter` วัตถุจัดการกระบวนการแปลง ทำให้แน่ใจว่าทรัพยากรได้รับการจัดการและกำจัดอย่างถูกต้อง

#### ขั้นตอนที่ 3: ตั้งค่าตัวเลือกการแปลง
ระบุรูปแบบเป้าหมายโดยใช้ตัวเลือกการแปลง:

```csharp
var options = new WordProcessingConvertOptions { Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt };
```
**คำอธิบาย**: เดอะ `WordProcessingConvertOptions` ช่วยให้คุณกำหนดค่าการตั้งค่าที่เฉพาะเจาะจงสำหรับการแปลงข้อความ

#### ขั้นตอนที่ 4: ดำเนินการแปลง
ดำเนินการแปลงและบันทึกผลลัพธ์:

```csharp
converter.Convert(outputFile, options);
```
**คำอธิบาย**:วิธีนี้จะดำเนินการแปลงไฟล์จริงโดยเขียนผลลัพธ์ไปยังตำแหน่งที่ระบุ

- **ปัญหาการอนุญาต**ตรวจสอบว่าแอปพลิเคชันของคุณมีสิทธิ์อ่าน/เขียนสำหรับไดเร็กทอรีที่เกี่ยวข้อง
