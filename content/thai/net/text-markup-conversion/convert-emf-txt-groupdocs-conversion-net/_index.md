## คู่มือการใช้งาน

มาแปลงไฟล์ EMF เป็นรูปแบบ TXT โดยใช้ GroupDocs.Conversion กัน:

### ขั้นตอนที่ 1: กำหนดเส้นทางอินพุตและเอาต์พุต

ระบุเส้นทางต้นทางและปลายทางสำหรับไฟล์ของคุณ:

```csharp
string inputEmfFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.emf");
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", "ConvertedFiles");

// ตรวจสอบให้แน่ใจว่ามีไดเร็กทอรีเอาท์พุตอยู่
if (!Directory.Exists(outputFolder))
{
    Directory.CreateDirectory(outputFolder);
}

string outputFile = Path.Combine(outputFolder, "emf-converted-to.txt");
```
ขั้นตอนนี้รับประกันว่ากระบวนการแปลงของคุณจะไม่ล้มเหลวเนื่องจากขาดไดเร็กทอรี

### ขั้นตอนที่ 2: กำหนดค่าตัวเลือกการแปลง

ตั้งค่าตัวเลือกการแปลงเฉพาะ TXT:

```csharp
using GroupDocs.Conversion.Options.Convert;
WordProcessingConvertOptions options = new WordProcessingConvertOptions { Format = WordProcessingFileType.Txt };
```
การ `WordProcessingConvertOptions` ระบุว่ารูปแบบเอาต์พุตเป็นข้อความธรรมดา

### ขั้นตอนที่ 3: ดำเนินการแปลง

ดำเนินการแปลงตามขั้นตอนเหล่านี้:

```csharp
using (var converter = new Converter(inputEmfFile))
{
    // แปลงไฟล์โดยใช้ตัวเลือกที่ระบุ
    converter.Convert(outputFile, options);
}
```
ตัวอย่างนี้สาธิตการสร้าง `Converter` อินสแตนซ์และการแปลงไฟล์ EMF ของคุณเป็น TXT
