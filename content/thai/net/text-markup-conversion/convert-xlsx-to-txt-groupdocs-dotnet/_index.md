## คู่มือการใช้งาน

### คุณสมบัติ: โหลดและแปลง XLSX เป็น TXT

#### ภาพรวม
ฟีเจอร์นี้ช่วยให้คุณโหลดไฟล์ Excel ในรูปแบบ XLSX และแปลงเป็นไฟล์ข้อความโดยใช้ GroupDocs.Conversion สำหรับ .NET ซึ่งมีประโยชน์อย่างยิ่งเมื่อประมวลผลข้อมูลจากสเปรดชีตในสภาพแวดล้อมที่รองรับเฉพาะไฟล์ข้อความเท่านั้น

#### การดำเนินการแบบทีละขั้นตอน

##### 1. กำหนดไดเรกทอรีเอาต์พุตและเส้นทางไฟล์
ตั้งค่าเส้นทางสำหรับไฟล์ TXT เอาท์พุตของคุณ:
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
File outputFile = new File(outputFolder, "xlsx-converted-to.txt");
```

##### 2. โหลดเอกสาร XLSX
ใช้ GroupDocs.Conversion เพื่อโหลดไฟล์ XLSX ของคุณ:

```csharp
try (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_XLSX")) {
    // ตรรกะการแปลงจะไปที่นี่
} catch (Exception e) {
    Console.WriteLine("Error during conversion: " + e.Message);
}
```

##### 3. ตั้งค่าตัวเลือกการแปลงสำหรับรูปแบบ TXT
กำหนดค่าการตั้งค่าการแปลงเพื่อกำหนดเป้าหมายรูปแบบ TXT:

```csharp
WordProcessingConvertOptions options = new WordProcessingConvertOptions();
options.setFormat(WordProcessingFileType.Txt);
```

##### 4. ดำเนินการแปลง
ดำเนินการแปลงและบันทึกไฟล์เอาท์พุต:

```csharp
converter.convert(outputFile.getPath(), options);
```

- ตรวจสอบให้แน่ใจว่าเส้นทางไปยังไฟล์อินพุต XLSX และไฟล์เอาต์พุต TXT ของคุณถูกต้อง
- ตรวจสอบว่าคุณมีสิทธิ์ที่จำเป็นในการอ่าน/เขียนไฟล์ในไดเร็กทอรีที่ระบุ
