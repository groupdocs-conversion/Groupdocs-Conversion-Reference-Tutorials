## คู่มือการใช้งาน

ตอนนี้คุณได้ตั้งค่าสภาพแวดล้อมและติดตั้งแพ็คเกจที่จำเป็นแล้ว มาดำเนินการแปลง MPX เป็น TEX กัน

### การตั้งค่าการแปลง

#### ภาพรวม

หัวข้อนี้จะแนะนำคุณเกี่ยวกับการตั้งค่าไดเร็กทอรีแหล่งที่มาและเอาต์พุต การโหลดไฟล์ MPX การกำหนดค่าตัวเลือกการแปลงสำหรับรูปแบบ TeX และการดำเนินการแปลง

#### การดำเนินการแบบทีละขั้นตอน

##### กำหนดไดเรกทอรีและเส้นทางไฟล์

ก่อนอื่น ให้ระบุว่าไฟล์ MPX ต้นทางของคุณอยู่ที่ไหนและจะบันทึกไฟล์ TEX ที่แปลงแล้วไว้ที่ใด:
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// กำหนดเส้นทางไฟล์ MPX ต้นทางและเส้นทางไฟล์ TEX เป้าหมาย
string sourceMpxFilePath = Path.Combine(documentDirectory, "sample.mpx");
string outputTexFilePath = Path.Combine(outputDirectory, "mpx-converted-to.tex");
```

##### โหลดและแปลง

โหลดไฟล์ MPX ของคุณโดยใช้ GroupDocs.Conversion และตั้งค่าตัวเลือกการแปลง:
```csharp
using (var converter = new GroupDocs.Conversion.Converter(sourceMpxFilePath))
{
    // กำหนดค่าตัวเลือกการแปลงสำหรับรูปแบบ TEX
    var convertOptions = new PageDescriptionLanguageConvertOptions { Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Tex };
    
    // ดำเนินการแปลงจาก MPX เป็น TEX
    converter.Convert(outputTexFilePath, convertOptions);
}
```

**คำอธิบาย:**
- **การเริ่มต้นตัวแปลง**: เดอะ `Converter` คลาสจะถูกสร้างอินสแตนซ์โดยใช้เส้นทางของไฟล์ต้นฉบับ
- **ตัวเลือกการแปลง**:เราระบุว่าเราต้องการแปลงไฟล์ของเราเป็นรูปแบบ TeX โดยใช้ `PageDescriptionLanguageConvertOptions`-
- **ดำเนินการแปลง**ในที่สุดวิธีการแปลงจะแปลงไฟล์ MPX ของคุณเป็นไฟล์ TEX

##### การแก้ไขปัญหา
หากคุณพบปัญหา ให้ตรวจสอบข้อผิดพลาดทั่วไป เช่น เส้นทางไฟล์ไม่ถูกต้องหรือขาดการอ้างอิง ตรวจสอบให้แน่ใจว่า GroupDocs.Conversion ได้รับการติดตั้งและได้รับอนุญาตอย่างถูกต้อง
