## คู่มือการใช้งาน

### แปลงไฟล์ PNG เป็น TXT

#### ภาพรวม

ฟีเจอร์นี้ช่วยให้คุณโหลดภาพ PNG และแปลงเป็นรูปแบบข้อความโดยใช้ GroupDocs.Conversion สำหรับ .NET

#### การดำเนินการแบบทีละขั้นตอน

**1. โหลดไฟล์ต้นฉบับ**

เริ่มต้นโดยโหลดไฟล์ PNG ต้นฉบับของคุณลงในอ็อบเจ็กต์ Converter:

```csharp
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY\\sample.png"))
{
    // ดำเนินการตามขั้นตอนการแปลงที่นี่
}
```

**2. ตั้งค่าตัวเลือกการแปลง**

กำหนดตัวเลือกการแปลงเพื่อระบุว่าคุณกำลังแปลงเป็นรูปแบบ TXT:

```csharp
WordProcessingConvertOptions options = new WordProcessingConvertOptions { Format = FileTypes.WordProcessingFileType.Txt };
```

- **คำอธิบายพารามิเตอร์**- `options` กำหนดค่าวิธีการจัดการการแปลง โดยระบุเอาท์พุตเป็น TXT

**3. ดำเนินการแปลง**

ดำเนินการแปลงและบันทึกผลลัพธ์ไปยังตำแหน่งที่คุณต้องการ:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "png-converted-to.txt");

converter.Convert(outputFile, options);
```

- **ค่าส่งคืน**: เดอะ `Convert` วิธีการบันทึกไฟล์ที่แปลงแล้วที่เส้นทางที่ระบุ

#### เคล็ดลับการแก้ไขปัญหา

- ตรวจสอบให้แน่ใจว่าเส้นทาง PNG ต้นทางของคุณถูกต้อง
- ตรวจสอบสิทธิ์การเขียนที่เพียงพอในไดเร็กทอรีเอาต์พุต
- ตรวจสอบการติดตั้งแพ็กเกจ GroupDocs.Conversion หากพบข้อผิดพลาด
