## คู่มือการใช้งาน
### การแปลงไฟล์ EMLX เป็นรูปแบบ TXT
**ภาพรวม:** ฟีเจอร์นี้จะแปลงไฟล์ Apple Mail Message (.emlx) เป็นข้อความธรรมดา (.txt) ทำให้จัดการและประมวลผลอีเมลผ่านโปรแกรมได้ง่ายยิ่งขึ้น

#### ขั้นตอนที่ 1: กำหนดค่าตัวแปลง
ขั้นแรกให้เริ่มต้น `Converter` คลาส ระบุไฟล์อินพุต .emlx ของคุณ:

```csharp
string inputFile = "YOUR_INPUT_FILE_PATH.emlx";
using (Converter converter = new Converter(inputFile))
{
    // การตั้งค่าคอนฟิกูเรชันจะถูกเพิ่มที่นี่
}
```
**ทำไม** การ `Converter` วัตถุเป็นสิ่งสำคัญเนื่องจากจะจัดการการโหลดและการแปลงไฟล์

#### ขั้นตอนที่ 2: ตั้งค่าตัวเลือกการแปลง
กำหนดพารามิเตอร์การแปลงของคุณเพื่อระบุรูปแบบผลลัพธ์:

```csharp
TxtConvertOptions options = new TxtConvertOptions();
```
ขั้นตอนนี้จะตั้งค่าวิธีแปลงไฟล์ .emlx ให้เป็นไฟล์ TXT ช่วยให้ปรับแต่งเพิ่มเติมได้หากจำเป็น

#### ขั้นตอนที่ 3: ดำเนินการแปลง
ดำเนินการแปลงและบันทึกผลลัพธ์:

```csharp
converter.Convert(() => File.Create(Path.Combine(outputPath, "output.txt")), options);
```
บรรทัดนี้จะแปลงไฟล์ EMLX ของคุณเป็นรูปแบบ TXT โดยใช้ตัวเลือกที่ระบุ และบันทึกลงในไดเร็กทอรีเอาต์พุตที่คุณกำหนด ซึ่งเป็นสิ่งสำคัญเพราะจะทำให้กระบวนการแปลงเสร็จสมบูรณ์
