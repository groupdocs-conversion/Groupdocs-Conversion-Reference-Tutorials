## คู่มือการใช้งาน

### คุณสมบัติ: แปลงไฟล์ PPSM เป็น TXT

#### ภาพรวม
การแปลง PPSM (ไฟล์ PowerPoint Show) เป็นรูปแบบข้อความถือเป็นสิ่งจำเป็นสำหรับการแยกเนื้อหาหรือเตรียมเนื้อหาสำหรับการประมวลผลในระบบอื่น

##### ขั้นตอนที่ 1: โหลดไฟล์ PPSM ต้นฉบับ
เริ่มต้นด้วยการโหลดไฟล์ PPSM ต้นฉบับของคุณ ตรวจสอบให้แน่ใจว่าคุณระบุเส้นทางที่ถูกต้อง:

```csharp
string documentPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.ppsm");
using (var converter = new Converter(documentPath))
{
    // ขั้นตอนต่อไปจะตามมาที่นี่...
}
```

*ทำไม* การโหลดไฟล์เป็นสิ่งสำคัญเพื่อเตรียมพร้อมสำหรับการดำเนินการแปลง

##### ขั้นตอนที่ 2: กำหนดค่าตัวเลือกการแปลง
ตั้งค่าตัวเลือกการแปลงของคุณเพื่อระบุว่าคุณต้องการเอาท์พุต TXT:

```csharp
WordProcessingConvertOptions options = new WordProcessingConvertOptions { Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt };
```

*ทำไม* การระบุรูปแบบจะช่วยให้ไลบรารีทราบวิธีประมวลผลและแปลงไฟล์

##### ขั้นตอนที่ 3: ดำเนินการแปลง
ดำเนินการแปลงและบันทึกผลลัพธ์ของคุณ:

```csharp
string outputPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "ppsm-converted-to.txt");
converter.Convert(outputPath, options);
```

*ทำไม* ขั้นตอนนี้จะดำเนินการแปลงและเขียนไฟล์ข้อความผลลัพธ์ลงในดิสก์

#### เคล็ดลับการแก้ไขปัญหา:
- ตรวจสอบให้แน่ใจว่าเส้นทางได้รับการตั้งค่าอย่างถูกต้องสำหรับไดเร็กทอรีอินพุตและเอาต์พุต
- ตรวจสอบว่าใบอนุญาต GroupDocs.Conversion ของคุณยังถูกต้องหากคุณเคยใช้งานแบบทดลองใช้มาก่อน

2. **การย้ายข้อมูล:** ย้ายข้อมูลการนำเสนอไปยังฐานข้อมูลข้อความเพื่อให้สามารถสอบถามและจัดการได้ง่ายขึ้น
