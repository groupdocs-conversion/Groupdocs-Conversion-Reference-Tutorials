## คู่มือการใช้งาน

มาแบ่งการใช้งานออกเป็นส่วนๆ ที่สามารถจัดการได้:

### คุณสมบัติ 1: โหลดไฟล์ PS

การโหลดไฟล์ PostScript เป็นขั้นตอนแรกก่อนการแปลง ฟีเจอร์นี้สาธิตวิธีการเริ่มต้นและโหลดไฟล์ PS โดยใช้ GroupDocs.Conversion

#### ภาพรวม
การโหลดไฟล์ PS จะช่วยเตรียมไฟล์สำหรับการดำเนินการถัดไป และรับรองว่าคุณสามารถเข้าถึงเนื้อหาของไฟล์ได้

#### โค้ดตัวอย่าง:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;

string psFilePath = "YOUR_DOCUMENT_DIRECTORY/sample.ps";

// เริ่มต้นตัวแปลงด้วยเส้นทางไฟล์ PS
using (var converter = new Converter(psFilePath))
{
    // ตอนนี้วัตถุตัวแปลงพร้อมที่จะดำเนินการกับไฟล์ PS ที่โหลดแล้ว
}
```

**คำอธิบาย**- 
- **`psFilePath`**: เส้นทางที่ไฟล์ PS ของคุณอยู่
- **`Converter` ระดับ**: จัดการการโหลดและการแปลง

### คุณสมบัติ 2: แปลง PS เป็น TEX

ตอนนี้คุณได้โหลดไฟล์ PS แล้ว มาแปลงเป็นรูปแบบ LaTeX Source Document (.tex) กัน

#### ภาพรวม
การแปลงจาก PS เป็น TEX เกี่ยวข้องกับการตั้งค่าตัวเลือกการแปลงที่เหมาะสมและดำเนินการตามกระบวนการ

#### โค้ดตัวอย่าง:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

string outputFolder = "YOUR_OUTPUT_DIRECTORY/"; // ระบุไดเรกทอรีเอาท์พุตของคุณที่นี่
string outputFile = Path.Combine(outputFolder, "ps-converted-to.tex");

// โหลดไฟล์ PS ต้นฉบับ
using (var converter = new Converter(psFilePath))
{
    // ตั้งค่าตัวเลือกการแปลงสำหรับรูปแบบ TEX
    PageDescriptionLanguageConvertOptions options = new PageDescriptionLanguageConvertOptions 
    {
        Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Tex
    };

    // ดำเนินการแปลงและบันทึกไฟล์เอาท์พุต
    converter.Convert(outputFile, options);
}
```

**คำอธิบาย**-
- **`outputFolder`**:ไดเร็กทอรีที่คุณต้องการบันทึกไฟล์ TEX ที่แปลงแล้ว
- **`PageDescriptionLanguageConvertOptions`**: กำหนดค่ารูปแบบการแปลงเป็น TEX

#### เคล็ดลับการแก้ไขปัญหา

- ตรวจสอบให้แน่ใจว่าเส้นทางถูกระบุอย่างถูกต้องและมีไดเร็กทอรีอยู่
- ตรวจสอบว่าคุณมีสิทธิ์ที่จำเป็นในการอ่าน/เขียนไฟล์
