```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main(string[] args)
    {
        string DocumentPath = "YOUR_DOCUMENT_DIRECTORY/sample.epub";
        string OutputDirectory = "YOUR_OUTPUT_DIRECTORY";

        using (var converter = new Converter(DocumentPath))
        {
            var options = new PageDescriptionLanguageConvertOptions { Format = PageDescriptionLanguageFileType.Tex };
            string outputFile = System.IO.Path.Combine(OutputDirectory, "epub-converted-to.tex");
            converter.Convert(outputFile, options);
        }
    }
}
```

### ภาพรวมคุณลักษณะ: การแปลง EPUB เป็น TEX

ฟังก์ชันหลักที่เรามุ่งเน้นคือการแปลงไฟล์ EPUB เป็นรูปแบบ TEX โดยใช้ GroupDocs.Conversion ฟีเจอร์นี้จำเป็นสำหรับผู้ที่ต้องการแปลงหนังสืออิเล็กทรอนิกส์หรือเอกสารที่คล้ายคลึงกันเป็นรูปแบบที่เหมาะสมสำหรับการเรียงพิมพ์

#### การดำเนินการแบบทีละขั้นตอน

**1. กำหนดเส้นทาง**

เริ่มต้นด้วยการกำหนดเส้นทางสำหรับเอกสาร EPUB ต้นทางของคุณและไดเร็กทอรีเอาต์พุตที่จะบันทึกไฟล์ TEX ที่แปลงแล้ว

```csharp
const string DocumentPath = "YOUR_DOCUMENT_DIRECTORY/sample.epub";
const string OutputDirectory = "YOUR_OUTPUT_DIRECTORY";
```

**2. โหลดไฟล์ต้นฉบับ**

สร้างอินสแตนซ์ของ `Converter` คลาสสำหรับโหลดเอกสาร EPUB ของคุณ:

```csharp
using (var converter = new Converter(DocumentPath))
{
    // รหัสการแปลงจะถูกวางไว้ที่นี่
}
```

**3. ตั้งค่าตัวเลือกการแปลง**

ระบุว่าคุณต้องการแปลงไฟล์เป็นรูปแบบ TEX โดยการตั้งค่าตัวเลือกการแปลง:

```csharp
var options = new PageDescriptionLanguageConvertOptions { Format = PageDescriptionLanguageFileType.Tex };
```

**4. ดำเนินการแปลง**

ดำเนินการแปลงจริงและบันทึกไฟล์เอาต์พุตเป็นรูปแบบ TEX:

```csharp
string outputFile = Path.Combine(OutputDirectory, "epub-converted-to.tex");
converter.Convert(outputFile, options);
```
