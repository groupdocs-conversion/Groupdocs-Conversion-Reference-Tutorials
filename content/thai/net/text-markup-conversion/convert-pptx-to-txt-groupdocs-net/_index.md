## คู่มือการใช้งาน

### การแปลง PPTX เป็น TXT

แปลงการนำเสนอ PowerPoint เป็นรูปแบบข้อความธรรมดาเพื่อการประมวลผลและวิเคราะห์ที่ง่ายดายยิ่งขึ้น

#### ขั้นตอนที่ 1: ระบุไดเรกทอรีและไฟล์เอาต์พุต
กำหนดปลายทางสำหรับไฟล์ข้อความที่คุณแปลง:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY\";
string outputFile = Path.Combine(outputFolder, "pptx-converted-to.txt");
```
**ทำไม:** ขั้นตอนนี้จะตั้งค่าตำแหน่งที่จะบันทึกไฟล์ TXT ที่คุณแปลงแล้ว

#### ขั้นตอนที่ 2: โหลดไฟล์ PPTX ต้นฉบับ
โหลดไฟล์ PowerPoint ของคุณโดยใช้ `Converter` ระดับ:

```csharp
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY\sample.pptx"))
{
    // ขั้นตอนการแปลงจะตามมาที่นี่
}
```
**ทำไม:** การ `Converter` วัตถุเป็นสิ่งสำคัญสำหรับการเข้าถึงและจัดการเอกสารต้นฉบับ

#### ขั้นตอนที่ 3: กำหนดตัวเลือกการแปลง
ตั้งค่าตัวเลือกในการแปลงไฟล์ของคุณเป็นรูปแบบ TXT:

```csharp
WordProcessingConvertOptions options = new WordProcessingConvertOptions { Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt };
```
**ทำไม:** การกำหนดค่านี้จะช่วยให้แน่ใจว่าเอาต์พุตเป็นข้อความธรรมดา โดยไม่ต้องมีการจัดรูปแบบใดๆ

#### ขั้นตอนที่ 4: แปลงและบันทึกไฟล์
ดำเนินการตามกระบวนการแปลง:

```csharp
converter.Convert(outputFile, options);
Console.WriteLine("Conversion completed successfully.");
```
**ทำไม:** การ `Convert` วิธีการประมวลผลไฟล์ตามตัวเลือกที่คุณระบุและบันทึกเป็นเอกสาร TXT
