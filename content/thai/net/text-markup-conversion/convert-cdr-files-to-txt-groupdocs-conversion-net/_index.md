## คู่มือการใช้งาน
หัวข้อนี้จะแนะนำคุณเกี่ยวกับการโหลดและการแปลงไฟล์ CDR เป็นรูปแบบ TXT โดยใช้ GroupDocs.Conversion สำหรับ .NET

### โหลดและแปลง CDR เป็น TXT
การแปลงไฟล์ CorelDRAW (CDR) เป็นข้อความธรรมดา (TXT) ช่วยให้จัดการหรือดึงข้อมูลได้ง่ายขึ้น ทำตามขั้นตอนเหล่านี้:

#### ขั้นตอนที่ 1: กำหนดไดเรกทอรีผลลัพธ์
กำหนดว่าไฟล์ที่คุณแปลงจะถูกบันทึกไว้ที่ไหน:

```csharp
// กำหนดเส้นทางไดเรกทอรีเอาท์พุต
string outputFolder = "YOUR_OUTPUT_DIRECTORY/";
string outputFile = Path.Combine(outputFolder, "cdr-converted-to.txt");
```
*ทำไม* การระบุไดเร็กทอรีเอาต์พุตจะทำให้แน่ใจว่าไฟล์ของคุณถูกจัดเก็บอย่างถูกต้องและสามารถเข้าถึงได้ง่ายในภายหลัง

#### ขั้นตอนที่ 2: โหลดไฟล์ CDR
โหลดไฟล์ CDR ต้นฉบับของคุณลงใน GroupDocs.Conversion แทนที่ `'YOUR_DOCUMENT_DIRECTORY\\sample.cdr'` พร้อมเส้นทางไปยังไฟล์ CDR จริงของคุณ:

```csharp
// โหลดไฟล์ CDR ต้นฉบับ
class Program
{
    static void Main(string[] args)
    {
        using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY\sample.cdr"))
        {
            // การแปลงจะดำเนินการในบล็อคนี้
        }
    }
}
```
*ทำไม* การโหลดไฟล์อย่างถูกต้องเป็นสิ่งสำคัญสำหรับการแปลงที่ประสบความสำเร็จ และการใช้ `using` คำชี้แจงเพื่อให้แน่ใจว่าทรัพยากรได้รับการจัดการอย่างถูกต้อง

#### ขั้นตอนที่ 3: กำหนดค่าตัวเลือกการแปลง
ตั้งค่าตัวเลือกที่จำเป็นในการแปลงไฟล์ CDR ของคุณเป็นรูปแบบ TXT:

```csharp
// กำหนดค่าตัวเลือกการแปลงสำหรับรูปแบบ TXT
class Program
{
    static void Main(string[] args)
    {
        WordProcessingConvertOptions options = new WordProcessingConvertOptions 
        {
            Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt
        };
    }
}
```
*ทำไม* การกำหนดค่าตัวเลือกช่วยให้คุณระบุรูปแบบเป้าหมายและการตั้งค่าเพิ่มเติมที่จำเป็นในระหว่างการแปลง

#### ขั้นตอนที่ 4: ดำเนินการแปลง
ดำเนินการแปลงและบันทึกไฟล์ของคุณ:

```csharp
// แปลงไฟล์ CDR เป็น TXT และบันทึกไว้
class Program
{
    static void Main(string[] args)
    {
        using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY\sample.cdr"))
        {
            WordProcessingConvertOptions options = new WordProcessingConvertOptions 
            {
                Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt
            };
            converter.Convert(outputFile, options);
        }
    }
}
```
*ทำไม* การดำเนินการตามขั้นตอนนี้จะทำให้กระบวนการแปลงเสร็จสมบูรณ์และสร้างไฟล์ TXT จากเอกสาร CDR ต้นฉบับของคุณ

### เคล็ดลับการแก้ไขปัญหา
- **ไม่พบไฟล์**: ตรวจสอบให้แน่ใจว่าเส้นทางไปยังไฟล์ CDR ต้นทางของคุณถูกต้อง
- **รูปแบบไม่ถูกต้อง**: ตรวจสอบอีกครั้งว่าคุณกำลังใช้สิ่งที่เหมาะสม `WordProcessingConvertOptions`-
- **ปัญหาการอนุญาต**: ตรวจสอบการอนุญาตไดเรกทอรีสำหรับทั้งเส้นทางอินพุตและเอาต์พุต
