## คู่มือการใช้งาน

### แปลงไฟล์ VDX เป็นรูปแบบ TXT

ฟีเจอร์นี้มุ่งเน้นที่การแปลงไฟล์ Microsoft Visio XML Drawing (.vdx) เป็นไฟล์ Plain Text (.txt) มาแบ่งขั้นตอนกันดังนี้:

#### 1. กำหนดเส้นทางเอาต์พุตและแหล่งที่มา
เริ่มต้นด้วยการระบุตำแหน่งไฟล์ VDX อินพุตของคุณ และตำแหน่งที่คุณต้องการบันทึกไฟล์ TXT เอาต์พุต

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY"; // กำหนดไดเรกทอรีเอาท์พุต
string sourceFile = "YOUR_DOCUMENT_DIRECTORY\\sample.vdx"; // เส้นทางไปยังไฟล์ VDX ของคุณ
string outputFile = Path.Combine(outputFolder, "vdx-converted-to.txt"); // เส้นทางไฟล์ TXT เอาท์พุต
```

#### 2. โหลดและแปลงไฟล์
สร้าง `Converter` อินสแตนซ์ที่มีไฟล์ต้นฉบับและตั้งค่าตัวเลือกการแปลง

```csharp
// โหลดและแปลงไฟล์ VDX เป็นรูปแบบ TXT
using (var converter = new GroupDocs.Conversion.Converter(sourceFile))
{
    WordProcessingConvertOptions options = new WordProcessingConvertOptions { Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt };
    // แปลงและบันทึกไฟล์เป็น TXT
    converter.Convert(outputFile, options);
}
```

- **พารามิเตอร์:** `sourceFile` คือเส้นทางไฟล์ VDX ของคุณ `WordProcessingConvertOptions` ระบุรูปแบบเป้าหมาย
- **ค่าส่งคืน:** วิธีการแปลงไฟล์เป็นรูปแบบที่ระบุและบันทึกไว้ที่ `outputFile`-

#### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่าเส้นทางทั้งหมดถูกต้องและสามารถเข้าถึงได้
- ตรวจสอบว่าเวอร์ชันไลบรารี GroupDocs.Conversion ตรงกับสภาพแวดล้อม .NET ของคุณ
