## คู่มือการใช้งาน

ตอนนี้เรามาดูการแปลงไฟล์ JPM เป็น TEX โดยใช้ GroupDocs.Conversion สำหรับ .NET กัน

### แปลง JPM เป็น TEX

**ภาพรวม:**
ฟีเจอร์นี้ช่วยให้คุณแปลงไฟล์ภาพ JPEG 2000 เป็นเอกสาร LaTeX ได้ เราจะใช้ตัวเลือกการแปลงที่ GroupDocs จัดเตรียมไว้ให้เพื่อดำเนินการนี้

#### การดำเนินการแบบทีละขั้นตอน
1. **โหลดไฟล์ต้นฉบับ**
   เริ่มต้นด้วยการโหลดไฟล์ .jpm ของคุณโดยใช้เส้นทางตัวแทนสำหรับไดเร็กทอรีเอกสาร:
   ```csharp
   string inputFilePath = @"YOUR_DOCUMENT_DIRECTORY\\sample.jpm";
   ```
2. **ตั้งค่าตัวเลือกการแปลง**
   ระบุรูปแบบผลลัพธ์เป็น TEX โดยตั้งค่าตัวเลือกการแปลงที่เหมาะสม:
   ```csharp
   var options = new PageDescriptionLanguageConvertOptions { Format = PageDescriptionLanguageFileType.Tex };
   ```
3. **ดำเนินการแปลง**
   ดำเนินการแปลงและบันทึกไฟล์ .tex ที่ได้ไปยังตำแหน่งที่คุณต้องการ:
   ```csharp
   string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
   string outputFile = Path.Combine(outputFolder, "jpm-converted-to.tex");

   using (var converter = new Converter(inputFilePath))
   {
       converter.Convert(outputFile, options);
   }
   ```
**คำอธิบายพารามิเตอร์และวิธีการ:**
- `PageDescriptionLanguageConvertOptions`: กำหนดค่าการแปลงเพื่อเอาท์พุตไฟล์ TEX
- `converter.Convert()`: ดำเนินการกระบวนการแปลง

### เคล็ดลับการแก้ไขปัญหา
- **ปัญหาทั่วไป**: ข้อผิดพลาดเส้นทางไฟล์ ตรวจสอบว่าไดเร็กทอรีของคุณมีอยู่และระบุเส้นทางอย่างถูกต้อง
- **สารละลาย**: ใช้เส้นทางแบบสัมบูรณ์หรือตรวจสอบเส้นทางสัมพันธ์ระหว่างการรันไทม์
