## คู่มือการใช้งาน

ปฏิบัติตามขั้นตอนเหล่านี้เพื่อแปลงไฟล์ TIFF เป็นรูปแบบ TEX โดยใช้ GroupDocs.Conversion:

### แปลง TIFF เป็น TEX

กระบวนการนี้จะแปลงไฟล์ TIFF เป็นรูปแบบ TEX อย่างมีประสิทธิภาพ โปรดปฏิบัติตามคำแนะนำเหล่านี้:

#### ขั้นตอนที่ 1: ตั้งค่าไดเรกทอรี
กำหนดไดเรกทอรีอินพุตและเอาต์พุตสำหรับการประมวลผลที่ราบรื่น:
```csharp
using System.IO;

public static string GetDocumentDirectoryPath()
{
    return Path.Combine(Directory.GetCurrentDirectory(), "YOUR_DOCUMENT_DIRECTORY");
}

public static string GetOutputDirectoryPath()
{
    return Path.Combine(Directory.GetCurrentDirectory(), "YOUR_OUTPUT_DIRECTORY");
}
```

#### ขั้นตอนที่ 2: โหลดไฟล์ TIFF และกำหนดค่าตัวเลือกการแปลง
โหลดไฟล์ TIFF ต้นทางของคุณโดยใช้ GroupDocs.Conversion และตั้งค่าตัวเลือกการแปลง:
```csharp
string documentDirectory = GetDocumentDirectoryPath();
string outputDirectory = GetOutputDirectoryPath();

// กำหนดเส้นทางสำหรับไฟล์อินพุตและเอาต์พุต
string inputFilePath = Path.Combine(documentDirectory, "sample.tiff");
string outputFilePath = Path.Combine(outputDirectory, "tiff-converted-to.tex");

using (var converter = new Converter(inputFilePath))
{
    // ตั้งค่าตัวเลือกการแปลงสำหรับรูปแบบ TEX
    PageDescriptionLanguageConvertOptions options = new PageDescriptionLanguageConvertOptions
    {
        Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Tex
    };

    // แปลงและบันทึกไฟล์ในรูปแบบ TEX
    converter.Convert(outputFilePath, options);
}
```

#### ตัวเลือกการกำหนดค่าคีย์:
- **คำอธิบายหน้าภาษาประเภทไฟล์**: ระบุรูปแบบเอาท์พุตเป็น `Tex`-
- **เส้นทางอินพุต/เอาท์พุต**: ตรวจสอบให้แน่ใจว่าเส้นทางเหล่านี้ได้รับการกำหนดอย่างถูกต้องสำหรับสภาพแวดล้อมของคุณ

### เคล็ดลับการแก้ไขปัญหา

- **ข้อผิดพลาดไม่พบไฟล์**: ตรวจสอบว่าไฟล์ TIFF ต้นฉบับมีอยู่ในไดเร็กทอรีที่ระบุ
- **ปัญหาการอนุญาต**:ยืนยันว่าแอปพลิเคชันของคุณมีสิทธิ์ที่จำเป็นในการอ่าน/เขียนไฟล์ในไดเร็กทอรีที่กำหนด
