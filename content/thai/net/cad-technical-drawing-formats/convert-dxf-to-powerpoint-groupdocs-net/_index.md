## คู่มือการใช้งาน

ตอนนี้มาแบ่งขั้นตอนการแปลงออกเป็นขั้นตอนที่ชัดเจนกัน

### โหลดและแปลงไฟล์ DXF เป็น PPT

**ภาพรวม:**
หัวข้อนี้จะแสดงวิธีโหลดไฟล์ DXF และแปลงไฟล์ดังกล่าวเป็นงานนำเสนอ PowerPoint โดยใช้ GroupDocs.Conversion

#### ขั้นตอนที่ 1: กำหนดไดเรกทอรีเอาต์พุตและเส้นทางไฟล์

เริ่มต้นด้วยการกำหนดว่าไฟล์ที่แปลงแล้วจะถูกบันทึกที่ไหน:
```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "dxf-converted-to.ppt");
```

#### ขั้นตอนที่ 2: โหลดไฟล์ต้นฉบับ DXF

โหลดไฟล์ DXF โดยใช้ `Converter` คลาสสำหรับการเริ่มต้นการแปลง:
```csharp
using (var converter = new Converter(Path.Combine("YOUR_DOCUMENT_DIRECTORY", "your-dxf-file.dxf")))
{
    // ตรรกะการแปลงจะถูกนำไปใช้ที่นี่
}
```

#### ขั้นตอนที่ 3: ตั้งค่าตัวเลือกการแปลง

ระบุว่าคุณต้องการแปลงไฟล์ของคุณเป็นรูปแบบ PowerPoint:
```csharp
var options = new PresentationConvertOptions();
```

#### ขั้นตอนที่ 4: ดำเนินการแปลง

ดำเนินการแปลงและบันทึกไฟล์เอาท์พุต
```csharp
converter.Convert(outputFile, options);
```

**ตัวเลือกการกำหนดค่าคีย์:**
- **รูปแบบเอาท์พุต**: ตั้งค่านี้ภายใน `PresentationConvertOptions` เพื่อกำหนดรูปแบบ PowerPoint ที่แน่นอน (เช่น PPTX)
