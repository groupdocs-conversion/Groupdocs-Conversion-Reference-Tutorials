### การโหลดและการแปลงไฟล์ IGS

#### ภาพรวม
คุณลักษณะนี้เกี่ยวข้องกับการโหลดไฟล์ IGS ต้นทางของคุณและแปลงเป็นรูปแบบ CSV ซึ่งมีประโยชน์สำหรับการวิเคราะห์หรือจัดการข้อมูลด้านวิศวกรรมในแอปพลิเคชันสเปรดชีต

#### การดำเนินการแบบทีละขั้นตอน

**1. ตั้งค่าเส้นทางไดเรกทอรี**
กำหนดเส้นทางสำหรับไฟล์ IGS อินพุตและไฟล์ CSV เอาท์พุตของคุณ:

```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY\";
string outputDir = @"YOUR_OUTPUT_DIRECTORY\";

if (!Directory.Exists(outputDir))
{
    Directory.CreateDirectory(outputDir);
}

string inputFile = Path.Combine(dataDir, "sample.igs");
string outputFile = Path.Combine(outputDir, "igs-converted-to.csv");
```

**2. เริ่มต้น GroupDocs.Conversion**
โหลดไฟล์ IGS โดยใช้ GroupDocs.Conversion:

```csharp
using (var converter = new Converter(inputFile))
{
    // โค้ดการแปลงจะอยู่ที่นี่
}
```

**3. กำหนดตัวเลือกการแปลง CSV**
ระบุตัวเลือกการแปลงสำหรับการแปลงเป็นรูปแบบ CSV:

```csharp
var options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Csv };
```

**4. ดำเนินการแปลง**
ดำเนินการตามกระบวนการแปลงโดยแปลงไฟล์ IGS ของคุณเป็นรูปแบบ CSV:

```csharp
converter.Convert(outputFile, options);
```
