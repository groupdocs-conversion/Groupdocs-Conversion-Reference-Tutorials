## คู่มือการใช้งาน
### แปลงไฟล์ DWF เป็นรูปแบบ HTML
ฟีเจอร์นี้สาธิตวิธีการแปลงไฟล์ DWF เป็นรูปแบบ HTML ทำให้สามารถเข้าถึงได้จากเว็บเบราว์เซอร์ใดก็ได้
#### ขั้นตอนที่ 1: กำหนดเส้นทางสำหรับอินพุตและเอาต์พุต
ขั้นแรก ตั้งค่าเส้นทางสำหรับไฟล์ DWF อินพุตของคุณและตำแหน่งที่คุณต้องการบันทึกไฟล์ HTML ที่แปลงแล้ว:
```csharp
string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.dwf");
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "dwf-converted-to.html");
```
#### ขั้นตอนที่ 2: โหลดและแปลงไฟล์
โหลดไฟล์ DWF โดยใช้ `Converter` คลาสและระบุตัวเลือกการแปลงสำหรับ HTML:
```csharp
using (var converter = new Converter(inputFilePath))
{
    // ตัวเลือกการเริ่มต้นการแปลงเป็นรูปแบบ HTML
    var options = new WebConvertOptions();
    
    // ดำเนินการแปลงและบันทึกเป็นไฟล์ HTML
    converter.Convert(outputFile, options);
}
```
### คำอธิบายชิ้นส่วนโค้ด
- **`Converter` ระดับ**: เริ่มต้นด้วยเส้นทางไฟล์ DWF คลาสนี้จัดการการโหลดไฟล์เพื่อการแปลง
- **`WebConvertOptions`**: ระบุว่ารูปแบบผลลัพธ์ควรจะเป็น HTML
- **`converter.Convert` วิธี**: ดำเนินการแปลงโดยบันทึกผลลัพธ์เป็นไฟล์ HTML
