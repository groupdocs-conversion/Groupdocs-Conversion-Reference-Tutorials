```csharp
using System;
using GroupDocs.Conversion;

// เริ่มต้นตัวแปลงด้วยเส้นทางไฟล์ต้นฉบับ
```

## คู่มือการใช้งาน

### คุณสมบัติ: แปลง PPTM เป็น DOCX

#### ภาพรวม

ฟังก์ชันหลักของฟีเจอร์นี้คือการแปลงการนำเสนอ PowerPoint ให้เป็นเอกสาร Word ที่สามารถแก้ไขได้ ช่วยให้จัดการและแบ่งปันเนื้อหาได้อย่างราบรื่น

#### กระบวนการแปลงทีละขั้นตอน

**1. การตั้งค่าเส้นทางต้นทางและปลายทางออก**
กำหนดเส้นทางที่ไฟล์ต้นฉบับของคุณอยู่และตำแหน่งที่คุณต้องการบันทึกเอกสารที่แปลงแล้ว:
```csharp
string sourceFilePath = "YOUR_DOCUMENT_DIRECTORY\\source.pptm";
string outputFilePath = "YOUR_DOCUMENT_DIRECTORY\\output.docx";
```

**2. การเริ่มต้น GroupDocs.Conversion**
โหลดไฟล์ PPTM ของคุณโดยใช้ `Converter` ระดับ:
```csharp
using (var converter = new Converter(sourceFilePath))
{
    // ระบุตัวเลือกการแปลงสำหรับรูปแบบ Word
    var convertOptions = new WordProcessingConvertOptions();
    
    // แปลงและบันทึกไฟล์ DOCX เอาท์พุต
    converter.Convert(outputFilePath, convertOptions);
}
```

**คำอธิบาย:**
- การ `Converter` วัตถุอ่านไฟล์ PPTM ของคุณ
- `WordProcessingConvertOptions` ระบุว่าเราต้องการแปลงเป็นเอกสาร Word (DOCX)
- `converter.Convert()` ดำเนินการแปลงและบันทึกผลลัพธ์
