## คู่มือการใช้งาน

### แปลงไฟล์ XLAM เป็นรูปแบบ PPT

หัวข้อนี้เน้นที่การแปลงไฟล์ XLAM เป็นงานนำเสนอ PowerPoint เราจะแบ่งขั้นตอนต่างๆ ออกเป็นขั้นตอนต่างๆ เพื่อความชัดเจนและง่ายต่อการนำไปใช้งาน

#### การเริ่มต้นวัตถุตัวแปลง

ขั้นแรก ให้กำหนดเส้นทางเอกสารของคุณและเริ่มต้นใช้งาน `Converter` วัตถุที่มีไฟล์ XLAM ต้นฉบับของคุณ:
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY"; // ตั้งค่าเส้นทางไดเร็กทอรีแหล่งที่มาของคุณ
string outputDirectory = "YOUR_OUTPUT_DIRECTORY"; // ตั้งค่าเส้นทางไดเร็กทอรีเป้าหมายของคุณ

// กำหนดเส้นทางไฟล์
string inputFile = Path.Combine(documentDirectory, "sample.xlam");
string outputFile = Path.Combine(outputDirectory, "xlam-converted-to.ppt");

// เริ่มต้นตัวแปลงด้วยไฟล์ XLAM ต้นฉบับ
using (var converter = new Converter(inputFile))
{
    // โค้ดการแปลงมีดังนี้...
}
```

#### ตั้งค่าตัวเลือกการแปลงการนำเสนอ

ขั้นตอนต่อไปคือการกำหนดค่าตัวเลือกการแปลงของคุณเพื่อระบุเอาท์พุต PowerPoint PPT:

#### ดำเนินการแปลง

```csharp

converter.Convert(outputFile, options);
```
