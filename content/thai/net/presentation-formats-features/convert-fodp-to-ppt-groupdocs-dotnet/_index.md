# แปลงไฟล์ FODP เป็น PPT โดยใช้ GroupDocs สำหรับ .NET

## การแนะนำ

คุณเคยพบไฟล์ OpenDocument Flat XML Presentation (.fodp) และต้องการแปลงไฟล์ดังกล่าวเป็นงานนำเสนอ PowerPoint ได้อย่างราบรื่นหรือไม่ คำแนะนำทีละขั้นตอนนี้จะช่วยตอบสนองความต้องการดังกล่าวโดยใช้ไลบรารี GroupDocs.Conversion สำหรับ .NET ที่มีประสิทธิภาพ ไม่ว่าคุณจะเป็นนักพัฒนาหรือผู้ที่ต้องการปรับปรุงการจัดการเอกสารให้มีประสิทธิภาพ บทช่วยสอนนี้จะแนะนำขั้นตอนที่จำเป็นในการแปลงไฟล์ดังกล่าวอย่างมีประสิทธิภาพ

### แปลง FODP เป็น PPT

ฟีเจอร์นี้ช่วยให้คุณแปลงไฟล์ OpenDocument Flat XML Presentation ให้เป็นงานนำเสนอ PowerPoint ได้ ดังต่อไปนี้:

#### ขั้นตอนที่ 1: โหลดไฟล์ FODP ต้นฉบับ

ระบุเส้นทางไฟล์ต้นฉบับและชื่อไฟล์เอาท์พุต:

```csharp
var fodpFilePath = System.IO.Path.Combine(DocumentDirectory, "sample.fodp");
string outputFile = System.IO.Path.Combine(OutputDirectory, "fodp-converted-to.ppt");
```

#### ขั้นตอนที่ 2: ตั้งค่าตัวเลือกการแปลง

กำหนดค่าตัวเลือกการแปลงสำหรับรูปแบบ PowerPoint:

```csharp
using (var converter = new GroupDocs.Conversion.Converter(fodpFilePath))
{
    var options = new PresentationConvertOptions { Format = GroupDocs.Conversion.FileTypes.PresentationFileType.Ppt };
```

#### ขั้นตอนที่ 3: ดำเนินการแปลง

ดำเนินการแปลงและบันทึกไฟล์ PPT ที่ได้:

```csharp
    converter.Convert(outputFile, options);
}
```
