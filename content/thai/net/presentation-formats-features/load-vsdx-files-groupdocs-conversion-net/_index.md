## คู่มือการใช้งาน
ในหัวข้อนี้เราจะแบ่งกระบวนการออกเป็นขั้นตอนที่สามารถจัดการได้

### คุณสมบัติ: การโหลดไฟล์ VSDX
#### ภาพรวม
การโหลดไฟล์ VSDX เป็นขั้นตอนแรกก่อนที่จะทำการแปลงใดๆ ได้ ซึ่งเกี่ยวข้องกับการเริ่มต้น `Converter` คลาสและส่งผ่านเส้นทางไปยังเอกสาร VSDX ของคุณ
##### ขั้นตอนที่ 1: นำเข้าเนมสเปซที่จำเป็น
ตรวจสอบให้แน่ใจว่าคุณได้นำเข้าเนมสเปซที่จำเป็นในตอนต้นของไฟล์ C# ของคุณ:
```csharp
using System;
using GroupDocs.Conversion;
```

##### ขั้นตอนที่ 2: กำหนดเส้นทางเอกสาร
ระบุไดเรกทอรีที่จัดเก็บไฟล์ VSDX ของคุณ แทนที่ `YOUR_DOCUMENT_DIRECTORY` ด้วยเส้นทางที่แท้จริงของคุณ
```csharp
string documentPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "example.vsdx");
```

##### ขั้นตอนที่ 3: เริ่มต้นตัวแปลง
สร้างอินสแตนซ์ของ `Converter` คลาสและโหลดไฟล์ของคุณ:
```csharp
var converter = new Converter(documentPath);
Console.WriteLine("VSDX file loaded successfully!");
```
**คำอธิบายพารามิเตอร์:**
- `documentPath`: เส้นทางเต็มไปยังไฟล์ VSDX ที่คุณต้องการแปลง
