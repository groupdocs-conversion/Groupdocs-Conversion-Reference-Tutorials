## คู่มือการใช้งาน

ทำตามคู่มือนี้เพื่อแปลงไฟล์รูปภาพ JPG เป็นไฟล์ XLS โดยใช้ฟีเจอร์ GroupDocs.Conversion

### โหลดและแปลงไฟล์

หากต้องการแปลง JPG เป็นสเปรดชีต Excel ให้ทำตามขั้นตอนเหล่านี้:

#### ขั้นตอนที่ 1: กำหนดเส้นทางอินพุตและเอาต์พุต
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
string sourceFilePath = Path.Combine(documentDirectory, "sample.jpg");
string outputFile = Path.Combine(outputDirectory, "jpg-converted-to.xls");
```

ระบุไฟล์ JPG อินพุตและตำแหน่งจัดเก็บไฟล์ XLS ที่จะแปลง

#### ขั้นตอนที่ 2: เริ่มต้นตัวแปลง
```csharp
using (var converter = new Converter(sourceFilePath))
{
    // ขั้นตอนการแปลงมีดังนี้
}
```
การ `Converter` คลาสโหลดไฟล์รูปภาพต้นฉบับของคุณ โดยตั้งค่าบริบทการแปลงสำหรับ GroupDocs.Conversion

#### ขั้นตอนที่ 3: ตั้งค่าตัวเลือกการแปลง
```csharp
SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Xls };
```
กำหนดรูปแบบผลลัพธ์โดยใช้ `SpreadsheetConvertOptions`โดยระบุไฟล์ XLS ตัวเลือกนี้อนุญาตให้ปรับแต่งได้ เช่น ตั้งค่าคุณสมบัติสเปรดชีตเฉพาะ

#### ขั้นตอนที่ 4: ดำเนินการแปลง
```csharp
compiler.Convert(outputFile, options);
```
วิธีนี้จะดำเนินการแปลงจาก JPG เป็น XLS และบันทึกลงในไดเร็กทอรีเอาต์พุตที่ระบุ
