## คู่มือการใช้งาน

### คุณสมบัติ: การแปลง OTT เป็น XLS

แปลงเอกสาร OTT ให้เป็นสเปรดชีต Excel เพื่อการวิเคราะห์ข้อมูลหรือการรายงาน

#### ขั้นตอนที่ 1: โหลดไฟล์ OTT

โหลดไฟล์ OTT ของคุณโดยใช้ `Converter` ระดับ:

```csharp
using (Converter converter = new Converter(inputFilePath))
{
    // ดำเนินการตั้งค่าการแปลงต่อ
}
```

**คำอธิบาย:** การ `Converter` วัตถุจะถูกเริ่มต้นด้วยเส้นทางไปยังไฟล์ OTT ของคุณ เพื่อเตรียมพร้อมสำหรับการประมวลผล

#### ขั้นตอนที่ 2: ตั้งค่าตัวเลือกการแปลง

ระบุรูปแบบผลลัพธ์ที่ต้องการโดยใช้ `SpreadsheetConvertOptions`-

```csharp
var options = new SpreadsheetConvertOptions();
options.Format = GroupDocs.Conversion.FileTypes.SpreadsheetFileType.Xls;
```

**คำอธิบาย:** การดำเนินการนี้จะตั้งค่าการกำหนดการแปลงของคุณ โดยระบุว่าผลลัพธ์ควรอยู่ในรูปแบบ XLS

#### ขั้นตอนที่ 3: ดำเนินการแปลง

ดำเนินการแปลงและบันทึกผลลัพธ์:

```csharp
converter.Convert(outputFilePath, options);
```

**คำอธิบาย:** การ `Convert` วิธีการจัดการการแปลงจาก OTT เป็น XLS ตามตัวเลือกที่ระบุ
