## คู่มือการใช้งาน

### การแปลง VDX เป็น XLSX

การแปลงไฟล์ VDX เป็นรูปแบบ XLSX เกี่ยวข้องกับการตั้งค่าเครื่องมือแปลงและระบุการตั้งค่าเอาต์พุต

#### ขั้นตอนที่ 1: ตั้งค่าเส้นทางไฟล์

กำหนดเส้นทางสำหรับไฟล์อินพุต (VDX) และเอาต์พุต (XLSX):

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputFileDirectory = Path.Combine(documentDirectory, "ConvertedFiles");

// ตรวจสอบให้แน่ใจว่ามีไดเร็กทอรีเอาท์พุตอยู่
if (!Directory.Exists(outputFileDirectory))
{
    Directory.CreateDirectory(outputFileDirectory);
}

string inputFilePath = Path.Combine(documentDirectory, "sample.vdx");
string outputFilePath = Path.Combine(outputFileDirectory, "vdx-converted-to.xlsx");
```

#### ขั้นตอนที่ 2: เริ่มต้นและกำหนดค่าตัวแปลง

ตั้งค่า `Converter` ตัวอย่าง:

```csharp
using (var converter = new Converter(inputFilePath))
{
    // ตั้งค่าตัวเลือกการแปลงสำหรับรูปแบบ XLSX
    var options = new SpreadsheetConvertOptions();
    
    // ดำเนินการตามกระบวนการแปลงจริง
    converter.Convert(outputFilePath, options);
}
```
- **พารามิเตอร์**- `inputFilePath` และ `outputFilePath` กำหนดว่าไฟล์ของคุณอยู่หรือบันทึกไว้ที่ไหน
- **ตัวเลือกการแปลงสเปรดชีต**: ระบุว่าคุณกำลังแปลงเป็นรูปแบบที่เข้ากันได้กับ Excel

#### ขั้นตอนที่ 3: การจัดการปัญหาทั่วไป

หากมีปัญหาเกิดขึ้น:
- ตรวจสอบว่าเส้นทางไฟล์ถูกต้องและสามารถเข้าถึงได้
- ตรวจสอบให้แน่ใจว่า GroupDocs.Conversion ได้รับการติดตั้งและอ้างอิงอย่างถูกต้องในโครงการของคุณ
- ตรวจสอบการอัปเดตหรือแพตช์หากข้อผิดพลาดยังคงมีอยู่
