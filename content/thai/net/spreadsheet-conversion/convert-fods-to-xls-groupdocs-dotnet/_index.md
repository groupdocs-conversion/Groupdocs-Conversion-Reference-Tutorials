**สิ่งที่คุณจะได้เรียนรู้:**

- แนวทางปฏิบัติที่ดีที่สุดสำหรับการเพิ่มประสิทธิภาพระหว่างการแปลง

## คู่มือการใช้งาน

มาแบ่งกระบวนการแปลงออกเป็นขั้นตอนที่ชัดเจนกัน

### การโหลดไฟล์ FODS ต้นฉบับ

เริ่มต้นโดยระบุไดเร็กทอรีสำหรับไฟล์ต้นฉบับและไฟล์เอาท์พุตของคุณ:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// โหลดไฟล์ FODS ต้นฉบับ
string sourceFilePath = Path.Combine(documentDirectory, "sample.fods");
```

### การตั้งค่าตัวเลือกการแปลง

กำหนดค่าตัวเลือกสำหรับการแปลงเป็นรูปแบบ XLS:

```csharp
using GroupDocs.Conversion.Options.Convert;

var converter = new Converter(sourceFilePath);

// ตั้งค่าตัวเลือกการแปลงสำหรับรูปแบบ XLS
SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Xls };
```

### การแปลงและบันทึกไฟล์ XLS

ดำเนินการแปลงและบันทึกไฟล์เอาท์พุต:

```csharp
string outputFile = Path.Combine(outputDirectory, "fods-converted-to.xls");

// แปลงและบันทึกไฟล์ XLS
converter.Convert(outputFile, options);

Console.WriteLine("Conversion completed successfully!");
```
