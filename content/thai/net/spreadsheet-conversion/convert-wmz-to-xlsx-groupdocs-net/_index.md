## คู่มือการใช้งาน
มาแบ่งกระบวนการแปลงออกเป็นขั้นตอนที่เป็นตรรกะกัน

### โหลดไฟล์ WMZ
**ภาพรวม**ขั้นตอนแรกคือโหลดไฟล์ WMZ ของคุณโดยใช้ GroupDocs.Conversion ซึ่งจะเป็นการตั้งค่าสภาพแวดล้อมสำหรับการดำเนินการเพิ่มเติม

#### ขั้นตอนที่ 1: ระบุเส้นทางเอกสาร
```csharp
string documentPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.wmz");
```

#### ขั้นตอนที่ 2: เริ่มต้นวัตถุตัวแปลง
```csharp
using (var converter = new Converter(documentPath))
{
    // ตอนนี้ไฟล์ WMZ โหลดเสร็จแล้วและพร้อมสำหรับการแปลงแล้ว
}
```

### กำหนดค่าตัวเลือกการแปลง
**ภาพรวม**:กำหนดวิธีการแปลงไฟล์ WMZ เป็น XLSX รวมถึงการตั้งค่าเฉพาะ

#### ขั้นตอนที่ 1: สร้างอินสแตนซ์ SpreadsheetConvertOptions
```csharp
var options = new SpreadsheetConvertOptions();
// ปรับแต่งตัวเลือกหากจำเป็น (เช่น หมายเลขหน้า)
```

### ดำเนินการแปลง
**ภาพรวม**: ดำเนินการแปลงจาก WMZ เป็น XLSX และบันทึกไฟล์เอาต์พุต

#### ขั้นตอนที่ 1: กำหนดเส้นทางเอาต์พุต
```csharp
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputDirectory, "wmz-converted-to.xlsx");
```

#### ขั้นตอนที่ 2: แปลงและบันทึกไฟล์
```csharp
using (var converter = new Converter(documentPath))
{
    var options = new SpreadsheetConvertOptions();
    // แปลงและบันทึกไฟล์ XLSX
    converter.Convert(outputFile, options);
}
```
