# แปลงไฟล์ WMZ เป็นรูปแบบ XLS ด้วย GroupDocs.Conversion สำหรับ .NET

## การแนะนำ

**บทเรียนที่สำคัญ:**
- การตั้งค่าและติดตั้ง GroupDocs.Conversion สำหรับ .NET
- การแปลงไฟล์ WMZ เป็นรูปแบบ XLS ทีละขั้นตอน
- การประยุกต์ใช้งานจริงของการแปลงนี้
- ข้อควรพิจารณาด้านประสิทธิภาพด้วยชุดข้อมูลขนาดใหญ่

## การตั้งค่า GroupDocs.Conversion สำหรับ .NET

เมื่อติดตั้งแล้ว ตั้งค่าไลบรารีในโครงการของคุณ:

```csharp
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

// กำหนดไดเรกทอรีสำหรับไฟล์ต้นฉบับและไฟล์เอาท์พุต
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY"; // แทนที่ด้วยไดเร็กทอรีจริงของคุณ
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// โหลดไฟล์ WMZ จากไดเร็กทอรีที่ระบุ
string documentPath = Path.Combine(documentDirectory, "sample.wmz");

using (var converter = new Converter(documentPath))
{
    // ตั้งค่าตัวเลือกการแปลงสำหรับรูปแบบ XLS
    var convertOptions = new SpreadsheetConvertOptions
    {
        Format = SpreadsheetFileType.Xls
    };

    // กำหนดเส้นทางเอาท์พุตที่ไฟล์ที่แปลงจะถูกบันทึก
    string outputFile = Path.Combine(outputDirectory, "wmz-converted-to.xls");

    // ดำเนินการแปลงและบันทึกไฟล์ XLS ในตำแหน่งที่ระบุ
    converter.Convert(outputFile, convertOptions);
}
```

โค้ดตัวอย่างนี้จะเริ่มต้น `Converter` วัตถุที่มีเส้นทางไฟล์ WMZ ตั้งค่าตัวเลือกการแปลงสำหรับรูปแบบ XLS และดำเนินการแปลง ไฟล์ที่แปลงแล้วจะถูกบันทึกไว้ในไดเร็กทอรีเอาต์พุตที่คุณระบุ

## คู่มือการใช้งาน

### แปลง WMZ เป็นรูปแบบ XLS

#### ภาพรวม

การแปลงไฟล์ WMZ เป็น XLS เกี่ยวข้องกับการตั้งค่าตัวเลือกการแปลงที่ถูกต้องและระบุทั้งเส้นทางต้นทางและปลายทาง

#### ขั้นตอน:
1. **ตัวแปลงการเริ่มต้น**
   - สร้าง `Converter` อินสแตนซ์ที่มีเส้นทางไฟล์ WMZ ของคุณ ซึ่งจัดการงานการแปลงทั้งหมด
2. **กำหนดค่าตัวเลือกการแปลง**
   - ใช้ `SpreadsheetConvertOptions` เพื่อกำหนดรูปแบบผลลัพธ์เป็น XLS โดยระบุว่าข้อมูลจะปรากฏใน Excel อย่างไร
3. **ดำเนินการและบันทึกการแปลง**
   - โทรหา `Convert` วิธีการที่มีเส้นทางไฟล์เอาท์พุตและตัวเลือกการแปลงเพื่อดำเนินการตามกระบวนการ ไฟล์ XLS ที่ได้จะถูกบันทึกไว้ในตำแหน่งที่กำหนด

#### การแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่าเส้นทางได้รับการตั้งค่าอย่างถูกต้องสำหรับไดเร็กทอรีต้นทางและปลายทาง
- ตรวจสอบว่าคุณมีสิทธิ์ที่ถูกต้องในการอ่าน/เขียนไฟล์ในตำแหน่งเหล่านั้น

2. **การวางแผนโครงการ:** ใช้ไฟล์ XLS สำหรับการจัดกำหนดการโครงการและการจัดสรรทรัพยากรตามข้อมูลไทล์แผนที่
