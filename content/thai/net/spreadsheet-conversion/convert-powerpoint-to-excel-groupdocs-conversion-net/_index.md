## คู่มือการใช้งาน

### โหลดไฟล์นำเสนอ PowerPoint
เริ่มต้นด้วยการโหลดไฟล์ PPT ที่คุณต้องการแปลง

#### ภาพรวม
ฟีเจอร์นี้สาธิตวิธีการโหลดงานนำเสนอ PowerPoint โดยใช้ GroupDocs.Conversion สำหรับ .NET เพื่อเตรียมพร้อมสำหรับการแปลง

##### ขั้นตอนที่ 1: ตั้งค่าสภาพแวดล้อมของคุณ
กำหนดเส้นทางไปยังไฟล์ PPT อินพุตของคุณ แทนที่ `@YOUR_DOCUMENT_DIRECTORY` ด้วยไดเร็กทอรีจริงหรือใช้ตัวแทน

```csharp
string pptFilePath = "@YOUR_DOCUMENT_DIRECTORY/sample.ppt";
```

##### ขั้นตอนที่ 2: เริ่มต้นวัตถุตัวแปลง
ใช้ GroupDocs.Conversion `Converter` คลาสที่จะโหลดไฟล์ PPT

```csharp
using (var converter = new GroupDocs.Conversion.Converter(pptFilePath))
{
    // ตอนนี้ตัวแปลงพร้อมสำหรับการดำเนินการเพิ่มเติม เช่น การแปลงแล้ว
}
```

### แปลง PPT เป็น XLS
เมื่อโหลดงานนำเสนอเสร็จแล้ว ให้เราแปลงเป็นรูปแบบสเปรดชีต Excel

#### ภาพรวม
หัวข้อนี้จะแสดงวิธีการแปลงไฟล์ PPT เป็นรูปแบบ XLS โดยใช้ GroupDocs.Conversion

##### ขั้นตอนที่ 1: กำหนดไดเรกทอรีเอาต์พุตและเส้นทางไฟล์
ตั้งค่าไดเร็กทอรีเอาต์พุตของคุณและระบุตำแหน่งที่จะบันทึกไฟล์ที่แปลงแล้ว

```csharp
string outputFolder = "@YOUR_OUTPUT_DIRECTORY";
string outputFile = System.IO.Path.Combine(outputFolder, "ppt-converted-to.xls");
```

##### ขั้นตอนที่ 2: สร้างตัวเลือกการแปลง
ระบุว่าคุณต้องการแปลงเป็นรูปแบบ XLS โดยใช้ `SpreadsheetConvertOptions`-

```csharp
using (var converter = new GroupDocs.Conversion.Converter("@YOUR_DOCUMENT_DIRECTORY/sample.ppt"))
{
    SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = GroupDocs.Conversion.FileTypes.SpreadsheetFileType.Xls };
    
    // ดำเนินการแปลงและบันทึกไปยังตำแหน่งที่ระบุ
    converter.Convert(outputFile, options);
}
```
