หัวข้อนี้สาธิตวิธีโหลดไฟล์รูปภาพ JPEG 2000 (.jpm) โดยใช้ GroupDocs.Conversion สำหรับ .NET และแปลงเป็นรูปแบบไฟล์ Excel (.xls)

ก่อนการแปลงให้ระบุเส้นทางอินพุตและเอาต์พุต

```csharp

string outputFile = Path.Combine(outputPath, "jpm-converted-to.xls");
```

##### ขั้นตอนที่ 2: เริ่มต้นตัวแปลง
สร้างอินสแตนซ์ใหม่ของ `Converter` คลาสสำหรับจัดการการแปลงไฟล์

```csharp
using (var converter = new Converter(inputFile))
{
    // ตรรกะการแปลงจะถูกเพิ่มที่นี่

```

กำหนดค่าตัวเลือกการแปลงไฟล์เป็นรูปแบบที่ต้องการ

##### ขั้นตอนที่ 3: ตั้งค่าตัวเลือกการแปลงสเปรดชีต

ดำเนินการแปลงจริงและเก็บผลลัพธ์ไว้ในตำแหน่งที่ระบุ

