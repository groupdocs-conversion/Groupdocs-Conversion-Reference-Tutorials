## คู่มือการใช้งาน
มาแบ่งกระบวนการแปลงออกเป็นขั้นตอนที่สามารถดำเนินการได้

### แปลง XLTX เป็น DOC
**ภาพรวม**:คุณลักษณะนี้สาธิตวิธีการแปลงไฟล์เทมเพลต Excel (.xltx) ให้เป็นเอกสาร Word (DOC) โดยใช้ GroupDocs.Conversion สำหรับ .NET

#### ขั้นตอนที่ 1: ตั้งค่าเส้นทางสำหรับการแปลงเอกสาร
กำหนดเส้นทางสำหรับไฟล์อินพุตและเอาต์พุตของคุณ แทนที่ `"YOUR_DOCUMENT_DIRECTORY"` และ `"YOUR_OUTPUT_DIRECTORY"` ด้วยไดเร็กทอรีจริงบนระบบของคุณ

```csharp
using System.IO;

string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.xltx");
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "xltx-converted-to.doc");
```

#### ขั้นตอนที่ 2: โหลดและแปลงไฟล์
โหลดไฟล์ .xltx ของคุณโดยใช้ `Converter` แบ่งคลาสและกำหนดตัวเลือกการแปลงสำหรับรูปแบบการประมวลผลคำ

```csharp
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

// โหลดไฟล์ต้นฉบับ XLTX
using (var converter = new Converter(inputFilePath))
{
    // กำหนดตัวเลือกการแปลงสำหรับรูปแบบ DOC
    var options = new WordProcessingConvertOptions { Format = WordProcessingFileType.Doc };
    
    // ดำเนินการแปลงและบันทึกไฟล์เอาท์พุต
    converter.Convert(outputFile, options);
}
```

**คำอธิบาย**- 
- **คลาสตัวแปลง**: จัดการการโหลดไฟล์ต้นฉบับ
- **ตัวเลือกการแปลงการประมวลผลคำ**: กำหนดค่าการตั้งค่าเฉพาะสำหรับการแปลงรูปแบบ DOC

#### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่าเส้นทางไปยังไดเร็กทอรีอินพุตและเอาต์พุตถูกต้องและสามารถเข้าถึงได้
- ตรวจสอบว่าคุณมีสิทธิ์เพียงพอในการอ่าน/เขียนไฟล์ในไดเร็กทอรีที่ระบุหรือไม่
- สำหรับข้อผิดพลาด โปรดตรวจสอบเอกสาร GroupDocs.Conversion หรือฟอรัมชุมชนเพื่อรับการสนับสนุนเพิ่มเติม
