## คู่มือการใช้งาน

### การแปลง PPTX เป็น DOCX ทีละขั้นตอน

คุณลักษณะนี้ช่วยให้สามารถแปลงจากงานนำเสนอ PowerPoint เป็นเอกสาร Word ได้อย่างราบรื่น

#### ขั้นตอนที่ 1: เตรียมไฟล์ของคุณ

ตรวจสอบให้แน่ใจว่าเส้นทางอินพุตและเอาต์พุตของคุณถูกต้อง:

```csharp
string inputFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "presentation.pptx");
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
```

#### ขั้นตอนที่ 2: กำหนดค่าตัวเลือกการแปลง

ระบุรูปแบบเป้าหมาย (DOCX):

```csharp
var convertOptions = new WordProcessingConvertOptions();
```

#### ขั้นตอนที่ 3: ดำเนินการแปลง

ดำเนินการแปลงและบันทึกไฟล์ DOCX:

```csharp
string outputFile = Path.Combine(outputFolder, "output.docx");
class ConverterExample {
    public void ConvertPptxToDocx() {
        var converter = new Converter(inputFile);
        converter.Convert(() => File.Create(outputFile), convertOptions);
    }
}
```

**คำอธิบาย**: เดอะ `Convert` วิธีนี้ใช้ฟังก์ชันแลมบ์ดาสำหรับการสร้างไฟล์ เพื่อให้แน่ใจว่าเส้นทางเอาต์พุตถูกต้อง ตัวเลือกการแปลงจะกำหนดโครงสร้าง DOCX

#### เคล็ดลับการแก้ไขปัญหา
- **ข้อผิดพลาดเส้นทางไฟล์**: ตรวจสอบเส้นทางอินพุตและเอาต์พุตอีกครั้ง
- **การขาดสิ่งที่ต้องพึ่งพา**:ตรวจสอบให้แน่ใจว่าแพ็คเกจที่จำเป็นทั้งหมดได้รับการติดตั้งผ่าน NuGet
