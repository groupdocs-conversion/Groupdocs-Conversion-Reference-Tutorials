## คู่มือการใช้งาน
### โหลดไฟล์ EML
**ภาพรวม:**
ขั้นตอนแรกในการแปลงคือการโหลดไฟล์ EML ซึ่งเกี่ยวข้องกับการเริ่มต้น `Converter` วัตถุที่มีเส้นทางไฟล์ EML ของคุณ

#### ขั้นตอนที่ 1: ระบุเส้นทางเอกสารของคุณ
กำหนดตำแหน่งไฟล์ EML ของคุณ:

```csharp
string sampleEmlPath = "YOUR_DOCUMENT_DIRECTORY\\sample.eml";
```

#### ขั้นตอนที่ 2: เริ่มต้นและโหลดไฟล์
โหลดไฟล์ EML ของคุณโดยใช้ `Converter` ระดับ:

```csharp
using (var converter = new Converter(sampleEmlPath))
{
    // EML ที่โหลดพร้อมสำหรับการแปลงหรือการดำเนินการอื่น ๆ
}
```
**คำอธิบาย:** 
รหัสนี้จะเริ่มต้น `Converter` วัตถุที่มีไฟล์ EML ที่คุณระบุ โดยใช้ `using` คำชี้แจงเพื่อให้แน่ใจว่าทรัพยากรได้รับการปล่อยออกอย่างถูกต้อง

### แปลง EML เป็น DOCX
**ภาพรวม:**
การแปลงไฟล์ EML เป็นรูปแบบ DOCX ช่วยเพิ่มความเข้ากันได้และการบูรณาการกับแอปพลิเคชันประมวลผลคำ

#### ขั้นตอนที่ 1: ตั้งค่าเส้นทางเอาต์พุตและเริ่มต้นตัวแปลง
ระบุตำแหน่งที่จะบันทึกไฟล์ที่แปลงแล้ว:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = System.IO.Path.Combine(outputFolder, "eml-converted-to.docx");

using (var converter = new Converter(sampleEmlPath))
{
    // ขั้นตอนการแปลงมีดังนี้
}
```
#### ขั้นตอนที่ 2: ระบุตัวเลือกการแปลง
ตั้งค่าตัวเลือกการแปลงสำหรับ DOCX:

```csharp
var options = new WordProcessingConvertOptions();
```

#### ขั้นตอนที่ 3: ดำเนินการแปลง
ดำเนินการและบันทึกไฟล์เอาท์พุต:

```csharp
converter.Convert(outputFile, options);
```
**คำอธิบาย:** 
โค้ดนี้จะแปลง EML ที่โหลดไปเป็นรูปแบบ DOCX โดยใช้ที่ระบุ `WordProcessingConvertOptions`โดยบันทึกไว้ในเส้นทางที่คุณกำหนด
