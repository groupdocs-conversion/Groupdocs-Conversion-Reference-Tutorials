## คู่มือการใช้งาน

ตอนนี้เรามาแปลงไฟล์ IGS เป็นรูปแบบ HTML ทีละขั้นตอนกัน

### ขั้นตอนที่ 1: กำหนดเส้นทางไฟล์

ขั้นแรก ให้ระบุเส้นทางสำหรับไฟล์อินพุตและเอาต์พุตของคุณ:

```csharp
using System.IO;

// ตั้งค่าเส้นทางไฟล์ IGS ต้นทางโดยใช้ตัวแทนไดเร็กทอรีเอกสารของคุณ
string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.igs");

// กำหนดเส้นทางไฟล์ HTML เอาท์พุตโดยใช้ตัวแทนไดเร็กทอรีเอาท์พุตของคุณ
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "igs-converted-to.html");
```

### ขั้นตอนที่ 2: เริ่มต้นตัวแปลง

ตั้งค่า `Converter` วัตถุที่มีไฟล์อินพุต IGS:

```csharp
using GroupDocs.Conversion;

// เริ่มต้นวัตถุ Converter ด้วยไฟล์ IGS อินพุต
using (var converter = new Converter(inputFilePath))
{
    // โค้ดการแปลงจะอยู่ที่นี่
}
```

### ขั้นตอนที่ 3: ตั้งค่าตัวเลือกการแปลง

กำหนดค่าการตั้งค่าการแปลงสำหรับรูปแบบ HTML โดยใช้ `WebConvertOptions`-

```csharp
using GroupDocs.Conversion.Options.Convert;

// ตั้งค่าตัวเลือกการแปลงสำหรับรูปแบบ HTML
var options = new WebConvertOptions();
```

### ขั้นตอนที่ 4: ดำเนินการแปลง

สุดท้ายให้ดำเนินการแปลงและบันทึกผลลัพธ์ไปยังเส้นทางที่คุณระบุ:

```csharp
// แปลง IGS เป็น HTML และบันทึกไว้
converter.Convert(outputFile, options);
```

โค้ดสั้นๆ นี้จะแปลงไฟล์ IGS เป็นเอกสาร HTML ได้อย่างมีประสิทธิภาพ `WebConvertOptions` ช่วยให้คุณระบุการตั้งค่าเพิ่มเติมเช่นช่วงหน้าหรือเทมเพลตที่กำหนดเองหากจำเป็น

### เคล็ดลับการแก้ไขปัญหา

- ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์อินพุตของคุณถูกต้อง
- ตรวจสอบว่าไดเรกทอรีเอาต์พุตมีอยู่ มิฉะนั้นให้สร้างขึ้นโดยใช้โปรแกรม
- ตรวจสอบข้อยกเว้นใดๆ ที่เกิดขึ้นจาก GroupDocs.Conversion และจัดการอย่างเหมาะสมในโค้ดของคุณ
