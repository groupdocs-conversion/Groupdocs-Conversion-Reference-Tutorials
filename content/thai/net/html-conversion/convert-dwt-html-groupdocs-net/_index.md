## คู่มือการใช้งาน

มาแบ่งกระบวนการแปลงออกเป็นขั้นตอนที่สามารถจัดการได้:

### โหลดไฟล์ DWT ต้นฉบับ

**ภาพรวม**การโหลดไฟล์ DWT ต้นฉบับเป็นขั้นตอนแรกในการเตรียมการแปลง

#### การเริ่มต้นวัตถุตัวแปลง
```csharp
using System;
using GroupDocs.Conversion;

string documentPath = @"YOUR_DOCUMENT_DIRECTORY\sample.dwt";

// เริ่มต้นวัตถุ Converter ด้วยเส้นทางไฟล์ DWT
using (var converter = new Converter(documentPath))
{
    // สามารถดำเนินการเพิ่มเติมกับเอกสารที่โหลดไว้ที่นี่ได้หากจำเป็น
}
```

**คำอธิบาย**: เดอะ `Converter` คลาสจาก GroupDocs.Conversion จัดการการโหลดไฟล์และตั้งค่าสภาพแวดล้อมสำหรับการดำเนินการแปลงเพิ่มเติม

### แปลง DWT เป็น HTML

**ภาพรวม**:คุณสมบัตินี้จะแปลงไฟล์ DWT เป็นรูปแบบ HTML ทำให้สามารถเข้าถึงได้บนแพลตฟอร์มเว็บ

#### โหลดและกำหนดค่าตัวเลือกการแปลง
```csharp
using System;
using System.IO;
using GroupDocs.Conversion.Options.Convert;

string documentPath = @"YOUR_DOCUMENT_DIRECTORY\sample.dwt";
string outputFolder = @"YOUR_OUTPUT_DIRECTORY\";
string outputFile = Path.Combine(outputFolder, "dwt-converted-to.html");

// โหลดไฟล์ DWT ต้นฉบับโดยใช้คลาส Converter
using (var converter = new Converter(documentPath))
{
    // กำหนดค่าตัวเลือกการแปลงสำหรับรูปแบบ HTML
    var options = new WebConvertOptions();
    
    // ตรวจสอบให้แน่ใจว่ามีไดเร็กทอรีเอาท์พุตอยู่
    Directory.CreateDirectory(outputFolder);
    
    // ดำเนินการแปลงและบันทึกผลลัพธ์ไปยังเส้นทางที่ระบุ
    converter.Convert(outputFile, options);
}
```

**คำอธิบาย**- 
- **ตัวเลือกการแปลงเว็บ**: กำหนดค่าการตั้งค่าที่เฉพาะเจาะจงสำหรับการแปลง HTML
- **ไดเรกทอรี.สร้างไดเรกทอรี()**: ช่วยให้แน่ใจว่าโฟลเดอร์เอาต์พุตพร้อมใช้งาน ป้องกันข้อผิดพลาดขณะรันไทม์

### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ได้รับการตั้งค่าอย่างถูกต้องเพื่อหลีกเลี่ยง `FileNotFoundException`-
- ตรวจสอบว่าโครงการของคุณอ้างอิงไลบรารี GroupDocs ที่จำเป็นทั้งหมด
