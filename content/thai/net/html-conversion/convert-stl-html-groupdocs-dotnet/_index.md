## คู่มือการใช้งาน

### คุณสมบัติ: การแปลง STL เป็น HTML
ฟีเจอร์นี้ช่วยให้คุณแปลงไฟล์ STL เป็นรูปแบบ HTML เพิ่มการเข้าถึงและการบูรณาการในสภาพแวดล้อมเว็บ

#### ขั้นตอนที่ 1: เตรียมเส้นทางไฟล์ของคุณ
ตรวจสอบให้แน่ใจว่าไดเร็กทอรีอินพุตและเอาต์พุตของคุณถูกตั้งค่าอย่างถูกต้องโดยใช้ตัวแทนเพื่อความยืดหยุ่น
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// กำหนดเส้นทางไฟล์
string inputFilePath = Path.Combine(documentDirectory, "sample.stl");
string outputFile = Path.Combine(outputDirectory, "stl-converted-to.html");
```

#### ขั้นตอนที่ 2: กำหนดค่าตัวเลือกการแปลง
ตั้งค่าตัวเลือกที่จำเป็นสำหรับการแปลงเป็นรูปแบบ HTML
```csharp
using GroupDocs.Conversion.Options.Convert;

var options = new WebConvertOptions();
// นี่ระบุว่าเรากำลังกำหนดเป้าหมายผลลัพธ์ HTML
```

#### ขั้นตอนที่ 3: ดำเนินการแปลง
ดำเนินการตามกระบวนการแปลงและบันทึกผลลัพธ์เป็นไฟล์ HTML
```csharp
using (var converter = new Converter(inputFilePath))
{
    converter.Convert(outputFile, options); // แปลง STL เป็น HTML และบันทึก
}
```

3. **เครื่องมือทางการศึกษา**:ใช้ไฟล์ HTML ที่แปลงแล้วเป็นส่วนหนึ่งของทรัพยากรการศึกษาออนไลน์ ช่วยให้ผู้เรียนสามารถโต้ตอบกับโครงสร้างสามมิติได้
