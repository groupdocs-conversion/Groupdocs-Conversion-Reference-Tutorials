## คู่มือการใช้งาน

ตอนนี้เรามาแบ่งกระบวนการแปลงออกเป็นขั้นตอนที่สามารถจัดการได้:

### แปลงไฟล์ XLTX เป็น HTML

#### ภาพรวม
ฟีเจอร์นี้ช่วยให้คุณแปลงไฟล์เทมเพลต Excel (XLTX) เป็นรูปแบบ HTML เพื่อการดูและแบ่งปันบนเว็บได้อย่างง่ายดาย

#### การดำเนินการแบบทีละขั้นตอน

**1. โหลดไฟล์ต้นฉบับ XLTX**

```csharp
string inputFilePath = @"path\to\your\xltxfile.xltx";
using (var converter = new Converter(inputFilePath))
{
    // การแปลงจะเกิดขึ้นภายในบล็อคนี้
}
```

*ทำไม*:การโหลดไฟล์จะเป็นการเริ่มต้นทำการแปลง และตั้งค่าสภาพแวดล้อมเพื่อใช้การแปลง

**2. ตั้งค่าตัวเลือกการแปลง**

```csharp
var options = new WebConvertOptions();
```

*คำอธิบาย*- `WebConvertOptions` ได้รับการออกแบบมาให้เหมาะกับรูปแบบที่เป็นมิตรต่อเว็บ เช่น HTML เพื่อให้มั่นใจได้ว่าจะเข้ากันได้กับเบราว์เซอร์

**3. ดำเนินการแปลงและบันทึกผลลัพธ์**

```csharp
string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "xltx-converted-to.html");

converter.Convert(outputFile, options);
```

*ทำไม*วิธีการนี้จะดำเนินการแปลงโดยใช้ตัวเลือกที่ระบุเพื่อสร้างไฟล์ HTML

**เคล็ดลับการแก้ไขปัญหา**
- ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์อินพุตของคุณถูกต้อง
- ตรวจสอบว่าไดเร็กทอรีเอาต์พุตของคุณมีอยู่และมีสิทธิ์ในการเขียน
