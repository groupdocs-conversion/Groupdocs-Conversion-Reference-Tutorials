## คู่มือการใช้งาน

### คุณสมบัติ: แปลง JPF เป็น HTML

#### ภาพรวม
การแปลงไฟล์ JPF เป็นเอกสาร HTML ช่วยให้แสดงรูปภาพบนเว็บได้อย่างราบรื่น ทำตามขั้นตอนเหล่านี้โดยใช้ GroupDocs.Conversion สำหรับ .NET:

##### ขั้นตอนที่ 1: กำหนดการตั้งค่าเอาท์พุต

ระบุไดเรกทอรีและชื่อไฟล์ที่จะบันทึกไฟล์ HTML ที่แปลงแล้ว:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "jpf-converted-to.html");
```
**คำอธิบาย**- `outputFolder` คือสถานที่จัดเก็บที่คุณกำหนดไว้ `outputFile` รวมเส้นทางนี้กับชื่อไฟล์ที่ต้องการ

##### ขั้นตอนที่ 2: โหลดไฟล์ต้นฉบับ JPF

โหลดและเตรียมไฟล์ต้นฉบับสำหรับการแปลง:

```csharp
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY/sample.jpf"))
{
    var options = new WebConvertOptions();
    
    // แปลงและบันทึกไฟล์ JPF เป็นเอกสาร HTML
    converter.Convert(outputFile, options);
}
```
**คำอธิบาย**- `Converter` เริ่มต้นด้วยเส้นทางแหล่งที่มาของ JPF `WebConvertOptions()` กำหนดค่าการตั้งค่าที่เฉพาะเจาะจงสำหรับการแปลง HTML

##### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่าเส้นทางไดเร็กทอรีของคุณถูกต้องเพื่อป้องกันข้อผิดพลาดไม่พบไฟล์
