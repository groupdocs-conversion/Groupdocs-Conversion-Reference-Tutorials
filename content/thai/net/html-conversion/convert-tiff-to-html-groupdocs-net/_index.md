## คู่มือการใช้งาน

ให้เราแบ่งการใช้งานออกเป็นขั้นตอนที่ชัดเจน โดยเน้นที่คุณลักษณะแต่ละอย่าง

### คุณสมบัติ: การแปลง TIFF เป็น HTML

#### ภาพรวม:
การแปลงไฟล์ TIFF เป็นรูปแบบ HTML ช่วยให้มีความยืดหยุ่นและเข้าถึงเนื้อหาภาพบนแพลตฟอร์มเว็บได้มากขึ้น

##### ขั้นตอนที่ 1: กำหนดค่าเส้นทางของคุณ
สร้างค่าคงที่สำหรับไดเรกทอรีเอกสารและโฟลเดอร์เอาท์พุตของคุณ:

```csharp
public static class Constants
{
    public const string SAMPLE_TIFF = @"YOUR_DOCUMENT_DIRECTORY\sample.tiff";

    public static string GetOutputDirectoryPath()
    {
        return Path.Combine(Directory.GetCurrentDirectory(), "output");
    }
}
```
*หมายเหตุ: เปลี่ยน `YOUR_DOCUMENT_DIRECTORY` พร้อมเส้นทางจริงไปยังไฟล์ TIFF ของคุณ*

##### ขั้นตอนที่ 2: ดำเนินการแปลง
ใช้โค้ดสั้นๆ ต่อไปนี้เพื่อแปลงภาพ TIFF เป็น HTML:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

string outputFolder = Constants.GetOutputDirectoryPath();
string outputFile = Path.Combine(outputFolder, "tiff-converted-to.html");

// โหลดไฟล์ TIFF ต้นฉบับ
using (var converter = new Converter(Constants.SAMPLE_TIFF))
{
    // กำหนดค่าตัวเลือกการแปลงสำหรับรูปแบบ HTML
    var options = new WebConvertOptions();

    // ดำเนินการแปลงและบันทึกผลลัพธ์เป็นไฟล์ HTML
    converter.Convert(outputFile, options);
}
```
- **พารามิเตอร์**- `Constants.SAMPLE_TIFF` ระบุไฟล์ TIFF อินพุต 
- **ค่าส่งคืน**:วิธีการนี้จะไม่ส่งคืนอะไรเลยนอกจากจะบันทึก HTML ที่แปลงแล้วไปยังเส้นทางที่ระบุ
- **วิธีการ วัตถุประสงค์**: เดอะ `Convert` วิธีการแปลงไฟล์ TIFF ให้เป็นเอกสาร HTML โดยใช้การตั้งค่าการแปลงที่กำหนดไว้ล่วงหน้า

##### เคล็ดลับการแก้ไขปัญหา:
- ตรวจสอบให้แน่ใจว่าเส้นทางทั้งหมดได้รับการตั้งค่าอย่างถูกต้องและสามารถเข้าถึงได้
- ตรวจสอบสิทธิ์ไฟล์หากคุณพบปัญหาในการเข้าถึง
- ตรวจสอบว่าไลบรารี GroupDocs มีการอ้างอิงอย่างถูกต้องในโครงการของคุณ

### คุณสมบัติ: การตั้งค่าเส้นทางไดเรกทอรี
การกำหนดค่าไดเร็กทอรีอย่างแม่นยำช่วยให้การทำงานราบรื่น คุณลักษณะนี้แสดงให้เห็นการตั้งค่าเส้นทางเอกสารอินพุตและไดเร็กทอรีเอาท์พุตอย่างมีประสิทธิภาพ

ต่อไปนี้คือสถานการณ์จริงบางสถานการณ์ที่การแปลง TIFF เป็น HTML สามารถเป็นประโยชน์ได้:
