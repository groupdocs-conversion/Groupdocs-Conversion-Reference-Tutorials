## คู่มือการใช้งาน

ตอนนี้เรามาแบ่งกระบวนการแปลงออกเป็นขั้นตอนตรรกะ:

### คุณสมบัติ: การแปลง FODP เป็น HTML

#### ภาพรวม
ฟีเจอร์นี้สาธิตวิธีการแปลงไฟล์ .fodp เป็นรูปแบบ HTML โดยใช้ GroupDocs.Conversion สำหรับ .NET

##### ขั้นตอนที่ 1: โหลดเอกสาร

```csharp
class Program
{
    static void Main()
    {
        string inputPath = "path/to/your/fodpfile.fodp";
        Converter converter = new Converter(inputPath);
        
        Console.WriteLine("Document loaded successfully.");
    }
}
```

การ `Converter` คลาสจะจัดการการโหลดเอกสาร เส้นทางอินพุตจะระบุตำแหน่งของไฟล์ต้นฉบับของคุณ

##### ขั้นตอนที่ 2: ตั้งค่าตัวเลือกการแปลง

```csharp
class Program
{
    static void Main()
    {
        string inputPath = "path/to/your/fodpfile.fodp";
        Converter converter = new Converter(inputPath);
        
        var options = new MarkupConvertOptions();
        
        Console.WriteLine("Conversion options set for HTML format.");
    }
}
```

ที่นี่, `MarkupConvertOptions` กำหนดค่าการแปลงให้กำหนดเป้าหมายเอาท์พุต HTML

##### ขั้นตอนที่ 3: ดำเนินการแปลง

```csharp
class Program
{
    static void Main()
    {
        string inputPath = "path/to/your/fodpfile.fodp";
        Converter converter = new Converter(inputPath);
        var options = new MarkupConvertOptions();
        
        string outputPath = Path.Combine("output/directory", "output.html");
        converter.Convert(outputPath, options);
        
        Console.WriteLine($"Document converted and saved to {outputPath}.");
    }
}
```

การ `Convert` วิธีการดำเนินการแปลง ตรวจสอบให้แน่ใจว่าคุณ `outputPath` ได้ถูกตั้งค่าอย่างถูกต้องไปยังตำแหน่งที่คุณต้องการบันทึกไฟล์ที่แปลงแล้ว

##### เคล็ดลับการแก้ไขปัญหา:
- **ไม่พบไฟล์**: ตรวจสอบเส้นทางอินพุตว่ามีการพิมพ์ผิดหรือโครงสร้างไดเร็กทอรีไม่ถูกต้อง
- **ปัญหาการอนุญาต**: ตรวจสอบสิทธิ์ไฟล์หากคุณพบข้อผิดพลาดในการเข้าถึง
