## คู่มือการใช้งาน

มาเรียนรู้การแปลงไฟล์ CGM เป็น HTML โดยใช้ GroupDocs.Conversion กัน

### แปลง CGM เป็น HTML
คุณสมบัตินี้ช่วยให้คุณแปลงไฟล์รูปแบบ Computer Graphics Metafile (.cgm) เป็น Hyper Text Markup Language (.html) ได้

#### ภาพรวมทีละขั้นตอน
1. **ตั้งค่าไดเรกทอรีเอาท์พุต**
2. **เริ่มต้นตัวแปลงและโหลดไฟล์ CGM**
3. **กำหนดค่าตัวเลือกการแปลง**
4. **ดำเนินการแปลง**

#### ตั้งค่าไดเรกทอรีเอาท์พุต
กำหนดเส้นทางไดเร็กทอรีเอาท์พุตของคุณที่จะบันทึกไฟล์ HTML:

```csharp
string outputFolder = @"YOUR_OUTPUT_DIRECTORY/";
```

#### เริ่มต้นตัวแปลงและโหลดไฟล์ CGM
โหลดไฟล์ CGM ต้นทางของคุณโดยใช้ GroupDocs.Conversion:

```csharp
using (var converter = new GroupDocs.Conversion.Converter(@"YOUR_DOCUMENT_DIRECTORY/sample.cgm"))
{
    // ขั้นตอนการแปลงจะตามมาที่นี่
}
```

#### กำหนดค่าตัวเลือกการแปลง
สร้างตัวเลือกการแปลงเฉพาะสำหรับรูปแบบ HTML:

```csharp
var options = new WebConvertOptions();
```

#### ดำเนินการแปลง
ดำเนินการแปลงและบันทึกผลลัพธ์เป็นไฟล์ HTML:

```csharp
string outputFile = Path.Combine(outputFolder, "cgm-converted-to.html");
converter.Convert(outputFile, options);
```
