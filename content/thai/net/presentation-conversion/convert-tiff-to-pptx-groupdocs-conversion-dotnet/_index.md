## คู่มือการใช้งาน
มาแบ่งการใช้งานออกเป็นส่วนๆ ตามตรรกะกัน

### แปลง TIFF เป็น PPTX
ฟีเจอร์นี้ช่วยให้คุณแปลงไฟล์ TIFF เป็นรูปแบบ PowerPoint (PPTX) ได้ ช่วยเพิ่มความสามารถในการนำเสนอภาพในแบบสไลด์โชว์หรือการนำเสนอต่างๆ

#### ขั้นตอนที่ 1: กำหนดเส้นทาง
ตั้งค่าเส้นทางสำหรับไดเร็กทอรีทั้งอินพุตและเอาต์พุต:
```csharp
using System.IO;

string documentDirectory = Path.Combine("YOUR_DOCUMENT_DIRECTORY");
string outputDirectory = Path.Combine("YOUR_OUTPUT_DIRECTORY");

string inputFilePath = Path.Combine(documentDirectory, "sample.tiff");
string outputFile = Path.Combine(outputDirectory, "tiff-converted-to.pptx");
```

#### ขั้นตอนที่ 2: โหลดไฟล์ TIFF ต้นฉบับ
ใช้ GroupDocs.Conversion เพื่อโหลดไฟล์ TIFF ต้นฉบับของคุณ:
```csharp
using (var converter = new Converter(inputFilePath))
{
    Console.WriteLine("TIFF file loaded successfully.");
}
```

#### ขั้นตอนที่ 3: เริ่มต้นตัวเลือกการแปลง
ตั้งค่าตัวเลือกการแปลงที่เหมาะกับรูปแบบ PPTX:
```csharp
var options = new PresentationConvertOptions();
Console.WriteLine("Conversion options set.");
```

#### ขั้นตอนที่ 4: ดำเนินการแปลง
ดำเนินการแปลงและบันทึกผลลัพธ์เป็นไฟล์ PPTX:
```csharp
converter.Convert(outputFile, options);
Console.WriteLine("Conversion completed successfully.");
```
