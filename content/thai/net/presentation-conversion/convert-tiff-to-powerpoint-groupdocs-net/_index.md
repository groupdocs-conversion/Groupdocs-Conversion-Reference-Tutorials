## คู่มือการใช้งาน

เมื่อตั้งค่า GroupDocs.Conversion เรียบร้อยแล้ว ให้เราใช้งานฟีเจอร์การแปลง TIFF เป็น PPT ได้เลย

### แปลงไฟล์ TIFF เป็นไฟล์นำเสนอ PowerPoint
ปฏิบัติตามขั้นตอนเหล่านี้:

#### ขั้นตอนที่ 1: กำหนดเส้นทางไดเร็กทอรี
ระบุเอกสารต้นฉบับและตำแหน่งไฟล์เอาท์พุตของคุณ:
```csharp
const string DocumentDirectory = "YOUR_DOCUMENT_DIRECTORY";
const string OutputDirectory = "YOUR_OUTPUT_DIRECTORY/";
```

#### ขั้นตอนที่ 2: โหลดไฟล์ TIFF ต้นฉบับ
โหลดไฟล์ TIFF เพื่อการแปลง:
```csharp
string inputFilePath = Path.Combine(DocumentDirectory, "sample.tiff");
```

#### ขั้นตอนที่ 3: ตั้งค่าตัวเลือกการแปลงสำหรับ PowerPoint
เริ่มต้นตัวเลือกและตั้งค่ารูปแบบเป้าหมายเป็น PPT:
```csharp
var options = new PresentationConvertOptions
{
    Format = PresentationFileType.Ppt // ส่งออกเป็นรูปแบบ PowerPoint
};
```

#### ขั้นตอนที่ 4: แปลงและบันทึกไฟล์
ใช้ `Converter` คลาสสำหรับการแปลงโดยส่งตัวเลือกที่คุณระบุ:
```csharp
using (var converter = new Converter(inputFilePath))
{
    string outputFile = Path.Combine(OutputDirectory, "tiff-converted-to.ppt");
    converter.Convert(outputFile, options);
}
```
