### โหลดและแปลงไฟล์ MOBI เป็น PPT
ฟีเจอร์นี้สาธิตการโหลดไฟล์ MOBI และการแปลงเป็นรูปแบบ PPT โดยคุณสามารถทำตามขั้นตอนต่อไปนี้:

#### ขั้นตอนที่ 1: กำหนดเส้นทาง
ขั้นแรก กำหนดเส้นทางไดเร็กทอรีสำหรับเอกสารและไฟล์เอาต์พุตของคุณ

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
```

ตัวแทนเหล่านี้จะถูกแทนที่ด้วยเส้นทางจริงในสภาพแวดล้อมของคุณ

#### ขั้นตอนที่ 2: ตั้งค่าเส้นทางไฟล์
สร้างเส้นทางแบบเต็มไปยังไฟล์ MOBI ต้นทางและไฟล์ PPT เอาท์พุตที่ต้องการ

```csharp
string mobiFilePath = Path.Combine(documentDirectory, "sample.mobi");
string outputFile = Path.Combine(outputDirectory, "mobi-converted-to.ppt");
```

#### ขั้นตอนที่ 3: เริ่มต้นตัวแปลง
เริ่มต้นวัตถุ GroupDocs.Converter ด้วยเส้นทางไฟล์ MOBI ของคุณ

```csharp
using (var converter = new Converter(mobiFilePath))
{
    // โค้ดการแปลงจะอยู่ที่นี่
}
```

ขั้นตอนนี้จะโหลดไฟล์ MOBI และเตรียมพร้อมสำหรับการแปลง

#### ขั้นตอนที่ 4: กำหนดค่าตัวเลือกการแปลง
ตั้งค่าตัวเลือกการแปลงการนำเสนอ โดยระบุ PPT เป็นรูปแบบเป้าหมาย

```csharp
PresentationConvertOptions options = new PresentationConvertOptions { Format = PresentationFileType.Ppt };
```

การ `PresentationConvertOptions` คลาสนี้ช่วยให้คุณสามารถปรับแต่งด้านต่างๆ ของไฟล์เอาต์พุตได้ เช่น การตั้งค่ารูปแบบไฟล์ที่ต้องการ

#### ขั้นตอนที่ 5: ดำเนินการแปลง
สุดท้ายดำเนินการแปลงและบันทึกไฟล์ PPT เอาท์พุต

```csharp
converter.Convert(outputFile, options);
Console.WriteLine("Conversion completed successfully.");
```

ขั้นตอนนี้จะเปลี่ยน MOBI eBook ของคุณให้เป็นงานนำเสนอ PowerPoint ทำให้พร้อมสำหรับการใช้ในการนำเสนอหรือแอปพลิเคชันอื่นๆ
