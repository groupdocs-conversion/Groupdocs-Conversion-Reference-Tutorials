using System;

    // ดำเนินการแปลงและบันทึกไฟล์ PPTX ที่ได้

### คุณสมบัติ: แปลง XLSM เป็น PPTX

**ภาพรวม**

ฟีเจอร์นี้จะทำการแปลงสเปรดชีต Excel ที่รองรับแมโคร (XLSM) เป็นงานนำเสนอ PowerPoint ได้โดยอัตโนมัติ ช่วยให้กระบวนการจัดการเอกสารของคุณมีประสิทธิภาพมากขึ้น

#### 1. โหลดไฟล์ XLSM ต้นฉบับ

เริ่มต้นด้วยการโหลดไฟล์ต้นฉบับของคุณโดยใช้ `Converter` ระดับ:

```csharp
string inputFile = "path/to/your/sample.xlsm";
using (var converter = new Converter(inputFile))
{
    // ตรรกะการแปลงอยู่ที่นี่
}
```

*ทำไม:* การโหลดไฟล์จะเริ่มกระบวนการแปลง ซึ่งจะทำให้คุณสามารถใช้การแปลงและตัวเลือกเอาต์พุตต่างๆ ได้

#### 2. เริ่มต้นตัวเลือกการแปลง

กำหนดพารามิเตอร์การแปลงเฉพาะโดยใช้ `PresentationConvertOptions`-

```csharp
var options = new PresentationConvertOptions();
```

*ทำไม:* ขั้นตอนนี้จะกำหนดค่าว่าเอกสารของคุณจะถูกแปลงเป็นรูปแบบ PPTX อย่างไร โดยให้แน่ใจว่าได้ใช้การตั้งค่าที่จำเป็นทั้งหมดแล้ว

#### 3. บันทึกไฟล์ที่แปลงแล้ว

สุดท้ายให้บันทึกไฟล์ที่แปลงแล้วไปยังตำแหน่งที่คุณต้องการ:

```csharp
string outputFile = Path.Combine(outputFolder, "xlsm-converted-to.pptx");
converter.Convert(outputFile, options);
```

*ทำไม:* ขั้นตอนสำคัญนี้จะเขียนไฟล์เอาต์พุตลงในไดเร็กทอรีที่ระบุ ทำให้กระบวนการแปลงเสร็จสมบูรณ์

**เคล็ดลับการแก้ไขปัญหา**

- **ไฟล์ที่หายไป:** ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ทั้งหมดถูกต้องและสามารถเข้าถึงได้
- **เวอร์ชันไม่ตรงกัน:** ตรวจสอบให้แน่ใจว่าคุณกำลังใช้ GroupDocs.Conversion เวอร์ชันที่เข้ากันได้สำหรับ .NET
