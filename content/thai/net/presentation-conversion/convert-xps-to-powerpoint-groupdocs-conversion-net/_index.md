## คู่มือการใช้งาน
มาแปลงไฟล์ XPS เป็นงานนำเสนอ PowerPoint กัน

### ขั้นตอนที่ 1: ตั้งค่าเส้นทางและแปลงตัวเลือก
กำหนดเส้นทางสำหรับไฟล์ XPS อินพุตและไฟล์ PPT เอาท์พุต:
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY"; // แทนที่ด้วยเส้นทางจริง
string outputDirectory = "YOUR_OUTPUT_DIRECTORY"; // แทนที่ด้วยเส้นทางจริง

// กำหนดเส้นทางไฟล์ต้นทางและปลายทาง
string sourceXpsFilePath = Path.Combine(documentDirectory, "sample.xps");
string convertedPptFilePath = Path.Combine(outputDirectory, "xps-converted-to.ppt");

using (var converter = new Converter(sourceXpsFilePath))
{
    PresentationConvertOptions options = new PresentationConvertOptions { Format = PresentationFileType.Ppt };

    // แปลงและบันทึกไฟล์เอาท์พุต
    converter.Convert(convertedPptFilePath, options);
}
```
#### คำอธิบาย:
- **เส้นทาง**: ชุด `documentDirectory` และ `outputDirectory` ไปยังไดเร็กทอรีเฉพาะของคุณ
- **การนำเสนอการแปลงตัวเลือก**: ระบุการแปลงเป็นรูปแบบ PowerPoint (PPT)

- ตรวจสอบว่าเวอร์ชัน GroupDocs.Conversion เข้ากันได้กับ .NET framework ของคุณ
