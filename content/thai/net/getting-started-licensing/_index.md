# บทช่วยสอนการเริ่มต้นใช้งานและการออกใบอนุญาต GroupDocs.Conversion สำหรับนักพัฒนา .NET

เริ่มต้นการเดินทางของคุณด้วยบทช่วยสอน GroupDocs.Conversion ที่จำเป็นเหล่านี้ ซึ่งจะแนะนำคุณเกี่ยวกับการติดตั้ง การกำหนดค่าใบอนุญาต และการแปลงเอกสารครั้งแรกของคุณใน .NET คำแนะนำสำหรับผู้เริ่มต้นใช้งานของเราจะแสดงวิธีการตั้งค่าสภาพแวดล้อมการพัฒนาของคุณ เริ่มต้นไลบรารี GroupDocs.Conversion กำหนดค่าใบอนุญาตจากไฟล์หรือสตรีม ตั้งค่าใบอนุญาตแบบคิดค่าบริการตามปริมาณ และดำเนินการแปลงขั้นพื้นฐานโดยใช้ทั้ง API มาตรฐานและ API ที่คล่องแคล่ว ทำตามบทช่วยสอนเหล่านี้เพื่อเรียนรู้พื้นฐานอย่างรวดเร็วและเตรียมพร้อมสำหรับงานแปลงเอกสารขั้นสูงในแอปพลิเคชัน .NET ของคุณ

### [วิธีตั้งค่าและสมัครใบอนุญาตสำหรับ GroupDocs.Conversion .NET: คำแนะนำทีละขั้นตอน](./groupdocs-conversion-net-license-setup/)

### [การนำระบบอนุญาตใช้งานแบบมิเตอร์ไปใช้งานกับ GroupDocs.Conversion สำหรับ .NET: คู่มือฉบับสมบูรณ์](./metered-licensing-groupdocs-conversion-dotnet/)
เรียนรู้วิธีการนำระบบออกใบอนุญาตแบบคิดค่าบริการตามปริมาณการใช้งานมาใช้กับ GroupDocs.Conversion สำหรับ .NET จัดการต้นทุนอย่างมีประสิทธิภาพพร้อมใช้ประโยชน์จากฟีเจอร์การแปลงเอกสารอันทรงพลัง
