## การนำเข้าเนมสเปซ
ในโครงการ .NET ของคุณ นำเข้าเนมสเปซที่จำเป็นเพื่อเข้าถึงฟังก์ชันการทำงานของ GroupDocs.Conversion
```csharp
using System;
using System.IO;
using GroupDocs.Conversion.Options.Convert;
```
## ขั้นตอนที่ 1: กำหนดตำแหน่งไฟล์เอาท์พุต
ตั้งค่าไดเร็กทอรีที่คุณต้องการจัดเก็บไฟล์ PDF ที่แปลงแล้ว
```csharp

string outputFile = Path.Combine(outputFolder, "igs-converted-to.pdf");
```

ใช้ไลบรารี GroupDocs.Conversion เพื่อโหลดไฟล์ IGS แหล่งที่มาที่คุณต้องการแปลง
```csharp
using (var converter = new GroupDocs.Conversion.Converter(Constants.SAMPLE_IGS))
```

ระบุตัวเลือกการแปลง เช่น เลือก PDF เป็นรูปแบบเป้าหมาย
```csharp
var options = new PdfConvertOptions();
```
## ขั้นตอนที่ 4: ดำเนินการแปลง
ดำเนินการตามกระบวนการแปลงด้วยตัวเลือกที่ระบุ
```csharp
converter.Convert(outputFile, options);
```
