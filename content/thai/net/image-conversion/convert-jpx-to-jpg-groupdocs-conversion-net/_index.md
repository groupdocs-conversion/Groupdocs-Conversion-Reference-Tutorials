ในภูมิทัศน์ดิจิทัลของปัจจุบัน การแปลง JPX เป็น JPG ถือเป็นสิ่งสำคัญสำหรับการเพิ่มประสิทธิภาพเว็บ เนื่องจาก JPG ได้รับการรองรับอย่างกว้างขวางในแพลตฟอร์มและแอปพลิเคชันต่างๆ ปฏิบัติตามคำแนะนำทีละขั้นตอนนี้เพื่อนำการแปลงนี้ไปใช้อย่างราบรื่นในแอปพลิเคชัน .NET ของคุณ

## คู่มือการใช้งาน

หัวข้อนี้จะอธิบายวิธีนำฟีเจอร์ต่างๆ ที่เกี่ยวข้องกับการแปลงไฟล์ JPX เป็น JPG มาใช้

### โหลดไฟล์ JPX เพื่อการแปลง
**ภาพรวม**
การโหลดไฟล์ JPX เป็นขั้นตอนแรกก่อนการแปลง ซึ่งจะเป็นตัวกำหนดแหล่งที่มาของไฟล์ที่จะแปลง

```csharp
using System;
using GroupDocs.Conversion;

string sourceFilePath = "YOUR_DOCUMENT_DIRECTORY/sample.jpx";
Converter converter = new Converter(sourceFilePath);
```
- **พารามิเตอร์**- `sourceFilePath` ควรเป็นเส้นทางไปยังไฟล์ JPX ของคุณ
- **วัตถุประสงค์**เริ่มกระบวนการแปลงด้วยไฟล์ที่ระบุ

### แปลง JPX เป็น JPG
**ภาพรวม**
ขั้นตอนถัดไปคือการแปลงไฟล์ JPX ที่โหลดเป็นรูปแบบ JPG ซึ่งจะทำให้เข้าถึงได้ง่ายขึ้นสำหรับแอปพลิเคชันต่างๆ

```csharp
using System;
using System.IO;
using GroupDocs.Conversion.Options.Convert;

string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");

Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);

ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Jpg };

converter.Convert(getPageStream, options);
```
- **พารามิเตอร์**-
  - `outputFolder`:ไดเร็กทอรีสำหรับบันทึกไฟล์ JPG ที่แปลงแล้ว
  - `getPageStream`:ผู้มอบหมายระบุวิธีการบันทึกแต่ละหน้าในระหว่างการแปลง
  - `options`: การตั้งค่าคอนฟิกสำหรับการแปลงภาพ

- **วัตถุประสงค์**:แปลงและบันทึกหน้า JPX เป็นไฟล์ JPG แยกกันในโฟลเดอร์เอาต์พุตที่ระบุ

**เคล็ดลับการแก้ไขปัญหา**
- ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ต้นทางของคุณถูกต้องเพื่อหลีกเลี่ยงข้อผิดพลาดในการโหลด
- ตรวจสอบสิทธิ์ไดเร็กทอรีหากคุณพบปัญหาในการบันทึกไฟล์
