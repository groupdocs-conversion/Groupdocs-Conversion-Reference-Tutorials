## คู่มือการใช้งาน

ในสไนปเป็ตนี้ เราจะเริ่มต้น `Converter` วัตถุที่มีเส้นทางไปยังไฟล์ XLSM ขั้นตอนนี้จะเตรียมเอกสารของคุณสำหรับการดำเนินการแปลงในขั้นถัดไป

### แปลงเป็นรูปแบบ PSD

#### ภาพรวม

การแปลงเอกสารเป็นรูปแบบต่างๆ จะช่วยปลดล็อกการใช้งานและแอปพลิเคชันใหม่ๆ ที่นี่ เราจะแปลงไฟล์ XLSM ที่โหลดไว้เป็นรูปแบบ PSD โดยใช้ GroupDocs.Conversion

##### ตั้งค่าตัวเลือกการแปลง

```csharp
using System;
using System.IO;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main()
    {
        string outputFolder = "YOUR_OUTPUT_DIRECTORY";
        string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");

        // กำหนดวิธีการจัดการสตรีมเอาท์พุต
        Func<SavePageContext, Stream> getPageStream = savePageContext => 
            new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);

        ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd };

        // แปลงเป็นรูปแบบ PSD โดยใช้ตัวเลือกการแปลงและตัวจัดการสตรีมเอาท์พุตที่ให้มา
        using (Converter converter = new Converter(sampleXlsmPath))
        {
            converter.Convert(getPageStream, options);
        }
    }
}
```

ในโค้ดนี้ `ImageConvertOptions` ได้รับการกำหนดค่าให้แปลงเป็น PSD `getPageStream` ฟังก์ชันนี้จะจัดการว่าแต่ละหน้าของเอกสารที่แปลงแล้วจะถูกบันทึกอย่างไร
