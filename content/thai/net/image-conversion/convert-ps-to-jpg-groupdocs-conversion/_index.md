## คู่มือการใช้งาน

ตอนนี้ มาแบ่งกระบวนการใช้งานออกเป็นขั้นตอนที่จัดการได้สำหรับการแปลงไฟล์ PS เป็น JPG โดยใช้ GroupDocs.Conversion สำหรับ .NET

### ภาพรวมการแปลง PS เป็น JPG

เป้าหมายคือการแปลงแต่ละหน้าของไฟล์ PostScript เป็นภาพ JPG แต่ละภาพ ซึ่งอาจมีประโยชน์โดยเฉพาะอย่างยิ่งสำหรับการเก็บถาวรหรือแชร์เอกสารในรูปแบบที่เข้าถึงได้ทั่วไปมากขึ้น

#### ขั้นตอนที่ 1: กำหนดเส้นทางไฟล์

ก่อนอื่น ให้ตั้งค่าเส้นทางสำหรับไฟล์ PS อินพุตและไดเร็กทอรีเอาต์พุต:
```csharp
using System;
using System.IO;

string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.ps");
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", ".");
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");
```
#### ขั้นตอนที่ 2: สร้างฟังก์ชันสตรีม

กำหนดฟังก์ชันเพื่อรับสตรีมสำหรับการบันทึกแต่ละหน้าของเอกสารที่แปลงแล้ว:
```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext =>
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```
ฟังก์ชันนี้จะทำให้แน่ใจว่าแต่ละหน้าจะได้รับการบันทึกเป็นไฟล์ JPG แต่ละไฟล์

#### ขั้นตอนที่ 3: โหลดและแปลงไฟล์ PS

โหลดไฟล์ PS โดยใช้ GroupDocs.Conversion และตั้งค่าตัวเลือกการแปลง:
```csharp
using (Converter converter = new Converter(inputFilePath))
{
    ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Jpg };
    converter.Convert(getPageStream, options);
}
```
โค้ดชิ้นนี้จัดการการโหลดไฟล์ PS ของคุณ การระบุรูปแบบเอาต์พุตที่ต้องการ และดำเนินการแปลง

### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่าเส้นทางทั้งหมดได้รับการตั้งค่าอย่างถูกต้องเพื่อหลีกเลี่ยง `FileNotFoundException`-
