## คู่มือการใช้งาน

ตอนนี้ทุกอย่างตั้งค่าเสร็จเรียบร้อยแล้ว มาดำเนินการแปลงไฟล์ BMP เป็นรูปแบบ TEX กัน

### แปลงไฟล์ BMP เป็นรูปแบบ TEX

ฟีเจอร์นี้ใช้ GroupDocs.Conversion เพื่อแปลงภาพ BMP เป็นเอกสาร TEX ได้อย่างราบรื่น

#### โหลดไฟล์ BMP ต้นฉบับ
ขั้นแรก โหลดไฟล์ BMP ของคุณโดยใช้ `Converter` คลาส ขั้นตอนนี้จะเริ่มกระบวนการแปลง:

```csharp
using (var converter = new GroupDocs.Conversion.Converter(inputFile))
{
    // ตรรกะการแปลงอยู่ที่นี่
}
```

#### ตั้งค่าตัวเลือกการแปลงสำหรับรูปแบบ TEX
ถัดไปกำหนดรูปแบบผลลัพธ์โดยการตั้งค่า `PageDescriptionLanguageConvertOptions`-

```csharp
PageDescriptionLanguageConvertOptions options = new PageDescriptionLanguageConvertOptions { Format = PageDescriptionLanguageFileType.Tex };
```

การ `options.Format` พารามิเตอร์ระบุว่าเราจะแปลงเป็น TEX ซึ่งจะกำหนดกระบวนการแปลง

#### แปลงและบันทึก
สุดท้ายดำเนินการแปลงและบันทึกผลลัพธ์:

```csharp
converter.Convert(outputFile, options);
```

วิธีนี้จะแปลงไฟล์ BMP เป็นรูปแบบ TEX และบันทึกไว้ในตำแหน่งที่ระบุ
