### การแปลงไฟล์ BMP เป็น PSD
คุณลักษณะนี้เน้นถึงวิธีการแปลงไฟล์ BMP เป็นรูปแบบ PSD โดยใช้ GroupDocs.Conversion

#### ขั้นตอนที่ 1: โหลดไฟล์ BMP ต้นฉบับ
ขั้นแรก ให้ระบุเส้นทางไปยังไฟล์ BMP ของคุณ แทนที่ `"YOUR_DOCUMENT_DIRECTORY"` พร้อมไดเร็กทอรีจริงที่มีไฟล์ BMP ของคุณ
```csharp
string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.bmp");
```

#### ขั้นตอนที่ 2: ตั้งค่าตัวเลือกการแปลงสำหรับรูปแบบ PSD
กำหนดค่าตัวเลือกการแปลงเพื่อกำหนดเป้าหมายรูปแบบ PSD:
```csharp
using GroupDocs.Conversion.Options.Convert;

ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd };
```

#### ขั้นตอนที่ 3: กำหนดเส้นทางเอาต์พุตและแปลงไฟล์
สร้างไดเร็กทอรีสำหรับไฟล์เอาต์พุตของคุณ ถ้ายังไม่มีอยู่ จากนั้นดำเนินการแปลง:
```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", "converted");
if (!Directory.Exists(outputFolder))
    Directory.CreateDirectory(outputFolder);

string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");

Func<SavePageContext, Stream> getPageStream = savePageContext => new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
converter.Convert(getPageStream, options);
```

#### ขั้นตอนที่ 1: กำหนดเส้นทางไดเรกทอรีฐาน


#### ขั้นตอนที่ 2: ตรวจสอบว่ามีไดเร็กทอรีอยู่
