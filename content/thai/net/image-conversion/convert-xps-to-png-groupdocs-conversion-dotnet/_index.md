## คู่มือการใช้งาน
ตอนนี้คุณได้ตั้งค่าสภาพแวดล้อมเรียบร้อยแล้ว เรามาเริ่มดำเนินการแปลงกันเลย ในส่วนนี้จะอธิบายขั้นตอนต่างๆ อย่างชัดเจนเพื่อให้เข้าใจได้ง่าย

### ขั้นตอนที่ 1: กำหนดไดเรกทอรีเอาต์พุตและเทมเพลตการตั้งชื่อไฟล์
ตั้งค่าว่าจะเก็บไฟล์ที่แปลงแล้วไว้ที่ไหนและตั้งชื่อตามแบบแผน:

```csharp
string outputFolder = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "YOUR_OUTPUT_DIRECTORY");
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.png");
```
*เหตุใดจึงก้าวขั้นนี้?* ช่วยให้แน่ใจว่าแต่ละหน้าของไฟล์ XPS จะได้รับไฟล์ PNG ที่ไม่ซ้ำกันในไดเร็กทอรีที่มีการจัดระเบียบ

### ขั้นตอนที่ 2: สร้างฟังก์ชันสตรีมสำหรับเอาท์พุต
กำหนดวิธีการบันทึกแต่ละหน้าที่แปลงแล้ว:

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```
*วัตถุประสงค์:* ฟังก์ชันนี้จะสร้างสตรีมไฟล์สำหรับทุกหน้า ช่วยให้ตัวแปลงเขียนข้อมูล PNG ได้โดยตรง

### ขั้นตอนที่ 3: โหลดไฟล์ XPS ต้นฉบับ
โหลดไฟล์ XPS ต้นทางของคุณโดยใช้ GroupDocs.Conversion:

```csharp
using (Converter converter = new Converter(Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.xps")))
{
    // ตรรกะการแปลงจะถูกวางไว้ที่นี่
}
```
*เหตุใดจึงก้าวขั้นนี้?* เริ่มกระบวนการแปลงโดยการโหลดเอกสารที่คุณต้องการแปลง

### ขั้นตอนที่ 4: ตั้งค่าตัวเลือกการแปลงและแปลง
กำหนดตัวเลือกการแปลงของคุณสำหรับรูปแบบ PNG และดำเนินการแปลง:

```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Png };
converter.Convert(getPageStream, options);
```
*การกำหนดค่าที่สำคัญ:* การ `ImageConvertOptions` คลาสระบุว่าเอาต์พุตของคุณควรอยู่ในรูปแบบ PNG

### เคล็ดลับการแก้ไขปัญหา
- **ปัญหาทั่วไป**: ไม่พบไฟล์ ข้อผิดพลาด ตรวจสอบให้แน่ใจว่าเส้นทางถูกต้องและสามารถเข้าถึงได้
- **สารละลาย**ตรวจสอบชื่อไดเร็กทอรีและการมีอยู่ของไฟล์อีกครั้งก่อนดำเนินการแปลง

## การประยุกต์ใช้งานจริง
ต่อไปนี้เป็นสถานการณ์บางอย่างที่การแปลง XPS เป็น PNG อาจเป็นประโยชน์ได้:
1. **การจัดเก็บเอกสารดิจิทัล**:แปลงเอกสารเก็บถาวรเป็นรูปแบบที่สามารถดูได้สากลมากขึ้น เช่น PNG
2. **การบูรณาการเว็บไซต์**:ใช้ PNG เพื่อฝังรูปภาพลงในหน้าเว็บเนื่องจากรองรับเบราว์เซอร์ได้หลากหลาย
3. **การแบ่งปันเอกสาร**:แบ่งปันการแสดงตัวอย่างเอกสารเป็นรูปภาพ PNG กับผู้ใช้ที่อาจไม่ได้ติดตั้งโปรแกรมดู XPS
