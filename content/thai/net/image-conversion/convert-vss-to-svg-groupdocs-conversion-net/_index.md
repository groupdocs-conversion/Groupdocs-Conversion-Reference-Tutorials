## คู่มือการใช้งาน

### โหลดไฟล์ VSS

**ภาพรวม:** ก่อนที่จะแปลง ให้โหลดไฟล์ VSS ของคุณเพื่อให้แน่ใจว่าสามารถเข้าถึงได้และพร้อมสำหรับการแปลง

#### ขั้นตอนที่ 1: เริ่มต้นตัวแปลง
```csharp
string sampleVssPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.vss");
using (var converter = new Converter(sampleVssPath))
{
    // ตอนนี้โหลดไฟล์ VSS แล้ว
}
```
- **ทำไม:** การเริ่มต้นใช้งาน `Converter` วัตถุที่มีเส้นทาง VSS ของคุณจะโหลดเอกสารลงในหน่วยความจำเพื่อเตรียมพร้อมสำหรับการแปลง

### แปลง VSS เป็น SVG

**ภาพรวม:** ขั้นตอนนี้เกี่ยวข้องกับการแปลงไฟล์ VSS ที่โหลดเป็นรูปแบบ SVG โดยใช้ตัวเลือก GroupDocs.Conversion ที่ปรับแต่งสำหรับเอาท์พุต SVG

#### ขั้นตอนที่ 2: ตั้งค่าตัวเลือกการแปลง
```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "vss-converted-to.svg");

using (var converter = new Converter(sampleVssPath))
{
    var options = new PageDescriptionLanguageConvertOptions { Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Svg };
    
    // ดำเนินการแปลง
    converter.Convert(outputFile, options);
}
```
- **ทำไม:** `PageDescriptionLanguageConvertOptions` ระบุ SVG เป็นรูปแบบเป้าหมาย การกำหนดค่านี้ช่วยให้แน่ใจว่ามีการตั้งค่าที่จำเป็นทั้งหมดเพื่อการแปลงที่แม่นยำ

### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ VSS ถูกต้องและสามารถเข้าถึงได้
- ยืนยันว่าคุณมีสิทธิ์เขียนลงในไดเร็กทอรีเอาต์พุตของคุณ
