# แปลง XLSM เป็น JPG ด้วย GroupDocs.Conversion .NET
## การแนะนำ
คุณกำลังมองหาวิธีแปลงแมโคร Excel (XLSM) ของคุณเป็นสแน็ปช็อตภาพในรูปแบบรูปภาพอย่างราบรื่นหรือไม่ การแปลงไฟล์ XLSM เป็น JPG อาจมีความสำคัญสำหรับการแชร์ข้อมูลกับผู้ใช้ที่ไม่ใช่ Excel หรือการรวมสเปรดชีตเข้ากับงานนำเสนอและเอกสาร บทช่วยสอนนี้จะแนะนำคุณเกี่ยวกับการใช้ GroupDocs.Conversion .NET ซึ่งเป็นไลบรารีที่มีประสิทธิภาพซึ่งช่วยลดความซับซ้อนของกระบวนการแปลงนี้

**สิ่งที่คุณจะได้เรียนรู้:**
- วิธีโหลดไฟล์ XLSM โดยใช้ GroupDocs.Conversion
- การตั้งค่าตัวเลือกการแปลง JPG ด้วย API
- การดำเนินการแปลงจริงจาก XLSM เป็น JPG
- การประยุกต์ใช้งานจริงและการพิจารณาประสิทธิภาพ

ก่อนจะเริ่มใช้งาน ให้แน่ใจว่าคุณมีทุกอย่างพร้อมแล้ว

### ข้อกำหนดการตั้งค่าสภาพแวดล้อม
ตรวจสอบให้แน่ใจว่าสภาพแวดล้อมการพัฒนาของคุณได้รับการตั้งค่าด้วย:

## การตั้งค่า GroupDocs.Conversion สำหรับ .NET
ขั้นแรก ให้ติดตั้ง GroupDocs.Conversion ในโครงการ .NET ของคุณโดยใช้คอนโซลตัวจัดการแพ็กเกจ NuGet หรือ .NET CLI
**คอนโซลตัวจัดการแพ็กเกจ NuGet:**
```bash
Install-Package GroupDocs.Conversion -Version 25.3.0
```
**.NET CLI:**
```bash
dotnet add package GroupDocs.Conversion --version 25.3.0
```
### การขอใบอนุญาต
ในการใช้ GroupDocs.Conversion โปรดรับใบอนุญาต:
- **ทดลองใช้งานฟรี**: เข้าถึงคุณสมบัติที่จำกัดได้โดยไม่ต้องซื้อ
- **ใบอนุญาตชั่วคราว**:ขอสิทธิ์การเข้าถึงแบบเต็มรูปแบบในระหว่างการประเมินผล
- **ซื้อ**:ซื้อลิขสิทธิ์เต็มรูปแบบเพื่อใช้งานฟังก์ชั่นต่างๆ ได้ครบถ้วน
เมื่อติดตั้งและได้รับอนุญาตแล้ว ให้เริ่มต้นไลบรารีด้วยการตั้งค่าพื้นฐาน:
```csharp
using GroupDocs.Conversion;
// การเริ่มต้นวัตถุตัวแปลง
var converter = new Converter("path/to/your/sample.xlsm");
```
## คู่มือการใช้งาน
เราจะแบ่งกระบวนการแปลงออกเป็นขั้นตอนโดยใช้คุณลักษณะ GroupDocs.Conversion
### โหลดไฟล์ XLSM ต้นฉบับ
ก่อนอื่น โหลดไฟล์ XLSM ของคุณ:
#### กำหนดไดเรกทอรีเอกสาร
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
```
#### เริ่มต้นและโหลดไฟล์ XLSM
```csharp
using (Converter converter = new Converter(Path.Combine(documentDirectory, "sample.xlsm")))
{
    // วัตถุตัวแปลงพร้อมสำหรับการแปลงแล้ว
}
```
โค้ดตัวอย่างนี้จะเริ่มต้น `Converter` ตัวอย่างเช่นโดยระบุเส้นทางไฟล์ XLSM ของคุณ
### ตั้งค่าตัวเลือกการแปลงสำหรับรูปแบบ JPG
ถัดไป กำหนดค่ากระบวนการแปลง:
#### กำหนดไดเรกทอรีเอาท์พุต
```csharp
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
```
#### กำหนดค่าตัวเลือกการแปลงรูปภาพ
```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Jpg };
```
ที่นี่, `options` ถูกตั้งค่าให้แปลงไฟล์ XLSM ของคุณเป็นรูปภาพรูปแบบ JPG
### แปลงไฟล์ XLSM เป็นรูปแบบ JPG
ดำเนินการแปลงจริง:
#### กำหนดเทมเพลตชื่อไฟล์เอาท์พุต
```csharp
string outputFileTemplate = Path.Combine(outputDirectory, "converted-page-{0}.jpg");
```
#### ฟังก์ชั่นสร้างเพจสตรีม
```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext =>
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```
ฟังก์ชั่นนี้จะสร้างสตรีมสำหรับแต่ละหน้าที่ถูกแปลง
#### ดำเนินการแปลง
```csharp
using (Converter converter = new Converter(Path.Combine(documentDirectory, "sample.xlsm")))
{
    converter.Convert(getPageStream, options);
}
```
## การประยุกต์ใช้งานจริง
พิจารณาสถานการณ์เหล่านี้ซึ่งการแปลงนี้อาจเป็นประโยชน์:
- **รายงานทางธุรกิจ**:เปลี่ยนรายงาน Excel ที่ซับซ้อนให้เป็นรูปภาพที่เผยแพร่ได้ง่ายสำหรับผู้มีส่วนได้ส่วนเสีย
- **การแสดงภาพข้อมูล**:แปลงตารางข้อมูลใน XLSM เป็น JPG เพื่อการนำเสนอหรือใช้งานบนเว็บ
- **เอกสารประกอบ**:ฝังการแสดงภาพของสเปรดชีตในเอกสารทางเทคนิค
## การพิจารณาประสิทธิภาพ
เมื่อต้องจัดการกับไฟล์ขนาดใหญ่หรือการแปลงเป็นชุด ควรพิจารณาสิ่งต่อไปนี้:
- **การจัดการหน่วยความจำ**: กำจัดสิ่งของอย่างถูกวิธีโดยใช้ `using` คำกล่าว
- **การประมวลผลแบบขนาน**:แปลงเอกสารหลายฉบับพร้อมกันเพื่อประหยัดเวลาหากจำเป็น
