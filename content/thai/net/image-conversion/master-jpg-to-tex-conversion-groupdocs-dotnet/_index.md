# คู่มือการแปลงไฟล์ JPG เป็น TEX โดยใช้ GroupDocs.Conversion สำหรับ .NET

## การแนะนำ

- กระบวนการดำเนินการแบบทีละขั้นตอน

#### การเริ่มต้นและการตั้งค่าเบื้องต้น

วิธีตั้งค่าโครงการของคุณสำหรับการแปลง JPG เป็น TEX มีดังนี้:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

namespace ConversionTutorial
{
    class Program
    {
        static void Main(string[] args)
        {
            string documentDirectory = "YOUR_DOCUMENT_DIRECTORY"; // แทนที่ด้วยเส้นทางไดเร็กทอรีแหล่งที่มาของคุณ
            string outputDirectory = "YOUR_OUTPUT_DIRECTORY"; // แทนที่ด้วยเส้นทางไดเร็กทอรีเอาท์พุตของคุณ

            using (var converter = new Converter(Path.Combine(documentDirectory, "sample.jpg")))
            {
                var options = new PageDescriptionLanguageConvertOptions { Format = PageDescriptionLanguageFileType.Tex };
                string outputFile = Path.Combine(outputDirectory, "converted-to.tex");
                converter.Convert(outputFile, options);
            }
        }
    }
}
```

## คู่มือการใช้งาน

เมื่อทุกอย่างพร้อมแล้ว เรามาแบ่งการใช้งานออกเป็นขั้นตอนที่สามารถจัดการได้

### โหลดและแปลงไฟล์ JPG ของคุณ

#### ภาพรวม

ฟีเจอร์นี้ช่วยให้คุณโหลดไฟล์ JPG ต้นฉบับและแปลงเป็นรูปแบบ TEX โดยใช้ GroupDocs.Conversion ซึ่งมีประสิทธิภาพในการแปลงเอกสารอัตโนมัติในแอปพลิเคชันของคุณ

##### ขั้นตอนที่ 1: เริ่มต้นวัตถุตัวแปลง
สร้างอินสแตนซ์ของ `Converter` คลาสส่งผ่านเส้นทางไปยังไฟล์ JPG ต้นทางของคุณ:

```csharp
using (var converter = new Converter(Path.Combine(documentDirectory, "sample.jpg")))
```

##### ขั้นตอนที่ 2: ตั้งค่าตัวเลือกการแปลง
กำหนดตัวเลือกการแปลงของคุณโดยระบุ TEX เป็นรูปแบบเป้าหมายโดยใช้ `PageDescriptionLanguageConvertOptions`-

```csharp
var options = new PageDescriptionLanguageConvertOptions { Format = PageDescriptionLanguageFileType.Tex };
```

##### ขั้นตอนที่ 3: ดำเนินการแปลง
โทรหา `Convert` วิธีการที่มีเส้นทางไฟล์เอาท์พุตและตัวเลือกการแปลง ขั้นตอนนี้จะแปลง JPG ของคุณเป็นเอกสาร TEX:

```csharp
converter.Convert(outputFile, options);
```
