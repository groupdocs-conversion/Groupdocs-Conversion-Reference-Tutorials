## การแนะนำ

## คู่มือการใช้งาน

### แปลง BMP เป็น TXT
ส่วนนี้มีรายละเอียดเกี่ยวกับขั้นตอนการแปลง

#### ขั้นตอนที่ 1: โหลดไฟล์ BMP
เตรียมไฟล์ BMP ต้นฉบับของคุณเพื่อการแปลง:
```csharp
using (var converter = new Converter(sourceBmpPath))
{
    // ตอนนี้ BMP พร้อมสำหรับการแปลงแล้ว
}
```

#### ขั้นตอนที่ 2: กำหนดตัวเลือกการแปลง
ระบุการตั้งค่าการแปลงโดยใช้ `WordProcessingConvertOptions`-
```csharp
var options = new WordProcessingConvertOptions { Format = FileType.Txt };
```

#### ขั้นตอนที่ 3: ดำเนินการแปลง
ดำเนินการและบันทึกผลลัพธ์การแปลง:
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "bmp-converted-to.txt");
converter.Convert(outputFile, options);
```
**เคล็ดลับการแก้ไขปัญหา:** ตรวจสอบให้แน่ใจว่าเส้นทางถูกต้องและสามารถเข้าถึงได้ จัดการข้อยกเว้นระหว่างการดำเนินการไฟล์เพื่อป้องกันข้อผิดพลาดระหว่างการทำงาน

### ตั้งค่าเส้นทางไดเรกทอรีเอาท์พุต
จัดระเบียบไฟล์เอาท์พุตอย่างมีประสิทธิภาพ:
#### ขั้นตอนที่ 1: กำหนดไดเรกทอรีฐาน
ตั้งค่าว่าจะเก็บไฟล์ที่แปลงไว้ที่ไหน:
```csharp
string baseOutputDirectory = "YOUR_OUTPUT_DIRECTORY";
```

#### ขั้นตอนที่ 2: สร้างและตรวจสอบเส้นทางผลลัพธ์
ตรวจสอบให้แน่ใจว่ามีไดเร็กทอรีอยู่หรือสร้างขึ้นใหม่หากจำเป็น:
```csharp
string GetOutputDirectoryPath()
{
    string outputPath = Path.Combine(baseOutputDirectory, "ConvertedFiles");
    
    // ตรวจสอบการมีอยู่ สร้างถ้าไม่มีอยู่
    if (!Directory.Exists(outputPath))
        Directory.CreateDirectory(outputPath);
    
    return outputPath;
}
```

2. **ระบบเอกสาร:** ปรับปรุงกระบวนการแปลงรูปแบบเอกสารดั้งเดิมเป็น TXT เพื่อประสิทธิภาพการจัดเก็บข้อมูลที่ดีขึ้น
