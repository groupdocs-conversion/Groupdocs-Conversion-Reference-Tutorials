# แปลง XLSB เป็น JPG โดยใช้ GroupDocs.Conversion สำหรับ .NET: คำแนะนำทีละขั้นตอน

การแปลงไฟล์รูปแบบไบนารีของ Excel (XLSB) เป็นรูปแบบภาพ เช่น JPG ถือเป็นสิ่งสำคัญสำหรับการปรับปรุงการเข้าถึงและความเข้าใจ ไม่ว่าคุณจะเป็นนักพัฒนาหรือมืออาชีพทางธุรกิจ คู่มือนี้จะแนะนำคุณเกี่ยวกับการแปลงไฟล์ XLSB เป็นชุดรูปภาพ JPG โดยใช้ GroupDocs.Conversion สำหรับ .NET

## สิ่งที่คุณจะได้เรียนรู้

- แปลงไฟล์ XLSB เป็นรูปแบบ JPG ได้อย่างราบรื่น
- ใช้ฟังก์ชัน GroupDocs.Conversion ได้อย่างมีประสิทธิภาพ
- เพิ่มประสิทธิภาพการทำงานและแก้ไขปัญหาทั่วไป
- สำรวจการใช้งานจริงของฟีเจอร์นี้

ก่อนที่เราจะเจาะลึก มาดูข้อกำหนดเบื้องต้นที่จำเป็นสำหรับประสบการณ์ที่ราบรื่นกันก่อน

## คู่มือการใช้งาน

เรามาแบ่งการใช้งานออกเป็นคุณสมบัติหลักๆ และอธิบายทีละขั้นตอน

### กำหนดไดเรกทอรีเอาท์พุตและเทมเพลตไฟล์

ขั้นแรก ตั้งค่าไดเร็กทอรีเอาต์พุตที่จะบันทึกไฟล์ JPG:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");
```

การตั้งค่านี้ช่วยจัดระเบียบรูปภาพที่สร้างขึ้นอย่างมีประสิทธิภาพ

#### สร้างฟังก์ชันเพื่อสร้างสตรีมสำหรับการบันทึกหน้าเป็น JPG

หากต้องการบันทึกแต่ละหน้าของไฟล์ XLSB ของคุณเป็นภาพ JPG แยกกัน ให้ทำดังนี้:

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

### โหลดไฟล์ XLSB ต้นฉบับและตั้งค่าตัวเลือกการแปลง

โหลดไฟล์ XLSB ของคุณและระบุว่าคุณต้องการแปลงเป็นรูปแบบ JPG:

```csharp
using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/sample.xlsb"))
{
    ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Jpg };

    // ดำเนินการแปลง
    converter.Convert(getPageStream, options);
}
```

#### คำอธิบายพารามิเตอร์

- `SavePageContext`: ให้บริบทสำหรับแต่ละหน้าที่ถูกบันทึก
- `ImageConvertOptions`: กำหนดค่ารูปแบบเอาต์พุตและการตั้งค่าการแปลงอื่น ๆ
