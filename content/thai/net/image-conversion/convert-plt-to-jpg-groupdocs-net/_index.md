## คู่มือการใช้งาน

### แปลง PLT เป็น JPG
คุณสมบัตินี้ช่วยให้คุณแปลงไฟล์ PLT เป็นรูปแบบ JPG ทำให้การแชร์และดูการออกแบบสะดวกยิ่งขึ้นโดยไม่ต้องใช้ซอฟต์แวร์เฉพาะ

#### โหลดไฟล์ PLT ต้นฉบับ
ขั้นแรก ให้ระบุเส้นทางไดเรกทอรีสำหรับเอกสารและผลลัพธ์ของคุณ ขั้นตอนนี้เกี่ยวข้องกับการโหลดไฟล์ต้นฉบับโดยใช้ `Converter` ระดับ:
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY/";

using (Converter converter = new Converter(Path.Combine(documentDirectory, "yourfile.plt")))
{
    // ตรรกะการแปลงจะไปที่นี่
}
```

#### ตั้งค่าตัวเลือกการแปลงสำหรับรูปแบบ JPG
กำหนดตัวเลือกการแปลงเพื่อระบุว่าคุณต้องการเอาต์พุตเป็นรูปแบบ JPG:
```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Jpg };
```

#### กำหนดฟังก์ชั่นสตรีมเอาท์พุต
สร้างฟังก์ชันเพื่อจัดการสตรีมเอาต์พุตสำหรับแต่ละหน้าของไฟล์ PLT ของคุณ เพื่อให้แน่ใจว่าแต่ละหน้าที่แปลงแล้วจะถูกบันทึกเป็นไฟล์ JPG แยกกัน:
```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(Path.Combine(outputDirectory, $"converted-page-{savePageContext.Page}.jpg"), FileMode.Create);
```

#### ดำเนินการแปลง
สุดท้ายดำเนินการแปลงโดยเรียกใช้ `Convert` วิธีการด้วยตัวเลือกที่คุณกำหนด:
```csharp
converter.Convert(getPageStream, options);
```
