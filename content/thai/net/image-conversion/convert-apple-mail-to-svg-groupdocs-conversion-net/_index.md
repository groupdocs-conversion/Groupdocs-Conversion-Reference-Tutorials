# แปลงข้อความ Apple Mail เป็น SVG ด้วย GroupDocs.Conversion สำหรับ .NET

## คู่มือการใช้งาน
### การแปลง EMLX เป็นรูปแบบ SVG
หัวข้อนี้จะแนะนำคุณเกี่ยวกับการแปลงไฟล์ข้อความ Apple Mail (.emlx) เป็น Scalable Vector Graphics (SVG)

#### กำหนดเส้นทางสำหรับไฟล์อินพุตและเอาต์พุต
ก่อนอื่น ตั้งค่าไดเร็กทอรีอินพุตและเอาต์พุตของคุณ:

```csharp
string inputDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// กำหนดเส้นทาง
string inputFile = Path.Combine(inputDirectory, "sample.emlx");
string outputFile = Path.Combine(outputDirectory, "emlx-converted-to.svg");
```

#### โหลดและแปลงโดยใช้ GroupDocs.Conversion
โหลดไฟล์ EMLX ของคุณและระบุตัวเลือกการแปลงสำหรับ SVG:

```csharp
using (var converterInstance = new Converter(inputFile))
{
    // ระบุรูปแบบเอาท์พุตเป็น SVG
    var convertOptions = new PageDescriptionLanguageConvertOptions
    {
        Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Svg
    };
    
    // แปลงและบันทึกไฟล์
    converterInstance.Convert(outputFile, convertOptions);
}
```

**คำอธิบายพารามิเตอร์:**
- **ไฟล์อินพุต:** เส้นทางไปยังไฟล์ EMLX ต้นทางของคุณ
- **ไฟล์เอาท์พุต:** เส้นทางปลายทางสำหรับเอาท์พุต SVG
- **แปลงตัวเลือก.รูปแบบ:** ระบุว่าเป้าหมายการแปลงเป็น SVG

3. **การบูรณาการกับแอปเว็บ:** แสดงอีเมลในรูปแบบกราฟิกภายในแอปพลิเคชันเว็บ โดยใช้ประโยชน์จากความสามารถในการปรับขนาดของ SVG
