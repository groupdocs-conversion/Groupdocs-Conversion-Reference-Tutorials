ในบทความนี้เราจะกล่าวถึงเรื่อง:
- การตั้งค่า GroupDocs.Conversion สำหรับ .NET
- การโหลดไฟล์ POTM และเตรียมพร้อมสำหรับการแปลง
- การกำหนดค่าตัวเลือกการแปลงสำหรับเอาท์พุต PNG
- การนำกระบวนการแปลงไปใช้ด้วยตัวอย่างเชิงปฏิบัติ

มาทำให้การแปลงเอกสารของคุณราบรื่นและมีประสิทธิภาพกันเถอะ!

## คู่มือการใช้งาน

มาแบ่งกระบวนการแปลงออกเป็นขั้นตอนที่ชัดเจนและจัดการได้

### โหลดไฟล์ POTM แหล่งที่มา

**ภาพรวม:** ขั้นตอนนี้เกี่ยวข้องกับการโหลดไฟล์ POTM ต้นทางของคุณโดยใช้ `Converter` คลาสที่จัดทำโดย GroupDocs.Conversion

#### ขั้นตอนที่ 1: ตั้งค่าเส้นทางไฟล์

ตรวจสอบให้แน่ใจว่าคุณมีเส้นทางที่ถูกต้องไปยังไฟล์ POTM ของคุณ:

```csharp
using System.IO;

string samplePotmPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.potm");
```

#### ขั้นตอนที่ 2: โหลดไฟล์

โหลดไฟล์ POTM โดยใช้ `Converter` ชั้นเรียนนี้เตรียมความพร้อมสำหรับการแปลง

```csharp
using GroupDocs.Conversion;

using (Converter converter = new Converter(samplePotmPath))
{
    // ตรรกะการแปลงจะถูกเพิ่มที่นี่
}
```

### ตั้งค่าตัวเลือกการแปลงสำหรับรูปแบบ PNG

**ภาพรวม:** กำหนดค่าการตั้งค่าการแปลงเอกสารของคุณให้ส่งออกเป็นรูปแบบ PNG โดยใช้ `ImageConvertOptions`-

#### ขั้นตอนที่ 3: กำหนดตัวเลือกการแปลงรูปภาพ

ตั้งค่ารูปแบบเป้าหมายเป็น PNG:

```csharp
using GroupDocs.Conversion.Options.Convert;

// สร้างอินสแตนซ์ของ ImageConvertOptions และระบุรูปแบบเป้าหมายเป็น PNG
ImageConvertOptions options = new ImageConvertOptions { Format = FileTypes.ImageFileType.Png };
```

### แปลง POTM เป็น PNG

**ภาพรวม:** เมื่อโหลดไฟล์และตั้งค่าตัวเลือกการแปลงแล้ว ก็ถึงเวลาดำเนินการแปลง

#### ขั้นตอนที่ 4: ตั้งค่าไดเรกทอรีเอาต์พุตและเทมเพลตการตั้งชื่อ

กำหนดว่าไฟล์ที่แปลงของคุณจะถูกบันทึกที่ไหน:

```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.png");
```

#### ขั้นตอนที่ 5: สร้างสตรีมสำหรับแต่ละหน้า

สร้างฟังก์ชันสำหรับจัดการสตรีมไฟล์สำหรับแต่ละหน้า PNG:

```csharp
using System.IO;

Func<SavePageContext, Stream> getPageStream = savePageContext => new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

#### ขั้นตอนที่ 6: ดำเนินการแปลง

ดำเนินการกระบวนการแปลงโดยใช้การตั้งค่าที่คุณกำหนดและฟังก์ชันสตรีม:

```csharp
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

// โหลดไฟล์ POTM ต้นฉบับ (ดังที่แสดงในหัวข้อก่อนหน้า)
using (Converter converter = new Converter(samplePotmPath))
{
    // แปลงเป็นรูปแบบ PNG โดยใช้ตัวเลือกที่ตั้งไว้ก่อนหน้านี้และฟังก์ชั่นสตรีมหน้า
    converter.Convert(getPageStream, options);
}
```
