## คู่มือการใช้งาน

เราจะแบ่งขั้นตอนการแปลงออกเป็นขั้นตอนที่ชัดเจน

### ภาพรวมคุณลักษณะ: การแปลง PPTM เป็น PSD

GroupDocs.Conversion ช่วยให้คุณแปลงรูปแบบเอกสารต่างๆ ได้อย่างง่ายดาย ที่นี่ เราจะเน้นการแปลงงานนำเสนอ PowerPoint (PPTM) เป็นเอกสาร Adobe Photoshop (PSD)

#### ขั้นตอนที่ 1: เริ่มต้นตัวแปลง
```csharp
using (var converter = new GroupDocs.Conversion.Converter("input.pptm"))
{
    // จะเพิ่มโค้ดการแปลงไว้ที่นี่
}
```
- **วัตถุประสงค์**: เริ่มต้น a `Converter` วัตถุสำหรับไฟล์ต้นฉบับของคุณ

#### ขั้นตอนที่ 2: กำหนดตัวเลือกการแปลง

ระบุรูปแบบเป้าหมายและตัวเลือกเฉพาะใดๆ ที่จำเป็นสำหรับการแปลง:
```csharp
var options = new PsdConvertOptions();
```
- **พารามิเตอร์**- `PsdConvertOptions()` ตั้งค่าเริ่มต้นสำหรับเอาท์พุต PSD

#### ขั้นตอนที่ 3: ดำเนินการแปลง
เรียกใช้วิธีการแปลงเพื่อแปลงไฟล์ PPTM เป็นรูปแบบ PSD:
```csharp
converter.Convert("output.psd", options);
```
- **คำอธิบาย**:แปลงและบันทึกงานนำเสนอเป็นเอกสาร Adobe Photoshop
