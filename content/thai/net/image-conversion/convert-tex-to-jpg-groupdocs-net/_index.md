## คู่มือการใช้งาน

เราจะแบ่งการใช้งานออกเป็นหลายส่วนตามตรรกะ โดยเน้นที่คุณลักษณะเฉพาะของกระบวนการแปลง

### โหลดไฟล์ TEX ต้นฉบับ
**ภาพรวม**ขั้นตอนนี้เกี่ยวข้องกับการโหลดไฟล์ TEX ต้นทางของคุณโดยใช้ GroupDocs.Conversion 

#### ขั้นตอนที่ 1: เริ่มต้นวัตถุตัวแปลง
```csharp
using System;
using GroupDocs.Conversion;

string documentPath = "YOUR_DOCUMENT_DIRECTORY";

using (Converter converter = new Converter(documentPath + "/sample.tex"))
{
    // ตัวแทนสำหรับขั้นตอนการแปลง
}
```
**คำอธิบาย**: เดอะ `Converter` คลาสจะถูกเริ่มต้นด้วยเส้นทางไปยังไฟล์ TEX ของคุณ อ็อบเจ็กต์นี้จะจัดการการดำเนินการทั้งหมดที่ตามมาในเอกสารนี้

### ตั้งค่าตัวเลือกการแปลงสำหรับรูปแบบ JPG
**ภาพรวม**:ที่นี่เราจะกำหนดว่าจะต้องแปลงเอกสารของเราเป็นรูปแบบ JPG อย่างไร

#### ขั้นตอนที่ 2: กำหนดตัวเลือกการแปลง
```csharp
using GroupDocs.Conversion.Options.Convert;

string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Jpg };
```
**คำอธิบาย**- `ImageConvertOptions` ช่วยให้คุณสามารถระบุรูปแบบภาพที่ต้องการได้ โดยตั้งค่าเป็น `Jpg` รับประกันว่าผลลัพธ์ของเราจะอยู่ในรูปแบบ JPG

### แปลงไฟล์ TEX เป็นรูปแบบ JPG
**ภาพรวม**ขั้นตอนนี้เกี่ยวข้องกับการแปลงไฟล์ TEX แต่ละหน้าเป็นภาพ JPG แต่ละภาพ

#### ขั้นตอนที่ 3: กำหนดตรรกะผลลัพธ์และการแปลง
```csharp
using System.IO;
using GroupDocs.Conversion;

string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");

Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);

using (Converter converter = new Converter(documentPath + "/sample.tex"))
{
    converter.Convert(getPageStream, options);
}
```
**คำอธิบาย**: เดอะ `getPageStream` ฟังก์ชันนี้จะสร้างสตรีมสำหรับแต่ละหน้าของเอกสาร เพื่อให้แน่ใจว่าทุกหน้าจะถูกบันทึกเป็นไฟล์ JPG แยกกันในไดเรกทอรีเอาต์พุตที่ระบุ

### เคล็ดลับการแก้ไขปัญหา
