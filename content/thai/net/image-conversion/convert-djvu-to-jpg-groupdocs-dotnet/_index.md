## คู่มือการใช้งาน

มาแบ่งการใช้งานออกเป็นคุณลักษณะเฉพาะต่างๆ กัน แต่ละส่วนจะแนะนำคุณตลอดการโหลดไฟล์ การกำหนดค่าตัวเลือก และการดำเนินการแปลง

### คุณสมบัติ 1: โหลดไฟล์ DJVU

#### ภาพรวม
การโหลดไฟล์ DJVU เป็นขั้นตอนแรกในกระบวนการแปลงเอกสารโดยใช้ GroupDocs.Conversion สำหรับ .NET

##### ขั้นตอนที่ 1: เตรียมเส้นทางแหล่งที่มาของคุณ
ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ DJVU ต้นทางของคุณได้รับการตั้งค่าอย่างถูกต้อง:

```csharp
string sourceFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.djvu");
```

##### ขั้นตอนที่ 2: โหลดไฟล์
ใช้ `Converter` คลาสสำหรับโหลดไฟล์ DJVU ของคุณ:

```csharp
using (Converter converter = new Converter(sourceFilePath))
{
    Console.WriteLine("The file is ready for conversion.");
}
```
**คำอธิบาย**:สไนปเป็ตนี้จะเริ่มต้นสิ่งใหม่ `Converter` อินสแตนซ์ที่มีเส้นทางไฟล์ต้นฉบับ เพื่อเตรียมพร้อมสำหรับการดำเนินการต่อไป

### คุณสมบัติที่ 2: ตั้งค่าตัวเลือกการแปลงเป็นรูปแบบ JPG

#### ภาพรวม
กำหนดค่าตัวเลือกการแปลงของคุณเพื่อแปลงเอกสารเป็นภาพ JPEG อย่างมีประสิทธิภาพ

##### ขั้นตอนที่ 1: กำหนดค่าตัวเลือกการแปลงรูปภาพ
ตั้งค่าตัวเลือกที่จำเป็นสำหรับการแปลงเป็นรูปแบบ JPG:

```csharp
using GroupDocs.Conversion.Options.Convert;

// สร้างและกำหนดค่าตัวเลือกการแปลงภาพ
ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Jpg };
```
**คำอธิบาย**: เดอะ `ImageConvertOptions` คลาสอนุญาตให้คุณระบุรูปแบบภาพเป้าหมาย ในกรณีนี้คือ JPG

### คุณสมบัติที่ 3: ดำเนินการแปลงจาก DJVU เป็น JPG

#### ภาพรวม
ดำเนินการตามกระบวนการแปลงและบันทึกรูปภาพเอาต์พุตของคุณ

##### ขั้นตอนที่ 1: กำหนดเส้นทางเอาต์พุตและตรรกะ
ตั้งค่าตรรกะสำหรับการบันทึกแต่ละหน้าที่แปลงเป็นไฟล์ JPEG แยกกัน:

```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");

Func<SavePageContext, Stream> getPageStream = savePageContext =>
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

##### ขั้นตอนที่ 2: ดำเนินการแปลง
ใช้ `Converter` อินสแตนซ์ที่จะแปลงและบันทึกผลลัพธ์:

```csharp
using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/sample.djvu"))
{
    // แปลงและบันทึกแต่ละหน้าเป็นไฟล์ JPG
    converter.Convert(getPageStream, options);
}
```
**คำอธิบาย**:สไนปเป็ตนี้จะแปลงแต่ละหน้าของเอกสาร DJVU ให้เป็นภาพ JPEG แยกกันโดยใช้ลอจิกสตรีมที่กำหนดไว้
