## คู่มือการใช้งาน

เราจะแนะนำแต่ละขั้นตอนที่จำเป็นในการแปลงไฟล์ DOCX เป็นรูปแบบ JPG โดยใช้ GroupDocs.Conversion

### โหลดไฟล์ DOCX แหล่งที่มา

**ภาพรวม:**
การโหลดไฟล์ DOCX ต้นฉบับของคุณเป็นขั้นตอนแรกของกระบวนการแปลงใดๆ ก็ตาม วิธีนี้จะช่วยให้คุณมีเอกสารพร้อมสำหรับการแปลงเป็นรูปแบบต่างๆ

**เริ่มต้นวัตถุตัวแปลง:**

```csharp
// ตรวจสอบให้แน่ใจว่ามีการรวมเนมสเปซที่จำเป็น
using System;
using GroupDocs.Conversion;

string documentPath = "YOUR_DOCUMENT_DIRECTORY\sample.docx";
Converter converter = new Converter(documentPath);
```
- **พารามิเตอร์:** `documentPath` เป็นเส้นทางไฟล์ของ DOCX ของคุณ
- **วัตถุประสงค์:** นี่จะสร้างอินสแตนซ์ของ `Converter`ซึ่งช่วยให้กระบวนการแปลงง่ายขึ้น

### แปลงไฟล์ DOCX เป็นรูปแบบ JPG

**ภาพรวม:**
ตอนนี้เราได้โหลดเอกสารแล้ว มาตั้งค่าตัวเลือกการแปลงและดำเนินการแปลงเป็นรูปแบบ JPG กัน

**กำหนดพารามิเตอร์เอาต์พุต:**

```csharp
using System.IO;
using GroupDocs.Conversion.Options.Convert;

string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");

Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```
- **วัตถุประสงค์:** สิ่งนี้จะกำหนดว่าแต่ละหน้าของเอกสารจะถูกบันทึกเป็นไฟล์ JPG ที่ไหนและอย่างไร
  
**ตั้งค่าตัวเลือกการแปลง:**

```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Jpg };
```
- **พารามิเตอร์:** `Format` ระบุว่าเราจะแปลงเป็น JPG

**ดำเนินการแปลง:**

```csharp
converter.Convert(getPageStream, options);
```
- **วัตถุประสงค์:** ดำเนินการกระบวนการแปลงโดยใช้การตั้งค่าที่กำหนด และส่งออกแต่ละหน้าเป็นไฟล์ JPG แยกกัน

### เคล็ดลับการแก้ไขปัญหา

- ตรวจสอบให้แน่ใจว่าไดเร็กทอรีเอาต์พุตของคุณมีอยู่ก่อนที่จะพยายามบันทึกไฟล์
- ตรวจสอบข้อยกเว้นที่เกี่ยวข้องกับสิทธิ์ไฟล์หรือเส้นทางที่ไม่ถูกต้อง
