## คู่มือการใช้งาน

### แปลงไฟล์ PPSX เป็นรูปแบบ SVG

#### ภาพรวม
การแปลงไฟล์ .ppsx เป็นรูปแบบ SVG นั้นเกี่ยวข้องกับการโหลดไฟล์ต้นฉบับ การกำหนดค่าการตั้งค่าการแปลง และการบันทึกผลลัพธ์ หัวข้อนี้จะแนะนำคุณในแต่ละขั้นตอนพร้อมคำอธิบายโดยละเอียดและตัวอย่างโค้ด

#### ขั้นตอนที่ 1: กำหนดเส้นทางสำหรับไดเรกทอรีอินพุต/เอาต์พุต
เริ่มต้นด้วยการระบุตำแหน่งไฟล์อินพุตของคุณและตำแหน่งที่คุณต้องการบันทึกไฟล์ที่แปลงแล้ว:

```csharp
string documentPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.ppsx");
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "ppsx-converted-to.svg");
```

#### ขั้นตอนที่ 2: โหลดไฟล์ PPSX ต้นฉบับ
โหลดไฟล์ .ppsx ของคุณโดยใช้ GroupDocs.Conversion `Converter` ระดับ:

```csharp
using (var converter = new Converter(documentPath))
{
    // ตรรกะการแปลงจะไปที่นี่
}
```
ขั้นตอนนี้จะช่วยให้แน่ใจว่าไฟล์ของคุณพร้อมสำหรับการประมวลผล

#### ขั้นตอนที่ 3: กำหนดค่าตัวเลือกการแปลง
ตั้งค่าตัวเลือกการแปลงเพื่อระบุ SVG เป็นรูปแบบเอาต์พุต:

```csharp
var options = new PageDescriptionLanguageConvertOptions { Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Svg };
```
ตัวเลือกเหล่านี้กำหนดว่าควรจัดการกระบวนการแปลงอย่างไร

#### ขั้นตอนที่ 4: ดำเนินการแปลงและบันทึก
ดำเนินการแปลงและบันทึกไฟล์ SVG ที่ได้:

```csharp
csvr.Convert(outputFile, options);
```
คำสั่งนี้จะแปลงการนำเสนอของคุณเป็นไฟล์ SVG และบันทึกในตำแหน่งที่กำหนด
