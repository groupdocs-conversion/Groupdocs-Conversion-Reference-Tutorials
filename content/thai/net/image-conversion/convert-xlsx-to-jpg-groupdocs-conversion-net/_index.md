## คู่มือการใช้งาน

### โหลดและแปลง XLSX เป็น JPG
ฟีเจอร์นี้สาธิตวิธีการโหลดไฟล์ XLSX และแปลงแผ่นงานแต่ละแผ่นเป็นภาพ JPG แยกกัน

#### กำหนดไดเรกทอรีเอาท์พุตและเทมเพลตไฟล์
ตั้งค่าเส้นทางไดเร็กทอรีเอาท์พุตและเทมเพลตสำหรับการตั้งชื่อรูปภาพที่แปลงแล้ว:
```csharp
using System.IO;
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string inputFile = "YOUR_DOCUMENT_DIRECTORY/SAMPLE_XLSX.xlsx"; // แทนที่ด้วยเส้นทางไฟล์ XLSX ของคุณ

// กำหนดรูปแบบการตั้งชื่อไฟล์เอาท์พุต\สตริง outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");
```

#### สร้างฟังก์ชันสตรีมสำหรับไฟล์เอาท์พุต
กำหนดฟังก์ชันสำหรับจัดการการสร้างสตรีมเอาท์พุตสำหรับแต่ละเพจที่ถูกแปลง:
```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext =>
{
    string filePath = string.Format(outputFileTemplate, savePageContext.Page);
    return new FileStream(filePath, FileMode.Create);
};
```

#### โหลดและแปลงไฟล์ XLSX
ใช้ `Converter` คลาสสำหรับโหลดไฟล์ของคุณและแปลงเป็นรูปแบบ JPG:
```csharp
using (Converter converter = new Converter(inputFile))
{
    // ระบุตัวเลือกการแปลงสำหรับรูปแบบ JPG
    ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Jpg };

    // ดำเนินการแปลง
    converter.Convert(getPageStream, options);
}
```

การตั้งค่านี้จะช่วยให้แน่ใจว่าแต่ละแผ่นงานในไฟล์ XLSX ของคุณได้รับการบันทึกเป็นภาพ JPG ที่ไม่ซ้ำกันตามหมายเลขหน้า

### การกำหนดค่าไดเรกทอรีเอาท์พุตและเทมเพลตไฟล์
การกำหนดค่าไดเร็กทอรีเอาต์พุตและเทมเพลตการตั้งชื่ออย่างถูกต้องถือเป็นสิ่งสำคัญสำหรับการจัดระเบียบไฟล์ที่แปลงแล้วอย่างมีประสิทธิภาพ หัวข้อนี้สร้างขึ้นจากเนื้อหาที่เราได้กล่าวถึงไปแล้ว

#### การตั้งค่าโครงสร้างไดเร็กทอรี
ตรวจสอบให้แน่ใจว่าไดเร็กทอรีเอาท์พุตของคุณมีอยู่ก่อนที่จะรันการแปลง:
```csharp
if (!Directory.Exists(outputFolder))
{
    Directory.CreateDirectory(outputFolder);
}
```

#### การกำหนดค่าเทมเพลต
เทมเพลตไฟล์มีตัวแทน `{0}` ซึ่งจะถูกแทนที่ด้วยหมายเลขหน้าแต่ละหน้าในระหว่างการแปลง ตรวจสอบให้แน่ใจว่าการตั้งค่านี้ตรงกับความต้องการขององค์กรของคุณ

## การประยุกต์ใช้งานจริง

- **การแบ่งปันเอกสาร**:แปลงสเปรดชีตเป็นรูปภาพเพื่อแชร์ในงานนำเสนอหรืออีเมลได้อย่างง่ายดาย
- **การแสดงภาพข้อมูล**:ใช้รูปแบบภาพสำหรับการแสดงภาพของแผนภูมิข้อมูลและกราฟภายในแผ่นงาน Excel
- **ความเข้ากันได้**:กระจายข้อมูลข้ามแพลตฟอร์มที่อาจไม่รองรับไฟล์ XLSX แต่สามารถแสดงรูปภาพได้

## การพิจารณาประสิทธิภาพ

เมื่อจัดการกับชุดข้อมูลขนาดใหญ่ ควรพิจารณาสิ่งต่อไปนี้:
- **การประมวลผลแบบแบตช์**ประมวลผลเอกสารเป็นชุดเพื่อจัดการการใช้หน่วยความจำอย่างมีประสิทธิภาพ
- **การดำเนินการแบบอะซิงโครนัส**:ใช้งานการแปลงแบบอะซิงโครนัสเพื่อให้แอปพลิเคชันของคุณตอบสนองได้ดี
