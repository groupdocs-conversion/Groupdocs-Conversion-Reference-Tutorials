## คู่มือการใช้งาน

เราจะแบ่งขั้นตอนการแปลงออกเป็นขั้นตอนที่ทำตามได้ง่าย

### การแปลง XLAM เป็น XLS

#### ภาพรวม

ฟีเจอร์นี้จะแปลงไฟล์ Add-In ของ Excel (XLAM) ให้เป็นเวิร์กบุ๊ก Excel มาตรฐาน (XLS) เพื่อให้มั่นใจได้ว่าจะเข้ากันได้กับ Microsoft Office เวอร์ชันต่างๆ

#### ขั้นตอนที่ 1: ตั้งค่าเส้นทางไฟล์

กำหนดค่าเอกสารและไดเรกทอรีเอาท์พุตของคุณโดยใช้ตัวแทน:

```csharp
string sourcePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "source.xlam");
string outputPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "xlam-converted-to.xls");
```

#### ขั้นตอนที่ 2: โหลดไฟล์ XLAM ต้นฉบับ

ใช้ GroupDocs.Conversion เพื่อโหลดไฟล์ XLAM ของคุณโดยระบุเส้นทาง:

```csharp
using (var converter = new Converter(sourcePath))
{
    // ตรรกะการแปลงจะตามมาที่นี่
}
```

#### ขั้นตอนที่ 3: กำหนดตัวเลือกการแปลง

ระบุการตั้งค่าการแปลงสำหรับรูปแบบ XLS โดยใช้ `SpreadsheetConvertOptions`-

```csharp
SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Xls };
```

การกำหนดค่านี้จะช่วยให้แน่ใจว่าไฟล์เอาต์พุตอยู่ในรูปแบบ XLS ที่ต้องการ

#### ขั้นตอนที่ 4: แปลงและบันทึกผลลัพธ์

ดำเนินการแปลงและบันทึกไฟล์ XLS ที่ได้:

```csharp
csv converter.Convert(outputPath, options);
```

### การกำหนดค่าเส้นทางไฟล์

#### ภาพรวม

การตั้งค่าเส้นทางไฟล์ของคุณอย่างถูกต้องถือเป็นสิ่งสำคัญสำหรับการแปลงที่ประสบความสำเร็จ ฟีเจอร์นี้ช่วยให้คุณกำหนดไดเรกทอรีอินพุตและเอาต์พุตได้อย่างถูกต้อง

#### ขั้นตอนที่ 1: กำหนดไดเรกทอรี

สร้างโครงสร้างไดเร็กทอรีสำหรับไฟล์ต้นฉบับและไฟล์เป้าหมาย:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
```

#### ขั้นตอนที่ 2: รวมเส้นทาง

สร้างเส้นทางไฟล์แบบเต็มโดยใช้ `Path.Combine`-

```csharp
string sourceFilePath = Path.Combine(documentDirectory, "source.xlam");
string targetFilePath = Path.Combine(outputDirectory, "xlam-converted-to.xls");
```
