## คู่มือการใช้งาน

มาแบ่งกระบวนการแปลงออกเป็นขั้นตอนที่สามารถจัดการได้

### โหลดไฟล์ต้นฉบับ

**ภาพรวม**โหลดไฟล์ XLT ของคุณเพื่อเตรียมการแปลงโดยใช้ GroupDocs.Conversion

#### การดำเนินการแบบทีละขั้นตอน
1. **กำหนดเส้นทางไดเรกทอรี**
   ระบุเส้นทางที่ไฟล์ต้นฉบับและไฟล์เอาท์พุตของคุณจะอยู่
   ```csharp
   const string DocumentDirectory = "YOUR_DOCUMENT_DIRECTORY";
   const string OutputDirectory = "YOUR_OUTPUT_DIRECTORY";
   ```
2. **โหลดไฟล์ XLT ต้นฉบับ**
   ใช้ `Converter` คลาสที่จะโหลดไฟล์ของคุณเพื่อเตรียมพร้อมสำหรับการแปลง
   ```csharp
   var xltFilePath = Path.Combine(DocumentDirectory, "sample.xlt");
   using (var converter = new GroupDocs.Conversion.Converter(xltFilePath))
   {
       // ตอนนี้โหลดไฟล์ XLT แล้ว
   }
   ```

### กำหนดค่าตัวเลือกการแปลง

**ภาพรวม**:ตั้งค่าตัวเลือกการแปลงเพื่อกำหนดว่า XLT จะถูกแปลงเป็น XLSX อย่างไร

#### การดำเนินการแบบทีละขั้นตอน
1. **สร้างตัวเลือกการแปลง**
   กำหนดรูปแบบและการตั้งค่าอื่น ๆ สำหรับไฟล์เป้าหมายของคุณโดยใช้ `SpreadsheetConvertOptions`-
   ```csharp
   var options = new SpreadsheetConvertOptions();
   ```

### ดำเนินการแปลง

**ภาพรวม**:ดำเนินการตามกระบวนการแปลงเพื่อแปลง XLT ของคุณเป็นไฟล์ XLSX

#### การดำเนินการแบบทีละขั้นตอน
1. **ดำเนินการและบันทึกการแปลง**
   ใช้ `Converter` ชั้นเรียน `Convert` วิธีการส่งผ่านเส้นทางเอาท์พุตและตัวเลือก
   ```csharp
   var xltFilePath = Path.Combine(DocumentDirectory, "sample.xlt");
   var outputFilePath = Path.Combine(OutputDirectory, "xlt-converted-to.xlsx");

   using (var converter = new GroupDocs.Conversion.Converter(xltFilePath))
   {
       var options = new SpreadsheetConvertOptions();
       converter.Convert(outputFilePath, options);
   }
   ```
