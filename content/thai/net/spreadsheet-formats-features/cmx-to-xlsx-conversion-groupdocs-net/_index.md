## คู่มือการใช้งาน
ตอนนี้เรามาดูขั้นตอนการแปลงไฟล์ CMX เป็นรูปแบบ XLSX กัน

### การโหลดและการแปลงไฟล์ CMX
**ภาพรวม**:คุณลักษณะนี้ช่วยให้คุณโหลดเอกสาร CMX และแปลงเป็นไฟล์ XLSX ที่เข้ากันได้กับ Excel โดยใช้ GroupDocs.Conversion สำหรับ .NET

#### ขั้นตอนที่ 1: ระบุเส้นทางและโหลดไฟล์ CMX ต้นทาง
ขั้นแรก ให้กำหนดไดเรกทอรีต้นทางและปลายทาง จากนั้นจึงเริ่มต้น `Converter` คลาสที่มีเส้นทางของไฟล์ CMX ของคุณ:

```csharp
using System.IO;
// กำหนดเส้นทางเอกสาร
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

string sourceFilePath = Path.Combine(documentDirectory, "sample.cmx"); // แทนที่ 'sample.cmx' ด้วยชื่อไฟล์ CMX จริงของคุณ

// โหลดไฟล์ CMX
using (var converter = new Converter(sourceFilePath))
{
    Console.WriteLine("CMX file loaded successfully.");
}
```

#### ขั้นตอนที่ 2: ระบุตัวเลือกการแปลงสำหรับรูปแบบ XLSX
ขั้นตอนต่อไป ให้ตั้งค่าตัวเลือกการแปลงเพื่อระบุว่าคุณต้องการแปลงเอกสารของคุณเป็นรูปแบบ XLSX:

```csharp
var options = new SpreadsheetConvertOptions();
Console.WriteLine("Conversion options set to XLSX.");
```

#### ขั้นตอนที่ 3: แปลงและบันทึกไฟล์ผลลัพธ์
สุดท้ายให้ดำเนินการแปลงและบันทึกไฟล์เอาท์พุตในไดเร็กทอรีที่กำหนด:

```csharp
string outputFile = Path.Combine(outputDirectory, "cmx-converted-to.xlsx");

// ดำเนินการแปลง
converter.Convert(outputFile, options);
Console.WriteLine("Conversion completed. Check the output directory.");
```

**เคล็ดลับการแก้ไขปัญหา:**
- ตรวจสอบให้แน่ใจว่าไฟล์ CMX ของคุณไม่เสียหาย
- ตรวจสอบเส้นทางไฟล์และการอนุญาตสำหรับไดเร็กทอรีทั้งต้นทางและปลายทาง

2. **การรายงานอัตโนมัติ**:ทำให้การสร้างรายงาน Excel จากไฟล์ CMX ภายในแอปพลิเคชัน .NET เป็นไปโดยอัตโนมัติ
