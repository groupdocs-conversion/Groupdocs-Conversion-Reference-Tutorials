**สิ่งที่คุณจะได้เรียนรู้:**

## คู่มือการใช้งาน

เราจะแบ่งการใช้งานออกเป็นส่วนๆ ตามคุณลักษณะ และแนะนำคุณเกี่ยวกับการแปลงไฟล์ PST เป็น XLS

### แปลง PST เป็น XLS

คุณลักษณะนี้จะแปลงไฟล์ Personal Storage Table (.pst) เป็น Excel Spreadsheet (.xls)

#### ภาพรวม
กระบวนการนี้เกี่ยวข้องกับการเริ่มต้นตัวแปลงด้วยไฟล์ PST ต้นฉบับของคุณและกำหนดค่าสำหรับเอาต์พุตในรูปแบบ XLS

**ขั้นตอนการดำเนินการ:**

##### ขั้นตอนที่ 1: เริ่มต้นตัวแปลง
ตั้งค่าตัวแปลงโดยใช้เส้นทางไปยังไฟล์ PST ของคุณ:

```csharp
string sourceFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.pst");
using (var converter = new GroupDocs.Conversion.Converter(sourceFilePath))
{
    // ขั้นตอนการกำหนดค่าเพิ่มเติมจะตามมา
}
```

##### ขั้นตอนที่ 2: กำหนดค่าตัวเลือกการแปลง
กำหนดตัวเลือกการแปลงเพื่อระบุรูปแบบผลลัพธ์เป็น XLS:

```csharp
SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Xls };
```

##### ขั้นตอนที่ 3: ดำเนินการแปลง
ดำเนินการแปลงและบันทึกผลลัพธ์:

```csharp
string outputFileTemplate = Path.Combine("YOUR_OUTPUT_DIRECTORY", "pst-converted-{0}-to.xls");
converter.Convert(
    (SaveContext saveContext) => new FileStream(string.Format(outputFileTemplate, counter++), FileMode.Create),
    options
);
```

**คำอธิบาย:**
- **โหลดบริบท:** กำหนดการจัดการพิเศษสำหรับไฟล์ OST
- **ตัวเลือกการแปลง:** ระบุรูปแบบเป้าหมายเป็น XLS

### โหลดและกำหนดค่าตัวเลือกการแปลง

คุณสมบัตินี้ช่วยให้สามารถกำหนดค่าการตั้งค่าการโหลดตามประเภทไฟล์และเงื่อนไขได้

#### ภาพรวม
การโหลดไฟล์ PST ด้วยเงื่อนไขเฉพาะจะช่วยให้มั่นใจได้ว่าการแปลงจะแม่นยำ โดยเฉพาะอย่างยิ่งเมื่อต้องจัดการกับรูปแบบอีเมลที่แตกต่างกัน เช่น PST หรือ OST

**ขั้นตอนการดำเนินการ:**

##### ขั้นตอนที่ 1: กำหนดตัวเลือกการโหลด
ตั้งค่าตัวเลือกในการจัดการรูปแบบแหล่งข้อมูลที่แตกต่างกัน:

```csharp
var loadOptions = new PersonalStorageLoadOptions();
```

##### ขั้นตอนที่ 2: กำหนดประเภทไฟล์
ตรวจสอบว่าไฟล์เป็น OST หรือไม่ และใช้การกำหนดค่าเฉพาะ:

```csharp
bool isOstFile = (sourceFilePath.EndsWith(".ost", StringComparison.OrdinalIgnoreCase));
if (isOstFile)
{
    // การจัดการเพิ่มเติมสำหรับไฟล์ OST
}
```

### กำหนดตัวเลือกการแปลง

คุณลักษณะนี้มุ่งเน้นที่การตั้งค่าตัวเลือกการแปลงเพื่อให้แน่ใจว่าไฟล์เอาต์พุตตรงตามความต้องการของคุณ

#### ภาพรวม
การกำหนดค่าตัวเลือกการแปลงช่วยให้คุณกำหนดได้ว่าควรจัดรูปแบบและจัดเก็บข้อมูลในไฟล์ XLS อย่างไร

**ขั้นตอนการดำเนินการ:**

##### ขั้นตอนที่ 1: ตั้งค่ารูปแบบผลลัพธ์
ระบุว่ารูปแบบเป้าหมายคือ XLS:

```csharp
SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Xls };
```
