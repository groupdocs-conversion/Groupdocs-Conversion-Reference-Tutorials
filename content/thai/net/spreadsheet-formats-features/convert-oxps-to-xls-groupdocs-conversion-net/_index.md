### การเริ่มต้นและการตั้งค่าเบื้องต้น

หากต้องการเริ่มต้น GroupDocs.Conversion สำหรับ .NET ในแอปพลิเคชัน C# ของคุณ ให้ปฏิบัติตามการตั้งค่าง่ายๆ นี้:

```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main()
    {
        string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
        string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

        string inputFile = Path.Combine(documentDirectory, "sample.oxps");
        string outputFile = Path.Combine(outputDirectory, "oxps-converted-to.xls");

        using (var converter = new Converter(inputFile))
        {
            var options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Xls };
            converter.Convert(outputFile, options);
        }
    }
}
```

## คู่มือการใช้งาน

### แปลง OXPS เป็น XLS

#### ภาพรวม
หัวข้อนี้จะแนะนำคุณเกี่ยวกับการแปลงไฟล์ OXPS เป็นสเปรดชีต Excel โดยใช้ GroupDocs.Conversion สำหรับ .NET

##### ขั้นตอนที่ 1: กำหนดเส้นทางไดเร็กทอรี
ตั้งค่าไดเร็กทอรีอินพุตและเอาต์พุตที่ไฟล์ของคุณอยู่

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
```

##### ขั้นตอนที่ 2: ระบุเส้นทางไฟล์
ระบุเส้นทางสำหรับทั้งไฟล์ OXPS ต้นทางและไฟล์ XLS เป้าหมาย

```csharp
string inputFile = Path.Combine(documentDirectory, "sample.oxps");
string outputFile = Path.Combine(outputDirectory, "oxps-converted-to.xls");
```

##### ขั้นตอนที่ 3: เริ่มต้นตัวแปลงและตั้งค่าตัวเลือก
สร้างอินสแตนซ์ของ `Converter` แบ่งชั้นเรียนและกำหนดค่าตัวเลือกการแปลง

```csharp
using (var converter = new Converter(inputFile))
{
    var options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Xls };
    converter.Convert(outputFile, options);
}
```

##### ขั้นตอนที่ 4: ดำเนินการแปลง
เรียกใช้กระบวนการแปลงเพื่อสร้างไฟล์ XLS ของคุณ
- **พารามิเตอร์**-
  - `inputFile`:เส้นทางสู่เอกสาร OXPS
  - `outputFile`:เส้นทางที่ต้องการสำหรับไฟล์ Excel ที่แปลงแล้ว

- **เคล็ดลับการแก้ไขปัญหา**-
  - ตรวจสอบให้แน่ใจว่าเส้นทางได้รับการตั้งค่าอย่างถูกต้องและมีไฟล์อยู่
  - ตรวจสอบว่าสภาพแวดล้อมของคุณมีสิทธิ์อ่าน/เขียนสำหรับไดเร็กทอรีที่ระบุ

## การประยุกต์ใช้งานจริง
1. **การรายงานข้อมูล**:แปลงข้อมูลการนำเสนอจาก OXPS เป็น XLS เพื่อการรายงานและการวิเคราะห์ที่ง่ายขึ้น
2. **การเก็บเอกสารถาวร**:จัดเก็บเวิร์กโฟลว์เอกสารในรูปแบบ Excel มาตรฐานเพื่อวัตถุประสงค์ในการเก็บถาวร
3. **ความเข้ากันได้ข้ามแพลตฟอร์ม**:รับรองความเข้ากันได้ระหว่างระบบต่างๆ โดยการแปลงเอกสารเป็นรูปแบบที่ได้รับการยอมรับอย่างกว้างขวาง เช่น XLS
