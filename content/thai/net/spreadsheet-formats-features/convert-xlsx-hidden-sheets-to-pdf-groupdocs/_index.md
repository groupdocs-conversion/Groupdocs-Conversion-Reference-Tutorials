## การแนะนำ

กำลังมองหาวิธีแปลงสเปรดชีต Excel รวมถึงสเปรดชีตที่ซ่อนอยู่เป็น PDF คุณภาพสูงโดยใช้ C# อยู่ใช่หรือไม่ บทช่วยสอนนี้จะแนะนำคุณตลอดขั้นตอนการใช้ GroupDocs.Conversion สำหรับ .NET ไม่ว่าเป้าหมายของคุณคือการรักษาความสมบูรณ์ของเอกสารหรือการรับรองการนำเสนอข้อมูลที่ครอบคลุมในรูปแบบที่เข้าถึงได้ทั่วไป คู่มือนี้เหมาะสำหรับคุณ

ในบทช่วยสอนทีละขั้นตอนนี้ เราจะครอบคลุม:

- การแปลงไฟล์ Excel (.xlsx) เป็น PDF ในขณะที่ใช้การตั้งค่าการแปลงขั้นสูง

## คู่มือการใช้งาน

### โหลดสเปรดชีตด้วยแผ่นงานที่ซ่อนอยู่

**ภาพรวม**:ฟีเจอร์นี้สาธิตวิธีโหลดสเปรดชีตเพื่อให้แน่ใจว่ามีการรวมแผ่นงานที่ซ่อนอยู่ในระหว่างกระบวนการแปลง

#### ขั้นตอนที่ 1: กำหนดตัวเลือกการโหลด

เริ่มต้นด้วยการสร้างฟังก์ชันเพื่อกำหนดตัวเลือกการโหลดของคุณ:

```csharp
Func<LoadContext, LoadOptions> getLoadOptions = loadContext => new SpreadsheetLoadOptions
{
    แสดงแผ่นงานที่ซ่อนอยู่ = true,
    OnePagePerSheet = true
};
```

- **ShowHiddenSheets**:รับรองว่ามีการประมวลผลแผ่นงานที่ซ่อนไว้
- **หนึ่งหน้าต่อหนึ่งแผ่น**:แต่ละแผ่นจะถูกแปลงเป็นหน้า PDF เดียว

### แปลงสเปรดชีตเป็น PDF ด้วยตัวเลือกขั้นสูง

**ภาพรวม**:ส่วนนี้จะครอบคลุมถึงการแปลงสเปรดชีตที่โหลดไว้แล้วเป็นไฟล์ PDF พร้อมทั้งใช้การตั้งค่าการแปลงเฉพาะเพื่อให้ได้ผลลัพธ์ที่เหมาะสมที่สุด

#### ขั้นตอนที่ 2: ตั้งค่าพารามิเตอร์การแปลง

ระบุตำแหน่งเอาต์พุตที่ต้องการและเริ่มต้นตัวแปลง:

```csharp
using System.IO;
using GroupDocs.Conversion.Options.Convert;

string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "converted.pdf");

// เริ่มต้นตัวแปลงด้วยตัวเลือกการโหลด
using (Converter converter = new Converter("path/to/your/document.xlsx", getLoadOptions))
{
    // กำหนดค่าการตั้งค่าการแปลงสำหรับ PDF
    var convertOptions = new PdfConvertOptions();

    // ดำเนินการแปลง
    converter.Convert(() => File.Create(outputFile), convertOptions);
}
```

- **ไดเรกทอรีผลลัพธ์**: ตรวจสอบให้แน่ใจว่าเส้นทางเอาต์พุตของคุณถูกต้องและสามารถเขียนได้
- **ตัวเลือกการแปลง PDF**ปรับแต่งสิ่งเหล่านี้เพื่อปรับเปลี่ยนรูปลักษณ์ของ PDF

#### เคล็ดลับการแก้ไขปัญหา

- ให้แน่ใจว่ามีแผ่นงานที่ซ่อนอยู่ในเอกสารของคุณโดยการตรวจสอบด้วยตนเองก่อนการแปลง
- ตรวจสอบเส้นทางไฟล์สำหรับไดเร็กทอรีทั้งอินพุตและเอาต์พุตเพื่อหลีกเลี่ยงข้อผิดพลาดขณะรันไทม์
