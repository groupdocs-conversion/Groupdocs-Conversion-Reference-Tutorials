## คู่มือการใช้งาน

มาแบ่งกระบวนการออกเป็นขั้นตอนที่สามารถจัดการได้:

### โหลดไฟล์ DOCM ต้นฉบับ

เริ่มต้นด้วยการโหลดไฟล์ DOCM ต้นทางของคุณ ตรวจสอบให้แน่ใจว่าคุณได้แทนที่ `'YOUR_DOCUMENT_DIRECTORY'` ด้วยเส้นทางจริงที่ไฟล์ DOCM ของคุณตั้งอยู่

```csharp
string documentPath = @"YOUR_DOCUMENT_DIRECTORY\\sample.docm";
```

ขั้นตอนนี้มีความสำคัญเนื่องจากจะระบุตำแหน่งที่แน่นอนของไฟล์ที่เราต้องการแปลง

### กำหนดไดเรกทอรีเอาต์พุตและเส้นทางไฟล์

จากนั้นระบุตำแหน่งที่คุณต้องการบันทึกไฟล์ CSV ที่แปลงแล้ว ใช้เส้นทางตัวแทนที่สอดคล้องกันเพื่อให้ปรับเปลี่ยนได้ง่าย:

```csharp
string outputFolder = @"YOUR_OUTPUT_DIRECTORY\\";
string outputFile = Path.Combine(outputFolder, "docm-converted-to.csv");
```

### การกำหนดค่าตัวเลือกการแปลง

กำหนดค่าตัวเลือกการแปลงเพื่อกำหนดเป้าหมายรูปแบบ CSV โดยการตั้งค่า `SpreadsheetConvertOptions`-

```csharp
var options = new SpreadsheetConvertOptions
{
    Format = GroupDocs.Conversion.FileTypes.SpreadsheetFileType.Csv
};
```

การกำหนดค่านี้จะกำหนดทิศทางให้กระบวนการแปลงส่งออกเป็นไฟล์ CSV

### ดำเนินการแปลง

สุดท้ายให้ดำเนินการแปลงโดยใช้ `Converter` คลาส วิธีการนี้จะอ่านไฟล์ DOCM และเขียนเนื้อหาที่แปลงแล้วลงในเส้นทางไฟล์ CSV ที่คุณระบุ:

```csharp
using (var converter = new Converter(documentPath))
{
    converter.Convert(outputFile, options);
}
```

การห่อหุ้มตรรกะนี้ไว้ในวิธีการช่วยให้เราสามารถนำกลับมาใช้ซ้ำสำหรับการแปลงหลายๆ ครั้งได้อย่างง่ายดาย

### เคล็ดลับการแก้ไขปัญหา

- **ปัญหาทั่วไป**:ข้อผิดพลาด "ไม่พบไฟล์" มักเกิดขึ้นหากเส้นทางไม่ถูกต้อง
  - **สารละลาย**ตรวจสอบไดเรกทอรีและชื่อไฟล์ของคุณอีกครั้ง
  
- **ปัญหาด้านประสิทธิภาพ**ไฟล์ DOCM ขนาดใหญ่จะใช้เวลาในการแปลงนานกว่า
  - **สารละลาย**:ควรพิจารณาเพิ่มประสิทธิภาพเอกสารหรือใช้งานบนเครื่องที่มีประสิทธิภาพสูงกว่า
