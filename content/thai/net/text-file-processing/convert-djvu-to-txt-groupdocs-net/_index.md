## คู่มือการใช้งาน

ทำตามขั้นตอนเหล่านี้เพื่อแปลงไฟล์ DJVU เป็นรูปแบบ TXT

### การโหลดและการแปลงไฟล์

#### ภาพรวม

GroupDocs.Conversion ช่วยให้โหลดและแปลงไฟล์ DJVU เป็นข้อความได้อย่างง่ายดาย โดยใช้ตัวเลือกการแปลงอันแข็งแกร่ง

##### ขั้นตอนที่ 1: กำหนดเส้นทางไฟล์

ขั้นแรก ระบุเอกสารและไดเร็กทอรีผลลัพธ์ของคุณ:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// เส้นทางไปยังไฟล์ DJVU แหล่งที่มา
class string djvuFilePath = Path.Combine(documentDirectory, "sample.djvu");
```

##### ขั้นตอนที่ 2: ตั้งค่าตัวเลือกการแปลง

เลือกตัวเลือกการแปลงที่เหมาะกับการประมวลผลข้อความ:

```csharp
var convertOptions = new WordProcessingConvertOptions
{
    Format = FileTypes.WordProcessingFileType.Txt
};
```

##### ขั้นตอนที่ 3: ดำเนินการแปลง

ดำเนินการแปลงและบันทึกผลลัพธ์:

```csharp
string txtOutputFile = Path.Combine(outputDirectory, "djvu-converted-to.txt");

using (var converter = new Converter(djvuFilePath))
{
    converter.Convert(txtOutputFile, convertOptions);
}
```

**คำอธิบาย:** 
- **คลาสตัวแปลง:** เริ่มต้นด้วยไฟล์ DJVU ของคุณ
- **วิธีการแปลง:** แปลงไฟล์โดยใช้ตัวเลือกที่ระบุและบันทึกไว้

##### เคล็ดลับการแก้ไขปัญหา

- ตรวจสอบให้แน่ใจว่าเส้นทางได้รับการตั้งค่าอย่างถูกต้องเพื่อหลีกเลี่ยง `FileNotFoundException`-
- ตรวจสอบความเข้ากันได้ของเวอร์ชันของ GroupDocs.Conversion
