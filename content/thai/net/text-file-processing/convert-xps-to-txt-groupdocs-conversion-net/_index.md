## คู่มือการใช้งาน

### แปลง XPS เป็น TXT

หัวข้อนี้ให้รายละเอียดวิธีการแปลงไฟล์ XPS เป็นรูปแบบข้อความ

#### ภาพรวม
การแปลงไฟล์ XPS เป็น TXT ช่วยให้จัดการและบูรณาการกับระบบอื่นๆ ได้ง่ายขึ้น เนื่องจากสามารถเข้าถึงข้อความธรรมดาได้ทั่วไป

#### การดำเนินการแบบทีละขั้นตอน

**1. กำหนดเส้นทางไฟล์**
ตั้งค่าไดเร็กทอรีอินพุตและเอาต์พุตของคุณ:

```csharp
string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY\\";
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY\\";

// อินพุตเส้นทางไฟล์ XPS
string inputFile = Path.Combine(documentDirectory, "sample.xps");

// เส้นทางไฟล์ TXT เอาท์พุต
string outputFile = Path.Combine(outputDirectory, "xps-converted-to.txt");
```

**2. โหลดไฟล์ XPS ต้นฉบับ**
ใช้ GroupDocs.Conversion เพื่อโหลดไฟล์ต้นฉบับของคุณ:

```csharp
using (var converter = new Converter(inputFile))
{
    Console.WriteLine("XPS file loaded successfully.");
}
```

**3. ตั้งค่าตัวเลือกการแปลง**
กำหนดค่าการตั้งค่าการแปลงสำหรับเอาท์พุต TXT:

```csharp
var options = new WordProcessingConvertOptions { Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt };
```
ที่นี่, `WordProcessingConvertOptions` ระบุรูปแบบข้อความ

**4. ดำเนินการแปลง**
ดำเนินการแปลงและบันทึกผลลัพธ์:

```csharp
converter.Convert(outputFile, options);
Console.WriteLine("Conversion completed successfully.");
```
