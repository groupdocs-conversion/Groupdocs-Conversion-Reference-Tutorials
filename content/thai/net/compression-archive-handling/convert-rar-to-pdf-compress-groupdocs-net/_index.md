## คู่มือการใช้งาน

### คุณสมบัติ 1: แปลง RAR เป็น PDF

#### ภาพรวม
ฟีเจอร์นี้ช่วยให้คุณแปลงไฟล์ RAR เป็นรูปแบบ PDF ได้ ซึ่งมีประโยชน์สำหรับสถานการณ์ที่คุณต้องการแจกจ่ายหรือเก็บเอกสารในรูปแบบที่เข้าถึงได้ง่ายกว่า

#### การดำเนินการแบบทีละขั้นตอน

**ขั้นตอนที่ 3.1: โหลดไฟล์ RAR**

เริ่มต้นด้วยการโหลดไฟล์ RAR ของคุณโดยใช้ `FluentConverter`. แทนที่ `"YOUR_DOCUMENT_DIRECTORY\sample.rar"` พร้อมเส้นทางจริงไปยังไฟล์ RAR ของคุณ:

```csharp
using System.IO;
using GroupDocs.Conversion.FileTypes;
using GroupDocs.Conversion.Options.Convert;

// โหลดและแปลงไฟล์ RAR เป็น PDF
var convertedStream = FluentConverter.Load(@"YOUR_DOCUMENT_DIRECTORY\sample.rar")
    .ConvertTo((SaveContext p) => new MemoryStream())
    .WithOptions(new PdfConvertOptions());
```

**ขั้นตอนที่ 3.2: กำหนดค่าตัวเลือก PDF**

การ `PdfConvertOptions` คลาสนี้ช่วยให้คุณปรับแต่งกระบวนการแปลงได้ คุณสามารถระบุการตั้งค่าเพิ่มเติม เช่น ขนาดหน้า ขอบ และอื่นๆ หากจำเป็น

### คุณสมบัติ 2: บีบอัดไฟล์ที่แปลงแล้วเป็น Zip

#### ภาพรวม
หลังจากแปลงไฟล์ RAR เป็น PDF แล้ว การบีบอัดผลลัพธ์นี้ให้เป็นรูปแบบ ZIP จะช่วยให้จัดเก็บและแบ่งปันได้ง่ายขึ้น

#### การดำเนินการแบบทีละขั้นตอน

**ขั้นตอนที่ 3.3: กำหนดโฟลเดอร์ผลลัพธ์**

ตั้งค่าไดเรกทอรีที่จะบันทึกไฟล์ ZIP ที่ถูกบีบอัด:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
```

**ขั้นตอนที่ 3.4: บีบอัดเป็นรูปแบบ Zip**

ใช้ประโยชน์ `FluentConverter` อีกครั้งคราวนี้จะบีบอัด PDF ที่แปลงแล้วลงในไฟล์ ZIP

```csharp
using GroupDocs.Conversion.Options.Convert;

var compressedStream = FluentConverter.Load(@"YOUR_DOCUMENT_DIRECTORY\sample.rar")
    .ConvertTo((SaveContext p) => new MemoryStream())
    .Compress(new CompressionConvertOptions { Format = CompressionFileType.Zip })
    .OnCompressionCompleted(compressedStream =>
    {
        using (var fs = new FileStream(Path.Combine(outputFolder, "converted.zip"), FileMode.Create))
        {
            compressedStream.CopyTo(fs);
        }
    });
```

**ขั้นตอนที่ 3.5: บันทึกไฟล์ที่บีบอัด**

วิธีการโทรกลับ `OnCompressionCompleted` ช่วยให้แน่ใจว่าไฟล์ ZIP ถูกบันทึกไว้ในไดเร็กทอรีเอาท์พุตที่คุณระบุ

### เคล็ดลับการแก้ไขปัญหา

- ตรวจสอบให้แน่ใจว่าเส้นทางได้รับการตั้งค่าอย่างถูกต้องสำหรับไฟล์อินพุตและเอาต์พุต
- ตรวจสอบสิทธิ์ที่เหมาะสมสำหรับการอ่าน/เขียนในไดเร็กทอรีที่ระบุ
- ตรวจสอบว่าสิ่งที่ต้องติดตั้งทั้งหมดถูกต้อง
