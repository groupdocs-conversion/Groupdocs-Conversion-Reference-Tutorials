## คู่มือการใช้งาน

### คุณสมบัติการแปลงไฟล์ JPEG เป็น PDF

หัวข้อนี้จะแนะนำคุณเกี่ยวกับการแปลงภาพ JPEG เป็นเอกสาร PDF โดยใช้ GroupDocs.Conversion

#### ขั้นตอนที่ 1: โหลดไฟล์ต้นฉบับ

เริ่มต้นด้วยการโหลดไฟล์ JPEG ต้นฉบับของคุณ `Converter` คลาสจัดการกระบวนการนี้:

```csharp
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY\\sample.jpeg"))
{
    // ดำเนินการตามขั้นตอนการแปลง...
}
```

*ทำไม* การโหลดไฟล์จะเป็นการเริ่มต้นไฟล์ภายใน GroupDocs.Conversion เพื่อเตรียมพร้อมสำหรับการประมวลผล

#### ขั้นตอนที่ 2: กำหนดตัวเลือกการแปลง PDF

ระบุตัวเลือกที่คุณต้องการสำหรับการแปลงเป็น PDF:

```csharp
var options = new PdfConvertOptions();
```

*วัตถุประสงค์:* การ `PdfConvertOptions` คลาสอนุญาตให้ปรับแต่งกระบวนการแปลงได้ แม้ว่าการตั้งค่าเริ่มต้นมักจะเพียงพอก็ตาม

#### ขั้นตอนที่ 3: ดำเนินการแปลง

ดำเนินการแปลงและบันทึกผลลัพธ์ PDF:

```csharp
converter.Convert(outputFile, options);
```

*ทำไม* วิธีนี้จะทริกเกอร์กระบวนการแปลงจริงและเขียนผลลัพธ์ไปยังตำแหน่งที่ระบุในรูปแบบ PDF
