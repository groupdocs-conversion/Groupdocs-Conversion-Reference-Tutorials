# Chargement de documents depuis le stockage cloud : tutoriels pour GroupDocs.Conversion .NET

Nos tutoriels sur le stockage cloud vous expliquent comment intégrer GroupDocs.Conversion aux plateformes de stockage cloud les plus populaires dans vos applications .NET. Ces guides détaillés expliquent comment se connecter aux buckets Amazon S3, accéder aux conteneurs Azure Blob Storage, configurer les identifiants appropriés, gérer les paramètres régionaux, gérer le pool de connexions et implémenter une logique de nouvelle tentative pour les opérations cloud. Chaque tutoriel inclut des exemples pratiques de code C# pour la récupération de documents cloud, vous aidant à créer des applications robustes qui exploitent les référentiels de documents cloud dans vos workflows de conversion.

## Tutoriels disponibles

### [Automatiser la conversion de fichiers S3 avec GroupDocs.Conversion pour .NET : guide étape par étape](./automate-s3-file-conversion-groupdocs/)
Découvrez comment automatiser la conversion de fichiers depuis Amazon S3 grâce au SDK AWS et à GroupDocs.Conversion pour .NET. Optimisez efficacement votre gestion documentaire.

### [Convertir des fichiers Azure Blob Storage au format PDF à l'aide de .NET et de GroupDocs.Conversion](./convert-azure-blob-storage-files-to-pdf-net/)
Découvrez comment télécharger facilement des fichiers depuis Azure Blob Storage et les convertir au format PDF avec .NET et GroupDocs.Conversion. Suivez ce guide complet pour une gestion efficace de vos documents.
