# CAD和技术绘图格式教程（适用于GroupDocs.Conversion .NET）

在此中心，您将找到使用 GroupDocs.Conversion for .NET 将 **convert CAD to PDF**（转换 CAD 为 PDF）的逐步指南，涵盖自定义页面尺寸、3‑D 模型展平、图层保留以及大图纸处理。无论您是构建渲染工程图的 Web 服务，还是批量处理数千张图纸的桌面工具，这些教程都展示了如何在保持低内存使用的同时获得可靠的生产就绪结果。

GroupDocs.Conversion for .NET 是一个 .NET 库，可将超过 150 种格式（包括 CAD 图纸）转换为 PDF、图像和 Office 文件。它同时支持 .NET Framework 和 .NET Core，可在 Windows、Linux 和 macOS 上运行，并且无需在服务器上安装任何外部 CAD 软件。

**Quantified claim:** GroupDocs.Conversion 能够处理高达 500 MB 的 CAD 文件，并在典型的 4 核 VM 上在 30 秒内转换数百页的图纸，这得益于其流式架构，避免将整个源文件加载到内存中。
