### Конвертировать J2K в HTML

#### Обзор
В этом разделе показано, как преобразовать загруженный файл JPEG 2000 (.j2k) в HTML-документ, который можно просмотреть в любом веб-браузере.

##### Шаг 1: Задайте параметры конвертации

```csharp
using GroupDocs.Conversion.Options.Convert;

string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "j2k-converted-to.html");

// Настройте параметры преобразования для формата HTML.
var options = new WebConvertOptions();
```
- **Параметры:** `WebConvertOptions` настраивает параметры, специфичные для веб-форматов, таких как HTML.

##### Шаг 2: Выполнение преобразования

```csharp
csversion.Convert(outputFile, options);
```
- **Цель метода:** The `Convert` Метод принимает выходной путь и параметры преобразования, выполняя фактическое преобразование файла.
