## Руководство по внедрению

### Функция 1: Преобразование защищенного паролем документа в HTML

#### Обзор

Преобразование защищенного паролем документа в формат HTML обеспечивает более широкую доступность и более простую интеграцию с веб-приложениями. Вот как этого добиться с помощью GroupDocs.Conversion.

#### Пошаговые инструкции

**Шаг 1**: Определите каталоги и пути к файлам.

```csharp
string dataDir = Path.Combine(Directory.GetCurrentDirectory(), "YOUR_DOCUMENT_DIRECTORY");
string outputDir = Path.Combine(Directory.GetCurrentDirectory(), "YOUR_OUTPUT_DIRECTORY");

string inputFilePath = Path.Combine(dataDir, "SAMPLE_DOCX_WITH_PASSWORD.docx");
string outputFile = Path.Combine(outputDir, "converted.html");
```

**Шаг 2**Создайте функцию для извлечения параметров загрузки с подробностями защиты паролем.

```csharp
Func<LoadContext, LoadOptions> getLoadOptions = loadContext => new WordProcessingLoadOptions
{
    Password = "12345" // Укажите пароль документа здесь
};
```

**Шаг 3**: Инициализируйте конвертер и настройте параметры конвертации.

```csharp
using (Converter converter = new Converter(inputFilePath, getLoadOptions))
{
    WebConvertOptions options = new WebConvertOptions
    {
        PageNumber = 2, 
        FixedLayout = true,
        PagesCount = 1,
        FixedLayoutShowBorders = false
    };
    
    // Выполнить преобразование
    converter.Convert(outputFile, options);
}
```

В этом разделе `PageNumber`, `PagesCount`и другие параметры позволяют вам точно настроить, какие части вашего документа будут преобразованы.

### Функция 2: Укажите страницы для конвертации

#### Обзор

Иногда требуется конвертировать только определенные страницы. GroupDocs.Conversion упрощает указание точных страниц, которые вам нужны, в формате HTML.

#### Пошаговые инструкции

**Шаг 1**: Определите пути и инициализируйте конвертер, как показано ранее, но без защиты паролем.

```csharp
string inputFilePath = Path.Combine(dataDir, "SAMPLE_DOCX.docx");
string outputFile = Path.Combine(outputDir, "pages_converted.html");

using (Converter converter = new Converter(inputFilePath))
{
    WebConvertOptions options = new WebConvertOptions
    {
        PageNumber = 2,
        PagesCount = 1,
        FixedLayout = true,
        FixedLayoutShowBorders = false
    };
    
    // Конвертировать указанные страницы в формат HTML
    converter.Convert(outputFile, options);
}
```

### Советы по устранению неполадок
- **Неверный пароль**: Убедитесь, что пароль введен правильно и соответствует уровню защиты документа.
- **Отсутствующие зависимости**: Еще раз проверьте, что все необходимые пакеты установлены через NuGet.
