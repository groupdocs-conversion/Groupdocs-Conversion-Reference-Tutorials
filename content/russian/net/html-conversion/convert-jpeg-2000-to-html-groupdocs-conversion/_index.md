## Настройка GroupDocs.Conversion для .NET

### Информация об установке
Начните с установки библиотеки с помощью консоли диспетчера пакетов NuGet или .NET CLI.

**Консоль диспетчера пакетов NuGet**
```bash
Install-Package GroupDocs.Conversion -Version 25.3.0
```

**\.NET CLI**
```bash
dotnet add package GroupDocs.Conversion --version 25.3.0
```

### Этапы получения лицензии
GroupDocs предлагает бесплатную пробную версию, позволяющую вам изучить функции перед покупкой или получением временной лицензии.
- **Бесплатная пробная версия**: Протестируйте библиотеку со всеми включенными функциями.
- **Временная лицензия**: Получите, если вам необходимо более обширное тестирование без ограничений по оценке.
- **Покупка**: Если вас все устраивает, приобретите лицензию для постоянного использования.
