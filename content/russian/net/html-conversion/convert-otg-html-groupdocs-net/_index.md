### Базовая инициализация и настройка

Вот как настроить среду для преобразования:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

// Определить пути для входных и выходных каталогов
string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY\";

// Инициализируйте путь к исходному файлу OTG
string sourceOtgFilePath = Path.Combine(documentDirectory, "sample.otg");
// Настройте путь к выходному HTML-файлу
string outputHtmlFilePath = Path.Combine(outputDirectory, "otg-converted-to.html");

// Загрузите и подготовьте к конвертации
using (var converter = new Converter(sourceOtgFilePath))
{
    // Создать параметры преобразования для формата HTML
    var htmlConversionOptions = new WebConvertOptions();
    
    // Выполнить преобразование из OTG в HTML
    converter.Convert(outputHtmlFilePath, htmlConversionOptions);
}
```

## Руководство по внедрению

### Функция: загрузка и конвертация файла OTG в HTML

Эта функция позволяет загружать файл графического шаблона OpenDocument (OTG) и преобразовывать его в формат HTML с помощью GroupDocs.Conversion.

#### Шаг 1: Определите пути к документам

Определите входные и выходные пути для ваших документов, чтобы обеспечить правильную загрузку и сохранение файлов:

```csharp
// Определить пути документов
currentDirectory = @"YOUR_DOCUMENT_DIRECTORY";
externalDirectory = @"YOUR_OUTPUT_DIRECTORY\";

sourceOtgFilePath = Path.Combine(currentDirectory, "sample.otg");
outputHtmlFilePath = Path.Combine(externalDirectory, "otg-converted-to.html");
```

#### Шаг 2: Загрузите файл OTG

Использование GroupDocs.Conversion `Converter` класс, загрузите ваш OTG-файл:

```csharp
// Инициализируйте конвертер с исходным путем к файлу OTG
groupDocsConverter = new Converter(sourceOtgFilePath);
{
    // Подготовьтесь к конвертации с использованием параметров HTML
    htmlConversionOptions = new WebConvertOptions();
    
    // Конвертировать и сохранить как HTML
    groupDocsConverter.Convert(outputHtmlFilePath, htmlConversionOptions);
}
```

**Параметры и цели метода:**
- `Converter`: Загружает исходный файл для преобразования.
- `WebConvertOptions()`: Указывает, что выходной формат должен быть HTML.

#### Советы по устранению неполадок

Если у вас возникли проблемы:
- Убедитесь, что все пути правильно настроены и доступны.
- Убедитесь, что GroupDocs.Conversion правильно установлен и лицензирован.
