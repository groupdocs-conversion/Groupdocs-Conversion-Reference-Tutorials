### Базовая инициализация и настройка
После установки инициализируйте GroupDocs.Conversion в вашем проекте .NET:
```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

namespace ConvertXlsToHtml
{
    class Program
    {
        static void Main(string[] args)
        {
            string sourceFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.xls");
            string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", "ConvertedHtml");

            if (!Directory.Exists(outputFolder))
            {
                Directory.CreateDirectory(outputFolder);
            }

            string outputFile = Path.Combine(outputFolder, "xls-converted-to.html");

            using (var converter = new GroupDocs.Conversion.Converter(sourceFilePath))
            {
                var options = new WebConvertOptions();
                converter.Convert(outputFile, options);
            }
        }
    }
}
```

## Руководство по внедрению
Давайте разберем процесс конвертации на четкие шаги.

### Функция: загрузка и конвертация XLS в HTML

#### Обзор
Эта функция позволяет вам загружать файл XLS и конвертировать его в документ HTML с помощью GroupDocs.Conversion для .NET. Это особенно полезно, когда вам нужно отобразить данные электронной таблицы на веб-странице.

#### Этапы внедрения

**Шаг 1: Определите пути к файлам**
- Укажите путь к исходному XLS-файлу и выходной каталог, в котором будет сохранен преобразованный HTML-файл.
```csharp
string sourceFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.xls");
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", "ConvertedHtml");

if (!Directory.Exists(outputFolder))
{
    Directory.CreateDirectory(outputFolder);
}
```

**Шаг 2: Инициализация конвертера**
- Создайте экземпляр `Converter` class с вашим исходным путем файла. Это загрузит файл XLS в память.
```csharp
using (var converter = new GroupDocs.Conversion.Converter(sourceFilePath))
{
    // Процесс конвертации будет осуществляться здесь.
}
```

**Шаг 3: Укажите параметры преобразования HTML**
- Использовать `WebConvertOptions` для установки параметров преобразования для формата HTML. Это гарантирует соответствие преобразованного файла веб-стандартам.
```csharp
var options = new WebConvertOptions();
```

**Шаг 4: Выполнение преобразования**
- Позвоните `Convert` метод, передавая путь к выходному файлу и параметры преобразования. Результатом является HTML-документ, сгенерированный из вашего XLS-файла.
```csharp
string outputFile = Path.Combine(outputFolder, "xls-converted-to.html");
converter.Convert(outputFile, options);
```

#### Советы по устранению неполадок
- Убедитесь, что путь к исходному файлу правильный и доступный.
- Проверьте права доступа к каталогу для записи преобразованных файлов.
- Проверьте, указана ли в вашем проекте правильная версия GroupDocs.Conversion.
