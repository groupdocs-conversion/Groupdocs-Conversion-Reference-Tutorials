## Руководство по внедрению
Мы разделим реализацию на две основные функции: загрузку PST-файла и преобразование его в формат TEX.

### Функция 1: Загрузка файла PST
Эта функция демонстрирует, как загрузить файл PST с помощью GroupDocs.Conversion. Загрузка файлов с определенными параметрами может иметь решающее значение для эффективной обработки больших наборов данных.

#### Пошаговая реализация:
**3.1 Определите путь к документу**
Укажите путь к PST-файлу, убедившись, что он доступен вашему приложению.
```csharp
string documentPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.pst");
```

**3.2 Инициализация параметров загрузки**
Использовать `PersonalStorageLoadOptions` для загрузки параметров, адаптированных к файлам PST.
```csharp
var loadOptions = new PersonalStorageLoadOptions();
```

**3.3 Создание объекта-конвертера**
Создайте объект-преобразователь с определенными условиями контекста, используя лямбда-выражения.
```csharp
using (var converter = new GroupDocs.Conversion.Converter(documentPath, context => 
    context.SourceFormat == GroupDocs.Conversion.FileTypes.EmailFileType.Ost ? loadOptions : null))
{
    // Теперь конвертер готов к дальнейшим операциям, таким как преобразование.
}
```

**3.4 Объяснение**
- **`documentPath`**: Направляет приложение к местоположению вашего PST-файла.
- **Лямбда-функция**: Обеспечивает правильность `loadOptions` применяются в зависимости от типа файла.

### Функция 2: Преобразование файла PST в формат TEX
Далее мы преобразуем загруженный файл PST в формат TEX. Это особенно полезно для создания форматированных документов из данных электронной почты, хранящихся в файлах PST.

#### Пошаговая реализация:
**4.1 Установить выходной каталог**
Определите, где будут сохранены преобразованные файлы.
```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", "output");
string outputFileTemplate = Path.Combine(outputFolder, "pst-converted-{0}-to.tex");
```

**4.2 Инициализация конвертера и параметры конвертации**
Загрузите файл PST в объект-конвертер и укажите параметры преобразования для формата TEX.
```csharp
using (var converter = new GroupDocs.Conversion.Converter(Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.pst")))
{
    PageDescriptionLanguageConvertOptions options = new PageDescriptionLanguageConvertOptions { Format = PageDescriptionLanguageFileType.Tex };
    
    // Выполнить преобразование
    converter.Convert(
        (SaveContext saveContext) => new FileStream(string.Format(outputFileTemplate, counter++), FileMode.Create),
        options

```

**4.3 Объяснение**
- **`outputFolder`**: Содержит путь для хранения преобразованных файлов.
- **Параметры формата TEX**: Настраивает тип и структуру выходного файла.
