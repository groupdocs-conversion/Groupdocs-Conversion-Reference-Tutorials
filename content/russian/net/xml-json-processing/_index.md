# Учебники по обработке XML и JSON для GroupDocs.Conversion .NET

Изучите возможности преобразования форматов данных с помощью наших учебных пособий GroupDocs.Conversion для разработчиков .NET. Эти пошаговые руководства показывают, как использовать XML и JSON в качестве источников данных для создания электронных таблиц и других типов документов, сохранять иерархические структуры данных во время преобразования, поддерживать сложные объектные связи, обрабатывать массивы и коллекции, управлять пространствами имен и схемами, а также реализовывать операции преобразования и сопоставления данных. Каждое учебное пособие включает практические примеры кода C# для работы со структурированными форматами данных, помогая вам создавать приложения, которые могут преобразовывать машиночитаемые данные в удобные для человека форматы документов.
