## Руководство по внедрению

### Получение возможных конверсий

#### Обзор
Откройте для себя ряд форматов, в которые можно конвертировать документы с помощью GroupDocs.Conversion.

#### Пошаговая реализация
**Получить параметры преобразования**
```csharp
using System;
using GroupDocs.Conversion.Contracts;

string inputFile = "YOUR_DOCUMENT_DIRECTORY/sample.docx";
var possibleConversions = Converter.GetPossibleConversions(inputFile);

foreach (var conversion in possibleConversions)
{
    Console.WriteLine(conversion.Format);
}
```
**Объяснение**: Этот фрагмент извлекает все форматы, в которые можно преобразовать файл DOCX, используя `GetPossibleConversions` метод, который возвращает массив параметров преобразования.

### Настройка параметров загрузки для документов Word

#### Обзор
Узнайте, как безопасно работать с документами, защищенными паролем.

#### Пошаговая реализация
**Установить защиту паролем**
```csharp
using System;
using GroupDocs.Conversion.Options.Load;

string inputFile = "YOUR_DOCUMENT_DIRECTORY/sample.docx";
var loadOptions = (WordProcessingLoadOptions) Converter.GetPossibleConversions(inputFile).LoadOptions;
loadOptions.Password = "12345";

Console.WriteLine("Password set in load options: {0}", loadOptions.Password);
```
**Объяснение**: Здесь мы настраиваем `WordProcessingLoadOptions` указать пароль для защищенных документов. Это гарантирует, что только авторизованные пользователи смогут получить доступ к контенту.

### Преобразование документа в формат PDF

#### Обзор
Конвертируйте документы Word в PDF-файлы без особых усилий с помощью индивидуальных настроек конвертации.

#### Пошаговая реализация
**Конвертировать в PDF**
```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

string inputFile = "YOUR_DOCUMENT_DIRECTORY/sample.docx";
string outputFolder = "path/to/output/directory";
string outputFile = Path.Combine(outputFolder, "converted.pdf");

var loadOptions = new WordProcessingLoadOptions
{
    Password = "12345"
};

using (Converter converter = new Converter(inputFile, () => loadOptions))
{
    var convertOptions = new PdfConvertOptions();
    converter.Convert(outputFile, convertOptions);
}
```
**Объяснение**: Этот код демонстрирует преобразование файла DOCX в PDF. Он инициализирует `Converter` класс с параметрами ввода и загрузки, затем выполняет преобразование с использованием `PdfConvertOptions`.
