## Руководство по внедрению
Давайте разберемся, как получить все возможные преобразования с помощью GroupDocs.Conversion для .NET.

### Обзор функций: получение всех возможных преобразований
Понимание того, какие форматы файлов доступны для преобразования, имеет решающее значение. Эта функция позволяет программно определять эти параметры из заданного исходного файла.

#### Пошаговая реализация:
**1. Получить все возможные преобразования**
Используйте `GetAllPossibleConversions` метод для извлечения всех потенциальных путей конверсии для ваших документов.

```csharp
var possibleConversions = Converter.GetAllPossibleConversions();
```

**2. Повторите каждую возможность конверсии**
Выполните итерацию по каждому расширению исходного файла и связанным с ним преобразованиям:

```csharp
foreach (var conversions in possibleConversions)
{
    Console.WriteLine("{0} could be converted to:", conversions.Source.Extension);
    
    foreach (var conversion in conversions.All)
    {
        var conversionType = conversion.IsPrimary ? "primary" : "secondary";
        Console.WriteLine("\t {0} as {1} conversion.", conversion.Format, conversionType);
    }
}
```

**Параметры и возвращаемые значения:**
- `GetAllPossibleConversions`: Возвращает коллекцию потенциальных конверсий.
- Каждый объект преобразования включает в себя такие сведения, как формат и является ли он первичным или вторичным.

### Основные параметры конфигурации
Понимание различий между первичными и вторичными преобразованиями позволяет вам расставить приоритеты в отношении форматов, имеющих решающее значение для вашего приложения.
