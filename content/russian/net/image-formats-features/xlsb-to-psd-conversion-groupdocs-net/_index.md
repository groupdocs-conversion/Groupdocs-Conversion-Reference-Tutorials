## Руководство по внедрению

### Загрузить файл XLSB

**Обзор:**
Начните с загрузки файла XLSB с помощью `Converter` класс для доступа к его содержимому и его преобразования.

#### Пошаговое руководство:
1. **Инициализировать преобразователь**: Создать новый экземпляр `Converter` с путем к вашему файлу.
    ```csharp
    string documentPath = "YOUR_DOCUMENT_DIRECTORY/SAMPLE_XLSB";
    using (Converter converter = new Converter(documentPath))
    {
        // Здесь будет находиться код преобразования.
    }
    ```
2. **Распоряжаться ресурсами**: Обеспечьте высвобождение ресурсов после использования.

### Установить параметры преобразования для формата PSD

**Обзор:**
Определите параметры преобразования, специфичные для формата PSD, используя `ImageConvertOptions`.

#### Пошаговое руководство:
1. **Создать ImageConvertOptions**: Укажите вывод как PSD.
    ```csharp
    using GroupDocs.Conversion.Options.Convert;

    ImageConvertOptions options = new ImageConvertOptions 
    {
        Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd
    };
    ```
2. **Объяснение параметров**: `Format` определяет целевой формат преобразования, устанавливается значение «Psd».

### Конвертировать XLSB в PSD

**Обзор:**
Выполните преобразование файла, сохранив каждую страницу как отдельный PSD-файл в выходном каталоге.

#### Пошаговое руководство:
1. **Установить выходной каталог и шаблон**: Определите, где будут храниться преобразованные файлы.
    ```csharp
    string outputFolder = "YOUR_OUTPUT_DIRECTORY";
    string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");
    ```
2. **Создать потоковую функцию**: Динамически генерировать пути к файлам для каждой страницы.
    ```csharp
    Func<SavePageContext, Stream> getPageStream = savePageContext => 
        new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
    ```
3. **Выполнить преобразование**: Используйте `Converter` экземпляр и параметры для преобразования вашего файла.
    ```csharp
    using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_XLSB"))
    {
        ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd };
        converter.Convert(getPageStream, options);
    }
    ```

### Советы по устранению неполадок
- **Проблемы с путями к файлам**: Убедитесь, что пути к документам указаны правильно.
- **Управление памятью**: Утилизируйте ресурсы, чтобы предотвратить утечки памяти.
