## Руководство по внедрению
### Настройка преобразования файлов
Эта функция демонстрирует, как настроить процесс конвертации из формата JPM в PSD. Она включает определение выходного каталога и шаблона для именования конвертированных файлов.
#### Определить выходной каталог
Укажите желаемую выходную папку, в которую будут сохраняться преобразованные файлы:
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
```
#### Шаблон именования для преобразованных файлов
Создайте функцию, которая динамически генерирует пути к файлам на основе результатов преобразования:
```csharp
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");
Func<SavePageContext, Stream> getPageStream = savePageContext => new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```
### Загрузка исходного файла
Загрузите исходный файл JPM для конвертации с помощью `Converter` сорт.
#### Инициализировать конвертер с исходным файлом
```csharp
global using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_JPM"))
{
    // Настройка преобразования будет реализована далее...
}
```
### Настройка параметров конвертации
Настройте параметры, необходимые для преобразования изображения из формата JPM в PSD.
#### Определить параметры преобразования изображения
Задайте целевой формат файла и другие соответствующие параметры:
```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd };
```
### Выполнение преобразования файлов
Выполните преобразование, используя предопределенные параметры и функцию выходного потока.
#### Выполнить преобразование
Выполните фактическое преобразование файла с помощью `Convert` метод:
```csharp
current.Convert(getPageStream, options);
```
