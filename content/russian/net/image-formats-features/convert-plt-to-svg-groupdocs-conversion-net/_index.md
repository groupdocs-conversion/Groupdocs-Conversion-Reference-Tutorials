## Настройка GroupDocs.Conversion для .NET

Для начала вам нужно установить библиотеку GroupDocs.Conversion в вашем проекте. Вот как это сделать:

**Консоль диспетчера пакетов NuGet**
```bash
Install-Package GroupDocs.Conversion -Version 25.3.0
```

**.NET CLI**
```bash
dotnet add package GroupDocs.Conversion --version 25.3.0
```

## Руководство по внедрению

### Загрузите и преобразуйте файл PLT в SVG

Эта функция позволяет преобразовать файл PLT в формат SVG, который широко используется для масштабируемой векторной графики.

#### Шаг 1: Определите выходной каталог

Сначала укажите, где будут сохраняться преобразованные файлы:
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY/";
string outputFile = Path.Combine(outputFolder, "plt-converted-to.svg");
```

#### Шаг 2: Загрузите файл PLT

Используйте `Converter` класс для загрузки вашего PLT файла. Заменить `'sample.plt'` с фактическим путем к файлу.
```csharp
using (var converter = new GroupDocs.Conversion.Converter("YOUR_DOCUMENT_DIRECTORY/sample.plt"))
{
    // Логика преобразования будет здесь
}
```

#### Шаг 3: Укажите параметры конвертации

Определите параметры преобразования для формата SVG:
```csharp
var options = new PageDescriptionLanguageConvertOptions { Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Svg };
```

#### Шаг 4: Выполнение преобразования

Выполните преобразование и сохраните выходной файл.
```csharp
converter.Convert(outputFile, options);
```
