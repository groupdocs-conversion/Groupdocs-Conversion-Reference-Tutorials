## Руководство по внедрению

### Обзор функций: преобразование формата JPX в PSD
Эта функция предназначена для преобразования высококачественной графики из файлов JPX в файлы PSD, широко используемые пользователями Adobe Photoshop.

#### Шаг 1: Определите выходной каталог и шаблон файла
Во-первых, настройте выходной каталог, в котором будут храниться ваши преобразованные файлы. Вы можете определить шаблон для именования этих файлов, как показано ниже:
```csharp
using System.IO;
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");
```

#### Шаг 2: Создание потоков файлов для преобразованных страниц
Создайте функцию для динамического создания потоков файлов. Это поможет управлять каждой страницей преобразованного документа:
```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

#### Шаг 3: Загрузите исходный файл JPX
Используйте `Converter` класс для загрузки исходного файла JPX. Убедитесь, что вы указали правильный путь:
```csharp
using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_JPX"))
{
    // Последует процесс конвертации.
}
```

#### Шаг 4: Задайте параметры преобразования для формата PSD
Определите параметры преобразования, адаптированные для формата PSD, используя `ImageConvertOptions`:
```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Psd };
```

#### Шаг 5: Выполнение преобразования
Выполните процесс конвертации, в результате которого будет создан файл PSD для каждой страницы вашего документа JPX:
```csharp
converter.Convert(getPageStream, options);
```

### Советы по устранению неполадок
- **Ошибки пути к файлу**: Убедитесь, что все пути к каталогам верны и доступны.
- **Несоответствие версии библиотеки**: Убедитесь, что у вас установлена соответствующая версия GroupDocs.Conversion.
