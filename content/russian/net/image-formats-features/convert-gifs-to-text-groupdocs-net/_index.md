## Руководство по внедрению

### Шаг 1: Определите путь к выходной папке
Сначала укажите, где вы хотите сохранить преобразованный TXT-файл:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY"; // Замените на ваш фактический выходной каталог
```

### Шаг 2: Настройте параметры конвертации
Определите, как GIF будет преобразован в текстовый формат:

```csharp
WordProcessingConvertOptions options = new WordProcessingConvertOptions { Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt };
```
На этом этапе GroupDocs настраивается на использование параметров преобразования файлов TXT.

### Шаг 3: Загрузите и преобразуйте исходный GIF-файл
Загрузите исходный GIF-файл из его каталога:

```csharp
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY/sample.gif")) // Убедитесь, что путь правильный
{
    string outputFile = Path.Combine(outputFolder, "gif-converted-to.txt");
    converter.Convert(outputFile, options);
}
```
Здесь, `Converter` загружает GIF-файл. `Convert` Затем для выполнения преобразования используется метод.
