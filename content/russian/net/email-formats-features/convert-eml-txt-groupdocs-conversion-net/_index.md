## Руководство по внедрению

### Функция: конвертация EML в TXT
Эта функция позволяет преобразовать файл EML в более удобный формат TXT с помощью GroupDocs.Conversion.

#### Шаг 1: Определите пути к файлам
Начните с установки путей для ваших входных и выходных каталогов. Это поможет вашему приложению узнать, откуда читать и куда писать.
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY"; // Укажите каталог вашего файла EML
string outputDirectory = "YOUR_OUTPUT_DIRECTORY"; // Укажите выходной каталог TXT

// Определить имена файлов
class Program
{
    static void Main()
    {
        string inputFile = Path.Combine(documentDirectory, "sample.eml");
        string outputFile = Path.Combine(outputDirectory, "eml-converted-to.txt");
    }
}
```
#### Шаг 2: Загрузите исходный файл EML
Используйте GroupDocs.Conversion для загрузки исходного файла EML.
```csharp
using (var converter = new Converter(inputFile))
{
    // Параметры конвертации будут определены на следующем этапе.
}
```
*Зачем использовать `using`? Это гарантирует правильную утилизацию ресурсов после завершения преобразования.*

#### Шаг 3: Определите параметры преобразования
Настройте преобразование для вывода в виде файла TXT, используя определенные параметры формата.
```csharp
var options = new WordProcessingConvertOptions { Format = FileType.Txt };
```
*The `WordProcessingConvertOptions` класс предоставляет различные параметры конфигурации, позволяющие указать желаемый формат вывода.*

#### Шаг 4: Конвертировать и сохранить
Выполните преобразование и сохраните файл TXT с заданными параметрами.
```csharp
converter.Convert(outputFile, options);
```
### Советы по устранению неполадок
- Убедитесь, что пути указаны правильно, чтобы избежать `FileNotFoundException`.
- Убедитесь, что версия GroupDocs.Conversion соответствует зависимостям вашего проекта.
