## Руководство по внедрению

### Загрузите исходный файл XLTX

#### Обзор
Эта функция позволяет загрузить файл XLTX, подготавливая почву для любой операции преобразования.

#### Пошаговая реализация

**1. Путь установки:**
Во-первых, укажите путь к месту расположения вашего документа:

```csharp
const string DOCUMENT_DIRECTORY = \@"YOUR_DOCUMENT_DIRECTORY";
```

**2. Определите путь к файлу:**
Объедините имя каталога и файла, чтобы получить полный путь:

```csharp
string sourceFilePath = Path.Combine(DOCUMENT_DIRECTORY, "sample.xltx");
```

**3. Инициализируйте конвертер:**
Загрузите файл XLTX с помощью GroupDocs.Conversion:

```csharp
using (var converter = new GroupDocs.Conversion.Converter(sourceFilePath))
{
    // Здесь будут выполняться операции по конвертации.
}
```

#### Объяснение
- **Путь.Объединить:** Обеспечивает кроссплатформенную совместимость.
- **Инициализация преобразователя:** Использует `using` оператор для автоматической очистки ресурсов.

### Советы по устранению неполадок
- Убедитесь, что путь к файлу правильный и доступный.
- Убедитесь, что версия .NET соответствует требованиям (4.6.1+).
