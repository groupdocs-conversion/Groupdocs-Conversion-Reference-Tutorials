## Руководство по внедрению

### Конвертировать PPSM в XLSX

Эта функция поможет вам преобразовать слайд-шоу PowerPoint (.ppsm) в электронную таблицу Excel Open XML (.xlsx).

#### Обзор процесса преобразования
Мы будем использовать библиотеку GroupDocs.Conversion для эффективной загрузки, преобразования и сохранения ваших файлов.

##### Шаг 1: Определите пути для входных и выходных каталогов
Укажите пути расположения вашего файла PPSM и пути вывода XLSX:

```csharp
string documentPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "your-file.ppsm");
string outputFolder = "YOUR_OUTPUT_DIRECTORY"; 
string outputFile = Path.Combine(outputFolder, "ppsm-converted-to.xlsx");
```

##### Шаг 2: Загрузите документ PPSM
Создайте экземпляр `Converter` класс для загрузки вашего документа:

```csharp
using (var converter = new GroupDocs.Conversion.Converter(documentPath))
{
    // Продолжайте настройку и выполнение преобразования.
}
```

##### Шаг 3: Инициализация параметров преобразования для формата XLSX
Настройте необходимые параметры для преобразования в электронную таблицу Excel:

```csharp
var options = new SpreadsheetConvertOptions();
```

##### Шаг 4: Преобразование и сохранение выходного файла
Выполните процесс конвертации и сохраните файл в нужном формате:

```csharp
converter.Convert(outputFile, options);
```
