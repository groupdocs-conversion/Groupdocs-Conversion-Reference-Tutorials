## Настройка GroupDocs.Conversion для .NET

### Установка

Установите библиотеку GroupDocs.Conversion одним из следующих способов:

**Консоль менеджера пакетов NuGet:**
```bash
Install-Package GroupDocs.Conversion -Version 25.3.0
```

**.NET CLI:**
```bash
dotnet add package GroupDocs.Conversion --version 25.3.0
```

### Приобретение лицензии

Чтобы в полной мере использовать GroupDocs.Conversion без ограничений, рассмотрите:
- **Бесплатная пробная версия**: Идеально подходит для первоначального тестирования.
- **Временная лицензия**: Подходит для расширенной оценки.
- **Покупка**: Предлагает полную поддержку и обновления для постоянного коммерческого использования.

### Базовая инициализация

После установки настройте процесс конвертации с помощью простой настройки:
```csharp
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

// Укажите путь к входному файлу STL
string inputFile = @"path\to\your\file.stl";
string outputFolder = @"YOUR_OUTPUT_DIRECTORY";

// Создайте экземпляр класса Converter и загрузите файл STL.
using (Converter converter = new Converter(inputFile))
{
    // Настройте параметры преобразования для формата Excel
    SpreadsheetConvertOptions convertOptions = new SpreadsheetConvertOptions();

    // Выполните преобразование и сохраните файл XLSX в выходном каталоге.
    converter.Convert(Path.Combine(outputFolder, "output.xlsx"), convertOptions);
}
```
