## Руководство по внедрению

### Конвертация XLSM в XLS

#### Обзор

Эта функция позволяет преобразовать файл XLSM в более простой формат XLS, что делает его совместимым с системами, не поддерживающими макросы.

##### Шаг 1: Загрузите исходный файл

Сначала убедитесь, что ваш файл XLSM доступен в каталоге вашего проекта. Затем загрузите его с помощью GroupDocs.Conversion:

```csharp
using (var converter = new Converter("sample.xlsm"))
{
    // Здесь будет добавлена логика преобразования.
}
```

##### Шаг 2: Определите параметры преобразования

Укажите параметры конвертации для преобразования файла XLSM в формат XLS.

```csharp
var options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Xls };
```

Здесь, `SpreadsheetConvertOptions` позволяет указать целевой формат и `SpreadsheetFileType.Xls` устанавливает старый двоичный формат Excel (XLS).

##### Шаг 3: Выполнение преобразования

Наконец, выполните процесс конвертации и сохраните файл.

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "xlsm-converted-to.xls");

converter.Convert(outputFile, options);
```

Этот код сохраняет преобразованный файл в указанном вами каталоге. Убедитесь, что `outputFolder` настроен правильно, чтобы избежать ошибок пути.

### Советы по устранению неполадок

- **Распространенная проблема:** Ошибка «Файл не найден».
