## Руководство по внедрению

### Функция: загрузка и преобразование файла SXC в XLSX

Эта функция позволяет загрузить существующий файл SXC и преобразовать его в более широко используемый формат XLSX.

#### Шаг 1: Определите выходной каталог и путь к файлу

Начните с настройки выходного каталога:

```csharp
string outputFolder = Path.Combine(Directory.GetCurrentDirectory(), "ConvertedFiles");
string outputFile = Path.Combine(outputFolder, "sxc-converted-to.xlsx");

// Убедитесь, что выходной каталог существует
Directory.CreateDirectory(outputFolder);
```

#### Шаг 2: Загрузите и преобразуйте файл SXC

Затем загрузите файл SXC и инициализируйте параметры преобразования:

```csharp
using (var converter = new GroupDocs.Conversion.Converter("YOUR_DOCUMENT_DIRECTORY/sample.sxc"))
{
    var options = new SpreadsheetConvertOptions();
    // Выполнить преобразование
    converter.Convert(outputFile, options);
}
```

**Объяснение:**
- `GroupDocs.Conversion.Converter`: Загружает файл SXC из указанного вами пути.
- `SpreadsheetConvertOptions`: Устанавливает целевой формат преобразования на XLSX.

### Советы по устранению неполадок
