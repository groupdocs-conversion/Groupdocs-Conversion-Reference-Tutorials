## Руководство по внедрению

### Обзор преобразования XLSM в CSV

Эта функция позволяет преобразовывать электронную таблицу Excel с поддержкой макросов в формат CSV, который можно легко обрабатывать или переносить между системами.

#### Шаг 1: Определите пути к документам

Настройте входные и выходные каталоги:

```csharp
string inputDocumentPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.xlsm");
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputDirectory, "xlsm-converted-to.csv");
```

#### Шаг 2: Загрузите исходный файл XLSM

Используйте GroupDocs.Conversion для загрузки файла XLSM:

```csharp
using (var converter = new Converter(inputDocumentPath))
{
    // Логика преобразования будет реализована здесь.
}
```

#### Шаг 3: Укажите параметры преобразования CSV

Настройте параметры преобразования для формата CSV:

```csharp
var options = new SpreadsheetConvertOptions
{
    Format = SpreadsheetFileType.Csv // Установить целевой формат на CSV
};
```

#### Шаг 4: Выполнение преобразования

Выполните преобразование и сохраните выходной файл:

```csharp
converter.Convert(outputFile, options);
```
