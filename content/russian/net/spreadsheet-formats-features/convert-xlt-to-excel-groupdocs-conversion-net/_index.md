## Руководство по внедрению
Давайте рассмотрим процесс преобразования XLT в XLS с помощью GroupDocs.Conversion для .NET.

### Функция: конвертация XLT в XLS
Эта функция позволяет преобразовать электронную таблицу XLT в более распространенный формат XLS.

#### Шаг 1: Определите входные и выходные каталоги
Настройте пути к файлам:
```csharp
string sourceFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "your-source-file.xlt");
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "xlt-converted-to.xls");
```
Этот фрагмент кода задает путь как для входного XLT-файла, так и для выходного XLS-файла.

#### Шаг 2: Загрузите файл XLT
Используйте GroupDocs.Conversion для загрузки файла:
```csharp
using (var converter = new Converter(sourceFilePath))
{
    // Логика преобразования будет располагаться здесь.
}
```
The `Converter` объект управляет загрузкой и подготовкой документа к конвертации.

#### Шаг 3: Настройте параметры конвертации
Определите параметры преобразования:
```csharp
SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Xls };
```
Эта конфигурация указывает, что вы хотите преобразовать файл в формат XLS.

#### Шаг 4: Выполнение преобразования
Выполните преобразование и сохраните результат:
```csharp
csv.Converter.Convert(outputFile, options);
```
The `Convert` метод обрабатывает файл в соответствии с заданными настройками и сохраняет его в указанном месте вывода.
