## Руководство по внедрению

Чтобы преобразовать файл PPSM в формат CSV, выполните следующие действия.

### Загрузка исходного файла

Загрузите файл слайд-шоу PowerPoint (.ppsm) с помощью `Converter` сорт:
```csharp
string dataDirectory = "YOUR_DOCUMENT_DIRECTORY";
using (var converter = new Converter(Path.Combine(dataDirectory, "sample.ppsm")))
{
    // Файл PPSM теперь готов к конвертации.
}
```

### Указание параметров преобразования

Определите параметры конвертации для преобразования вашей презентации в формат CSV:
```csharp
var options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Csv };
// Эти параметры указывают, что наши выходные данные будут в формате CSV.
```

### Определение выходного пути и выполнение преобразования

Укажите выходной путь для преобразованного файла и выполните процесс преобразования:
```csharp
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputDirectory, "ppsm-converted-to.csv");

converter.Convert(outputFile, options);
// Это запишет преобразованный CSV-файл в указанное место.
```

### Советы по устранению неполадок

- Убедитесь, что путь к входному файлу PPSM правильный и доступный.
- Убедитесь, что у вас есть права на запись в выходной каталог.
