## Настройка GroupDocs.Conversion для .NET

Чтобы начать использовать GroupDocs.Conversion в вашем приложении .NET, установите библиотеку с помощью консоли диспетчера пакетов NuGet или .NET CLI:

**Консоль диспетчера пакетов NuGet**
```bash
Install-Package GroupDocs.Conversion -Version 25.3.0
```

**.NET CLI**
```bash
dotnet add package GroupDocs.Conversion --version 25.3.0
```

## Руководство по внедрению

В этом разделе представлено подробное пошаговое руководство по конвертации файлов PPSM в формат XLS с помощью GroupDocs.Conversion.

### Процесс преобразования файла

#### Обзор
Преобразование файлов PPSM, слайд-шоу PowerPoint в форматах с поддержкой макросов, в электронные таблицы XLS полезно для анализа и обработки данных. Вот как можно реализовать это преобразование:

#### Шаг 1: Определите пути к документам
Укажите пути для исходного файла PPSM и выходного каталога.

```csharp
string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY";

string sourceFilePath = Path.Combine(documentDirectory, "sample.ppsm");
string outputFile = Path.Combine(outputDirectory, "ppsm-converted-to.xls");
```

#### Шаг 2: Загрузите файл PPSM
Используйте `Converter` класс для загрузки вашего файла PPSM.

```csharp
using (var converter = new Converter(sourceFilePath))
{
    // Ваш код конвертации будет здесь.
}
```

#### Шаг 3: Задайте параметры конвертации
Определите и настройте параметры преобразования для формата XLS.

```csharp
SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Формат = SpreadsheetFileType.Xls };
```

- **Format**: Указывает целевой тип файла, в данном случае `Xls`.

#### Шаг 4: Выполнение преобразования
Выполните процесс преобразования и сохраните выходной файл.

```csharp
csv
converter.Convert(outputFile, options);
```

**Советы по устранению неполадок:**
- Убедитесь, что все пути указаны правильно, чтобы избежать ошибок «файл не найден».
- Проверьте, совпадает ли версия вашей библиотеки с установленной.
