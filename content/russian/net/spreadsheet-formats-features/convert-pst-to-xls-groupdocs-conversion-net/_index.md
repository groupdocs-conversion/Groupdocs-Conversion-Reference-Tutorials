```csharp
    // Далее последуют дальнейшие шаги по настройке.
```

##### Шаг 2: Настройте параметры конвертации
Определите параметры преобразования, чтобы указать выходной формат XLS:


##### Шаг 3: Выполнение преобразования
Выполните преобразование и сохраните результат:

```csharp
string outputFileTemplate = Path.Combine("YOUR_OUTPUT_DIRECTORY", "pst-converted-{0}-to.xls");
converter.Convert(
    (SaveContext saveContext) => new FileStream(string.Format(outputFileTemplate, counter++), FileMode.Create),
    options

```
