### Базовая инициализация и настройка

Вот как инициализировать ваш проект с помощью GroupDocs.Conversion в C#:

```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

namespace AiToXlsxConverter
{
    class Program
    {
        static void Main(string[] args)
        {
            string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY";
            string outputDirectory = @"YOUR_OUTPUT_DIRECTORY/";

            // Инициализируйте конвертер с путем к файлу AI
            using (var converter = new Converter(documentDirectory + "\sample.ai"))
            {
                var options = new SpreadsheetConvertOptions();
                string outputFile = System.IO.Path.Combine(outputDirectory, "ai-converted-to.xlsx");
                
                // Конвертируйте и сохраните файл XLSX
                converter.Convert(outputFile, options);
            }
        }
    }
}
```

В этом фрагменте демонстрируется загрузка файла AI и преобразование его в формат XLSX с использованием `SpreadsheetConvertOptions`.
