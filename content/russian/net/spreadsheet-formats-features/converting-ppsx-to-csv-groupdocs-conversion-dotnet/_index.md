### Функция: загрузка и конвертация файла PPSX
В этом разделе демонстрируется загрузка файла PPSX и преобразование его в CSV с помощью GroupDocs.Conversion.
#### Шаг 1: Инициализируйте конвертер с помощью файла PPSX
```csharp
using System.IO;
using GroupDocs.Conversion;

const string DocumentDirectory = @"YOUR_DOCUMENT_DIRECTORY";
const string OutputDirectory = @"YOUR_OUTPUT_DIRECTORY/";

string outputFolder = Path.Combine(OutputDirectory);
string outputFile = Path.Combine(outputFolder, "ppsx-converted-to.csv");

// Инициализируйте конвертер с помощью файла PPSX
using (var converter = new Converter(Path.Combine(DocumentDirectory, "sample.ppsx")))
{
    // Дополнительные шаги будут здесь.
}
```
**Объяснение:** Начнем с настройки наших документов и выходных каталогов. Затем инициализируем `Converter` класс, используя путь к нашему файлу PPSX.
#### Шаг 2: Установите параметры преобразования в формат CSV
```csharp
using GroupDocs.Conversion.Options.Convert;

// Определите параметры преобразования для формата CSV
class SpreadsheetConvertOptions : ConvertOptions
{
    public SpreadsheetFileType Format { get; set; }
}

var options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Csv };
```
**Объяснение:** Здесь мы указываем, что наши выходные данные должны быть в формате CSV.
#### Шаг 3: Выполните преобразование и сохраните результат
```csharp
// Выполните преобразование и сохраните выходной файл.
class Program
{
    static void Main()
    {
        // Предположим, что преобразователь инициализирован, как показано ранее.
        converter.Convert(outputFile, options);
    }
}
```
**Объяснение:** Наконец, мы выполняем фактическое преобразование и сохраняем результат в указанном месте. `Convert` Метод управляет этим процессом.
