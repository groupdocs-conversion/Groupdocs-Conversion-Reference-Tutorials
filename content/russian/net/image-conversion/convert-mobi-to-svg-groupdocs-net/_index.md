### Функция: конвертация MOBI в SVG

#### Обзор

Эта функция позволяет вам преобразовать файл MOBI в формат SVG с помощью GroupDocs.Conversion. Это полезно для рендеринга содержимого электронной книги в масштабируемой векторной графике, пригодной для использования на различных платформах и устройствах.

#### Шаг 1: Загрузите исходный файл MOBI

Сначала укажите путь к входному файлу MOBI:

```csharp
string inputPath = "YOUR_DOCUMENT_DIRECTORY\\sample.mobi";
```

#### Шаг 2: Настройка выходного пути и конфигурации

Определите, где вы хотите сохранить преобразованный SVG-файл:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY\";
string outputFile = System.IO.Path.Combine(outputFolder, "mobi-converted-to.svg");
```

#### Шаг 3: Настройте параметры конвертации

Настройте параметры преобразования, адаптированные для формата SVG:

```csharp
using GroupDocs.Conversion.Options.Convert;

// Установить параметры преобразования для формата SVG
PageDescriptionLanguageConvertOptions options = new PageDescriptionLanguageConvertOptions { Format = PageDescriptionLanguageFileType.Svg };
```

#### Шаг 4: Выполнение преобразования

Наконец, выполните процесс конвертации и сохраните выходной файл:

```csharp
using (var converter = new Converter(inputPath))
{
    // Сохраните преобразованный файл SVG.
    converter.Convert(outputFile, options);
}
```

**Параметры и возвращаемые значения:**
- `inputPath`: Путь к исходному файлу MOBI.
- `outputFile`: Путь назначения для преобразованного файла SVG.
- Процесс преобразования обеспечивает внутреннее распределение и управление ресурсами.

**Советы по устранению неполадок:**
- Убедитесь, что путь к входному файлу указан правильно.
- Проверьте наличие достаточных прав доступа к выходному каталогу.
- Убедитесь, что версия пакета GroupDocs.Conversion соответствует вашей среде .NET.
