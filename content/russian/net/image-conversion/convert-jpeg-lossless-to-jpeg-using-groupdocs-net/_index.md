## Руководство по внедрению

### Обзор функций: преобразование файлов JLS в формат JPG
Эта функция использует возможности GroupDocs.Conversion для .NET для преобразования файлов JPEG Lossless в формат JPEG, делая их более универсально совместимыми и удобными для хранения.

#### Шаг 1: Определите пути к каталогам
Настройте пути к каталогам для входных (JLS) и выходных (JPG) файлов:
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY/";
```
Эти заполнители помогут вам организовать поиск исходных файлов и хранение преобразованных изображений.

#### Шаг 2: Определите путь к исходному файлу
Создайте путь для вашего файла JLS:
```csharp
string sourceFilePath = System.IO.Path.Combine(documentDirectory, "sample.jls");
```
На этом шаге указывается точное местоположение файла, который необходимо преобразовать.

#### Шаг 3: Задайте шаблон именования выходного файла
Определите, как вы хотите называть ваши выходные файлы JPG:
```csharp
string outputFileTemplate = System.IO.Path.Combine(outputDirectory, "converted-page-{0}.jpg");
```
The `{0}` заполнитель позволяет выполнять динамическую нумерацию страниц во время конвертации.

#### Шаг 4: Создайте FileStream для каждой страницы
Создайте файловый поток для обработки каждой преобразованной страницы:
```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new System.IO.FileStream(string.Format(outputFileTemplate, savePageContext.Page), System.IO.FileMode.Create);
```
Это гарантирует, что каждая выходная страница будет сохранена как отдельный файл JPG.

#### Шаг 5: Загрузите и преобразуйте файл JLS
Используйте GroupDocs.Conversion для загрузки исходного файла и его преобразования:
```csharp
using (Converter converter = new Converter(sourceFilePath))
{
    ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Jpg };
    converter.Convert(getPageStream, options);
}
```
В этом фрагменте демонстрируется загрузка файла JLS, настройка параметров преобразования для формата JPG и выполнение процесса преобразования.
