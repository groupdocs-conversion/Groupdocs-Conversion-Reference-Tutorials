## Руководство по внедрению преобразования

### Конвертация JPEG в JPG: практическое руководство

#### Обзор
Преобразование изображений из одного формата в другой является распространенным требованием в программных приложениях. В этом разделе основное внимание уделяется преобразованию файлов JPEG в общепризнанный формат JPG с помощью GroupDocs.Conversion.

#### Пошаговый процесс
**1. Определите выходной каталог и шаблон файла**
   Укажите, где будут храниться преобразованные файлы, а также правила их именования.
   ```csharp
   string outputFolder = "YOUR_OUTPUT_DIRECTORY";
   string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");
   ```

**2. Настройка функции Page Stream**
   Эта функция генерирует файловый поток для каждой преобразованной страницы.
   ```csharp
   Func<SavePageContext, Stream> getPageStream = savePageContext => 
       new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
   ```

**3. Загрузите и преобразуйте исходный файл JPEG.**
   Инициализируйте `Converter` объект с путем к исходному файлу и укажите параметры преобразования.
   ```csharp
   using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/sample.jpeg"))
   {
       ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Jpg };
       converter.Convert(getPageStream, options);
   }
   ```

**Параметры и цели метода**
- `Converter`: Управляет процессом конвертации.
- `ImageConvertOptions`: Указывает целевой формат (в данном случае JPG).
  
#### Советы по устранению неполадок
- Убедитесь, что пути к файлам верны и доступны.
- Убедитесь, что библиотека GroupDocs.Conversion установлена правильно.
