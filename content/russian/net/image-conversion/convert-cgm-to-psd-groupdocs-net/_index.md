## Руководство по внедрению
В этом разделе мы рассмотрим практические шаги, необходимые для преобразования файла CGM в формат PSD с помощью GroupDocs.Conversion. Мы разберем процесс для ясности.

### Загрузить исходный файл
**Обзор**: Эта функция демонстрирует, как загрузить исходный файл CGM в процесс конвертации.

#### Шаг 1: Определите путь и инициализируйте конвертер
```csharp
using System;
using GroupDocs.Conversion;

public class LoadSourceFileFeature
{
    public void Run()
    {
        // Определите путь для входного CGM-файла
        string cgmFilePath = "YOUR_DOCUMENT_DIRECTORY/sample.cgm";

        // Инициализируйте объект Converter с указанием пути к исходному файлу.
        using (Converter converter = new Converter(cgmFilePath))
        {
            // Теперь преобразователь готов к выполнению операций преобразования.
        }
    }
}
```
- **Почему**: Инициализация `Converter` класс с вашим CGM-файлом подготавливает его к последующим этапам преобразования.

### Установить параметры конвертации
**Обзор**: Настройте необходимые параметры, чтобы указать вывод в формате PSD.

#### Шаг 2: Укажите формат вывода
```csharp
using GroupDocs.Conversion.Options.Convert;

public class SetConversionOptionsFeature
{
    public void Run()
    {
        // Создайте экземпляр ImageConvertOptions
        ImageConvertOptions options = new ImageConvertOptions();

        // Укажите выходной формат PSD.
        options.Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd;
    }
}
```
- **Почему**: Настройка `ImageConvertOptions` гарантирует преобразование вашего файла в нужный формат.

### Конвертировать файл
**Обзор**: Выполнить процесс преобразования, сохранив выходные файлы в указанном месте.

#### Шаг 3: Выполнение преобразования и сохранение вывода
```csharp
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

public class ConvertFileFeature
{
    public void Run()
    {
        // Определить выходной каталог и шаблон для выходных файлов
        string outputFolder = "YOUR_OUTPUT_DIRECTORY";
        string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");

        // Создать функцию для генерации выходных файловых потоков на основе контекста страницы.
        Func<SavePageContext, Stream> getPageStream = savePageContext => 
            new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);

        // Загрузите исходный файл CGM (предполагается, что он уже определен в LoadSourceFileFeature)
        string cgmFilePath = "YOUR_DOCUMENT_DIRECTORY/sample.cgm";
        using (Converter converter = new Converter(cgmFilePath))
        {
            // Создать параметры преобразования для формата PSD
            ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd };

            // Выполнить преобразование в формат PSD с указанной функцией выходного потока
            converter.Convert(getPageStream, options);
        }
    }
}
```
- **Почему**: Этот шаг связывает все воедино, выполняя преобразование файлов и сохраняя каждую страницу как отдельный PSD-файл.
