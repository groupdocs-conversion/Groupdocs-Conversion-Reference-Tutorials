## Руководство по внедрению
### Преобразование EMLX в формат SVG
В этом разделе вы узнаете, как преобразовать файл сообщения Apple Mail (.emlx) в масштабируемую векторную графику (SVG).

#### Определить пути для входных и выходных файлов
Сначала настройте входные и выходные каталоги:

```csharp
string inputDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// Определите пути
string inputFile = Path.Combine(inputDirectory, "sample.emlx");
string outputFile = Path.Combine(outputDirectory, "emlx-converted-to.svg");
```

#### Загрузка и конвертация с помощью GroupDocs.Conversion
Загрузите файл EMLX и укажите параметры преобразования для SVG:

```csharp
using (var converterInstance = new Converter(inputFile))
{
    // Укажите выходной формат SVG.
    var convertOptions = new PageDescriptionLanguageConvertOptions
    {
        Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Svg
    };
    
    // Конвертируйте и сохраните файл
    converterInstance.Convert(outputFile, convertOptions);
}
```

**Объясняемые параметры:**
- **входной файл:** Путь к исходному файлу EMLX.
- **выходной файл:** Путь назначения для выходных данных SVG.
- **convertOptions.Формат:** Указывает, что целью преобразования является SVG.
