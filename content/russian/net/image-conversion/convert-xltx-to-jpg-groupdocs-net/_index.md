## Руководство по внедрению

### Загрузите и конвертируйте файл XLTX в JPG
Эта функция преобразует файл XLTX в серию изображений JPG, что идеально подходит для визуального обмена данными электронных таблиц.

**Шаг 1: Укажите путь к выходному каталогу**
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
// Определите, где будут сохранены преобразованные файлы.
```

**Шаг 2: Определите шаблон для имен выходных файлов**
```csharp
string outputFileTemplate = System.IO.Path.Combine(outputFolder, "converted-page-{0}.jpg");
// Этот шаблон помогает присвоить каждой странице документа уникальный номер.
```

**Шаг 3: Создайте поток для каждой страницы результата конверсии**
```csharp
Func<SavePageContext, System.IO.Stream> getPageStream = savePageContext => new System.IO.FileStream(System.String.Format(outputFileTemplate, savePageContext.Page), System.IO.FileMode.Create);
// Эта функция гарантирует сохранение каждой страницы как отдельного файла.
```

**Шаг 4: Загрузите исходный файл XLTX и задайте параметры преобразования.**
```csharp
using (Converter converter = new Converter(filePath))
{
    // Определить параметры преобразования для формата JPG
    ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Jpg };

    // Выполнить конвертацию из XLTX в JPG
    converter.Convert(getPageStream, options);
}
```
