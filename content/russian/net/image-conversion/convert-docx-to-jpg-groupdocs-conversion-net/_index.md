## Руководство по внедрению

Мы рассмотрим каждый шаг, необходимый для преобразования файла DOCX в формат JPG с помощью GroupDocs.Conversion.

### Загрузить исходный файл DOCX

**Обзор:**
Загрузка исходного файла DOCX — это первый шаг в любом процессе конвертации. Это гарантирует, что документ готов к преобразованию в различные форматы.

**Инициализация объекта преобразователя:**

```csharp
// Обязательно включите необходимые пространства имен.
using System;
using GroupDocs.Conversion;

string documentPath = "YOUR_DOCUMENT_DIRECTORY\sample.docx";
Converter converter = new Converter(documentPath);
```
- **Параметры:** `documentPath` — это путь к файлу DOCX.
- **Цель:** Это создает экземпляр `Converter`, что облегчает процесс конвертации.

### Конвертировать DOCX в формат JPG

**Обзор:**
Теперь, когда наш документ загружен, давайте настроим параметры конвертации и выполним преобразование в формат JPG.

**Определите выходные параметры:**

```csharp
using System.IO;
using GroupDocs.Conversion.Options.Convert;

string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");

Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```
- **Цель:** Это определяет, где и как каждая страница документа будет сохранена в виде файла JPG.
  
**Установить параметры конвертации:**

```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Jpg };
```
- **Параметры:** `Format` указывает, что мы конвертируем в JPG.

**Выполнить преобразование:**

```csharp
converter.Convert(getPageStream, options);
```
- **Цель:** Выполняет процесс конвертации с использованием заданных настроек и выводит каждую страницу как отдельный файл JPG.
