## Руководство по внедрению

### Загрузите и преобразуйте файл VSSM в формат PSD

Эта функция позволяет загружать файл Microsoft Visio с поддержкой макросов (.vssm) и преобразовывать его в формат документа Adobe Photoshop (.psd).

#### Шаг 1: Загрузите исходный файл VSSM
Загрузите ваш файл .vssm с помощью GroupDocs.Conversion `Converter` сорт.

```csharp
using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_VSSM"))
{
    // Дальнейшие шаги преобразования будут следовать здесь
}
```

#### Шаг 2: Задайте параметры преобразования для формата PSD
Определите формат изображения, в который вы хотите преобразовать свой файл, используя `ImageConvertOptions`.

```csharp
var options = new ImageConvertOptions { Format = ImageFileType.Psd };
```

**Объяснение:** The `Format` свойство указывает, что вывод будет в формате PSD.

#### Шаг 3: Настройка выходного потока
Создайте функцию, которая определяет, как каждая страница сохраняется в потоке. Это позволяет вам эффективно управлять именованием и хранением файлов.

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = System.IO.Path.Combine(outputFolder, "converted-page-{0}.psd");

Func<SavePageContext, Stream> getPageStream = savePageContext =>
{
    return new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
};
```

**Объяснение:** Эта лямбда-функция форматирует имя выходного файла и создает новый файловый поток для каждой страницы.

#### Шаг 4: Выполнение преобразования
Наконец, выполните процесс преобразования с помощью `Convert` метод.

```csharp
converter.Convert(getPageStream, options);
```

**Объяснение:** The `Convert` Метод использует предоставленные параметры и обработчик потока для выполнения преобразования файла.
