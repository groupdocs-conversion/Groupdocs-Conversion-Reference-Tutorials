## Настройка GroupDocs.Conversion для .NET

### Информация об установке
Для начала установите библиотеку GroupDocs.Conversion. Это можно сделать с помощью консоли NuGet Package Manager или .NET CLI:

**Консоль диспетчера пакетов NuGet**
```plaintext
Install-Package GroupDocs.Conversion -Version 25.3.0
```

**.NET CLI**
```bash
dotnet add package GroupDocs.Conversion --version 25.3.0
```

### Базовая инициализация и настройка
Чтобы инициализировать GroupDocs.Conversion для .NET, выполните следующие действия:

```csharp
using GroupDocs.Conversion;
// Другие необходимые пространства имен

public void InitializeConversion()
{
    // Предполагая, что лицензия установлена, как показано выше.

    string inputFile = "path/to/your/sample.svgz";
    string outputFile = "path/to/output/svgz-converted-to.doc";

    using (var converter = new Converter(inputFile))
    {
        var options = new WordProcessingConvertOptions { Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc };
        converter.Convert(outputFile, options);
    }
}
```

## Руководство по внедрению
### Конвертировать SVGZ в DOC
Давайте разберем процесс конвертации:

#### Загрузить исходный файл
Начните с загрузки файла SVGZ:
```csharp
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY/sample.svgz"))
{
    // Перейти к вариантам конвертации
}
```

#### Установить параметры конвертации
Укажите, что вы хотите преобразовать в формат DOC:
```csharp
WordProcessingConvertOptions options = new WordProcessingConvertOptions { Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc };
```

#### Выполнить преобразование
Выполните преобразование и сохраните выходной файл:
```csharp
converter.Convert("YOUR_OUTPUT_DIRECTORY/svgz-converted-to.doc", options);
```
**Советы по устранению неполадок:**
- Убедитесь, что входной путь SVGZ указан правильно.
- Убедитесь, что ваше приложение имеет права записи в выходной каталог.
