## Руководство по внедрению
### Конвертировать OTT в JPG
Чтобы преобразовать OTT-файл в изображение JPG, выполните следующие действия:

#### Шаг 1: Загрузите исходный файл
Начните с загрузки вашего OTT-документа с помощью `Converter` класс. Это подготавливает его к конвертации.

```csharp
class Program
{
    static void Main(string[] args)
    {
        var converter = new Converter("path/to/your/document.ott");
    }
}
```

#### Шаг 2: Укажите параметры конвертации
Определите параметры преобразования, используя `ImageConvertOptions` для установки таких параметров, как разрешение и качество.

```csharp
class Program
{
    static void Main(string[] args)
    {
        var converter = new Converter("path/to/your/document.ott");
        var options = new ImageConvertOptions
        {
            Format = GroupDocs.Conversion.FileTypes.ImageFileType.Jpg,
            Width = 1920, // Дополнительно: при необходимости отрегулируйте ширину.
            Height = 1080 // Дополнительно: отрегулируйте высоту по мере необходимости.
        };
    }
}
```

#### Шаг 3: Выполнение преобразования
Выполните конвертацию и сохраните файл JPG:

```csharp
class Program
{
    static void Main(string[] args)
    {
        var converter = new Converter("path/to/your/document.ott");
        var options = new ImageConvertOptions
        {
            Format = GroupDocs.Conversion.FileTypes.ImageFileType.Jpg,
            Width = 1920, // Дополнительно: при необходимости отрегулируйте ширину.
            Height = 1080 // Дополнительно: отрегулируйте высоту по мере необходимости.
        };
        converter.Convert("output/path/document.jpg", options);
    }
}
```
Этот фрагмент преобразует OTT-файл в JPG и сохраняет его.
