## Руководство по внедрению
Мы разобьем процесс внедрения на выполнимые этапы, каждый из которых объяснит важную часть процесса.

### Загрузка и конвертация PPTX в PNG
#### Обзор
Основная задача — загрузить файл PowerPoint и преобразовать его в формат PNG. Это преобразование может быть полезно для веб-публикации или встраивания слайдов в документацию.

#### Шаг 1: Определите выходной каталог и шаблон
Начните с указания места, где будут находиться преобразованные файлы:
```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", "ConvertedPPTX");
if (!Directory.Exists(outputFolder))
{
    Directory.CreateDirectory(outputFolder);
}
```
Создайте шаблон для именования выходных PNG-файлов:
```csharp
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.png");
```

#### Шаг 2: Создание функции FileStream
Нам необходимо обрабатывать каждую преобразованную страницу как поток файлов:
```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext =>
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```
Эта функция динамически создает файловый поток для каждой страницы во время конвертации.

#### Шаг 3: Загрузите и преобразуйте файл
Теперь загрузите файл PPTX и укажите желаемый формат изображения:
```csharp
string pptxFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.pptx");

using (Converter converter = new Converter(pptxFilePath))
{
    ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Png };
    
    // Выполнить преобразование
    converter.Convert(getPageStream, options);
}
```
Здесь, `ImageConvertOptions` указывает, что мы хотим преобразовать в PNG. `converter.Convert()` Метод выполняет фактическое преобразование, используя нашу функцию потока файлов.
