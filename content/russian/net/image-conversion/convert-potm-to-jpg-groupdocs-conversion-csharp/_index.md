- Настройте свою среду с необходимыми зависимостями.

## Руководство по внедрению

Теперь, когда все готово, давайте конвертируем файлы POTM в JPG.

### Загрузка и преобразование файла POTM

#### Обзор

Цель здесь — загрузить файл POTM с помощью GroupDocs.Conversion и преобразовать его в изображение JPG. Этот процесс может создавать предварительные просмотры или делиться шаблонами PowerPoint с поддержкой макросов в доступном формате.

#### Пошаговая реализация

##### 1. Определите путь к выходному каталогу

Укажите путь, по которому вы хотите сохранить преобразованные файлы:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");
```

Гарантировать `outputFolder` заменяется на фактический путь к вашему каталогу.

##### 2. Создайте функцию для получения потока страниц

Эта функция сохраняет каждую страницу POTM как отдельное изображение JPG:

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

##### 3. Загрузка и конвертация с помощью GroupDocs.Conversion

Используйте `Converter` класс для загрузки вашего файла POTM и его преобразования:

```csharp
using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_POTM"))
{
    // Установить параметры конвертации для формата JPG
    ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Jpg };

    // Выполнить преобразование
    converter.Convert(getPageStream, options);
}
```

Заменять `"YOUR_DOCUMENT_DIRECTORY/SAMPLE_POTM"` с фактическим путем к файлу.

**Советы по устранению неполадок:**
- Убедитесь, что все пути верны и доступны.
- Убедитесь, что у вас есть права на запись в выходной каталог.
