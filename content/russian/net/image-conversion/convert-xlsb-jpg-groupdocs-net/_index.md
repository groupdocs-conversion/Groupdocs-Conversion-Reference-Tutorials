## Руководство по внедрению

Давайте разберем реализацию на ключевые функции и объясним их пошагово.

### Определить выходной каталог и шаблон файла

Сначала настройте выходной каталог, в котором будут сохраняться файлы JPG:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");
```

Такая настройка помогает эффективно организовывать сгенерированные изображения.

#### Создать функцию для генерации потоков для сохранения страниц в формате JPG

Чтобы сохранить каждую страницу файла XLSB как отдельное изображение JPG:

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

### Загрузите исходный файл XLSB и задайте параметры преобразования

Загрузите файл XLSB и укажите, что вы хотите преобразовать его в формат JPG:

```csharp
using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/sample.xlsb"))
{
    ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Jpg };

    // Выполнить преобразование
    converter.Convert(getPageStream, options);
}
```

#### Объяснение параметров

- `SavePageContext`: Предоставляет контекст для каждой сохраняемой страницы.
- `ImageConvertOptions`: Настраивает выходной формат и другие параметры преобразования.
