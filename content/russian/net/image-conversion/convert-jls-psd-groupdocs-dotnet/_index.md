## Руководство по внедрению

Давайте разберем каждую функцию, задействованную в конвертации формата JLS в PSD.

### Загрузить исходный файл

Этот шаг демонстрирует, как можно загрузить исходный файл JLS с помощью GroupDocs.Conversion. Он закладывает основу для любого процесса преобразования, гарантируя готовность файла к преобразованию.

**1. Укажите путь ввода**
```csharp
string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.jls");
```

**2. Инициализация объекта-конвертера**

The `Converter` класс имеет решающее значение, поскольку он инкапсулирует все функции, необходимые для преобразования файлов:

```csharp
using GroupDocs.Conversion;

Converter converter = new Converter(inputFilePath);
// Не забудьте утилизировать объект-конвертер после завершения работы.
```

### Установить параметры конвертации

Здесь мы определяем, как ваш файл JLS будет преобразован в формат PSD. Настройка параметров преобразования имеет решающее значение для указания типа целевого файла и других параметров.

**1. Создайте ImageConvertOptions**

Эта конфигурация указывает, что выходные данные должны быть в формате PSD:

```csharp
using GroupDocs.Conversion.Options.Convert;

ImageConvertOptions options = new ImageConvertOptions 
{
    Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd
};
```

### Определить функцию выходного потока

Чтобы управлять тем, где и как сохраняются ваши преобразованные файлы, определите функцию выходного потока. Это гарантирует, что каждая страница файла JLS будет правильно обработана и сохранена как PSD.

**1. Укажите выходной каталог**
```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
```

**2. Определить функцию обработки потока**

Эта функция управляет созданием потока для каждой преобразованной страницы:

```csharp
using System.IO;

Func<SavePageContext, Stream> getPageStream = savePageContext => 
{
    string outputPath = string.Format(Path.Combine(outputFolder, "converted-page-{0}.psd"), savePageContext.Page);
    return new FileStream(outputPath, FileMode.Create);
};
```

### Выполнить преобразование

Наконец, выполните процесс преобразования, используя инициализированный `Converter` объект, настроенные параметры и функция выходного потока.

```csharp
converter.Convert(getPageStream, options);
// Убедитесь, что все открытые потоки или ресурсы закрыты надлежащим образом после использования.
```
