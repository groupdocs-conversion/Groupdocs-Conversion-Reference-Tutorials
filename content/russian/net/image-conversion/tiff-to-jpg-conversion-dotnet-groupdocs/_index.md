## Руководство по внедрению
Давайте разобьем процесс конвертации на управляемые этапы.

### Настройка путей и шаблона вывода
Определите путь к входному файлу TIFF и выходной каталог:
```csharp
string sourceTiffPath = @"YOUR_DOCUMENT_DIRECTORY\sample.tiff";
string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
```
Создайте шаблон для именования каждого выходного файла JPG на основе номера страницы:
```csharp
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");
```

### Подготовка выходных потоков
Определите метод генерации потока для каждого файла JPG, используя номер страницы:
```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

### Выполнение преобразования
Загрузите файл TIFF и настройте параметры конвертации в целевой формат JPG:
```csharp
using (Converter converter = new Converter(sourceTiffPath))
{
    ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Jpg };
    
    // Конвертировать каждую страницу TIFF в отдельный файл JPG
    converter.Convert(getPageStream, options);
}
```
**Объяснение**: `Converter` Класс загружает ваше изображение TIFF. `ImageConvertOptions` указывает, что вы хотите преобразовать изображения в формат JPG. `converter.Convert()` Метод управляет реальным процессом преобразования, создавая отдельные файлы JPG для каждой страницы документа TIFF.
