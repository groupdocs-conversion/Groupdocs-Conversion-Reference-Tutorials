### Функция: преобразование GIF в SVG

#### Обзор

Конвертация GIF в формат SVG приносит пользу веб-приложениям, делая графику масштабируемой и уменьшая размер файла. Этот раздел проведет вас через каждый шаг этого процесса:

**Шаг 1: Определите выходной каталог и настройте параметры преобразования**

Убедитесь, что выходной каталог существует, затем настройте параметры преобразования:

```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", "output");
Directory.CreateDirectory(outputFolder); // Убедитесь, что выходной каталог существует

// Укажите путь к входному GIF-файлу
string inputFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.gif");

// Определить путь к выходному файлу SVG
string outputFile = Path.Combine(outputFolder, "gif-converted-to.svg");
```

**Шаг 2: Инициализация конвертера и выполнение преобразования**

Инициализируйте объект-конвертер с помощью исходного GIF-файла:

```csharp
using (var converter = new Converter(inputFile))
{
    // Определить параметры преобразования для формата SVG
    var options = new PageDescriptionLanguageConvertOptions { Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Svg };

    // Конвертируйте и сохраните выходной SVG-файл.
    converter.Convert(outputFile, options);
}
```

**Пояснение параметров:**
- `inputFile`: Путь к исходному GIF-файлу.
- `outputFolder`: Каталог, в котором будет сохранен преобразованный SVG.
- `options.Format`: Указывает преобразование в формат SVG.

### Советы по устранению неполадок
