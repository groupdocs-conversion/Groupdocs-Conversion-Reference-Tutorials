## Руководство по внедрению
### Функция: Преобразование VSTM в PSD
Эта функция предназначена для преобразования шаблона чертежа Visio с поддержкой макросов в документ Adobe Photoshop.
#### Шаг 1: Определите выходной каталог и шаблон файла
Настройте выходной каталог для сохранения преобразованных файлов. Укажите шаблон именования для каждого файла страницы:
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");
```
#### Шаг 2: Создайте потоки для каждой страницы
Определите функцию для создания потока для каждой преобразованной страницы. Это гарантирует, что каждый PSD-файл будет сгенерирован правильно:
```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```
#### Шаг 3: Загрузите исходный файл VSTM и задайте параметры преобразования.
Используйте `Converter` класс для загрузки вашего .vstm файла. Укажите параметры преобразования для формата PSD:
```csharp
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY\\sample.vstm"))
{
    var options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd };
    
    // Конвертируйте VSTM в PSD.
    converter.Convert(getPageStream, options);
}
```
**Объяснение:**
- `ImageConvertOptions` указывает, что выходной формат должен быть PSD.
- The `converter.Convert()` Метод обрабатывает процесс преобразования, используя определенную функцию потока.
