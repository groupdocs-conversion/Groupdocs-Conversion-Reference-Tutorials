## Руководство по внедрению

### Загрузите и конвертируйте XLSX в JPG
Эта функция демонстрирует, как можно загрузить файл XLSX и преобразовать каждый лист в отдельное изображение JPG.

#### Определить выходной каталог и шаблон файла
Настройте путь к выходному каталогу и шаблон для именования преобразованных изображений:
```csharp
using System.IO;
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string inputFile = "YOUR_DOCUMENT_DIRECTORY/SAMPLE_XLSX.xlsx"; // Замените на путь к вашему файлу XLSX.

// Определите шаблон именования выходного файла\string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");
```

#### Создать потоковую функцию для выходных файлов
Определите функцию для управления созданием выходных потоков для каждой преобразованной страницы:
```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext =>
{
    string filePath = string.Format(outputFileTemplate, savePageContext.Page);
    return new FileStream(filePath, FileMode.Create);
};
```

#### Загрузите и преобразуйте файл XLSX
Используйте `Converter` класс для загрузки вашего файла и преобразования его в формат JPG:
```csharp
using (Converter converter = new Converter(inputFile))
{
    // Укажите параметры конвертации для формата JPG
    ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Jpg };

    // Выполнить преобразование
    converter.Convert(getPageStream, options);
}
```

Такая настройка гарантирует, что каждый лист в файле XLSX будет сохранен как уникальное изображение JPG на основе номера страницы.

### Настройка выходного каталога и шаблона файла
Правильная настройка выходного каталога и шаблона именования имеет решающее значение для эффективной организации преобразованных файлов. Этот раздел основывается на том, что мы уже рассмотрели.

#### Настройка структуры каталога
Перед запуском преобразования убедитесь, что выходной каталог существует:
```csharp
if (!Directory.Exists(outputFolder))
{
    Directory.CreateDirectory(outputFolder);
}
```

#### Конфигурация шаблона
Шаблон файла включает заполнитель `{0}` который заменяется на каждый номер страницы во время преобразования. Убедитесь, что эта настройка соответствует вашим организационным потребностям.
