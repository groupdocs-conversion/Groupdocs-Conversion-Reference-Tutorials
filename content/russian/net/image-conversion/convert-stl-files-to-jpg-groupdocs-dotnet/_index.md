### Конвертация STL в JPG
#### Обзор
Преобразование файлов STL (используемых для 3D-моделирования) в изображения JPG полезно для обмена проектами в формате, который легко просматривать без специального программного обеспечения.

##### Шаг 1: Загрузите исходный STL-файл
Убедитесь, что у вас готов файл STL, и укажите путь к нему:

```csharp
using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY\\sample.stl")) // Замените фактическим путем к вашему STL-файлу
{
    Console.WriteLine("STL file loaded.");
}
```
##### Шаг 2: Задайте параметры конвертации
Настройте параметры преобразования для формата JPG:

```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Jpg };
Console.WriteLine("Conversion options set.");
```
Этот фрагмент устанавливает выходной формат JPG.

##### Шаг 3: Выполнение преобразования
Выполните конвертацию и сохраните файлы JPG:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY"; // Определите свой выходной каталог
string outputFileTemplate = System.IO.Path.Combine(outputFolder, "converted-page-{0}.jpg");

Func<SavePageContext, Stream> getPageStream = savePageContext => new FileStream(System.IO.Path.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);

converter.Convert(getPageStream, options);
Console.WriteLine("Conversion completed and files are saved.");
```

Эффективное управление путями к файлам гарантирует, что ваше приложение сможет беспрепятственно находить файлы и управлять ими.

Настройте входные и выходные каталоги:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY"; // Путь к входному каталогу
```
