## Настройка GroupDocs.Conversion для .NET

Для начала установите библиотеку GroupDocs.Conversion с помощью консоли диспетчера пакетов NuGet или .NET CLI.

**Консоль менеджера пакетов NuGet:**
```shell
Install-Package GroupDocs.Conversion -Version 25.3.0
```

**\.NET CLI:**
```bash
dotnet add package GroupDocs.Conversion --version 25.3.0
```

### Базовая инициализация с помощью C#

После установки инициализируйте GroupDocs.Conversion в вашем проекте .NET:
```csharp
using System;
using GroupDocs.Conversion;

class Program
{
    static void Main()
    {
        // Инициализировать объект Converter с указанием пути к входному файлу
        using (Converter converter = new Converter("path/to/your/input.epub"))
        {
            Console.WriteLine("GroupDocs.Conversion initialized successfully.");
        }
    }
}
```
