## Руководство по внедрению

### Конвертировать PLT в JPG
Эта функция позволяет конвертировать файлы PLT в формат JPG, что упрощает обмен проектами и их просмотр без использования специализированного программного обеспечения.

#### Загрузите исходный файл PLT
Сначала укажите пути к каталогам для ваших документов и вывода. Этот шаг включает загрузку исходного файла с помощью `Converter` сорт:
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY/";

using (Converter converter = new Converter(Path.Combine(documentDirectory, "yourfile.plt")))
{
    // Логика преобразования будет здесь
}
```

#### Установить параметры конвертации для формата JPG
Определите параметры преобразования, чтобы указать, что вы хотите получить вывод в формате JPG:
```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Jpg };
```

#### Определить функцию выходного потока
Создайте функцию для обработки выходного потока для каждой страницы вашего файла PLT. Это гарантирует, что каждая преобразованная страница будет сохранена как отдельный файл JPG:
```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(Path.Combine(outputDirectory, $"converted-page-{savePageContext.Page}.jpg"), FileMode.Create);
```

#### Выполнить преобразование
Наконец, выполните преобразование, вызвав `Convert` метод с заданными вами параметрами:
```csharp
converter.Convert(getPageStream, options);
```

### Советы по устранению неполадок
