## Руководство по внедрению

### Загрузите и конвертируйте файл PLT в PNG

**Обзор:**
Основная функция этого руководства — загрузка файла PLT и его конвертация в формат PNG. Этот процесс включает настройку параметров конвертации, специфичных для форматов изображений.

#### Шаг 1: Настройка выходного каталога и функции потока
Сначала укажите, где будут сохранены преобразованные файлы:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.png");

Func<SavePageContext, Stream> getPageStream = savePageContext =>
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

- **Объяснение**: `getPageStream` — это функция, которая возвращает поток для каждой преобразованной страницы. Она помогает сохранять выходные файлы PNG в указанном каталоге.

#### Шаг 2: Настройте параметры конвертации

Определите, как будет преобразован ваш PLT-файл:

```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Png };
```

- **Объяснение**: `options` указывает, что формат преобразования — PNG. Эта конфигурация гарантирует, что выходные файлы будут в желаемом формате изображения.

#### Шаг 3: Выполнение преобразования

Выполните преобразование с помощью экземпляра конвертера:

```csharp
using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_PLT"))
{
    converter.Convert(getPageStream, options);
}
```

- **Объяснение**: `Convert` Метод использует потоковую функцию и параметры преобразования для обработки и сохранения каждой страницы файла PLT в виде изображения PNG.

**Советы по устранению неполадок:**
- Убедитесь, что путь к выходному каталогу указан правильно.
- Убедитесь, что исходный PLT-файл существует по указанному пути.
