## Руководство по внедрению
### Загрузите и конвертируйте файл TSV в PNG
Давайте разберем шаги, необходимые для преобразования файла TSV в изображения PNG.
#### Обзор
В этом разделе демонстрируется загрузка файла TSV с помощью GroupDocs.Conversion и преобразование каждой его страницы в отдельные файлы PNG.
#### Шаг 1: Настройка выходного каталога
Сначала укажите выходной каталог, в котором будут сохранены преобразованные изображения:
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY"; // Установите желаемый путь
```
#### Шаг 2: Загрузите файл TSV
Используйте GroupDocs.Conversion для загрузки исходного файла TSV. Убедитесь, что вы указали правильный путь к файлу:
```csharp
string inputFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.tsv");
using (Converter converter = new Converter(inputFile))
{
    // Перейти к вариантам конвертации
}
```
#### Шаг 3: Настройте параметры конвертации
Определите, как каждая страница вашего TSV должна быть преобразована в файл PNG, указав параметры формата изображения:
```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Png };
```
#### Шаг 4: Конвертируйте и сохраните каждую страницу как PNG
Создайте функцию для управления выходным потоком для каждой страницы. Этот шаг включает генерацию уникального шаблона имени файла для каждой преобразованной страницы:
```csharp
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.png");

Func<SavePageContext, Stream> getPageStream = savePageContext =>
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);

converter.Convert(getPageStream, options);
```
**Объяснение:**
- `getPageStream`: Эта функция генерирует поток файлов для каждой преобразуемой страницы.
- The `outputFileTemplate` используется для создания уникальных имен файлов для изображений PNG.
