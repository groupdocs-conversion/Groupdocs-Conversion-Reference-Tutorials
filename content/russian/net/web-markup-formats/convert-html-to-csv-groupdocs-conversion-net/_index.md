## Руководство по внедрению

Когда все готово, давайте реализуем преобразование HTML в CSV.

### Загрузка и конвертация файла

1. **Настройка путей документов**
   Определите каталоги для исходных файлов и преобразованных выходных данных:
   ```csharp
   const string DocumentDirectory = "YOUR_DOCUMENT_DIRECTORY";
   const string OutputDirectory = "YOUR_OUTPUT_DIRECTORY";
   ```

2. **Загрузите исходный HTM-файл**
   Используйте `Converter` класс для загрузки вашего HTML-файла:
   ```csharp
   string inputFilePath = Path.Combine(DocumentDirectory, "sample.htm");

   using (var converter = new Converter(inputFilePath))
   {
       // Здесь будет находиться код преобразования.
   }
   ```

3. **Определить параметры преобразования**
   Задайте параметры преобразования для формата CSV:
   ```csharp
   SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Csv };
   ```

4. **Выполнить преобразование**
   Выполните преобразование и сохраните результат:
   ```csharp
   string outputFilePath = Path.Combine(OutputDirectory, "converted.csv");
   converter.Convert(outputFilePath, options);
   ```

### Советы по устранению неполадок
- Гарантировать `sample.htm` существует в вашем каталоге документов.
- Проверьте права доступа к файлам для входных и выходных каталогов, чтобы предотвратить проблемы с доступом.
