- **Библиотеки и зависимости**: Вам понадобится GroupDocs.Conversion версии 25.3.0.

- **Требования к знаниям**: Базовые знания C# и знакомство с концепциями обработки файлов.

## Настройка GroupDocs.Conversion для .NET

Для начала вам нужно установить библиотеку GroupDocs.Conversion. Вот два способа сделать это:

**Консоль диспетчера пакетов NuGet**
```shell
Install-Package GroupDocs.Conversion -Version 25.3.0
```

**\.NET CLI**
```bash
dotnet add package GroupDocs.Conversion --version 25.3.0
```
