## Руководство по внедрению

Давайте разобьем процесс конвертации на управляемые этапы.

### Шаг 1: Загрузка файла MPT

Сначала загрузите ваш MPT-файл с помощью `Converter` класс. Это подготавливает файл для последующих этапов преобразования.

```csharp
using (var converter = new Converter("path\to\your\file.mpt"))
{
    Console.WriteLine("MPT file loaded successfully.");
}
```

**Почему?**
Загрузка гарантирует, что все необходимые данные из файла MPT доступны и готовы к преобразованию в формат XLS.

### Шаг 2: Настройка параметров конвертации

Далее настройте параметры преобразования, чтобы указать целевой формат файла (XLS).

```csharp
var convertOptions = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Xls };

// Конвертировать загруженный документ в указанный формат.
converter.Convert("output\path\file.xls", convertOptions);
```

**Почему?**
Указав `SpreadsheetConvertOptions`вы определяете, как преобразование должно обрабатывать определенные атрибуты, такие как формат, обеспечивая совместимость и функциональность в Excel.

### Советы по устранению неполадок
- **Файл не найден**: Еще раз проверьте пути к файлам на предмет опечаток или неправильных разрешений.
- **Ошибки преобразования**: Убедитесь, что ваш файл MPT не поврежден. Попробуйте открыть его с помощью Microsoft Project, чтобы проверить целостность.
