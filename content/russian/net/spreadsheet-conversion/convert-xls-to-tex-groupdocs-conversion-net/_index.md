## Руководство по внедрению

### Функция: конвертация XLS в TEX

#### Обзор

Эта функция позволяет преобразовать файл Excel (.xls) в документ TeX, идеально подходящий для набора текста и математических презентаций.

#### Пошаговая реализация

##### 1. Определите пути к файлам
Настройте входные и выходные пути для ваших файлов:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
string xlsFilePath = Path.Combine(documentDirectory, "sample.xls");
string texOutputFile = Path.Combine(outputDirectory, "xls-converted-to.tex");
```

##### 2. Конвертировать файл
Загрузите ваш XLS-файл и конвертируйте его с помощью GroupDocs.Conversion:

```csharp
using (var converterInstance = new Converter(xlsFilePath))
{
    var convertOptions = new PageDescriptionLanguageConvertOptions { Format = FileType.Tex };
    converterInstance.Convert(texOutputFile, convertOptions);
}
```

- **Параметры:** 
  - `xlsFilePath`: Путь к исходному XLS-файлу.
  - `texOutputFile`: Где будет сохранен преобразованный TEX-файл.

##### Советы по устранению неполадок

- Убедитесь, что пути правильные и доступные.
- Проверьте необходимые разрешения на чтение/запись файлов.

### Функция: настройка пути к выходному каталогу

#### Обзор
