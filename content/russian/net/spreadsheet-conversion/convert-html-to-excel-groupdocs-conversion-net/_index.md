**.NET CLI**

## Руководство по внедрению

### Конвертировать HTML в XLS

Эта функция позволяет преобразовать HTML-файл в хорошо структурированную электронную таблицу Excel, идеально подходящую для анализа данных и составления отчетов.

#### Шаг 1: Загрузите исходный HTML-файл

Начните с загрузки исходного HTML-документа с помощью `Converter` class. Убедитесь, что путь к вашему HTML-файлу указан правильно:

```csharp
string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.html");
```

#### Шаг 2: Укажите параметры конвертации

Определите параметры преобразования для формата XLS, которые сообщат GroupDocs.Conversion, как вы хотите отформатировать свой выходной файл:

```csharp
using (var converter = new Converter(inputFilePath))
{
    SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Xls };
}
```

#### Шаг 3: Выполнение преобразования

Выполните процесс преобразования и сохраните полученный файл Excel в указанном вами каталоге:

```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "html-converted-to.xls");

converter.Convert(outputFile, options);
```
