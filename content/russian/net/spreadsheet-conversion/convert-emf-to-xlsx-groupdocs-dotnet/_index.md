### Конвертировать файл EMF в формат XLSX
#### Обзор
Эта функция позволяет легко преобразовывать графические данные из файла EMF в структурированный формат Excel, что упрощает обработку и анализ.
##### Шаг 1: Подготовка вариантов конвертации
Сначала настройте параметры конвертации, характерные для файлов XLSX:
```csharp
// Настройте параметры преобразования для формата XLSX
SpreadsheetConvertOptions options = new SpreadsheetConvertOptions();
```
#### Шаг 2: Выполнение преобразования
Далее выполните преобразование с помощью `Convert` метод:
```csharp
// Конвертировать файл EMF в XLSX
converter.Convert("output/path/file.xlsx", options);
```
**Объясняемые параметры:**
- **"выход/путь/файл.xlsx":** Целевой путь для вашего преобразованного файла.
- **параметры:** Указывает тип и свойства выходного документа.
