## Руководство по внедрению

### Функция: Загрузка исходного файла
Эта функция демонстрирует, как загрузить файл XPS для преобразования. Правильная загрузка документа имеет решающее значение перед любой обработкой.

#### Шаг 1: Определите путь к файлу
Укажите каталог и путь к файлу, где находится исходный XPS-файл:

```csharp
string dataDirectory = @"YOUR_DOCUMENT_DIRECTORY\";
string sourceFilePath = Path.Combine(dataDirectory, "SAMPLE_XPS.xps");
```

#### Шаг 2: Загрузите файл
Используйте GroupDocs.Conversion для загрузки документа XPS в память:

```csharp
var converter = new Converter(sourceFilePath);
composer.Dispose(); // Обеспечить освобождение ресурсов, когда они больше не нужны
```

### Функция: Преобразование XPS в Excel
Эта функция демонстрирует преобразование файла XPS в формат Excel (XLS).

#### Шаг 1: Подготовьте выходной каталог и путь к файлу
Убедитесь, что выходной каталог существует, или создайте его при необходимости:

```csharp
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY\\";
if (!Directory.Exists(outputDirectory))
{
    Directory.CreateDirectory(outputDirectory);
}

string outputFile = Path.Combine(outputDirectory, "xps-converted-to.xls");
```

#### Шаг 2: Задайте параметры конвертации
Настройте параметры преобразования для формата Excel (XLS):

```csharp
var options = new SpreadsheetConvertOptions
{
    Format = GroupDocs.Conversion.FileTypes.SpreadsheetFileType.Xls
};
```

#### Шаг 3: Выполнение преобразования
Выполните процесс преобразования из XPS в Excel (XLS) и сохраните выходной файл:

```csharp
using (var converterInstance = new Converter(sourceFilePath)) // Повторно использовать ранее загруженный исходный файл
{
    converterInstance.Convert(outputFile, options);
}
```

### Советы по устранению неполадок
- **Убедитесь, что пути верны**: Убедитесь, что все пути к каталогам указаны правильно.
- **Проверьте права доступа к файлу**: Убедитесь, что у вас есть необходимые разрешения на чтение и запись файлов в ваших каталогах.
