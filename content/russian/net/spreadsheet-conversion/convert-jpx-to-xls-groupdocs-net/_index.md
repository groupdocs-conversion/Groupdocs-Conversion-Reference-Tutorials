## Руководство по внедрению

### Функция: загрузка файла JPX
**Обзор**
Загрузка файла .jpx — это первый шаг в подготовке его к конвертации. Эта функция позволяет вам легко интегрировать данные изображения в рабочий процесс вашего приложения.

#### Шаг 1: Настройте каталог документов
Убедитесь, что путь к каталогу документов, в котором хранятся файлы JPX, указан правильно.
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
```

#### Шаг 2: Загрузите исходный файл
Создайте экземпляр `Converter` класс и откройте исходный файл JPX:
```csharp
using (var converter = new Converter(Path.Combine(documentDirectory, "sample.jpx")))
{
    // Файл теперь загружен и готов к конвертации.
}
```

### Функция: конвертация формата JPX в XLS
**Обзор**
Преобразование файла .jpx в формат .xls позволяет использовать данные изображений в приложениях для работы с электронными таблицами, таких как Microsoft Excel.

#### Шаг 1: Установка выходных путей
Определите пути для выходного каталога и имя преобразованного файла:
```csharp
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputDirectory, "jpx-converted-to.xls");
```

#### Шаг 2: Инициализация параметров преобразования
Укажите, что вы хотите преобразовать файл изображения в формат XLS, используя `SpreadsheetConvertOptions`:
```csharp
var options = new SpreadsheetConvertOptions { Format = GroupDocs.Conversion.FileTypes.SpreadsheetFileType.Xls };
```

#### Шаг 3: Выполнение преобразования
Выполните преобразование и сохраните выходной файл в желаемом месте:
```csharp
converter.Convert(outputFile, options);
```

### Советы по устранению неполадок
- **Распространенная проблема:** Файл не найден. Убедитесь, что путь к исходному файлу JPX указан правильно.
- **Решение:** Еще раз проверьте пути к каталогам и убедитесь, что файлы существуют в указанных местах.
