## Руководство по внедрению

### Функция: конвертация CGM в XLSX

Эта функция позволяет преобразовывать файлы CGM в широко используемый формат XLSX, что упрощает обработку и анализ графических данных.

#### Шаг 1: Определите пути к файлам

Сначала настройте входные и выходные каталоги. Заменить `YOUR_DOCUMENT_DIRECTORY` и `YOUR_OUTPUT_DIRECTORY` с соответствующими путями в вашей системе.

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// Пути для исходного файла CGM и целевого файла XLSX
string inputFile = Path.Combine(documentDirectory, "sample.cgm");
string outputFile = Path.Combine(outputDirectory, "cgm-converted-to.xlsx");
```

#### Шаг 2: Загрузка исходного файла

Загрузите исходный файл CGM с помощью GroupDocs.Conversion:

```csharp
using (var converter = new Converter(inputFile))
{
    // Логика преобразования будет здесь
}
```

#### Шаг 3: Укажите параметры конвертации

Определите параметры преобразования для формата XLSX:

```csharp
var options = new SpreadsheetConvertOptions();
```

#### Шаг 4: Выполнение преобразования

Выполните фактическое преобразование из CGM в XLSX:

```csharp
converter.Convert(outputFile, options);
```

Этот процесс сохранит преобразованный файл в указанном выходном каталоге.

### Советы по устранению неполадок

- **Проблемы с путем к файлу:** Убедитесь, что пути к входным и выходным файлам указаны правильно.
- **Совместимость библиотек:** Убедитесь, что у вас установлена правильная версия GroupDocs.Conversion.
