### Функция: конвертация DOCX в XLSX

#### Обзор
Конвертация документов Word в электронные таблицы Excel полезна при обработке текстовых данных в табличном формате. Выполните следующие шаги с помощью GroupDocs.Conversion:

##### Шаг 1: Определите пути к файлам и инициализируйте конвертер
Укажите пути для входного файла DOCX и выходного файла XLSX:
```csharp
string documentPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.docx");
string outputFile = Path.Combine("YOUR_OUTPUT_DIRECTORY", "docx-converted-to.xlsx");
```
Инициализируйте `Converter` объект с путем к исходному файлу DOCX:
```csharp
using (var converter = new GroupDocs.Conversion.Converter(documentPath))
{
    Console.WriteLine("Converter initialized successfully.");
}
```
##### Шаг 2: Создайте параметры преобразования
Настройте параметры преобразования для формата XLSX с помощью `SpreadsheetConvertOptions`:
```csharp
// Укажите параметры преобразования для целевого файла XLSX
```

##### Шаг 3: Выполнение преобразования
Выполните преобразование и сохраните результат в виде файла XLSX:
```csharp
// Конвертировать DOCX в XLSX и сохранить в указанном месте
converter.Convert(outputFile, options);

Console.WriteLine("Conversion completed successfully.");
```

- **Распространенная проблема**: Ошибки «Файл не найден». Убедитесь, что пути установлены правильно и доступны.
