## Руководство по внедрению
Теперь, когда ваша среда готова, давайте реализуем преобразование.
### Загрузка и конвертация DOCM в XLS
#### Обзор
Конвертируйте документы Word с макросами (.docm) в электронные таблицы Excel (.xls).
#### Пошаговая реализация
**1. Загрузите исходный файл DOCM**
Укажите путь к исходному документу:
```csharp
string documentPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.docm");
```
**2. Установите параметры конвертации**
Настройте параметры преобразования для формата Excel:
```csharp
using (var converter = new GroupDocs.Conversion.Converter(documentPath))
{
    SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = GroupDocs.Conversion.FileTypes.SpreadsheetFileType.Xls };
}
```
**3. Конвертировать и сохранить как XLS**
Выполните процесс конвертации:
```csharp
string outputFile = Path.Combine("YOUR_OUTPUT_DIRECTORY", "docm-converted-to.xls");
converter.Convert(outputFile, options);
```
#### Объяснение параметров
- **documentPath**: Полный путь к вашему файлу .docm.
- **Электронная таблицаКонвертироватьПараметры**: Определяет формат преобразования (в данном случае XLS).
