## Руководство по внедрению
### Конвертация PPS в XLSX
#### Обзор
Эта функция преобразует презентацию PowerPoint в файл Excel, что позволяет более эффективно обрабатывать и анализировать данные.

#### Шаг 1: Определите пути к файлам
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// Настройте пути к файлам для конвертации
string sourceFilePath = Path.Combine(documentDirectory, "sample.pps");
string outputFile = Path.Combine(outputDirectory, "converted-to.xlsx");
```
*Почему этот шаг?* Указание четких входных и выходных каталогов гарантирует, что ваши файлы будут организованы и легкодоступны.

#### Шаг 2: Загрузите исходный файл
```csharp
using (var converter = new Converter(sourceFilePath))
{
    // Инициализирует процесс конвертации с нашим исходным файлом PPS.
}
```

#### Шаг 3: Определите параметры преобразования
```csharp
var options = new SpreadsheetConvertOptions();
// Настройте параметры преобразования для формата XLSX.
```
*Почему этот шаг?* The `SpreadsheetConvertOptions` класс указывает, что в качестве выходных данных мы используем электронную таблицу Excel.

#### Шаг 4: Выполнение преобразования
```csharp
converter.Convert(outputFile, options);
// Конвертирует и сохраняет файл PPS в формат XLSX по указанному пути.
```
