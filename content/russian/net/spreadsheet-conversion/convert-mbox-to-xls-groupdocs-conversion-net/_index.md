## Руководство по внедрению
### Функция 1: Загрузка файла MBOX
**Обзор:**
Загрузка файла MBOX — первый шаг к конвертации его содержимого. В этом разделе показано, как загрузить файл MBOX с использованием условных параметров загрузки.
#### Шаг 1: Настройка параметров условной загрузки
```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Load;

string inputPath = "YOUR_DOCUMENT_DIRECTORY/sample.mbox";
var loadOptions = new MboxLoadOptions();
```
**Объяснение:**  
`MboxLoadOptions()` настраивает процесс загрузки, гарантируя обработку только файлов MBOX.
#### Шаг 2: Создание экземпляра конвертера
```csharp
var converter = new Converter(inputPath, (loadContext) => 
    loadContext.SourceFormat == EmailFileType.Mbox ? loadOptions : null);
```
**Объяснение:**
Перед применением указанных параметров загрузки конвертер проверяет, является ли исходный формат MBOX.
#### Шаг 3: Избавление от ресурсов
```csharp
current.Dispose();
```
**Объяснение:**  
Утилизируйте ресурсы, чтобы освободить память после завершения задач преобразования.
### Функция 2: Преобразование MBOX в XLS
**Обзор:**
В этой функции описывается, как преобразовать файл MBOX в формат XLS, что упрощает обработку и анализ данных.
#### Шаг 1: Настройка выходного пути
```csharp
string outputPath = "YOUR_OUTPUT_DIRECTORY/mbox-converted-{0}-to.xls";
```
#### Шаг 2: Инициализация конвертера с параметрами загрузки
```csharp
using (var converter = new Converter(Constants.SAMPLE_MBOX, 
    (loadContext) => loadContext.SourceFormat == EmailFileType.Mbox ? loadOptions : null))
{
    var options = new SpreadsheetConvertOptions { Format = GroupDocs.Conversion.FileTypes.SpreadsheetFileType.Xls };
    int counter = 1;
    
    converter.Convert(
        (saveContext) => new FileStream(string.Format(outputPath, counter++), FileMode.Create),
        options
    );
}
```
**Объяснение:**  
- **Электронная таблицаКонвертироватьПараметры**: Указывает формат преобразования как XLS.
- **ФайлПоток**: Управляет созданием файла для каждого преобразованного документа.
