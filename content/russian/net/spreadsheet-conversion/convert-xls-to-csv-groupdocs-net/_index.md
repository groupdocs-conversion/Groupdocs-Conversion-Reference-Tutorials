## Руководство по внедрению

### Загрузка исходного XLS-файла

#### Обзор
Загрузка исходного файла Excel — это первый шаг в процессе конвертации. В этом разделе показано, как использовать GroupDocs.Conversion для загрузки файла XLS.

##### Шаг 1: Определите входной путь и загрузите файл
```csharp
using System;
using GroupDocs.Conversion;

string inputFilePath = "YOUR_DOCUMENT_DIRECTORY\sample.xls";

// Загрузите исходный XLS-файл
typing (var converter = new Converter(inputFilePath))
{
    // Теперь преобразователь готов к операциям преобразования.
}
```

Этот фрагмент загружает ваш файл Excel в `Converter` объект, делая его готовым к дальнейшим действиям.

### Настройка параметров преобразования для CSV

#### Обзор
Настройка правильных параметров гарантирует, что процесс преобразования выводит правильно отформатированный CSV-файл. Вот как настроить эти параметры с помощью GroupDocs.Conversion.

##### Шаг 2: Настройте параметры конвертации
```csharp
using GroupDocs.Conversion.Options.Convert;

// Создайте экземпляр SpreadsheetConvertOptions и укажите формат CSV.
var options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Csv };
```

The `SpreadsheetConvertOptions` класс позволяет настраивать различные параметры преобразования, такие как установка типа выходного файла.

### Выполнение преобразования из XLS в CSV

#### Обзор
В этом разделе рассматривается выполнение фактического процесса конвертации и сохранение полученного CSV-файла.

##### Шаг 3: Определите выходной путь и выполните преобразование
```csharp
using System.IO;
using GroupDocs.Conversion;

string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "xls-converted-to.csv");

// Выполнить конвертацию из XLS в CSV
typing (var converter = new Converter(inputFilePath))
{
    // Выполните преобразование и сохраните выходной файл.
    converter.Convert(outputFile, options);
}
```

Этот код выполняет преобразование и записывает полученный CSV-файл в указанный вами каталог.
