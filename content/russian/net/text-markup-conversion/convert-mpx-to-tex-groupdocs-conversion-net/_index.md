## Руководство по внедрению

Теперь, когда вы настроили свою среду и установили необходимые пакеты, давайте реализуем преобразование MPX в TEX.

### Настройка преобразования

#### Обзор

В этом разделе вы узнаете, как настроить исходный и выходной каталоги, загрузить файл MPX, настроить параметры преобразования для формата TeX и выполнить преобразование.

#### Пошаговая реализация

##### Определить каталоги и пути к файлам

Сначала укажите, где находятся исходные файлы MPX и куда следует сохранять преобразованные файлы TEX:
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// Определите путь к исходному файлу MPX и путь к целевому файлу TEX
string sourceMpxFilePath = Path.Combine(documentDirectory, "sample.mpx");
string outputTexFilePath = Path.Combine(outputDirectory, "mpx-converted-to.tex");
```

##### Загрузить и преобразовать

Загрузите файл MPX с помощью GroupDocs.Conversion и настройте параметры конвертации:
```csharp
using (var converter = new GroupDocs.Conversion.Converter(sourceMpxFilePath))
{
    // Настройте параметры преобразования для формата TEX
    var convertOptions = new PageDescriptionLanguageConvertOptions { Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Tex };
    
    // Выполнить преобразование из MPX в TEX
    converter.Convert(outputTexFilePath, convertOptions);
}
```

**Объяснение:**
- **Инициализация преобразователя**: `Converter` экземпляр класса создается с использованием пути к исходному файлу.
- **Варианты конвертации**: Мы указываем, что хотим преобразовать наш файл в формат TeX, используя `PageDescriptionLanguageConvertOptions`.
- **Выполнить преобразование**: Наконец, метод конвертации преобразует ваш файл MPX в файл TEX.

##### Поиск неисправностей
Если у вас возникли проблемы, проверьте распространенные ошибки, такие как неправильные пути к файлам или отсутствующие зависимости. Убедитесь, что GroupDocs.Conversion правильно установлен и лицензирован.
