- Как загрузить исходный PSD-файл
- Настройка параметров преобразования для вывода в формате TXT

## Руководство по внедрению

### Загрузить исходный файл

**Обзор:** Загрузка PSD-файла необходима для доступа к исходному документу и управления им.

#### Шаг 1: Укажите путь к исходному файлу
```csharp
string sourceFilePath = @"YOUR_DOCUMENT_DIRECTORY\sample.psd";
```
*Объяснение:* Заменять `YOUR_DOCUMENT_DIRECTORY` с указанием пути к вашему PSD-файлу, что гарантирует точное определение местоположения.

### Настроить параметры преобразования

**Обзор:** Настройка параметров преобразования имеет решающее значение для настройки выходного формата TXT.

#### Шаг 2: Настройте параметры конвертации
```csharp
using GroupDocs.Conversion.Options.Convert;

WordProcessingConvertOptions options = new WordProcessingConvertOptions { Format = FileTypes.WordProcessingFileType.Txt };
```
*Объяснение:* Это определяет, что выходной формат должен быть TXT. `WordProcessingConvertOptions` класс используется для текстовых преобразований.

### Выполнить преобразование и сохранить вывод

**Обзор:** Конвертируйте из PSD в TXT и сохраните в указанном каталоге.

#### Шаг 3: Определите выходной каталог
```csharp
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY";

if (!Directory.Exists(outputDirectory))
{
    Directory.CreateDirectory(outputDirectory);
}
```
*Объяснение:* Убедитесь, что выходной путь существует, или создайте его, чтобы избежать ошибок при сохранении файла.

#### Шаг 4: Выполнение преобразования и сохранение файла
```csharp
string outputFile = Path.Combine(outputDirectory, "psd-converted-to.txt");

using (var converter = new Converter(@"YOUR_DOCUMENT_DIRECTORY\sample.psd"))
{
    WordProcessingConvertOptions options = new WordProcessingConvertOptions { Format = FileTypes.WordProcessingFileType.Txt };

    // Выполнить преобразование и сохранить вывод
    converter.Convert(outputFile, options);
}
```
*Объяснение:* Инициализируйте `Converter` с вашим PSD-файлом, задайте параметры преобразования, выполните преобразование и сохраните его как «psd-converted-to.txt».
