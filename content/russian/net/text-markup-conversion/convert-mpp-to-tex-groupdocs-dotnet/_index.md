### Конвертировать MPP в TEX
Преобразование файла MPP в документ TEX может оптимизировать рабочий процесс, позволяя интегрировать данные проекта в документы LaTeX, обычно используемые для написания технических и академических текстов.
#### Шаг 1: Определите выходной каталог
Настройте выходной каталог, в котором будут сохранены преобразованные файлы:
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = System.IO.Path.Combine(outputFolder, "mpp-converted-to.tex");
```
#### Шаг 2: Инициализация конвертера и настройка параметров конвертации
Инициализируйте конвертер с исходным путем к файлу MPP. Настройте параметры конвертации, указав TEX в качестве целевого формата:
```csharp
using (var converter = new GroupDocs.Conversion.Converter("YOUR_DOCUMENT_DIRECTORY\sample.mpp"))
{
    // Укажите параметры преобразования для формата TEX
    var options = new PageDescriptionLanguageConvertOptions { Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Tex };
    
    // Выполните преобразование и сохраните выходной файл.
    converter.Convert(outputFile, options);
}
```
**Пояснение параметров:**
- **ВыходнойПуть:** Путь назначения для сохранения преобразованного TEX-файла.
- **Формат:** Определяет целевой формат документа (в данном случае TEX).
#### Советы по устранению неполадок
- Убедитесь, что пути к MPP и выходному каталогу указаны правильно.
- Проверьте разрешения на чтение и запись в эти каталоги.
