### Базовая инициализация и настройка с помощью C#

Вот как можно инициализировать преобразование GroupDocs в вашем приложении .NET:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main()
    {
        // Инициализировать преобразователь
        string inputFilePath = "path/to/your/file.jpf";
        using (var converter = new Converter(inputFilePath))
        {
            var options = new TexConvertOptions();
            
            // Определить выходной путь
            string outputFilePath = Path.ChangeExtension(inputFilePath, ".tex");
            
            // Выполнить преобразование
            converter.Convert(outputFilePath, options);
        }
    }
}
```

В этом фрагменте:
- Мы инициализируем `Converter` объект с входным файлом JPF.
- Мы указываем TEX в качестве целевого формата, используя `TexConvertOptions`.
- Наконец, мы выполняем преобразование и сохраняем его по указанному выходному пути.
