### Базовая инициализация

Начните с настройки базовой среды для загрузки и преобразования ваших документов. Вот начальная настройка на C#:

```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main(string[] args)
    {
        string DocumentPath = "YOUR_DOCUMENT_DIRECTORY/sample.epub";
        string OutputDirectory = "YOUR_OUTPUT_DIRECTORY";

        using (var converter = new Converter(DocumentPath))
        {
            var options = new PageDescriptionLanguageConvertOptions { Format = PageDescriptionLanguageFileType.Tex };
            string outputFile = System.IO.Path.Combine(OutputDirectory, "epub-converted-to.tex");
            converter.Convert(outputFile, options);
        }
    }
}
```

Создайте экземпляр `Converter` класс для загрузки вашего документа EPUB:


**3. Установите параметры конвертации**

Укажите, что вы хотите преобразовать файл в формат TEX, настроив параметры преобразования:


**4. Выполнить преобразование**

Выполните фактическое преобразование и сохраните выходной файл в формате TEX:
