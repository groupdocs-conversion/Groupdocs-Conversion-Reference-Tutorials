### Загрузка и конвертация файла MOBI в формат TEX
Эта функция предназначена для загрузки исходного файла MOBI и преобразования его в формат TEX.

#### Шаг 1: Определите входные и выходные пути
Начните с настройки пути к входному файлу и выходному каталогу:

```csharp
string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY";

if (!Directory.Exists(outputDirectory))
{
    Directory.CreateDirectory(outputDirectory);
}

string inputFilePath = Path.Combine(documentDirectory, "sample.mobi");
string outputFile = Path.Combine(outputDirectory, "mobi-converted-to.tex");
```

#### Шаг 2: Загрузите и преобразуйте файл
Загрузите ваш MOBI-файл и конвертируйте его в формат TEX:

```csharp
using (var converter = new Converter(inputFilePath))
{
    // Определить параметры преобразования для формата TEX
    PageDescriptionLanguageConvertOptions options = new PageDescriptionLanguageConvertOptions { Format = PageDescriptionLanguageFileType.Tex };

    // Преобразовать документ
    converter.Convert(outputFile, options);

    Console.WriteLine("Conversion completed.");
}
```
**Объяснение**: 
- **Конвертер(inputFilePath)**: Инициализирует процесс конвертации с вашим файлом MOBI.
- **PageDescriptionLanguageConvertOptions**: Указывает, что мы хотим преобразовать в формат TEX. Настройте дополнительные параметры по мере необходимости.
- **converter.Convert(выходнойФайл, параметры)**: Выполняет преобразование и сохраняет вывод.

#### Шаг 3: Советы по устранению неполадок
- Убедитесь, что входной MOBI-файл существует в указанном вами каталоге.
- Проверьте права доступа к выходному каталогу, чтобы убедиться в возможности записи файлов.
- Проверьте версию пакета GroupDocs.Conversion на предмет проблем совместимости.
