## Руководство по внедрению
### Преобразование файла EMLX в формат TXT
**Обзор:** Эта функция преобразует файлы сообщений Apple Mail (.emlx) в обычный текст (.txt), что упрощает программную обработку электронных писем.

#### Шаг 1: Настройте конвертер
Сначала инициализируйте `Converter` класс, указав ваш входной файл .emlx:

```csharp
string inputFile = "YOUR_INPUT_FILE_PATH.emlx";
using (Converter converter = new Converter(inputFile))
{
    // Параметры конфигурации будут добавлены сюда
}
```
**Почему?** The `Converter` объект необходим, поскольку он обрабатывает загрузку и преобразование файлов.

#### Шаг 2: Задайте параметры конвертации
Определите параметры преобразования, чтобы указать выходной формат:

```csharp
TxtConvertOptions options = new TxtConvertOptions();
```
На этом этапе настраивается, как файл .emlx будет преобразован в файл TXT, что позволяет выполнить дальнейшую настройку при необходимости.

#### Шаг 3: Выполнение преобразования
Выполните преобразование и сохраните результат:

```csharp
converter.Convert(() => File.Create(Path.Combine(outputPath, "output.txt")), options);
```
Эта строка преобразует ваш файл EMLX в формат TXT с использованием указанных параметров и сохраняет его в указанном вами выходном каталоге. Это важно, поскольку завершает процесс преобразования.

#### Советы по устранению неполадок
- **Проблемы с путем к файлу:** Убедитесь, что все пути настроены правильно.
- **Ошибки разрешения:** Проверьте, имеет ли ваше приложение разрешения на чтение/запись для задействованных каталогов.
