## Руководство по внедрению

### Функция: конвертация PPTX в TEX

**Обзор**

Эта функция позволяет плавно конвертировать презентации PowerPoint в документы LaTeX с помощью библиотеки GroupDocs.Conversion. Вот шаги:

#### Шаг 1: Определите пути к файлам
```csharp
string inputFilePath = @"YOUR_DOCUMENT_DIRECTORY/sample.pptx"; // Путь к вашему файлу PPTX.
string outputFolder = @"YOUR_OUTPUT_DIRECTORY"; // Выходной каталог для файлов TEX.
string outputFile = System.IO.Path.Combine(outputFolder, "pptx-converted-to.tex"); // Полный путь к выходному TEX-файлу.
```

**Объяснение**: Укажите пути с помощью заполнителей. Замените их реальными каталогами в вашей среде.

#### Шаг 2: Загрузите и преобразуйте файл PPTX
```csharp
using (var converter = new Converter(inputFilePath))
{
    var options = new PageDescriptionLanguageConvertOptions { Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Tex };
    
    // Конвертируйте и сохраните TEX-файл
    converter.Convert(outputFile, options);
}
```

**Объяснение**: 
- **Класс преобразователя**: Инициализируется с вашим файлом PPTX.
- **PageDescriptionLanguageConvertOptions**: Настраивает параметры преобразования специально для формата TEX.
- **Метод конвертера.Конвертировать**: Выполняет процесс преобразования.

#### Советы по устранению неполадок

- Убедитесь, что все пути правильно определены и доступны в вашей среде.
- Убедитесь, что GroupDocs.Conversion правильно установлен через NuGet.
- Проверьте права доступа к файлам, чтобы избежать ошибок доступа при чтении/записи файлов.
