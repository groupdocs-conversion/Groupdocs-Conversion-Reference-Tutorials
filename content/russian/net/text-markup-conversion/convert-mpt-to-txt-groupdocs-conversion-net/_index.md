## Руководство по внедрению

### Загрузка и конвертация файла MPT в TXT

Выполните следующие действия, чтобы загрузить файл MPT и преобразовать его в формат TXT.

#### Шаг 1: Определите пути к каталогам

Начните с определения путей для входного документа и выходного каталога:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// Определите путь к исходному MPT-файлу и выходному TXT-файлу.
string mptFilePath = Path.Combine(documentDirectory, "sample.mpt");
string txtOutputFile = Path.Combine(outputDirectory, "mpt-converted-to.txt");
```

#### Шаг 2: Загрузите файл MPT

Используйте `Converter` класс из GroupDocs.Conversion для загрузки вашего файла MPT:

```csharp
using (var converter = new Converter(mptFilePath))
{
    // Конвертация будет выполнена здесь.
}
```

#### Шаг 3: Задайте параметры конвертации

Укажите параметры конвертации для преобразования формата MPT в формат TXT:

```csharp
WordProcessingConvertOptions options = new WordProcessingConvertOptions 
{ 
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt 
};
```

**Объяснение**: `WordProcessingConvertOptions` позволяет вам определить целевой тип файла. Здесь мы устанавливаем его на `Txt`.

#### Шаг 4: Выполнение преобразования

Выполните преобразование и сохраните результат:

```csharp
converter.Convert(txtOutputFile, options);
```
