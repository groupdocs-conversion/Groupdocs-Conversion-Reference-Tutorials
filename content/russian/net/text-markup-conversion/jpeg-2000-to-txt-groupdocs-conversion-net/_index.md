Определите путь к выходному каталогу:

Загрузите исходный файл, используя класс Converter, предоставленный GroupDocs:
```csharp
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_J2C"))

    // Логика преобразования будет здесь
```

#### Шаг 3: Настройте параметры конвертации
Установите параметры для целевого формата TXT, указав необходимые выходные параметры:
```csharp
var options = new WordProcessingConvertOptions { Format = FileTypes.WordProcessingFileType.Txt };
```

#### Шаг 4: Выполнение преобразования
Выполните преобразование и сохраните результат по указанному вами пути:

### Советы по устранению неполадок

- Проверьте права на запись в выходной каталог.
