## Руководство по внедрению

### Конвертировать файл FODS в формат TEX
Эта функция позволяет вам преобразовать OpenDocument Flat XML Spreadsheet в исходный документ LaTeX. Вот как:

#### Шаг 1: Настройте параметры конвертации
Сначала укажите параметры преобразования для формата TEX:
```csharp
using GroupDocs.Conversion.Options.Convert;

PageDescriptionLanguageConvertOptions options = new PageDescriptionLanguageConvertOptions { Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Tex };
```
**Объяснение**: Мы используем `PageDescriptionLanguageConvertOptions` указать TEX в качестве типа выходного файла для точного преобразования.

#### Шаг 2: Выполнение преобразования
Выполните преобразование и сохраните результат:
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "fods-converted-to.tex");

using (var converter = new GroupDocs.Conversion.Converter(filePath))
{
    // Конвертируйте и сохраните TEX-файл
    converter.Convert(outputFile, options);
}
```
**Объяснение**: `Convert` Метод преобразует ваш документ FODS в файл .tex на основе указанных параметров.
