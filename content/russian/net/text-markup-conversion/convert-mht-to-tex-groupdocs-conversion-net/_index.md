Сначала настройте библиотеку GroupDocs.Conversion в своем проекте с помощью консоли диспетчера пакетов NuGet или .NET CLI:

**Консоль менеджера пакетов NuGet:**

### Базовая инициализация и настройка
Инициализируйте API GroupDocs.Conversion в вашем проекте C# следующим образом:
```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main()
    {
        string outputFolder = "YOUR_OUTPUT_DIRECTORY";
        string inputFile = "YOUR_DOCUMENT_DIRECTORY\sample.mht";

        // Инициализируйте объект Converter с помощью MHT-файла.
        using (var converter = new Converter(inputFile))
        {
            Console.WriteLine("Converter initialized successfully.");
        }
    }
}
```
Заменять `YOUR_OUTPUT_DIRECTORY` и `YOUR_DOCUMENT_DIRECTORY\sample.mht` с вашими реальными путями.

## Руководство по внедрению
Вот как конвертировать файлы MHT в формат TEX:

### Шаг 1: Загрузите исходный MHT-файл
Загрузите исходный MHT-файл для преобразования:
```csharp
using (var converter = new Converter(@"YOUR_DOCUMENT_DIRECTORY\sample.mht"))
{
    Console.WriteLine("Source MHT file loaded.");
}
```

### Шаг 2: Настройте параметры конвертации
Настройте параметры, соответствующие формату TEX:
```csharp
PageDescriptionLanguageConvertOptions options = new PageDescriptionLanguageConvertOptions
{
    Format = PageDescriptionLanguageFileType.Tex // Укажите TEX в качестве целевого формата.
};
```

### Шаг 3: Выполнение преобразования
Выполните преобразование и сохраните результат как файл .tex:
```csharp
string outputFile = Path.Combine(outputFolder, "mht-converted-to.tex");
converter.Convert(outputFile, options);
Console.WriteLine("Conversion completed successfully.");
```
На этом этапе ваш документ преобразуется в нужный формат.
