### Функция конвертации PPSX в TXT

Эта функция позволяет вам преобразовывать презентации PowerPoint напрямую в простые текстовые файлы. Вот как это работает:

#### Шаг 1: Инициализация конвертера
Начните с инициализации конвертера, указав путь к файлу PPSX.

```csharp
using (var converter = new GroupDocs.Conversion.Converter("YOUR_DOCUMENT_DIRECTORY\\\\sample.ppsx"))
{
    // Шаги преобразования приведены здесь.
}
```
**Почему?**: Инициализация гарантирует, что библиотека готова обрабатывать ваш конкретный формат файла.

#### Шаг 2: Задайте параметры конвертации
Укажите, что вы хотите преобразовать в формат TXT, используя следующие параметры:

```csharp
WordProcessingConvertOptions options = new WordProcessingConvertOptions
{
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt
};
```
**Почему?**: Настройка параметров преобразования сообщает библиотеке, как именно вы хотите преобразовать файл.

#### Шаг 3: Конвертировать и сохранить
Выполните преобразование и сохраните результат:

```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "ppsx-converted-to.txt");

converter.Convert(outputFile, options);
```
**Почему?**На этом последнем этапе ваш файл обрабатывается и выводится в желаемом формате.
