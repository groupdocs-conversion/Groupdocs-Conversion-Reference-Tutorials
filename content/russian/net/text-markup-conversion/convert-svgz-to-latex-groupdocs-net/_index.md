Чтобы использовать GroupDocs.Conversion для .NET, установите библиотеку через NuGet Package Manager Console или с помощью .NET CLI. Убедитесь, что ваша система имеет доступ к Интернету для загрузки пакетов.

**Консоль диспетчера пакетов NuGet**
```bash
Install-Package GroupDocs.Conversion -Version 25.3.0
```

**\.NET CLI**
```bash
dotnet add package GroupDocs.Conversion --version 25.3.0
```
