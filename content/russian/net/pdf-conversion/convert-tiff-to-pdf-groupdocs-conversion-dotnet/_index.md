### Конвертировать TIFF в PDF

Эта функция позволяет вам эффективно конвертировать изображение TIFF в документ PDF. Вот как это работает:

#### Шаг 1: Загрузите исходный файл TIFF

Сначала укажите путь к исходному файлу TIFF:
```csharp
string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.tiff");
```
Этот шаг гарантирует, что GroupDocs.Conversion знает, какой файл вы хотите преобразовать.

#### Шаг 2: Настройте параметры конвертации

Далее определите параметры преобразования для формата PDF. Здесь вы настраиваете, как TIFF будет преобразован в PDF:
```csharp
var options = new PdfConvertOptions();
```
Эти параметры позволяют вам настраивать выходной PDF-файл в соответствии с вашими потребностями.

#### Шаг 3: Выполнение преобразования

Наконец, выполните преобразование и сохраните PDF-файл в желаемом месте:
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "tiff-converted-to.pdf");

using (var converter = new Converter(inputFilePath))
{
    // Конвертируйте и сохраните файл TIFF как PDF
    converter.Convert(outputFile, options);
}
```
На этом этапе ваш TIFF-файл преобразуется в PDF-файл и сохраняется в указанном каталоге.
