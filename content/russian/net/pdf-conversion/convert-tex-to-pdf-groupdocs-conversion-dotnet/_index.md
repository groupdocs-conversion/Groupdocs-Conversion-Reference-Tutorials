### Базовая инициализация и настройка с помощью C#
Вот как инициализировать конвертер:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY";

string texFilePath = Path.Combine(documentDirectory, "sample.tex"); // Путь к вашему TEX-файлу
string pdfOutputFile = Path.Combine(outputDirectory, "tex-converted-to.pdf");

// Загрузите и конвертируйте файл TEX в PDF
using (var converter = new Converter(texFilePath))
{
    var options = new PdfConvertOptions();
    converter.Convert(pdfOutputFile, options);
}
```

## Руководство по внедрению

### Загрузка файла LaTeX
Чтобы начать конвертацию файлов TEX в PDF, вам необходимо загрузить их в конвертер.

#### Шаг 1: Определите пути и загрузите файлы
- **Настройка пути**: Определите пути для входных (TEX) и выходных (PDF) файлов.
- **Загрузка файлов**: Использовать `Converter` класс из GroupDocs.Conversion для обработки файловых операций.

### Конвертация TEX в PDF
Теперь давайте преобразуем загруженный TEX-файл в PDF-документ.

#### Шаг 2: Задайте параметры конвертации
- **PdfConvertOptions**: Настройте параметры специально для преобразования PDF, такие как номера страниц и настройки макета.

```csharp
var options = new PdfConvertOptions();
// При необходимости здесь можно применить дополнительную конфигурацию.
```

#### Шаг 3: Выполнение преобразования
- **Выполнение преобразования**: Вызов `converter.Convert` метод выполнения фактического преобразования файла.

### Советы по устранению неполадок
- **Ошибки пути к файлу**: Убедитесь, что пути указаны правильно и доступны.
- **Конфликты версий библиотеки**: Используйте совместимые версии GroupDocs.Conversion, как указано в этом руководстве.
