## Базовая инициализация и настройка
Вот как инициализировать конвертер с помощью базовой настройки:

```csharp
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

string sourceFilePath = "YOUR_DOCUMENT_DIRECTORY/SAMPLE_EML";
string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "converted.pdf");

// Параметры загрузки для преобразования электронной почты
Func<LoadContext, LoadOptions> getLoadOptions = loadContext => new EmailLoadOptions { ConvertOwned = false };

using (Converter converter = new Converter(sourceFilePath, getLoadOptions))
{
    PdfConvertOptions options = new PdfConvertOptions();
    converter.Convert(outputFile, options);
}
```

## Руководство по внедрению
В этом разделе мы рассмотрим каждый шаг преобразования файла электронной почты в PDF.

### Загрузить файл электронной почты с определенными параметрами
**Обзор:**
Настройка параметров загрузки позволяет вам контролировать, как процесс преобразования обрабатывает ваши файлы электронной почты. Здесь вы указываете предпочтения, например, следует ли преобразовывать принадлежащие вам свойства.

```csharp
Func<LoadContext, LoadOptions> getLoadOptions = loadContext => new EmailLoadOptions
{
    ConvertOwned = false // Не конвертировать имеющуюся недвижимость по умолчанию
};
```
**Объяснение:**
- `ConvertOwned`: Если установлено значение false, обеспечивается преобразование стандартных атрибутов электронной почты без преобразования каких-либо фирменных форматов.

### Инициализировать конвертер и задать параметры конвертации
**Обзор:**
Основная работа происходит здесь. Вы инициализируете `Converter` класс с указанием пути к исходному файлу и параметров загрузки.

```csharp
using (Converter converter = new Converter(sourceFilePath, getLoadOptions))
{
    PdfConvertOptions options = new PdfConvertOptions();
    converter.Convert(outputFile, options);
}
```
**Объяснение:**
- **Параметры**: `sourceFilePath` указывает файл электронной почты, который необходимо преобразовать, и `getLoadOptions` предоставляет настройки преобразования.
- **Возвращаемое значение**: Эта операция возвращает PDF-файл, расположенный по адресу `outputFile`.

### Основные параметры конфигурации
Настройка вашего `PdfConvertOptions` позволяет вам настроить вывод. Вы можете указать размер страницы, поля и многое другое в зависимости от ваших требований.
