## Настройка GroupDocs.Conversion для .NET

Чтобы начать конвертировать файлы OneNote в PDF, вам сначала нужно установить библиотеку GroupDocs.Conversion. Вот как это сделать:

### Использование консоли диспетчера пакетов NuGet
```bash
Install-Package GroupDocs.Conversion -Version 25.3.0
```

### Использование .NET CLI
```bash
dotnet add package GroupDocs.Conversion --version 25.3.0
```

#### Этапы получения лицензии
- **Бесплатная пробная версия**: Вы можете начать с бесплатной пробной версии, чтобы изучить функции.

#### Базовая инициализация и настройка

После установки инициализируйте GroupDocs.Conversion следующим образом:


## Руководство по внедрению

Давайте разобьем процесс на четкие шаги.

### Загрузка и конвертация файла .one в PDF

#### Обзор
В этом разделе рассказывается, как загрузить файл OneNote и преобразовать его в формат PDF с помощью GroupDocs.Conversion для .NET.

##### Шаг 1: Определите пути

Начните с определения путей к вашему источнику `.one` файл и целевой вывод PDF:

```csharp
using System;
using System.IO;

string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// Укажите путь к файлу OneNote и полученному PDF-файлу.
string oneFilePath = Path.Combine(documentDirectory, "sample.one");
string pdfOutputFile = Path.Combine(outputDirectory, "one-converted-to.pdf");
```

##### Шаг 2: Загрузите исходный файл ONE

Загрузите ваш `.one` файл с использованием GroupDocs.Conversion:

```csharp
using (var converter = new Converter(oneFilePath))
{
    // Перейдите к указанию параметров конвертации.
}
```

##### Шаг 3: Укажите параметры преобразования для формата PDF

Настройте параметры преобразования PDF:

```csharp
var pdfOptions = new PdfConvertOptions();
```

##### Шаг 4: Преобразуйте и сохраните файл .one как PDF-документ

Выполните преобразование и сохраните выходной файл:

```csharp
converter.Convert(pdfOutputFile, pdfOptions);
Console.WriteLine("Conversion completed successfully.");
```

#### Основные параметры конфигурации
- **PdfConvertOptions**: Настройте диапазоны страниц, поворот и другие параметры, чтобы адаптировать выводимые данные.

#### Советы по устранению неполадок
- Убедитесь, что пути настроены правильно.
- Убедитесь, что `.one` файл доступен и не поврежден.
