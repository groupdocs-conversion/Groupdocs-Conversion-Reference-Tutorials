Для начала установите библиотеку GroupDocs.Conversion в свой проект с помощью диспетчера пакетов NuGet или .NET CLI.

**Консоль диспетчера пакетов NuGet**

После установки инициализируйте GroupDocs.Conversion с помощью базовой настройки с использованием C#:
```csharp
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

// Инициализировать преобразователь
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

string rtfFilePath = Path.Combine(documentDirectory, "sample.rtf");
string pdfOutputPath = Path.Combine(outputDirectory, "rtf-converted-to.pdf");

using (var converter = new Converter(rtfFilePath))
{
    var options = new PdfConvertOptions();
    converter.Convert(pdfOutputPath, options);
}
```

## Руководство по внедрению

### Функция: конвертация RTF в PDF

Эта функция позволяет преобразовывать документы RTF в PDF-файлы с помощью GroupDocs.Conversion.

#### Шаг 1: Настройка каталогов
Определите пути для ваших документов и выходных каталогов. Это помогает эффективно организовать входные и выходные файлы.
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
```

#### Шаг 2: Определите пути к файлам
Укажите путь к исходному файлу RTF и путь к целевому файлу PDF.
```csharp
string rtfFilePath = Path.Combine(documentDirectory, "sample.rtf");
string pdfOutputPath = Path.Combine(outputDirectory, "rtf-converted-to.pdf");
```

#### Шаг 3: Загрузка и преобразование документа
Используйте GroupDocs.Conversion для загрузки документа RTF и преобразования его в формат PDF.
```csharp
using (var converter = new Converter(rtfFilePath))
{
    var options = new PdfConvertOptions();
    converter.Convert(pdfOutputPath, options);
}
```

**Основные параметры конфигурации:**
- **Параметры PDFConvert:** При необходимости настройте параметры преобразования, такие как диапазон страниц или корректировки макета.

### Советы по устранению неполадок

- Убедитесь, что путь к входному RTF-файлу указан правильно и файл существует.
- Проверьте наличие достаточных прав на чтение/запись файлов в указанных каталогах.
- Убедитесь, что версия библиотеки GroupDocs.Conversion соответствует зависимостям вашего проекта.
