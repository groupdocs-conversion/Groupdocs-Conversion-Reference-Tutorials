## Руководство по внедрению

### Функция: конвертация GIF в PDF
Эта функция преобразует изображение GIF в общедоступный документ PDF. Вот как это можно реализовать:

#### Шаг 1: Загрузите исходный файл
Сначала загрузите исходный GIF-файл. Вам понадобится путь к GIF-файлу и выходной каталог для PDF-файла.
```csharp
using System.IO;
using GroupDocs.Conversion;

namespace ConversionFeatures {
    public static class GifToPdfConverter {
        public static void ConvertGifToPdf() {
            string outputFolder = "YOUR_OUTPUT_DIRECTORY";
            string outputFile = Path.Combine(outputFolder, "gif-converted-to.pdf");
            
            // Загрузите исходный GIF-файл
            using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY/Sample.gif")) {
                var options = new PdfConvertOptions();
                converter.Convert(outputFile, options);
            }
        }
    }
}
```

#### Параметры и методы
- `Converter`: Инициализирует процесс конвертации. Передайте ему путь к вашему GIF-файлу.
- `PdfConvertOptions`: Настраивает способ создания PDF-файла.
- `converter.Convert()`: Выполняет преобразование из GIF в PDF.

### Советы по устранению неполадок
- Убедитесь, что все пути указаны правильно и каталоги существуют.
