## Импорт пространств имен
Прежде чем приступить к процессу конвертации, убедитесь, что вы импортировали необходимые пространства имен для доступа к функциям GroupDocs.Conversion.


Теперь, когда мы подготовили почву, давайте перейдем к пошаговому процессу преобразования файла PPSM (слайд-шоу PowerPoint) в PDF с помощью GroupDocs.Conversion для .NET.
## 1. Определите выходную папку и путь к файлу
```csharp
string outputFolder = "Your Document Directory";
string outputFile = Path.Combine(outputFolder, "ppsm-converted-to.pdf");
```
## 2. Загрузите исходный файл PPSM
```csharp
using (var converter = new GroupDocs.Conversion.Converter(Constants.SAMPLE_PPSM))
{
    var options = new PdfConvertOptions();
```
## 3. Настройте параметры преобразования (необязательно)
В зависимости от ваших требований вы можете настроить различные параметры конвертации. В этом примере мы конвертируем в PDF без каких-либо определенных параметров.
## 4. Выполнить преобразование
```csharp
// Сохранить преобразованный PDF-файл

```
