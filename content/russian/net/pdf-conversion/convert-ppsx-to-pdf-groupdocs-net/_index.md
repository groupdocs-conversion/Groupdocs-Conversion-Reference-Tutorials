### Конвертировать PPSX в PDF

#### Обзор
Эта функция демонстрирует, как преобразовать презентацию PowerPoint (PPSX) в документ PDF. Процесс преобразования прост с библиотекой GroupDocs.

**Шаги по реализации:**

##### Шаг 1: Определите пути и загрузите файл
Сначала укажите каталоги для исходных файлов и место назначения выходных данных:
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputFileDirectory = "YOUR_OUTPUT_DIRECTORY";

// Укажите пути к файлам
string ppsxFilePath = Path.Combine(documentDirectory, "sample.ppsx");
string pdfOutputPath = Path.Combine(outputFileDirectory, "ppsx-converted-to.pdf");
```

##### Шаг 2: Настройте параметры конвертации
Далее настройте параметры преобразования PPSX в PDF:
```csharp
// Загрузите исходный файл и подготовьте настройки преобразования.
using (var converter = new Converter(ppsxFilePath))
{
    var convertOptions = new PdfConvertOptions();
    
    // Конвертируйте и сохраните вывод в формате PDF
    converter.Convert(pdfOutputPath, convertOptions);
}
```
**Объяснение**: `Converter` класс загружает ваш файл PPSX, в то время как `PdfConvertOptions` настраивает способ обработки преобразования.

### Советы по устранению неполадок
