## Руководство по внедрению

Теперь, когда у нас все готово, давайте реализуем функцию преобразования файлов XLS в PDF.

### Загрузка исходного файла

Определите пути для входных и выходных каталогов. Заменить `'YOUR_DOCUMENT_DIRECTORY'` и `'YOUR_OUTPUT_DIRECTORY'` с реальными путями в вашей системе:
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// Определите путь для исходного XLS-файла и выходного PDF-файла.
string xlsFilePath = Path.Combine(documentDirectory, "sample.xls");
string pdfOutputFile = Path.Combine(outputDirectory, "xls-converted-to.pdf");
```

### Выполнение преобразования

Чтобы преобразовать файл Excel в PDF:
```csharp
try
{
    // Загрузите исходный XLS-файл
    using (var converter = new Converter(xlsFilePath))
    {
        // Установить параметры конвертации для формата PDF
        var options = new PdfConvertOptions();

        // Выполните преобразование и сохраните выходной PDF-файл.
        converter.Convert(pdfOutputFile, options);
    }
}
catch(Exception ex)
{
    Console.WriteLine("An error occurred: " + ex.Message);
}
```

#### Объяснение

- **Класс преобразователя**: Ответственный за загрузку вашего документа.
- **PdfConvertOptions**: Указывает, что мы хотим преобразовать наш документ в формат PDF.
- **Обработка ошибок**: Блок try-catch гарантирует, что любые проблемы во время преобразования будут обнаружены и зарегистрированы.

### Советы по устранению неполадок

- Убедитесь, что пути к файлам указаны правильно, чтобы предотвратить `FileNotFoundException`.
- Убедитесь, что версия библиотеки GroupDocs.Conversion совместима с вашей средой .NET.
