## Настройка GroupDocs.Conversion для .NET

Когда ваша среда готова и библиотека установлена, приступайте к настройке GroupDocs.Conversion в вашем проекте:

### Базовая инициализация и настройка с помощью C#

Настройте пространства имен и инициализируйте объект преобразования:

```csharp
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

namespace JP2ToPdfConversion
{
    internal static class ConvertJp2ToPdfFeature
    {
        public static void Run()
        {
            string YOUR_DOCUMENT_DIRECTORY = "YOUR_DOCUMENT_DIRECTORY";
            string YOUR_OUTPUT_DIRECTORY = "YOUR_OUTPUT_DIRECTORY";

            // Путь к выходному PDF-файлу
            string outputFolder = YOUR_OUTPUT_DIRECTORY;
            string outputFile = Path.Combine(outputFolder, "jp2-converted-to.pdf");

            // Загрузите исходный файл JP2
            string sourceFilePath = Path.Combine(YOUR_DOCUMENT_DIRECTORY, "sample.jp2");
            using (var converter = new Converter(sourceFilePath))
            {
                // Инициализировать параметры преобразования PDF
                var options = new PdfConvertOptions();

                // Конвертировать и сохранить как PDF
                converter.Convert(outputFile, options);
            }
        }
    }
}
```
Эта настройка подчеркивает простоту конвертации файлов с помощью GroupDocs.Conversion.
