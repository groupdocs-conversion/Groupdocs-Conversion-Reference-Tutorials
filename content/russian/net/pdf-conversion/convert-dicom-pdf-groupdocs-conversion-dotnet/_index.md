## Руководство по внедрению
### Функция: конвертация DICOM в PDF
Эта функция позволяет конвертировать файлы DICOM в формат PDF, что упрощает их просмотр и обмен.

#### Шаг 1: Определите пути для входных и выходных каталогов
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY/";

// Путь к исходному DICOM-файлу
string dicomFilePath = System.IO.Path.Combine(documentDirectory, "your-dicom-file.dcm");

// Определите путь к выходному PDF-файлу
string pdfOutputFile = System.IO.Path.Combine(outputDirectory, "dicom-converted-to.pdf");
```
#### Шаг 2: Загрузите и преобразуйте файл DICOM
Вот как можно загрузить файл DICOM и преобразовать его в PDF:
```csharp
using (Converter converter = new Converter(dicomFilePath))
{
    // Инициализируйте параметры для конвертации в PDF
    PdfConvertOptions options = new PdfConvertOptions();
    
    // Выполнить конвертацию из DICOM в PDF и сохранить результат
    converter.Convert(pdfOutputFile, options);
}
```
**Объяснение**: 
- `Converter`: Загружает ваш DICOM-файл.
- `PdfConvertOptions`: Указывает, что вы хотите получить выходной файл в формате PDF.
- `converter.Convert()`: Выполняет преобразование.

#### Советы по устранению неполадок:
- Убедитесь, что пути к каталогам правильные и доступные.
- Проверьте наличие конфликтов зависимостей в ссылках проекта.
