## Руководство по внедрению
Теперь давайте преобразуем файл CGM в формат DOC с помощью GroupDocs.Conversion для .NET.

### Загрузка и конвертация файла CGM в формат DOC
#### Обзор
В этом разделе мы загружаем исходный файл CGM и преобразуем его в формат документа Microsoft Word (.doc).

#### Шаги по реализации:
**1. Определите входные и выходные пути**
Настройте каталоги и укажите пути для входного CGM-файла и выходного DOC-файла:
```csharp
string documentDirectory = \@"YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = \@"YOUR_OUTPUT_DIRECTORY";

// Укажите путь к исходному CGM-файлу и выходному DOC-файлу
string cgmFilePath = Path.Combine(documentDirectory, "sample.cgm");
string docOutputFile = Path.Combine(outputDirectory, "cgm-converted-to.doc");
```

**2. Загрузите исходный файл CGM**
Используйте GroupDocs.Conversion для загрузки файла CGM:
```csharp
using (var converter = new Converter(cgmFilePath))
{
    Console.WriteLine("CGM file loaded successfully.");
}
```
Этот шаг инициализирует `Converter` экземпляр с указанным путем к файлу CGM.

**3. Задайте параметры преобразования для формата DOC**
Настройте параметры преобразования, специфичные для формата обработки текста (.doc):
```csharp
// Настройте параметры преобразования для формата обработки текста (.doc)
var options = new WordProcessingConvertOptions 
{
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc
};
```
Здесь, `WordProcessingConvertOptions` указывает, что выходной формат должен быть DOC.

**4. Преобразуйте и сохраните вывод**
Выполните преобразование и сохраните полученный файл:
```csharp
converter.Convert(docOutputFile, options);
Console.WriteLine("Conversion completed successfully.");
```
Этот метод преобразует файл CGM в документ DOC с использованием указанных параметров и сохраняет его в нужном месте.
