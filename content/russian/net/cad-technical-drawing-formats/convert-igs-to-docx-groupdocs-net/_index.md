## Настройка GroupDocs.Conversion для .NET

Чтобы начать использовать GroupDocs.Conversion, сначала установите его через NuGet или .NET CLI:

**Консоль менеджера пакетов NuGet:**
```shell
Install-Package GroupDocs.Conversion -Version 25.3.0
```

**.NET CLI:**
```bash
dotnet add package GroupDocs.Conversion --version 25.3.0
```

После установки приобретите лицензию для полной функциональности. GroupDocs предлагает бесплатные пробные версии и временные лицензии для ознакомительных целей. При необходимости вы можете приобрести постоянную лицензию на их официальном сайте.

## Руководство по внедрению

### Шаг 1: Определите путь к выходному каталогу

Укажите, где следует сохранить преобразованный файл DOCX:
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
```
Этот путь имеет решающее значение, поскольку он определяет, где будет находиться полученный файл.

### Шаг 2: Создание полного пути к выходному файлу для DOCX

Создайте полный путь к файлу, чтобы гарантировать правильное наименование и сохранение выходных данных:
```csharp
string outputFile = Path.Combine(outputFolder, "igs-converted-to.docx");
```
Такой подход предотвращает возникновение конфликтов имен или ошибок, связанных со структурой каталогов.

### Шаг 3: Загрузите исходный файл IGS с помощью объекта-конвертера

Загрузите исходный файл с помощью GroupDocs.Conversion `Converter` класс. Это подразумевает создание экземпляра конвертера:
```csharp
using (var converter = new GroupDocs.Conversion.Converter("YOUR_DOCUMENT_DIRECTORY/your-source-file.igs"))
{
    // Дальнейшие шаги будут здесь.
}
```
Эта настройка считывает ваш файл IGS, подготавливая его к конвертации.

### Шаг 4: Задайте параметры преобразования для формата обработки текста

Определите конкретные параметры преобразования в формат DOCX:
```csharp
var options = new WordProcessingConvertOptions();
```
Эти параметры позволяют настроить способ преобразования содержимого IGS в формат DOCX.

### Шаг 5: Преобразуйте и сохраните файл IGS как DOCX

Выполните процесс конвертации и сохраните файл:
```csharp
converter.Convert(outputFile, options);
```
На этом этапе выполняется фактическое преобразование и вывод записывается по указанному пути.
