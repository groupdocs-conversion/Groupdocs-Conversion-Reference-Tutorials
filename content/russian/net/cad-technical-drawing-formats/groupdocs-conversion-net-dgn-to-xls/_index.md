#### Базовая инициализация и настройка

Вот как инициализировать и настроить GroupDocs.Conversion в вашем приложении .NET:
```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main(string[] args)
    {
        // Инициализируйте конвертер, указав путь к файлу DGN.
        using (Converter converter = new Converter("path/to/your/file.dgn"))
        {
            // Укажите параметры конвертации для формата XLS
            var convertOptions = new SpreadsheetConvertOptions();
            
            // Конвертировать и сохранить результат в файл Excel
            converter.Convert("output/path/file.xls", convertOptions);
        }
    }
}
```

### Функция: конвертация DGN в XLS

#### Обзор

Эта функция проведет вас через процесс преобразования файла DGN в формат XLS, гарантируя ясность на каждом этапе.

##### Шаг 1: Инициализация конвертера
Создать `Converter` объект с вашим исходным путем к файлу DGN. Этот класс управляет процессами преобразования.
```csharp
using (Converter converter = new Converter("path/to/your/file.dgn"))
{
    // Логика преобразования идет здесь
}
```
##### Шаг 2: Задайте параметры конвертации
Определите параметры преобразования для формата Excel, используя `SpreadsheetConvertOptions`. Этот объект позволяет вам указать такие настройки, как желаемый тип электронной таблицы.
```csharp
var convertOptions = new SpreadsheetConvertOptions();
```
##### Шаг 3: Выполнение преобразования
Вызовите `Convert` метод с вашим целевым путем к файлу и параметрами преобразования. Этот шаг выполняет фактический процесс преобразования.
```csharp
converter.Convert("output/path/file.xls", convertOptions);
```
#### Советы по устранению неполадок
- Убедитесь, что путь к файлу DGN указан правильно, чтобы избежать ошибок во время выполнения.
- Убедитесь, что выходной каталог имеет права на запись для успешного сохранения файла XLS.
