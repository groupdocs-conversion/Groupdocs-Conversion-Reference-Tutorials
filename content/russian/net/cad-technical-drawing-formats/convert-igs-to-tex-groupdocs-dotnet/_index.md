## Настройка GroupDocs.Conversion для .NET

### Установка
Для начала вам необходимо установить библиотеку GroupDocs.Conversion. Это можно сделать через NuGet Package Manager Console или с помощью .NET CLI:

**Консоль диспетчера пакетов NuGet**
```bash
Install-Package GroupDocs.Conversion -Version 25.3.0
```

**.NET CLI**
```bash
dotnet add package GroupDocs.Conversion --version 25.3.0
```

### Приобретение лицензии
GroupDocs предлагает различные варианты лицензирования, включая бесплатную пробную версию и временные лицензии для оценки. Чтобы получить доступ к полным функциям без ограничений, вы можете приобрести лицензию на их веб-сайте.

#### Базовая инициализация и настройка
После установки инициализация GroupDocs.Conversion проста. Вот как настроить его в вашем проекте C#:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main()
    {
        string documentPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "your-input.igs");
        string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
        string outputFile = Path.Combine(outputFolder, "converted-output.tex");

        Directory.CreateDirectory(outputFolder);

        using (var converter = new Converter(documentPath))
        {
            var options = new PageDescriptionLanguageConvertOptions { Format = FileType.Tex };
            converter.Convert(outputFile, options);
        }
    }
}
```

## Руководство по внедрению

### Загрузка и конвертация IGS в TEX

#### Обзор
В этом разделе рассматривается загрузка файла IGES (IGS) и его конвертация в формат TeX. GroupDocs.Conversion упрощает этот процесс с помощью интуитивно понятного API.

**Шаг 1: Укажите пути к файлам**
Начните с установки входных и выходных путей для ваших файлов. Убедитесь, что эти пути правильно указывают на ваш файл IGS и желаемый выходной каталог:

```csharp
string documentPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "your-input.igs");
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "converted-output.tex");

Directory.CreateDirectory(outputFolder); // Убедитесь, что выходная папка существует
```

**Шаг 2: Инициализация конвертера**
Загрузите ваш файл IGS с помощью `Converter` класс, предоставленный GroupDocs.Conversion:

```csharp
using (var converter = new Converter(documentPath))
{
    var options = new PageDescriptionLanguageConvertOptions { Format = FileType.Tex };
    converter.Convert(outputFile, options);
}
```

**Шаг 3: Настройте параметры конвертации**
Параметры преобразования позволяют указать выходной формат и другие параметры. Здесь мы устанавливаем целевой формат на TeX:

```csharp
var options = new PageDescriptionLanguageConvertOptions { Format = FileType.Tex };
```
