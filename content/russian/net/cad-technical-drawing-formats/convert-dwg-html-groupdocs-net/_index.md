## Руководство по внедрению

### Функция преобразования DWG в HTML

Эта функция позволяет вам конвертировать файлы DWG в формат HTML, делая их удобными для веб-сайтов. Давайте разберем шаги:

#### Шаг 1: Настройка входных и выходных каталогов

Во-первых, четко определите пути к документам:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY"; // Заменить фактическим путем к каталогу
string outputDirectory = "YOUR_OUTPUT_DIRECTORY"; // Заменить на желаемый выходной каталог
```

#### Шаг 2: Загрузите исходный файл DWG

Загрузите ваш файл DWG с помощью GroupDocs.Conversion `Converter` сорт:

```csharp
using (var converter = new Converter(Path.Combine(documentDirectory, "sample.dwg")))
{
    // Перейти к вариантам конвертации
}
```

#### Шаг 3: Задайте параметры преобразования для формата HTML

Настройте необходимые параметры для преобразования HTML с помощью `WebConvertOptions`:

```csharp
var options = new WebConvertOptions();
```

#### Шаг 4: Сохраните преобразованный HTML-файл.

Наконец, сохраните преобразованный файл в указанном выходном каталоге:

```csharp
string outputFile = Path.Combine(outputDirectory, "dwg-converted-to.html");
converter.Convert(outputFile, options);
```

### Советы по устранению неполадок

- **Отсутствующие библиотеки DLL**: Убедитесь, что установлены все необходимые пакеты.
- **Ошибки пути**: Еще раз проверьте путь к документу и выходным файлам.
