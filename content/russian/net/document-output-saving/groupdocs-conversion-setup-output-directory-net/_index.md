### Настроить путь к выходному каталогу
**Обзор:** Эта функция настраивает указанный каталог для хранения преобразованных файлов, позволяя динамически обрабатывать места хранения файлов.

#### Шаг 1: Определите метод
Определите метод, который извлекает или устанавливает путь к выходному каталогу:

```csharp
using System;
using System.IO;

// Определите метод получения пути к выходному каталогу.
class DirectorySetup {
    public static string GetOutputDirectoryPath() {
        // Установите базовый путь к выходной папке
        string baseOutputFolder = "YOUR_OUTPUT_DIRECTORY";

        // Проверьте, существует ли каталог, создайте его, если нет.
        if (!Directory.Exists(baseOutputFolder)) {
            Directory.CreateDirectory(baseOutputFolder);
        }

        return baseOutputFolder;
    }
}
```

**Объяснение:** Этот метод гарантирует доступность указанной вами выходной папки. Если она не существует, метод создает ее для вас.

#### Шаг 2: Использование при конверсии
При выполнении преобразования интегрируйте этот путь к каталогу:

```csharp
class ConversionExample {
    public static void ConvertFile() {
        Converter converter = new Converter("path/to/your/input/file");
        string outputPath = DirectorySetup.GetOutputDirectoryPath();
        converter.Convert("convertedFile.pdf", new PdfSaveOptions { OutputFilePath = Path.Combine(outputPath, "output.pdf") });
    }
}
```

**Объяснение:** В этом фрагменте показано, как применить конфигурацию выходного пути во время операции преобразования файла.
