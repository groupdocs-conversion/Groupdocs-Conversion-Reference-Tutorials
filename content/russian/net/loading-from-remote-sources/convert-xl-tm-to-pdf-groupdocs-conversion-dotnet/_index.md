### Конвертировать XLTM в PDF

**Обзор:** В этом разделе рассматривается преобразование файла XLTM в формат PDF. Выполнив эти шаги, вы сможете автоматизировать преобразование документов в своих приложениях.

#### Шаг 1: Определите пути к выходным и исходным файлам
Укажите пути как для исходного файла XLTM, так и для желаемого выходного файла PDF.

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string inputFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.xltm");
string outputFile = Path.Combine(outputFolder, "xltm-converted-to.pdf");
```

#### Шаг 2: Загрузите исходный файл XLTM
Загрузите исходный файл в `Converter` объект с использованием GroupDocs.Conversion.

```csharp
using (var converter = new Converter(inputFile))
{
    // Логика преобразования будет добавлена здесь
}
```

#### Шаг 3: Задайте параметры конвертации для формата PDF
Укажите, что вывод должен быть в формате PDF, используя `PdfConvertOptions`.

```csharp
var options = new PdfConvertOptions();
```

#### Шаг 4: Преобразование и сохранение выходного PDF-файла
Выполните преобразование и сохраните результат.

```csharp
converter.Convert(outputFile, options);
```

**Советы по устранению неполадок:**
- Убедитесь, что путь к выходному каталогу указан правильно.
- Убедитесь, что входной файл XLTM существует в указанном месте, чтобы избежать `FileNotFoundException`.
