```csharp
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "jpc-converted-to.pptx");

// Инициализируйте конвертер с исходным путем к файлу JPC
using (var converter = new Converter("@YOUR_DOCUMENT_DIRECTORY\\sample.jpc"))
{
    var options = new PresentationConvertOptions();
    converter.Convert(outputFile, options);
}
```

### Функция: конвертация JPC в PPTX

#### Обзор
Эта функция позволяет легко преобразовывать файлы изображений JPEG 2000 (.jpc) в презентации PowerPoint Open XML (.pptx), упрощая создание презентаций из данных изображений.

#### Пошаговая реализация

##### Загрузить исходный файл
Начните с загрузки исходного файла JPC с помощью `Converter` класс, который инициализирует процесс преобразования:

```csharp
string inputFile = "@YOUR_DOCUMENT_DIRECTORY\\sample.jpc";
using (var converter = new Converter(inputFile))
{
    // Параметры и логика преобразования будут представлены ниже.
}
```

##### Укажите параметры конвертации
Определите желаемый формат вывода, настроив `PresentationConvertOptions`. Эти параметры определяют, как ваш файл будет преобразован в PPTX.

```csharp
var options = new PresentationConvertOptions();
// При необходимости добавьте здесь любые особые настройки преобразования.
```

##### Выполнить преобразование
Выполните процесс преобразования и сохраните результат:

```csharp

converter.Convert(outputFile, options);
```

#### Параметры и пояснения методов
- `Converter`: Инициализируется с путем к файлу или потоком. Управляет загрузкой и преобразованием.
- `PresentationConvertOptions`: Задает параметры преобразования PowerPoint.
