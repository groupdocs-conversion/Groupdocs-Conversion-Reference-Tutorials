## Руководство по внедрению

### Конвертация определенных страниц

Функция Convert Specific Pages позволяет вам выбрать и преобразовать только определенные страницы вашего документа в формат PDF. Это особенно полезно для больших документов или при обмене краткой информацией.

#### Шаг 1: Инициализация конвертера
Начните с создания `Converter` объект с путем к исходному документу. Это загрузит документ, подготавливая его к конвертации.
```csharp
using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_DOCX"))
{
    // Мы добавим сюда больше конфигурации...
}
```

#### Шаг 2: Задайте параметры конвертации
Создавать `PdfConvertOptions` чтобы указать, какие страницы вы хотите преобразовать. Определите это с помощью списка номеров страниц.
```csharp
// Создайте PdfConvertOptions для указания параметров преобразования.
PdfConvertOptions options = new PdfConvertOptions
{
    Pages = new List<int> { 1, 3 } // Укажите страницы для конвертации (например, первую и третью страницу).
};
```

#### Шаг 3: Выполнение преобразования
Наконец, используйте `Converter` объект для выполнения преобразования и сохранения выходного PDF-файла.
```csharp
// Выполните преобразование и сохраните полученный PDF-файл.
converter.Convert(outputFile, options);
```

### Объяснение основных параметров
- **Страницы:** Этот параметр позволяет определить список номеров страниц, которые должны быть преобразованы. Это важно для целевого обмена документами.
- **Путь к выходному файлу:** Путь, по которому будет сохранен преобразованный PDF-файл.

### Советы по устранению неполадок
- Убедитесь, что пути к файлам верны и доступны.
- Проверьте, существуют ли указанные страницы в исходном документе.
