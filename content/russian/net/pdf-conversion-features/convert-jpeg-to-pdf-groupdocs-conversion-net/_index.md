## Руководство по внедрению

### Функция преобразования JPEG в PDF

В этом разделе вы узнаете, как преобразовать изображение JPEG в документ PDF с помощью GroupDocs.Conversion.

#### Шаг 1: Загрузите исходный файл

Начните с загрузки исходного файла JPEG. `Converter` класс обрабатывает этот процесс:

```csharp
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY\\sample.jpeg"))
{
    // Продолжайте выполнять шаги преобразования...
}
```

*Почему?* Загрузка файла инициализирует его в GroupDocs.Conversion, подготавливая его к обработке.

#### Шаг 2: Определите параметры преобразования PDF

Укажите желаемые параметры конвертации в PDF:

```csharp
var options = new PdfConvertOptions();
```

*Цель:* The `PdfConvertOptions` класс позволяет настраивать процесс преобразования, хотя часто бывает достаточно настроек по умолчанию.

#### Шаг 3: Выполнение преобразования

Выполните преобразование и сохраните полученный PDF-файл:

```csharp
converter.Convert(outputFile, options);
```

*Почему?* Этот метод запускает реальный процесс преобразования и записывает результат в указанное место в формате PDF.
