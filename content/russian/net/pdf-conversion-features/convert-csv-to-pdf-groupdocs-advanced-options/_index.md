## Руководство по внедрению

### Настройка расширенных параметров конвертации

Сосредоточьтесь на преобразовании CSV-файлов в PDF-файлы, одновременно выполняя преобразования дат и числовых данных.

#### 1. Подготовьте параметры загрузки

```csharp
using GroupDocs.Conversion.Options.Load;

// Настройте параметры загрузки для расширенных настроек\Func<LoadContext, LoadOptions> getLoadOptions = loadContext => new CsvLoadOptions
{
    ПреобразованиеДатыВремениДанные = true,
    ConvertNumericData = true
};
```
- **ConvertDateTimeData**: Преобразует поля даты в удобный для чтения формат.
- **ПреобразованиеЧисловыхДанных**: Обеспечивает правильность форматирования чисел.

#### 2. Инициализация конвертера

Инициализируйте преобразователь с расширенными параметрами загрузки:

```csharp
string sourceDirectory = @"YOUR_DOCUMENT_DIRECTORY";
string outputFolder = @"YOUR_OUTPUT_DIRECTORY\\";

string inputFile = Path.Combine(sourceDirectory, "sample.csv");
string outputFile = Path.Combine(outputFolder, "converted.pdf");

using (Converter converter = new Converter(inputFile, getLoadOptions))
{
    PdfConvertOptions options = new PdfConvertOptions();
    converter.Convert(outputFile, options);
}
```
- `inputFile`: Путь к вашему CSV-файлу.
- `outputFile`: Путь назначения для PDF-файла.
