## Руководство по внедрению

Мы разобьем процесс конвертации на управляемые этапы.

### Функция: Преобразование MHT в CSV

Эта функция позволяет преобразовать файл MHT в формат CSV, что делает данные более доступными для анализа и составления отчетов.

#### Шаг 1: Определите пути к файлам
Эффективно управляйте входными и выходными путями. Это гарантирует бесперебойную работу без ошибок, связанных с путями.

```csharp
using System.IO;

string sourceMhtPath = "YOUR_DOCUMENT_DIRECTORY\\sample.mht"; // Входной MHT-файл
string outputFolder = "YOUR_OUTPUT_DIRECTORY"; // Выходной каталог
if (!Directory.Exists(outputFolder))
{
    Directory.CreateDirectory(outputFolder); // Создать, если не существует
}
string outputFile = Path.Combine(outputFolder, "mht-converted-to.csv");
```

#### Шаг 2: Загрузите исходный MHT-файл
Загрузка исходного файла — это первый шаг в процессе конвертации.

```csharp
using (var converter = new Converter(sourceMhtPath))
{
    // Код преобразования будет здесь
}
```

#### Шаг 3: Определите параметры преобразования
Укажите, что вы хотите преобразовать в формат CSV, используя `SpreadsheetConvertOptions`.

```csharp
var convertOptions = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Csv };
```

#### Шаг 4: Выполнение преобразования и сохранение вывода
Наконец, выполните преобразование и сохраните файл.

```csharp
converter.Convert(outputFile, convertOptions);
Console.WriteLine("Conversion completed successfully!");
```

### Функция: Управление путями к файлам

Эффективное управление путями к файлам гарантирует, что файлы будут сохраняться в правильных каталогах без ошибок.

#### Шаг 1: Настройка каталогов
Прежде чем приступить к преобразованиям, убедитесь, что существуют как входной, так и выходной каталоги.

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string sampleMhtFilePath = Path.Combine(documentDirectory, "sample.mht");

string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
if (!Directory.Exists(outputDirectory))
{
    Directory.CreateDirectory(outputDirectory);
}

string csvOutputFilePath = Path.Combine(outputDirectory, "mht-converted-to.csv");
```
