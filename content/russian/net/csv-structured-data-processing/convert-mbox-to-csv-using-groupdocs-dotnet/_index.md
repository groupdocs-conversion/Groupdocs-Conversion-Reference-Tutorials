## Руководство по внедрению
В этом разделе вы узнаете, как загрузить файл MBOX и преобразовать его в формат CSV. Для ясности мы разберем его по функциям.

### Функция 1: Загрузка файла MBOX
#### Обзор
Загрузка файла MBOX — это первый шаг в нашем процессе конвертации. GroupDocs.Conversion предоставляет простой способ справиться с этим с помощью `MboxLoadOptions`.

#### Этапы внедрения
**Шаг 1**: Определите путь к файлу MBOX и параметры загрузки:
```csharp
string mboxFilePath = @"YOUR_DOCUMENT_DIRECTORY\\sample.mbox";
MboxLoadOptions loadOptions = new MboxLoadOptions();
```

**Шаг 2**: Создайте экземпляр Converter для обработки процесса конвертации:
```csharp
var converter = new Converter(mboxFilePath, (LoadContext context) => {
    return context.SourceFormat == EmailFileType.Mbox ? loadOptions : null;
});
```
*Почему этот шаг?* Это гарантирует, что будут обрабатываться только файлы MBOX с указанными параметрами.

### Функция 2: Преобразование MBOX в CSV
#### Обзор
После загрузки файла MBOX мы можем преобразовать его в формат CSV. Процесс преобразования использует `SpreadsheetConvertOptions`.

#### Этапы внедрения
**Шаг 1**: Настройте выходной каталог и шаблон:
```csharp
string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "mbox-converted-{0}-to.csv");
```

**Шаг 2**: Определите параметры преобразования для формата CSV:
```csharp
SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Csv };
int counter = 1;
```

**Шаг 3**: Выполните преобразование и сохраните каждый файл как CSV:
```csharp
counter = 1; // Сброс для ясности объяснения
converter.Convert(
    (SaveContext saveContext) => new FileStream(string.Format(outputFileTemplate, counter++), FileMode.Create),
    options
);
```
*Почему этот шаг?* Это обеспечивает фактическое преобразование и сохранение каждого электронного письма в новом CSV-файле.

### Советы по устранению неполадок
- Убедитесь, что путь к MBOX указан правильно; в противном случае вы столкнетесь с ошибками «Файл не найден».
- Убедитесь, что GroupDocs.Conversion правильно установлен и указан в вашем проекте.
