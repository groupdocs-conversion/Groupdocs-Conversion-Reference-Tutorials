## Руководство по внедрению

Мы разберем реализацию на две основные функции: загрузку PLT-файлов и преобразование их в CSV.

### Загрузить файл PLT

**Обзор**: Эта функция демонстрирует, как загрузить исходный файл PLT, подготовив его к преобразованию.

#### Шаг 1: Определите пути к файлам (H3)
Укажите каталог документов, в котором находится исходный файл PLT:
```csharp
private const string DocumentDirectory = @"YOUR_DOCUMENT_DIRECTORY\\/";
```

#### Шаг 2: Загрузите исходный файл PLT (H3)

Используйте GroupDocs.Conversion для загрузки вашего файла PLT:
```csharp
using System;
using GroupDocs.Conversion;

namespace FeatureLoadPltFile {
    internal static class LoadPlt {
        public static void Run() {
            string sourceFilePath = Path.Combine(DocumentDirectory, "sample.plt");
            
            using (var converter = new Converter(sourceFilePath)) {
                // Логика преобразования будет реализована в следующей функции.
            }
        }
    }
}
```
*Почему именно такой подход?* С использованием `Converter` инициализирует файловый поток и подготавливает его к последующим операциям.

### Конвертировать PLT в CSV

**Обзор**: В этом разделе показано, как преобразовать загруженный файл PLT в формат CSV, оптимизируя обработку данных.

#### Шаг 1: Определите выходные пути (H3)
Настройте пути для исходного и выходного каталогов:
```csharp
private const string OutputDirectory = @"YOUR_OUTPUT_DIRECTORY\\/";
string outputPath = Path.Combine(OutputDirectory, "plt-converted-to.csv");
```

#### Шаг 2: Установите параметры конвертации (H3)

Настройте параметры для преобразования файла в формат CSV:
```csharp
using GroupDocs.Conversion.Options.Convert;

var options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Csv };
```
*Зачем использовать `SpreadsheetConvertOptions`?* Он определяет параметры преобразования, адаптированные для форматов электронных таблиц, таких как CSV.

#### Шаг 3: Выполнение преобразования (H3)

Выполните преобразование и сохраните результат:
```csharp
using (var converter = new Converter(sourceFilePath)) {
    converter.Convert(outputPath, options);
}
```
Этот фрагмент эффективно обрабатывает преобразование файлов, используя надежный API GroupDocs.
