### Конвертировать VSDM в TXT

В этом разделе рассматривается преобразование чертежей Visio с поддержкой макросов в обычный текст.

#### Шаг 1: Настройка каталогов
Определите ваши входные и выходные каталоги. Замените заполнители фактическими путями в вашей среде.

```csharp
string YOUR_DOCUMENT_DIRECTORY = "/path/to/your/documents";
string YOUR_OUTPUT_DIRECTORY = "/path/to/output/directory/";
```

#### Шаг 2: Загрузите исходный файл VSDM
Создать `Converter` объект для загрузки исходного файла по указанному пути. Этот шаг инициализирует процесс преобразования.

```csharp
using (var converter = new Converter(Path.Combine(YOUR_DOCUMENT_DIRECTORY, "yourfile.vsdm")))
{
    // Здесь будут добавлены дополнительные этапы преобразования.
}
```

#### Шаг 3: Настройте параметры конвертации
Настройте параметры, необходимые для конвертации в формат TXT. Укажите, что целевой тип файла — TXT.

```csharp
WordProcessingConvertOptions options = new WordProcessingConvertOptions 
{
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt
};
```

#### Шаг 4: Выполнение преобразования
Выполните преобразование и сохраните вывод в виде файла TXT. Полученный файл будет сохранен в указанном выходном каталоге.

```csharp
string outputFile = Path.Combine(YOUR_OUTPUT_DIRECTORY, "vsdm-converted-to.txt");
converter.Convert(outputFile, options);
```

### Советы по устранению неполадок

- Убедитесь, что все пути установлены правильно, чтобы избежать `FileNotFoundException`.
