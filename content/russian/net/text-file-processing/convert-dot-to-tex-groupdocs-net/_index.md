### Преобразование файла DOT в формат TEX
Эта функция позволяет преобразовать шаблон документа Microsoft Word (.dot) в исходный документ LaTeX (.tex).

#### Шаг 1: Определите пути для входных и выходных каталогов
Перед конвертацией файлов укажите каталоги, в которых находятся исходные документы и куда вы хотите сохранить преобразованные файлы.
```csharp
string inputDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// Загрузите исходный файл DOT
using (var converter = new Converter(Path.Combine(inputDirectory, "sample.dot")))
{
    // Установить параметры преобразования для формата LaTeX
    var convertOptions = new PageDescriptionLanguageConvertOptions
    {
        Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Tex
    };

    // Определите путь к выходному файлу и выполните преобразование.
    string outputFile = Path.Combine(outputDirectory, "dot-converted-to.tex");
    converter.Convert(outputFile, convertOptions);
}
```
#### Параметры и конфигурация
- **Входной каталог**: Разместите здесь исходные файлы .dot.
- **Выходной каталог**: Преобразованные файлы .tex будут сохранены в этом месте.
- **Варианты конвертации**: Укажите формат для преобразования с помощью `PageDescriptionLanguageConvertOptions`.
