# Как преобразовать MHTML в текст в C# с помощью GroupDocs.Conversion для .NET

Это руководство проведет вас через использование GroupDocs.Conversion для .NET для преобразования файлов MHTML в простые файлы TXT.

## Руководство по внедрению

### Обзор функции преобразования

Ключевой функциональностью здесь является преобразование файла MHTML в обычный текстовый формат (.txt), который можно использовать для дальнейшей обработки или анализа.

#### Шаг 1: Определите пути к документам и выходной каталог
```csharp
using System;
using System.IO;

string sourceMhtmlPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.mhtml");
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "mhtml-converted-to.txt");
```

#### Шаг 2: Загрузите файл MHTML и задайте параметры конвертации
```csharp
using GroupDocs.Conversion.Options.Convert;

// Загрузите файл MHTML с помощью GroupDocs.Conversion
using (var converter = new Converter(sourceMhtmlPath))
{
    // Установите параметры конвертации для конвертации в формат TXT
    var options = new WordProcessingConvertOptions
    {
        Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt
    };
}
```

#### Шаг 3: Выполните преобразование и сохраните результат
```csharp
// Выполните преобразование и сохраните как файл .txt.
converter.Convert(outputFile, options);
Console.WriteLine("Conversion completed successfully.");
```

### Объяснение основных параметров

- **sourceMhtmlПуть:** Путь к исходному документу MHTML.
- **выходной файл:** Путь, по которому будет сохранен преобразованный TXT.
- **WordProcessingConvertOptions:** Параметры, указывающие целевой формат (в данном случае TXT).

#### Советы по устранению неполадок
- Убедитесь, что пути указаны правильно и каталоги существуют.
- Убедитесь, что версия пакета GroupDocs.Conversion совместима с вашей средой.
