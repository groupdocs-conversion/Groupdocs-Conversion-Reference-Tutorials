## Руководство по внедрению
Чтобы преобразовать PDF-файл в формат TXT, выполните следующие действия.

### 1. Определите каталоги для входных и выходных файлов
Укажите каталоги для исходного PDF-файла и выходного TXT-файла:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// Убедитесь, что каталоги существуют или создайте их.
if (!Directory.Exists(documentDirectory)) Directory.CreateDirectory(documentDirectory);
if (!Directory.Exists(outputDirectory)) Directory.CreateDirectory(outputDirectory);
```

### 2. Загрузите и подготовьте исходный PDF-файл
Загрузите исходный PDF-файл с помощью `Converter` сорт:

```csharp
string sourceFilePath = Path.Combine(documentDirectory, "sample.pdf");
string outputFile = Path.Combine(outputDirectory, "pdf-converted-to.txt");

using (var converter = new GroupDocs.Conversion.Converter(sourceFilePath))
{
    // Конвертация произойдет на следующем этапе
}
```

### 3. Задайте параметры конвертации и выполните конвертацию.
Настройте параметры преобразования для формата TXT и выполните преобразование:

```csharp
// Используйте WordProcessingConvertOptions для преобразования текстового формата
WordProcessingConvertOptions options = new WordProcessingConvertOptions 
{ 
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt 
};

// Выполнить преобразование и сохранить в выходной файл.
converter.Convert(outputFile, options);
```

### Советы по устранению неполадок
- **PDF-файл не найден**: Убедитесь, что исходный PDF-файл существует в указанном каталоге.
- **Проблемы с разрешениями**: Убедитесь, что ваше приложение имеет права на запись в выходной каталог.
