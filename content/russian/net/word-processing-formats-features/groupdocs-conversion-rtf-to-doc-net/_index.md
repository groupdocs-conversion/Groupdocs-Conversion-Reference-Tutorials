### Конвертировать RTF в DOC

Эта функция позволяет преобразовать файл RTF в формат DOC с помощью GroupDocs.Conversion для .NET. 

#### Шаг 1: Настройте свой проект и загрузите исходный файл

Убедитесь, что ваш проект настроен с необходимыми зависимостями, и загрузите исходный RTF-файл:

```csharp
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY\\sample.rtf"))
{
    // Параметры конвертации будут определены на следующем этапе.
}
```

#### Шаг 2: Определите параметры преобразования

Далее укажите настройки конвертации, используя `WordProcessingConvertOptions` для целевого формата DOC:

```csharp
WordProcessingConvertOptions options = new WordProcessingConvertOptions 
{ 
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc 
};
```

#### Шаг 3: Выполнение преобразования

Наконец, выполните преобразование и сохраните выходной файл:

```csharp
string outputFile = Path.Combine(outputFolder, "rtf-converted-to.doc");
converter.Convert(outputFile, options);
```

### Определить путь к выходному каталогу

Правильное управление выходными путями имеет решающее значение для организации преобразованных файлов. Этот метод гарантирует, что указанный вами каталог существует перед сохранением преобразованных документов:

```csharp
string GetOutputDirectoryPath()
{
    string outputFolder = "YOUR_OUTPUT_DIRECTORY";
    if (!Directory.Exists(outputFolder))
        Directory.CreateDirectory(outputFolder);
    return outputFolder;
}
```
