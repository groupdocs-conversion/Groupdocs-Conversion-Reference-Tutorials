```csharp
        }

}
```

## Руководство по внедрению
### Загрузка и конвертация файла DOCM в DOCX

**Обзор:** Эта функция позволяет вам загружать файл DOCM и конвертировать его в формат DOCX с помощью GroupDocs.Conversion. Она позволяет работать с универсальными документами Word в ваших приложениях.

**Шаги для преобразования**

**Шаг 1: Задайте пути**
Укажите пути для исходного файла DOCM и выходного файла DOCX:

```csharp
string sourceDocmPath = Path.Combine(@"YOUR_DOCUMENT_DIRECTORY", "sample.docm");
string outputFile = Path.Combine(@"YOUR_OUTPUT_DIRECTORY", "docm-converted-to.docx");
```

**Шаг 2: Загрузите исходный файл**
Создайте экземпляр `Converter` class с путем к вашему файлу DOCM:

```csharp
using (var converter = new Converter(sourceDocmPath))
{
    // Логика преобразования идет здесь
}
```
The `Converter` Объект обрабатывает загрузку документов и предоставляет методы для задач преобразования.

**Шаг 3: Укажите параметры конвертации**
Определите целевой формат, используя `WordProcessingConvertOptions`:

```csharp
var options = new WordProcessingConvertOptions();
```
Это означает, что вы конвертируете в файл DOCX, являющийся частью форматов текстового процессора Microsoft Office.

**Шаг 4: Выполнение преобразования**
Выполните преобразование и сохраните выходной файл DOCX:

```csharp
converter.Convert(outputFile, options);
```
Здесь, `converter.Convert` преобразует DOCM в DOCX, используя указанные параметры.
