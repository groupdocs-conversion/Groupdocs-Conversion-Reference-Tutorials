## Руководство по внедрению
### Функция: загрузка и конвертация MHTML в DOCX
#### Обзор
Преобразование файла MHTML в формат DOCX включает загрузку источника, настройку параметров и выполнение преобразования. Выполните следующие шаги:

**Шаг 1**: Определить выходной каталог
Настройте выходной каталог для хранения преобразованных файлов.

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
```

**Шаг 2**: Загрузить исходный файл MHTML
Определите и загрузите файл MHTML, который вы хотите преобразовать. Заменить `YOUR_DOCUMENT_DIRECTORY` с путем к папке с документами.

```csharp
var sampleMhtmlPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY\sample.mhtml");
using (Converter converter = new Converter(sampleMhtmlPath))
{
    // Здесь будут определены этапы преобразования.
}
```

**Шаг 3**: Настройка параметров преобразования DOCX
Укажите выходной формат и дополнительные настройки для конвертации.

```csharp
var convertOptions = new DocxConvertOptions();
string outputFile = Path.Combine(outputFolder, "mhtml-converted-to.docx");
converter.Convert(outputFile, convertOptions);
```

#### Объяснение параметров
- **Конвертер**: Представляет экземпляр конвертера файлов.
- **DocxConvertOptions**: Позволяет настраивать параметры вывода DOCX.
