## Руководство по внедрению

### Пошаговое преобразование PPTX в DOCX

Эта функция позволяет легко конвертировать презентации PowerPoint в документы Word.

#### Шаг 1: Подготовьте файлы

Убедитесь, что пути ввода и вывода указаны правильно:

```csharp
string inputFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "presentation.pptx");
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
```

#### Шаг 2: Настройте параметры конвертации

Укажите целевой формат (DOCX):

```csharp
var convertOptions = new WordProcessingConvertOptions();
```

#### Шаг 3: Выполнение преобразования

Выполните преобразование и сохраните файл DOCX:

```csharp
string outputFile = Path.Combine(outputFolder, "output.docx");
class ConverterExample {
    public void ConvertPptxToDocx() {
        var converter = new Converter(inputFile);
        converter.Convert(() => File.Create(outputFile), convertOptions);
    }
}
```

**Объяснение**: `Convert` Метод использует лямбда-функцию для создания файла, гарантируя, что выходной путь является допустимым. Параметры преобразования определяют структуру DOCX.

#### Советы по устранению неполадок
- **Ошибки пути к файлу**: Дважды проверьте входные и выходные пути.
- **Отсутствующие зависимости**: Убедитесь, что все необходимые пакеты установлены через NuGet.
