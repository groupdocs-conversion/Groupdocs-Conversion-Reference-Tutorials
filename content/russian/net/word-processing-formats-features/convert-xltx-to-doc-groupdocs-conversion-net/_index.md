### Конвертировать XLTX в DOC
**Обзор**: эта функция демонстрирует, как преобразовать файл шаблона Excel (.xltx) в документ Word (DOC) с помощью GroupDocs.Conversion для .NET.

#### Шаг 1: Настройка путей для преобразования документов
Определите пути для входных и выходных файлов. Заменить `"YOUR_DOCUMENT_DIRECTORY"` и `"YOUR_OUTPUT_DIRECTORY"` с реальными каталогами в вашей системе.

```csharp
using System.IO;

string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.xltx");
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "xltx-converted-to.doc");
```

#### Шаг 2: Загрузите и преобразуйте файл
Загрузите ваш файл .xltx с помощью `Converter` класс и определение параметров преобразования для форматов текстового процессора.

```csharp
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

// Загрузите исходный файл XLTX
using (var converter = new Converter(inputFilePath))
{
    // Определить параметры преобразования для формата DOC
    var options = new WordProcessingConvertOptions { Format = WordProcessingFileType.Doc };
    
    // Выполнить преобразование и сохранить выходной файл.
    converter.Convert(outputFile, options);
}
```

**Объяснение**: 
- **Класс преобразователя**: Обрабатывает загрузку исходных файлов.
- **WordProcessingConvertOptions**: Настраивает специальные параметры для преобразования формата DOC.

#### Советы по устранению неполадок
- Убедитесь, что пути к входным и выходным каталогам верны и доступны.
- Проверьте, есть ли у вас достаточные права на чтение/запись файлов в указанных каталогах.
