Сначала определите пути для входных и выходных каталогов:

**Инициализировать преобразователь**
```csharp

using (var converter = new Converter(Path.Combine(documentDirectory, "sample.vstx")))
```

Далее укажите параметры для преобразования в формат DOCX. Этот шаг настраивает, как должен быть отформатирован ваш выходной документ:

**Установить параметры конвертации**
```csharp
// Укажите параметры преобразования для обработки текста (DOCX)
```

### Выполнение преобразования
Наконец, выполните фактическое преобразование и сохраните полученный файл DOCX в желаемом месте. Вот где происходит волшебство:

**Конвертировать и сохранить**
```csharp
// Определить выходной путь для преобразованного файла DOCX
string outputFile = Path.Combine(outputDirectory, "vstx-converted-to.docx");

// Выполнить преобразование
```
