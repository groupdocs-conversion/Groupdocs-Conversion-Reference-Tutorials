## Руководство по внедрению
В этом разделе вы узнаете, как реализовать функцию преобразования ODT в TXT.
### 1. Подготовка файлов
Перед преобразованием убедитесь, что исходный ODT-файл правильно размещен в каталоге проекта.
#### Определить пути к файлам
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// Пути источника и назначения
string sourceFilePath = Path.Combine(documentDirectory, "sample.odt");
string outputFile = Path.Combine(outputDirectory, "odt-converted-to.txt");
```
### 2. Преобразование ODT в TXT
Указав пути к файлам, выполните преобразование:
#### Инициализировать конвертер и задать параметры
```csharp
using (var converter = new GroupDocs.Conversion.Converter(sourceFilePath))
{
    var options = new WordProcessingConvertOptions
    {
        Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt
    };

    // Выполнить преобразование
    converter.Convert(outputFile, options);
}
```
### Пояснение параметров:
- **Конвертер**: Берет путь к исходному файлу и инициализирует сеанс преобразования.
- **WordProcessingConvertOptions**: Настраивает параметры выходного формата (в данном случае TXT).
- **конвертер.Convert()**: Выполняет преобразование файла.
#### Советы по устранению неполадок
- Убедитесь, что пути указаны правильно, чтобы избежать `FileNotFoundException`.
- Проверьте наличие достаточных прав доступа к каталогам для чтения/записи файлов.
