### Базовая инициализация

Чтобы инициализировать и настроить GroupDocs.Conversion в вашем проекте C#, выполните следующие действия:

```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

// Укажите выходной каталог для преобразованных файлов.
string outputFolder = "YOUR_OUTPUT_DIRECTORY";

// Путь к вашему OTG-файлу.
string inputFile = "YOUR_DOCUMENT_DIRECTORY/sample.otg"; 

// Объедините пути, чтобы создать полный путь к выходному DOC-файлу.
string outputFile = Path.Combine(outputFolder, "otg-converted-to.doc");

using (var converter = new GroupDocs.Conversion.Converter(inputFile))
{
    // Параметры преобразования установлены для формата обработки Word как DOC
    var options = new WordProcessingConvertOptions { Format = WordProcessingFileType.Doc };
    
    // Выполните преобразование из OTG в DOC.
    converter.Convert(outputFile, options);
}
```

## Руководство по внедрению

### Конвертировать OTG в DOC

**Обзор**: эта функция позволяет конвертировать файлы графических шаблонов OpenDocument (.otg) в документы Microsoft Word (.doc).

#### Шаг 1: Настройте параметры конвертации
Создать экземпляр `WordProcessingConvertOptions` и укажите целевой формат как `Doc`.

```csharp
var options = new WordProcessingConvertOptions { Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc };
```

**Объяснение**: Эта конфигурация предписывает библиотеке преобразовать входной файл в формат DOC, который широко используется в Microsoft Word.

#### Шаг 2: Выполнение преобразования

Используйте `Converter` класс для загрузки вашего OTG-файла и применения параметров конвертации.

```csharp
using (var converter = new GroupDocs.Conversion.Converter(inputFile))
{
    converter.Convert(outputFile, options);
}
```

**Объяснение**: `Convert` Метод обрабатывает фактический процесс преобразования, сохраняя вывод в указанном каталоге.
