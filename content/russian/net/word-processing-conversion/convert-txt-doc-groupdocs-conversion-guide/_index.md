## Руководство по внедрению

### Функция преобразования TXT в DOC

Давайте разобьем реализацию на логические шаги:

#### Шаг 1: Загрузите исходный файл

Укажите путь к вашему источнику `.txt` файл и инициализируйте `Converter` объект:

```csharp
string sampleTxtPath = "YOUR_DOCUMENT_DIRECTORY\sample.txt";
var converter = new Converter(sampleTxtPath);
```

**Почему?** Загрузка вашего текстового файла в `Converter` объект необходим для доступа к его содержимому в рамках GroupDocs.Conversion.

#### Шаг 2: Задайте параметры конвертации

Определите параметры конвертации для преобразования из формата TXT в формат DOC:

```csharp
// Определить параметры преобразования Word
class WordProcessingConvertOptions { public WordProcessingFileType Format; }
var wordOptions = new WordProcessingConvertOptions { Format = WordProcessingFileType.Doc };
```

**Параметры:** `WordProcessingConvertOptions` указывает тип выходного файла, в данном случае `.doc`.

#### Шаг 3: Выполнение преобразования

Выполните преобразование и сохраните полученный документ:

```csharp
string outputFile = "YOUR_OUTPUT_DIRECTORY\txt-converted-to.doc";
// Конвертируйте и сохраните файл DOC
converter.Convert(outputFile, wordOptions);
```

**Почему?** The `Convert` метод обрабатывает входной файл в соответствии с указанными параметрами и выводит `.doc` файл в указанном вами каталоге.
