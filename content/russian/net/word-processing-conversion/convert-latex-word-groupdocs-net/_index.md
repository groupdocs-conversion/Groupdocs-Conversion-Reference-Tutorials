## Настройка GroupDocs.Conversion для .NET
Чтобы начать конвертировать документы LaTeX, сначала установите библиотеку GroupDocs.Conversion. Вот как это сделать:
### Консоль диспетчера пакетов NuGet
```bash
Install-Package GroupDocs.Conversion -Version 25.3.0
```
### .NET CLI
```bash
dotnet add package GroupDocs.Conversion --version 25.3.0
```
#### Приобретение лицензии:
- **Бесплатная пробная версия**Начните с бесплатной пробной версии, чтобы протестировать возможности библиотеки.
- **Временная лицензия**: Получите временную лицензию для расширенного тестирования.
- **Покупка**: Рассмотрите возможность приобретения полной лицензии для постоянного использования.
После установки инициализируйте и настройте GroupDocs.Conversion в своем проекте:
```csharp
using System.IO;
using GroupDocs.Conversion.Options.Convert;
using GroupDocs.Conversion;
// Определите пути для входных и выходных файлов
string outputFolder = "YOUR_OUTPUT_DIRECTORY"; // Укажите желаемый путь к выходному каталогу
string inputFile = "YOUR_DOCUMENT_DIRECTORY/sample.tex"; // Путь к входному .tex-файлу
string outputFile = Path.Combine(outputFolder, "tex-converted-to.doc"); // Путь к выходному файлу
// Загрузите исходный TEX-файл с помощью GroupDocs.Conversion
using (var converter = new Converter(inputFile)) 
{
    WordProcessingConvertOptions options = new WordProcessingConvertOptions { Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc };
    // Конвертировать и сохранить файл DOC по указанному выходному пути.
    converter.Convert(outputFile, options);
}
```
