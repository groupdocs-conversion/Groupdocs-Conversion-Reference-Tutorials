### Базовая инициализация

Инициализируйте GroupDocs.Conversion с помощью следующего кода:

```csharp
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

// Определите каталоги для ваших документов.
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// Объедините пути для формирования полных путей к файлам.
string inputFile = Path.Combine(documentDirectory, "sample.xlam");
string outputFile = Path.Combine(outputDirectory, "xlam-converted-to.doc");

using (var converter = new Converter(inputFile))
{
    // Параметры преобразования для формата текстового процессора.
    var options = new WordProcessingConvertOptions { Format = WordProcessingFileType.Doc };
    converter.Convert(outputFile, options);
}
```

Этот код настраивает процесс конвертации, загружает ваш файл XLAM и преобразует его в файл DOC.

## Руководство по внедрению

### Обзор процесса конверсии

Библиотека GroupDocs.Conversion упрощает преобразование файлов между различными форматами. В этом разделе основное внимание уделяется преобразованию файла XLAM в формат DOC с использованием C#.

#### Шаг 1: Определите пути к файлам

Укажите пути для входных и выходных файлов:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// Обязательно замените «sample.xlam» на фактическое имя файла.
string inputFile = Path.Combine(documentDirectory, "sample.xlam");
string outputFile = Path.Combine(outputDirectory, "xlam-converted-to.doc");
```

#### Шаг 2: Инициализация конвертера

Создайте экземпляр `Converter` класс и загрузите ваш файл XLAM:

```csharp
using (var converter = new Converter(inputFile))
{
    // Перейти к вариантам конвертации.
}
```

#### Шаг 3: Задайте параметры конвертации

Определите формат, в который вы хотите преобразовать, используя `WordProcessingConvertOptions`:

```csharp
var options = new WordProcessingConvertOptions { Format = WordProcessingFileType.Doc };
```

#### Шаг 4: Выполнение преобразования

Выполните преобразование и сохраните ваш DOC-файл:

```csharp
converter.Convert(outputFile, options);
```
