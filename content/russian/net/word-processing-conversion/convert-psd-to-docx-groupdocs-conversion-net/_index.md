## Руководство по внедрению

### Шаг 1: Загрузите исходный PSD-файл

Начните с загрузки исходного PSD-файла в приложение .NET для преобразования.

**Определить путь к документу:**
```csharp
string psdFilePath = "YOUR_DOCUMENT_DIRECTORY/sample.psd";
```

**Загрузите PSD-файл:**
Используйте GroupDocs.Conversion для загрузки файла и подготовки его к последующим операциям:
```csharp
using (var converter = new Converter(psdFilePath))
{
    // Готовы к выполнению конвертации.
}
```

### Шаг 2: Преобразование PSD в формат DOCX

Загрузив PSD-файл, приступайте к его конвертации в формат Word.

**Установите входные и выходные пути:**
Определите пути для входного PSD-файла и выходного DOCX-файла:
```csharp
string inputFilePath = "YOUR_DOCUMENT_DIRECTORY/sample.psd";
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "psd-converted-to.docx");
```

**Укажите параметры конвертации:**
Укажите преобразование в формат обработки Word (DOCX):
```csharp
var options = new WordProcessingConvertOptions();
```

**Выполнить преобразование:**
Сохраните преобразованный файл DOCX, используя указанные параметры:
```csharp
using (var converter = new Converter(inputFilePath))
{
    converter.Convert(outputFile, options);
}
```
