### Конвертировать JPEG 2000 в формат DOCX

#### Обзор
Узнайте, как легко преобразовать изображение JPEG 2000 (JP2) в документ Word в формате DOCX, который идеально подходит для редактирования или интеграции визуальных данных с текстовым контентом.

##### Шаг 1: Настройте пути к файлам
Определите пути для вашего файла JP2 и выходного DOCX:
```csharp
string sourceFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.jp2");
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "jp2-converted-to.docx");
```

##### Шаг 2: Загрузите исходный файл
Инициализируйте `Converter` объект:
```csharp
using (var converter = new Converter(sourceFilePath))
{
    // Перейдите к настройке параметров конвертации.
}
```

##### Шаг 3: Настройте параметры конвертации
Настройте параметры преобразования текстового процессора для формата DOCX:
```csharp
var options = new WordProcessingConvertOptions();
```

##### Шаг 4: Выполнение преобразования
Выполните и сохраните преобразование:
```csharp
converter.Convert(outputFile, options);
```
