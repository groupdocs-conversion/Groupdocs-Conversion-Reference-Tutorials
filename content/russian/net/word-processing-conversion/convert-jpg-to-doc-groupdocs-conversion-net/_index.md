Для начала установите пакет GroupDocs.Conversion в свой проект. Вот как это сделать:

### Консоль диспетчера пакетов NuGet

```bash
Install-Package GroupDocs.Conversion -Version 25.3.0
```

### .NET CLI
```bash
dotnet add package GroupDocs.Conversion --version 25.3.0
```

## Руководство по внедрению

Давайте разберем процесс конвертации на четкие шаги:

### Конвертировать JPG в DOC
Эта функция позволяет конвертировать изображения JPEG в документы Microsoft Word с помощью GroupDocs.Conversion.

#### Шаг 1: Определите пути к файлам
Настройте исходные и выходные каталоги, убедившись, что пути для входных и выходных файлов указаны правильно.
```csharp
private static readonly string SourceDirectory = "YOUR_DOCUMENT_DIRECTORY";
private static readonly string OutputDirectory = "YOUR_OUTPUT_DIRECTORY";

string sourceFilePath = Path.Combine(SourceDirectory, "sample.jpg");
string outputFile = Path.Combine(OutputDirectory, "jpg-converted-to.doc");
```
#### Шаг 2: Инициализация конвертера
Создайте экземпляр `Converter` class с путем к файлу JPG. Этот шаг подготавливает изображение к конвертации.
```csharp
using (var converter = new Converter(sourceFilePath))
{
    // Шаги преобразования будут описаны здесь.
}
```
#### Шаг 3: Задайте параметры конвертации
Укажите выходной формат, используя `WordProcessingConvertOptions`. Это сообщает GroupDocs.Conversion о необходимости создания файла DOC.
```csharp
var convertOptions = new WordProcessingConvertOptions { Format = WordProcessingFileType.Doc };
```
#### Шаг 4: Выполнение преобразования
Выполните преобразование и сохраните вывод по указанному вами пути. Этот шаг преобразует ваш JPG в формат DOC.
```csharp
converter.Convert(outputFile, convertOptions);
```
**Совет по устранению неполадок**: Убедитесь, что пути к файлам правильные и доступные, чтобы избежать `FileNotFoundException`.
