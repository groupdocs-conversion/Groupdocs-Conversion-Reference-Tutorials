### Базовая инициализация и настройка
Настройте свою среду с помощью следующего кода:
```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main(string[] args)
    {
        // Определить каталоги для входных и выходных файлов
        string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
        string inputFile = Path.Combine(@"YOUR_DOCUMENT_DIRECTORY", "example.webp");

        // Инициализируйте объект Converter, указав путь к файлу WEBP.
        using (Converter converter = new Converter(inputFile))
        {
            // Укажите параметры конвертации в формат DOC
            var convertOptions = new WordProcessingConvertOptions();

            // Конвертировать и сохранить выходной документ
            converter.Convert(outputFolder + "\output.doc", convertOptions);
        }
    }
}
```
В этом коде `Converter` инициализируется вашим файлом WEBP. Мы указываем вывод DOC с помощью `WordProcessingConvertOptions`.
