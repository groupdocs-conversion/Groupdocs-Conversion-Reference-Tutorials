## Руководство по внедрению

### Конвертировать XLSM в DOCX

#### Обзор
Эта функция позволяет преобразовывать файлы Excel Macro-Enabled Workbook (.xlsm) в документы Microsoft Word (.docx), что упрощает обмен документами и их редактирование.

**Шаг 1: Настройка каталогов**
Определите исходный и выходной каталоги:

```csharp
string sourceDirectory = @"YOUR_DOCUMENT_DIRECTORY";
string documentPath = Path.Combine(sourceDirectory, "sample.xlsm");
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputDirectory, "xlsm-converted-to.docx");
```

**Шаг 2: Загрузите и преобразуйте файл XLSM**
Загрузите файл XLSM и преобразуйте его в формат DOCX:

```csharp
using (var converter = new Converter(documentPath))
{
    var options = new WordProcessingConvertOptions();
    converter.Convert(outputFile, options);
}
```
- **Параметры**: `documentPath` — это путь к исходному файлу XLSM. `outputFile` это место, где вы хотите сохранить свой DOCX.
- **Метод Цель**: `converter.Convert()` выполняет преобразование с использованием указанных параметров.

**Совет по устранению неполадок**: Убедитесь, что указаны правильные пути и необходимые разрешения для чтения/записи файлов.
