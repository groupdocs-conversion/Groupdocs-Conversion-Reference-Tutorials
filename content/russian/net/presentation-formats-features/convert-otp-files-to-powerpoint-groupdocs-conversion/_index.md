Чтобы использовать GroupDocs.Conversion, установите пакет через консоль диспетчера пакетов NuGet или .NET CLI:


**\.NET CLI**

### Конвертировать OTP в формат PPT

**Обзор:**
В этом разделе демонстрируется преобразование файла OTP в презентацию PowerPoint с помощью GroupDocs.Conversion.

#### Шаг 1: Укажите выходной каталог и имя файла

Определите, где будет сохранен преобразованный файл:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = System.IO.Path.Combine(outputFolder, "otp-converted-to.ppt");
```

#### Шаг 2: Настройте параметры конвертации

Установите желаемый формат для конвертации с помощью `PresentationConvertOptions`:

```csharp
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY/sample.otp"))
{
    PresentationConvertOptions options = new PresentationConvertOptions { Format = GroupDocs.Conversion.FileTypes.PresentationFileType.Ppt };
    
    // Выполнить преобразование и сохранить по указанному пути выходного файла.
    converter.Convert(outputFile, options);
}
```

**Параметры и методы:**
- `sourceFilePath`: Путь к входному файлу OTP.
- `outputFolder`/`outputFile`: Каталоги для сохранения преобразованных файлов.
- `PresentationConvertOptions`: Задает параметры преобразования, специфичные для формата.
