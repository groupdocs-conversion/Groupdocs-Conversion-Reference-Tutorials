### Конвертировать CSV в презентацию PowerPoint

В этом разделе демонстрируется преобразование данных из формата CSV в слайды PPT с помощью GroupDocs.Conversion.

#### Пошаговый процесс преобразования

**1. Определите пути к файлам**

Укажите каталоги для исходного CSV-файла и выходного PPT-файла:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputFolder = "YOUR_OUTPUT_DIRECTORY";

string sourceFilePath = Path.Combine(documentDirectory, "sample.csv");
string outputFile = Path.Combine(outputFolder, "csv-converted-to.ppt");
```

**2. Загрузите исходный CSV-файл**

Загрузите ваш CSV-файл с помощью GroupDocs.Conversion `Converter` сорт:

```csharp
using (var converter = new Converter(sourceFilePath))
{
    // Перейти к настройке параметров конвертации
}
```

**3. Установите параметры преобразования для PowerPoint**

Настройте параметры для преобразования файла в формат презентации PowerPoint:

```csharp
PresentationConvertOptions options = new PresentationConvertOptions
{
    Format = PresentationFileType.Ppt // Укажите PPT в качестве выходного формата.
};
```

**4. Конвертируйте и сохраните файл**

Выполните преобразование и сохраните результат:

```csharp
converter.Convert(outputFile, options);
Console.WriteLine("Conversion completed successfully.");
```
