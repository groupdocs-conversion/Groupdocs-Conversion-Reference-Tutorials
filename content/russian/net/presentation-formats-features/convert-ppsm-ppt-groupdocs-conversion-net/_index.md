## Настройка GroupDocs.Conversion для .NET

Чтобы использовать GroupDocs.Conversion в своих проектах .NET, установите его через NuGet Package Manager Console или .NET CLI. Вот как:

### Установка с помощью консоли диспетчера пакетов NuGet
Откройте консоль менеджера пакетов и выполните:
```bash
Install-Package GroupDocs.Conversion -Version 25.3.0
```

### Установка с использованием .NET CLI
Выполните эту команду в терминале:
```bash
dotnet add package GroupDocs.Conversion --version 25.3.0
```

#### Этапы получения лицензии
- **Бесплатная пробная версия**: Начните с бесплатной пробной версии, чтобы протестировать функции.

    static void Main(string[] args)

## Руководство по внедрению
Мы разберем это руководство на отдельные функции и шаги, которые помогут вам эффективно конвертировать файлы PPSM в формат PPT.

### Функция: конвертация PPSM в PPT

#### Обзор
Преобразуйте файл .ppsm в более широко поддерживаемый формат .ppt с помощью GroupDocs.Conversion для .NET.

#### Шаг 1: Загрузите исходный файл PPSM
Укажите путь к исходному файлу и выходной каталог:
```csharp
string sourceFilePath = "YOUR_DOCUMENT_DIRECTORY/sample.ppsm"; // Измените это в соответствии с фактическим путем к документу.
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
```

#### Шаг 2: Настройте параметры конвертации
Настройте параметры преобразования для презентации PowerPoint (.ppt):
```csharp
using GroupDocs.Conversion.Options.Convert;

PresentationConvertOptions options = new PresentationConvertOptions
{
    Format = GroupDocs.Conversion.FileTypes.PresentationFileType.Ppt
};
```

#### Шаг 3: Выполните преобразование и сохраните выходной файл.
Выполните процесс преобразования и сохраните результат:
```csharp
string outputFile = Path.Combine(outputFolder, "ppsm-converted-to.ppt");

using (var converter = new Converter(sourceFilePath))
{
    converter.Convert(outputFile, options);
}

Console.WriteLine("Conversion completed successfully.");
```

#### Советы по устранению неполадок
Если у вас возникли проблемы:
- Убедитесь, что путь к исходному файлу правильный и доступный.
- Убедитесь, что выходной каталог существует, или создайте его при необходимости.

### Функция: Конфигурация пути к файлу

#### Обзор
Правильная настройка путей обеспечивает бесперебойную обработку входных документов и сохранение преобразованных файлов.

#### Шаг 1: Определите каталоги
Укажите каталоги для ваших документов:
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY"; // Путь заполнителя для исходных документов
string outputDirectory = "YOUR_OUTPUT_DIRECTORY"; // Путь заполнителя для выходных данных
```

#### Шаг 2: Объедините пути и убедитесь, что выходной каталог существует
```csharp
string sourceFilePath = Path.Combine(documentDirectory, "sample.ppsm");
string outputFile = Path.Combine(outputDirectory, "ppsm-converted-to.ppt");

if (!Directory.Exists(outputDirectory))
{
    Directory.CreateDirectory(outputDirectory);
}
```
