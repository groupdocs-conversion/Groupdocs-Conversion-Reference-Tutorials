После установки необходимых пакетов давайте инициализируем и настроим GroupDocs.Conversion в вашем проекте.

Вот как начать базовую настройку с использованием C#:

```csharp
using System;
using GroupDocs.Conversion;

class Program
{
    static void Main()
```

## Руководство по внедрению

Давайте разобьем процесс на логические шаги для лучшего понимания и реализации.

### Загрузить файл PPTM
**Обзор:** Начните с загрузки файла PowerPoint с помощью GroupDocs.Conversion. Это подготовит его к последующим операциям конвертации.

#### Шаг 1: Определите путь к документу
Укажите путь к вашему файлу .pptm:

```csharp
string documentPath = "YOUR_DOCUMENT_DIRECTORY/Sample.pptm";
```

#### Шаг 2: Инициализация конвертера
Создать `Converter` экземпляр для управления процессом загрузки:

```csharp
using (Converter converter = new Converter(documentPath))
{
    // Файл теперь готов к конвертации.
}
```

### Конвертировать PPTM в PNG
**Обзор:** После загрузки преобразуйте файл в формат PNG, используя специальные параметры конвертации.

#### Шаг 1: Определите выходной каталог и шаблон
Подготовьте выходной каталог и шаблон имени файла:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = System.IO.Path.Combine(outputFolder, "converted-page-{0}.png");
```

#### Шаг 2: Настройка функции потока
Создайте функцию для управления созданием PNG-файлов:

```csharp
Func<SavePageContext, System.IO.Stream> getPageStream = savePageContext =>
{
    return new System.IO.FileStream(System.String.Format(outputFileTemplate, savePageContext.Page), System.IO.FileMode.Create);
};
```

#### Шаг 3: Настройте параметры конвертации
Настройте параметры преобразования изображений для формата PNG:

```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Png };
```

#### Шаг 4: Выполнение преобразования
Выполните процесс конвертации:

```csharp
converter.Convert(getPageStream, options);
```

Это приведет к созданию файлов PNG в указанном вами каталоге.
