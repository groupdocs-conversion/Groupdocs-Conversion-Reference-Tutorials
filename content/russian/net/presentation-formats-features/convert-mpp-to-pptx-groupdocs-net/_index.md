## Руководство по внедрению

### Преобразование MPP в PPTX

Эта функция позволяет преобразовать файл Microsoft Project в презентацию PowerPoint.

#### Шаг 1: Определите выходной путь

Убедитесь, что ваш выходной путь установлен правильно. Используйте заполнители, чтобы определить, где будут сохранены преобразованные файлы:

```csharp
string outputFolder = Path.Combine(@"YOUR_OUTPUT_DIRECTORY");
```

#### Шаг 2: Настройте параметры конвертации

Укажите параметры конвертации для PowerPoint:

```csharp
var loadOptions = new MppLoadOptions();
var convertOptions = new PresentationConvertOptions();

// Загрузите и конвертируйте файл
using (var converter = new Converter("path/to/your/file.mpp", () => loadOptions))
{
    converter.Convert(outputFolder + "/converted.pptx", () => convertOptions);
}
```

**Объяснение:**
- `MppLoadOptions`: Настраивает способ загрузки файлов MPP.
- `PresentationConvertOptions`: Задает параметры преобразования PowerPoint.

#### Шаг 3: Выполнение преобразования

Выполнить преобразование, используя указанные параметры:

```csharp
converter.Convert(outputFolder + "/converted.pptx", () => convertOptions);
```

**Советы по устранению неполадок:**
- Убедитесь, что пути к файлам верны и доступны.
- Проверьте наличие отсутствующих зависимостей или версий библиотек.
