# Конвертируйте PPSX в PPTX с помощью GroupDocs.Conversion для .NET: подробное руководство

## Введение

Конвертация файлов PowerPoint Open XML Slide Show (.ppsx) в стандартные презентации PowerPoint (.pptx) является обычным требованием для многих профессионалов. Это руководство поможет вам освоить процесс конвертации с использованием GroupDocs.Conversion для .NET, обеспечивая плавные переходы между различными платформами и совместимость с более широкой аудиторией.

## Руководство по внедрению

### Конвертировать PPSX в PPTX

#### Обзор
Эта функция преобразует файл слайд-шоу PowerPoint Open XML (.ppsx) в файл презентации PowerPoint Open XML (.pptx), обеспечивая совместимость с большим количеством стандартных приложений.

#### Шаг 1: Определите входные и выходные каталоги
Начните с настройки входных и выходных каталогов:
```csharp
string documentDirectory = \@"YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = Path.Combine(Directory.GetCurrentDirectory(), "YOUR_OUTPUT_DIRECTORY");

if (!Directory.Exists(outputDirectory))
{
    Directory.CreateDirectory(outputDirectory);
}
```
#### Шаг 2: Загрузите и преобразуйте файл
Загрузите исходный файл PPSX и укажите параметры преобразования:
```csharp
using (var converter = new Converter(Path.Combine(documentDirectory, "sample.ppsx")))
{
    var options = new PresentationConvertOptions();
    
    string outputFile = Path.Combine(outputDirectory, "ppsx-converted-to.pptx");
    converter.Convert(outputFile, options);
}
```
**Объяснение**: `Converter` Класс управляет загрузкой и преобразованием файлов. `PresentationConvertOptions` определяет целевой формат как PPTX.

#### Советы по устранению неполадок
- Убедитесь, что путь к исходному файлу указан правильно.
- Проверьте права доступа к каталогу для чтения/записи файлов.
- Проверьте совместимость версий GroupDocs.Conversion.
