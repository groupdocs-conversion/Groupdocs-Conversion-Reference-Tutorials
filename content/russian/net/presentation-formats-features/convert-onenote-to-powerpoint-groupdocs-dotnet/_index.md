## Руководство по внедрению

### Функция: Преобразование OneNote в PowerPoint

Эта функция позволяет конвертировать файлы Microsoft OneNote (.one) в формат PowerPoint Presentation Open XML (.pptx).

#### Шаг 1: Настройте пути к документам

Определите пути для входных и выходных каталогов:
```csharp
string YOUR_DOCUMENT_DIRECTORY = "YOUR_DOCUMENT_DIRECTORY";
string YOUR_OUTPUT_DIRECTORY = "YOUR_OUTPUT_DIRECTORY";
// Убедитесь, что эти пути указаны правильно и соответствуют расположению ваших файлов.
```

#### Шаг 2: Загрузите исходный файл ONE

Загрузите файл .one, используя класс Converter из GroupDocs.Conversion:
```csharp
using (var converter = new Converter(YOUR_DOCUMENT_DIRECTORY + "/sample.one"))
{
    // Объект «конвертер» теперь готов к выполнению задач конвертации.
}
```

#### Шаг 3: Определите параметры преобразования

Настройте параметры преобразования в формат PowerPoint с помощью PresentationConvertOptions:
```csharp
var options = new PresentationConvertOptions();
// Эти параметры определяют, как будет преобразован файл.
```

#### Шаг 4: Выполнение преобразования

Выполните преобразование и сохраните результат как файл .pptx:
```csharp
string outputFile = YOUR_OUTPUT_DIRECTORY + "/one-converted-to.pptx";
converter.Convert(outputFile, options);
// Теперь файл сохранен в указанном вами выходном каталоге.
```

### Функция: Конфигурация пути к файлу

Убедитесь, что ваши каталоги правильно настроены для ввода и вывода.

#### Шаг 1: Настройка выходного каталога

Получите полный путь к месту сохранения преобразованных файлов:
```csharp
string outputFolder = Path.Combine(Directory.GetCurrentDirectory(), YOUR_OUTPUT_DIRECTORY);
if (!Directory.Exists(outputFolder))
{
    Directory.CreateDirectory(outputFolder); // Создайте каталог, если он не существует.
}
```

#### Шаг 2: Определите путь к выходному файлу

Укажите путь, куда вы хотите сохранить преобразованный файл:
```csharp
string outputFile = Path.Combine(outputFolder, "one-converted-to.pptx");
// Убедитесь, что этот путь в выходной папке указан правильно.
```
