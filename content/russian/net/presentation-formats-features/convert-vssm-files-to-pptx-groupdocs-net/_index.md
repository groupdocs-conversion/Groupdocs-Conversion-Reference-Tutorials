## Настройка GroupDocs.Conversion для .NET

Для начала давайте настроим **GroupDocs.Конверсия** в вашем проекте:


### Установка с использованием .NET CLI
```bash

```

После установки вы можете начать использовать его возможности для конвертации различных форматов файлов.

### Базовая инициализация и настройка с помощью C#

Чтобы начать конвертацию файлов, инициализируйте `Converter` класс и укажите ваш входной файл:

```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main()
    {
        string inputFilePath = "sample.vssm";
        string outputFolder = "output";
        string outputFile = System.IO.Path.Combine(outputFolder, "vssm-converted-to.pptx");

        using (var converter = new Converter(inputFilePath))
        {
            var options = new PresentationConvertOptions();
            converter.Convert(outputFile, options);
        }
    }
}
```

Эта функция фокусируется на конвертации файлов VSSM в формат PPTX. Процесс включает загрузку файла VSSM и применение настроек конвертации для вывода его в виде презентации PowerPoint.

   - Инициализировать `PresentationConvertOptions` для преобразования PPTX.
   
3. **Выполнить преобразование**
   - Позвоните `Convert` метод с указанными параметрами для сохранения в виде файла PPTX.

#### Пояснение кода


- **Converter**: Этот класс обрабатывает процесс преобразования. `using` заявление обеспечивает правильное распоряжение ресурсами.
  
- **ПрезентацияКонвертироватьПараметры**: Настраивает параметры, специфичные для преобразования PowerPoint.
  
- **Метод преобразования**: Выполняет преобразование и записывает вывод в файл PPTX, указанный `outputFile`.
