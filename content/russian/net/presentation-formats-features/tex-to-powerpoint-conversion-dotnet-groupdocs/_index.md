Вот как инициализировать GroupDocs.Conversion в вашем приложении C#:
```csharp

using GroupDocs.Conversion;

namespace LaTeXToPPTConversion
{
    class Program
    {
        static void Main(string[] args)
        {
            // Инициализируйте объект Converter, указав путь к вашему .tex-файлу.

            }
        }
    }
}
```
## Руководство по внедрению
Давайте разберем процесс конверсии на понятные и выполнимые шаги.
### Шаг 1: Определите выходной каталог и пути к файлам
Начните с настройки выходного каталога и пути к файлу:
```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY"); // Настроить этот путь
string outputFile = Path.Combine(outputFolder, "tex-converted-to.ppt");
```
Заменять `"YOUR_OUTPUT_DIRECTORY"` с желаемым местом хранения преобразованных презентаций.
### Шаг 2: Загрузите исходный файл LaTeX
Далее загрузите ваш TEX-документ с помощью `Converter` class. Убедитесь, что вы указали правильный путь:
```csharp
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY/sample.tex")) // Настроить этот путь
{
    Console.WriteLine("LaTeX file loaded successfully.");
}
```
Заменять `"YOUR_DOCUMENT_DIRECTORY"` с каталогом, содержащим ваши TEX-файлы.
### Шаг 3: Настройте параметры конвертации
Настройте параметры конвертации для формата PowerPoint:
```csharp
PresentationConvertOptions options = new PresentationConvertOptions
{
    Format = PresentationFileType.Ppt // Целевой формат установлен на PPT
};
```
Эта конфигурация указывает, что вывод должен быть в формате PPT.
### Шаг 4: Выполните преобразование и сохраните результат
Наконец, выполните преобразование и сохраните результат:
```csharp
converter.Convert(outputFile, options);
Console.WriteLine("Conversion completed successfully.");
```
