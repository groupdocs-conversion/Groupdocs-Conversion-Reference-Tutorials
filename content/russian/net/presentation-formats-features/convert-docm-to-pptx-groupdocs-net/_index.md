### Конвертировать формат DOCM в PPTX
#### Обзор:
Теперь давайте преобразуем загруженный файл DOCM в презентацию PowerPoint (формат PPTX).
```csharp
using System.IO;
using GroupDocs.Conversion.Options.Convert;

// Определить выходной каталог и путь к файлу
class Path
{
    public static string Combine(string path1, string path2)
    {
        // Логика объединения путей...
    }
}
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "docm-converted-to.pptx");

// Настройте параметры конвертации для формата PPTX
var options = new PresentationConvertOptions();

// Выполнить преобразование
converter.Convert(outputFile, options);
```
- **Параметры**:
  - `outputFile`: Путь, по которому будет сохранен преобразованный файл.
  - `options`: Настраивает способ преобразования содержимого DOCM в формат PPTX.

#### Основные параметры конфигурации:
- **ПрезентацияКонвертироватьПараметры**: Настройте параметры представления во время конвертации.

#### Советы по устранению неполадок:
- Убедитесь, что пути определены правильно, чтобы избежать ошибок «файл не найден».
- Проверьте права на запись для выходного каталога.
