## Руководство по внедрению

### Функция: Преобразование BMP в PPTX

Эта функция предназначена для преобразования файлов BMP в презентации PowerPoint с использованием мощных возможностей GroupDocs.Conversion.

#### Загрузка и конвертация файла

1. **Определить выходные пути**
   
   Начните с настройки путей к файлам для входных (BMP) и выходных (PPTX) файлов:
   ```csharp
   string outputFolder = "YOUR_OUTPUT_DIRECTORY/"; // Определите, где сохранить преобразованный PPTX.
   string sourceBmpPath = "YOUR_DOCUMENT_DIRECTORY/sample.bmp"; // Путь к файлу изображения BMP.
   ```

2. **Инициализировать преобразователь**
   
   Используйте GroupDocs.Conversion для загрузки и преобразования файла BMP:
   ```csharp
   using (var converter = new Converter(sourceBmpPath))
   {
       var options = new PresentationConvertOptions(); // Варианты конвертации в формат PPTX.
       string outputFile = System.IO.Path.Combine(outputFolder, "bmp-converted-to.pptx");
       converter.Convert(outputFile, options); // Выполните преобразование и сохраните как PPTX.
   }
   ```

#### Объяснение параметров

- **`Converter`:** Этот класс загружает ваш файл BMP для обработки. Убедитесь, что пути установлены правильно.
- **`PresentationConvertOptions`:** Настраивает способ преобразования файла BMP в презентацию PowerPoint.
