## Руководство по внедрению

### Функция: конвертация PPS в PNG
Эта функция позволяет конвертировать слайд-шоу PowerPoint (.pps) в формат Portable Network Graphic (.png).

#### Шаг 1: Определите пути к файлам
Начните с настройки пути к входному файлу и выходному каталогу:

```csharp
string documentPath = "YOUR_DOCUMENT_DIRECTORY/Sample.pps";
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
```

Замените заполнители вашими фактическими путями.

#### Шаг 2: Настройка выходного шаблона
Определите шаблон для наименования преобразованного PNG-файла каждой страницы:

```csharp
string outputFileTemplate = System.IO.Path.Combine(outputFolder, "converted-page-{0}.png");
```

#### Шаг 3: Определите функцию потока страниц
Создайте функцию для обработки выходного потока для каждой преобразованной страницы. Это гарантирует, что каждый слайд будет сохранен как отдельный файл PNG.

```csharp
Func<SavePageContext, System.IO.Stream> getPageStream = savePageContext =>
    new System.IO.FileStream(System.String.Format(outputFileTemplate, savePageContext.Page), System.IO.FileMode.Create);
```

#### Шаг 4: Настройте параметры конвертации
Настройте параметры преобразования для формата изображения:

```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Png };
```

#### Шаг 5: Выполнение преобразования
Выполните фактическое преобразование, вызвав `Convert` метод:

```csharp
using (Converter converter = new Converter(documentPath))
{
    // Конвертировать PPS в формат PNG
    converter.Convert(getPageStream, options);
}
```

**Совет по устранению неполадок:** Убедитесь, что пути указаны правильно и у вас есть права на запись в выходной каталог.
