## Настройка GroupDocs.Conversion для .NET

Чтобы преобразовать файлы CDR, установите **GroupDocs.Конверсия** библиотека.

### Установка через консоль диспетчера пакетов NuGet
Откройте консоль диспетчера пакетов NuGet в Visual Studio и выполните:
```bash
dotnet add package GroupDocs.Conversion --version 25.3.0
```

### Этапы получения лицензии

### Базовая инициализация и настройка
```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main()
    {
        // Определите путь к выходному каталогу и местонахождение файла CDR.
        string outputFolder = "YOUR_OUTPUT_DIRECTORY";
        string sampleCdrPath = "YOUR_DOCUMENT_DIRECTORY\sample.cdr"; // Замените на ваш реальный путь

        string outputFile = Path.Combine(outputFolder, "cdr-converted-to.pptx");

        // Инициализируйте объект-конвертер с исходным CDR-файлом
        using (var converter = new Converter(sampleCdrPath))
        {
            var options = new PresentationConvertOptions();
            
            // Конвертируйте и сохраните выходной файл PPTX
            converter.Convert(outputFile, options);
        }
    }
}
```

## Руководство по внедрению

Вот как преобразовать файлы CDR в презентации PowerPoint.

### Загрузка исходного файла
Начните с загрузки исходного файла CDR с помощью `Converter` сорт:
```csharp
using (var converter = new Converter(sampleCdrPath))
{
    // Код конвертации здесь...
}
```
**Почему**: `Converter` объект инициализируется с указанием пути к файлу, что позволяет выполнять последующие операции преобразования.

### Определение параметров преобразования
Далее укажите параметры преобразования формата PowerPoint, используя `PresentationConvertOptions`:
```csharp
var options = new PresentationConvertOptions();
```
**Почему**: Эти параметры указывают, что вы хотите преобразовать свой CDR в файл PPTX.

### Выполнение преобразования
Наконец, выполните преобразование и сохраните результат:
```csharp
converter.Convert(outputFile, options);
```
**Почему**Этот метод выполняет фактическое преобразование на основе определенных параметров и сохраняет его по указанному пути.
