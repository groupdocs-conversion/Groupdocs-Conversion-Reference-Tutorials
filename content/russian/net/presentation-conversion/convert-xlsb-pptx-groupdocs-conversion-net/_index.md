### Конвертировать XLSB в PPTX

#### Обзор
Эта функция позволяет автоматизировать преобразование двоичных файлов Excel (XLSB) в презентации PowerPoint (PPTX), экономя время и сокращая количество ручных ошибок.

##### Шаг 1: Определите пути
Начните с настройки выходной папки и путей к файлам, где будет сохранена преобразованная презентация. Замените заполнители фактическими путями к каталогам в вашей среде.

```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", "");
string outputFile = Path.Combine(outputFolder, "xlsb-converted-to.pptx");
```

##### Шаг 2: Загрузка исходного файла
Загрузите исходный файл XLSB с помощью класса Converter. Убедитесь, что вы указали правильный каталог, в котором находится исходный файл.

```csharp
using (var converter = new GroupDocs.Conversion.Converter(Path.Combine("YOUR_DOCUMENT_DIRECTORY", "SAMPLE_XLSB")))
{
    // Продолжайте шаги преобразования
}
```

##### Шаг 3: Инициализация параметров преобразования
Инициализировать `PresentationConvertOptions` чтобы указать, что вы конвертируете в формат PowerPoint.

```csharp
var options = new PresentationConvertOptions();
```

##### Шаг 4: Выполнение преобразования
Выполните преобразование и сохраните файл PPTX по указанному вами пути.

```csharp
converter.Convert(outputFile, options);
```
