## Руководство по внедрению

### Загрузка PST-файлов
#### Обзор
Загрузка файла PST — это начальный шаг в процессе конвертации. Этот раздел поможет вам подготовить файлы к конвертации.

#### Пошаговая реализация
**1. Определите тип файла**
   - Определите, является ли исходный файл OST или PST, и соответствующим образом задайте параметры загрузки.

```csharp
string sourceFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.pst");
var converterOptions = new LoadOptions();

if (Path.GetExtension(sourceFilePath).Equals(".ost", StringComparison.OrdinalIgnoreCase)) {
    // Используйте специальные конфигурации для OST-файлов.
    converterOptions = new PersonalStorageLoadOptions();
}
```

**2. Инициализация конвертера**
   - Создать `Converter` экземпляр с использованием пути к исходному файлу и параметров загрузки.

```csharp
using (var converter = new GroupDocs.Conversion.Converter(sourceFilePath, converterOptions)) {
    // Готовы к выполнению конвертации.
}
```

### Конвертация PST в PPTX
#### Обзор
После загрузки преобразуйте файлы PST в презентабельный формат PPTX.

#### Пошаговая реализация
**1. Установите параметры конвертации**
   - Определите выходной каталог и шаблон именования файлов.

```csharp
string outputDirectory = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFilePattern = Path.Combine(outputDirectory, "pst-converted-{0}-to.pptx");
var options = new PresentationConvertOptions();
int counter = 1;
```

**2. Выполнить преобразование**
   - Используйте `Converter` экземпляр для преобразования и сохранения файла.

```csharp
using (var converter = new GroupDocs.Conversion.Converter(sourceFilePath)) {
    converter.Convert(
        savePath => new FileStream(string.Format(outputFilePattern, counter++), FileMode.Create),
        options
    );
}
```

### Советы по устранению неполадок
- **Файл не найден**: Убедитесь, что путь к исходному файлу указан правильно.
- **Проблемы с разрешением**Убедитесь, что ваше приложение имеет разрешения на чтение/запись для задействованных каталогов.
