## Руководство по внедрению

### Функция преобразования анимированного GIF в PPTX

Эта функция позволяет преобразовать GIF-файл в формат презентации PowerPoint. Давайте разберем необходимые шаги.

#### Шаг 1: Определите пути к файлам и настройте конвертер

Сначала определите входные и выходные каталоги:

```csharp
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

string outputFolder = "YOUR_OUTPUT_DIRECTORY"; // Определите путь к выходному каталогу
string inputFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.gif"); // Путь к входному GIF-файлу
string outputFile = Path.Combine(outputFolder, "gif-converted-to.pptx"); // Определите имя выходного файла PPTX

// Загрузите исходный GIF-файл
using (var converter = new Converter(inputFile))
{
    var options = new PresentationConvertOptions(); // Установить параметры преобразования для формата PowerPoint
}
```

#### Шаг 2: Выполнение преобразования и сохранение вывода

В блоке using выполните преобразование:

```csharp
converter.Convert(outputFile, options); // Конвертируйте и сохраните файл PPTX
Console.WriteLine("Conversion Successful!");
```

Для эффективного управления путями создайте служебный класс для хранения констант каталога.
