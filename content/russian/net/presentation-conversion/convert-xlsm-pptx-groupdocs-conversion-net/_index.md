### Базовая инициализация и настройка

Вот как можно инициализировать и настроить GroupDocs.Conversion в проекте C#:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

// Инициализируйте объект Converter, указав путь к файлу XLSM.
string inputFile = "path/to/your/sample.xlsm";
string outputFolder = "path/to/output/directory";

// Создать экземпляр класса Converter
using (var converter = new Converter(inputFile))
{
    // Определить параметры преобразования для формата PPTX
    var options = new PresentationConvertOptions();
    
    // Укажите путь к выходному файлу
    string outputFile = Path.Combine(outputFolder, "xlsm-converted-to.pptx");
    
    // Выполните преобразование и сохраните полученный файл PPTX.
    converter.Convert(outputFile, options);
}
```

### Функция: конвертация XLSM в PPTX

**Обзор**

Эта функция автоматизирует преобразование таблиц Excel с поддержкой макросов (XLSM) в презентации PowerPoint, оптимизируя процесс управления документами.

#### 1. Загрузите исходный файл XLSM

Начните с загрузки исходного файла с помощью `Converter` сорт:

```csharp
string inputFile = "path/to/your/sample.xlsm";
using (var converter = new Converter(inputFile))
{
    // Логика преобразования идет здесь
}
```

*Почему:* Загрузка файла инициализирует процесс преобразования, позволяя применять различные преобразования и параметры вывода.

#### 2. Инициализация параметров преобразования

Определите конкретные параметры преобразования, используя `PresentationConvertOptions`:

```csharp
var options = new PresentationConvertOptions();
```

*Почему:* На этом этапе настраивается, как ваш документ будет преобразован в формат PPTX, и гарантируются все необходимые настройки.

#### 3. Сохраните преобразованный файл

Наконец, сохраните преобразованный файл в желаемом месте:

```csharp
string outputFile = Path.Combine(outputFolder, "xlsm-converted-to.pptx");
converter.Convert(outputFile, options);
```

*Почему:* На этом важном этапе выходной файл записывается в указанный каталог, завершая процесс преобразования.

**Советы по устранению неполадок**

- **Отсутствующие файлы:** Убедитесь, что все пути к файлам верны и доступны.
