### Базовая инициализация
Чтобы настроить GroupDocs.Conversion в приложении C#:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

// Определите путь к выходному каталогу как заполнитель
string outputFolder = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "Output");

// Инициализируйте обработчик преобразования с помощью входного файла
using (var converter = new Converter("input.ifc"))
{
    var options = new PresentationConvertOptions();
    
    // Конвертировать и сохранить презентацию
    converter.Convert(() => File.Create(Path.Combine(outputFolder, "output.pptx")), options);
}
```

Здесь мы инициализируем `Converter` объект с файлом IFC в качестве входных данных. Мы устанавливаем параметры преобразования для формата PowerPoint, используя `PresentationConvertOptions`. Этот фрагмент кода демонстрирует простоту преобразования файлов с помощью GroupDocs.Conversion.
