Чтобы начать конвертировать документы, вам нужно установить библиотеку GroupDocs.Conversion. Вот как это можно сделать:


**.NET CLI:**

#### Базовая инициализация и настройка

Давайте инициализируем GroupDocs.Conversion в приложении C#. Убедитесь, что вы настроили среду проекта, как описано выше:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

// Определите путь к исходному файлу PPTM
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

string sourceFilePath = Path.Combine(documentDirectory, "sample.pptm");
string outputFolder = Path.Combine(outputDirectory, "ConvertedFiles");

// Убедитесь, что выходной каталог существует
Directory.CreateDirectory(outputFolder);

// Определите путь к выходному файлу XLSX
string outputFile = Path.Combine(outputFolder, "pptm-converted-to.xlsx");
```

**2. Настройка параметров конвертации**

Инициализируйте параметры преобразования для формата электронной таблицы:


**3. Выполнение преобразования**

```csharp
converter.Convert(outputFile, options);
```
