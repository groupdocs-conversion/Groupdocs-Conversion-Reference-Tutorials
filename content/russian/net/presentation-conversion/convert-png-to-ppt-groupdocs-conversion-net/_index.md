## Руководство по внедрению

Теперь давайте рассмотрим, как шаг за шагом преобразовать изображение PNG в формат PPT.

### Функция: конвертация PNG в PPT

**Обзор**: эта функция позволяет конвертировать файлы PNG в презентации PowerPoint, сохраняя визуальные материалы в динамическом формате.

#### Шаг 1: Определите входные и выходные пути

Укажите пути для входного PNG-файла и желаемый выходной каталог:

```csharp
string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.png"); // Заменить на фактический путь
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", ".\" );
string outputFile = Path.Combine(outputFolder, "png-converted-to.ppt");
```

**Объяснение**: `inputFilePath` должен указывать на файл PNG для преобразования. При необходимости отрегулируйте пути.

#### Шаг 2: Загрузите и преобразуйте PNG-файл

Загрузите исходный PNG с помощью GroupDocs.Conversion `Converter` класс, затем настройте параметры для формата PowerPoint:

```csharp
using (var converter = new Converter(inputFilePath))
{
    PresentationConvertOptions options = new PresentationConvertOptions 
    {
        Format = PresentationFileType.Ppt // Целевой формат PPT
    };
    
    // Выполните преобразование и сохраните выходной файл.
    converter.Convert(outputFile, options);
}
```

**Объяснение**: `Converter` класс загружает ваш PNG. Установка `Format` к `PresentationFileType.Ppt` указывает презентацию PowerPoint в качестве целевого формата.
