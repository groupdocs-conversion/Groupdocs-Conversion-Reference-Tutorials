## Руководство по внедрению

### Функция 1: Преобразование PPSX в DOC

Узнайте, как преобразовать файл слайд-шоу PowerPoint Open XML (.ppsx) в документ Microsoft Word (.doc).

#### Шаг 1: Определите свои пути
Настройте пути для входных и выходных файлов.
```csharp
string documentPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.ppsx");
string outputFolder = GetOutputDirectoryPath();
string outputFile = Path.Combine(outputFolder, "ppsx-converted-to.doc");
```

#### Шаг 2: Загрузите и преобразуйте файл PPSX
Используйте GroupDocs.Conversion для загрузки файла PPSX и укажите параметры преобразования для форматов текстового процессора.
```csharp
using (var converter = new Converter(documentPath))
{
    // Укажите параметры конвертации для формата DOC.
    var options = new WordProcessingConvertOptions
    {
        Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc
    };
    
    // Конвертируйте и сохраните файл PPSX как файл формата DOC.
    converter.Convert(outputFile, options);
}
```

### Функция 2: Определение пути к выходному каталогу
Динамически управляйте выходными путями, определяя функцию для управления созданием каталогов.
```csharp
string GetOutputDirectoryPath()
{
    string baseOutputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
    
    // Создайте каталог, если он не существует.
    if (!Directory.Exists(baseOutputFolder))
    {
        Directory.CreateDirectory(baseOutputFolder);
    }
    
    return baseOutputFolder;
}
```
