## Руководство по внедрению

### Функция 1: Конвертация RAR в PDF

#### Обзор
Эта функция позволяет преобразовать файл RAR в формат PDF. Это полезно для сценариев, где вам нужно распространять или архивировать документы в более доступном формате.

#### Пошаговая реализация

**Шаг 3.1: Загрузите файл RAR**

Начните с загрузки вашего файла RAR с помощью `FluentConverter`. Заменять `"YOUR_DOCUMENT_DIRECTORY\sample.rar"` с фактическим путем к вашему RAR-файлу:

```csharp
using System.IO;
using GroupDocs.Conversion.FileTypes;
using GroupDocs.Conversion.Options.Convert;

// Загрузите и конвертируйте файл RAR в PDF
var convertedStream = FluentConverter.Load(@"YOUR_DOCUMENT_DIRECTORY\sample.rar")
    .ConvertTo((SaveContext p) => new MemoryStream())
    .WithOptions(new PdfConvertOptions());
```

**Шаг 3.2: Настройка параметров PDF**

The `PdfConvertOptions` класс позволяет вам настраивать процесс конвертации. Вы можете указать дополнительные настройки, такие как размер страницы, поля и многое другое, если это необходимо.

### Функция 2: Сжатие преобразованного файла в ZIP-архив

#### Обзор
После преобразования файла RAR в PDF сжатие выходных данных в формат ZIP упрощает хранение и обмен ими.

#### Пошаговая реализация

**Шаг 3.3: Определите выходную папку**

Укажите каталог, в котором будет сохранен сжатый ZIP-файл:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
```

**Шаг 3.4: Сжатие в формат ZIP**

Использовать `FluentConverter` снова, на этот раз для сжатия преобразованного PDF-файла в ZIP-архив.

```csharp
using GroupDocs.Conversion.Options.Convert;

var compressedStream = FluentConverter.Load(@"YOUR_DOCUMENT_DIRECTORY\sample.rar")
    .ConvertTo((SaveContext p) => new MemoryStream())
    .Compress(new CompressionConvertOptions { Format = CompressionFileType.Zip })
    .OnCompressionCompleted(compressedStream =>
    {
        using (var fs = new FileStream(Path.Combine(outputFolder, "converted.zip"), FileMode.Create))
        {
            compressedStream.CopyTo(fs);
        }
    });
```

**Шаг 3.5: Сохраните сжатый файл**

Метод обратного вызова `OnCompressionCompleted` гарантирует, что ZIP-файл будет сохранен в указанном вами выходном каталоге.

### Советы по устранению неполадок

- Убедитесь, что пути для входных и выходных файлов заданы правильно.
- Проверьте наличие достаточных прав на чтение/запись в указанных каталогах.
