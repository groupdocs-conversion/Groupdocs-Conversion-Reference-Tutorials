# Учебники по сжатию и обработке архивов для GroupDocs.Conversion .NET

Узнайте, как обрабатывать форматы архивов с помощью наших руководств GroupDocs.Conversion для разработчиков .NET. Эти подробные руководства научат вас, как преобразовывать сжатые форматы, такие как RAR, в другие типы документов, извлекать содержимое из архивов, сжимать результаты преобразования, указывать файлы для извлечения из архивов, обрабатывать защищенные паролем архивы, управлять многотомными архивами и реализовывать выборочное извлечение файлов. Каждое руководство содержит практические примеры кода C# для работы со сжатыми файлами, помогая вам создавать приложения, которые могут эффективно получать доступ и преобразовывать содержимое, хранящееся в различных форматах архивов.
