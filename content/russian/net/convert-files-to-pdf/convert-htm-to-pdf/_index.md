## Импорт пространств имен
Прежде чем углубляться в процесс преобразования, обязательно импортируйте необходимые пространства имен для доступа к функциям GroupDocs.Conversion в вашем коде C#:
```csharp
using System;
using System.IO;
using GroupDocs.Conversion.Options.Convert;
```

Чтобы преобразовать веб-страницы HTM в PDF с помощью GroupDocs.Conversion для .NET, выполните следующие действия:
## Шаг 1: Укажите выходную папку и имя файла
Определите выходную папку, в которую будет сохранен преобразованный PDF-файл, а также желаемое имя файла:
```csharp
string outputFolder = "Your Document Directory";
string outputFile = Path.Combine(outputFolder, "htm-converted-to.pdf");
```
## Шаг 2: Загрузите исходный HTM-файл
Используйте `Converter` класс из библиотеки GroupDocs.Conversion для загрузки исходного HTM-файла:
```csharp
using (var converter = new GroupDocs.Conversion.Converter(Constants.SAMPLE_HTM))
{
    // Логика преобразования будет реализована здесь.
}
```
## Шаг 3: Настройте параметры конвертации
Укажите параметры конвертации, в данном случае, `PdfConvertOptions`, чтобы настроить процесс конвертации:
```csharp
var options = new PdfConvertOptions();
```
## Шаг 4: Выполнение преобразования
Вызовите `Convert` Метод `Converter` класс, передающий путь к выходному файлу и параметры преобразования:
```csharp
converter.Convert(outputFile, options);
```
## Шаг 5: Отображение сообщения об успешном завершении
После успешного преобразования отобразите сообщение о завершении и местоположении преобразованного PDF-файла:
```csharp
Console.WriteLine("\nConversion to PDF completed successfully. \nCheck output in {0}", outputFolder);
```
