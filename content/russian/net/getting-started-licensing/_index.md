# GroupDocs.Conversion. Руководства по началу работы и лицензированию для разработчиков .NET

Наши руководства для новичков покажут, как настроить среду разработки, инициализировать библиотеку GroupDocs.Conversion, настроить лицензию из файла или потока, настроить измеренное лицензирование и реализовать базовые операции преобразования с использованием как стандартных, так и гибких API.
