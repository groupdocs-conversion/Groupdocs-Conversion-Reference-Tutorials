## Руководство по внедрению: определение возможных форматов документов
Теперь, когда вы все настроили, давайте реализуем функцию определения возможных конверсий.
### Обзор функций
Функция «Получить возможные преобразования» позволяет вам определить все форматы, в которые может быть преобразован документ. Это важно для приложений, которым требуются гибкие решения для управления документами.
#### Шаг 1: Определите путь к документу
Начните с указания пути к вашему документу:
```csharp
string documentPath = @"YOUR_DOCUMENT_DIRECTORY\sample.docx";
```
#### Шаг 2: Инициализация класса преобразователя
Инициализируйте новый экземпляр `Converter` класс с определенным путем:
```csharp
using (Converter converter = new Converter(documentPath))
{
    // Дальнейшая обработка будет производиться здесь.
}
```
#### Шаг 3: Получите возможные конверсии
Используйте `GetPossibleConversions` Метод получения возможностей преобразования для вашего документа:
```csharp
PossibleConversions conversions = converter.GetPossibleConversions();
```
#### Шаг 4: Повторите и отобразите сведения о конверсии
Пройдитесь по каждой возможности преобразования, определяя, является ли она первичной или вторичной. Затем отобразите результаты:
```csharp
foreach (var conversion in conversions.All)
{
    string conversionType = conversion.IsPrimary ? "primary" : "secondary";
    Console.WriteLine($"\t {conversion.Format} as {conversionType} conversion.");
}
```
