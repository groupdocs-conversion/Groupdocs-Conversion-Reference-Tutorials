
## Руководство по внедрению
### Шаг 1: Укажите пути к файлам и макеты
Настройте пути для входного файла САПР и выходного PDF-файла, указав, какие макеты вы хотите преобразовать:
```java
String sourceFilePath = "YOUR_DOCUMENT_DIRECTORY/sample.dwg";
String convertedFile = "YOUR_OUTPUT_DIRECTORY/ConvertCadAndSpecifyLayouts.pdf";

// Укажите желаемые названия макетов
cadLoadOptions loadOptions = new CadLoadOptions();
loadOptions.setLayoutNames(new String[] { "Layout1", "Layout3" });
```
### Шаг 2: Инициализация конвертера
Инициализируйте `Converter` класс с указанием пути к файлу и параметров загрузки:
```java
Converter converter = new Converter(sourceFilePath, () -> loadOptions);
```
Это позволяет указать, какие макеты следует включить в преобразование.
### Шаг 3: Задайте параметры конвертации
Настройте параметры преобразования PDF с помощью `PdfConvertOptions`:
```java
PdfConvertOptions convertOptions = new PdfConvertOptions();
```
Эти параметры позволяют выполнять дальнейшую настройку, например, задавать DPI или конкретные диапазоны страниц.
### Шаг 4: Выполнение преобразования
Выполните процесс преобразования, вызвав `convert` метод:
```java
converter.convert(convertedFile, convertOptions);
```
Это создаст PDF-файл, содержащий только указанные макеты из вашего документа САПР.
