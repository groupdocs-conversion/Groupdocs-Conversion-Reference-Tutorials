# Учебники по конвертации форматов электронной почты для GroupDocs.Conversion Java

Освойте преобразование файлов электронной почты с помощью наших подробных руководств GroupDocs.Conversion для разработчиков Java. Эти пошаговые руководства показывают, как преобразовывать сообщения электронной почты в форматы документов, извлекать вложения электронной почты, сохранять метаданные электронной почты, обрабатывать заголовки и тела электронной почты и реализовывать операции пакетного преобразования электронной почты. Каждое руководство содержит практические примеры кода Java для работы с архивами и сообщениями электронной почты, помогая вам создавать приложения, которые могут эффективно обрабатывать и преобразовывать сообщения электронной почты в различные форматы документов.
