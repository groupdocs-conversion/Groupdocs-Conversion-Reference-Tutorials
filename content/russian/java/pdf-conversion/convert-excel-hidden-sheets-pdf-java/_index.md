# Одна страница на лист: Конвертация скрытых листов Excel в PDF (Java)

Преобразование книги Excel в PDF с сохранением всех листов — включая скрытые — может быть сложной задачей. В этом руководстве вы узнаете, как выполнять конвертацию **одна страница на лист** с помощью **GroupDocs.Conversion for Java**, чтобы ни одна информация не была утеряна. Мы пройдём через настройку, конфигурацию и предоставим точный код, а также рассмотрим реальные сценарии, где такой подход проявляет себя наилучшим образом.

## Быстрые ответы
- **Можно ли включить скрытые листы?** Да — установите `setShowHiddenSheets(true)`.
- **Сколько страниц PDF создаётся?** Одна страница на лист, когда используется `setOnePagePerSheet(true)`.
- **Какая версия Java требуется?** JDK 8 или выше.
- **Нужна ли лицензия?** Бесплатная пробная версия подходит для тестирования; для продакшн‑использования требуется коммерческая лицензия.
- **Является ли Maven единственным инструментом сборки?** В примере используется Maven, но Gradle можно применять аналогично.

## Что означает «одна страница на лист»?
Опция **одна страница на лист** указывает конвертеру отрисовывать каждый лист Excel на отдельной странице PDF. Такой макет идеален для отчётов, аудитов и любых ситуаций, когда требуется чёткое постраничное представление исходной книги.

## Руководство по реализации

Ниже приведён полностью готовый к запуску Java‑код, который конвертирует файл Excel — включая скрытые листы — в PDF, где каждый лист отображается на отдельной странице.

### Шаг 1: Укажите путь к исходному документу
Укажите конвертеру путь к книге Excel, содержащей скрытые листы.

```java
String sourceDocumentPath = "YOUR_DOCUMENT_DIRECTORY/SAMPLE_XLSX_WITH_HIDDEN_SHEET";
```

### Шаг 2: Настройте параметры загрузки
Включите обработку скрытых листов и макет «одна страница на лист».

```java
SpreadsheetsLoadOptions loadOptions = new SpreadsheetsLoadOptions();
loadOptions.setShowHiddenSheets(true); // Include hidden sheets
loadOptions.setOnePagePerSheet(true);   // One page per sheet in PDF output
```

### Шаг 3: Инициализируйте конвертер
Создайте экземпляр `Converter`, указав путь к источнику и параметры загрузки.

```java
Converter converter = new Converter(sourceDocumentPath, () -> loadOptions);
```

### Шаг 4: Настройте параметры конвертации
Подготовьте конфигурацию конвертации в PDF.

```java
PdfConvertOptions convertOptions = new PdfConvertOptions();
```

### Шаг 5: Выполните конвертацию
Запустите процесс конвертации и запишите PDF в нужное место.

```java
String outputFilePath = "YOUR_OUTPUT_DIRECTORY/ConvertSpreadsheetWithHiddenSheetsIncluded.pdf";
converter.convert(outputFilePath, convertOptions);
```

#### Краткое резюме конфигурации
- `setShowHiddenSheets(true)`: Делает скрытые листы видимыми для конвертера.
- `setOnePagePerSheet(true)`: Обеспечивает отдельную страницу PDF для каждого листа.

#### Советы по устранению неполадок
- **Ошибки «файл не найден»:** Проверьте абсолютный или относительный путь, который вы указали.
- **Конфликты зависимостей:** Убедитесь, что координаты Maven соответствуют установленной версии.
- **Проблемы с памятью при больших книгах:** Увеличьте размер кучи JVM (`-Xmx`), если возникнет `OutOfMemoryError`.
