## Руководство по внедрению

Теперь, когда вы настроили GroupDocs.Conversion, давайте реализуем функцию преобразования электронных таблиц Excel, включая скрытые листы, в файлы PDF.

### Обзор функций
Эта функция позволяет вам включать все листы из файла Excel в вывод PDF, даже те, которые скрыты при обычном использовании. Это особенно полезно для комплексных отчетов и документирования, где каждая деталь имеет значение.

#### Шаг 1: Определите путь к исходному документу
Укажите путь к исходной электронной таблице, содержащей скрытые листы.

```java
String sourceDocumentPath = "YOUR_DOCUMENT_DIRECTORY/SAMPLE_XLSX_WITH_HIDDEN_SHEET";
```

#### Шаг 2: Настройте параметры загрузки
Здесь вы можете настроить параметры загрузки, чтобы обеспечить включение скрытых листов:

```java
SpreadsheetsLoadOptions loadOptions = new SpreadsheetsLoadOptions();
loadOptions.setShowHiddenSheets(true); // Включить скрытые листы
loadOptions.setOnePagePerSheet(true);   // Одна страница на листе в PDF-файле
```

#### Шаг 3: Инициализация конвертера
Создайте экземпляр конвертера с указанным вами исходным документом и параметрами загрузки.

```java
Converter converter = new Converter(sourceDocumentPath, () -> loadOptions);
```

#### Шаг 4: Настройте параметры конвертации
Настройте параметры преобразования для вывода в PDF:

```java
PdfConvertOptions convertOptions = new PdfConvertOptions();
```

#### Шаг 5: Выполнение преобразования
Выполните процесс конвертации и сохраните результат в нужном вам месте.

```java
String outputFilePath = "YOUR_OUTPUT_DIRECTORY/ConvertSpreadsheetWithHiddenSheetsIncluded.pdf";
converter.convert(outputFilePath, convertOptions);
```

### Основные параметры конфигурации:
- `setShowHiddenSheets(true)`: Обеспечивает обработку скрытых листов.
- `setOnePagePerSheet(true)`: Преобразует каждый лист в отдельную страницу PDF.

#### Советы по устранению неполадок:
- Убедитесь, что путь к исходному файлу указан правильно, чтобы избежать ошибок «файл не найден».
- Проверьте правильность версий в настройках Maven, если у вас возникли проблемы с зависимостями.
