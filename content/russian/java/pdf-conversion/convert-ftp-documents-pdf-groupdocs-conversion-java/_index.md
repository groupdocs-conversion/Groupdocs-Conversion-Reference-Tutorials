### Функция: конвертация документа из FTP в PDF
#### Обзор
Эта функция демонстрирует, как загрузить документ с FTP-сервера и преобразовать его в PDF-файл с помощью GroupDocs.Conversion для Java.
##### Шаг 1: Извлечение файла с FTP-сервера
Для извлечения файлов мы будем использовать Apache Commons Net. Подключитесь к вашему FTP-серверу, перейдите в нужный каталог и извлеките документ:

```java
private static InputStream getFileFromFtp(String server, String dirname, String fileName) throws Exception {
    FTPClient client = new FTPClient();
    
    // Подключитесь к FTP-серверу
    client.connect(server);
    
    // Войдите в систему, используя свои учетные данные (замените «анонимный» и «» при необходимости)
    client.login("username", "password");
    
    // Изменить рабочий каталог на сервере
    client.changeWorkingDirectory(dirname);
    
    // Извлечь файл и вернуть его InputStream
    return client.retrieveFileStream(fileName);
}
```
Этот метод подключается к FTP-серверу, входит в систему с указанными учетными данными, меняет каталог и извлекает файл как `InputStream`.
##### Шаг 2: Преобразование документа в PDF
Получив файл, используйте GroupDocs.Conversion, чтобы преобразовать его в PDF:

```java
public static void run() {
    String server = "127.0.0.1"; // Адрес FTP-сервера
    String convertedFile = YOUR_OUTPUT_DIRECTORY + "/LoadDocumentFromFtp.pdf";
    String dirname = "pub"; // Каталог на FTP-сервере
    String fileName = "sample.docx"; // Файл для извлечения и преобразования

    try {
        // Инициализируйте Converter с помощью лямбда-функции для извлечения файла с FTP
        Converter converter = new Converter(() -> getFileFromFtp(server, dirname, fileName));
        
        // Установить параметры преобразования PDF
        PdfConvertOptions options = new PdfConvertOptions();
        
        // Конвертировать документ и сохранить как PDF
        converter.convert(convertedFile, options);
    } catch (Exception e) {
        throw new RuntimeException(e.getMessage());
    }
}
```
Этот фрагмент кода инициализирует `Converter` объект с потоком файлов FTP и устанавливает параметры преобразования PDF по умолчанию.
