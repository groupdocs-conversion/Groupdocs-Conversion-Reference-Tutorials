## Почему стоит выбрать GroupDocs.Conversion для Java?

* **Broad format support** – Обрабатывает Word, Excel, CSV, ZIP, email и многие другие форматы, предоставляя универсальное решение для всех ваших потребностей в конвертации.  
* **High fidelity** – Сохраняет макет, шрифты и сложные элементы (таблицы, диаграммы, аннотации), поэтому полученный PDF выглядит точно так же, как исходный.  
* **Performance‑oriented** – Оптимизированные API позволяют конвертировать большие партии или отдельные файлы с минимальным потреблением памяти — идеально для облачных сервисов.  
* **Security‑ready** – Встроенные параметры для работы с PDF, защищённым паролем, удаления вложенных файлов и безопасного извлечения метаданных.

## Быстрый старт

1. **Add the Maven dependency** (or Gradle equivalent) for GroupDocs.Conversion.. **Instantiate the `Converter`** with your license key.  
3. **Choose the source format** (e.g., `.docx`, `.xlsx`, `.csv`).  
4. **Configure `PdfConvertOptions`** – set page size, font substitution, or password handling.  
5. **Call `convert`** and write the output stream to a file or response.

> **Pro tip:** When you need to **convert excel to pdf java** with custom fonts, set `PdfConvertOptions.setFontSubstitution(true)` and provide a `FontMapper` to guarantee visual consistency.
