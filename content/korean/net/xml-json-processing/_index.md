# GroupDocs.Conversion .NET을 위한 XML 및 JSON 처리 튜토리얼

.NET 개발자를 위한 GroupDocs.Conversion 튜토리얼을 통해 데이터 형식 변환 기능을 살펴보세요. 이 단계별 가이드는 XML과 JSON을 데이터 소스로 사용하여 스프레드시트 및 기타 문서 유형을 생성하고, 변환 과정에서 계층적 데이터 구조를 유지하고, 복잡한 개체 관계를 유지하고, 배열과 컬렉션을 처리하고, 네임스페이스와 스키마를 관리하고, 데이터 변환 및 매핑 작업을 구현하는 방법을 보여줍니다. 각 튜토리얼에는 구조화된 데이터 형식을 다루는 데 유용한 실용적인 C# 코드 예제가 포함되어 있어, 기계가 읽을 수 있는 데이터를 사람이 이해하기 쉬운 문서 형식으로 변환하는 애플리케이션을 구축하는 데 도움을 줍니다.
