# GroupDocs.Conversion .NET을 위한 저장 파일 및 PST 처리 튜토리얼

스토리지 파일 처리 튜토리얼은 .NET 애플리케이션에서 이메일 컨테이너를 처리하는 방법에 대한 자세한 지침을 제공합니다. 이 실용적인 가이드는 PST 및 OST와 같은 개인 스토리지 파일에서 콘텐츠를 추출 및 변환하고, 폴더 구조를 검색하고, 스토리지 콘텐츠를 다른 형식으로 변환하고, 중첩된 콘텐츠 계층 구조를 처리하고, 메시지 메타데이터를 보존하고, 여러 메시지를 일괄 처리하고, 선택적 추출 작업을 구현하는 방법을 보여줍니다. 각 튜토리얼에는 스토리지 파일 변환 시나리오에 대한 완전한 C# 코드 예제가 포함되어 있어 이메일 컨테이너 형식으로 저장된 데이터에 효과적으로 액세스하고 변환할 수 있는 애플리케이션을 구축하는 데 도움이 됩니다.
