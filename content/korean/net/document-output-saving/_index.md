# GroupDocs.Conversion .NET을 위한 문서 출력 및 저장 튜토리얼

문서 저장 튜토리얼은 .NET 애플리케이션에서 변환 출력을 관리하는 방법에 대한 자세한 지침을 제공합니다. 이 단계별 가이드는 변환된 문서를 스트림에 저장하고, 사용자 지정 파일 명명 패턴을 지정하고, 출력 파일 덮어쓰기 정책을 구현하고, 대용량 문서에 대한 다중 파트 출력을 생성하고, 변환 프로세스 중 임시 파일을 관리하는 방법을 보여줍니다. 각 튜토리얼에는 다양한 출력 시나리오에 대한 C# 코드 예제가 포함되어 있어 리소스 사용 및 파일 구성을 최적화하면서 문서 변환 작업 결과를 유연하게 처리할 수 있는 애플리케이션을 구축하는 데 도움이 됩니다.
