# GroupDocs.Conversion .NET을 위한 CSV 및 구조화된 데이터 처리 튜토리얼

CSV 및 구조화된 데이터 튜토리얼은 .NET 애플리케이션에서 테이블 형식 및 데이터 중심 형식을 처리하는 방법에 대한 자세한 지침을 제공합니다. 이 실용적인 가이드는 CSV 파일을 JSON 및 XML 형식으로 변환하고, CSV를 다양한 문서 유형으로 변환하고, 구분 기호 및 인코딩 옵션을 지정하고, 날짜/시간 및 숫자 데이터를 적절하게 변환하고, 헤더 및 필드 이름을 관리하고, 변환 중 데이터 유형을 유지하고, 대량 데이터 변환 작업을 구현하는 방법을 보여줍니다. 각 튜토리얼에는 데이터 형식 변환 시나리오에 대한 완전한 C# 코드 예제가 포함되어 있어 원시 데이터 형식과 사람이 읽을 수 있는 문서를 효과적으로 연결하는 애플리케이션을 구축하는 데 도움이 됩니다.
