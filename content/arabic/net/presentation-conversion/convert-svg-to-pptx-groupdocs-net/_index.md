### تحميل الملفات وتحويلها
#### ملخص
في هذا الجزء من البرنامج التعليمي، سنحمّل ملف SVG ونحفظه كعرض تقديمي في PowerPoint. يتضمن ذلك تهيئة كائن التحويل وإعداد خيارات التحويل.

#### الخطوة 1: تحديد المسارات وتحميل ملف SVG
ابدأ بتحديد المسارات إلى ملف SVG المصدر وملف PPTX الناتج:
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY"; // استبدله بمسار الدليل الفعلي الخاص بك.
string outputDirectory = "YOUR_OUTPUT_DIRECTORY"; // استبدله بمسار الإخراج المطلوب.

string svgFilePath = Path.Combine(documentDirectory, "sample.svg"); 
string pptxOutputPath = Path.Combine(outputDirectory, "svg-converted-to.pptx");
```

#### الخطوة 2: تهيئة المحول وخيارات التحويل
إنشاء `Converter` كائن لتحميل ملف SVG، ثم قم بتهيئة خيارات التحويل:
```csharp
using (var converter = new Converter(svgFilePath))
{
    var options = new PresentationConvertOptions();
}
```
ال `PresentationConvertOptions` يتم استخدام الفئة هنا لتحديد أننا نقوم بالتحويل إلى تنسيق PowerPoint.

#### الخطوة 3: إجراء التحويل وحفظ الناتج
وأخيرًا، قم بتنفيذ التحويل وحفظ ملف PPTX:
```csharp
converter.Convert(pptxOutputPath, options);
```
