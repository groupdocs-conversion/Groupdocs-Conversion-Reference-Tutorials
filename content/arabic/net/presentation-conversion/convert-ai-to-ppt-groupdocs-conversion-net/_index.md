توضح هذه الميزة كيفية تحويل ملف Adobe Illustrator (.ai) إلى عرض تقديمي بتنسيق PowerPoint (.ppt).


قم بتحميل ملف AI باستخدام GroupDocs.Conversion:

```csharp
using (var converter = new Converter(Path.Combine(documentDirectory, "sample.ai")))

    // سيتم تعريف عملية التحويل هنا.

```

**لماذا؟** يعد تحميل الملف أمرًا بالغ الأهمية لإعداده للتحويل.

#### الخطوة 3: تكوين خيارات التحويل

قم بإعداد الخيارات لتحديد تنسيق الإخراج كـ PPT:

```csharp
PresentationConvertOptions options = new PresentationConvertOptions { Format = PresentationFileType.Ppt };
```

**توضيح:** يخبر هذا التكوين GroupDocs.Conversion بنوع الملف الذي نريد تحويل مستند AI الخاص بنا إليه.

#### الخطوة 4: إجراء التحويل والحفظ

قم بتنفيذ التحويل وحفظ ملف PPT الناتج:

```csharp
string outputFile = Path.Combine(outputDirectory, "ai-converted-to.ppt");
converter.Convert(outputFile, options);
```

**لماذا؟** تنهي هذه الخطوة العملية عن طريق إنشاء ملف PowerPoint.
