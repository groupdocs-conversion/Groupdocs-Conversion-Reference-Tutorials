## دليل التنفيذ

الآن، دعونا نقسم التنفيذ إلى أقسام منطقية.

### الميزة: تحويل PNG إلى PPTX

تتيح لك هذه الميزة تحويل ملف PNG إلى تنسيق PPTX بسهولة. 

#### الخطوة 1: إعداد الدلائل

أولاً، تأكد من إعداد أدلة الإدخال والإخراج بشكل صحيح:
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// تأكد من وجود دليل الإخراج أو قم بإنشائه إذا لم يكن موجودًا
if (!Directory.Exists(outputDirectory))
{
    Directory.CreateDirectory(outputDirectory);
}
```

#### الخطوة 2: تحميل الملف وتحويله

إليك كيفية تحميل ملف PNG وتحويله إلى PPTX:
```csharp
string sourceFile = Path.Combine(documentDirectory, "sample.png");
string outputFile = Path.Combine(outputDirectory, "png-converted-to.pptx");

// قم بتحميل ملف PNG المصدر باستخدام GroupDocs.Conversion
using (var converter = new Converter(sourceFile))
{
    var options = new PresentationConvertOptions(); // تعيين خيارات التحويل لتنسيق PPTX
    
    // قم بإجراء التحويل وحفظ النتيجة في ملف إخراج محدد
    converter.Convert(outputFile, options);
}
```

**توضيح:**
- **`Converter`:** يتم التهيئة باستخدام ملف PNG المصدر الخاص بك.
- **`PresentationConvertOptions`:** يحدد أن تنسيق الهدف هو PPTX.
- **`converter.Convert()`:** ينفذ التحويل ويحفظ الناتج.

#### نصائح استكشاف الأخطاء وإصلاحها

- تأكد من أن مسار ملف PNG المدخل صحيح لتجنب `FileNotFoundException`.
- التحقق من أذونات الكتابة لدليل الإخراج لمنع `UnauthorizedAccessException`.

### الميزة: إعداد الدلائل
