## دليل التنفيذ

### تحويل XLTM إلى PPTX

يوضح هذا القسم كيفية تحويل ملف XLTM إلى تنسيق PPTX باستخدام GroupDocs.Conversion.

#### التنفيذ خطوة بخطوة

##### تحميل ملف XLTM المصدر

أولاً، قم بتحديد المسارات الخاصة بدليل الإدخال والإخراج الخاص بك:

```csharp
string sourceFilePath = "YOUR_DOCUMENT_DIRECTORY\\sample.xltm"; // استبدله بمسار ملف XLTM الخاص بك
string outputFolder = "YOUR_OUTPUT_DIRECTORY"; // المسار لحفظ PPTX المحول
```

تهيئة `Converter` الكائن باستخدام مسار ملف المصدر:

```csharp
using (var converter = new Converter(sourceFilePath))
{
    string outputFile = Path.Combine(outputFolder, "xltm-converted-to.pptx");
    
    var options = new PresentationConvertOptions();
    converter.Convert(outputFile, options);
}
```

**توضيح:**
- ال `Converter` يتم تهيئة المسار إلى ملف XLTM الخاص بك.
- `PresentationConvertOptions()` يحدد أن تنسيق الإخراج يجب أن يكون PPTX.
- ال `converter.Convert()` تتعامل الطريقة مع عملية التحويل وتحفظ النتيجة في الدليل المحدد.

##### إعداد دليل الإخراج

تأكد من وجود الدليل الذي تريد حفظ الملفات المحولة فيه:

```csharp
string outputDirectory = "YOUR_OUTPUT_DIRECTORY"; // استبدله بمسار دليل الإخراج المطلوب

if (!Directory.Exists(outputDirectory))
{
    Directory.CreateDirectory(outputDirectory);
}
```

**توضيح:**
- يتحقق هذا المقطع من وجود دليل الإخراج.
- إذا لم يحدث ذلك، `Directory.CreateDirectory()` يتم استدعاؤه لإنشاء المجلد الضروري.
