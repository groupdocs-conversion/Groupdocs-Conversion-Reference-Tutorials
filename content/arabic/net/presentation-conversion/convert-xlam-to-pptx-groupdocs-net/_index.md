## دليل التنفيذ

يرشدك هذا القسم خلال عملية تحويل ملف XLAM إلى تنسيق PPTX، مع التركيز على كل خطوة من خطوات العملية.

### تحميل وتحويل المستند

#### ملخص
تتضمن عملية تحويل المستندات تحميل ملف المصدر وتحديد خيارات التحويل لتحويله إلى التنسيق المطلوب.

#### التنفيذ خطوة بخطوة

**1. تحديد مسارات الملفات**
ابدأ بإعداد المسارات لكل من ملفات الإدخال والإخراج:

```csharp
string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY/";

// إدخال ملف XLAM
string inputFile = Path.Combine(documentDirectory, "sample.xlam");
// إخراج ملف PPTX
string outputFile = Path.Combine(outputDirectory, "xlam-converted-to.pptx");
```

**2. تهيئة المحول**
قم بتحميل XLAM الخاص بك باستخدام `Converter` الفئة وتحديد الخيارات لتحويل PowerPoint.

```csharp
using (var converter = new Converter(inputFile))
{
    var options = new PresentationConvertOptions(); // تكوين خيارات التحويل لـ PPTX
    converter.Convert(outputFile, options); // تنفيذ التحويل وحفظ الناتج
}
```

**المعلمات موضحة:**
- `inputFile`:المسار إلى ملف XLAM الخاص بك.
- `outputFile`:مسار الوجهة لملف PPTX المُحوّل.
- `PresentationConvertOptions()`:تعيين الخيارات الافتراضية لتحويل PowerPoint.

### نصائح استكشاف الأخطاء وإصلاحها

- **لم يتم العثور على الملف:** تأكد من صحة المسارات وأسماء الملفات. تحقق من أذونات المجلدات إذا لزم الأمر.
