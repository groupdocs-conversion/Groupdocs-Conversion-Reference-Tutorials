## دليل التنفيذ
دعونا نقسم التنفيذ إلى خطوات واضحة لتحويل ملف TSV إلى تنسيق PPT.

### الخطوة 1: تحميل ملف TSV
ابدأ بتحديد مجلدات الإدخال والإخراج، مع التأكد من ضبط المسارات بشكل صحيح لكلٍّ من ملفات المصدر والوجهة. تتضمن هذه الخطوة إنشاء مثيل لفئة المحول مع مسار ملف TSV.
```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string inputFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.tsv");
```

### الخطوة 2: تحديد خيارات التحويل
بعد ذلك، حدد خيارات التحويل لبرنامج PowerPoint. يتضمن ذلك إنشاء مثيل لـ `PresentationConvertOptions`، مع تحديد التنسيق المطلوب (PPT في هذه الحالة).
```csharp
using GroupDocs.Conversion.Options.Convert;
PresentationConvertOptions options = new PresentationConvertOptions { Format = PresentationFileType.Ppt };
```

### الخطوة 3: التحويل والحفظ
أخيرًا، قم بإجراء التحويل عن طريق استدعاء `Convert` على نسخة المحول لديك. سيؤدي هذا إلى حفظ ملف PPT المُحوّل في دليل الإخراج المُحدد.
```csharp
string outputFile = Path.Combine(outputFolder, "tsv-converted-to.ppt");
using (var converter = new Converter(inputFile))
{
    converter.Convert(outputFile, options);
}
```
**نصائح استكشاف الأخطاء وإصلاحها:**
- تأكد من أن جميع المسارات صحيحة ويمكن الوصول إليها.
- تأكد من تثبيت مكتبة GroupDocs.Conversion بشكل صحيح.
