**ما سوف تتعلمه:**


## دليل التنفيذ

### نظرة عامة على ميزة تحويل DOTM إلى PPTX

يوضح هذا القسم تحويل ملف .dotm إلى صيغة .pptx. يُبسط GroupDocs.Conversion لـ .NET هذه العملية، موفرًا تحويلًا عالي الجودة بأقل قدر من التعليمات البرمجية.

#### الخطوة 1: إعداد الدلائل

ابدأ بتحديد أدلة الإدخال والإخراج:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY/";
```

#### الخطوة 2: تحميل ملف DOTM المصدر

قم بتحميل ملف .dotm الخاص بك باستخدام GroupDocs.Conversion's `Converter` هذه الخطوة هي بمثابة تهيئة لعملية التحويل.

```csharp
using (var converter = new Converter(Path.Combine(documentDirectory, "sample.dotm")))
{
    // سيتم تعريف خيارات التحويل هنا
}
```

#### الخطوة 3: تهيئة خيارات التحويل

قم بإعداد خيارات التحويل لتحديد تنسيق الإخراج كـ PPTX:

```csharp
var options = new PresentationConvertOptions();
```

#### الخطوة 4: تحديد مسار الإخراج والتحويل

حدد المكان الذي يجب حفظ الملف المحول فيه، ثم قم بإجراء التحويل:

```csharp
string outputFile = Path.Combine(outputDirectory, "dotm-converted-to.pptx");
converter.Convert(outputFile, options);
```
