## دليل التنفيذ

### الميزة: تحويل PDF إلى PSD

يُعد تحويل ملف PDF إلى صيغة PSD أمرًا بالغ الأهمية لمصممي الجرافيك الذين يحتاجون إلى طبقات قابلة للتعديل. لنشرح العملية بالتفصيل:

#### الخطوة 1: تحديد مسارات المجلد والملف الناتج

إعداد مجلدات لملفات الإدخال والإخراج. تعديل المسارات حسب الحاجة.

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string inputFile = "YOUR_DOCUMENT_DIRECTORY\\sample.pdf";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");
```

#### الخطوة 2: إنشاء دالة التدفق

سنستخدم دالةً لإنشاء تدفقات لكل صفحة يتم تحويلها. هذا يضمن تسمية جميع ملفات PSD بشكل صحيح.

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext =>
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

#### الخطوة 3: إعداد خيارات التحويل

قم بتحديد خيارات التحويل لتحديد أننا نقوم بالتحويل إلى تنسيق PSD.

```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Psd };
```

#### الخطوة 4: تنفيذ التحويل

قم بتنفيذ التحويل باستخدام `Converter` الكائن والإعدادات المحددة الخاصة بك.

```csharp
using (Converter converter = new Converter(inputFile))
{
    // تحويل كل صفحة من ملف PDF إلى صيغة PSD
    converter.Convert(getPageStream, options);
}
```
