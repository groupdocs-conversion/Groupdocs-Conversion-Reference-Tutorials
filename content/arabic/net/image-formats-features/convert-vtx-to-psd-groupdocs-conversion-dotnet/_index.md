## دليل التنفيذ
يرشدك هذا القسم خلال تحويل ملفات VTX إلى تنسيق PSD باستخدام GroupDocs.Conversion.
### تحميل الملفات وتحويلها
#### ملخص
تعلّم كيفية تحميل ملف .vtx وتحويله إلى ملفات .psd متعددة، كل منها يُمثّل صفحةً من المستند الأصلي. هذا مفيدٌ لإعداد قوالب Visio لأعمال التصميم الجرافيكي في Photoshop.
#### التنفيذ خطوة بخطوة
**1. إعداد المسارات**
```csharp
string documentPath = Path.Combine(@"YOUR_DOCUMENT_DIRECTORY", "sample.vtx");
string outputFolder = Path.Combine(@"YOUR_OUTPUT_DIRECTORY");
```
**2. قم بتعريف دالة إنشاء التدفق**
تعمل هذه الوظيفة على إنشاء مجرى جديد لكل صفحة سيتم تحويلها:
```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext =>
{
    return new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
};
```
**3. تحميل وتحويل ملف VTX**
قم بتحميل ملف VTX وحدد خيارات التحويل:
```csharp
using (Converter converter = new Converter(documentPath))
{
    ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Psd };
    converter.Convert(getPageStream, options);
}
```
**توضيح:**
- `SavePageContext`:يوفر سياقًا حول الصفحة التي يتم تحويلها.
- `ImageConvertOptions`:تكوين إعدادات التحويل، وتحديد PSD كتنسيق مستهدف.
