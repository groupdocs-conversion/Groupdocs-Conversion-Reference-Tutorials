## دليل التنفيذ

### نظرة عامة على الميزة: تحويل تنسيق JPX إلى تنسيق PSD
ترتكز هذه الميزة على تحويل الرسومات عالية الجودة من ملفات JPX إلى ملفات PSD المستخدمة على نطاق واسع من قبل مستخدمي Adobe Photoshop.

#### الخطوة 1: تحديد دليل الإخراج وقالب الملف
أولاً، حدّد مجلد الإخراج الذي ستُخزّن فيه ملفاتك المُحوّلة. يمكنك تحديد قالب لتسمية هذه الملفات كما هو موضح أدناه:
```csharp
using System.IO;
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");
```

#### الخطوة 2: إنشاء تدفقات الملفات للصفحات المُحوّلة
أنشئ دالة لتوليد تدفقات الملفات ديناميكيًا. سيساعدك هذا على إدارة كل صفحة من المستند المُحوَّل:
```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

#### الخطوة 3: تحميل ملف JPX المصدر
استخدم `Converter` استخدم class لتحميل ملف JPX المصدر. تأكد من تحديد المسار الصحيح:
```csharp
using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_JPX"))
{
    // سوف تتبع عملية التحويل.
}
```

#### الخطوة 4: تعيين خيارات التحويل لتنسيق PSD
قم بتحديد خيارات التحويل المخصصة لتنسيق PSD باستخدام `ImageConvertOptions`:
```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Psd };
```

#### الخطوة 5: تنفيذ التحويل
قم بتنفيذ عملية التحويل، والتي ستؤدي إلى إنشاء ملف PSD لكل صفحة من مستند JPX الخاص بك:
```csharp
converter.Convert(getPageStream, options);
```
