# تحويل DNG إلى PSD باستخدام GroupDocs.Conversion لـ .NET: دليل خطوة بخطوة

## مقدمة

هل ترغب في تحويل ملفات النيجاتيف الرقمي (DNG) إلى صيغة مستندات أدوبي فوتوشوب (PSD) بكفاءة؟ سيوضح لك هذا الدليل المفصل كيفية استخدام GroupDocs.Conversion لـ .NET، وهي أداة فعّالة تُبسّط تحويلات الملفات. سواء كنت مصورًا محترفًا أو مصمم جرافيك، فإن إتقان هذا التحويل يُسهّل سير عملك.

في هذا البرنامج التعليمي، سنغطي:
- فهم تحويل DNG إلى PSD
- إعداد بيئتك باستخدام GroupDocs.Conversion لـ .NET
- تنفيذ عملية التحويل خطوة بخطوة
- التطبيقات الواقعية واعتبارات الأداء

باتباع هذا الدليل، ستتعلم كيفية تحويل ملفات DNG إلى PSD باستخدام C#. لنبدأ بمراجعة المتطلبات الأساسية.

## المتطلبات الأساسية

- **المكتبات والتبعيات**GroupDocs.Conversion لـ .NET (الإصدار 25.3.0)

### التهيئة والإعداد الأساسي

قم بتضمين المساحات الأساسية اللازمة في مشروع C# الخاص بك:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;
```

## دليل التنفيذ

يوفر هذا القسم دليلاً مفصلاً لتنفيذ تحويل DNG إلى PSD.

### نظرة عامة على ميزة التحويل

تتيح لك هذه الميزة تحويل ملف Digital Negative (DNG) إلى تنسيق Adobe Photoshop Document (PSD)، مما يتيح المزيد من التحرير والتلاعب في برامج التصميم الجرافيكي مثل Adobe Photoshop.

#### الخطوة 1: تحديد دليل الإخراج

قم بتعيين دليل الإخراج الذي سيتم حفظ الملفات المحولة فيه:

```csharp
string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
```

#### الخطوة 2: إنشاء تدفق لكل صفحة مُحوَّلة

استخدم دالة لإنشاء تدفق لكل صفحة من الملف المُحوَّل. هذا ضروري للتعامل مع صفحات متعددة إن أمكن.

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext =>
    new FileStream(string.Format(outputFolder + "\\converted-page-{0}.psd", savePageContext.Page), FileMode.Create);
```

#### الخطوة 3: تحميل ملف DNG المصدر

حمّل ملف DNG المصدر باستخدام GroupDocs.Conversion. تأكد من استبدال `"YOUR_DOCUMENT_DIRECTORY/SAMPLE_DNG"` مع المسار الفعلي لملف DNG الخاص بك:

```csharp
using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_DNG"))
{
    // سيتم وضع كود التكوين والتحويل هنا.
}
```

#### الخطوة 4: تعيين خيارات التحويل

حدّد خيارات التحويل لصيغة PSD. هذا يعني أن الناتج يجب أن يكون ملف PSD:

```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Psd };
```

#### الخطوة 5: تنفيذ التحويل

قم بتنفيذ التحويل عن طريق استدعاء `Convert` الطريقة، تمرير وظيفة التدفق وخيارات التحويل الخاصة بك:

```csharp
converter.Convert(getPageStream, options);
```

- **التحقق من صحة الترخيص**:تأكد من إعداد ترخيصك بشكل صحيح إذا واجهت قيودًا على الاستخدام.
