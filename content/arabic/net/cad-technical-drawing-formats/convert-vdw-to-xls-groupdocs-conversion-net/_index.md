### التهيئة والإعداد الأساسي
لتهيئة GroupDocs.Conversion في تطبيق C# الخاص بك:

```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

namespace VdwToXlsConverter
{
    class Program
    {
        static void Main(string[] args)
        {
            string documentPath = @"YOUR_DOCUMENT_DIRECTORY\sample.vdw"; // المسار إلى ملف VDW الخاص بك
            string outputPath = @"YOUR_OUTPUT_DIRECTORY\output.xls"; // مسار الإخراج لملف XLS

            using (Converter converter = new Converter(documentPath))
            {
                SpreadsheetConvertOptions options = new SpreadsheetConvertOptions();
                
                converter.Convert(outputPath, options);
            }
        }
    }
}
```
يقوم هذا الكود بإعداد بيئة تحويل أساسية. `Converter` يتم تهيئة الفئة باستخدام مسار ملف VDW الخاص بك، ويتم تنفيذ التحويل باستخدام `SpreadsheetConvertOptions`.
