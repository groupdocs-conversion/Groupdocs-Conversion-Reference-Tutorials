## دليل التنفيذ
### تحويل ملف DWF إلى تنسيق HTML
توضح هذه الميزة كيفية تحويل ملف DWF إلى تنسيق HTML، مما يجعله متاحًا على أي متصفح ويب.
#### الخطوة 1: تحديد مسارات الإدخال والإخراج
أولاً، قم بإعداد المسارات لملف DWF المدخل والمكان الذي تريد حفظ ملف HTML المحول فيه:
```csharp
string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.dwf");
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "dwf-converted-to.html");
```
#### الخطوة 2: تحميل الملف وتحويله
قم بتحميل ملف DWF باستخدام `Converter` الفئة وتحديد خيارات التحويل لـ HTML:
```csharp
using (var converter = new Converter(inputFilePath))
{
    // خيارات تهيئة التحويل إلى تنسيق HTML
    var options = new WebConvertOptions();
    
    // قم بإجراء التحويل وحفظه كملف HTML
    converter.Convert(outputFile, options);
}
```
### شرح مقتطفات التعليمات البرمجية
- **`Converter` فصل**: يتم التهيئة بمسار ملف DWF. تتولى هذه الفئة تحميل الملف للتحويل.
- **`WebConvertOptions`**:يحدد أن تنسيق الإخراج يجب أن يكون HTML.
- **`converter.Convert` طريقة**:تنفيذ عملية التحويل وحفظ النتيجة كملف HTML.
