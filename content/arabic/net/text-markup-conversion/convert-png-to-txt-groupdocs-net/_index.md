## دليل التنفيذ

### تحويل PNG إلى TXT

#### ملخص

تتيح لك هذه الميزة تحميل صورة PNG وتحويلها إلى تنسيق نصي باستخدام GroupDocs.Conversion لـ .NET.

#### التنفيذ خطوة بخطوة

**1. قم بتحميل ملف المصدر**

ابدأ بتحميل ملف PNG المصدر الخاص بك إلى كائن المحول:

```csharp
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY\\sample.png"))
{
    // تابع خطوات التحويل هنا
}
```

**2. تعيين خيارات التحويل**

قم بتحديد خيارات التحويل لتحديد أنك تقوم بالتحويل إلى تنسيق TXT:

```csharp
WordProcessingConvertOptions options = new WordProcessingConvertOptions { Format = FileTypes.WordProcessingFileType.Txt };
```

- **شرح المعلمة**: `options` يقوم بتكوين كيفية التعامل مع التحويل، وتحديد الإخراج كـ TXT.

**3. تنفيذ التحويل**

قم بإجراء التحويل وحفظ النتيجة في الموقع المطلوب:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "png-converted-to.txt");

converter.Convert(outputFile, options);
```

- **قيم الإرجاع**: ال `Convert` تحفظ الطريقة الملف المُحوّل في المسار المحدد.

- تأكد من أن مسار المصدر الخاص بك صحيح.
