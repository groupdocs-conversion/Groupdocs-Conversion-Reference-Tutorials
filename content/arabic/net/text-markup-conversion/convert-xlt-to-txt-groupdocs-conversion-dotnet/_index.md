### نظرة عامة على الميزة

تتضمن الوظيفة الأساسية تحميل مستند XLT وتحويله إلى ملف نصي مع الحفاظ على سلامة محتواه. تُعد هذه العملية مفيدة عند الحاجة إلى استخراج البيانات من قوالب Excel بصيغة نصية بسيطة.

#### الخطوة 1: تحديد مسارات الملفات

ابدأ بتحديد الدلائل ومسارات الملفات لملفات XLT المصدر وملفات TXT الناتجة:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputFileDir = "YOUR_OUTPUT_DIRECTORY";

// تعيين المسار لملفات المصدر والإخراج
class SourceFilePath = Path.Combine(documentDirectory, "sample.xlt");

```

#### الخطوة 2: تهيئة المحول

بعد ذلك، قم بتهيئة GroupDocs.Converter باستخدام ملف XLT المصدر:

```csharp
using (var converter = new GroupDocs.Conversion.Converter(sourceFilePath))
{
    // خيارات التحويل والمنطق سوف تتبع هنا
}
```

#### الخطوة 3: تحديد خيارات التحويل

لتحويل إلى تنسيق TXT، قم بتحديد خيارات التحويل على النحو التالي:


تعمل هذه الخطوة على تكوين المحول لإخراج ملف نصي.

#### الخطوة 4: تنفيذ التحويل

وأخيرًا، قم بتنفيذ التحويل وحفظ النتيجة:

```csharp

class Converter(outputPath, options);
```
