## دليل التنفيذ

### الميزة: تحويل ملف PS إلى صيغة TXT
توضح هذه الميزة كيفية تحويل ملف PostScript إلى تنسيق نص عادي.

#### الخطوة 1: تحميل ملف PS المصدر
ابدأ بتحميل ملف PS المصدر باستخدام GroupDocs.Conversion. `Converter` الفئة مسؤولة عن التعامل مع هذه العملية:
```csharp
using System.IO;
using GroupDocs.Conversion;

string documentPath = "YOUR_DOCUMENT_DIRECTORY/sample.ps";
```
**توضيح**:تأكد من أن `documentPath` يشير إلى الموقع الصحيح لملف PS الخاص بك.

#### الخطوة 2: تكوين خيارات التحويل
إعداد خيارات التحويل لتحديد TXT كتنسيق الهدف:
```csharp
var options = new WordProcessingConvertOptions { Format = WordProcessingFileType.Txt };
```
**توضيح**: ال `WordProcessingConvertOptions` يتيح لك تكوين إعدادات متنوعة لتحويل المستندات. هنا، نحدد `.txt` كتنسيق الإخراج المطلوب لدينا.

#### الخطوة 3: تنفيذ التحويل
قم بتنفيذ التحويل وحفظ النتيجة في مجلد الإخراج المخصص لك:
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "ps-converted-to.txt");

using (var converter = new Converter(documentPath))
{
    // قم بإجراء التحويل
    converter.Convert(outputFile, options);
}
```
**توضيح**: ال `Convert` تتولى الطريقة عملية تحويل مستندك وحفظه في المسار المحدد.
