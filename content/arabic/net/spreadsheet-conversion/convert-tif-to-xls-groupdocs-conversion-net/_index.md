## دليل التنفيذ

### تحميل ملف TIF

**ملخص**
تحميل ملف TIF هو الخطوة الأولى. تُجهّز هذه العملية صورتك للتحويل باستخدام واجهة برمجة التطبيقات القوية من GroupDocs.

#### الخطوة 1: تحديد مسار ملف TIF

تأكد من أن لديك المسار الصحيح لملف TIF الخاص بك:

```csharp
string tifFilePath = @"YOUR_DOCUMENT_DIRECTORY\sample.tif";
```

#### الخطوة 2: تحميل ملف TIF

استخدم `Converter` فئة لتحميل ملفك. يمثل هذا الكائن ملف TIF المُحمّل، وهو جاهز للتحويل.

```csharp
using (var converter = new Converter(tifFilePath))
{
    // تم الآن تحميل ملف TIF وأصبح جاهزًا.
}
```

### تحويل تنسيق TIF إلى تنسيق XLS

**ملخص**
بعد تحميل ملف TIF، يمكنك تحويله إلى تنسيق XLS. يتناول هذا القسم إعداد خيارات التحويل وتنفيذ العملية.

#### الخطوة 1: تحديد مسار الإخراج

حدد المكان الذي تريد حفظ الملف المحول فيه:

```csharp
string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "tif-converted-to.xls");
```

#### الخطوة 2: تعيين خيارات التحويل

إنشاء `SpreadsheetConvertOptions` الكائن لتحديد تنسيق الهدف (XLS في هذه الحالة):

```csharp
SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Xls };
```

#### الخطوة 3: تنفيذ التحويل

استدعاء طريقة التحويل باستخدام مسار الملف والخيارات الخاصة بك:

```csharp
using (var converter = new Converter(@"YOUR_DOCUMENT_DIRECTORY\sample.tif"))
{
    // تحويل وحفظ ملف الإخراج.
    converter.Convert(outputFile, options);
}
```
