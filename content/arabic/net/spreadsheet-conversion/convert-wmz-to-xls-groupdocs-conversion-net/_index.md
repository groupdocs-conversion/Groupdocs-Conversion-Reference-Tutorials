## إعداد GroupDocs.Conversion لـ .NET

بمجرد التثبيت، قم بإعداد المكتبة في مشروعك:

```csharp
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

// تحديد الدلائل للملفات المصدر والإخراج
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY"; // استبدل بالدليل الفعلي الخاص بك
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// قم بتحميل ملف WMZ من الدليل المحدد
string documentPath = Path.Combine(documentDirectory, "sample.wmz");

using (var converter = new Converter(documentPath))
{
    // تعيين خيارات التحويل لتنسيق XLS
    var convertOptions = new SpreadsheetConvertOptions
    {
        Format = SpreadsheetFileType.Xls
    };

    // قم بتحديد مسار الإخراج حيث سيتم حفظ الملف المحول
    string outputFile = Path.Combine(outputDirectory, "wmz-converted-to.xls");

    // قم بإجراء التحويل وحفظ ملف XLS في الموقع المحدد
    converter.Convert(outputFile, convertOptions);
}
```

يقوم مقتطف التعليمات البرمجية هذا بتهيئة `Converter` كائن بمسار ملف WMZ، ويُهيئ خيارات التحويل لتنسيق XLS، ويُجري التحويل. يُحفظ الملف المُحوّل في مجلد الإخراج المُحدد.

### تحويل تنسيق WMZ إلى تنسيق XLS

#### ملخص

تتضمن عملية تحويل ملفات WMZ إلى XLS تعيين خيارات التحويل الصحيحة وتحديد مسارات المصدر والوجهة.

#### خطوات:
1. **تهيئة المحول**
   - إنشاء `Converter` مثيل مع مسار ملف WMZ الخاص بك، ومعالجة كافة مهام التحويل.
2. **تكوين خيارات التحويل**
   - يستخدم `SpreadsheetConvertOptions` لتحديد تنسيق الإخراج كـ XLS، مع تحديد كيفية ظهور البيانات في Excel.
3. **تنفيذ وحفظ التحويل**
   - اتصل بـ `Convert` استخدم الطريقة مع مسار ملف الإخراج وخيارات التحويل لتنفيذ العملية. يُحفظ ملف XLS الناتج في الموقع المحدد.

#### استكشاف الأخطاء وإصلاحها
- تأكد من تعيين المسارات بشكل صحيح لكل من الدليل المصدر والدليل الوجهة.
