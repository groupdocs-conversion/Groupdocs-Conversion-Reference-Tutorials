## دليل التنفيذ
### الميزة 1: تحميل ملف MBOX
**ملخص:**
تحميل ملف MBOX هو الخطوة الأولى لتحويل محتوياته. يوضح هذا القسم كيفية تحميل ملف MBOX باستخدام خيارات التحميل المشروط.
#### الخطوة 1: إعداد خيارات التحميل المشروط
```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Load;

string inputPath = "YOUR_DOCUMENT_DIRECTORY/sample.mbox";
var loadOptions = new MboxLoadOptions();
```
**توضيح:**  
`MboxLoadOptions()` يقوم بتكوين عملية التحميل، مما يضمن معالجة ملفات MBOX فقط.
#### الخطوة 2: إنشاء مثيل محول
```csharp
var converter = new Converter(inputPath, (loadContext) => 
    loadContext.SourceFormat == EmailFileType.Mbox ? loadOptions : null);
```
**توضيح:**
يتحقق المحول من أن تنسيق المصدر هو MBOX قبل تطبيق خيارات التحميل المحددة.
#### الخطوة 3: التخلص من الموارد
```csharp
current.Dispose();
```
**توضيح:**  
تخلص من الموارد لتحرير الذاكرة بمجرد الانتهاء من مهام التحويل.
### الميزة 2: تحويل MBOX إلى XLS
**ملخص:**
تشرح هذه الميزة كيفية تحويل ملف MBOX إلى تنسيق XLS، مما يتيح معالجة البيانات وتحليلها بسهولة.
#### الخطوة 1: إعداد مسار الإخراج
```csharp
string outputPath = "YOUR_OUTPUT_DIRECTORY/mbox-converted-{0}-to.xls";
```
#### الخطوة 2: تهيئة المحول باستخدام خيارات التحميل
```csharp
using (var converter = new Converter(Constants.SAMPLE_MBOX, 
    (loadContext) => loadContext.SourceFormat == EmailFileType.Mbox ? loadOptions : null))
{
    var options = new SpreadsheetConvertOptions { Format = GroupDocs.Conversion.FileTypes.SpreadsheetFileType.Xls };
    int counter = 1;
    
    converter.Convert(
        (saveContext) => new FileStream(string.Format(outputPath, counter++), FileMode.Create),
        options
    );
}
```
**توضيح:**  
- **خيارات تحويل جدول البيانات**:يحدد تنسيق التحويل كـXLS.
- **تدفق الملفات**:إدارة إنشاء الملف لكل مستند مُحوّل.
