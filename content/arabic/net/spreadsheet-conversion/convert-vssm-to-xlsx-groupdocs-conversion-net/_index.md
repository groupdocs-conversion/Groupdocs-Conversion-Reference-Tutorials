### التهيئة والإعداد الأساسي
قم بتهيئة GroupDocs.Conversion لـ .NET في تطبيق C# الخاص بك:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

// تحديد الدلائل لملفات الإدخال والإخراج
string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY\";

// قم بتحميل ملف VSSM المصدر باستخدام المسار المحدد.
using (var converter = new Converter(Path.Combine(documentDirectory, "sample.vssm")))
{
    // إعداد خيارات التحويل لتنسيق Excel.
    var options = new SpreadsheetConvertOptions();
    
    // حدد مسار ملف الإخراج وقم بتحويله.
    string outputFile = Path.Combine(outputDirectory, "vssm-converted-to.xlsx");
    converter.Convert(outputFile, options);
}
```
في مقتطف التعليمات البرمجية هذا، نقوم بتعريف أدلة الإدخال والإخراج، وتحميل ملف VSSM، وإعداد خيارات التحويل الخاصة بجداول بيانات Excel، وإجراء التحويل.
