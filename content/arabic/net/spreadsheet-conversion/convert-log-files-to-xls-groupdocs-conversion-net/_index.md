## دليل التنفيذ

الآن، دعونا نقوم بتقسيم عملية التحويل إلى خطوات قابلة للإدارة.

using (var converter = new Converter(inputFile))
{
    // تم الانتهاء من إعداد التحويل

### تعيين خيارات التحويل لتنسيق XLS

بعد ذلك، قم بتكوين خيارات التحويل لتحديد ما إذا كنت تريد إخراج XLS:

#### الخطوة 2: تكوين خيارات تحويل جدول البيانات

```csharp
SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Xls };
```

**توضيح:**
- `SpreadsheetConvertOptions`:تتيح هذه الفئة تحديد إعدادات تنسيق جدول البيانات المختلفة.
- `Format`:يحدد نوع الملف المستهدف للتحويل.

### إجراء التحويل وحفظ الناتج

وأخيرًا، قم بتنفيذ التحويل وحفظ النتيجة:

#### الخطوة 3: تحويل الملف وحفظه

```csharp
string outputFile = "path/to/output/log-converted-to.xls";
converter.Convert(outputFile, options);
```

**المعلمات موضحة:**
- `outputFile`:المسار الذي سيتم حفظ الملف المُحوّل فيه.
- `options`:يحتوي على إعدادات التحويل.

- تأكد من أن المسارات صحيحة ويمكن الوصول إليها.
