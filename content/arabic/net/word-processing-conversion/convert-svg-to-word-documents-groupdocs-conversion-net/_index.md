### تحويل SVG إلى مستند Word
تتيح لك هذه الميزة تحويل ملفات SVG إلى صيغة مستند Word أكثر سهولة في الاستخدام. تُنجز مكتبة GroupDocs.Conversion هذه المهمة بكفاءة وبأقل قدر من التعليمات البرمجية.

#### الخطوة 1: تحديد مسارات الملفات وتحميل مصدر SVG
أولاً، حدد المسارات الخاصة بدليل الإدخال والإخراج الخاص بك:

```csharp
using System.IO;

// تحديد مسارات الملفات باستخدام العناصر النائبة
string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.svg");
string outputFolder = Constants.GetOutputDirectoryPath(); // تعيين مسار متسق مثل "YOUR_OUTPUT_DIRECTORY"
string outputFile = Path.Combine(outputFolder, "svg-converted-to.doc");

// تحميل ملف SVG المصدر
using (var converter = new Converter(inputFilePath))
{
    // سيتم تعريف خيارات التحويل والعملية هنا...
}
```

**توضيح:**
- ال `inputFilePath` تشير النقاط المتغيرة إلى ملف SVG الخاص بك.
- `outputFile` هو المكان الذي سيتم حفظ ملف DOC المُحوّل فيه.

#### الخطوة 2: تكوين خيارات التحويل
بعد ذلك، قم بإعداد خيارات التحويل لتحويل SVG إلى مستند Word:

```csharp
// إنشاء خيارات تحويل معالجة الكلمات لتنسيق .doc
var options = new WordProcessingConvertOptions { Format = WordProcessingFileType.Doc };

// تنفيذ التحويل وحفظ ملف الإخراج
converter.Convert(outputFile, options);
```

**توضيح:**
- `WordProcessingConvertOptions` يحدد تنسيق DOC المستهدف.
- ال `Format` تم تعيين الخاصية إلى `Doc` للتوافق مع Microsoft Word.
