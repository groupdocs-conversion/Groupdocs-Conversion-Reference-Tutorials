## دليل التنفيذ

### تحميل وتحويل ملف POTM

#### ملخص
سنحمّل ملف POTM ونحوّله إلى صيغة DOC باستخدام GroupDocs.Conversion. تتضمن هذه العملية تحديد خيارات التحويل وتنفيذ طريقة التحويل.

#### التنفيذ خطوة بخطوة

**1. تحديد مسارات المستندات**
ابدأ بتحديد المسارات لملف POTM المدخل وملف DOC المخرج:

```csharp
const string DocumentDirectory = "YOUR_DOCUMENT_DIRECTORY";
const string OutputDirectory = "YOUR_OUTPUT_DIRECTORY/";
```

**2. تهيئة المحول**
قم بتحميل ملف POTM إلى كائن GroupDocs.Conversion:

```csharp
using (var converter = new GroupDocs.Conversion.Converter(Path.Combine(DocumentDirectory, "sample.potm")))
{
    // منطق التحويل سوف يذهب هنا.
}
```

**3. تعيين خيارات التحويل**
حدد الخيارات لتحويل المستند إلى تنسيق DOC:

```csharp
var options = new WordProcessingConvertOptions
{
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc
};
```

**4. قم بإجراء التحويل**
قم بتنفيذ التحويل وحفظ ملف الإخراج:

```csharp
string outputFile = Path.Combine(OutputDirectory, "potm-converted-to.doc");
converter.Convert(outputFile, options);
```
