## دليل التنفيذ

### الميزة: تحميل وتحويل RTF إلى TXT

تتيح لك هذه الميزة تحميل ملف RTF وتحويله إلى تنسيق نص عادي باستخدام GroupDocs.Conversion.

#### الخطوة 1: تحديد مسارات الملفات

ابدأ بإعداد مسارات الدليل لملفات الإدخال والإخراج:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// تحديد مسار ملف RTF المصدر
string sourceRtfFilePath = Path.Combine(documentDirectory, "sample.rtf");
// تحديد مسار ملف TXT الناتج
string outputFile = Path.Combine(outputDirectory, "rtf-converted-to.txt");
```

#### الخطوة 2: تهيئة المحول

استخدم `Converter` الفئة لتحميل مستند RTF الخاص بك:

```csharp
using (var converter = new Converter(sourceRtfFilePath))
{
    // سيتم تعريف عمليات التحويل هنا
}
```

#### الخطوة 3: تعيين خيارات التحويل

تكوين خيارات التحويل لتنسيق TXT:

```csharp
// إنشاء WordProcessingConvertOptions وتعيين التنسيق إلى TXT
var options = new WordProcessingConvertOptions
{
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt
};
```

#### الخطوة 4: تنفيذ التحويل

قم بتنفيذ عملية التحويل وحفظ ملف الإخراج:

```csharp
// تحويل RTF إلى TXT وحفظه في الدليل المحدد
converter.Convert(outputFile, options);
```
