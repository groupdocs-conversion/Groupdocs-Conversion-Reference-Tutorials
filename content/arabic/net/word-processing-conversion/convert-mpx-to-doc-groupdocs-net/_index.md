## دليل التنفيذ

### نظرة عامة على الميزة: تحويل MPX إلى DOC

تستفيد هذه الميزة من الإمكانات القوية لبرنامج GroupDocs.Conversion لتحويل ملفات بيانات المشروع من صيغة MPX إلى مستندات Word قابلة للتعديل. لنستعرض كل خطوة من خطوات التنفيذ:

#### الخطوة 1: تحديد أدلة الإدخال والإخراج

ابدأ بتحديد مكان وجود ملف MPX المصدر والمكان الذي تريد حفظ ملف DOC المحول فيه.

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY"; // استبداله بمسار الدليل الفعلي
string outputFileDirectory = "YOUR_OUTPUT_DIRECTORY"; // استبداله بمسار الدليل الفعلي

string sourceMpxFilePath = Path.Combine(documentDirectory, "sample.mpx"); 
string outputFile = Path.Combine(outputFileDirectory, "mpx-converted-to.doc");
```

#### الخطوة 2: تهيئة المحول

إنشاء مثيل لـ `Converter` الفئة باستخدام ملف MPX الخاص بك.

```csharp
using (var converter = new GroupDocs.Conversion.Converter(sourceMpxFilePath))
{
    // سيتم إضافة منطق التحويل هنا
}
```

#### الخطوة 3: تعيين خيارات التحويل لتنسيق DOC

قم بتحديد خيارات التحويل لتحديد تنسيق الهدف كـ DOC.

```csharp
WordProcessingConvertOptions options = new WordProcessingConvertOptions 
{ 
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc 
};
```

#### الخطوة 4: تنفيذ التحويل

استخدم `Convert` طريقة مثيل المحول، تمرير مسار ملف الإخراج وخيارات التحويل.

```csharp
converter.Convert(outputFile, options);
```
