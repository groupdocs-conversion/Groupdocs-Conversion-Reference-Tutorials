**1. مسار الإعداد:**

قم بدمج الدليل واسم الملف للحصول على المسار الكامل:

**3. تهيئة المحول:**
قم بتحميل ملف XLTX باستخدام GroupDocs.Conversion:

```csharp
using (var converter = new GroupDocs.Conversion.Converter(sourceFilePath))
{
    // سيتم إجراء عمليات التحويل هنا
}
```

#### توضيح
- **المسار.الجمع:** يضمن التوافق بين الأنظمة الأساسية.
- **تهيئة المحول:** يستخدم `using` عبارة للتعامل مع تنظيف الموارد تلقائيًا.
