## دليل التنفيذ

الآن بعد إعداد بيئتك، دعنا نقوم بتحويل ملفات CSV إلى JSON.

### الميزة: تحويل CSV إلى JSON
تتيح هذه الميزة تحويل بيانات CSV بكفاءة إلى صيغة JSON منظمة. اتبع الخطوات التالية:

#### الخطوة 1: تحديد مسارات الدليل وأسماء الملفات
قم بتحديد المكان الذي سيتم فيه وضع ملفات الإدخال والإخراج لإدارة مسار الملف بشكل فعال في الكود الخاص بك.

```csharp
// تعيين مسارات الدليل لملفات الإدخال والإخراج
cstring documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
cstring outputDirectory = "YOUR_OUTPUT_DIRECTORY/";

// تحديد أسماء الملفات
cstring inputCsvFile = Path.Combine(documentDirectory, "sample.csv");
cstring outputFile = Path.Combine(outputDirectory, "converted.json");
```

#### الخطوة 2: تهيئة خيارات تحميل CSV
قم بتكوين خيارات التحميل الخاصة بك لتحديد الفاصل المستخدم في ملف CSV (الفاصلة في هذا المثال).

```csharp
// تهيئة خيارات تحميل CSV باستخدام فاصل محدد
var loadOptions = new CsvLoadOptions
{
    Separator = ','
};
```

#### الخطوة 3: إنشاء مثيل لفئة المحول
باستخدام ملف الإدخال وخيارات التحميل، قم بإنشاء مثيل `Converter` الفئة لإعداد منطق التحويل الخاص بك.

```csharp
// إنشاء مثيل لفئة المحول مع سياق التحميل
using (Converter converter = new Converter(inputCsvFile, (LoadContext loadContext) => loadOptions))
{
    // الخطوة 4: تعيين خيارات التحويل لتنسيق JSON
    WebConvertOptions convertOptions = new WebConvertOptions
    {
        Format = WebFileType.Json
    };

    // تحويل CSV إلى JSON وحفظ ملف الإخراج
    converter.Convert(outputFile, convertOptions);
}
```

### شرح معلمات الكود
- **`CsvLoadOptions`**: يُهيئ كيفية قراءة بيانات CSV. يُحدد الفاصل تقسيمات الحقول.
- **`Converter` فصل**:يتعامل مع عمليات التحويل بشكل مركزي.
- **`WebConvertOptions`**:يحدد تنسيق الإخراج، JSON في هذه الحالة.

### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من أن مسارات الملفات صحيحة ويمكن الوصول إليها بواسطة تطبيقك.
- التحقق من سلامة بيانات CSV لمنع مخرجات JSON المشوهة.
- التحقق من وجود أي استثناءات أثناء التنفيذ لتشخيص مشكلات الإعداد.
