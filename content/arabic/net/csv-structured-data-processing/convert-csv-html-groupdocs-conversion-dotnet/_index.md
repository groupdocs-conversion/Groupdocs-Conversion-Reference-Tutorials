## دليل التنفيذ

دعنا نستعرض عملية تحويل ملف CSV إلى HTML باستخدام GroupDocs.Conversion لـ .NET.

### التهيئة والإعداد

ابدأ بتهيئة مكتبة التحويل. إليك إعداد بسيط بلغة C#:

```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main(string[] args)
    {
        string sourceCsvPath = @"YOUR_DOCUMENT_DIRECTORY\sample.csv";
        string outputPath = Path.Combine(@"YOUR_OUTPUT_DIRECTORY", "output.html");

        // قم بتهيئة المحول باستخدام مسار ملف CSV الخاص بك
        using (Converter converter = new Converter(sourceCsvPath))
        {
            var options = new MarkupConvertOptions(); // خيارات لتحويل HTML

            // تحويل وحفظ الإخراج إلى المسار المحدد
            converter.Convert(outputPath, options);
        }
    }
}
```

**توضيح:**
- **محول:** تتعامل هذه الفئة مع تحويل الملفات.
- **خيارات التحويل MarkupConvert:** يقوم بتكوين الإعدادات خصيصًا لتحويل الملفات إلى تنسيق HTML.

### خيارات تكوين المفاتيح

سيساعدك فهم هذه الخيارات على تخصيص التحويل وفقًا لاحتياجاتك:
- **تخطيط ثابت:** يحافظ على تخطيط CSV الأصلي في HTML الناتج.
- **FixedLayoutShowBorders:** يحدد ما إذا كان سيتم عرض الحدود حول الخلايا.
