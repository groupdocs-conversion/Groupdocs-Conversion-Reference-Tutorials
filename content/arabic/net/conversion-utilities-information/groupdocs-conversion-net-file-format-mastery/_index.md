## دليل التنفيذ
دعنا نوضح كيفية استرداد كل التحويلات الممكنة باستخدام GroupDocs.Conversion لـ .NET.

### نظرة عامة على الميزة: استرداد جميع التحويلات الممكنة
من الضروري فهم صيغ الملفات المتاحة للتحويل. تتيح لك هذه الميزة تحديد هذه الخيارات برمجيًا من ملف مصدر مُحدد.

#### التنفيذ خطوة بخطوة:
**1. استرداد جميع التحويلات الممكنة**
استخدم `GetAllPossibleConversions` طريقة لجلب جميع مسارات التحويل المحتملة لمستنداتك.

```csharp
var possibleConversions = Converter.GetAllPossibleConversions();
```

**2. كرر كل إمكانية تحويل**
كرر كل ملحق ملف المصدر والتحويلات المرتبطة به:

```csharp
foreach (var conversions in possibleConversions)
{
    Console.WriteLine("{0} could be converted to:", conversions.Source.Extension);
    
    foreach (var conversion in conversions.All)
    {
        var conversionType = conversion.IsPrimary ? "primary" : "secondary";
        Console.WriteLine("\t {0} as {1} conversion.", conversion.Format, conversionType);
    }
}
```

**المعلمات وقيم الإرجاع:**
- `GetAllPossibleConversions`:إرجاع مجموعة من التحويلات المحتملة.
- يتضمن كل كائن تحويل تفاصيل مثل التنسيق وما إذا كان أساسيًا أم ثانويًا.
