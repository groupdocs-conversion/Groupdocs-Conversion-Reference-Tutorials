## دليل التنفيذ

اتبع الخطوات التالية لتحويل ملف ODP إلى HTML باستخدام GroupDocs.Conversion.

### تحميل وتحويل الملف

#### ملخص

قم بتحميل مستند ODP الخاص بك وتحويله إلى تنسيق HTML عن طريق تحديد مسارات الإدخال والإخراج، إلى جانب خيارات التحويل.

**الخطوة 1: إعداد دليل الإخراج الخاص بك**

قم بتحديد المكان الذي تريد حفظ ملفاتك المحولة فيه:

```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
```

**الخطوة 2: تحديد مسار ملف الإخراج**

إنشاء مسار لملف HTML الناتج:

```csharp
string outputFile = Path.Combine(outputFolder, "odp-converted-to.html");
```

**الخطوة 3: تحميل ملف ODP وتحويله**

قم بتحميل ملف ODP الخاص بك وقم بإعداد عملية التحويل:

```csharp
using (var converter = new Converter(Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.odp")))
{
    var options = new WebConvertOptions();
    converter.Convert(outputFile, options);
}
```

- **محول**: يتولى تحميل ملف ODP. يتطلب مسار المستند المصدر.
- **خيارات تحويل الويب**:يحدد إعدادات التحويل لتنسيقات الويب مثل HTML.
