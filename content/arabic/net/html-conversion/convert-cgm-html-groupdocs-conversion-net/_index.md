## دليل التنفيذ

دعنا نتعمق في تحويل ملفات CGM إلى HTML باستخدام GroupDocs.Conversion.

### تحويل CGM إلى HTML
تتيح لك هذه الميزة تحويل تنسيقات ملفات الرسومات الحاسوبية (.cgm) إلى لغة ترميز النص التشعبي (.html).

#### نظرة عامة خطوة بخطوة
1. **إعداد دليل الإخراج**
2. **تهيئة المحول وتحميل ملف CGM**
3. **تكوين خيارات التحويل**
4. **قم بإجراء التحويل**

#### إعداد دليل الإخراج
قم بتحديد مسار دليل الإخراج الذي سيتم حفظ ملف HTML فيه:

```csharp
string outputFolder = @"YOUR_OUTPUT_DIRECTORY/";
```

#### تهيئة المحول وتحميل ملف CGM
قم بتحميل ملف CGM المصدر الخاص بك باستخدام GroupDocs.Conversion:

```csharp
using (var converter = new GroupDocs.Conversion.Converter(@"YOUR_DOCUMENT_DIRECTORY/sample.cgm"))
{
    // سيتم اتباع خطوات التحويل هنا.
}
```

#### تكوين خيارات التحويل
إنشاء خيارات التحويل الخاصة بتنسيق HTML:

```csharp
var options = new WebConvertOptions();
```

#### قم بإجراء التحويل
قم بتنفيذ التحويل وحفظ النتيجة كملف HTML:

```csharp
string outputFile = Path.Combine(outputFolder, "cgm-converted-to.html");
converter.Convert(outputFile, options);
```
