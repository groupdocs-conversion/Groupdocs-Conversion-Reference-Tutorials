## دليل التنفيذ

الآن، دعنا نستعرض عملية تحويل ملف HTM إلى تنسيق TXT باستخدام GroupDocs.Conversion.

### تحويل HTML إلى نص

تتيح لك هذه الميزة تحويل ملفات HTML إلى نص عادي. اتبع الخطوات التالية:

#### الخطوة 1: تحديد مسارات الملفات
أولاً، قم بتحديد مسارات الإدخال والإخراج للملفات الخاصة بك.

```csharp
string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.htm");
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "htm-converted-to.txt");
```

#### الخطوة 2: تحميل ملف HTML
إنشاء مثيل لـ `Converter` لتحميل ملف المصدر الخاص بك.

```csharp
using (var converter = new Converter(inputFilePath))
{
    // سيتم إضافة منطق التحويل هنا.
}
```

#### الخطوة 3: تعيين خيارات التحويل
تكوين خيارات التحويل لتنسيق TXT باستخدام `WordProcessingConvertOptions`.

```csharp
var options = new WordProcessingConvertOptions { Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt };
```

#### الخطوة 4: تنفيذ التحويل
استخدم `Convert` طريقة لتحويل ملف HTML وحفظه على هيئة نص.

```csharp
converter.Convert(outputFile, options);
```

- تأكد من أن مسار ملف HTM المدخل صحيح.
- التحقق من أذونات الدليل لكل من قراءة ملفات المصدر وكتابة ملفات الإخراج.
