## دليل التنفيذ
الآن بعد إعداد بيئتنا، دعنا نقوم بتحويل ملفات FODS إلى PDF.

### تحويل FODS إلى PDF
تتضمن الوظيفة الأساسية تحميل ملف المصدر وتحديد خيارات التحويل. إليك الطريقة:

#### الخطوة 1: تحديد مسارات الملفات
تعيين المسارات لملفات الإدخال والإخراج الخاصة بك:
```csharp
string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.fods");
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", ".\\");
string outputFile = Path.Combine(outputFolder, "fods-converted-to.pdf");
```

#### الخطوة 2: تحميل ملف FODS المصدر
استخدم GroupDocs.Conversion لتحميل مستندك:
```csharp
using (var converter = new Converter(inputFilePath))
{
    // متابعة التحويل...
}
```
ال `Converter` تتيح الفئة التعامل مع أنواع الملفات المختلفة والتحويلات.

#### الخطوة 3: تعيين خيارات التحويل
حدد الخيارات المخصصة لإخراج PDF:
```csharp
var options = new PdfConvertOptions();
```
تسمح لك هذه الخيارات بتخصيص مستند PDF الناتج وفقًا لاحتياجاتك.

#### الخطوة 4: التحويل والحفظ
قم بتنفيذ عملية التحويل وحفظ النتيجة:
```csharp
converter.Convert(outputFile, options);
```
تنهي هذه الخطوة عملية التحويل عن طريق كتابة ملف PDF الناتج إلى المسار المحدد.
