## دليل التنفيذ

سنقوم بتقسيم التنفيذ إلى أقسام منطقية للتأكد من فهمك لكل ميزة بشكل كامل.

### تحويل أوراق محددة إلى PDF

تتيح لك هذه الميزة تحويل أوراق محددة من جدول بيانات Excel إلى ملف PDF.

#### ملخص

الهدف هنا هو تحديد أوراق معينة حسب فهارسها وتحويلها إلى مستند PDF واحد. سنستخدم وظائف مكتبة GroupDocs.Conversion القوية لهذا الغرض.

#### التنفيذ خطوة بخطوة

1. **تحديد خيارات التحميل**
   
   حدد الأوراق التي تريد تحويلها باستخدام `SpreadsheetLoadOptions`. هنا، سوف نختار الورقتين الأولى والثالثة:
    
   ```csharp
   Func<LoadContext, LoadOptions> getLoadOptions = loadContext => new SpreadsheetLoadOptions
   {
       SheetIndexes = new[] { 0, 2 }, // تحديد الأوراق ذات الفهرسين 0 و2
       OnePagePerSheet = true          // إنشاء صفحة PDF واحدة لكل ورقة جدول بيانات
   };
   ```
    
   **لماذا هذا مهم:** من خلال تحديد `SheetIndexes`، يمكنك التأكد من معالجة البيانات الضرورية فقط، وتحسين وقت التحويل وحجم ملف الإخراج.

2. **تهيئة المحول**
   
   قم بإنشاء مثيل محول لمستند Excel الخاص بك باستخدام خيارات التحميل المحددة:
    
   ```csharp
   string outputFolder = "YOUR_OUTPUT_DIRECTORY";
   string outputFile = Path.Combine(outputFolder, "converted.pdf");

   using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY\\sample.xlsx", getLoadOptions))
   {
       Console.WriteLine("Converter initialized for specific sheets.");
   }
   ```
    
   **لماذا هذا مهم:** يؤثر التهيئة باستخدام خيارات التحميل بشكل مباشر على عملية التحويل من خلال التصفية المسبقة للأجزاء التي تريد تحويلها من المستند.

3. **تحديد خيارات تحويل PDF**
   
   إعداد إعدادات التحويل باستخدام `PdfConvertOptions`:
    
   ```csharp
   PdfConvertOptions options = new PdfConvertOptions();
   
   // تحويل وحفظ الأوراق المحددة إلى مستند PDF واحد
   converter.Convert(outputFile, options);
   Console.WriteLine("Conversion completed successfully.");
   ```
    
   **لماذا هذا مهم:** يتيح لك تكوين هذه الخيارات تخصيص تنسيق الإخراج وعملية التحويل.
