## دليل التنفيذ

الآن بعد أن أصبح الإعداد جاهزًا، دعنا ننفذ ميزة تحويل ملفات XLS إلى PDF.

### تحميل ملف المصدر

حدد مسارات مجلدات الإدخال والإخراج. استبدل `'YOUR_DOCUMENT_DIRECTORY'` و `'YOUR_OUTPUT_DIRECTORY'` مع المسارات الفعلية على نظامك:
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// قم بتحديد المسار لملف XLS المصدر وملف PDF الناتج
string xlsFilePath = Path.Combine(documentDirectory, "sample.xls");
string pdfOutputFile = Path.Combine(outputDirectory, "xls-converted-to.pdf");
```

### إجراء التحويل

لتحويل ملف Excel إلى PDF:
```csharp
try
{
    // تحميل ملف XLS المصدر
    using (var converter = new Converter(xlsFilePath))
    {
        // تعيين خيارات التحويل لتنسيق PDF
        var options = new PdfConvertOptions();

        // قم بإجراء التحويل وحفظ ملف PDF الناتج
        converter.Convert(pdfOutputFile, options);
    }
}
catch(Exception ex)
{
    Console.WriteLine("An error occurred: " + ex.Message);
}
```

#### توضيح

- **فئة المحول**:المسؤول عن تحميل مستندك.
- **خيارات تحويل PDF**:يشير إلى أننا نريد تحويل مستندنا إلى تنسيق PDF.
- **معالجة الأخطاء**:تضمن كتلة try-catch اكتشاف أي مشكلات أثناء التحويل وتسجيلها.

### نصائح استكشاف الأخطاء وإصلاحها

- تأكد من صحة مسارات الملفات لمنع `FileNotFoundException`.
- تأكد من أن إصدار مكتبة GroupDocs.Conversion متوافق مع بيئة .NET الخاصة بك.
