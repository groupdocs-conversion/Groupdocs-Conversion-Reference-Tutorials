## دليل التنفيذ

### تحميل ملف BMP المصدر

يوضح هذا القسم كيفية تحميل ملف BMP المصدر، وهو أمر بالغ الأهمية قبل تحويله.

#### الخطوة 1: تحديد مسار ملف BMP
أولاً، حدد مكان تخزين ملف BMP. تأكد من ضبط المسار الصحيح في تطبيقك:

```csharp
string bmpFilePath = "YOUR_DOCUMENT_DIRECTORY/sample.bmp";
```

#### الخطوة 2: تحميل ملف BMP

قم بتحميل BMP في مثيل المحول لبدء عملية التحويل.

```csharp
using (var converter = new GroupDocs.Conversion.Converter(bmpFilePath))
{
    // تم الآن تحميل ملف BMP وهو جاهز للتحويل.
}
```
**توضيح:** هنا، نقوم بإنشاء مثيل لـ `Converter` باستخدام مسار الملف لتحميل صورتنا. يتيح لنا هذا الإعداد إجراء عمليات إضافية على هذا المستند.

### تحويل BMP إلى PDF

بمجرد تحميل ملف المصدر، حان الوقت لتحويله إلى تنسيق PDF.

#### الخطوة 1: تحديد مسار الإخراج
قم بإعداد المكان الذي سيتم حفظ ملف PDF المُحوّل فيه:

```csharp
string pdfOutputPath = "YOUR_OUTPUT_DIRECTORY/bmp-converted-to.pdf";
```

#### الخطوة 2: تعيين خيارات التحويل
قم بتكوين إعدادات التحويل خصيصًا لإخراج PDF:

```csharp
var options = new PdfConvertOptions();
```
**توضيح:** `PdfConvertOptions` يوفر معلمات تخصيص مختلفة مثل الهوامش وحجم الصفحة والمزيد.

#### الخطوة 3: تنفيذ التحويل
تنفيذ عملية التحويل من BMP إلى PDF:

```csharp
using (var converter = new GroupDocs.Conversion.Converter(bmpFilePath))
{
    // تحويل ملف BMP المحمّل إلى تنسيق PDF
    converter.Convert(pdfOutputPath, options);
}
```
**توضيح:** ال `Convert` تنفذ الطريقة التحويل الفعلي، مع أخذ مسار الوجهة وإعدادات التحويل كمعلمات.
