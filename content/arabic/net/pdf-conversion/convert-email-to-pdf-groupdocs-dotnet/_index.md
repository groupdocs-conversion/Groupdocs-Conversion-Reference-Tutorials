## دليل التنفيذ

دعنا نقسم التنفيذ إلى أقسام منطقية، مع التركيز على تحويل البريد الإلكتروني مع مرفقاته.

### تكوين خيارات التحميل

أولاً، قم بتكوين خيارات التحميل لتحديد كيفية التعامل مع مستندات بريدك الإلكتروني أثناء التحويل. يتضمن ذلك ضبط خصائص مثل `ConvertOwner` و `ConvertOwned`.

```csharp
Func<LoadContext, LoadOptions> getLoadOptions = loadContext => new EmailLoadOptions
{
    ConvertOwner = true,
    ConvertOwned = true,
    Depth = 2 // يتضمن المرفقات في عملية التحويل
};
```

### تهيئة المحول

بعد ذلك، قم بتهيئة `Converter` قم بإنشاء فصل دراسي باستخدام مستند البريد الإلكتروني الخاص بك وخيارات التحميل المحددة مسبقًا.

```csharp
using (Converter converter = new Converter(inputFilePath, getLoadOptions))
{
    int index = 1; // فهرس لتسمية ملفات الإخراج
    
    PdfConvertOptions options = new PdfConvertOptions(); // تعيين خيارات التحويل إلى PDF
    
    // قم بتعريف وظيفة استرجاع لحفظ كل مستند أو مرفق تم تحويله
    converter.Convert((SaveContext saveContext) =>
    {
        string fileName = index == 1 ? "converted.pdf" : $"converted-attachment-{index - 1}.pdf";
        index++;
        string outputFile = Path.Combine(outputFolder, fileName); // إنشاء مسار الإخراج الكامل
        return new FileStream(outputFile, FileMode.Create); // إنشاء تدفق ملف لكل مستند مُحوّل
    }, options);
}
```

#### توضيح:
- **خيارات التحميل**:يتحكم في كيفية معالجة البريد الإلكتروني ومرفقاته.
- **فئة المحول**:إدارة عملية التحويل من الإدخال إلى PDF.
- **خيارات تحويل PDF**:يحدد أن تنسيق الإخراج يجب أن يكون PDF.
- **استدعاء حفظ السياق**:يتولى تسمية الملفات وتخزينها لكل مستند أو مرفق تم تحويله.

### نصائح استكشاف الأخطاء وإصلاحها
تأكد من جميع المسارات في `inputFilePath` و `outputFolder` تم ضبطها بشكل صحيح. تأكد من أن معلمة العمق كافية لتشمل جميع المرفقات.
