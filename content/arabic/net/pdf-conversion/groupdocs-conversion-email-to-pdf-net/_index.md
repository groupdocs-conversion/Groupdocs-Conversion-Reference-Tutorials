# تحويل رسائل البريد الإلكتروني إلى PDF باستخدام GroupDocs.Conversion لـ .NET: دليل خطوة بخطوة

## مقدمة

**ما سوف تتعلمه:**

## التهيئة والإعداد الأساسي
فيما يلي كيفية تهيئة المحول باستخدام الإعداد الأساسي:

```csharp
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

string sourceFilePath = "YOUR_DOCUMENT_DIRECTORY/SAMPLE_EML";
string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "converted.pdf");

// خيارات التحميل لتحويل البريد الإلكتروني
Func<LoadContext, LoadOptions> getLoadOptions = loadContext => new EmailLoadOptions { ConvertOwned = false };

using (Converter converter = new Converter(sourceFilePath, getLoadOptions))
{
    PdfConvertOptions options = new PdfConvertOptions();
    converter.Convert(outputFile, options);
}
```

## دليل التنفيذ
في هذا القسم، سنتناول كل خطوة من خطوات تحويل ملف البريد الإلكتروني إلى ملف PDF.

### تحميل ملف البريد الإلكتروني مع خيارات محددة
**ملخص:**
يتيح لك إعداد خيارات التحميل التحكم في كيفية تعامل عملية التحويل مع ملفات بريدك الإلكتروني. هنا يمكنك تحديد تفضيلاتك، مثل تحويل الخصائص المملوكة.

```csharp
Func<LoadContext, LoadOptions> getLoadOptions = loadContext => new EmailLoadOptions
{
    ConvertOwned = false // لا تقم بتحويل العقارات المملوكة افتراضيًا
};
```
**توضيح:**
- `ConvertOwned`:عند ضبطه على "خطأ"، فإنه يضمن تحويل سمات البريد الإلكتروني القياسية دون تحويل أي تنسيقات خاصة.

### تهيئة المحول وتعيين خيارات التحويل
**ملخص:**
العمل الرئيسي يحدث هنا. تقوم بتهيئة `Converter` الفئة مع مسار ملف المصدر وخيارات التحميل.

```csharp
using (Converter converter = new Converter(sourceFilePath, getLoadOptions))
{
    PdfConvertOptions options = new PdfConvertOptions();
    converter.Convert(outputFile, options);
}
```
**توضيح:**
- **حدود**: ال `sourceFilePath` يحدد ملف البريد الإلكتروني المراد تحويله، و `getLoadOptions` يوفر إعدادات التحويل.
- **قيمة الإرجاع**:تعيد هذه العملية ملف PDF الموجود في `outputFile`.

### خيارات تكوين المفاتيح
تكوين الخاص بك `PdfConvertOptions` يتيح لك تخصيص المخرجات. يمكنك تحديد حجم الصفحة والهوامش وغيرها بناءً على متطلباتك.
