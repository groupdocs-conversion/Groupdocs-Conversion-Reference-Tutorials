```csharp
// قم بتهيئة المحول باستخدام ملف RAR

{
    // معالجة كل ملف في الأرشيف

```

#### الخطوة 2: تكوين خيارات التحويل
```csharp
PdfConvertOptions options = new PdfConvertOptions();

// تخصيص إعدادات التحويل حسب الحاجة
options.MarginTop = 10;
options.MarginBottom = 10;

// تحويل وحفظ ملف PDF
converter.Convert("output.pdf", options);
```

- **تحميل أرشيف RAR**:تهيئة `Converter` الكائن مع مسار ملف RAR الخاص بك.
