## دليل التنفيذ
### تحويل ملف AI إلى صيغة TXT
تتيح لك هذه الميزة تحويل ملفات Adobe Illustrator إلى تنسيق نص عادي لتسهيل معالجة البيانات أو تحليلها.

#### الخطوة 1: تعيين دليل الإخراج وتحديد مسار الإخراج
ابدأ بتحديد دليل الإخراج الذي سيتم حفظ الملف المُحوّل فيه. استبدل `YOUR_OUTPUT_DIRECTORY` مع المسار الفعلي على نظامك.

```csharp
string outputFolder = @"C:\OutputDirectory\";
string outputFile = System.IO.Path.Combine(outputFolder, "ai-converted-to.txt");
```

#### الخطوة 2: تحميل ملف AI المصدر
قم بتحميل ملف AI المصدر الخاص بك باستخدام `GroupDocs.Conversion.Converter` الصف. استبدال `YOUR_DOCUMENT_DIRECTORY` مع المسار إلى ملف الذكاء الاصطناعي الخاص بك.

```csharp
using (var converter = new GroupDocs.Conversion.Converter(@"C:\DocumentDirectory\sample.ai"))
{
    // وسوف يتبع ذلك منطق التحويل.
}
```

#### الخطوة 3: تعيين خيارات التحويل
حدّد خيارات التحويل لتحديد تنسيق الإخراج المطلوب TXT. هذا ضروري لتحديد كيفية تحويل ملفك.

```csharp
var options = new GroupDocs.Conversion.Options.Convert.WordProcessingConvertOptions 
{ 
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt 
};
```

#### الخطوة 4: تنفيذ التحويل
وأخيرًا، قم بتنفيذ عملية التحويل وحفظ الناتج كملف نصي باستخدام الإعدادات المحددة.

```csharp
converter.Convert(outputFile, options);
```
