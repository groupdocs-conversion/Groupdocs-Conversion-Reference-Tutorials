### تحويل PST إلى XLS

تعمل هذه الميزة على تحويل ملف جدول التخزين الشخصي (.pst) إلى جدول بيانات Excel (.xls).

#### ملخص
تتضمن العملية تهيئة المحول باستخدام ملف PST المصدر الخاص بك وتكوينه لإخراجه بتنسيق XLS.

**خطوات التنفيذ:**

##### الخطوة 1: تهيئة المحول
قم بإعداد المحول باستخدام المسار إلى ملف PST الخاص بك:

```csharp
string sourceFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.pst");
using (var converter = new GroupDocs.Conversion.Converter(sourceFilePath))
{
    // سيتم اتباع خطوات التكوين الأخرى.
}
```

##### الخطوة 2: تكوين خيارات التحويل
قم بتحديد خيارات التحويل لتحديد تنسيق الإخراج كـ XLS:

```csharp
SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Xls };
```

##### الخطوة 3: تنفيذ التحويل
قم بتنفيذ التحويل وحفظ النتيجة:

```csharp
string outputFileTemplate = Path.Combine("YOUR_OUTPUT_DIRECTORY", "pst-converted-{0}-to.xls");
converter.Convert(
    (SaveContext saveContext) => new FileStream(string.Format(outputFileTemplate, counter++), FileMode.Create),
    options
);
```

**توضيح:**
- **سياق التحميل:** تحديد المعالجة الخاصة لملفات OST.
- **خيارات التحويل:** يحدد تنسيق الهدف كـXLS.

تتيح هذه الميزة تكوين إعدادات التحميل استنادًا إلى نوع الملف والظروف.
