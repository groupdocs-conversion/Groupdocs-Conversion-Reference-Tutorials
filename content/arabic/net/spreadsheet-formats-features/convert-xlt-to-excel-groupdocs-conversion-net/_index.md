#### الخطوة 1: تحديد أدلة الإدخال والإخراج
إعداد مسارات الملفات الخاصة بك:
```csharp

string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "xlt-converted-to.xls");
```
يحدد مقتطف التعليمات البرمجية هذا المسار لكل من ملف XLT المدخل وملف XLS المخرج.

#### الخطوة 2: تحميل ملف XLT

```csharp

{
```
