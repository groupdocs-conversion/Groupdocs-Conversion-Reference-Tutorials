#### التهيئة الأساسية والإعداد باستخدام C#
فيما يلي كيفية تهيئة GroupDocs.Conversion في تطبيقك:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

namespace DocumentConversionToSpreadsheet
{
    internal static class ConvertDocumentToSpreadsheet
    {
        public static void Run()
        {
            // قم بتحديد مسار دليل الإخراج باستخدام عنصر نائب.
            string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");

            // قم بدمج مجلد الإخراج واسم الملف لإنشاء المسار الكامل للملف المحول.
            string outputFile = Path.Combine(outputFolder, "converted.xlsx");

            // قم بتهيئة كائن المحول باستخدام مسار المستند المصدر باستخدام عنصر نائب.
            using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY"))
            {
                // قم بإنشاء مثيل لـ SpreadsheetConvertOptions لتحديد خيارات التحويل.
                SpreadsheetConvertOptions options = new SpreadsheetConvertOptions();

                // قم بإجراء التحويل من مستند الإدخال إلى ملف الإخراج المحدد باستخدام الخيارات.
                converter.Convert(outputFile, options);
            }
        }
    }
}
```

#### شرح المعلمات والطرق

### ضبط خيارات التحويل

#### ملخص
ال `SpreadsheetConvertOptions` تتيح لك الفئة تحديد معلمات تحويل متنوعة. بينما يستخدم مثالنا الأساسي الإعدادات الافتراضية، يمكنك تخصيص هذه الخيارات حسب الحاجة.

```csharp
// قم بإنشاء مثيل لـ SpreadsheetConvertOptions لتحديد خيارات التحويل.
SpreadsheetConvertOptions options = new SpreadsheetConvertOptions();

// قم بإجراء التحويل من مستند الإدخال إلى ملف الإخراج المحدد باستخدام الخيارات.
converter.Convert(outputFile, options);
```

#### خيارات تكوين المفاتيح
- **الإعدادات الافتراضية**يستخدم الكود الإعدادات الافتراضية للتبسيط. للحصول على إعدادات متقدمة، مثل تحديد الأوراق أو الصفحات، راجع وثائق GroupDocs.
