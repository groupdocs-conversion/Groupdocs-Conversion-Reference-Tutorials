## دليل التنفيذ

الآن بعد أن قمت بإعداد بيئتك، دعنا ننتقل إلى تنفيذ الميزة لتحويل ملفات STL إلى تنسيق XLSX.

### تحويل STL إلى XLSX

#### ملخص
الوظيفة الأساسية هنا هي تحويل ملف STL إلى جدول بيانات Excel. يتضمن ذلك قراءة بيانات ثلاثية الأبعاد من ملف STL وعرضها في شكل جدول داخل ملف XLSX.

#### التنفيذ خطوة بخطوة

##### **1. تحديد مسارات الإدخال والإخراج**
قم بتحديد المسارات لملف STL المدخل وملف XLSX المخرج:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputFile = Path.Combine("YOUR_OUTPUT_DIRECTORY", "output.xlsx");
```

##### **2. قم بتحميل ملف STL**
قم بتحميل ملف STL الخاص بك باستخدام `Converter` فصل.

```csharp
using (var converter = new Converter(Path.Combine(documentDirectory, "file.stl")))
{
    Console.WriteLine("STL file loaded.");
}
```

##### **3. تعيين خيارات التحويل**
حدد خيارات التحويل لتنسيق XLSX:

```csharp
var convertOptions = new SpreadsheetConvertOptions();
```

##### **4. قم بإجراء التحويل**
قم بتنفيذ التحويل وحفظ الناتج كملف XLSX.

```csharp
converter.Convert(outputFile, convertOptions);
Console.WriteLine("Conversion to XLSX completed.");
```

#### خيارات تكوين المفاتيح
- `SpreadsheetConvertOptions` يتيح لك ضبط معلمات متنوعة، مثل أرقام الصفحات أو أوراق محددة. خصّص هذه الخيارات وفقًا لاحتياجاتك.

#### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من أن مسار ملف STL المدخل صحيح.
- التحقق من وجود أذونات كافية في دلائل الإخراج.
- التحقق من تثبيت جميع التبعيات الضرورية.
