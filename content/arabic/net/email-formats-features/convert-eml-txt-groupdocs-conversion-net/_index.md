# كيفية تحويل ملفات EML إلى TXT باستخدام GroupDocs.Conversion لـ .NET

## دليل التنفيذ

### الميزة: تحويل EML إلى TXT
تتيح لك هذه الميزة تحويل ملف EML إلى تنسيق TXT أكثر قابلية للإدارة باستخدام GroupDocs.Conversion.

#### الخطوة 1: تحديد مسارات الملفات
ابدأ بتحديد مسارات مجلدات الإدخال والإخراج. هذا يُساعد تطبيقك على معرفة مكان القراءة والكتابة.
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY"; // حدد دليل ملف EML الخاص بك
string outputDirectory = "YOUR_OUTPUT_DIRECTORY"; // حدد دليل إخراج TXT الخاص بك

// تحديد أسماء الملفات
class Program
{
    static void Main()
    {
        string inputFile = Path.Combine(documentDirectory, "sample.eml");
        string outputFile = Path.Combine(outputDirectory, "eml-converted-to.txt");
    }
}
```
#### الخطوة 2: تحميل ملف EML المصدر
استخدم GroupDocs.Conversion لتحميل ملف EML المصدر الخاص بك.
```csharp
using (var converter = new Converter(inputFile))
{
    // سيتم تحديد خيارات التحويل في الخطوة التالية.
}
```
*لماذا تستخدم `using`؟ فهو يضمن التخلص من الموارد بشكل صحيح بمجرد اكتمال التحويل.*

#### الخطوة 3: تحديد خيارات التحويل
قم بتكوين التحويل الخاص بك لإخراج ملف TXT باستخدام إعدادات التنسيق المحددة.
```csharp
var options = new WordProcessingConvertOptions { Format = FileType.Txt };
```
*ال `WordProcessingConvertOptions` توفر الفئة معلمات تكوين مختلفة، مما يسمح لك بتحديد تنسيق الإخراج المطلوب.*

#### الخطوة 4: التحويل والحفظ
قم بإجراء التحويل وحفظ ملف TXT الخاص بك بالإعدادات التي قمت بتكوينها.
```csharp
converter.Convert(outputFile, options);
```
### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من تحديد المسارات بشكل صحيح لتجنب `FileNotFoundException`.
- تأكد من أن إصدار GroupDocs.Conversion يتطابق مع تبعيات مشروعك.
