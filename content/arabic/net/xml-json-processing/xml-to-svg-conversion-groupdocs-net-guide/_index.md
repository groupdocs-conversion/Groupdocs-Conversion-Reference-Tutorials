### تحويل XML إلى SVG

يوضح هذا القسم كيفية تحويل ملف XML إلى صيغة SVG بسهولة. تتضمن العملية تحميل ملف XML وتحديد صيغة الإخراج.

#### تحميل ملف XML المصدر

ابدأ بتحديد المسارات لملفات الإدخال والإخراج الخاصة بك:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY"; // تحديد المسار إلى دليل المستندات الخاص بك
string outputDirectory = "YOUR_OUTPUT_DIRECTORY"; // حدد المكان الذي تريد حفظ الإخراج فيه

// تأكد من وجود دليل الإخراج أو قم بإنشائه إذا لزم الأمر
if (!Directory.Exists(outputDirectory))
{
    Directory.CreateDirectory(outputDirectory);
}

string inputFilePath = Path.Combine(documentDirectory, "sample.xml");
string outputFile = Path.Combine(outputDirectory, "xml-converted-to.svg");
```

#### تعيين خيارات التحويل

بعد ذلك، قم بتهيئة المحول وإعداد خيارات التحويل:

```csharp
using (var converter = new Converter(inputFilePath))
{
    // حدد تنسيق SVG كنوع الإخراج
    var options = new PageDescriptionLanguageConvertOptions
    {
        Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Svg
    };

    // تنفيذ التحويل وحفظ ملف الإخراج
    converter.Convert(outputFile, options);
}
```

- **ملف الإخراج:** مسار الوجهة لملف SVG المُحوّل.
- **وصف الصفحةاللغةالتحويلالخيارات:** يحدد تنسيق الهدف للتحويل.
