## دليل التنفيذ
### تحميل ملف PPSX المصدر
#### ملخص:
إن تحميل ملف PowerPoint المصدر الخاص بك هو الخطوة الأولى لتحويله إلى تنسيق PSD.

#### التعليمات خطوة بخطوة:
**H3: تهيئة كائن المحول**
```csharp
using System;
using GroupDocs.Conversion;

namespace Example
{
    public class LoadSourcePpsx
    {
        // استبدل 'YOUR_DOCUMENT_DIRECTORY' بمسار المستند الفعلي الخاص بك.
        private const string SourceFilePath = "@YOUR_DOCUMENT_DIRECTORY/sample.ppsx";
        
        public static void Run()
        {
            using (Converter converter = new GroupDocs.Conversion.Converter(SourceFilePath))
            {
                // تم الآن تحميل الملف لعمليات التحويل
            }
        }
    }
}
```
**توضيح:**
- **مسار الملف المصدر**:تأكد من أن هذا يشير إلى الدليل الصحيح الذي توجد به ملفات PPSX.
- `using` تضمن هذه العبارة التخلص السليم من الموارد، مما يساعد في إدارة الذاكرة.

### ضبط خيارات التحويل لتنسيق PSD
#### ملخص:
يعد تكوين إعدادات التحويل أمرًا بالغ الأهمية لتحديد تنسيق الإخراج.

#### التعليمات خطوة بخطوة:

```csharp
using System;
using GroupDocs.Conversion.Options.Convert;

namespace Example
{
    public class SetConversionOptionsPsd
    {
        public static void Run()
        {
            ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd };
            
            // تحتوي الآن "الخيارات" على تكوينات التحويل إلى PSD.
        }
    }
}
```
**توضيح:**
- **خيارات تحويل الصورة**:يحدد هذا الكائن تنسيق الصورة الناتجة (PSD في هذه الحالة).
- `Format`:يحدد نوع الملف المستهدف، وهو أمر ضروري لتحديد نتائج التحويل.

### تحويل PPSX إلى PSD
#### ملخص:
بعد تحميل المصدر وتعيين الخيارات، قم بإجراء التحويل الفعلي من PPSX إلى PSD.

#### التعليمات خطوة بخطوة:
**H3: تنفيذ التحويل**
```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

namespace Example
{
    public class ConvertPpsxToPsdFeature
    {
        // استبدل 'YOUR_OUTPUT_DIRECTORY' بمسار الإخراج المطلوب.
        private const string OutputDirectory = "@YOUR_OUTPUT_DIRECTORY";
        
        public static void Run()
        {
            string outputFileTemplate = Path.Combine(OutputDirectory, "converted-page-{0}.psd");
            
            Func<SavePageContext, Stream> getPageStream = savePageContext => new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);

            using (Converter converter = new GroupDocs.Conversion.Converter("YOUR_DOCUMENT_DIRECTORY/sample.ppsx"))
            {
                ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd };
                
                // تحويل كل شريحة إلى ملف PSD
                converter.Convert(getPageStream, options);
            }
        }
    }
}
```
**توضيح:**
- **قالب ملف الإخراج**:يحدد اتفاقيات التسمية لملفات الإخراج.
- `getPageStream`:تولد الوظيفة تدفقات لكل صفحة تم تحويلها؛ وهو أمر ضروري لحفظ النتائج.
- **المحول.تحويل()**:ينفذ التحويل باستخدام الخيارات المحددة.
