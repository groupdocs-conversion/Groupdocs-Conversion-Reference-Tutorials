## دليل التنفيذ

### تحميل وتحويل ملف TEX إلى PNG
توضح هذه الميزة كيفية تحميل ملف TEX وتحويله إلى تنسيق PNG باستخدام GroupDocs.Conversion لـ .NET.

#### إعداد دليل الإخراج ومسار قالب الملف
أولاً، قم بتحديد المكان الذي سيتم حفظ الملفات المحولة فيه:
```csharp
string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.png");

// وظيفة للحصول على التدفق لكل صفحة
Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```
#### تحميل ملف TEX المصدر
قم بتحميل ملف TEX الخاص بك باستخدام `Converter` فصل:
```csharp
using (Converter converter = new Converter(inputFile))
{
    // سيتم تعيين خيارات التحويل في الخطوة التالية.
}
```
#### تعيين خيارات التحويل لتنسيق PNG
قم بتكوين إعدادات التحويل لإخراج تنسيق PNG:
```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Png };
```
#### قم بإجراء التحويل
تنفيذ عملية التحويل:
```csharp
converter.Convert(getPageStream, options);
```
