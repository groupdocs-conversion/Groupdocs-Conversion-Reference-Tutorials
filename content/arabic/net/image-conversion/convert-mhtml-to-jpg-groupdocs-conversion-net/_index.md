## دليل التنفيذ
### تحويل MHTML إلى JPG
سوف يرشدك هذا القسم خلال تحويل ملف MHTML إلى صور JPG باستخدام GroupDocs.Conversion لـ .NET.

#### الخطوة 1: تحديد مسارات الملفات وقالب الإخراج
حدّد مسارات المصدر والإخراج. هذا يضمن حفظ كل صفحة على حدة.

```csharp
string sourceFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.mhtml");
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");
```

#### الخطوة 2: إنشاء وظيفة لتوليد تدفق الصفحة
حدّد دالةً تُولّد تدفقًا لكل صفحة تحويل. هذا ضروريٌّ لحفظ كل صفحة كملف JPG منفصل.

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

#### الخطوة 3: تحميل ملف MHTML وتحويله
قم بتحميل ملف المصدر الخاص بك وقم بتكوين خيارات التحويل لاستهداف تنسيق JPG.

```csharp
using (Converter converter = new Converter(sourceFilePath))
{
    ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Jpg };
    converter.Convert(getPageStream, options);
}
```
**خيارات تكوين المفتاح:**
- `ImageConvertOptions`:يشير إلى أننا نقوم بالتحويل إلى تنسيق صورة.
- `Format = ImageFileType.Jpg`:يضبط تنسيق الهدف على أنه JPG.

#### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من تحديد مسارات الملفات بشكل صحيح وإمكانية الوصول إليها.
- تأكد من أن لديك أذونات الكتابة لدليل الإخراج.
