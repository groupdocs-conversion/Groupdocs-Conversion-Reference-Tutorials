## دليل التنفيذ

دعونا نقسم عملية التحويل إلى خطوات قابلة للإدارة.

### الميزة: تحويل PNG إلى PSD

تتيح لك هذه الميزة تحويل ملف PNG إلى تنسيق PSD باستخدام GroupDocs.Conversion.

#### ملخص

ستتعلم كيفية إعداد بيئتك، وإنشاء التدفقات اللازمة لملفات الإخراج، وإجراء التحويل الفعلي.

#### التنفيذ خطوة بخطوة

**1. إعداد دليل الإخراج**

حدد المكان الذي سيتم حفظ الملفات المحولة فيه:

```csharp
string outputFolder = @"YOUR_OUTPUT_DIRECTORY\"; // قم بتعيين دليل الإخراج المطلوب هنا
```

**2. تحميل ملف الإدخال**

حدد المسار إلى ملف PNG المدخل الخاص بك:

```csharp
string inputFile = @"YOUR_DOCUMENT_DIRECTORY\sample.png"; // المسار إلى ملف PNG المدخل
```

**3. إنشاء تدفق لكل صفحة يتم تحويلها**

تعمل هذه الوظيفة على إنشاء تدفق لكل صفحة مُحوّلة، مما يضمن التعامل السليم مع الملف:

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(Path.Combine(outputFolder, $"converted-page-{savePageContext.Page}.psd"), FileMode.Create);
```

**4. تحميل ملف PNG المصدر وتكوين خيارات التحويل**

قم بتهيئة المحول وإعداد إعدادات التحويل:

```csharp
using (Converter converter = new Converter(inputFile))
{
    ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Psd };

    // قم بإجراء التحويل من صيغة PNG إلى PSD.
    converter.Convert(getPageStream, options);
}
```

#### شرح الكود

- **حفظ سياق الصفحة**:يوفر سياقًا لكل صفحة يتم تحويلها.
- **خيارات تحويل الصورة**:تكوين الإعدادات الخاصة بتنسيقات الصور.
