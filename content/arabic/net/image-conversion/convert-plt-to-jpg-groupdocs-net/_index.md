## دليل التنفيذ

### تحويل PLT إلى JPG
تتيح لك هذه الميزة تحويل ملفات PLT إلى صيغة JPG، مما يجعل من السهل مشاركة التصميمات وعرضها دون الحاجة إلى برامج متخصصة.

#### تحميل ملف PLT المصدر
أولاً، حدد مسارات المجلدات لمستنداتك ومخرجاتك. تتضمن هذه الخطوة تحميل ملف المصدر باستخدام `Converter` فصل:
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY/";

using (Converter converter = new Converter(Path.Combine(documentDirectory, "yourfile.plt")))
{
    // منطق التحويل سوف يذهب هنا
}
```

#### تعيين خيارات التحويل لتنسيق JPG
قم بتحديد خيارات التحويل لتحديد ما إذا كنت تريد الإخراج بتنسيق JPG:
```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Jpg };
```

#### تعريف دالة تدفق الإخراج
أنشئ دالة لمعالجة تدفق الإخراج لكل صفحة من ملف PLT. هذا يضمن حفظ كل صفحة مُحوّلة كملف JPG منفصل:
```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(Path.Combine(outputDirectory, $"converted-page-{savePageContext.Page}.jpg"), FileMode.Create);
```

#### تنفيذ التحويل
أخيرًا، قم بإجراء التحويل عن طريق استدعاء `Convert` الطريقة مع الخيارات المحددة لديك:
```csharp
converter.Convert(getPageStream, options);
```
