## دليل التنفيذ

الآن بعد إعداد كل شيء، دعنا ننتقل إلى تحويل ملفات BMP إلى تنسيق TEX.

### تحويل ملف BMP إلى صيغة TEX

تستخدم هذه الميزة GroupDocs.Conversion لتحويل صور BMP إلى مستندات TEX بسلاسة.

#### تحميل ملف BMP المصدر
أولاً، قم بتحميل ملف BMP الخاص بك باستخدام `Converter` هذه الخطوة تُمهّد عملية التحويل:

```csharp
using (var converter = new GroupDocs.Conversion.Converter(inputFile))
{
    // منطق التحويل يذهب هنا
}
```

#### تعيين خيارات التحويل لتنسيق TEX
بعد ذلك، قم بتحديد تنسيق الإخراج عن طريق الإعداد `PageDescriptionLanguageConvertOptions`:

```csharp
PageDescriptionLanguageConvertOptions options = new PageDescriptionLanguageConvertOptions { Format = PageDescriptionLanguageFileType.Tex };
```

ال `options.Format` يُحدد هذا المُعامل أننا نُحوّل إلى TEX. هذا يُوجّه عملية التحويل.

#### تحويل وحفظ
وأخيرًا، قم بتنفيذ التحويل وحفظ الناتج:

```csharp
converter.Convert(outputFile, options);
```

تقوم هذه الطريقة بتحويل ملف BMP إلى صيغة TEX وحفظه في الموقع المحدد.
