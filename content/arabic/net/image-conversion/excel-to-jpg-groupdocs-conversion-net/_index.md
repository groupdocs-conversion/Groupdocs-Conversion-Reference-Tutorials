# تحويل Excel إلى JPG في .NET: دليل خطوة بخطوة باستخدام GroupDocs.Conversion

## مقدمة

يُعد عرض بيانات Excel بصريًا أمرًا أساسيًا للتواصل الفعال، خاصةً عندما لا يتوفر لدى الجهات المعنية برنامج جداول بيانات. يقدم هذا الدليل شرحًا تفصيليًا لتحويل ملفات Excel (.xls) إلى صور JPEG باستخدام GroupDocs.Conversion for .NET، وهي مكتبة قوية مصممة لتحويل الملفات بسلاسة.

**ما سوف تتعلمه:**
- إعداد GroupDocs.Conversion في مشروع .NET الخاص بك.
- تحويل ملف Excel إلى JPEG خطوة بخطوة.
- تخصيص تكوينات الإخراج.
- تحسين الأداء باستخدام مجموعات البيانات الكبيرة.

لنبدأ بالتأكد من أن لديك المتطلبات الأساسية اللازمة!

## دليل التنفيذ

اتبع الخطوات التالية لتحويل ملف Excel إلى صور JPEG باستخدام GroupDocs.Conversion لـ .NET.

### الخطوة 1: تحضير دالة تدفق الإخراج

إنشاء وظيفة للتعامل مع كتابة كل صفحة مُحوّلة كملف JPG منفصل:

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

**توضيح:**
- `getPageStream` هو مندوب يقوم بإنشاء تدفق لكل صفحة.
- يتضمن مسار الإخراج `{0}` للتمييز بين الصفحات.

### الخطوة 2: تحميل ملف XLS

قم بتحميل ملف Excel الخاص بك باستخدام فئة GroupDocs.Converter:

```csharp
using (Converter converter = new Converter(inputFilePath))
{
    // المضي قدما في خطوات التحويل
}
```

**توضيح:**
- ال `Converter` يتعامل الكائن مع عملية التحويل.
- تأكد من ذلك `inputFilePath` هو الصحيح.

### الخطوة 3: تكوين خيارات التحويل

إعداد الخيارات لتحويل الملفات إلى تنسيق JPG:

```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Jpg };
```

**توضيح:**
- `ImageConvertOptions` يحدد الإعدادات لتحويلات الصور.
- ال `Format` تقوم الخاصية بتعيين الإخراج إلى JPEG.

### الخطوة 4: تنفيذ التحويل

تنفيذ عملية التحويل:

```csharp
converter.Convert(getPageStream, options);
```

**توضيح:**
- تعمل هذه المكالمة على بدء التحويل من XLS إلى JPG باستخدام وظيفة التدفق والخيارات المحددة.
- يضمن `outputFileTemplate` تم ضبطه بشكل صحيح لحفظ الصور.
