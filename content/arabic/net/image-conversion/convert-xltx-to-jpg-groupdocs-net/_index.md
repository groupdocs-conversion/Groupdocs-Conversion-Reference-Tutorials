## دليل التنفيذ

### تحميل وتحويل ملف XLTX إلى JPG
تعمل هذه الميزة على تحويل ملف XLTX إلى سلسلة من صور JPG، وهي مثالية لمشاركة بيانات جدول البيانات بصريًا.

**الخطوة 1: تعيين مسار دليل الإخراج**
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
// قم بتحديد المكان الذي سيتم حفظ الملفات المحولة فيه.
```

**الخطوة 2: تحديد قالب أسماء ملفات الإخراج**
```csharp
string outputFileTemplate = System.IO.Path.Combine(outputFolder, "converted-page-{0}.jpg");
// يساعد هذا القالب في تسمية كل صفحة من المستند برقم فريد.
```

**الخطوة 3: إنشاء تدفق لكل صفحة من نتائج التحويل**
```csharp
Func<SavePageContext, System.IO.Stream> getPageStream = savePageContext => new System.IO.FileStream(System.String.Format(outputFileTemplate, savePageContext.Page), System.IO.FileMode.Create);
// تضمن هذه الوظيفة حفظ كل صفحة كملف منفصل.
```

**الخطوة 4: تحميل ملف XLTX المصدر وتعيين خيارات التحويل**
```csharp
using (Converter converter = new Converter(filePath))
{
    // تحديد خيارات التحويل لتنسيق JPG
    ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Jpg };

    // قم بإجراء التحويل من XLTX إلى JPG
    converter.Convert(getPageStream, options);
}
```
