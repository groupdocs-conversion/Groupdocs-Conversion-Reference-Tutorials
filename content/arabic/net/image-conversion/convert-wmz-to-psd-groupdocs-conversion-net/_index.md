## دليل التنفيذ

### تحويل WMZ إلى PSD

توضح هذه الميزة تحويل ملف WMZ إلى مستند Adobe Photoshop. لنشرحها خطوة بخطوة.

#### الخطوة 1: تحديد مسار الإخراج وقالب الملف

ابدأ بتحديد دليل الإخراج الذي سيتم حفظ ملفاتك المحولة فيه:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY"; // تحديد مسار دليل الإخراج
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");
```

#### الخطوة 2: إنشاء تدفق لكل صفحة

قم بتعريف وظيفة تقوم بإنشاء مجرى ملف جديد لكل صفحة يتم تحويلها:

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext =>
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

#### الخطوة 3: تحميل ملف WMZ وتحويله

الآن، قم بتحميل ملف WMZ المصدر الخاص بك باستخدام `Converter` الفئة وتحديد خيارات التحويل:

```csharp
using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_WMZ")) // حدد دليل المستند الخاص بك هنا
{
    ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd };
    
    // قم بإجراء التحويل من WMZ إلى PSD باستخدام التدفق والخيارات المحددة
    converter.Convert(getPageStream, options);
}
```

**المعلمات الرئيسية موضحة:**
- `outputFileTemplate`: قالب لتسمية ملفات الإخراج.
- `getPageStream`:وظيفة للتعامل مع إنشاء الملف لكل صفحة.
- `ImageConvertOptions`:يحدد أن تنسيق الهدف هو PSD.

#### نصائح استكشاف الأخطاء وإصلاحها

- تأكد من أن مسار دليل الإخراج الخاص بك صحيح وقابل للكتابة.
- تأكد من عدم تلف ملفات WMZ قبل التحويل.
- تحقق من صحة ترخيص GroupDocs إذا واجهت قيودًا على الاستخدام.
