# تحويل ملفات MOBI إلى JPG باستخدام GroupDocs.Conversion لـ .NET

## مقدمة
في عالم الكتب الإلكترونية، يُمكن لتحويل الصيغ الشائعة مثل MOBI إلى صيغ متاحة للجميع مثل JPG أن يُحسّن بشكل كبير من مشاركة الملفات وعرضها على مختلف الأجهزة. يُوضح هذا الدليل الشامل كيفية إجراء هذا التحويل باستخدام **GroupDocs.Conversion** مع تقنيات .NET.

**ما سوف تتعلمه:**
- تحميل ملف MOBI باستخدام GroupDocs.Conversion
- إعداد خيارات تحويل الصور إلى صيغة JPG
- تنفيذ عملية التحويل ومعالجة المخرجات

دعونا نتأكد من اكتمال الإعداد الخاص بك قبل الغوص فيه.

## المتطلبات الأساسية
قبل البدء، تأكد من أنك قمت بتغطية المتطلبات الأساسية التالية:

### المكتبات والإصدارات المطلوبة:
- **GroupDocs.Conversion لـ .NET** الإصدار 25.3.0
- المعرفة الأساسية ببيئات تطوير C# و.NET

### متطلبات إعداد البيئة:
تأكد من أن البيئة الخاصة بك جاهزة باستخدام Visual Studio أو أي IDE آخر يدعم مشاريع .NET.

### المتطلبات المعرفية:
سيكون من المفيد الحصول على فهم أساسي للتعامل مع الملفات في .NET، ولكن ليس ضروريًا، حيث سنرشدك خلال كل خطوة.

## إعداد GroupDocs.Conversion لـ .NET
لتحويل ملفات MOBI إلى JPG باستخدام **GroupDocs.Conversion**ثبّت المكتبة أولاً. إليك الطريقة:

**وحدة تحكم مدير حزمة NuGet:**
```shell
Install-Package GroupDocs.Conversion -Version 25.3.0
```

**.NET CLI:**
```bash
dotnet add package GroupDocs.Conversion --version 25.3.0
```

### خطوات الحصول على الترخيص
ابدأ بفترة تجريبية مجانية أو اطلب ترخيصًا مؤقتًا من موقع الويب الخاص بهم للتجربة **GroupDocs.Conversion**للاستخدام التجاري، قم بشراء الترخيص الكامل.

بمجرد التثبيت، قم بدمج GroupDocs.Conversion في مشروع .NET الخاص بك:
```csharp
using GroupDocs.Conversion;
```
يقوم هذا الإعداد بتجهيزك لمهام التحويل المحددة.

## دليل التنفيذ
سنقوم بتقسيم العملية إلى أقسام منطقية تتوافق مع كل ميزة من ميزات تنفيذ الكود.

### تحميل ملف MOBI
**ملخص:** تتضمن هذه الخطوة تحميل ملف MOBI المصدر الخاص بك باستخدام **GroupDocs.Conversion**.

1. **تحديد المسارات:**
   قم بإعداد دليل المستندات الخاص بك حيث يتم تخزين ملفات MOBI.
   ```csharp
   string documentDirectory = "@YOUR_DOCUMENT_DIRECTORY";
   string mobiFilePath = Path.Combine(documentDirectory, "sample.mobi");
   ```
2. **تحميل الملف:**
   استخدم `Converter` الفئة لتحميل ملف MOBI.
   ```csharp
   using (Converter converter = new Converter(mobiFilePath))
   {
       // تم الآن تحميل ملف MOBI وهو جاهز للتحويل.
   }
   ```
*لماذا هذه الخطوة؟* يضمن تحميل الملف بشكل صحيح تنفيذ جميع العمليات اللاحقة على مجموعة البيانات الصحيحة.

### تعيين خيارات تحويل JPG
**ملخص:** هنا، نقوم بتكوين كيفية تحويل ملفنا إلى صيغة JPG.

1. **إنشاء خيارات التحويل:**
   قم بتحديد خيارات تحويل الصور الخاصة بك باستخدام `ImageConvertOptions`.
   ```csharp
   ImageConvertOptions options = new ImageConvertOptions();
   options.Format = GroupDocs.Conversion.FileTypes.ImageFileType.Jpg;
   ```
2. **فهم المعلمات:**
ال `Format` تحدد المعلمة استهداف تحويل تنسيق ملف JPG.

### تحويل MOBI إلى JPG
**ملخص:** تؤدي هذه الخطوة الأخيرة إلى إجراء التحويل الفعلي من MOBI إلى JPG باستخدام الخيارات المحددة.

1. **إعداد مسارات الإخراج:**
   قم بتحديد المسارات لكل من مجلدات المستندات والإخراج.
   ```csharp
   string outputDirectory = "@YOUR_OUTPUT_DIRECTORY";
   string outputFileTemplate = Path.Combine(outputDirectory, "converted-page-{0}.jpg");
   ```
2. **تعريف وظيفة للحصول على تدفق الصفحة:**
   تضمن هذه الوظيفة تحويل كل صفحة من ملف MOBI وحفظها بشكل صحيح.
   ```csharp
   Func<SavePageContext, Stream> getPageStream = savePageContext => 
       new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
   ```
3. **تنفيذ التحويل:**
   قم بدمج كافة العناصر لتحويل ملف MOBI إلى صيغة JPG.
   ```csharp
   using (Converter converter = new Converter(mobiFilePath))
   {
       converter.Convert(getPageStream, options);
   }
   ```
*لماذا هذا البناء؟* يضمن هذا النهج معالجة كل صفحة من كتاب MOBI الخاص بك بشكل فردي وحفظها كملف JPG منفصل.

### نصائح استكشاف الأخطاء وإصلاحها

- **لم يتم العثور على الملف:** تأكد من وجود "sample.mobi" في دليل المستند المحدد قبل تشغيل الكود.
