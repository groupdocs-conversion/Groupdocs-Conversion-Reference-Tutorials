## دليل التنفيذ

### تحميل وتحويل XLS إلى PNG

#### ملخص:

يركز هذا القسم على تحميل ملف Excel وتحويل كل ورقة إلى صور PNG منفصلة.

#### الخطوة 1: تحديد المسارات

تأكد من ضبط مسار المستند ومجلد الإخراج بشكل صحيح. هذا ضروري لتحديد موقع ملف الإدخال وتخزين الصور المُحوّلة.

```csharp
string documentPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.xls");
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");

if (!Directory.Exists(outputFolder))
{
    Directory.CreateDirectory(outputFolder);
}
```

#### الخطوة 2: تهيئة المحول

إنشاء `Converter` مثال لمعالجة ملف XLS. يُدير هذا الكائن عملية التحويل.

```csharp
using (Converter converter = new Converter(documentPath))
{
    // منطق التحويل يذهب هنا
}
```

#### الخطوة 3: تعيين خيارات التحويل

قم بتحديد تنسيق الإخراج والإعدادات الإضافية باستخدام `ImageConvertOptions`.

```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Png };
```

#### الخطوة 4: تحديد قالب الإخراج

قم بإنشاء قالب تسمية لكل صفحة PNG مُحوّلة لضمان تخزين الملفات بشكل منظم.

```csharp
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.png");
```

#### الخطوة 5: إنشاء تدفقات الصفحات

إعداد دالة لإنشاء تدفقات إخراج لكل ملف PNG. هذا ضروري لكتابة الصور على القرص.

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext =>
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

#### الخطوة 6: تنفيذ التحويل

تنفيذ عملية التحويل عن طريق الاتصال `Convert`، من خلال تمرير وظيفة تدفق الصفحة والخيارات الخاصة بك.

```csharp
converter.Convert(getPageStream, options);
```
