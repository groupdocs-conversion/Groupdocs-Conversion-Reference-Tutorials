## دليل التنفيذ

### الميزة: تحميل ملف JLS
#### ملخص:
الخطوة الأولى في عملية التحويل هي تحميل ملف صورة JPEG بدون فقدان (JLS) باستخدام GroupDocs.Conversion. هذا يُهيئ مستندك للتحويلات اللاحقة.

**الخطوة 1:** تعيين المسار لدليل المستندات الخاص بك
- يُعرِّف `sourceFilePath` مع المسار إلى ملف JLS الخاص بك.

```csharp
string sourceFilePath = @"YOUR_DOCUMENT_DIRECTORY/sample.jls";
```

**الخطوة 2:** تهيئة المحول
- استخدم GroupDocs.Conversion's `Converter` الفئة لتحميل ملف JLS الخاص بك.
  
```csharp
using (var converter = new Converter(sourceFilePath))
{
    // المحول جاهز الآن للعمليات على ملف JLS المحمل
}
```
**توضيح:**
ال `Converter` يتم تهيئة الكائن بمسار ملف JLS المحدد، مما يُهيئه لمهام التحويل. تضمن هذه الخطوة أن يتم التعرف على مستندك بشكل صحيح وأن المكتبة قادرة على الوصول إليه.

### الميزة: تحويل تنسيق JLS إلى تنسيق PPTX
#### ملخص:
الآن، دعنا نقوم بتحويل ملف JLS المحمّل إلى تنسيق عرض تقديمي بتنسيق PowerPoint (PPTX) باستخدام GroupDocs.Conversion.

**الخطوة 1:** تعيين مسارات لدليل الإخراج
- يُعرِّف `outputFolder` و `outputFile` المسارات التي سيتم حفظ المستند المُحوّل فيها.

```csharp
string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "jls-converted-to.pptx");
```

**الخطوة 2:** تأكد من وجود دليل الإخراج

```csharp
Directory.CreateDirectory(outputFolder);
```
**توضيح:**
تمنع هذه الخطوة حدوث الأخطاء عن طريق إنشاء الدليل إذا لم يكن موجودًا بالفعل.

**الخطوة 3:** تحويل وحفظ بتنسيق PPTX

```csharp
using (var converter = new Converter(sourceFilePath))
{
    // إعداد خيارات التحويل لتنسيق PPTX
    var options = new PresentationConvertOptions();
    
    // قم بإجراء التحويل
    converter.Convert(outputFile, options);
}
```
**توضيح:**
ال `PresentationConvertOptions` تقوم الفئة بتكوين عملية التحويل لإخراج ملف PPTX. `Convert` تنفذ الطريقة التحويل وتحفظ النتيجة في الموقع المحدد.
