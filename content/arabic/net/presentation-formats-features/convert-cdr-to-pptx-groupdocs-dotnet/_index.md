## إعداد GroupDocs.Conversion لـ .NET

لتحويل ملفات CDR الخاصة بك، قم بتثبيت **GroupDocs.Conversion** مكتبة.

### التثبيت عبر وحدة تحكم NuGet Package Manager
افتح وحدة التحكم NuGet Package Manager في Visual Studio وقم بتشغيل:
```bash

```

2. **رخصة مؤقتة**:الحصول على ترخيص مؤقت للاختبار الموسع في [ترخيص GroupDocs المؤقت](https://purchase.groupdocs.com/temporary-license/).
3. **شراء**:للحصول على الوصول الكامل، قم بشراء ترخيص من [صفحة شراء GroupDocs](https://purchase.groupdocs.com/buy).

### التهيئة والإعداد الأساسي
```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main()
    {
        // قم بتحديد مسار دليل الإخراج وموقع ملف CDR
        string outputFolder = "YOUR_OUTPUT_DIRECTORY";
        string sampleCdrPath = "YOUR_DOCUMENT_DIRECTORY\sample.cdr"; // استبدل بالمسار الفعلي الخاص بك

        string outputFile = Path.Combine(outputFolder, "cdr-converted-to.pptx");

        // قم بتهيئة كائن المحول باستخدام ملف CDR المصدر
        using (var converter = new Converter(sampleCdrPath))
        {
            var options = new PresentationConvertOptions();
            
            // تحويل وحفظ ملف PPTX الناتج
            converter.Convert(outputFile, options);
        }
    }
}
```
