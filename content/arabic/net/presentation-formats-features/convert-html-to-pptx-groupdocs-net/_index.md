## دليل التنفيذ
### الميزة: تحويل HTML إلى PPTX
#### ملخص
في هذا القسم، سنحوّل ملف HTML إلى عرض تقديمي بصيغة PowerPoint. تُسهّل هذه الميزة مشاركة محتوى الويب بتنسيقات شرائح احترافية.
#### الخطوة 1: تحميل ملف HTML المصدر
ابدأ بتحميل ملف HTML الخاص بك باستخدام `Converter` فصل:
```csharp
string documentPath = Path.Combine(@"YOUR_DOCUMENT_DIRECTORY", "sample.html");
```
يُحدد هذا المقطع من التعليمات البرمجية مسار ملف HTML المصدر. استبدل `"YOUR_DOCUMENT_DIRECTORY"` مع الدليل الفعلي الذي يحتوي على HTML الخاص بك.
#### الخطوة 2: تعيين خيارات التحويل
بعد ذلك، قم بتكوين خيارات التحويل لبرنامج PowerPoint:
```csharp
var options = new PresentationConvertOptions();
```
ال `PresentationConvertOptions` تتيح لك الفئة تحديد إعدادات متنوعة لتحويل المستندات إلى صيغة PPTX. في أبسط صورها، تُوجّه GroupDocs.Conversion لإخراج ملف PowerPoint.
#### الخطوة 3: تحويل وحفظ الناتج
وأخيرًا، قم بتنفيذ التحويل:
```csharp
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputDirectory, "html-converted-to.pptx");

using (var converter = new Converter(documentPath))
{
    converter.Convert(outputFile, options);
}
```
يقوم هذا المقطع بتحويل ملف HTML إلى PPTX ويحفظه في الدليل المحدد. تأكد `"YOUR_OUTPUT_DIRECTORY"` تم ضبطه على المكان الذي تريد تخزين ملفاتك المحولة فيه.
