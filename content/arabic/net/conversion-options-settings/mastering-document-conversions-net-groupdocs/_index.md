## دليل التنفيذ: تحديد تنسيقات المستندات المحتملة
الآن بعد أن قمت بإعداد كل شيء، دعنا ننفذ الميزة لتحديد التحويلات المحتملة.
### نظرة عامة على الميزة
تتيح لك وظيفة "الحصول على التحويلات الممكنة" تحديد جميع التنسيقات التي يمكن تحويل المستند إليها. يُعد هذا ضروريًا للتطبيقات التي تتطلب حلولًا مرنة لإدارة المستندات.
#### الخطوة 1: تحديد مسار المستند
ابدأ بتحديد المسار إلى مستندك:
```csharp
string documentPath = @"YOUR_DOCUMENT_DIRECTORY\sample.docx";
```
#### الخطوة 2: تهيئة فئة المحول
تهيئة مثيل جديد من `Converter` الفئة ذات المسار المحدد:
```csharp
using (Converter converter = new Converter(documentPath))
{
    // سيتم إجراء المعالجة الإضافية هنا.
}
```
#### الخطوة 3: استرداد التحويلات المحتملة
استخدم `GetPossibleConversions` طريقة لجلب إمكانيات التحويل لمستندك:
```csharp
PossibleConversions conversions = converter.GetPossibleConversions();
```
#### الخطوة 4: تكرار وعرض تفاصيل التحويل
كرر كل احتمال تحويل، وحدد ما إذا كان خيارًا أساسيًا أم ثانويًا. ثم اعرض النتائج:
```csharp
foreach (var conversion in conversions.All)
{
    string conversionType = conversion.IsPrimary ? "primary" : "secondary";
    Console.WriteLine($"\t {conversion.Format} as {conversionType} conversion.");
}
```
