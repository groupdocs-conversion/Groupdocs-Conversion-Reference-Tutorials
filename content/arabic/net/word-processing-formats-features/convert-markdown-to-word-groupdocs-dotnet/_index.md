سيرشدك هذا القسم إلى كيفية تحويل ملف Markdown إلى مستند Microsoft Word باستخدام GroupDocs.Conversion لـ .NET. سنشرح كل خطوة بالتفصيل لضمان الوضوح والفهم.

#### الخطوة 1: تهيئة المحول

أولاً، نقوم بتهيئة المحول باستخدام مسار ملف Markdown المدخل لدينا:

```csharp
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY/sample.md"))
{
    // سيتم وضع رمز التحويل هنا
}
```

**توضيح:** هذه الخطوة تخلق `Converter` الكائن المسؤول عن التعامل مع عملية تحويل المستندات.

#### الخطوة 2: تعيين خيارات التحويل

بعد ذلك، قم بتحديد خيارات التحويل لتحديد أننا نريد إخراجًا بتنسيق DOC:

```csharp
var options = new WordProcessingConvertOptions
{
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc
};
```

**توضيح:** هنا، نقوم بتكوين `WordProcessingConvertOptions` لاستهداف نوع ملف DOC.

#### الخطوة 3: تنفيذ التحويل

وأخيرًا، قم بتنفيذ التحويل وحفظ الناتج:

```csharp
string outputFile = System.IO.Path.Combine("YOUR_OUTPUT_DIRECTORY", "md-converted-to.doc");

converter.Convert(outputFile, options);
```

**توضيح:** يقوم هذا السطر بتحويل مستند Markdown إلى ملف Word، وحفظه في الدليل المحدد.
