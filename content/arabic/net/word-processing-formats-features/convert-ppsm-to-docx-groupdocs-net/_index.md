## دليل التنفيذ

دعونا نقسم التنفيذ إلى خطوات منطقية.

### نظرة عامة على الميزة: تحويل PPSM إلى DOCX
تتيح لك هذه الميزة تحويل ملفات عرض الشرائح في PowerPoint (.ppsm) إلى مستندات Word (.docx) بسلاسة، مما يجعل مشاركة المستندات أكثر تنوعًا وسهولة في الوصول إليها.

#### الخطوة 1: تحديد دليل الإخراج
أنشئ دليل إخراج لتخزين ملفاتك المُحوّلة. استبدل `YOUR_OUTPUT_DIRECTORY` مع المسار الذي تريده:
```csharp
string outputFolder = Path.Combine(@"YOUR_OUTPUT_DIRECTORY");
```
**لماذا هذا مهم؟**:يساعد تحديد موقع إخراج واضح في إدارة المستندات المحولة واسترجاعها بكفاءة.

#### الخطوة 2: إعداد المحول
إنشاء مثيل لـ `Converter` الفئة، تمرير مسار ملف PPSM المصدر:
```csharp
using (var converter = new Converter("source.ppsm"))
{
    // منطق التحويل يذهب هنا
}
```
**توضيح**:يتولى هذا الكائن إدارة عملية التحويل، ومعالجة جميع العمليات الضرورية داخليًا.

#### الخطوة 3: تكوين خيارات DOCX
قم بتحديد تنسيق الإخراج المطلوب باستخدام `WordProcessingConvertOptions`:
```csharp
var options = new WordProcessingConvertOptions();
```
**تكوين المفتاح**:يؤدي هذا إلى إعداد معلمات محددة لتنسيق مستند Word وسلوك التحويل.

#### الخطوة 4: تنفيذ التحويل
قم بتنفيذ التحويل عن طريق استدعاء `Convert` الطريقة، تمرير خياراتك ومسار الإخراج:
```csharp
converter.Convert(Path.Combine(outputFolder, "output.docx"), options);
```
**حصيلة**:تم الآن تحويل ملف PPSM الخاص بك إلى تنسيق DOCX جاهزًا للاستخدام أو التوزيع.
