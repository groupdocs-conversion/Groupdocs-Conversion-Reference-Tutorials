## دليل التنفيذ

دعونا نقسم عملية تحويل MHT إلى DOCX إلى خطوات قابلة للإدارة:

### تحميل ملف المصدر وإعداده

أولاً، قم بتحميل ملف MHT المصدر الخاص بك عن طريق إعداد مسار إلى دليل المستند الخاص بك:

```csharp
string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY\sample.mht");
```

### تحويل MHT إلى DOCX

بعد ذلك، قم بإجراء التحويل باستخدام الميزات المضمنة في GroupDocs.Conversion.

**تهيئة المحول**

```csharp
using (var converter = new GroupDocs.Conversion.Converter(inputFilePath))
{
    // تحديد خيارات التحويل إلى تنسيق DOCX
    var convertOptions = converter.GetPossibleConversions()["docx"].ConvertOptions;

    // قم بإجراء التحويل
    string outputFile = Path.Combine("YOUR_OUTPUT_DIRECTORY", "output.docx");
    converter.Convert(outputFile, convertOptions);
}
```

**توضيح**: 
- `GetPossibleConversions()` يسترجع جميع صيغ الملفات المستهدفة. هنا، نحدد DOCX كصيغة الملف المطلوبة.
- ال `converter.Convert()` تعمل الوظيفة على معالجة تحويل الملف الفعلي.
