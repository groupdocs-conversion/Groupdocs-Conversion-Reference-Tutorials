# CAD 和技術圖紙格式教程（適用於 GroupDocs.Conversion .NET）

在此中心，您會找到使用 GroupDocs.Conversion for .NET **將 CAD 轉換為 PDF** 的逐步指南，涵蓋自訂頁面尺寸、3‑D 模型展平、圖層保留以及大型圖紙處理。無論您是構建渲染工程圖的 Web 服務，還是批量處理數千張圖紙的桌面工具，這些教程都會示範如何在保持低記憶體使用量的同時，獲得可靠、可投入生產的結果。

GroupDocs.Conversion for .NET 是一個 .NET 函式庫，可將超過 150 種格式（包括 CAD 圖紙）轉換為 PDF、影像和 Office 檔案。它同時支援 .NET Framework 與 .NET Core，可在 Windows、Linux 與 macOS 上執行，且不需要在伺服器上安裝任何外部 CAD 軟體。

**Quantified claim:** GroupDocs.Conversion 能處理高達 500 MB 的 CAD 檔案，並在一般 4 核心 VM 上於 30 秒內轉換數百頁的圖紙，這歸功於其串流架構，可避免將整個來源檔案載入記憶體。
