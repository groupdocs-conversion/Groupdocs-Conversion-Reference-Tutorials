* **廣泛的格式支援** – 支援 Word、Excel、CSV、ZIP、電子郵件等多種格式，成為您所有轉換需求的一站式解決方案。  
* **高保真度** – 保留版面、字型與複雜元素（表格、圖表、註釋），使輸出 PDF 與原始檔案完全相同。  
* **效能導向** – 精簡的 API 讓您以最小記憶體佔用轉換大量或單一檔案，適合雲端服務。

## 快速入門

1. **新增 Maven 相依性**（或相等的 Gradle 設定）以使用 GroupDocs.Conversion。  
2. **實例化 `Converter`**，並使用您的授權金鑰。  
3. **選擇來源格式**（例如 `.docx`、`.xlsx`、`.csv`）。  
4. **設定 `PdfConvertOptions`** – 設定頁面大小、字型替換或密碼處理等。  
5. **呼叫 `convert`**，並將輸出串流寫入檔案或回應。  

> **專業提示：** 當您需要 **convert excel to pdf java** 並使用自訂字型時，請設定 `PdfConvertOptions.setFontSubstitution(true)` 並提供 `FontMapper` 以確保視覺一致性。
