## 使用 GroupDocs.Conversion Java 將試算表轉換為 PDF 的方法
在本節中，我們說明將任何支援的試算表格式轉換為高品質 PDF 文件的整體方法。無論您是從 XLSX、ODS 或 CSV 檔案開始，流程皆相同——只需載入來源檔案、設定轉換選項（例如隱藏註解、保留頁面佈局），然後呼叫轉換 API。

### 為什麼要將試算表轉換為 PDF？
- **Universal access** – PDF 可在任何裝置上開啟，無需 Excel。
- **Preserve layout** – 儲存格寬度、顏色與邊框保持不變。
- **Security** – 您可以對產生的 PDF 設定密碼保護或限制編輯。
- **Automation** – 適用於在 Java 應用程式中批次處理報告、發票或資料匯出。

### GroupDocs.Conversion Java 支援的主要功能
- **Preserve cell formatting** – 顏色、字型、邊框與條件格式皆會保留。
- **Preserve Excel formulas** – 公式在轉換過程中會被計算，PDF 會顯示計算後的數值。
- **Convert ODS to XLSX** – 您可以先將 OpenDocument 試算表轉換為 Excel，然後再產生 PDF。
- **Convert CSV to PDF Java** – 可自訂分隔符號與欄寬。
- **GroupDocs Conversion Java** API – 完全相容 Maven，並提供詳細文件與範例。
