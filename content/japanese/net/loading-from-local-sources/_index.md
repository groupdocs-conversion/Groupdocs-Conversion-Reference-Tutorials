# GroupDocs.Conversion .NET のローカル ソースからのドキュメントの読み込みチュートリアル

ローカルドキュメント読み込みチュートリアルでは、.NETアプリケーションでローカルシステムに保存されているドキュメントを操作するための包括的なガイダンスを提供しています。これらのステップバイステップガイドに従って、ファイルシステムパスやメモリストリームからドキュメントを読み込む方法を学習してください。これらのチュートリアルでは、相対パスと絶対パス、ストリーム管理の考慮事項、メモリ最適化手法など、さまざまなシナリオを網羅しています。各ガイドには、ローカルドキュメント読み込みシナリオに対応した完全なC#コードサンプルが含まれており、ファイルシステムやメモリからコンテンツに効率的にアクセスできるアプリケーションの構築に役立ちます。

## 利用可能なチュートリアル

### [GroupDocs.Conversion for .NET を使用して EMZ ファイルを読み込む方法: 包括的なガイド](./load-emz-files-groupdocs-conversion-dotnet/)

### [GroupDocs.Conversion for .NET を使用して XLTX ファイルを読み込む方法: 包括的なガイド](./load-xltx-file-groupdocs-conversion-net/)
GroupDocs.Conversion for .NET を使用して、Excel テンプレートファイル（XLTX）を効率的に読み込み、変換する方法を学びます。このガイドでは、詳細な手順、コード例、トラブルシューティングのヒントを紹介します。

### [GroupDocs.Conversion for .NET を使用して OTG ファイルを読み込み、変換する: 開発者ガイド](./load-otg-files-groupdocs-conversion-net/)
