## O que é Excel to PDF Java?

Converter uma pasta de trabalho Excel (`.xlsx` ou `.xls`) em um documento PDF em um ambiente Java é uma necessidade comum para relatórios, arquivamento e compartilhamento de dados sem expor as fórmulas originais da planilha. GroupDocs.Conversion abstrai os detalhes de renderização de baixo nível, permitindo que você se concentre na lógica de negócios.

## Por que usar GroupDocs.Conversion para Excel to PDF Java?

- **Alta fidelidade** – preserva a formatação das células, células mescladas, gráficos e imagens.  
- **Multiplataforma** – funciona em qualquer SO que suporte Java 8+.  
- **Extensível** – combine com outros recursos como **html to pdf java**, **convert word pdf java**, ou **add pdf watermark java** em um único fluxo de trabalho.  
- **Segura** – manipule arquivos **password protected pdf java** com segurança, com criptografia opcional na saída.
