# Tutoriais de Formatos de CAD e Desenho Técnico para GroupDocs.Conversion .NET

Neste hub você encontrará guias passo a passo para **converter CAD para PDF** usando GroupDocs.Conversion para .NET, abordando dimensões de página personalizadas, achatamento de modelos 3‑D, preservação de camadas e manipulação de desenhos grandes. Seja construindo um serviço web que renderiza esquemas de engenharia ou uma ferramenta desktop que processa milhares de desenhos, estes tutoriais mostram como obter resultados confiáveis e prontos para produção, mantendo o uso de memória baixo.

GroupDocs.Conversion para .NET é uma biblioteca .NET que converte mais de 150 formatos — incluindo desenhos CAD — em PDF, imagens e arquivos Office. Ela suporta tanto .NET Framework quanto .NET Core, funciona em Windows, Linux e macOS, e não requer a instalação de software CAD externo no servidor.

**Afirmação quantificada:** GroupDocs.Conversion pode processar arquivos CAD de até 500 MB e converter desenhos com centenas de páginas em menos de 30 segundos em uma VM típica de 4 núcleos, graças à sua arquitetura de streaming que evita carregar todo o arquivo fonte na memória.
