## Οδηγός Εφαρμογής
Ας αναλύσουμε τη διαδικασία μετατροπής σε διαχειρίσιμα βήματα:

### Ρύθμιση μετατροπής εγγράφων
**Επισκόπηση:** Αυτή η ενότητα προετοιμάζει το περιβάλλον σας για τη μετατροπή ενός εγγράφου σε μορφή εικόνας.

#### Βήμα 1: Ορισμός διαδρομών αρχείων
Καθορίστε πού βρίσκεται το PDF προέλευσης και πού θέλετε να αποθηκεύσετε το αρχείο PSD που έχει μετατραπεί.
```java
String sourceFilePath = "YOUR_DOCUMENT_DIRECTORY/SAMPLE_PDF"; // Αντικαταστήστε με τη διαδρομή PDF σας
String outputFolder = "YOUR_OUTPUT_DIRECTORY"; // Φάκελος προορισμού για το αρχείο PSD
```

### Διαμόρφωση επιλογών μετατροπής εικόνας
**Επισκόπηση:** Ορίστε συγκεκριμένες επιλογές μετατροπής για να στοχεύσετε μια μορφή PSD.

#### Βήμα 2: Διαμόρφωση επιλογών μετατροπής
Διαμορφώστε το `ImageConvertOptions` κλάση για να ορίσετε τη μορφή εξόδου και τον αριθμό των σελίδων.
```java
import com.groupdocs.conversion.options.convert.ImageConvertOptions;
import com.groupdocs.conversion.filetypes.ImageFileType;

ImageConvertOptions options = new ImageConvertOptions();
options.setFormat(ImageFileType.Psd); // Ορισμός μορφής σε PSD
options.setPagesCount(1); // Μετατροπή μόνο της πρώτης σελίδας
```

### Εκτέλεση Διαδικασίας Μετατροπής
**Επισκόπηση:** Εκτελέστε τη μετατροπή χρησιμοποιώντας τις διαμορφωμένες ρυθμίσεις και αποθηκεύστε το αποτέλεσμα.

#### Βήμα 3: Εκτέλεση μετατροπής
Δημιουργήστε ένα `FileOutputStream` για να γράψετε το αρχείο που έχει μετατραπεί, να αρχικοποιήσετε τον μετατροπέα και να εκτελέσετε τη διαδικασία.
```java
import com.groupdocs.conversion.Converter;
import java.io.FileOutputStream;

String outputFileTemplate = String.format("%s/converted-page-%d.psd", outputFolder, 1);

try (FileOutputStream getPageStream = new FileOutputStream(outputFileTemplate)) {
    Converter converter = new Converter(sourceFilePath); // Αρχικοποίηση με το PDF προέλευσης
    converter.convert(() -> getPageStream, options); // Μετατροπή και αποθήκευση σε PSD
} catch (IOException e) {
    System.out.println(e.getMessage());
}
```
