## Οδηγός Εφαρμογής
Τώρα, ας δούμε τα βήματα για την απόκρυψη σχολίων κατά τη μετατροπή:

### Απόκρυψη σχολίων κατά τη μετατροπή
Αυτή η λειτουργία είναι ζωτικής σημασίας για τη διατήρηση του απορρήτου και του επαγγελματισμού σε κοινόχρηστα έγγραφα. Δείτε πώς μπορείτε να την εφαρμόσετε:

#### Βήμα 1: Φόρτωση ρυθμίσεων επιλογών
Αρχικά, ρυθμίστε τις επιλογές φόρτωσης για να καθορίσετε ότι τα σχόλια θα πρέπει να είναι κρυμμένα.

```java
import com.groupdocs.conversion.Converter;
import com.groupdocs.conversion.options.convert.PdfConvertOptions;
import com.groupdocs.conversion.options.load.WordProcessingLoadOptions;

// Ρύθμιση παραμέτρων επιλογών φόρτωσης
WordProcessingLoadOptions loadOptions = new WordProcessingLoadOptions();
loadOptions.setHideComments(true); // Απόκρυψη σχολίων στο PDF εξόδου
```

#### Βήμα 2: Αρχικοποίηση μετατροπέα
Στη συνέχεια, αρχικοποιήστε τον μετατροπέα με τη διαδρομή του εγγράφου προέλευσης και τις διαμορφωμένες επιλογές φόρτωσης.

```java
String sourceDocument = "YOUR_DOCUMENT_DIRECTORY/SAMPLE_DOCX_WITH_TRACKED_CHANGES";
Converter converter = new Converter(sourceDocument, () -> loadOptions);
```

#### Βήμα 3: Μετατροπή σε PDF
Τέλος, ρυθμίστε τις επιλογές μετατροπής και εκτελέστε τη μετατροπή.

```java
PdfConvertOptions convertOptions = new PdfConvertOptions(); // Προεπιλεγμένες ρυθμίσεις PDF
String outputPdf = "YOUR_OUTPUT_DIRECTORY/ConvertWordProcessingByHiddingComments.pdf";

// Εκτέλεση μετατροπής
converter.convert(outputPdf, convertOptions);
```
