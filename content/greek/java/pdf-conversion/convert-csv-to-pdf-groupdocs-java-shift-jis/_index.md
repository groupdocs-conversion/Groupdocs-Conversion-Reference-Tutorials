# csv to pdf java – Μετατροπή CSV σε PDF σε Java χρησιμοποιώντας GroupDocs με κωδικοποίηση Shift_JIS

Η μετατροπή ενός αρχείου CSV σε PDF διατηρώντας το σωστό σύνολο χαρακτήρων είναι μια κοινή απαίτηση για πολλές εφαρμογές Java. Σε αυτόν τον οδηγό θα μάθετε **πώς να εκτελέσετε μια μετατροπή csv σε pdf java** με το GroupDocs.Conversion, εξασφαλίζοντας ότι τα δεδομένα κωδικοποιημένα σε Shift_JIS (συχνά χρησιμοποιούμενα για ιαπωνικό κείμενο) αποδίδονται σωστά.

## Γρήγορες Απαντήσεις
- **Τι βιβλιοθήκη χρειάζεται;** GroupDocs.Conversion for Java (v25.2+).

- **Μπορώ να δημιουργήσω PDF από CSV με άλλες κωδικοποιήσεις;** Ναι – απλώς αλλάξτε το charset στο `CsvLoadOptions`.  
- **Χρειάζομαι άδεια;** Μια δωρεάν δοκιμή λειτουργεί για ανάπτυξη· απαιτείται μόνιμη άδεια για παραγωγή.  
- **Είναι ο κώδικας thread‑safe;** Κάθε αντικείμενο `Converter` είναι ανεξάρτητο, έτσι μπορείτε να εκτελείτε μετατροπές σε παράλληλα νήματα.

Η διαδικασία μετατρέπει δεδομένα CSV απλού κειμένου σε ένα μορφοποιημένο έγγραφο PDF. Αυτό είναι χρήσιμο όταν χρειάζεστε μια μη επεξεργάσιμη, εκτυπώσιμη αναπαράσταση των πινάκων δεδομένων, ειδικά όταν η πηγή περιέχει ειδικούς χαρακτήρες που πρέπει να διατηρηθούν.

## Πώς να δημιουργήσετε PDF από CSV χρησιμοποιώντας κωδικοποίηση Shift_JIS

### Διαμόρφωση Επιλογών Φόρτωσης CSV με Συγκεκριμένη Κωδικοποίηση

Καθορίστε την κωδικοποίηση του εισερχόμενου αρχείου CSV χρησιμοποιώντας Shift_JIS:

```java
CsvLoadOptions loadOptions = new CsvLoadOptions();
loadOptions.setEncoding(java.nio.charset.Charset.forName("shift_jis")); // Set encoding to Shift_JIS
```

**Η κλάση `CsvLoadOptions` σας επιτρέπει να ορίσετε παραμέτρους όπως η κωδικοποίηση χαρακτήρων, διασφαλίζοντας ότι τα δεδομένα CSV ερμηνεύονται σωστά πριν από τη μετατροπή.**

### Αρχικοποίηση του Αντικειμένου Converter

Αρχικοποιήστε το αντικείμενο `Converter` με τη διαδρομή του πηγαίου αρχείου CSV και τις επιλογές φόρτωσης:

```java
String sourceCsvPath = "YOUR_DOCUMENT_DIRECTORY/your-input-file.csv";
Converter converter = new Converter(sourceCsvPath, () -> loadOptions);
```

**Η κλάση `Converter` διαχειρίζεται τη διαδικασία μετατροπής. Με τη μεταβίβαση της διαδρομής του αρχείου CSV και των επιλογών φόρτωσης, προετοιμάζουμε τα δεδομένα για μετατροπή.**

### Διαμόρφωση Επιλογών Μετατροπής

Ρυθμίστε τις επιλογές μετατροπής PDF:

```java
PdfConvertOptions pdfConvertOptions = new PdfConvertOptions();
```

**Το `PdfConvertOptions` μπορεί να προσαρμοστεί ώστε να διαμορφώσει το παραγόμενο PDF, όπως ορίζοντας το μέγεθος σελίδας ή τα περιθώρια.**

### Μετατροπή Αρχείου CSV σε PDF

Εκτελέστε τη μετατροπή χρησιμοποιώντας τις καθορισμένες επιλογές:

```java
String targetPdfPath = "YOUR_OUTPUT_DIRECTORY/output-file.pdf";
converter.convert(targetPdfPath, pdfConvertOptions);
```

**Η μέθοδος `convert` λαμβάνει τη διαδρομή του αρχείου εξόδου και τις επιλογές μετατροπής, επεξεργαζόμενη τα δεδομένα CSV σε PDF ενώ σέβεται την κωδικοποίηση Shift_JIS.**

## Κοινά Προβλήματα και Λύσεις

- **Προβλήματα Διαδρομής Αρχείου:** Βεβαιωθείτε ότι οι φάκελοι πηγής και προορισμού υπάρχουν και ότι η εφαρμογή έχει δικαιώματα ανάγνωσης/εγγραφής.  
- **Ασυμφωνία Έκδοσης:** Χρησιμοποιήστε το GroupDocs.Conversion 25.2 ή νεότερο· παλαιότερες εκδόσεις μπορεί να μην υποστηρίζουν τη ρύθμιση κωδικοποίησης του `CsvLoadOptions`.
