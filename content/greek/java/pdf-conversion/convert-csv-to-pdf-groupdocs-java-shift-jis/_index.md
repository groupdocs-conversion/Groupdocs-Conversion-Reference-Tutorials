## Οδηγός Εφαρμογής

Ας δούμε βήμα προς βήμα τη διαδικασία υλοποίησης.

### Ρύθμιση παραμέτρων επιλογών φόρτωσης CSV με συγκεκριμένη κωδικοποίηση

Καθορίστε την κωδικοποίηση του αρχείου CSV εισόδου μας χρησιμοποιώντας το Shift_JIS:

```java
CsvLoadOptions loadOptions = new CsvLoadOptions();
loadOptions.setEncoding(java.nio.charset.Charset.forName("shift_jis")); // Ορισμός κωδικοποίησης σε Shift_JIS
```

**Γιατί να χρησιμοποιήσετε επιλογές φόρτωσης;**
Ο `CsvLoadOptions` Η κλάση σάς επιτρέπει να ορίσετε διάφορες παραμέτρους για τη φόρτωση αρχείων CSV, συμπεριλαμβανομένης της κωδικοποίησης χαρακτήρων. Αυτό διασφαλίζει ότι τα δεδομένα σας ερμηνεύονται και μετατρέπονται σωστά.

### Αρχικοποίηση του αντικειμένου μετατροπέα

Αρχικοποίηση του `Converter` αντικείμενο με τη διαδρομή του αρχείου CSV πηγής και τις επιλογές φόρτωσης:

```java
String sourceCsvPath = "YOUR_DOCUMENT_DIRECTORY/your-input-file.csv";
Converter converter = new Converter(sourceCsvPath, () -> loadOptions);
```

**Τι κάνει αυτό το βήμα:**
Ο `Converter` Η κλάση διαχειρίζεται τη διαδικασία μετατροπής. Μεταβιβάζοντας τη διαδρομή του αρχείου CSV και τις επιλογές φόρτωσης, προετοιμάζουμε τα δεδομένα μας για μετατροπή.

### Ρύθμιση παραμέτρων επιλογών μετατροπής

Ρύθμιση επιλογών μετατροπής PDF:

```java
PdfConvertOptions pdfConvertOptions = new PdfConvertOptions();
```

**Βασικές επιλογές διαμόρφωσης:**
Ο `PdfConvertOptions` μπορεί να προσαρμοστεί για να προσαρμόσει το PDF εξόδου, όπως ορίζοντας το μέγεθος σελίδας ή τα περιθώρια.

### Μετατροπή αρχείου CSV σε PDF

Εκτελέστε τη μετατροπή χρησιμοποιώντας τις καθορισμένες επιλογές:

```java
String targetPdfPath = "YOUR_OUTPUT_DIRECTORY/output-file.pdf";
converter.convert(targetPdfPath, pdfConvertOptions);
```

**Πώς λειτουργεί:**
Ο `convert` Η μέθοδος λαμβάνει τη διαδρομή του αρχείου εξόδου και τις επιλογές μετατροπής. Επεξεργάζεται τα δεδομένα CSV σε μορφή PDF τηρώντας παράλληλα τις ρυθμίσεις κωδικοποίησης.
