## Οδηγός Εφαρμογής

Θα αναλύσουμε την υλοποίηση σε χαρακτηριστικά, καθοδηγώντας σας σε κάθε βήμα.

### Δυνατότητα 1: Μετατροπή TXT σε PDF με επιλογές για προχωρημένους

Αυτή η λειτουργία επισημαίνει τον τρόπο μετατροπής ενός αρχείου κειμένου σε μορφή PDF, ελέγχοντας παράλληλα συμπεριφορές μορφοποίησης, όπως τα κενά στην αρχή και την ανίχνευση αρίθμησης.

#### Ρύθμιση επιλογών φόρτωσης για μετατροπή TXT

Διαμορφώστε τις επιλογές φόρτωσης χρησιμοποιώντας `TxtLoadOptions`:

```java
import com.groupdocs.conversion.options.load.TxtLoadOptions;
import com.groupdocs.conversion.options.load.TxtLeadingSpacesOptions;

// Δημιουργήστε την παρουσία TxtLoadOptions.
TxtLoadOptions loadOptions = new TxtLoadOptions();
loadOptions.setLeadingSpacesOptions(TxtLeadingSpacesOptions.ConvertToIndent); // Μετατροπή κενών στην αρχή σε εσοχές.
loadOptions.setDetectNumberingWithWhitespaces(true); // Εντοπισμός αρίθμησης με κενά διαστήματα.
```

#### Αρχικοποίηση του μετατροπέα

Μεταβιβάστε το έγγραφο TXT και φορτώστε τις επιλογές στο `Converter`:

```java
import com.groupdocs.conversion.Converter;

// Δημιουργήστε μια παρουσία μετατροπέα με συγκεκριμένες επιλογές.
Converter converter = new Converter(inputFile, () -> loadOptions);
```

#### Ρύθμιση παραμέτρων επιλογών μετατροπής PDF

Ρύθμιση επιλογών για μετατροπή σε PDF χρησιμοποιώντας `PdfConvertOptions`:

```java
import com.groupdocs.conversion.options.convert.PdfConvertOptions;

// Προετοιμάστε επιλογές μετατροπής PDF.
PdfConvertOptions convertOptions = new PdfConvertOptions();
```

#### Εκτελέστε τη μετατροπή

Εκτελέστε τη μετατροπή και αποθηκεύστε το PDF εξόδου:

```java
// Μετατρέψτε TXT σε PDF με την εφαρμογή προηγμένων ρυθμίσεων.
converter.convert(outputFile, convertOptions);
System.out.println("Conversion complete. Check YOUR_OUTPUT_DIRECTORY for the result.");
```

### Λειτουργία 2: Ρύθμιση επιλογών φόρτωσης για προηγμένη επεξεργασία κειμένου

Αυτή η λειτουργία επιδεικνύει τη ρύθμιση επιλογών φόρτωσης ειδικά για αρχεία κειμένου που χρειάζονται ακριβή έλεγχο μορφοποίησης κατά τη μετατροπή.

#### Ρύθμιση παραμέτρων TxtLoadOptions

Δείτε πώς μπορείτε να ορίσετε επιλογές φόρτωσης για να χειριστείτε την προηγμένη επεξεργασία κειμένου:

```java
TxtLoadOptions txtLoadOptions = new TxtLoadOptions();
txtLoadOptions.setLeadingSpacesOptions(TxtLeadingSpacesOptions.ConvertToIndent);
txtLoadOptions.setDetectNumberingWithWhitespaces(true);

// Αυτές οι επιλογές διασφαλίζουν ότι η μετατροπή PDF σέβεται τις αρχικές λεπτομέρειες μορφοποίησης.
```
