## Οδηγός Εφαρμογής

### Μετατροπή email σε PDF με επιλογές για προχωρημένους

Αυτή η λειτουργία σάς επιτρέπει να μετατρέπετε έγγραφα email σε μορφή PDF, προσαρμόζοντας παράλληλα τα ορατά πεδία. Ας αναλύσουμε τη διαδικασία βήμα προς βήμα:

#### Βήμα 1: Ρύθμιση παραμέτρων επιλογών φόρτωσης email
Για να ξεκινήσετε, δημιουργήστε μια παρουσία του `EmailLoadOptions` και να το διαμορφώσετε σύμφωνα με τις απαιτήσεις σας. Αυτό περιλαμβάνει την απόφαση για το ποιες κεφαλίδες και διευθύνσεις email θα πρέπει να εμφανίζονται.

```java
import com.groupdocs.conversion.options.load.EmailLoadOptions;

EmailLoadOptions loadOptions = new EmailLoadOptions();
loadOptions.setDisplayHeader(false);
loadOptions.setDisplayFromEmailAddress(false);
loadOptions.setDisplayToEmailAddress(false);
loadOptions.setDisplayEmailAddress(false);
loadOptions.setDisplayCcEmailAddress(false);
loadOptions.setDisplayBccEmailAddress(false);
loadOptions.setConvertOwned(false); // Αποτροπή μετατροπής πεδίων που ανήκουν στο έγγραφο
```

#### Βήμα 2: Αρχικοποίηση μετατροπέα
Χρησιμοποιώντας το διαμορφωμένο `EmailLoadOptions`, αρχικοποιήστε μια νέα παρουσία του `Converter`.

```java
import com.groupdocs.conversion.Converter;
import com.groupdocs.conversion.options.convert.PdfConvertOptions;

Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_MSG", () -> loadOptions);
```

#### Βήμα 3: Ορισμός επιλογών μετατροπής PDF
Στη συνέχεια, ορίστε τις επιλογές μετατροπής χρησιμοποιώντας `PdfConvertOptions`Αυτό επιτρέπει περαιτέρω προσαρμογή του PDF εξόδου.

```java
PdfConvertOptions options = new PdfConvertOptions();
```

#### Βήμα 4: Μετατροπή email σε PDF
Τέλος, εκτελέστε τη μετατροπή καλώντας το `convert` μέθοδο με τις καθορισμένες διαδρομές εισόδου και εξόδου.

```java
converter.convert("YOUR_OUTPUT_DIRECTORY/ConvertEmailWithAlteringFieldsVisibility.pdf", options);
```

### Φόρτωση επιλογών ανά τύπο εγγράφου
Η κατανόηση του τρόπου φόρτωσης διαφορετικών τύπων εγγράφων χρησιμοποιώντας εξειδικευμένες επιλογές φόρτωσης είναι ζωτικής σημασίας για αποτελεσματικές μετατροπές. Εδώ, εστιάζουμε στα email:

#### Βήμα 1: Ρύθμιση παραμέτρων επιλογών φόρτωσης email
Όπως αναφέρθηκε προηγουμένως, διαμορφώστε το `EmailLoadOptions` σύμφωνα με τις απαιτήσεις ορατότητας των πεδίων email σας.

```java
import com.groupdocs.conversion.options.load.EmailLoadOptions;

EmailLoadOptions emailLoadOptions = new EmailLoadOptions();
emailLoadOptions.setDisplayHeader(false);
emailLoadOptions.setDisplayFromEmailAddress(false);
emailLoadOptions.setDisplayToEmailAddress(false);
emailLoadOptions.setDisplayEmailAddress(false);
emailLoadOptions.setDisplayCcEmailAddress(false);
emailLoadOptions.setDisplayBccEmailAddress(false);
emailLoadOptions.setConvertOwned(false); // Μην μετατρέπετε τα ιδιόκτητα πεδία
```

#### Βήμα 2: Αρχικοποίηση του μετατροπέα με επιλογές φόρτωσης email
Χρησιμοποιήστε το διαμορφωμένο `EmailLoadOptions` κατά την αρχικοποίηση του `Converter`.

```java
Converter emailConverter = new Converter("EMAIL_FILE_PATH", () -> emailLoadOptions);
```
