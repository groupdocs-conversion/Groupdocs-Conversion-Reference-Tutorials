## Οδηγός Υλοποίησης

### Μετατροπή CSV σε PDF με Προηγμένες Επιλογές

Η χρήση προηγμένων επιλογών διασφαλίζει ότι τα δεδομένα σας διατηρούν την ακεραιότητά τους κατά τη μετατροπή, διατηρώντας την αρχική μορφοποίηση για ημερομηνίες‑ώρας και αριθμητικές τιμές.

#### Υλοποίηση βήμα‑βήμα

**1. Διαμόρφωση Επιλογών Φόρτωσης**  
Ξεκινήστε ρυθμίζοντας το `CsvLoadOptions` για να διαχειριστεί ειδικούς τύπους δεδομένων:
```java
// Initialize load options for the CSV
CsvLoadOptions loadOptions = new CsvLoadOptions();
loadOptions.setConvertDateTimeData(true); // Enable conversion of date-time data
loadOptions.setConvertNumericData(true);  // Enable conversion of numeric data
```

**2. Δημιουργία Αντικειμένου Converter**  
Αρχικοποιήστε το αντικείμενο `Converter` με τη διαδρομή του εισαγόμενου αρχείου CSV και τις επιλογές φόρτωσης:
```java
String inputCsvPath = "YOUR_DOCUMENT_DIRECTORY/sample.csv";
Converter converter = new Converter(inputCsvPath, () -> loadOptions);
```

**3. Ρύθμιση Επιλογών Μετατροπής PDF**  
Προετοιμαστείτε για τη μετατροπή του εγγράφου διαμορφώνοντας το `PdfConvertOptions`:
```java
// Initialize PDF conversion options
PdfConvertOptions pdfConvertOptions = new PdfConvertOptions();
```

**4. Εκτέλεση της Μετατροπής**  
Τέλος, εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο PDF στην επιθυμητή τοποθεσία:
```java
String outputPdfPath = "YOUR_OUTPUT_DIRECTORY/converted_file.pdf";
converter.convert(outputPdfPath, pdfConvertOptions);
```
