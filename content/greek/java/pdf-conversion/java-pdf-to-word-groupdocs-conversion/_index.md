## Οδηγός Εφαρμογής

Τώρα ας αναλύσουμε την υλοποίηση σε διαχειρίσιμα τμήματα, εστιάζοντας σε κάθε χαρακτηριστικό.

### Φόρτωση PDF με Προηγμένες Επιλογές

**Επισκόπηση:**
Αυτή η λειτουργία σάς επιτρέπει να φορτώσετε ένα αρχείο PDF και να το ρυθμίσετε ώστε να αποκρύπτει τις σχολιασμούς πριν από τη μετατροπή, εξασφαλίζοντας ένα πιο καθαρό αποτέλεσμα εγγράφου.

#### Βήμα 1: Ρύθμιση παραμέτρων PdfLoadOptions
Δημιουργήστε μια παρουσία του `PdfLoadOptions` και ορίστε την επιλογή απόκρυψης σχολίων:
```java
// Δημιουργία και ρύθμιση παραμέτρων επιλογών φόρτωσης για το έγγραφο PDF
double createPdfLoadOptionsWithHiddenAnnotations() {
    // Δημιουργία αρχικών επιλογών φόρτωσης Pdf
    PdfLoadOptions loadOptions = new PdfLoadOptions();
    
    // Ορισμός επιλογής για απόκρυψη σχολιασμών στο PDF
    loadOptions.setHidePdfAnnotations(true);
    
    return 0; // Τιμή επιστροφής κράτησης θέσης
}
```
**Εξήγηση:**
- **`setHidePdfAnnotations(true)`:** Αυτή η μέθοδος αποκρύπτει τυχόν σχόλια που υπάρχουν στο PDF σας, διασφαλίζοντας ότι δεν εμφανίζονται στο έγγραφο που έχει μετατραπεί.

### Μετατροπή PDF σε μορφή επεξεργασίας κειμένου

**Επισκόπηση:**
Αφού φορτώσετε και ρυθμίσετε το αρχείο PDF, μπορείτε να το μετατρέψετε σε μορφή επεξεργασίας κειμένου χρησιμοποιώντας συγκεκριμένες επιλογές για βέλτιστα αποτελέσματα.

#### Βήμα 2: Ορισμός διαδρομών εισόδου και εξόδου
Ορίστε placeholders για διαδρομές εισόδου και εξόδου:
```java
// Ορίστε τη διαδρομή για έγγραφα εισόδου και εξόδου χρησιμοποιώντας placeholders
void definePaths() {
    String pdfInputPath = "YOUR_DOCUMENT_DIRECTORY/SAMPLE_PDF.pdf"; // Διαδρομή αρχείου PDF με κράτηση θέσης
    String wordOutputPath = "YOUR_OUTPUT_DIRECTORY/ConvertedToWord.docx"; // Διαδρομή DOCX εξόδου κράτησης θέσης
}
```
**Εξήγηση:**
- **`pdfInputPath`:** Η τοποθεσία του εγγράφου PDF προέλευσης.
- **`wordOutputPath`:** Ο επιθυμητός προορισμός για το αρχείο Word που έχει μετατραπεί.

#### Βήμα 3: Εκτέλεση μετατροπής
Χρησιμοποιήστε το `Converter` κλάση για να χειριστεί τη διαδικασία μετατροπής:
```java
// Εκτελέστε τη μετατροπή από PDF σε μορφή επεξεργασίας κειμένου
double convertPdfToWordProcessing(PdfLoadOptions loadOptions) {
    // Ορίστε διαδρομές εισόδου και εξόδου για τη διαδικασία μετατροπής
    String pdfInputPath = "YOUR_DOCUMENT_DIRECTORY/SAMPLE_PDF.pdf"; 
    String wordOutputPath = "YOUR_OUTPUT_DIRECTORY/ConvertedToWord.docx";

    // Μετατροπέας Instantiate με διαδρομή εισόδου PDF και επιλογές φόρτωσης
    Converter converter = new Converter(pdfInputPath, () -> loadOptions);

    // Ορισμός επιλογών μετατροπής για τη μορφή επεξεργασίας κειμένου
    WordProcessingConvertOptions options = new WordProcessingConvertOptions();

    // Μετατροπή εγγράφου από PDF σε μορφή επεξεργασίας κειμένου
    converter.convert(wordOutputPath, options);
    
    return 0; // Τιμή επιστροφής κράτησης θέσης
}
```
**Εξήγηση:**
- **`Converter`:** Αρχικοποιείται με τις επιλογές διαδρομής και φόρτωσης.
- **`WordProcessingConvertOptions`:** Ρυθμίζει τις παραμέτρους για το έγγραφο Word προορισμού.
