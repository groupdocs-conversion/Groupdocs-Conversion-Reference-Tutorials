## Οδηγός Εφαρμογής

Ας αναλύσουμε την υλοποίηση σε λογικά βήματα:

### Εξαγωγή αρχείων από ZIP και μετατροπή σε PDF

Αυτή η λειτουργία επιδεικνύει τον χειρισμό κάθε αρχείου σε ένα συμπιεσμένο αρχείο zip και τη μετατροπή του σε μεμονωμένο έγγραφο PDF χρησιμοποιώντας το GroupDocs.Conversion.

#### Βήμα 1: Αρχικοποίηση μετατροπέα

Δημιουργήστε ένα `Converter` παράδειγμα με τη διαδρομή του αρχείου ZIP σας:

```java
import com.groupdocs.conversion.Converter;
import com.groupdocs.conversion.options.convert.PdfConvertOptions;

import java.io.FileNotFoundException;
import java.io.FileOutputStream;
import java.nio.file.Paths;

String sampleZipPath = "YOUR_DOCUMENT_DIRECTORY/SAMPLE_ZIP";
String outputFolder = "YOUR_OUTPUT_DIRECTORY";

try (Converter converter = new Converter(sampleZipPath)) {
    // Συνέχεια με τη μετατροπή
}
```

#### Βήμα 2: Διαμόρφωση επιλογών μετατροπής

Ρυθμίστε τις επιλογές μετατροπής PDF για να καθορίσετε τον τρόπο μετατροπής κάθε αρχείου:

```java
PdfConvertOptions options = new PdfConvertOptions();
final int[] i = {0};
```

#### Βήμα 3: Εκτέλεση μετατροπής

Επαναλάβετε κάθε αρχείο στο ZIP και μετατρέψτε το σε ξεχωριστό έγγραφο PDF:

```java
converter.convert(() -> {
    try {
        // Δημιουργήστε μοναδικά ονόματα αρχείων για PDF που έχουν μετατραπεί χρησιμοποιώντας ένα αυξανόμενο ευρετήριο
        return new FileOutputStream(Paths.get(outputFolder, String.format("converted-%d.pdf", ++i[0])).toFile());
    } catch (FileNotFoundException e) {
        throw new RuntimeException(e);
    }
}, options);
```

### Εξήγηση

- **`Converter`:** Αρχικοποιεί τη διαδικασία μετατροπής με το καθορισμένο αρχείο ZIP.
- **`PdfConvertOptions`:** Ρυθμίζει τον τρόπο μετατροπής των αρχείων σε μορφή PDF.
- **Αυξανόμενος Δείκτης:** Διασφαλίζει ότι κάθε PDF έχει ένα μοναδικό όνομα αρχείου.
