## Οδηγός Εφαρμογής
Ας αναλύσουμε την υλοποίηση σε διαχειρίσιμα τμήματα:

### Δυνατότητα: Μετατροπή εγγράφου από FTP σε PDF
#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς να κατεβάσετε ένα έγγραφο από έναν διακομιστή FTP και να το μετατρέψετε σε αρχείο PDF χρησιμοποιώντας το GroupDocs.Conversion για Java.
##### Βήμα 1: Ανάκτηση αρχείου από διακομιστή FTP
Για την ανάκτηση αρχείων, θα χρησιμοποιήσουμε το Apache Commons Net. Συνδεθείτε στον διακομιστή FTP σας, μεταβείτε στον επιθυμητό κατάλογο και ανακτήστε το έγγραφο:

```java
private static InputStream getFileFromFtp(String server, String dirname, String fileName) throws Exception {
    FTPClient client = new FTPClient();
    
    // Σύνδεση με τον διακομιστή FTP
    client.connect(server);
    
    // Συνδεθείτε με τα διαπιστευτήριά σας (αντικαταστήστε τα "ανώνυμος" και "" εάν χρειάζεται)
    client.login("username", "password");
    
    // Αλλαγή καταλόγου εργασίας στον διακομιστή
    client.changeWorkingDirectory(dirname);
    
    // Ανακτήστε το αρχείο και επιστρέψτε το InputStream του
    return client.retrieveFileStream(fileName);
}
```
Αυτή η μέθοδος συνδέεται με τον διακομιστή FTP, συνδέεται με συγκεκριμένα διαπιστευτήρια, αλλάζει τον κατάλογο και ανακτά το αρχείο ως `InputStream`.
##### Βήμα 2: Μετατροπή εγγράφου σε PDF
Μόλις έχετε το αρχείο, χρησιμοποιήστε το GroupDocs.Conversion για να το μετατρέψετε σε PDF:

```java
public static void run() {
    String server = "127.0.0.1"; // Διεύθυνση διακομιστή FTP
    String convertedFile = YOUR_OUTPUT_DIRECTORY + "/LoadDocumentFromFtp.pdf";
    String dirname = "pub"; // Κατάλογος στον διακομιστή FTP
    String fileName = "sample.docx"; // Αρχείο για ανάκτηση και μετατροπή

    try {
        // Αρχικοποίηση του μετατροπέα με μια συνάρτηση λάμδα για ανάκτηση αρχείου από FTP
        Converter converter = new Converter(() -> getFileFromFtp(server, dirname, fileName));
        
        // Ορισμός επιλογών μετατροπής PDF
        PdfConvertOptions options = new PdfConvertOptions();
        
        // Μετατρέψτε το έγγραφο και αποθηκεύστε το ως PDF
        converter.convert(convertedFile, options);
    } catch (Exception e) {
        throw new RuntimeException(e.getMessage());
    }
}
```
Αυτό το απόσπασμα κώδικα αρχικοποιεί ένα `Converter` αντικείμενο με μια ροή αρχείων FTP και ορίζει τις προεπιλεγμένες επιλογές μετατροπής PDF.

Η προσαρμογή των επιλογών μετατροπής μπορεί να βελτιώσει την ποιότητα του αποτελέσματος. Δείτε πώς μπορείτε να διαμορφώσετε αυτές τις ρυθμίσεις:

```java
public class PdfConversionOptions {

    }
}
```
Αυτή η ρύθμιση επιτρέπει προσαρμοσμένες διαμορφώσεις όπως η προσαρμογή μεγεθών σελίδας και περιθωρίων, αν και οι προεπιλογές επαρκούν για βασικές ανάγκες.
