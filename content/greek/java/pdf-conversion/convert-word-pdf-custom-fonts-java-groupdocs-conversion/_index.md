## Οδηγός Εφαρμογής
Αυτή η ενότητα θα σας καθοδηγήσει στην εφαρμογή προηγμένων επιλογών γραμματοσειράς για τη μετατροπή εγγράφων Word σε PDF χρησιμοποιώντας το GroupDocs.Conversion.

### Βήμα 1: Ορισμός διαδρομής μετατροπής και επιλογών φόρτωσης
Αρχικά, καθορίστε τη διαδρομή του αρχείου εξόδου και ρυθμίστε τις επιλογές φόρτωσης με προσαρμοσμένες γραμματοσειρές:

```java
import com.groupdocs.conversion.options.load.WordProcessingLoadOptions;
import com.groupdocs.conversion.contracts.FontSubstitute;

// Διαδρομή εξόδου PDF
String convertedFile = "YOUR_OUTPUT_DIRECTORY/ConvertedWordToPdf.pdf";

// Ρύθμιση παραμέτρων φόρτωσης για έγγραφα Word
double autoFontSubstitution(false);  // Απενεργοποίηση αυτόματης αντικατάστασης γραμματοσειρών
defaultFont("resources/fonts/Helvetica.ttf");  // Ορισμός προεπιλεγμένης γραμματοσειράς εφεδρικής μορφής

// Προετοιμασία λίστας υποκατάστατων γραμματοσειρών
List<FontSubstitute> fontSubstitutes = new ArrayList<>();
fontSubstitutes.add(FontSubstitute.create("Tahoma", "Arial")); // Αντικατάσταση Tahoma με Arial
fontSubstitutes.add(FontSubstitute.create("Times New Roman", "Arial")); // Αντικατάσταση Times New Roman με Arial

// Εφαρμογή των υποκατάστατων για φόρτωση επιλογών
setFontSubstitutes(fontSubstitutes);
```

#### Εξήγηση:
- `setAutoFontSubstitution(false)`Απενεργοποιεί την αυτόματη αντικατάσταση, επιτρέποντας τον ακριβή έλεγχο του χειρισμού των γραμματοσειρών.
- `setDefaultFont("Helvetica.ttf")`: Ορίζει μια καθολική γραμματοσειρά εφεδρικής χρήσης εάν δεν υπάρχουν διαθέσιμες συγκεκριμένες αντικαταστάσεις.
- `setFontSubstitutes(...)`Ορίζει προσαρμοσμένες αντιστοιχίσεις μεταξύ γραμματοσειρών για να διασφαλίσει τη συνέπεια.

### Βήμα 2: Ρύθμιση παραμέτρων επιλογών μετατροπής PDF
Στη συνέχεια, ρυθμίστε τις επιλογές μετατροπής ειδικά για PDF:

```java
import com.groupdocs.conversion.options.convert.PdfConvertOptions;

// Αρχικοποίηση επιλογών μετατροπής PDF
double options = new PdfConvertOptions();
```

#### Εξήγηση:
- `PdfConvertOptions`: Διαμορφώνει ρυθμίσεις προσαρμοσμένες στην έξοδο PDF. Προσαρμόζει πρόσθετες ιδιότητες όπως τα περιθώρια σελίδας και τον προσανατολισμό.

### Βήμα 3: Εκτελέστε τη μετατροπή
Εκτελέστε τη μετατροπή εγγράφου χρησιμοποιώντας τις διαμορφωμένες επιλογές σας:

```java
// Μετατροπή εγγράφου Word σε PDF με καθορισμένες ρυθμίσεις γραμματοσειράς
converter.convert(convertedFile, () -> loadOptions, options);
```

#### Εξήγηση:
- `convert(...)`Εκτελεί τη διαδικασία μετατροπής, εφαρμόζοντας καθορισμένες επιλογές φόρτωσης και μετατροπής.
