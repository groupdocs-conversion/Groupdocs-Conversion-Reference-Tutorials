## Οδηγός Εφαρμογής

### Απόκρυψη αλλαγών που παρακολουθούνται στη μετατροπή Word σε PDF

Αυτή η λειτουργία σάς επιτρέπει να μετατρέπετε έγγραφα, διατηρώντας παράλληλα το τελικό PDF απαλλαγμένο από εντοπισμένες αλλαγές. Δείτε πώς μπορείτε να την εφαρμόσετε:

#### Βήμα 1: Ρύθμιση επιλογών φόρτωσης
Αρχικά, διαμορφώστε τις επιλογές φόρτωσης ειδικά για έγγραφα επεξεργασίας Word.

```java
import com.groupdocs.conversion.Converter;
import com.groupdocs.conversion.options.convert.PdfConvertOptions;
import com.groupdocs.conversion.options.load.WordProcessingLoadOptions;

// Δημιουργήστε επιλογές φόρτωσης για να αποκρύψετε τις εντοπισμένες αλλαγές
WordProcessingLoadOptions loadOptions = new WordProcessingLoadOptions();
loadOptions.setHideWordTrackedChanges(true); // Απόκρυψη αλλαγών που παρακολουθούνται κατά τη μετατροπή
```

#### Βήμα 2: Αρχικοποίηση μετατροπέα με επιλογές φόρτωσης

```java
String inputFile = "YOUR_DOCUMENT_DIRECTORY/SAMPLE_DOCX_WITH_TRACKED_CHANGES";
String outputFile = "YOUR_OUTPUT_DIRECTORY/ConvertWordProcessingByHiddingTrackedChanges.pdf";

// Δημιουργήστε ένα αντικείμενο Converter χρησιμοποιώντας το αρχείο εισόδου και τις επιλογές φόρτωσης
Converter converter = new Converter(inputFile, () -> loadOptions);
```

#### Βήμα 3: Ρύθμιση παραμέτρων επιλογών μετατροπής PDF

```java
PdfConvertOptions pdfOptions = new PdfConvertOptions(); // Προσαρμόστε τις επιλογές όπως απαιτείται
converter.convert(outputFile, pdfOptions); // Εκτελέστε τη μετατροπή
```

### Φόρτωση εγγράφου με προσαρμοσμένες επιλογές φόρτωσης

Αυτή η λειτουργία επιδεικνύει τη φόρτωση εγγράφων χρησιμοποιώντας προσαρμοσμένες διαμορφώσεις. Δείτε πώς μπορείτε να τη ρυθμίσετε:
