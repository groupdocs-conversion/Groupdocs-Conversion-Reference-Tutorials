# Πώς να μετατρέψετε έγγραφα που προστατεύονται με κωδικό πρόσβασης χρησιμοποιώντας το GroupDocs.Conversion για Java

Δυσκολεύεστε να μετατρέψετε έγγραφα που προστατεύονται με κωδικό πρόσβασης σε μορφές υπολογιστικών φύλλων σε Java; Με το GroupDocs.Conversion API, η μετατροπή προστατευμένων εγγράφων Word σε υπολογιστικά φύλλα Excel είναι απρόσκοπτη και αποτελεσματική. Αυτό το σεμινάριο σας καθοδηγεί στη φόρτωση και μετατροπή αρχείων Word που προστατεύονται με κωδικό πρόσβασης χρησιμοποιώντας προηγμένες επιλογές.

- Ρύθμιση παραμέτρων για προχωρημένους ρυθμίσεων μετατροπής, όπως επίπεδο ζουμ και εύρος σελίδων.

Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε έτοιμες τις απαραίτητες προϋποθέσεις.

## Οδηγός Εφαρμογής

Θα καλύψουμε τη φόρτωση εγγράφων που προστατεύονται με κωδικό πρόσβασης και τη μετατροπή τους σε μορφές υπολογιστικών φύλλων με προηγμένες επιλογές.

### Φόρτωση εγγράφου που προστατεύεται με κωδικό πρόσβασης

**Επισκόπηση:**
Η φόρτωση περιλαμβάνει τη ρύθμιση επιλογών φόρτωσης που περιλαμβάνουν τον κωδικό πρόσβασης του εγγράφου. Αυτό το βήμα είναι κρίσιμο για τη μετατροπή.

#### Βήμα 1: Ρύθμιση επιλογών φόρτωσης
```java
import com.groupdocs.conversion.options.load.WordProcessingLoadOptions;

WordProcessingLoadOptions loadOptions = new WordProcessingLoadOptions();
loadOptions.setPassword("12345"); // Αντικαταστήστε το με τον κωδικό πρόσβασης του εγγράφου σας.
```
- **Εξήγηση:** Ο `WordProcessingLoadOptions` Η κλάση χειρίζεται έγγραφα που προστατεύονται με κωδικό πρόσβασης. Δώστε τον σωστό κωδικό πρόσβασης για να αποκτήσετε πρόσβαση στο περιεχόμενο.

### Μετατροπή σε υπολογιστικό φύλλο με επιλογές για προχωρημένους

**Επισκόπηση:**
Μετά τη φόρτωση, μετατρέψτε το έγγραφο σε υπολογιστικό φύλλο Excel, εστιάζοντας σε συγκεκριμένες σελίδες και προσαρμοσμένες ρυθμίσεις, όπως τα επίπεδα ζουμ.

#### Βήμα 2: Διαμόρφωση επιλογών μετατροπής
```java
import com.groupdocs.conversion.Converter;
import com.groupdocs.conversion.filetypes.SpreadsheetFileType;
import com.groupdocs.conversion.options.convert.SpreadsheetConvertOptions;

SpreadsheetConvertOptions options = new SpreadsheetConvertOptions();
options.setPageNumber(2); // Ξεκινήστε από τη σελίδα 2.
options.setPagesCount(1); // Μετατρέψτε μόνο μία σελίδα.
options.setFormat(SpreadsheetFileType.Xls); // Μορφή εξόδου ως XLS.
options.setZoom(150); // Ορισμός επιπέδου ζουμ για μετατροπή (συντελεστής κλίμακας).
```
- **Εξήγηση:** Καθορίστε σελίδες για μετατροπή (`setPageNumber` και `setPagesCount`) και τη μορφή εξόδου. Η `setZoom` Η επιλογή προσαρμόζει την κλιμάκωση περιεχομένου κατά τη μετατροπή.

#### Βήμα 3: Εκτέλεση μετατροπής
```java
String convertedFile = Constants.getConvertedPath("ConvertToSpreadsheetWithAdvancedOptions.xls");
Converter converter = new Converter(Constants.SAMPLE_DOCX_WITH_PASSWORD, () -> loadOptions);
converter.convert(convertedFile, options);
```
- **Εξήγηση:** Ο `convert` Η μέθοδος εκτελεί τον μετασχηματισμό χρησιμοποιώντας προηγουμένως ορισμένες διαμορφώσεις. Βεβαιωθείτε ότι οι διαδρομές αρχείων είναι σωστές για είσοδο και έξοδο.
