## Οδηγός Υλοποίησης

### Πώς Να Μετατρέψετε DWG Σε TIFF Με Προσαρμοσμένες Διαστάσεις

#### Βήμα 1: Εισαγωγή Απαραίτητων Βιβλιοθηκών
```java
import com.groupdocs.conversion.Converter;
import com.groupdocs.conversion.filetypes.ImageFileType;
import com.groupdocs.conversion.options.convert.ImageConvertOptions;
import com.groupdocs.conversion.options.load.CadLoadOptions;
```

#### Βήμα 2: Ρύθμιση Επιλογών Φόρτωσης Με Προσαρμοσμένες Διαστάσεις
```java
String sourceDocumentPath = "YOUR_DOCUMENT_DIRECTORY/SAMPLE_DWG_WITH_LAYOUTS_AND_LAYERS";
CadLoadOptions loadOptions = new CadLoadOptions();
loadOptions.setWidth(1920); // Specify the desired width in pixels
loadOptions.setHeight(1080); // Specify the desired height in pixels
Converter converter = new Converter(sourceDocumentPath, () -> loadOptions);
```
**Επεξήγηση**: Ρυθμίζουμε το `CadLoadOptions` για να ορίσουμε τις διαστάσεις εξόδου, διασφαλίζοντας ότι όταν το έγγραφο CAD φορτώνεται, συμμορφώνεται με αυτές τις καθορισμένες μετρήσεις.

#### Βήμα 3: Διαμόρφωση Επιλογών Μετατροπής
```java
String convertedFilePath = "YOUR_OUTPUT_DIRECTORY/ConvertCadAndSpecifyWidthAndHeight.tiff";
ImageConvertOptions options = new ImageConvertOptions();
options.setFormat(ImageFileType.Tiff); // Set the conversion target to TIFF format
```

#### Βήμα 4: Εκτέλεση της Μετατροπής
```java
converter.convert(convertedFilePath, options);
```
**Επεξήγηση**: Καθορίζοντας το `ImageFileType.Tiff`, κατευθύνετε το GroupDocs.Conversion να δημιουργήσει μια εικόνα TIFF. Η διαδικασία αξιοποιεί αυτές τις ρυθμίσεις για να παραγάγει ένα βελτιστοποιημένο αρχείο.
