## Οδηγός Εφαρμογής

### Φόρτωση εγγράφων CAD με προσαρμοσμένες διαστάσεις

**Επισκόπηση**Αυτή η λειτουργία επιτρέπει τη φόρτωση ενός εγγράφου CAD, ενώ παράλληλα καθορίζετε τις διαστάσεις του πριν από τη μετατροπή. Είναι χρήσιμη για την προετοιμασία αρχείων για συγκεκριμένες απαιτήσεις εμφάνισης.

#### Βήμα 1: Εισαγωγή απαραίτητων βιβλιοθηκών
```java
import com.groupdocs.conversion.Converter;
import com.groupdocs.conversion.filetypes.ImageFileType;
import com.groupdocs.conversion.options.convert.ImageConvertOptions;
import com.groupdocs.conversion.options.load.CadLoadOptions;
```

#### Βήμα 2: Ρύθμιση επιλογών φόρτωσης με προσαρμοσμένες διαστάσεις
```java
String sourceDocumentPath = "YOUR_DOCUMENT_DIRECTORY/SAMPLE_DWG_WITH_LAYOUTS_AND_LAYERS";
CadLoadOptions loadOptions = new CadLoadOptions();
loadOptions.setWidth(1920); // Καθορίστε το επιθυμητό πλάτος σε pixel
loadOptions.setHeight(1080); // Καθορίστε το επιθυμητό ύψος σε pixel
Converter converter = new Converter(sourceDocumentPath, () -> loadOptions);
```
**Εξήγηση**: Στήσαμε `CadLoadOptions` για να ορίσετε τις διαστάσεις εξόδου, διασφαλίζοντας ότι όταν φορτώνεται το έγγραφο CAD, αυτό συμμορφώνεται με αυτές τις καθορισμένες μετρήσεις.

### Μετατροπή εικόνας CAD σε TIFF

**Επισκόπηση**Αφού φορτώσετε το αρχείο CAD με προσαρμοσμένες διαστάσεις, μετατρέψτε το σε μορφή εικόνας TIFF, ιδανική για αποτελέσματα υψηλής ποιότητας.

#### Βήμα 3: Διαμόρφωση επιλογών μετατροπής
```java
String convertedFilePath = "YOUR_OUTPUT_DIRECTORY/ConvertCadAndSpecifyWidthAndHeight.tiff";
ImageConvertOptions options = new ImageConvertOptions();
options.setFormat(ImageFileType.Tiff); // Ορίστε τον στόχο μετατροπής σε μορφή TIFF
```

#### Βήμα 4: Εκτελέστε τη μετατροπή
```java
converter.convert(convertedFilePath, options);
```
**Εξήγηση**: Καθορίζοντας `ImageFileType.Tiff`, κατευθύνετε το GroupDocs.Conversion για να εκτυπώσει μια εικόνα TIFF. Η διαδικασία αξιοποιεί αυτές τις ρυθμίσεις για να δημιουργήσει ένα βελτιστοποιημένο αρχείο.
