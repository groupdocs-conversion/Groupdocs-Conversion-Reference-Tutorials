## Οδηγός Υλοποίησης

### Βήμα 1: Καθορισμός Διαδρομών Αρχείων και Διατάξεων
Ορίστε τις διαδρομές για το αρχείο CAD εισόδου και το PDF εξόδου, ορίζοντας ποιες διατάξεις θέλετε να μετατρέψετε:
```java
String sourceFilePath = "YOUR_DOCUMENT_DIRECTORY/sample.dwg";
String convertedFile = "YOUR_OUTPUT_DIRECTORY/ConvertCadAndSpecifyLayouts.pdf";

// Specify desired layout names
cadLoadOptions loadOptions = new CadLoadOptions();
loadOptions.setLayoutNames(new String[] { "Layout1", "Layout3" });
```

### Βήμα 2: Αρχικοποίηση του Converter
Αρχικοποιήστε την κλάση `Converter` με τη διαδρομή του αρχείου και τις επιλογές φόρτωσης:
```java
Converter converter = new Converter(sourceFilePath, () -> loadOptions);
```
Αυτό επιτρέπει τον καθορισμό των διατάξεων που θα συμπεριληφθούν στη μετατροπή.

### Βήμα 3: Ορισμός Επιλογών Μετατροπής
Διαμορφώστε τις ρυθμίσεις μετατροπής PDF χρησιμοποιώντας `PdfConvertOptions`:
```java
PdfConvertOptions convertOptions = new PdfConvertOptions();
```
Αυτές οι επιλογές επιτρέπουν περαιτέρω προσαρμογές, όπως ο καθορισμός DPI ή συγκεκριμένων περιοχών σελίδων.

### Βήμα 4: Εκτέλεση της Μετατροπής
Εκτελέστε τη διαδικασία μετατροπής καλώντας τη μέθοδο `convert`:
```java
converter.convert(convertedFile, convertOptions);
```
Αυτό δημιουργεί ένα αρχείο PDF που περιέχει μόνο τις καθορισμένες διατάξεις από το έγγραφο CAD σας.
