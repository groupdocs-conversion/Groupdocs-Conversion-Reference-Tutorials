1. **Ρύθμιση διαδρομών εγγράφων**
   Ορίστε καταλόγους για τα αρχεία πηγής και την έξοδο που έχετε μετατρέψει:
   ```csharp

   const string OutputDirectory = "YOUR_OUTPUT_DIRECTORY";
   ```

2. **Φόρτωση του αρχείου προέλευσης HTM**
   Χρησιμοποιήστε το `Converter` κλάση για να φορτώσετε το αρχείο HTML σας:
   ```csharp
   string inputFilePath = Path.Combine(DocumentDirectory, "sample.htm");

   using (var converter = new Converter(inputFilePath))
   {
       // Ο κώδικας μετατροπής θα τοποθετηθεί εδώ.
   }
   ```

3. **Ορισμός επιλογών μετατροπής**
   Ορίστε τις επιλογές μετατροπής για τη μορφή CSV:
   ```csharp
   SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Csv };
   ```

4. **Εκτελέστε τη μετατροπή**
   Εκτελέστε τη μετατροπή και αποθηκεύστε την έξοδο:
   ```csharp
   string outputFilePath = Path.Combine(OutputDirectory, "converted.csv");
   converter.Convert(outputFilePath, options);
   ```
