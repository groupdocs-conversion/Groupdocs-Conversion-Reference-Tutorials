## Οδηγός Εφαρμογής

Τώρα, ας εμβαθύνουμε στη μετατροπή ενός XLTM σε HTML βήμα προς βήμα.

### Φόρτωση του αρχείου προέλευσης
Αρχικά, καθορίστε το αρχείο .xltm πηγής σας ορίζοντας διαδρομές και προετοιμάζοντας για μετατροπή.

#### Ρύθμιση διαδρομών
```csharp
using System.IO;

string inputFilePath = Path.Combine(@"YOUR_DOCUMENT_DIRECTORY", "sample.xltm");
string outputFolder = Path.Combine(@"YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "xltm-converted-to.html");
```

### Μετατροπή XLTM σε HTML
Στη συνέχεια, εκτελέστε τη μετατροπή χρησιμοποιώντας το GroupDocs.Conversion.

#### Ρύθμιση παραμέτρων επιλογών μετατροπής
```csharp
using GroupDocs.Conversion.Options.Convert;

var options = new WebConvertOptions();
```

#### Εκτελέστε τη μετατροπή
```csharp
using (var converter = new Converter(inputFilePath))
{
    // Μετατροπή και αποθήκευση σε μορφή HTML
    converter.Convert(outputFile, options);
}
```

### Επεξήγηση βασικών παραμέτρων
- **Μετατροπέας:** Αρχικοποιείται με το αρχείο προέλευσης.
- **Επιλογές WebConvert:** Καθορίζει ότι μετατρέπουμε σε μορφή φιλική προς το web (HTML).
- **Μέθοδος μετατροπής:** Εκτελεί τη διαδικασία μετατροπής.
