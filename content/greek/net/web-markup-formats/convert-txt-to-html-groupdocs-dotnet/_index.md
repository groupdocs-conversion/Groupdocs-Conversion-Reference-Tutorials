### Μετατροπή αρχείου TXT σε HTML
Αυτή η λειτουργία σάς επιτρέπει να μετατρέπετε εύκολα έγγραφα απλού κειμένου σε αρχεία HTML φιλικά προς το web, ιδανικά για δημοσίευση ή επεξεργασία σε προγράμματα επεξεργασίας WYSIWYG.

#### Βήμα 1: Ορισμός καταλόγου εξόδου και διαδρομών αρχείων
Δημιουργήστε καταλόγους εάν δεν υπάρχουν και καθορίστε διαδρομές για τα αρχεία εισόδου και εξόδου:

```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", "Output");
Directory.CreateDirectory(outputFolder); // Βεβαιωθείτε ότι ο κατάλογος υπάρχει

string outputFile = Path.Combine(outputFolder, "txt-converted-to.html");
string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.txt");
```

**Εξήγηση**Χρησιμοποιούμε `Path.Combine` για τη δημιουργία διαδρομών αρχείων συμβατών με διαφορετικά λειτουργικά συστήματα. Η διασφάλιση της ύπαρξης καταλόγων εκ των προτέρων αποτρέπει σφάλματα χρόνου εκτέλεσης.

#### Βήμα 2: Φόρτωση και μετατροπή του αρχείου TXT
Χρησιμοποιήστε το GroupDocs.Conversion για να φορτώσετε το έγγραφο TXT και να το μετατρέψετε χρησιμοποιώντας επιλογές που αφορούν συγκεκριμένα την HTML:

```csharp
using (var converter = new Converter(inputFilePath))
{
    var options = new WebConvertOptions(); // Ορισμός επιλογών μετατροπής για μορφή HTML
    
    // Εκτέλεση μετατροπής από TXT σε HTML
    converter.Convert(outputFile, options);
}
```

**Εξήγηση**: Το `Converter` Το αντικείμενο χειρίζεται την είσοδο αρχείου. `WebConvertOptions` Διαμορφώνει ρυθμίσεις ειδικά για μορφές ιστού όπως η HTML.
