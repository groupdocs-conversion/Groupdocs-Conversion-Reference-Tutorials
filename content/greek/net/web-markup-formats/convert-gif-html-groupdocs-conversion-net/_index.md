## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα δούμε πώς να μετατρέψετε ένα GIF σε HTML χρησιμοποιώντας το GroupDocs.Conversion.

### Επισκόπηση

Η μετατροπή ενός GIF σε HTML σάς επιτρέπει να μετατρέψετε κινούμενες εικόνες σε μορφές φιλικές προς το web, βελτιώνοντας την διαδραστικότητα και τη συμβατότητα με τα σύγχρονα προγράμματα περιήγησης.

#### Βήμα 1: Ορισμός διαδρομών για αρχεία εισόδου και εξόδου

Καθορίστε τις διαδρομές για το αρχείο GIF πηγής και τον κατάλογο HTML εξόδου:

```csharp
string documentPath = Path.Combine(@"YOUR_DOCUMENT_DIRECTORY", "sample.gif");
string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "gif-converted-to.html");
```

#### Βήμα 2: Φορτώστε το αρχείο GIF πηγής

Χρησιμοποιήστε το `Converter` κλάση για να φορτώσετε το αρχείο GIF σας. Αυτό το βήμα ξεκινά τη διαδικασία μετατροπής:

```csharp
using (var converter = new GroupDocs.Conversion.Converter(documentPath))
{
    // Συνέχεια με τη ρύθμιση των επιλογών μετατροπής
}
```

#### Βήμα 3: Ορισμός επιλογών μετατροπής για μορφή HTML

Ορίστε τις παραμέτρους μετατροπής χρησιμοποιώντας `WebConvertOptions`Αυτές οι ρυθμίσεις καθορίζουν τον τρόπο με τον οποίο το GIF σας μετατρέπεται σε HTML:

```csharp
var options = new WebConvertOptions();
// Μετατρέψτε και αποθηκεύστε το αρχείο εξόδου
converter.Convert(outputFile, options);
```
