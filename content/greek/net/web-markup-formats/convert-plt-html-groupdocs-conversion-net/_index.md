## Οδηγός Εφαρμογής

### Χαρακτηριστικό 1: Φόρτωση αρχείου PLT πηγής

#### Επισκόπηση
Φορτώστε ένα αρχείο PLT πηγής για να το προετοιμάσετε για μετατροπή HTML.

**Βήμα 1: Εισαγωγή της απαραίτητης βιβλιοθήκης**
Βεβαιωθείτε ότι έχετε συμπεριλάβει τον χώρο ονομάτων GroupDocs.Conversion:
```csharp
using GroupDocs.Conversion;
```

**Βήμα 2: Καθορισμός καταλόγου εγγράφων και φόρτωση αρχείου**
Ορίστε τον κατάλογο εγγράφων σας και φορτώστε το αρχείο PLT χρησιμοποιώντας το `Converter` κλάση. Αυτό το βήμα αρχικοποιεί τη διαδικασία μετατροπής.
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
GroupDocs.Conversion.Converter converter = new GroupDocs.Conversion.Converter(Path.Combine(documentDirectory, "sample.plt"));
```

### Λειτουργία 2: Ρύθμιση παραμέτρων επιλογών μετατροπής HTML

#### Επισκόπηση
Διαμορφώστε τις ρυθμίσεις για τη μετατροπή αρχείων PLT σε μορφή HTML.

**Βήμα 1: Εισαγωγή χώρου ονομάτων επιλογών μετατροπής**
```csharp
using GroupDocs.Conversion.Options.Convert;
```

**Βήμα 2: Αρχικοποίηση του WebConvertOptions**
Στήνω `WebConvertOptions` για να προσαρμόσετε τον τρόπο με τον οποίο θα μετατραπεί το έγγραφό σας σε HTML:
```csharp
WebConvertOptions htmlOptions = new WebConvertOptions();
```

### Χαρακτηριστικό 3: Μετατροπή PLT σε HTML και αποθήκευση εξόδου

#### Επισκόπηση
Χειριστείτε τη μετατροπή ενός φορτωμένου αρχείου PLT σε μορφή HTML και αποθηκεύστε το στην επιθυμητή θέση.

**Βήμα 1: Καθορισμός διαδρομών**
Προσδιορίστε τόσο τον κατάλογο εγγράφων όσο και τον κατάλογο εξόδου:
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
```

**Βήμα 2: Εκτέλεση μετατροπής και αποθήκευση εξόδου**
Μετατρέψτε το αρχείο PLT χρησιμοποιώντας τις επιλογές που έχουν διαμορφωθεί προηγουμένως και αποθηκεύστε την έξοδο HTML:
```csharp
GroupDocs.Conversion.Converter converter = new GroupDocs.Conversion.Converter(Path.Combine(documentDirectory, "sample.plt"));
WebConvertOptions htmlOptions = new WebConvertOptions();
string outputFile = Path.Combine(outputDirectory, "plt-converted-to.html");
converter.Convert(outputFile, htmlOptions);
```
