## Οδηγός Εφαρμογής
### Μετατροπή HTML σε LaTeX
#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε έγγραφα HTML σε μορφή TEX, καθιστώντας την κατάλληλη για διάφορους τεχνικούς και ακαδημαϊκούς σκοπούς.
#### Βήματα για την εφαρμογή
##### Βήμα 1: Ορισμός διαδρομών και επιλογών
Αρχικά, ορίστε τις απαραίτητες διαδρομές και επιλογές μετατροπής:

```csharp
using System.IO;
using GroupDocs.Conversion.Options.Convert;

// Ορίστε διαδρομές για τους καταλόγους εγγράφων και εξόδου
string sourceHtmlPath = "YOUR_DOCUMENT_DIRECTORY/sample.html"; // Αντικατάσταση με την πραγματική διαδρομή αρχείου HTML
string outputFolder = "YOUR_OUTPUT_DIRECTORY"; // Καθορίστε τον επιθυμητό κατάλογο εξόδου
string outputFile = Path.Combine(outputFolder, "html-converted-to.tex");

// Αρχικοποίηση του μετατροπέα
using (var converter = new Converter(sourceHtmlPath))
{
    Console.WriteLine("Converter initialized.");
}
```
##### Βήμα 2: Διαμόρφωση επιλογών μετατροπής
Ρυθμίστε τις επιλογές μετατροπής για τη μορφή LaTeX:

```csharp
// Ορισμός επιλογών μετατροπής για τη μορφή LaTeX
PageDescriptionLanguageConvertOptions options = new PageDescriptionLanguageConvertOptions { Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Tex };
```
##### Βήμα 3: Εκτέλεση μετατροπής
Εκτελέστε τη διαδικασία μετατροπής:

```csharp
// Μετατροπή HTML σε LaTeX
converter.Convert(outputFile, options);

Console.WriteLine($"Conversion complete. Output saved at {outputFile}");
```
