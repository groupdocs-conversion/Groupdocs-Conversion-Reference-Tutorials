## Οδηγός Εφαρμογής
Τώρα, ας εξερευνήσουμε πώς να μετατρέψουμε αρχεία MHTML σε μορφή TEX.

### Φόρτωση του αρχείου MHTML πηγής
Ξεκινήστε φορτώνοντας το αρχείο MHTML πηγής στον μετατροπέα. Καθορίστε τη διαδρομή προς το έγγραφό σας:

```csharp
string documentPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.mhtml");
```

### Ορισμός καταλόγου εξόδου και διαδρομής αρχείου
Καθορίστε πού θέλετε να αποθηκευτεί το αρχείο TEX που έχει μετατραπεί:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "mhtml-converted-to.tex");
```

### Ορισμός επιλογών μετατροπής
Ορίστε επιλογές μετατροπής για τη μετατροπή MHTML σε μορφή TEX. Καθορίστε τον τύπο μορφής και τυχόν πρόσθετες διαμορφώσεις:

```csharp
using (var converter = new Converter(documentPath))
{
    var options = new PageDescriptionLanguageConvertOptions { Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Tex };
}
```

### Εκτελέστε τη μετατροπή
Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο εξόδου:

```csharp
converter.Convert(outputFile, options);
```
