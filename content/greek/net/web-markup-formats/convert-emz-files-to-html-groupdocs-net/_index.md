## Οδηγός Εφαρμογής

### Μετατροπή EMZ σε HTML

Αυτή η λειτουργία εστιάζει στη μετατροπή ενός αρχείου EMZ σε ένα προσβάσιμο έγγραφο HTML.

#### Βήμα 1: Ρύθμιση διαδρομών αρχείων

Ορίστε διαδρομές για το αρχείο εισόδου EMZ και τον κατάλογο εξόδου:

```csharp
string emzFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.emz");
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "emz-converted-to.html");
```

#### Βήμα 2: Φόρτωση του αρχείου EMZ προέλευσης

Χρησιμοποιήστε το `Converter` κλάση για να φορτώσετε το αρχείο EMZ:

```csharp
using (var converter = new Converter(emzFilePath))
{
    var options = new WebConvertOptions(); // Επιλογές μετατροπής HTML
    converter.Convert(outputFile, options); // Εκτελέστε τη μετατροπή
}
```

### Επεξήγηση παραμέτρων κώδικα

- **Επιλογές Μετατροπής Web**: Διαμορφώνει ρυθμίσεις για την έξοδο HTML. Προσαρμόστε τες με βάση τις ανάγκες σας.
- **μετατροπέας.Μετατροπή()**Αυτή η μέθοδος εκτελεί την πραγματική μετατροπή αρχείου και το αποθηκεύει στην καθορισμένη διαδρομή.
