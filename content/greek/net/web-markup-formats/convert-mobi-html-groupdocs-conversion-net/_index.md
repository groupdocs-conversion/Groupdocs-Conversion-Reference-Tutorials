## Οδηγός Εφαρμογής

Τώρα που έχουμε όλα τα απαραίτητα έτοιμα, ας αναλύσουμε την υλοποίηση σε διαχειρίσιμα βήματα.

### Χαρακτηριστικό: Μετατροπή αρχείου MOBI σε HTML

Αυτή η λειτουργία επιτρέπει τη μετατροπή μιας μορφής αρχείου MOBI, που χρησιμοποιείται συνήθως για ηλεκτρονικά βιβλία, σε ένα έγγραφο HTML. Αυτή η μετατροπή είναι ιδιαίτερα χρήσιμη όταν θέλετε να εμφανίσετε περιεχόμενο βιβλίου σε ιστοσελίδες ή σε εφαρμογές με υποστήριξη εμπλουτισμένου κειμένου.

#### Βήμα 1: Προετοιμάστε τους καταλόγους σας
Αρχικά, βεβαιωθείτε ότι οι κατάλογοι εισόδου και εξόδου έχουν ρυθμιστεί σωστά:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// Διαδρομή προς το αρχείο MOBI πηγής
string mobiFilePath = Path.Combine(documentDirectory, "sample.mobi");
// Διαδρομή όπου θα αποθηκευτεί το HTML που έχει μετατραπεί
string htmlOutputPath = Path.Combine(outputDirectory, "mobi-converted-to.html");
```

#### Βήμα 2: Αρχικοποίηση του μετατροπέα
Χρησιμοποιήστε το `Converter` κλάση για να φορτώσετε το αρχείο MOBI σας:

```csharp
try {
    using (var converter = new Converter(mobiFilePath)) {
        // Τα βήματα μετατροπής θα προστεθούν εδώ.
    }
} catch (Exception ex) {
    Console.WriteLine("An error occurred during conversion: " + ex.Message);
}
```

#### Βήμα 3: Ορισμός επιλογών μετατροπής
Ρυθμίστε τις παραμέτρους των επιλογών μετατροπής για να καθορίσετε ότι θέλετε μια έξοδο HTML:

```csharp
var options = new WebConvertOptions();
```
Ο `WebConvertOptions` Η κλάση έχει σχεδιαστεί για μετατροπές μορφών ιστού, διασφαλίζοντας ότι η HTML εξόδου σας διατηρεί την απαραίτητη μορφοποίηση και δομή.

#### Βήμα 4: Εκτελέστε τη μετατροπή
Τέλος, μετατρέψτε και αποθηκεύστε το αρχείο χρησιμοποιώντας τις καθορισμένες ρυθμίσεις:

```csharp
converter.Convert(htmlOutputPath, options);
```

### Συμβουλές αντιμετώπισης προβλημάτων:
- Βεβαιωθείτε ότι η διαδρομή του αρχείου MOBI που εισαγάγατε είναι σωστή για να αποφύγετε `FileNotFoundException`.
- Ελέγξτε ότι έχετε δικαιώματα εγγραφής για τον κατάλογο εξόδου.
- Χρησιμοποιήστε μπλοκ try-catch για να χειριστείτε με ομαλό τρόπο τις μη αναμενόμενες εξαιρέσεις.
