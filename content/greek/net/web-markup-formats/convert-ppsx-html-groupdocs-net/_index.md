## Οδηγός Εφαρμογής

Θα αναλύσουμε κάθε λειτουργία σε διαχειρίσιμα βήματα για λόγους σαφήνειας.

### Φόρτωση αρχείου PPSX

Αυτή η ενότητα παρουσιάζει τον τρόπο φόρτωσης ενός αρχείου PPSX χρησιμοποιώντας το GroupDocs.Conversion.

#### Βήμα 1: Ορίστε τη διαδρομή εγγράφου

Βεβαιωθείτε ότι έχετε ορίσει `documentDirectory` με τη διαδρομή του εγγράφου σας:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
```

#### Βήμα 2: Αρχικοποίηση κλάσης μετατροπέα

Δημιουργήστε μια παρουσία του `Converter` κλάση περνώντας τη διαδρομή αρχείου PPSX.

```csharp
var converter = new Converter(Path.Combine(documentDirectory, "sample.ppsx"));
```

### Μετατροπή PPSX σε HTML

Τώρα ας μετατρέψουμε το φορτωμένο αρχείο PPSX σε μορφή HTML.

#### Βήμα 1: Ορισμός διαδρομής καταλόγου εξόδου

Ορίστε τον κατάλογο εξόδου όπου θα αποθηκευτεί το αρχείο που έχει μετατραπεί:

```csharp
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
```

#### Βήμα 2: Διαμόρφωση επιλογών μετατροπής

Δημιουργήστε μια παρουσία του `WebConvertOptions` για να διαμορφώσετε τις ρυθμίσεις μετατροπής σας:

```csharp
var options = new WebConvertOptions();
```

#### Βήμα 3: Εκτελέστε τη μετατροπή

Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο HTML στον καθορισμένο κατάλογο.

```csharp
converter.Convert(Path.Combine(outputDirectory, "ppsx-converted-to.html"), options);
```
