## Ρύθμιση του GroupDocs.Conversion για .NET

Για να ξεκινήσετε, θα χρειαστεί να εγκαταστήσετε τη βιβλιοθήκη GroupDocs.Conversion. Δείτε πώς:

**Χρησιμοποιώντας την Κονσόλα Διαχείρισης Πακέτων NuGet:**
```bash

```

**Χρησιμοποιώντας το .NET CLI:**
```bash
dotnet add package GroupDocs.Conversion --version 25.3.0
```

Αρχικά, βεβαιωθείτε ότι ο κατάλογος εξόδου σας υπάρχει:

```csharp

    Directory.CreateDirectory(outputFolder);
}
```

**Εξήγηση:** Αυτός ο κώδικας ελέγχει εάν ο καθορισμένος κατάλογος υπάρχει και τον δημιουργεί εάν όχι. Είναι κρίσιμος για την αποθήκευση των αρχείων που έχουν μετατραπεί.

### Φόρτωση και μετατροπή αρχείου MPT

Τώρα, φορτώστε το αρχείο MPT πηγής σας και μετατρέψτε το:

```csharp
string outputFile = Path.Combine(outputFolder, "mpt-converted-to.html");

using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY\\sample.mpt"))
{
    var options = new WebConvertOptions(); // Επιλογές μετατροπής σε μορφή HTML.
    
    converter.Convert(outputFile, options);
}
```

**Εξήγηση:** Ο `Converter` Η κλάση φορτώνει το αρχείο MPT. Η `WebConvertOptions` καθορίζει ότι μετατρέπουμε σε HTML.
