## Οδηγός Εφαρμογής

### Μετατροπή DJVU σε HTML

#### Επισκόπηση
Αυτή η λειτουργία μετατρέπει ένα αρχείο DJVU σε έγγραφο HTML χρησιμοποιώντας το GroupDocs.Conversion, διευκολύνοντας την προβολή και την κοινή χρήση του περιεχομένου σε πλατφόρμες ιστού.

#### Βήμα προς βήμα εφαρμογή
**1. Ορισμός Διαδρομών**
Ρυθμίστε τους καταλόγους εισόδου και εξόδου:
```csharp
string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY";
string outputFileDirectory = @"YOUR_OUTPUT_DIRECTORY/";

// Καθορίστε τη διαδρομή του αρχείου DJVU
string djvuFilePath = Path.Combine(documentDirectory, "sample.djvu");

// Ορίστε τη διαδρομή του αρχείου εξόδου HTML
string htmlOutputFile = Path.Combine(outputFileDirectory, "djvu-converted-to.html");
```
**2. Αρχικοποίηση μετατροπέα**
Δημιουργήστε μια παρουσία του `Converter` τάξη:
```csharp
using (var converter = new GroupDocs.Conversion.Converter(djvuFilePath))
{
    Console.WriteLine("Conversion process initialized.");
}
```
**3. Καθορίστε Επιλογές Μετατροπής**
Ρύθμιση επιλογών μετατροπής για μορφή HTML:
```csharp
var options = new WebConvertOptions();
```
**4. Εκτελέστε τη μετατροπή**
Εκτελέστε τη μετατροπή καλώντας το `Convert` μέθοδος:
```csharp
converter.Convert(htmlOutputFile, options);
Console.WriteLine("Conversion completed successfully.");
```
