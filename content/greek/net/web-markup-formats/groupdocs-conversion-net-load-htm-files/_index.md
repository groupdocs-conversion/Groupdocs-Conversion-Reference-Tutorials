## Οδηγός Εφαρμογής

### Φόρτωση αρχείου HTM πηγής

Μάθετε πώς να φορτώνετε και να προετοιμάζετε ένα έγγραφο HTML για μετατροπή χρησιμοποιώντας το GroupDocs.Conversion.

#### Βήμα 1: Ορισμός καταλόγου εγγράφων
Αρχικά, καθορίστε τον κατάλογο στον οποίο βρίσκονται τα έγγραφά σας:

```csharp
string documentDirectory = "/path/to/your/documents";
```

#### Βήμα 2: Αρχικοποίηση αντικειμένου μετατροπέα
Δημιουργήστε ένα `Converter` αντικείμενο για τη διαχείριση της διαδικασίας φόρτωσης αρχείων:

```csharp
using (Converter converter = new Converter(Path.Combine(documentDirectory, "sample.htm")))
{
    // Η διαμόρφωση και η εκτέλεση της μετατροπής θα πραγματοποιηθούν εδώ.
}
```

**Επεξήγηση παραμέτρων:**
- `documentDirectory`: Διαδρομή όπου βρίσκονται τα αρχεία προέλευσης.
- `Path.Combine(...)`: Συνδυάζει τη διαδρομή καταλόγου με το όνομα αρχείου για να δημιουργήσει μια πλήρη διαδρομή.

#### Βήμα 3: Διαμόρφωση επιλογών μετατροπής
Διαμορφώστε τις ρυθμίσεις μετατροπής ανάλογα με τις ανάγκες σας (π.χ., μορφή στόχου):

```csharp
var options = new PdfConvertOptions(); // Παράδειγμα μετατροπής σε PDF
```

**Βασικές επιλογές διαμόρφωσης:**
- `PdfConvertOptions`: Καθορίζει τις ρυθμίσεις για τη μετατροπή PDF.
