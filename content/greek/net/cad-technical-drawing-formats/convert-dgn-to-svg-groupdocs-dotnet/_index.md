## Οδηγός Εφαρμογής

### Μετατροπή αρχείου DGN σε μορφή SVG

Ακολουθήστε αυτά τα βήματα για απρόσκοπτη μετατροπή αρχείων DGN σε μορφή SVG χρησιμοποιώντας το GroupDocs.Conversion:

#### Βήμα 1: Ορισμός καταλόγου εξόδου και διαδρομής αρχείου
Καθορίστε πού θα αποθηκευτεί το αρχείο εξόδου σας:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = System.IO.Path.Combine(outputFolder, "dgn-converted-to.svg");
```

#### Βήμα 2: Φόρτωση του αρχείου DGN προέλευσης
Φορτώστε το αρχείο DGN πηγής σας από έναν καθορισμένο κατάλογο:

```csharp
using (var converter = new GroupDocs.Conversion.Converter("YOUR_DOCUMENT_DIRECTORY\sample.dgn"))
{
    // Η λογική μετατροπής θα προστεθεί εδώ.
}
```

#### Βήμα 3: Διαμόρφωση επιλογών μετατροπής
Ρυθμίστε τις επιλογές μετατροπής για να καθορίσετε το SVG ως τη μορφή προορισμού:

```csharp
var options = new PageDescriptionLanguageConvertOptions { Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Svg };
```

#### Βήμα 4: Εκτελέστε τη μετατροπή
Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο εξόδου:

```csharp
caller.Convert(outputFile, options);
```
