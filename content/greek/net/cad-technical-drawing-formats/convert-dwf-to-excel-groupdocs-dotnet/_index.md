## Οδηγός Εφαρμογής

### Λειτουργία 1: Φόρτωση αρχείου DWF προέλευσης

#### Επισκόπηση

Αυτή η λειτουργία δείχνει πώς να φορτώσετε ένα αρχείο DWF χρησιμοποιώντας τη βιβλιοθήκη GroupDocs.Conversion, προετοιμάζοντάς το για μετατροπή.

**Βήμα 3.1: Αρχικοποίηση του μετατροπέα**

Για να φορτώσετε το αρχείο DWF:
```csharp
using System;
using System.IO;
using GroupDocs.Conversion;

string sourceFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.dwf");

// Φορτώστε το αρχείο DWF στην κλάση Converter χρησιμοποιώντας μια εντολή 'using' για σωστή διάθεση πόρων.
using (var converter = new Converter(sourceFilePath))
{
    // Το αρχείο προέλευσης έχει πλέον φορτωθεί και είναι έτοιμο για διαδικασίες μετατροπής.
}
```

**Εξήγηση:** 
- `Converter` Αρχικοποιείται με τη διαδρομή του αρχείου DWF σας, φορτώνοντάς το για επόμενες λειτουργίες. Η χρήση μιας εντολής 'using' διασφαλίζει ότι οι πόροι διαχειρίζονται σωστά.

### Λειτουργία 2: Ορισμός επιλογών μετατροπής σε μορφή XLS

#### Επισκόπηση

Ρυθμίστε τις απαραίτητες επιλογές για τη μετατροπή ενός εγγράφου σε υπολογιστικό φύλλο Excel (XLS).

**Βήμα 3.2: Ρύθμιση παραμέτρων του SpreadsheetConvertOptions**

```csharp
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

string outputFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "dwf-converted-to.xls");

// Ρύθμιση επιλογών μετατροπής για τη μορφή XLS
SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Xls };
```

**Εξήγηση:** 
- `SpreadsheetConvertOptions` καθορίζει την επιθυμητή μορφή εξόδου, εδώ ορισμένη σε XLS.

### Χαρακτηριστικό 3: Εκτέλεση της διαδικασίας μετατροπής

#### Επισκόπηση

Εκτελέστε την πραγματική μετατροπή από DWF σε XLS και αποθηκεύστε το αποτέλεσμα.

**Βήμα 3.3: Μετατροπή και αποθήκευση**

```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

string sourceFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.dwf");
string outputFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "dwf-converted-to.xls");

// Φορτώστε το αρχείο DWF στην κλάση Converter χρησιμοποιώντας μια εντολή 'using' για σωστή διάθεση πόρων.
using (var converter = new Converter(sourceFilePath))
{
    // Ρύθμιση επιλογών μετατροπής για τη μορφή XLS
    SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Xls };
    
    // Εκτελέστε τη μετατροπή και αποθηκεύστε την έξοδο σε μια καθορισμένη διαδρομή
    converter.Convert(outputFilePath, options);
}
```

**Εξήγηση:**
- Ο `converter.Convert` Η μέθοδος εκτελεί τη διαδικασία μετατροπής χρησιμοποιώντας προκαθορισμένες επιλογές.
