- Απρόσκοπτη μετατροπή από CGM σε μορφή LaTeX (.tex)
- Ρύθμιση περιβάλλοντος και βελτιστοποίηση απόδοσης

```csharp
string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.cgm");
```

**Βήμα 2:** Δημιουργήστε μια παρουσία του `Converter` τάξη.
```csharp
using (var converter = new Converter(inputFilePath))
{
    // Το αρχείο CGM έχει πλέον φορτωθεί.
}
```
#### Εξήγηση
Ο `Converter` Το αντικείμενο διατηρεί το φορτωμένο αρχείο CGM, επιτρέποντας τις επόμενες εργασίες μετατροπής. Αυτή η ρύθμιση διασφαλίζει ότι εργάζεστε με το προβλεπόμενο έγγραφο πριν το επεξεργαστείτε περαιτέρω.

### Μετατροπή CGM σε μορφή TEX

#### Επισκόπηση
Αυτή η λειτουργία εστιάζει στη μετατροπή ενός φορτωμένου αρχείου CGM σε μορφή LaTeX (.tex) χρησιμοποιώντας το GroupDocs.Conversion.

#### Βήματα για την εφαρμογή
**Βήμα 1:** Ρυθμίστε τον κατάλογο εξόδου και τη διαδρομή αρχείου.
```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "cgm-converted-to.tex");
```

**Βήμα 2:** Φορτώστε το αρχείο CGM πηγής για μετατροπή.
```csharp
string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.cgm");

using (var converter = new Converter(inputFilePath))
{
    // Η ρύθμιση της μετατροπής ξεκινά εδώ.
}
```

**Βήμα 3:** Ρυθμίστε τις παραμέτρους των επιλογών μετατροπής για να καθορίσετε τη μορφή προορισμού (.tex).
```csharp
PageDescriptionLanguageConvertOptions options = new PageDescriptionLanguageConvertOptions
{
    Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Tex
};
```

**Βήμα 4:** Εκτελέστε τη μετατροπή και αποθηκεύστε το αποτέλεσμα.
```csharp
converter.Convert(outputFile, options);
```
#### Εξήγηση
Ο `PageDescriptionLanguageConvertOptions` Η κλάση χρησιμοποιείται για τον ορισμό συγκεκριμένων ρυθμίσεων μετατροπής. Εδώ, ορίζουμε τη μορφή σε `.tex`, επιτρέποντας τον μετασχηματισμό από CGM σε LaTeX.
