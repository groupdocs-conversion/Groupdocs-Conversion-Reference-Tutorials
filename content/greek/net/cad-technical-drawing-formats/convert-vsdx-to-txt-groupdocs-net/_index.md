### Χαρακτηριστικό: Μετατροπή αρχείου VSDX σε TXT

Δημιουργήστε μια παρουσία του `Converter` κλάση με τη διαδρομή του αρχείου πηγαίου κώδικα:

```csharp
string documentPath = "YOUR_DOCUMENT_DIRECTORY/sample.vsdx";
using (var converter = new Converter(documentPath))
{
    // Ο κώδικας μετατροπής θα τοποθετηθεί εδώ.
}
```
**Εξήγηση:** Ο `Converter` Το αντικείμενο αρχικοποιείται με τη διαδρομή προς το αρχείο VSDX σας, προετοιμάζοντάς το για επεξεργασία.

#### Βήμα 2: Καθορισμός επιλογών μετατροπής
Ορίστε επιλογές για μετατροπή σε μορφή TXT:

```csharp
var options = new WordProcessingConvertOptions { Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt };
```
**Εξήγηση:** `WordProcessingConvertOptions` επιτρέπει τον ορισμό της μορφής εξόδου ως TXT, καθορίζοντας πώς θα μετασχηματιστεί το περιεχόμενο VSDX σας.

#### Βήμα 3: Εκτελέστε τη μετατροπή
Εκτελέστε τη μετατροπή και αποθηκεύστε το αποτέλεσμα:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "vsdx-converted-to.txt");
converter.Convert(outputFile, options);
```
**Εξήγηση:** Ο `Convert` Η μέθοδος δέχεται τις καθορισμένες επιλογές σας και εξάγει ένα αρχείο TXT στην καθορισμένη θέση.
