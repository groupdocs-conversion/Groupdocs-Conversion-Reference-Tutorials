## Οδηγός Εφαρμογής
Τώρα, ας μετατρέψουμε ένα αρχείο CGM σε μορφή DOC χρησιμοποιώντας το GroupDocs.Conversion για .NET.

### Φόρτωση και μετατροπή αρχείου CGM σε μορφή DOC
#### Επισκόπηση
Σε αυτήν την ενότητα, φορτώνουμε ένα αρχείο CGM πηγής και το μετατρέπουμε σε μορφή εγγράφου του Microsoft Word (.doc).

#### Βήματα για την εφαρμογή:
**1. Ορισμός διαδρομών εισόδου και εξόδου**
Ρυθμίστε καταλόγους και καθορίστε διαδρομές για το αρχείο CGM εισόδου και το αρχείο DOC εξόδου:
```csharp
string documentDirectory = \@"YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = \@"YOUR_OUTPUT_DIRECTORY";

// Καθορίστε τη διαδρομή προς το αρχείο CGM προέλευσης και το αρχείο DOC εξόδου
string cgmFilePath = Path.Combine(documentDirectory, "sample.cgm");
string docOutputFile = Path.Combine(outputDirectory, "cgm-converted-to.doc");
```

**2. Φορτώστε το αρχείο CGM πηγής**
Χρησιμοποιήστε το GroupDocs.Conversion για να φορτώσετε το αρχείο CGM:
```csharp
using (var converter = new Converter(cgmFilePath))
{
    Console.WriteLine("CGM file loaded successfully.");
}
```
Αυτό το βήμα αρχικοποιεί ένα `Converter` παράδειγμα με την καθορισμένη διαδρομή αρχείου CGM.

**3. Ορίστε τις επιλογές μετατροπής για τη μορφή DOC**
Διαμορφώστε τις ρυθμίσεις μετατροπής ειδικά για τη μορφή επεξεργασίας κειμένου (.doc):
```csharp
// Ρύθμιση επιλογών μετατροπής για τη μορφή επεξεργασίας κειμένου (.doc)
var options = new WordProcessingConvertOptions 
{
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc
};
```
Εδώ, `WordProcessingConvertOptions` Καθορίζει ότι η μορφή εξόδου θα πρέπει να είναι DOC.

**4. Μετατροπή και αποθήκευση της εξόδου**
Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο που προκύπτει:
```csharp
converter.Convert(docOutputFile, options);
Console.WriteLine("Conversion completed successfully.");
```
Αυτή η μέθοδος μετατρέπει το αρχείο CGM σε έγγραφο DOC χρησιμοποιώντας καθορισμένες επιλογές και το αποθηκεύει στην επιθυμητή θέση.
