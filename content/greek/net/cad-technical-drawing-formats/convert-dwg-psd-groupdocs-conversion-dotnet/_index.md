### Ορισμός επιλογών μετατροπής για τη μορφή PSD

#### Επισκόπηση

Στη συνέχεια, διαμορφώστε τις επιλογές μετατροπής για να καθορίσετε ότι θέλετε να μετατρέψετε το έγγραφό σας σε μορφή PSD.

**Ρύθμιση παραμέτρων επιλογών μετατροπής**

```csharp
using GroupDocs.Conversion.Options.Convert;

// Ορίστε επιλογές μετατροπής για τη μορφή PSD
ImageConvertOptions psdOptions = new ImageConvertOptions 
{ 
    Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd  // Ορισμός μορφής εξόδου ως PSD
};
```

### Μετατροπή DWG σε PSD

#### Επισκόπηση

Αφού φορτωθεί το αρχείο προέλευσης και οριστούν οι επιλογές μετατροπής, μπορείτε πλέον να μετατρέψετε το αρχείο DWG σε PSD.

**Εκτέλεση μετατροπής**

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

// Ορίστε τη διαδρομή καταλόγου εξόδου για τα αρχεία που έχουν μετατραπεί
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");

Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);

// Εκτελέστε τη μετατροπή από DWG σε PSD
using (Converter converter = new Converter(sampleDwgPath))
{
    // Μετατροπή χρησιμοποιώντας καθορισμένες επιλογές και χειριστή ροής
    converter.Convert(getPageStream, psdOptions);
}
```
