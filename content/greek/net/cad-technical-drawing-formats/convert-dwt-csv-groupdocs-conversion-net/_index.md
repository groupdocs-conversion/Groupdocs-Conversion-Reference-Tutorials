### Βασική Αρχικοποίηση
Αρχικοποιήστε το GroupDocs.Conversion στο έργο C# σας με τον ακόλουθο κώδικα:
```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

namespace DWTToCSVConverter
{
class Program
{
    static void Main(string[] args)
    {
        // Ορίστε διαδρομές για αρχεία εισόδου και εξόδου
        string sourceFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.dwt");
        string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
        string outputFile = Path.Combine(outputFolder, "dwt-converted-to.csv");

        // Φορτώστε το αρχείο DWT χρησιμοποιώντας το GroupDocs.Conversion
        using (var converter = new Converter(sourceFilePath))
        {
            // Ορισμός επιλογών μετατροπής σε μορφή CSV
            var options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Csv };

            // Μετατροπή και αποθήκευση ως αρχείο CSV
            converter.Convert(outputFile, options);
        }
    }
}
```
Αυτό το απόσπασμα δείχνει πώς να φορτώσετε ένα αρχείο DWT και να το μετατρέψετε σε μορφή CSV χρησιμοποιώντας `SpreadsheetConvertOptions`.
