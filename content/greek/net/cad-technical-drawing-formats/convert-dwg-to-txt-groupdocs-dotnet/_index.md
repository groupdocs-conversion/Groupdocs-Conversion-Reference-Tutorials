### Ρύθμιση επιλογών μετατροπής για μορφή TXT
Η ρύθμιση παραμέτρων των επιλογών μετατροπής είναι απαραίτητη για τον ορισμό της μορφής και των ρυθμίσεων εξόδου.

#### Βήμα 1: Ορισμός του WordProcessingConvertOptions
Στήνω `WordProcessingConvertOptions` για να μετατρέψετε αρχεία σε μορφή TXT:
```csharp
using GroupDocs.Conversion.Options.Convert;

var options = new WordProcessingConvertOptions
{
    Format = FileTypes.WordProcessingFileType.Txt
};
```
Αυτή η ρύθμιση διασφαλίζει ότι η έξοδος είναι ένα απλό αρχείο κειμένου.

### Εκτέλεση μετατροπής και αποθήκευση της εξόδου
Το τελικό βήμα περιλαμβάνει τη μετατροπή του φορτωμένου αρχείου DWG σε μορφή TXT και την αποθήκευσή του στην επιθυμητή θέση.

#### Βήμα 1: Καθορισμός θέσης εξόδου
Ορίστε πού θέλετε να αποθηκευτεί το αρχείο που έχει μετατραπεί:
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = System.IO.Path.Combine(outputFolder, "dwg-converted-to.txt");
```

#### Βήμα 2: Εκτέλεση μετατροπής
Χρησιμοποιήστε το `Convert` μέθοδος για την εκτέλεση της μετατροπής και την αποθήκευση του αποτελέσματος:
```csharp
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY/sample.dwg"))
{
    var options = new WordProcessingConvertOptions { Format = FileTypes.WordProcessingFileType.Txt };
    
    // Αποθήκευση του αρχείου TXT που έχει μετατραπεί
    converter.Convert(outputFile, options);
}
```
