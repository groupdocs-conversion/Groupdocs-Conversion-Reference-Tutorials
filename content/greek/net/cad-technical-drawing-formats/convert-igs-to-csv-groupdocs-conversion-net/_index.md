#### Επισκόπηση
Αυτή η λειτουργία περιλαμβάνει τη φόρτωση του αρχείου IGS πηγής και τη μετατροπή του σε μορφή CSV, κάτι που είναι χρήσιμο για την ανάλυση ή τον χειρισμό δεδομένων μηχανικής σε εφαρμογές υπολογιστικών φύλλων.

#### Βήμα προς βήμα εφαρμογή

**1. Ρύθμιση διαδρομών καταλόγου**
Ορίστε διαδρομές τόσο για το αρχείο IGS εισόδου όσο και για το αρχείο CSV εξόδου:

```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY\";
string outputDir = @"YOUR_OUTPUT_DIRECTORY\";

if (!Directory.Exists(outputDir))
{
    Directory.CreateDirectory(outputDir);
}

string inputFile = Path.Combine(dataDir, "sample.igs");
string outputFile = Path.Combine(outputDir, "igs-converted-to.csv");
```

**2. Αρχικοποίηση του GroupDocs.Conversion**
Φορτώστε το αρχείο IGS χρησιμοποιώντας το GroupDocs.Conversion:

```csharp
using (var converter = new Converter(inputFile))
{
    // Ο κώδικας μετατροπής θα τοποθετηθεί εδώ.
}
```

**3. Ορίστε τις επιλογές μετατροπής CSV**
Καθορίστε επιλογές μετατροπής για μετατροπή σε μορφή CSV:

```csharp
var options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Csv };
```

**4. Εκτελέστε τη μετατροπή**
Εκτελέστε τη διαδικασία μετατροπής, μετατρέποντας το αρχείο IGS σε μορφή CSV:

```csharp
converter.Convert(outputFile, options);
```
