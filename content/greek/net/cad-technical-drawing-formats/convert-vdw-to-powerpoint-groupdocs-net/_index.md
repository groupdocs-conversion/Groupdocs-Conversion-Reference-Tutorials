## Οδηγός Εφαρμογής
### Φόρτωση και μετατροπή VDW σε PPTX
Ας μετατρέψουμε ένα αρχείο VDW σε μορφή PowerPoint (PPTX).

#### Επισκόπηση
Μετατρέψτε οποιοδήποτε αρχείο VDW σε επεξεργάσιμη μορφή παρουσίασης για το Microsoft PowerPoint.

#### Βήμα προς βήμα εφαρμογή
**1. Προετοιμάστε το περιβάλλον σας**
Βεβαιωθείτε ότι οι διαδρομές εγγράφων έχουν οριστεί σωστά:
```csharp
public static class Constants
{
    public static string YOUR_DOCUMENT_DIRECTORY = "YOUR_DOCUMENT_DIRECTORY";
    public static string SAMPLE_VDW = Path.Combine(YOUR_DOCUMENT_DIRECTORY, "sample.vdw");
}
```
**2. Αρχικοποίηση του μετατροπέα**
Φορτώστε το αρχείο VDW:
```csharp
using (var converter = new Converter(Constants.SAMPLE_VDW))
{
    // Τα βήματα μετατροπής ακολουθούν...
}
```
**3. Ρύθμιση επιλογών μετατροπής**
Ορισμός παραμέτρων μετατροπής:
```csharp
var options = new PresentationConvertOptions();
```
**4. Ορισμός διαδρομής εξόδου και μετατροπή**
Καθορίστε την τοποθεσία εξόδου και μετατρέψτε:
```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "vdw-converted-to.pptx");
converter.Convert(outputFile, options);
```
Αυτή η διαδικασία μετατρέπει αποτελεσματικά το αρχείο VDW σε μορφή PPTX.
