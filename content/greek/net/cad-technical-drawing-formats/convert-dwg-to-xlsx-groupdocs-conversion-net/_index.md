### Μετατροπή DWG σε XLSX
#### Επισκόπηση
Αφού φορτώσετε το αρχείο DWG, μπορείτε να προχωρήσετε στη μετατροπή του σε μορφή XLSX.

#### Βήμα 2: Δημιουργία επιλογών μετατροπής
```csharp
using System.IO;
using GroupDocs.Conversion.Options.Convert;

// Καθορίστε τον κατάλογο εξόδου και τη διαδρομή αρχείου για το XLSX που έχει μετατραπεί
string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "dwg-converted-to.xlsx");
var options = new SpreadsheetConvertOptions();
// Προσαρμόστε τις «επιλογές» όπως απαιτείται (π.χ., όνομα φύλλου)
```
Ο `SpreadsheetConvertOptions` Η κλάση σάς επιτρέπει να προσαρμόσετε τη διαδικασία μετατροπής ανάλογα με τις ανάγκες σας.

#### Βήμα 3: Εκτέλεση μετατροπής και αποθήκευση
```csharp
using(var converter = new Converter(@"YOUR_DOCUMENT_DIRECTORY\sample.dwg"))
{
    // Μετατρέψτε το αρχείο DWG σε μορφή XLSX
    converter.Convert(outputFile, options);
}

Console.WriteLine("Conversion complete. Check output folder for result.");
```
Αυτό το βήμα εκτελεί τη μετατροπή και αποθηκεύει το αρχείο XLSX που προκύπτει στον καθορισμένο κατάλογο.
