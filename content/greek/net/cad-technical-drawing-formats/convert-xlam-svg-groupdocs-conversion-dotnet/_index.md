Η λειτουργία μετατρέπει αρχεία πρόσθετων με δυνατότητα μακροεντολών του Microsoft Excel (.xlam) σε κλιμακώσιμα διανυσματικά γραφικά (SVG), επιτρέποντας υψηλής ποιότητας, κλιμακώσιμες απεικονίσεις.

#### Βήμα 1: Ορίστε τις διαδρομές αρχείων

Ορίστε διαδρομές για το αρχείο XLAM πηγής και τον κατάλογο εξόδου. Βεβαιωθείτε ότι ο κατάλογος εξόδου υπάρχει:

```csharp
string sourceFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.xlam");
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", "ConvertedFiles");

if (!Directory.Exists(outputFolder)) 
{
    Directory.CreateDirectory(outputFolder);
}
```

#### Βήμα 2: Αρχικοποίηση μετατροπέα

Φορτώστε το αρχείο XLAM χρησιμοποιώντας το GroupDocs.Conversion:

```csharp
using (var converter = new Converter(sourceFilePath))
{
    // Η λογική μετατροπής θα εφαρμοστεί εδώ.

```

#### Βήμα 3: Ρύθμιση παραμέτρων επιλογών SVG

Ορίστε τις επιλογές μετατροπής για να στοχεύσετε συγκεκριμένα τη μορφή SVG:

```csharp
var options = new PageDescriptionLanguageConvertOptions { Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Svg };
```

#### Βήμα 4: Εκτελέστε τη μετατροπή

Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο εξόδου:

```csharp

converter.Convert(outputFile, options);
```
