## Οδηγός Εφαρμογής

### Βήμα 1: Ορίστε τη διαδρομή καταλόγου εξόδου

Καθορίστε πού θα πρέπει να αποθηκευτεί το αρχείο DOCX που έχετε μετατρέψει:
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
```
Αυτή η διαδρομή είναι κρίσιμη, καθώς καθορίζει πού θα βρίσκεται το αρχείο που προκύπτει.

### Βήμα 2: Κατασκευάστε την πλήρη διαδρομή αρχείου εξόδου για το DOCX

Δημιουργήστε μια πλήρη διαδρομή αρχείου για να βεβαιωθείτε ότι η έξοδος έχει σωστό όνομα και αποθηκεύεται:
```csharp
string outputFile = Path.Combine(outputFolder, "igs-converted-to.docx");
```
Αυτή η προσέγγιση αποτρέπει τυχόν διενέξεις ονοματοδοσίας ή σφάλματα που σχετίζονται με τη δομή καταλόγου.

### Βήμα 3: Φόρτωση του αρχείου IGS πηγής χρησιμοποιώντας ένα αντικείμενο μετατροπέα

Φορτώστε το αρχείο προέλευσης χρησιμοποιώντας το GroupDocs.Conversion `Converter` κλάση. Αυτό περιλαμβάνει τη δημιουργία μιας παρουσίας του μετατροπέα:
```csharp
using (var converter = new GroupDocs.Conversion.Converter("YOUR_DOCUMENT_DIRECTORY/your-source-file.igs"))
{
    // Περαιτέρω βήματα θα ακολουθήσουν εδώ
}
```
Αυτή η ρύθμιση διαβάζει το αρχείο IGS σας, προετοιμάζοντάς το για μετατροπή.

### Βήμα 4: Ορισμός επιλογών μετατροπής για τη μορφή επεξεργασίας κειμένου

Ορίστε τις συγκεκριμένες επιλογές για τη μετατροπή σε μορφή DOCX:
```csharp
var options = new WordProcessingConvertOptions();
```
Αυτές οι επιλογές σάς επιτρέπουν να προσαρμόσετε τον τρόπο με τον οποίο το περιεχόμενο IGS μετατρέπεται σε μορφή DOCX.

### Βήμα 5: Μετατροπή και αποθήκευση του αρχείου IGS ως DOCX

Εκτελέστε τη διαδικασία μετατροπής και αποθηκεύστε το αρχείο σας:
```csharp
converter.Convert(outputFile, options);
```
Αυτό το βήμα εκτελεί την πραγματική μετατροπή και εγγράφει την έξοδο στην καθορισμένη διαδρομή.
