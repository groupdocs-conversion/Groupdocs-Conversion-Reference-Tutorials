## Οδηγός Εφαρμογής
Θα αναλύσουμε τη διαδικασία μετατροπής σε διαχειρίσιμα βήματα για λόγους σαφήνειας.

### Φόρτωση αρχείου DXF πηγής
**Επισκόπηση:**
Η φόρτωση ενός αρχείου DXF είναι το πρώτο βήμα στη διαδικασία μετατροπής. Αυτή η λειτουργία μας επιτρέπει να προετοιμάσουμε το έγγραφο προέλευσης για μετασχηματισμό.

#### Βήμα προς βήμα εφαρμογή:
1. **Ορισμός διαδρομής:**
   Ορίστε τη διαδρομή προς το αρχείο DXF.
   ```csharp
   string sourceFilePath = Path.Combine(@"YOUR_DOCUMENT_DIRECTORY", "sample.dxf");
   ```
2. **Φόρτωση αρχείου:**
   Χρησιμοποιήστε το `Converter` κλάση για να φορτώσετε το αρχείο σας.
   ```csharp
   using (Converter converter = new Converter(sourceFilePath))
   {
       // Το αρχείο έχει πλέον φορτωθεί και είναι έτοιμο για μετατροπή.
   }
   ```

### Ορισμός επιλογών μετατροπής για μορφή JPG
**Επισκόπηση:**
Η ρύθμιση παραμέτρων των επιλογών μετατροπής προσαρμόζει τη μορφή εξόδου, διασφαλίζοντας ότι τα αρχεία DXF σας μετατρέπονται σε εικόνες JPG υψηλής ποιότητας.

#### Βήμα προς βήμα εφαρμογή:
1. **Δημιουργία επιλογών μετατροπής:**
   Δημιουργία στιγμιαίας εικόνας `ImageConvertOptions` και καθορίστε τη μορφή προορισμού ως JPG.
   ```csharp
   using GroupDocs.Conversion.Options.Convert;
   ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Jpg };
   ```

### Μετατροπή DXF σε μορφή JPG
**Επισκόπηση:**
Αυτή η λειτουργία ενορχηστρώνει τη διαδικασία μετατροπής, χρησιμοποιώντας ρυθμίσεις και διαδρομές που έχουν καθοριστεί προηγουμένως.

#### Βήμα προς βήμα εφαρμογή:
1. **Ορισμός λεπτομερειών εξόδου:**
   Ρυθμίστε τον κατάλογο εξόδου και το πρότυπο αρχείου.
   ```csharp
   string outputFolder = Path.Combine(@"YOUR_OUTPUT_DIRECTORY");
   string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");

   Func<SavePageContext, Stream> getPageStream = savePageContext =>
       new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
   ```
2. **Εκτέλεση μετατροπής:**
   Μετατρέψτε το αρχείο DXF σε JPG χρησιμοποιώντας το `Converter` αντικείμενο.
   ```csharp
   using (Converter converter = new Converter(sourceFilePath))
   {
       converter.Convert(getPageStream, options);
   }
   ```
