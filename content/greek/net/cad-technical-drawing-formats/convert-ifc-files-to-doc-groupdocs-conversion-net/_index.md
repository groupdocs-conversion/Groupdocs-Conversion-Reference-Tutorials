**Βήμα 1:** Ορίστε τη διαδρομή του καταλόγου εγγράφων σας όπου αποθηκεύονται τα αρχεία IFC πηγής σας. Προσαρμόστε `"YOUR_DOCUMENT_DIRECTORY"` για να υποδείξει τον πραγματικό σας φάκελο.

```csharp
private static string GetDocumentDirectoryPath()
{
    return @"YOUR_DOCUMENT_DIRECTORY";
}
```

**Βήμα 2:** Φορτώστε το αρχείο IFC χρησιμοποιώντας το GroupDocs.Conversion:

```csharp
using (var converter = new Converter(ifcFilePath))
{
    // Το αρχείο IFC έχει πλέον φορτωθεί και είναι έτοιμο για μετατροπή ή άλλη επεξεργασία.
}
```

### Μετατροπή IFC σε μορφή DOC
#### Επισκόπηση
Μόλις φορτωθεί το αρχείο IFC, μετατρέψτε το σε μορφή εγγράφου Word. Αυτό βελτιώνει τη συνεργασία, καθιστώντας τα σύνθετα μοντέλα εύκολα κοινοποιήσιμα.

**Βήμα 1:** Ορίστε διαδρομές εξόδου για το μετατρεπόμενο έγγραφο:

```csharp
private static string GetOutputDirectoryPath()
{
    return Path.Combine(Directory.GetCurrentDirectory(), "YOUR_OUTPUT_DIRECTORY");
}
```

**Βήμα 2:** Ρυθμίστε τις επιλογές μετατροπής και εκτελέστε τη μετατροπή:

```csharp
using (var converter = new Converter(ifcFilePath))
{
    var convertOptions = new WordProcessingConvertOptions();
    
    string outputFile = Path.Combine(outputFolder, "ifc-converted-to.doc");
    converter.Convert(outputFile, convertOptions);
}
```
