# Μαθήματα μορφών CAD και τεχνικών σχεδίων για το GroupDocs.Conversion .NET

Σε αυτό το κέντρο θα βρείτε οδηγούς βήμα‑βήμα για **μετατροπή CAD σε PDF** χρησιμοποιώντας το GroupDocs.Conversion για .NET, καλύπτοντας προσαρμοσμένες διαστάσεις σελίδας, επίπεδωση 3‑Δ μοντέλων, διατήρηση στρωμάτων και διαχείριση μεγάλων σχεδίων. Είτε δημιουργείτε μια υπηρεσία web που αποδίδει τεχνικά σχήματα είτε ένα επιτραπέζιο εργαλείο που επεξεργάζεται χιλιάδες σχέδια, αυτά τα μαθήματα σας δείχνουν πώς να έχετε αξιόπιστα, έτοιμα για παραγωγή αποτελέσματα ενώ διατηρείτε τη χρήση μνήμης χαμηλή.

Το GroupDocs.Conversion για .NET είναι μια βιβλιοθήκη .NET που μετατρέπει πάνω από 150 μορφές—συμπεριλαμβανομένων των σχεδίων CAD—σε PDF, εικόνες και αρχεία Office. Υποστηρίζει τόσο το .NET Framework όσο και το .NET Core, λειτουργεί σε Windows, Linux και macOS, και δεν απαιτεί εξωτερικό λογισμικό CAD να εγκατασταθεί στον διακομιστή.

**Ποσοτικοποιημένη δήλωση:** Το GroupDocs.Conversion μπορεί να επεξεργαστεί αρχεία CAD έως 500 MB σε μέγεθος και να μετατρέπει σχέδια με εκατοντάδες σελίδες σε λιγότερο από 30 δευτερόλεπτα σε μια τυπική VM με 4 πυρήνες, χάρη στην αρχιτεκτονική ροής του που αποφεύγει τη φόρτωση ολόκληρου του αρχείου προέλευσης στη μνήμη.
