### Βασική Αρχικοποίηση και Ρύθμιση
Για να αρχικοποιήσετε το GroupDocs.Conversion στην εφαρμογή C# που χρησιμοποιείτε:

```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

namespace VdwToXlsConverter
{
    class Program
    {
        static void Main(string[] args)
        {
            string documentPath = @"YOUR_DOCUMENT_DIRECTORY\sample.vdw"; // Διαδρομή προς το αρχείο VDW
            string outputPath = @"YOUR_OUTPUT_DIRECTORY\output.xls"; // Διαδρομή εξόδου για το αρχείο XLS

            using (Converter converter = new Converter(documentPath))
            {
                SpreadsheetConvertOptions options = new SpreadsheetConvertOptions();
                
                converter.Convert(outputPath, options);
            }
        }
    }
}
```
Αυτός ο κώδικας δημιουργεί ένα βασικό περιβάλλον μετατροπής. `Converter` Η κλάση αρχικοποιείται με τη διαδρομή του αρχείου VDW και η μετατροπή εκτελείται χρησιμοποιώντας `SpreadsheetConvertOptions`.
