## Οδηγός Εφαρμογής

### Φόρτωση και μετατροπή STL σε DOCX

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να φορτώνετε απρόσκοπτα ένα αρχείο STL και να το μετατρέπετε σε έγγραφο DOCX, διευκολύνοντας την επεξεργασία ή την κοινή χρήση περιεχομένου κειμένου που προέρχεται από τρισδιάστατα μοντέλα.

#### Βήμα 1: Ορισμός διαδρομών αρχείων
Βεβαιωθείτε ότι οι διαδρομές του καταλόγου σας έχουν ρυθμιστεί σωστά. Αντικαταστήστε τα placeholders με τις πραγματικές διαδρομές:
```csharp
string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY";

if (!Directory.Exists(documentDirectory))
    Directory.CreateDirectory(documentDirectory);
if (!Directory.Exists(outputDirectory))
    Directory.CreateDirectory(outputDirectory);

string inputFilePath = Path.Combine(documentDirectory, "sample.stl");
string outputFilePath = Path.Combine(outputDirectory, "stl-converted-to.docx");
```

#### Βήμα 2: Αρχικοποίηση μετατροπέα και ορισμός επιλογών
Φορτώστε το αρχείο STL χρησιμοποιώντας το `Converter` κλάση και διαμορφώστε τις ρυθμίσεις μετατροπής:
```csharp
using (var converter = new Converter(inputFilePath))
{
    var options = new WordProcessingConvertOptions(); // Ρύθμιση παραμέτρων επιλογών μετατροπής DOCX
    
    converter.Convert(outputFilePath, options);
}
```
Αυτό το απόσπασμα κώδικα δείχνει τη φόρτωση ενός αρχείου STL και τη μετατροπή του χρησιμοποιώντας το `Converter` τάξη. Το `WordProcessingConvertOptions` επιτρέπει την προσαρμογή του αρχείου DOCX εξόδου σας.

#### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι οι διαδρομές έχουν οριστεί σωστά για να αποφύγετε σφάλματα "δεν βρέθηκε αρχείο".
- Βεβαιωθείτε ότι έχετε δικαιώματα ανάγνωσης/εγγραφής σε συγκεκριμένους καταλόγους.
