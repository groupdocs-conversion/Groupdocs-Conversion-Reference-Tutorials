## Οδηγός Εφαρμογής

### Φόρτωση και μετατροπή του αρχείου DWF
#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να φορτώσετε ένα αρχείο DWF και να το μετατρέψετε εύκολα σε μορφή κειμένου. Ας αναλύσουμε κάθε βήμα αυτής της διαδικασίας.

#### Βήμα 1: Ορισμός διαδρομών
Αρχικά, καθορίστε πού βρίσκονται τα αρχεία DWF εισόδου και πού θέλετε να αποθηκευτούν τα αρχεία TXT εξόδου:
```csharp
using System.IO;

string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY"; // Βεβαιωθείτε ότι αυτοί οι κατάλογοι υπάρχουν στο σύστημα αρχείων σας
```
#### Βήμα 2: Φόρτωση του αρχείου DWF προέλευσης
Στη συνέχεια, φορτώστε το αρχείο DWF που θέλετε να μετατρέψετε χρησιμοποιώντας το `Converter` τάξη:
```csharp
string inputFile = Path.Combine(documentDirectory, "sample.dwf"); // Αντικαταστήστε το 'sample.dwf' με το πραγματικό σας όνομα αρχείου
string outputFile = Path.Combine(outputDirectory, "dwf-converted-to.txt");
```
#### Βήμα 3: Διαμόρφωση επιλογών μετατροπής
Ορίστε επιλογές μετατροπής για τη μορφή TXT. Αυτό περιλαμβάνει τον καθορισμό ότι η έξοδος θα πρέπει να είναι ένα αρχείο κειμένου:
```csharp
using GroupDocs.Conversion.Options.Convert;

WordProcessingConvertOptions options = new WordProcessingConvertOptions
{
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt
};
```
#### Βήμα 4: Εκτελέστε τη μετατροπή
Τέλος, μετατρέψτε το αρχείο DWF και αποθηκεύστε το ως TXT:
```csharp
using (var converter = new Converter(inputFile))
{
    converter.Convert(outputFile, options);
}
```
