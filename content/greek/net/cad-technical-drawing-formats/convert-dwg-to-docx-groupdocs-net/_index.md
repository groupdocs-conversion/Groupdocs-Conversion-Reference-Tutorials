## Οδηγός Εφαρμογής

Τώρα που έχετε ρυθμίσει, ας προχωρήσουμε στη διαδικασία μετατροπής.

### Φόρτωση και μετατροπή DWG σε DOCX

Αυτή η λειτουργία σάς επιτρέπει να φορτώσετε ένα αρχείο DWG και να το μετατρέψετε σε μορφή DOCX αποτελεσματικά.

#### Βήμα 1: Ορισμός καταλόγου εξόδου

Αρχικά, καθορίστε πού θα αποθηκευτούν τα αρχεία που έχετε μετατρέψει:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
```

#### Βήμα 2: Κατασκευάστε την πλήρη διαδρομή για το αρχείο εξόδου

Δημιουργήστε μια παρουσία του `File` για να αναπαραστήσει τη διαδρομή του αρχείου DOCX:

```csharp
using System.IO;

File outputFile = new File(Path.Combine(outputFolder, "dwg-converted-to.docx"));
```

#### Βήμα 3: Φόρτωση του αρχείου DWG

Χρησιμοποιήστε το `Converter` κλάση για να φορτώσετε το αρχείο DWG πηγής σας:

```csharp
using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_DWG")) {
    // Ο κώδικας μετατροπής θα τοποθετηθεί εδώ
}
```

#### Βήμα 4: Ορισμός επιλογών μετατροπής

Ρύθμιση παραμέτρων επιλογών για μετατροπή σε επεξεργασία Word:

```csharp
WordProcessingConvertOptions options = new WordProcessingConvertOptions();
```

#### Βήμα 5: Εκτελέστε τη μετατροπή

Τέλος, εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο DOCX:

```csharp
converter.Convert(outputFile.FullName, options);
Console.WriteLine("Conversion completed successfully.");
```
