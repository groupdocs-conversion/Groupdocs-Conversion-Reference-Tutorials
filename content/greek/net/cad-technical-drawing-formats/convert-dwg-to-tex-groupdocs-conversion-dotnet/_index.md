### Βασική Αρχικοποίηση και Ρύθμιση

Αρχικά, ας ρυθμίσουμε το έργο μας. Ακολουθεί ένα απλό απόσπασμα κώδικα που αρχικοποιεί τη βιβλιοθήκη GroupDocs.Conversion:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main(string[] args)
    {
        // Ορισμός καταλόγου εξόδου
        string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");

        // Βεβαιωθείτε ότι ο κατάλογος υπάρχει
        Directory.CreateDirectory(outputFolder);

        // Αρχικοποίηση χειριστή μετατροπής
        using (var converter = new Converter("input.dwg"))
        {
            var options = new MarkupConvertOptions { Format = MarkupFileType.Tex };
            converter.Convert(Path.Combine(outputFolder, "output.tex"), options);
        }
    }
}
```

Αυτός ο κώδικας παρουσιάζει τα βασικά στοιχεία της φόρτωσης ενός αρχείου DWG και της μετατροπής του σε μορφή TEX.
