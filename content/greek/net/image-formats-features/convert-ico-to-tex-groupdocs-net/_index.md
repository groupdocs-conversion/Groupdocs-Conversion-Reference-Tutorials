### Μετατροπή ICO σε TEX

#### Επισκόπηση
Στη συνέχεια, θα μετατρέψουμε το φορτωμένο αρχείο ICO σε μορφή TEX. Αυτό περιλαμβάνει τη ρύθμιση επιλογών μετατροπής και την εκτέλεση της μετατροπής.

**Βήμα 1: Ορισμός διαδρομής εξόδου**

Καθορίστε πού θέλετε να αποθηκεύσετε το αρχείο TEX που έχετε μετατρέψει:

```csharp
using System.IO;

const string OutputDirectory = "YOUR_OUTPUT_DIRECTORY/";
const string ConvertedTexFileName = "ico-converted-to.tex";

string outputFolder = OutputDirectory;
string outputFile = Path.Combine(outputFolder, ConvertedTexFileName);
```

**Βήμα 2: Ορισμός επιλογών μετατροπής**

Ρυθμίστε τις επιλογές μετατροπής για τη μορφή TEX:

```csharp
using GroupDocs.Conversion.Options.Convert;
using GroupDocs.Conversion.FileTypes;

var options = new PageDescriptionLanguageConvertOptions { Format = PageDescriptionLanguageFileType.Tex };
```

**Βήμα 3: Εκτελέστε τη μετατροπή**

Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο εξόδου:

```csharp
converter.Convert(outputFile, options);
```
