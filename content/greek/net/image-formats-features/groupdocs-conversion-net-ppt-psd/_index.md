## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα αναλύσουμε τη διαδικασία μετατροπής σε διαχειρίσιμα βήματα.

### Φόρτωση και μετατροπή PPT σε PSD

#### Επισκόπηση:
Αυτή η λειτουργία σάς επιτρέπει να φορτώσετε ένα αρχείο PowerPoint και να μετατρέψετε κάθε διαφάνεια σε ξεχωριστό έγγραφο Photoshop.

#### Βήμα προς βήμα εφαρμογή:

**Προετοιμασία διαδρομών αρχείων:**
Ορίστε τη διαδρομή του αρχείου προέλευσης και τον κατάλογο εξόδου. Βεβαιωθείτε ότι οι κατάλογοι υπάρχουν για να αποτρέψετε σφάλματα χρόνου εκτέλεσης.

```csharp
string sourceFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.ppt");
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", "ConvertedPSD");

// Βεβαιωθείτε ότι ο κατάλογος εξόδου υπάρχει
Directory.CreateDirectory(outputFolder);
```

**Δημιουργήστε μια ροή για αρχεία εξόδου:**
Ρυθμίστε μια συνάρτηση για τη δημιουργία ροών όπου θα αποθηκεύεται κάθε αρχείο PSD.

```csharp
Func<SavePageContext, Stream> getPageStream = (savePageContext) => 
    new FileStream(Path.Combine(outputFolder, $"converted-page-{savePageContext.Page}.psd"), FileMode.Create);
```

**Ρύθμιση παραμέτρων επιλογών μετατροπής:**
Καθορίστε τις επιλογές μετατροπής για να διασφαλίσετε ότι τα αρχεία αποθηκεύονται ως PSD.

```csharp
using GroupDocs.Conversion.Options.Convert;

ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Psd };
```

**Εκτελέστε τη μετατροπή:**
Φορτώστε το αρχείο PPT και εκτελέστε τη μετατροπή χρησιμοποιώντας τις καθορισμένες ρυθμίσεις.

```csharp
using (Converter converter = new Converter(sourceFilePath))
{
    converter.Convert(getPageStream, options);
}
```

#### Επεξήγηση παραμέτρων:
- `sourceFilePath`: Διαδρομή προς το αρχείο PPT εισόδου σας.
- `outputFolder`: Κατάλογος όπου θα αποθηκευτούν τα αρχεία PSD που έχουν μετατραπεί.
- `getPageStream`: Συνάρτηση που καθορίζει τον τρόπο χειρισμού των ροών εξόδου.
- `options`: Ρύθμιση παραμέτρων για τη μετατροπή κάθε διαφάνειας σε PSD.

#### Συμβουλές αντιμετώπισης προβλημάτων:
- Βεβαιωθείτε ότι όλες οι διαδρομές και οι κατάλογοι έχουν καθοριστεί σωστά.
- Ελέγξτε τις εξαρτήσεις του GroupDocs.Conversion για να αποφύγετε σφάλματα που λείπουν από τη βιβλιοθήκη.
