## Οδηγός Εφαρμογής

Ας εξερευνήσουμε πώς να εφαρμόσουμε κάθε χαρακτηριστικό.

### Φόρτωση και μετατροπή DWF σε PSD

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να φορτώσετε ένα αρχείο DWF και να το μετατρέψετε σε μορφή PSD, η οποία χρησιμοποιείται ευρέως σε εφαρμογές γραφιστικής όπως το Adobe Photoshop.

**Βήμα 1: Ορισμός διαδρομών αρχείων**

Αρχικά, ορίστε τη διαδρομή του εγγράφου προέλευσης και τον φάκελο εξόδου:

```csharp
string documentPath = @"YOUR_DOCUMENT_DIRECTORY\sample.dwf";
string outputFolder = @"YOUR_OUTPUT_DIRECTORY\";
```

**Βήμα 2: Δημιουργία προτύπου αρχείου εξόδου**

Ορίστε ένα πρότυπο για την ονομασία των αρχείων που έχουν μετατραπεί:

```csharp
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");
```

**Βήμα 3: Διαχείριση ροών σελίδων**

Δημιουργήστε μια συνάρτηση για τη διαχείριση ροών αρχείων κατά τη μετατροπή:

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

**Βήμα 4: Ορισμός επιλογών μετατροπής και εκτέλεση**

Διαμορφώστε τις ρυθμίσεις μετατροπής για τη μορφή PSD και εκτελέστε τη μετατροπή:

```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd };

// Εκτελέστε τη μετατροπή σε PSD
converter.Convert(getPageStream, options);
```

### Αποθήκευση εξόδου μετατροπής σε συγκεκριμένο κατάλογο

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να καθορίσετε έναν κατάλογο εξόδου όπου θα αποθηκευτούν τα αρχεία που έχετε μετατρέψει.

**Βήμα 1: Ορισμός καταλόγων**

Καθορίστε τους καταλόγους εγγράφων και εξόδου:

```csharp
string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY\";
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY\";
```

**Βήμα 2: Χρήση προτύπου αρχείου εξόδου**

Κατασκευάστε τη διαδρομή για το πρότυπο αρχείου εξόδου για να βεβαιωθείτε ότι τα αρχεία αποθηκεύονται σε μια καθορισμένη τοποθεσία:

```csharp
string outputFileTemplate = Path.Combine(outputDirectory, "converted-page-{0}.psd");
```
