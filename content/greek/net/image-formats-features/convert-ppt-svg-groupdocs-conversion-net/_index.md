## Οδηγός Εφαρμογής

### Μετατροπή PPT σε SVG

Αυτή η ενότητα θα σας καθοδηγήσει στη μετατροπή ενός αρχείου PowerPoint σε μορφή SVG.

#### Βήμα 1: Ορισμός διαδρομών

Καθορίστε τους καταλόγους εισόδου και εξόδου για τα αρχεία σας:

```csharp
string documentPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.ppt");
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "ppt-converted-to.svg");

// Βεβαιωθείτε ότι ο κατάλογος εξόδου υπάρχει
Directory.CreateDirectory(outputFolder);
```

**Εξήγηση:** Ορίζουμε διαδρομές για το αρχείο προέλευσης και πού θέλετε να αποθηκεύσετε το μετατρεπόμενο SVG. `Directory.CreateDirectory` Η μέθοδος διασφαλίζει ότι ο φάκελος εξόδου είναι διαθέσιμος.

#### Βήμα 2: Φόρτωση του αρχείου PPT προέλευσης

```csharp
using (var converter = new GroupDocs.Conversion.Converter(documentPath))
{
    // Ρύθμιση παραμέτρων επιλογών μετατροπής για τη μορφή SVG
    PageDescriptionLanguageConvertOptions options = new PageDescriptionLanguageConvertOptions { Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Svg };
    
    // Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο εξόδου
    converter.Convert(outputFile, options);
}
```

**Εξήγηση:** Εδώ, φορτώνουμε το αρχείο PPT χρησιμοποιώντας το `Converter` κλάση. Ορίζουμε επιλογές μετατροπής ειδικά για τη μορφή SVG και εκτελούμε τη μετατροπή.
