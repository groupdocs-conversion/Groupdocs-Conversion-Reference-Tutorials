## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Φόρτωση αρχείου πηγής

**Επισκόπηση:** Ξεκινήστε φορτώνοντας ένα αρχείο Visio (.vsd) προέλευσης στο GroupDocs.Conversion `Converter` αντικείμενο. Αυτό είναι το πρώτο σας βήμα προς τη μετατροπή αρχείων.

#### Βήμα 1: Φόρτωση αρχείου VSD
```csharp
using System;
using GroupDocs.Conversion;

string sourceFilePath = "YOUR_DOCUMENT_DIRECTORY/SAMPLE_VSD";

// Αρχικοποιήστε τον Μετατροπέα με τη διαδρομή προς το αρχείο VSD σας
using (Converter converter = new Converter(sourceFilePath))
{
    Console.WriteLine("VSD file loaded successfully.");
}
```
**Εξήγηση:** Ο `Converter` Η κλάση χειρίζεται ολόκληρη τη διαδικασία μετατροπής και εδώ φορτώνουμε ένα συγκεκριμένο αρχείο VSD. Βεβαιωθείτε ότι η διαδρομή του αρχείου σας είναι σωστή.

### Χαρακτηριστικό: Ορισμός επιλογών μετατροπής

**Επισκόπηση:** Ορίστε τις παραμέτρους μετατροπής για να βεβαιωθείτε ότι η έξοδος είναι σε μορφή PSD.

#### Βήμα 1: Ορισμός επιλογών μετατροπής
```csharp
using System;
using GroupDocs.Conversion.Options.Convert;

// Δημιουργία ImageConvertOptions για μορφή PSD
ImageConvertOptions options = new ImageConvertOptions();
options.Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd; // Ορισμός μορφής στόχου σε PSD

Console.WriteLine("Conversion options set to PSD.");
```
**Εξήγηση:** Ο `ImageConvertOptions` Η κλάση σάς επιτρέπει να καθορίσετε τη μορφή εξόδου. Εδώ, τη διαμορφώνουμε για PSD.

### Χαρακτηριστικό: Μετατροπή σε μορφή στόχου

**Επισκόπηση:** Τέλος, εκτελέστε τη μετατροπή και αποθηκεύστε κάθε σελίδα του αρχείου VSD ως ξεχωριστό αρχείο PSD.

#### Βήμα 2: Εκτέλεση μετατροπής
```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");

// Συνάρτηση για τη δημιουργία ροών αρχείων για κάθε σελίδα που μετατρέπεται
Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);

using (Converter converter = new Converter(sourceFilePath))
{
    // Μετατροπή VSD σε PSD χρησιμοποιώντας τις καθορισμένες επιλογές και το πρότυπο εξόδου
    converter.Convert(getPageStream, options);
}

Console.WriteLine("Conversion completed successfully.");
```
**Εξήγηση:** Ο `Convert` Η μέθοδος επεξεργάζεται κάθε σελίδα του αρχείου VSD σας σε ξεχωριστό αρχείο PSD, χρησιμοποιώντας τις καθορισμένες επιλογές.
