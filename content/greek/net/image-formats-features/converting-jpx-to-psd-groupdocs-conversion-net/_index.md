## Οδηγός Εφαρμογής

### Επισκόπηση λειτουργιών: Μετατροπή JPX σε μορφή PSD
Αυτή η λειτουργία εστιάζει στη μετατροπή γραφικών υψηλής ποιότητας από αρχεία JPX σε αρχεία PSD που χρησιμοποιούνται εκτενώς από τους χρήστες του Adobe Photoshop.

#### Βήμα 1: Ορισμός καταλόγου εξόδου και προτύπου αρχείου
Αρχικά, ορίστε τον κατάλογο εξόδου όπου θα αποθηκευτούν τα αρχεία που έχετε μετατρέψει. Μπορείτε να ορίσετε ένα πρότυπο για την ονομασία αυτών των αρχείων, όπως φαίνεται παρακάτω:
```csharp
using System.IO;
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");
```

#### Βήμα 2: Δημιουργία ροών αρχείων για σελίδες που έχουν μετατραπεί
Δημιουργήστε μια συνάρτηση για τη δυναμική δημιουργία ροών αρχείων. Αυτό θα βοηθήσει στη διαχείριση κάθε σελίδας του μετατρεπόμενου εγγράφου:
```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

#### Βήμα 3: Φόρτωση του αρχείου JPX πηγής
Χρησιμοποιήστε το `Converter` κλάση για να φορτώσετε το αρχείο JPX πηγής σας. Βεβαιωθείτε ότι έχετε καθορίσει τη σωστή διαδρομή:
```csharp
using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_JPX"))
{
    // Θα ακολουθήσει η διαδικασία μετατροπής.
}
```

#### Βήμα 4: Ορισμός επιλογών μετατροπής για τη μορφή PSD
Ορίστε επιλογές μετατροπής προσαρμοσμένες για τη μορφή PSD χρησιμοποιώντας `ImageConvertOptions`:
```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Psd };
```

#### Βήμα 5: Εκτελέστε τη μετατροπή
Εκτελέστε τη διαδικασία μετατροπής, η οποία θα δημιουργήσει ένα αρχείο PSD για κάθε σελίδα του εγγράφου JPX:
```csharp
converter.Convert(getPageStream, options);
```
