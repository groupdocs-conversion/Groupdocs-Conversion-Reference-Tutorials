## Οδηγός Εφαρμογής

Αυτή η ενότητα περιγράφει τα βήματα για τη μετατροπή αρχείων CF2 σε μορφή PSD χρησιμοποιώντας το GroupDocs.Conversion, εστιάζοντας σε βασικά χαρακτηριστικά της βιβλιοθήκης.

### Φόρτωση αρχείου CF2

**Επισκόπηση:**
Η φόρτωση ενός αρχείου CF2 είναι το πρώτο σας βήμα. Αυτό περιλαμβάνει τη ρύθμιση διαδρομών και τη χρήση του `Converter` κλάση για να ανοίξετε το αρχείο σας.

**Βήματα Υλοποίησης:**
1. **Ορισμός σταθερών για διαδρομές αρχείων:**
   ```csharp
   private const string DocumentDirectory = "YOUR_DOCUMENT_DIRECTORY";
   private const string SampleCf2FilePath = Path.Combine(DocumentDirectory, "sample.cf2");
   ```
2. **Φόρτωση του αρχείου CF2:**
   Χρησιμοποιήστε το `Converter` κλάση για να φορτώσετε το αρχείο CF2.
   
   ```csharp
   using (Converter converter = new Converter(SampleCf2FilePath))
   {
       // Το αρχείο CF2 έχει πλέον φορτωθεί και είναι έτοιμο για μετατροπή.
   }
   ```

### Ορισμός επιλογών μετατροπής

**Επισκόπηση:**
Για να μετατρέψετε ένα αρχείο σε μορφή PSD, πρέπει να ορίσετε συγκεκριμένες επιλογές που θα χρησιμοποιήσει η βιβλιοθήκη κατά τη μετατροπή.

**Βήματα Υλοποίησης:**
1. **Ορισμός επιλογών μετατροπής εικόνας:**
   
   ```csharp
   ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd };
   ```

   Αυτό ρυθμίζει το αρχείο σας για μετατροπή σε μορφή PSD, καθορίζοντας βασικές ιδιότητες της εικόνας εξόδου.

### Μετατροπή CF2 σε PSD

**Επισκόπηση:**
Αυτή η λειτουργία συνδυάζει τις επιλογές φόρτωσης και ρύθμισης με την εκτέλεση της διαδικασίας μετατροπής. Είναι το σημείο όπου όλα συνδυάζονται για να δημιουργηθεί ένα αρχείο PSD από την είσοδο CF2.

**Βήματα Υλοποίησης:**
1. **Ρύθμιση καταλόγου εξόδου και προτύπου αρχείου:**
   
   ```csharp
   private const string OutputDirectory = "YOUR_OUTPUT_DIRECTORY";
   private const string outputFileTemplate = Path.Combine(OutputDirectory, "converted-page-{0}.psd");
   
   Func<SavePageContext, Stream> getPageStream = savePageContext => 
       new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
   ```
2. **Εκτελέστε τη μετατροπή:**
   Εκτελέστε τη μετατροπή με καθορισμένες επιλογές.

   ```csharp
   using (Converter converter = new Converter(SampleCf2FilePath))
   {
       ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd };
       
       // Μετατρέψτε και αποθηκεύστε κάθε σελίδα ως αρχείο PSD
       converter.Convert(getPageStream, options);
   }
   ```

**Συμβουλές αντιμετώπισης προβλημάτων:**
- Βεβαιωθείτε ότι όλες οι διαδρομές έχουν ρυθμιστεί σωστά για να αποφύγετε `FileNotFoundException`.
- Βεβαιωθείτε ότι υπάρχουν τα απαραίτητα δικαιώματα για την ανάγνωση/εγγραφή αρχείων.
