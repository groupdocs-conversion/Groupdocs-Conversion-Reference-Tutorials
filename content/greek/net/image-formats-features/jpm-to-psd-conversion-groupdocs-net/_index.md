## Οδηγός Εφαρμογής
### Ρύθμιση μετατροπής αρχείων
Αυτή η λειτουργία δείχνει πώς να ρυθμίσετε τη διαδικασία μετατροπής από μορφή JPM σε PSD. Περιλαμβάνει τον ορισμό ενός καταλόγου εξόδου και ενός προτύπου για την ονομασία των αρχείων που έχουν μετατραπεί.
#### Ορισμός καταλόγου εξόδου
Ορίστε τον επιθυμητό φάκελο εξόδου όπου θα αποθηκευτούν τα αρχεία που έχουν μετατραπεί:
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
```
#### Ονομασία προτύπου για αρχεία που έχουν μετατραπεί
Δημιουργήστε μια συνάρτηση που δημιουργεί δυναμικά διαδρομές αρχείων με βάση τα αποτελέσματα μετατροπής:
```csharp
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");
Func<SavePageContext, Stream> getPageStream = savePageContext => new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

Φορτώστε το αρχείο JPM πηγής σας για μετατροπή χρησιμοποιώντας το `Converter` τάξη.

```csharp

    // Η ρύθμιση μετατροπής θα υλοποιηθεί στη συνέχεια...
}
```
### Ρύθμιση επιλογών μετατροπής
Ρυθμίστε τις παραμέτρους των επιλογών που απαιτούνται για τη μετατροπή μιας εικόνας από μορφή JPM σε PSD.
#### Ορισμός επιλογών μετατροπής εικόνας
Ορίστε τη μορφή αρχείου προορισμού και άλλες σχετικές παραμέτρους:
```csharp

```
### Εκτέλεση μετατροπής αρχείων
Εκτελέστε τη μετατροπή χρησιμοποιώντας προκαθορισμένες επιλογές και μια συνάρτηση ροής εξόδου.

```csharp

```
