## Οδηγός Εφαρμογής

### Φόρτωση αρχείου XLSB

**Επισκόπηση:**
Ξεκινήστε φορτώνοντας το αρχείο XLSB χρησιμοποιώντας το `Converter` κλάση για πρόσβαση και μετατροπή των περιεχομένων της.

#### Βήμα προς βήμα:
1. **Αρχικοποίηση μετατροπέα**: Δημιουργήστε μια νέα παρουσία του `Converter` με τη διαδρομή του αρχείου σας.
    ```csharp
    string documentPath = "YOUR_DOCUMENT_DIRECTORY/SAMPLE_XLSB";
    using (Converter converter = new Converter(documentPath))
    {
        // Ο κώδικας μετατροπής θα τοποθετηθεί εδώ.
    }
    ```
2. **Απόρριψη πόρων**Βεβαιωθείτε ότι οι πόροι απελευθερώνονται μετά τη χρήση.

### Ορισμός επιλογών μετατροπής για τη μορφή PSD

**Επισκόπηση:**
Ορίστε ρυθμίσεις μετατροπής ειδικά για τη μορφή PSD χρησιμοποιώντας `ImageConvertOptions`.

#### Βήμα προς βήμα:
1. **Δημιουργία ImageConvertOptions**Καθορίστε την έξοδο ως PSD.
    ```csharp
    using GroupDocs.Conversion.Options.Convert;

    ImageConvertOptions options = new ImageConvertOptions 
    {
        Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd
    };
    ```
2. **Επεξήγηση παραμέτρων**: `Format` καθορίζει τη μορφή μετατροπής-στόχο, ορισμένη σε 'Psd'.

### Μετατροπή XLSB σε PSD

**Επισκόπηση:**
Εκτελέστε τη μετατροπή αρχείου, διασφαλίζοντας ότι κάθε σελίδα αποθηκεύεται ως ξεχωριστό PSD στον κατάλογο εξόδου σας.

#### Βήμα προς βήμα:
1. **Ορισμός καταλόγου εξόδου και προτύπου**: Ορίστε πού θα αποθηκευτούν τα αρχεία που έχουν μετατραπεί.
    ```csharp
    string outputFolder = "YOUR_OUTPUT_DIRECTORY";
    string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");
    ```
2. **Δημιουργία συνάρτησης ροής**: Δημιουργήστε δυναμικά διαδρομές αρχείων για κάθε σελίδα.
    ```csharp
    Func<SavePageContext, Stream> getPageStream = savePageContext => 
        new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
    ```
3. **Εκτέλεση μετατροπής**: Χρησιμοποιήστε το `Converter` παράδειγμα και επιλογές για τη μετατροπή του αρχείου σας.
    ```csharp
    using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_XLSB"))
    {
        ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd };
        converter.Convert(getPageStream, options);
    }
    ```
