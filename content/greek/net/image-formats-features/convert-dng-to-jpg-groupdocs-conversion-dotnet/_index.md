## Οδηγός Εφαρμογής

Θα αναλύσουμε τη διαδικασία μετατροπής σε δύο κύρια χαρακτηριστικά: τη φόρτωση ενός αρχείου DNG και τη μετατροπή του σε μορφή JPG.

### Φόρτωση αρχείου DNG
Η φόρτωση του αρχείου DNG πηγής είναι απλή. Ξεκινάτε αρχικοποιώντας το `Converter` κλάση με τη διαδρομή προς το αρχείο DNG σας, όπως φαίνεται παραπάνω. Αυτό το βήμα προετοιμάζει το αρχείο σας για μετατροπή.

```csharp
string sampleDngPath = "YOUR_DOCUMENT_DIRECTORY/sample.dng";
Converter converter = new Converter(sampleDngPath);
```

### Μετατροπή DNG σε JPG
#### Επισκόπηση:
Αυτή η λειτουργία περιλαμβάνει τον ορισμό επιλογών μετατροπής και την επεξεργασία του αρχείου DNG σε μορφή JPEG. Θα χρησιμοποιήσουμε έναν κατάλογο εξόδου και ένα πρότυπο για την ονομασία κάθε σελίδας που έχει μετατραπεί.

#### Βήμα προς βήμα εφαρμογή:
**Ορισμός παραμέτρων εξόδου**
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");
```

**Δημιουργία συνάρτησης ροής για αποθήκευση σελίδας**
Αυτή η λειτουργία διασφαλίζει ότι κάθε σελίδα αποθηκεύεται ως ξεχωριστό αρχείο κατά τη διάρκεια της διαδικασίας μετατροπής.
```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext =>
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

**Ορισμός επιλογών μετατροπής**
Εδώ, καθορίζουμε ότι η μορφή-στόχος μας είναι JPG και ορίζουμε τυχόν πρόσθετες επιλογές εικόνας, εάν χρειάζεται.
```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Jpg };
```

**Εκτελέστε τη μετατροπή**
Τέλος, καλέστε τον `Convert` μέθοδος για την εκτέλεση του μετασχηματισμού αρχείου χρησιμοποιώντας τις καθορισμένες παραμέτρους.
```csharp
converter.Convert(getPageStream, options);
```
