## Οδηγός Εφαρμογής

Τώρα, ας δούμε πώς να μετατρέψετε ένα αρχείο DIB σε μορφή TXT.

### Φόρτωση και μετατροπή του αρχείου DIB

#### Βήμα 1: Αρχικοποίηση κλάσης μετατροπέα

Ο `Converter` Η κλάση φορτώνει το αρχείο πηγαίου κώδικα:

```csharp
using System;
using GroupDocs.Conversion;

string inputFilePath = "YOUR_DOCUMENT_DIRECTORY/Sample.dib";
using (var converter = new Converter(inputFilePath))
{
    Console.WriteLine("DIB file loaded.");
}
```

#### Βήμα 2: Ορισμός επιλογών μετατροπής

Ρυθμίστε τις επιλογές μετατροπής για τη μορφή TXT:

```csharp
using GroupDocs.Conversion.Options.Convert;

WordProcessingConvertOptions options = new WordProcessingConvertOptions 
{ 
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt 
};
Console.WriteLine("Conversion options set.");
```

#### Βήμα 3: Εκτέλεση μετατροπής

Μετατρέψτε το αρχείο και αποθηκεύστε το στην επιθυμητή τοποθεσία:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = System.IO.Path.Combine(outputFolder, "dib-converted-to.txt");

converter.Convert(outputFile, options);
Console.WriteLine("Conversion complete. File saved at: " + outputFile);
```
