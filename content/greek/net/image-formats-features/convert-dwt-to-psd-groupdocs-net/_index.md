- Δημιουργήστε μια μεταβλητή συμβολοσειράς για να διατηρήσετε τη διαδρομή του αρχείου εισόδου σας.

```csharp
string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.dwt");
```

**Βήμα 2: Αρχικοποίηση αντικειμένου μετατροπέα**
- Χρησιμοποιήστε το `Converter` κλάση από το GroupDocs.Conversion για να φορτώσετε το αρχείο DWT.

```csharp
using (Converter converter = new Converter(inputFilePath))
{
    // Η διαδικασία μετατροπής είναι έτοιμη να ξεκινήσει.
}
```
*Εξήγηση*: Το `Converter` Το αντικείμενο χειρίζεται τη φόρτωση και τη μετατροπή, χρησιμεύοντας ως γέφυρα μεταξύ των αρχείων πηγής και των μορφών εξόδου τους.

### Ορισμός επιλογών μετατροπής για μορφή PSD

#### Επισκόπηση
Διαμορφώστε τις ρυθμίσεις που απαιτούνται για τη μετατροπή αρχείων DWT σε μορφή PSD χρησιμοποιώντας τις επιλογές του GroupDocs.Conversion.

**Βήμα 1: Δημιουργία ImageConvertOptions**
- Δημιουργία στιγμιαίας εικόνας `ImageConvertOptions` για να καθορίσετε λεπτομέρειες μετατροπής.

```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd };
```
*Εξήγηση*Ο ορισμός της μορφής εδώ διασφαλίζει ότι το αρχείο σας μετατρέπεται ειδικά σε PSD, επιτρέποντας περαιτέρω προσαρμογή όπως απαιτείται.

### Μετατροπή αρχείου DWT σε PSD

#### Επισκόπηση
Εκτελέστε τη διαδικασία μετατροπής και αποθηκεύστε κάθε σελίδα του εγγράφου ξεχωριστά σε μορφή PSD.

**Βήμα 1: Ορισμός καταλόγου εξόδου**
- Ορίστε έναν κατάλογο όπου θα αποθηκεύονται τα αρχεία που έχουν μετατραπεί. Βεβαιωθείτε ότι υπάρχει ή δημιουργήστε τον μέσω προγραμματισμού.

```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", "ConvertedFiles");
Directory.CreateDirectory(outputFolder);
```

**Βήμα 2: Δημιουργία συνάρτησης ροής σελίδας**
- Αυτή η συνάρτηση δημιουργεί ροές για κάθε σελίδα που μετατρέπεται, διασφαλίζοντας ότι αποθηκεύονται σωστά.

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => new FileStream(string.Format("converted-page-{0}.psd", savePageContext.Page), FileMode.Create);
```

**Βήμα 3: Εκτελέστε τη μετατροπή**
- Χρησιμοποιήστε το `Convert` μέθοδος του `Converter` αντικείμενο για την εκτέλεση μετατροπής χρησιμοποιώντας επιλογές που έχουν οριστεί προηγουμένως.

```csharp
using (Converter converter = new Converter(inputFilePath))
{
    converter.Convert(getPageStream, options);
}
```
*Εξήγηση*: Το `Convert` Η μέθοδος επαναλαμβάνεται σε κάθε σελίδα του εγγράφου σας, εφαρμόζοντας την καθορισμένη μορφή και αποθηκεύοντας την καθεμία ως ξεχωριστό αρχείο.
