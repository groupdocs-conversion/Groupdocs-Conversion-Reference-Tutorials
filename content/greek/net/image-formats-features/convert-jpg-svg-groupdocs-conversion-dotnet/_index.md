### Χαρακτηριστικό: Μετατροπή JPG σε SVG

Αυτή η λειτουργία δείχνει πώς να μετατρέψετε ένα αρχείο JPG σε μορφή SVG χρησιμοποιώντας τις ισχυρές δυνατότητες του GroupDocs.Conversion .NET.

#### Βήμα 1: Ορισμός διαδρομών αρχείων

Ξεκινήστε ορίζοντας διαδρομές για τα αρχεία εισόδου και εξόδου:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string inputFile = "YOUR_DOCUMENT_DIRECTORY\\\\sample.jpg"; // Διαδρομή προς το αρχείο JPG εισόδου
string outputFile = Path.Combine(outputFolder, "jpg-converted-to.svg"); // Όνομα αρχείου SVG εξόδου
```

#### Βήμα 2: Φόρτωση του αρχείου προέλευσης

Φορτώστε το αρχείο JPG πηγής χρησιμοποιώντας το GroupDocs.Conversion API:

```csharp
using (var converter = new Converter(inputFile))
{
    // Τα βήματα μετατροπής θα μεταφερθούν εδώ
}
```

#### Βήμα 3: Καθορισμός επιλογών μετατροπής

Στη συνέχεια, καθορίστε τις επιλογές μετατροπής για τη μορφή SVG:

```csharp
var options = new ΣελίδαΠεριγραφήΓλώσσαΜετατροπήΕπιλογές { Format = PageDescriptionLanguageFileType.Svg };
```

- **PageDescriptionLanguageConvertOptions**Αυτή η κλάση σάς επιτρέπει να ορίσετε ρυθμίσεις ειδικά για τη δημιουργία αρχείων SVG.
- **Ιδιότητα μορφής**Καθορίζει ότι η έξοδος πρέπει να είναι σε μορφή SVG.

#### Βήμα 4: Εκτελέστε τη μετατροπή

Τέλος, εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο σας:

```csharp
converter.Convert(outputFile, options);
```
