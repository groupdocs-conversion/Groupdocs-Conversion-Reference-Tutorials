## Ρύθμιση του GroupDocs.Conversion για .NET

Για να ξεκινήσετε, πρέπει να εγκαταστήσετε τη βιβλιοθήκη GroupDocs.Conversion στο έργο σας. Δείτε πώς:

**Κονσόλα διαχείρισης πακέτων NuGet**
```bash

```

**.NET CLI**
```bash
dotnet add package GroupDocs.Conversion --version 25.3.0
```

## Οδηγός Εφαρμογής

### Φόρτωση και μετατροπή αρχείου PLT σε SVG

Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε ένα αρχείο PLT σε μορφή SVG, η οποία χρησιμοποιείται ευρέως για κλιμακώσιμα διανυσματικά γραφικά.

#### Βήμα 1: Ορίστε τον κατάλογο εξόδου

Αρχικά, ορίστε πού θα αποθηκευτούν τα αρχεία που έχετε μετατρέψει:
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY/";
string outputFile = Path.Combine(outputFolder, "plt-converted-to.svg");
```

#### Βήμα 2: Φόρτωση του αρχείου PLT

Χρησιμοποιήστε το `Converter` κλάση για να φορτώσετε το αρχείο PLT. Αντικαταστήστε `'sample.plt'` με την πραγματική διαδρομή του αρχείου σας.
```csharp
using (var converter = new GroupDocs.Conversion.Converter("YOUR_DOCUMENT_DIRECTORY/sample.plt"))
{
    // Η λογική μετατροπής θα εφαρμοστεί εδώ.
}
```

#### Βήμα 3: Καθορισμός επιλογών μετατροπής

Ορίστε τις επιλογές μετατροπής για τη μορφή SVG:
```csharp
var options = new PageDescriptionLanguageConvertOptions { Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Svg };
```

#### Βήμα 4: Εκτελέστε τη μετατροπή

Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο εξόδου.
```csharp
converter.Convert(outputFile, options);
```
