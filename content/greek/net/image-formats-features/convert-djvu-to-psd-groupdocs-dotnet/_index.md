## Οδηγός Εφαρμογής

Τώρα, ας εμβαθύνουμε στη μετατροπή ενός αρχείου DJVU σε μορφή PSD. Θα αναλύσουμε τη διαδικασία βήμα προς βήμα.

### Βήμα 1: Φόρτωση του αρχείου DJVU

Αρχικά, φορτώστε το αρχείο DJVU χρησιμοποιώντας το `Converter` κλάση. Αυτό είναι κρίσιμο καθώς προετοιμάζει το έγγραφο προέλευσης για μετατροπή.

```csharp
using (var converter = new Converter("input.djvu"))
{
    // Η λογική μετατροπής θα προστεθεί εδώ
}
```

### Βήμα 2: Ρύθμιση επιλογών PSD

Στη συνέχεια, διαμορφώστε τις επιλογές για τη μετατροπή σε μορφή PSD. Αυτό περιλαμβάνει τον καθορισμό βασικών παραμέτρων όπως η λειτουργία χρώματος και η ανάλυση.

```csharp
var convertOptions = new PsdConvertOptions()
{
    ColorMode = GroupDocs.Conversion.FileTypes.PsdColorMode.Rgb,
    Width = 1024,
    Height = 768
};
```

### Βήμα 3: Εκτελέστε τη μετατροπή

Τέλος, εκτελέστε τη μετατροπή χρησιμοποιώντας το `Convert` μέθοδος. Αυτό το βήμα μετατρέπει το αρχείο DJVU σας σε PSD.

```csharp
using (var converter = new Converter("input.djvu"))
{
    converter.Convert("output.psd", convertOptions);
    Console.WriteLine("Conversion completed successfully.");
}
```
