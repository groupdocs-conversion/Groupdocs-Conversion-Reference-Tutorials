## Οδηγός Εφαρμογής

### Μετατροπή αρχείου TXT σε μορφή PSD

#### Επισκόπηση:
Θα μετατρέψουμε ένα `.txt` Μετατρέψτε το αρχείο σε μορφή εγγράφου Adobe Photoshop χρησιμοποιώντας το GroupDocs.Conversion, καταδεικνύοντας την απλότητα και τη δύναμη αυτής της βιβλιοθήκης.

##### Βήμα 1: Προετοιμασία σταθερών καταλόγου
Ορίστε καταλόγους για τα αρχεία εισόδου και εξόδου:

```csharp
public static class Constants
{
    public static string YOUR_DOCUMENT_DIRECTORY = "path_to_your_txt_file";
    public static string YOUR_OUTPUT_DIRECTORY = "path_to_output_directory";

    // Μέθοδος για να λάβετε τη διαδρομή του καταλόγου εξόδου
    public static string GetOutputDirectoryPath()
    {
        return Path.Combine(YOUR_OUTPUT_DIRECTORY);
    }
}
```

##### Βήμα 2: Ρύθμιση επιλογών μετατροπής
Φορτώστε την πηγή σας `.txt` αρχείο και διαμορφώστε τις επιλογές μετατροπής:

```csharp
using System.IO;
using GroupDocs.Conversion.Options.Convert;

string inputFilePath = Constants.YOUR_DOCUMENT_DIRECTORY + "/sample.txt";

// Φόρτωση του αρχείου TXT
using (Converter converter = new Converter(inputFilePath))
{
    // Ρύθμιση επιλογών μετατροπής για μορφή PSD
    ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd };

    string outputFolder = Constants.GetOutputDirectoryPath();
    string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");

    // Συνάρτηση για τη διαχείριση ροών σελίδων κατά τη μετατροπή
    Func<SavePageContext, Stream> getPageStream = savePageContext => new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);

    // Εκτελέστε τη μετατροπή TXT σε PSD
    converter.Convert(getPageStream, options);
}
```

**Εξήγηση:**
- Ο `Converter` το αντικείμενο αρχικοποιείται με το δικό σας `.txt` αρχείο.
- Οι ρυθμίσεις μετατροπής καθορίζουν το PSD ως μορφή εξόδου.
- Μια προσαρμοσμένη συνάρτηση χειρίζεται ροές σελίδων για κάθε σελίδα που έχει μετατραπεί.

### Συμβουλές αντιμετώπισης προβλημάτων:
