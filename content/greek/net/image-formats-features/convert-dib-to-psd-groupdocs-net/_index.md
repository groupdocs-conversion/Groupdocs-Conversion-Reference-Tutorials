## Οδηγός Εφαρμογής

### Μετατροπή αρχείων DIB σε μορφή PSD

#### Επισκόπηση
Η μετατροπή ενός DIB σε PSD σάς επιτρέπει να αξιοποιήσετε τις ισχυρές δυνατότητες επεξεργασίας της Adobe. Ας αναλύσουμε τη διαδικασία βήμα προς βήμα:

#### Βήμα 1: Ρύθμιση του καταλόγου εξόδου σας (H3)
Ορίστε πού θα αποθηκευτούν τα αρχεία που έχετε μετατρέψει χρησιμοποιώντας `outputFolder` και `outputFileTemplate`.

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");
```
**Εξήγηση**Αυτή η διαμόρφωση διασφαλίζει ότι κάθε σελίδα ενός πολυσέλιδου DIB αποθηκεύεται ξεχωριστά.

#### Βήμα 2: Δημιουργία της συνάρτησης ροής (H3)
Ορίστε πώς θα αποθηκεύεται κάθε αρχείο που έχει μετατραπεί:

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```
**Εξήγηση**Αυτή η συνάρτηση δημιουργεί δυναμικά μια ροή αρχείων για κάθε σελίδα χρησιμοποιώντας `SavePageContext`, επιτρέποντάς σας να καθορίσετε τη διαδρομή και τη λειτουργία του αρχείου.

#### Βήμα 3: Φόρτωση του αρχείου DIB πηγής (H3)
Αρχικοποιήστε το `Converter` αντικείμενο με το αρχείο DIB προέλευσης:

```csharp
using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY\\\\sample.dib"))
{
    // Η λογική μετατροπής θα προστεθεί εδώ
}
```
**Εξήγηση**Αυτό το βήμα περιλαμβάνει τη φόρτωση της αρχικής σας εικόνας στη μνήμη για μετατροπή.

#### Βήμα 4: Ρύθμιση επιλογών μετατροπής (H3)
Καθορίστε τη μορφή εξόδου ως PSD:

```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = FileType.Psd };
```
**Εξήγηση**: Με ρύθμιση `FileType.Psd`, κατευθύνετε το GroupDocs να μετατρέψει το αρχείο DIB σε PSD.

#### Βήμα 5: Εκτέλεση μετατροπής (H3)
Εκτελέστε τη διαδικασία μετατροπής χρησιμοποιώντας την καθορισμένη ροή και τις επιλογές:

```csharp
converter.Convert(getPageStream, options);
```
**Εξήγηση**Αυτή η κλήση μεθόδου εκτελεί την πραγματική μετατροπή, αποθηκεύοντας κάθε σελίδα σε μορφή PSD στον καθορισμένο κατάλογο εξόδου.
