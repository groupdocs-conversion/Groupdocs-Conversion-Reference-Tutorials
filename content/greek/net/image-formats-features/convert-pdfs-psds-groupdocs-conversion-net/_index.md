## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Μετατροπή PDF σε PSD

Η μετατροπή ενός αρχείου PDF σε μορφή PSD είναι ανεκτίμητη για τους γραφίστες που χρειάζονται επεξεργάσιμα επίπεδα. Ας αναλύσουμε τη διαδικασία:

#### Βήμα 1: Ορισμός φακέλου εξόδου και διαδρομών αρχείων

Ρυθμίστε καταλόγους για αρχεία εισόδου και εξόδου. Προσαρμόστε τις διαδρομές όπως απαιτείται.

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string inputFile = "YOUR_DOCUMENT_DIRECTORY\\sample.pdf";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");
```

#### Βήμα 2: Δημιουργήστε μια συνάρτηση ροής

Θα χρησιμοποιήσουμε μια συνάρτηση για να δημιουργήσουμε ροές για κάθε σελίδα που μετατρέπεται. Αυτό διασφαλίζει ότι κάθε αρχείο PSD έχει το σωστό όνομα.

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext =>
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

#### Βήμα 3: Ρύθμιση επιλογών μετατροπής

Ορίστε τις επιλογές μετατροπής για να καθορίσετε ότι μετατρέπουμε σε μορφή PSD.

```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Psd };
```

#### Βήμα 4: Εκτελέστε τη μετατροπή

Εκτελέστε τη μετατροπή χρησιμοποιώντας το `Converter` αντικείμενο και τις καθορισμένες ρυθμίσεις σας.

```csharp
using (Converter converter = new Converter(inputFile))
{
    // Μετατρέψτε κάθε σελίδα του PDF σε μορφή PSD
    converter.Convert(getPageStream, options);
}
```
