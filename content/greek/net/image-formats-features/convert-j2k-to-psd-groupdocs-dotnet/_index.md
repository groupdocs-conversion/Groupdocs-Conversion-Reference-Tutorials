#### Βασική Αρχικοποίηση και Ρύθμιση
Δείτε πώς μπορείτε να ρυθμίσετε το GroupDocs.Conversion στο έργο σας C#:

```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main(string[] args)
    {
        // Αρχικοποιήστε τον μετατροπέα με μια διαδρομή αρχείου J2K
        using (Converter converter = new Converter("path/to/your/file.j2k"))
        {
            // Ρύθμιση επιλογών μετατροπής για μορφή PSD
            var convertOptions = new PsdConvertOptions();
            
            // Μετατρέψτε και αποθηκεύστε το αρχείο PSD εξόδου
            converter.Convert("output/path/output.psd", convertOptions);
        }
    }
}
```

Αυτός ο κώδικας αρχικοποιεί ένα `Converter` αντικείμενο με το αρχείο J2K σας και το μετατρέπει σε PSD χρησιμοποιώντας καθορισμένες επιλογές.
