Μόλις εγκατασταθεί, αρχικοποιήστε το GroupDocs.Conversion με αυτήν τη ρύθμιση σε C#:

```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main()
    {
        string pptxFilePath = "sample-presentation.pptx";
        string pdfOutputPath = "pptx-converted-to.pdf";

        using (var converter = new Converter(pptxFilePath))
        {
            var options = new PdfConvertOptions();
            converter.Convert(pdfOutputPath, options);
        }
    }
}
```

## Οδηγός Εφαρμογής

### Μετατροπή PPTX σε PDF

Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε παρουσιάσεις PowerPoint σε μια καθολικά προσβάσιμη μορφή: PDF.

#### Βήμα 1: Ορισμός διαδρομών αρχείων

Ορίστε τις διαδρομές για τα αρχεία PPTX πηγής και PDF εξόδου. Αντικαταστήστε `"YOUR_DOCUMENT_DIRECTORY"` και `"YOUR_OUTPUT_DIRECTORY"` με πραγματικούς καταλόγους:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

string pptxFilePath = Path.Combine(documentDirectory, "sample-presentation.pptx");
string pdfOutputPath = Path.Combine(outputDirectory, "pptx-converted-to.pdf");
```

#### Βήμα 2: Φόρτωση του αρχείου PPTX

Χρησιμοποιήστε το GroupDocs.Conversion για να φορτώσετε το αρχείο PowerPoint. Αυτό το βήμα ξεκινά τη διαδικασία μετατροπής:

```csharp
using (var converter = new Converter(pptxFilePath))
{
    // Η λογική μετατροπής θα εφαρμοστεί εδώ.
}
```

#### Βήμα 3: Ορισμός επιλογών μετατροπής PDF

Ρυθμίστε επιλογές για τη μετατροπή σε PDF, επιτρέποντας την προσαρμογή, όπως τον ορισμό εύρους σελίδων ή συγκεκριμένων μορφών:

```csharp
var options = new PdfConvertOptions();
```

#### Βήμα 4: Εκτελέστε τη μετατροπή

Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο εξόδου. Εδώ είναι που το GroupDocs.Conversion επεξεργάζεται το PPTX εισόδου σε ένα PDF:

```csharp
converter.Convert(pdfOutputPath, options);
```
