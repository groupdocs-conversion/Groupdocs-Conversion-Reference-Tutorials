## Οδηγός Εφαρμογής
### Μετατροπή ODT σε PDF
Η βασική λειτουργικότητα που θα καλύψουμε είναι η μετατροπή ενός εγγράφου ODT σε μορφή PDF. Ακολουθήστε τα παρακάτω βήματα για μια επιτυχημένη μετατροπή:

#### 1. Ορισμός καταλόγου εξόδου
Καθορίστε πού θέλετε να αποθηκευτούν τα PDF που έχετε μετατρέψει.

```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "odt-converted-to.pdf");
```
Εδώ, `outputFolder` είναι η διαδρομή καταλόγου για την αποθήκευση αρχείων, και `outputFile` συνδυάζει αυτήν τη διαδρομή με ένα όνομα αρχείου για εύκολη αναγνώριση.

#### 2. Φορτώστε το αρχείο ODT πηγής
Χρήση του GroupDocs.Conversion `Converter` κλάση, φορτώστε το έγγραφο ODT σας:

```csharp
class Program
{
    static void Main(string[] args)
    {
        using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY\sample.odt"))
        {
            // Τα βήματα μετατροπής θα ακολουθήσουν εδώ
        }
    }
}
```
Αυτό το απόσπασμα αρχικοποιεί μια νέα παρουσία του `Converter`που δείχνει στο αρχείο ODT πηγής σας.

#### 3. Αρχικοποίηση επιλογών μετατροπής PDF
Προετοιμαστείτε για μετατροπή ρυθμίζοντας `PdfConvertOptions`.

```csharp
class Program
{
    static void Main(string[] args)
    {
        using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY\sample.odt"))
        {
            var options = new PdfConvertOptions();
            // Τα βήματα μετατροπής θα ακολουθήσουν εδώ
        }
    }
}
```
`PdfConvertOptions` σας επιτρέπει να διαμορφώσετε συγκεκριμένες ρυθμίσεις όπως εύρος σελίδων, περιθώρια και άλλα, αν και οι προεπιλογές λειτουργούν καλά στις περισσότερες περιπτώσεις.

#### 4. Μετατροπή και αποθήκευση του PDF
Εκτελέστε τη διαδικασία μετατροπής με αυτές τις διαμορφώσεις:

```csharp
class Program
{
    static void Main(string[] args)
    {
        using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY\sample.odt"))
        {
            var options = new PdfConvertOptions();
            string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
            string outputFile = Path.Combine(outputFolder, "odt-converted-to.pdf");
            converter.Convert(outputFile, options);
        }
    }
}
```
Αυτή η κλήση μεθόδου μετατρέπει το αρχείο ODT σας σε PDF χρησιμοποιώντας τις καθορισμένες επιλογές και το αποθηκεύει στο `outputFile`.
