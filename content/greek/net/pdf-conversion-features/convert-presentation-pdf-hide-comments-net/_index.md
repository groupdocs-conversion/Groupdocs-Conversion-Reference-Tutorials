## Οδηγός Εφαρμογής

### Απόκρυψη σχολίων κατά τη μετατροπή PDF

Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε μια παρουσίαση σε PDF αποκρύπτοντας τα σχόλια, διασφαλίζοντας ότι δεν θα εμφανίζονται στο αρχείο εξόδου.

#### Βήμα 1: Ορισμός επιλογών φόρτωσης

Αρχικά, ορίστε επιλογές φόρτωσης που καθορίζουν τον τρόπο χειρισμού των σχολίων κατά τη μετατροπή. Ορίζοντας `CommentsPosition` να `None`, διασφαλίζουμε ότι τα σχόλια είναι κρυφά.

```csharp
Func<LoadContext, LoadOptions> getLoadOptions = loadContext => new PresentationLoadOptions
{
    CommentsPosition = PresentationCommentsPosition.None // Απόκρυψη σχολίων.
};
```

#### Βήμα 2: Αρχικοποίηση μετατροπέα

Αρχικοποιήστε τον μετατροπέα με το αρχείο εισόδου σας και τις προσαρμοσμένες επιλογές φόρτωσης. Αυτό το βήμα προετοιμάζει το έγγραφο για μετατροπή.

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = System.IO.Path.Combine(outputFolder, "converted.pdf");

using (Converter converter = new Converter(inputFile, getLoadOptions))
{
    // Ακολουθεί η λογική μετατροπής.
}
```

#### Βήμα 3: Ορισμός επιλογών μετατροπής PDF

Ορίστε τις επιλογές μετατροπής που αφορούν συγκεκριμένα το PDF. Αυτό καθορίζει τον τρόπο με τον οποίο θα μορφοποιηθεί το έγγραφό σας στο αρχείο εξόδου.

```csharp
PdfConvertOptions options = new PdfConvertOptions();
```

#### Βήμα 4: Εκτέλεση μετατροπής

Τέλος, εκτελέστε τη διαδικασία μετατροπής από παρουσίαση σε PDF χρησιμοποιώντας τις διαμορφωμένες επιλογές.

```csharp
converter.Convert(outputFile, options);
```

**Συμβουλές αντιμετώπισης προβλημάτων:** Βεβαιωθείτε ότι τα αρχεία εισόδου σας είναι προσβάσιμα και ότι οι διαδρομές έχουν καθοριστεί σωστά. Ελέγξτε για τυχόν εξαιρέσεις που δημιουργούνται κατά την αρχικοποίηση ή τη μετατροπή, για ενδείξεις σχετικά με την επίλυση προβλημάτων.
