### Φόρτωση εγγράφου πηγής
Ξεκινήστε φορτώνοντας το έγγραφο PDF:

```csharp

using (var converter = new Converter(documentPath))

    // Η λογική μετατροπής θα προστεθεί εδώ
}
```
#### Εξήγηση:
- **Μετατροπέας**: Χειρίζεται τη διαδικασία μετατροπής.
- **Διαδρομήεγγράφου**: Διαδρομή προς το αρχείο PDF προέλευσης.

### Ρύθμιση παραμέτρων επιλογών μετατροπής
Καθορίστε το CSV ως τη μορφή στόχου σας χρησιμοποιώντας το `SpreadsheetConvertOptions`:

```csharp
// Ορίστε επιλογές μετατροπής για CSV
var options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Csv };
```
#### Εξήγηση:
- **Επιλογές Μετατροπής Υπολογιστικού Φύλλου**: Διαμορφώνει ρυθμίσεις που αφορούν συγκεκριμένες μορφές υπολογιστικών φύλλων.
- **Σχήμα και διάταξις βιβλίου**: Καθορίζει τον τύπο αρχείου εξόδου, σε αυτήν την περίπτωση, CSV.

### Εκτέλεση μετατροπής
Εκτελέστε τη μετατροπή και αποθηκεύστε το αποτέλεσμα:

```csharp
string outputFile = Path.Combine("YOUR_OUTPUT_DIRECTORY", "pdf-converted-to.csv");
converter.Convert(outputFile, options);
```
#### Εξήγηση:
- **Μέθοδος μετατροπής**: Εκτελεί τον μετασχηματισμό του εγγράφου.
- **Αρχείο εξόδου**: Διαδρομή προορισμού για το αρχείο CSV που έχει μετατραπεί.
