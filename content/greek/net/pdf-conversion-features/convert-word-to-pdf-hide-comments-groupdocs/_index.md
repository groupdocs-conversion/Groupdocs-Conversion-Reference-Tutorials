### Βασική Αρχικοποίηση και Ρύθμιση

Αρχικοποιήστε το GroupDocs.Conversion στο έργο σας C#:
```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;
using GroupDocs.Conversion.Options.Load;

class Program
{
    static void Main()
    {
        // Ορίστε τον κατάλογο εξόδου και τη διαδρομή αρχείου.
        string outputFolder = "YOUR_OUTPUT_DIRECTORY";
        string outputFile = Path.Combine(outputFolder, "converted.pdf");

        // Αρχικοποιήστε τις επιλογές φόρτωσης για να αποκρύψετε τα σχόλια κατά τη μετατροπή.
        Func<LoadContext, LoadOptions> getLoadOptions = loadContext => new WordProcessingLoadOptions
        {
            CommentDisplayMode = WordProcessingCommentDisplay.Hidden // Ορισμός σχολίων ώστε να είναι κρυμμένα
        };

        // Μετατρέψτε το έγγραφο χρησιμοποιώντας το GroupDocs.Conversion.
        using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_DOCX_WITH_TRACKED_CHANGES", getLoadOptions))
        {
            PdfConvertOptions options = new PdfConvertOptions(); // Αρχικοποίηση επιλογών μετατροπής PDF
            converter.Convert(outputFile, options); // Εκτελέστε τη μετατροπή σε PDF
        }
    }
}
```

## Οδηγός Εφαρμογής

### Μετατροπή εγγράφου επεξεργασίας κειμένου σε PDF με κρυφά σχόλια

Ακολουθήστε αυτά τα βήματα για να μετατρέψετε ένα `.docx` αρχείο σε PDF ενώ αποκρύπτονται τα σχόλια.

#### Βήμα 1: Ορισμός επιλογών φόρτωσης
Σειρά `CommentDisplayMode` να `Hidden`:
```csharp
Func<LoadContext, LoadOptions> getLoadOptions = loadContext => new WordProcessingLoadOptions
{
    CommentDisplayMode = WordProcessingCommentDisplay.Hidden // Απόκρυψη σχολίων κατά τη μετατροπή
};
```

#### Βήμα 2: Αρχικοποίηση μετατροπέα
Δημιουργήστε ένα `Converter` παράδειγμα:
```csharp
using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_DOCX_WITH_TRACKED_CHANGES", getLoadOptions))
{
    // Συνεχίστε με τη ρύθμιση της μετατροπής.
}
```

#### Βήμα 3: Ορισμός επιλογών μετατροπής PDF
Αρχικοποίηση `PdfConvertOptions`:
```csharp
PdfConvertOptions options = new PdfConvertOptions();
```

#### Βήμα 4: Εκτέλεση μετατροπής
Εκτελέστε τη διαδικασία μετατροπής:
```csharp
converter.Convert(outputFile, options); // Μετατροπή σε PDF με κρυφά σχόλια
```
