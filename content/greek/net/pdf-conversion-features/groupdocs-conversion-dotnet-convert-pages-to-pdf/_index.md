## Οδηγός Εφαρμογής

### Μετατροπή εύρους σελίδων
#### Επισκόπηση
Η δυνατότητα μετατροπής συγκεκριμένων εύρων σελίδων είναι ανεκτίμητη όταν πρόκειται για μεγάλα έγγραφα. Αυτή η λειτουργία σάς επιτρέπει να καθορίσετε την αρχική σελίδα και τον αριθμό των διαδοχικών σελίδων που θα μετατραπούν, βελτιστοποιώντας τόσο τον χρόνο όσο και τους πόρους.

#### Βήματα Υλοποίησης
**Βήμα 1:** Ορισμός καταλόγου εξόδου
- **Σκοπός**: Ορίστε πού θα αποθηκευτούν τα αρχεία PDF που έχετε μετατρέψει.
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
```

**Βήμα 2:** Αρχικοποίηση παρουσίας μετατροπέα
- **Γιατί;** Αυτό δημιουργεί ένα περιβάλλον μετατροπής για το έγγραφό σας.
```csharp
using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_DOCX"))
{
    // Η λογική μετατροπής ισχύει εδώ
}
```

**Βήμα 3:** Ρύθμιση παραμέτρων επιλογών μετατροπής PDF
- **Βασικές παράμετροι**: `PageNumber` και `PagesCount`.
- **Σκοπός**: Καθορίζει ποιες σελίδες θα μετατραπούν.
```csharp
PdfConvertOptions options = new PdfConvertOptions
{
    PageNumber = 2, // Έναρξη μετατροπής από τη σελίδα 2
    PagesCount = 2  // Μετατροπή δύο συνεχόμενων σελίδων
};
```

**Βήμα 4:** Εκτέλεση μετατροπής και αποθήκευση εξόδου
- **Μέθοδος**: `converter.Convert()`
- **Σκοπός**: Εκτελεί τη διαδικασία μετατροπής.
```csharp
string outputFile = Path.Combine(outputFolder, "converted.pdf");
converter.Convert(outputFile, options);
```

#### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι οι διαδρομές αρχείων έχουν καθοριστεί σωστά για να αποφύγετε σφάλματα "δεν βρέθηκε αρχείο".
- Επιβεβαιώστε ότι το έγγραφο εισαγωγής σας υποστηρίζεται από το GroupDocs.Conversion.
