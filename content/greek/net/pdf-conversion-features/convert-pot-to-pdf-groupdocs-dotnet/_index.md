### Βασική Αρχικοποίηση

Μόλις εγκατασταθεί, αρχικοποιήστε το GroupDocs.Conversion για .NET στο έργο C# σας:

```csharp
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

// Ορισμός διαδρομών
const string sourceFilePath = "YOUR_DOCUMENT_DIRECTORY\\sample.pot";
const string outputDirectory = "YOUR_OUTPUT_DIRECTORY\";
string outputFile = System.IO.Path.Combine(outputDirectory, "pot-converted-to.pdf");

// Αρχικοποίηση μετατροπέα
using (var converter = new Converter(sourceFilePath))
{
    var options = new PdfConvertOptions();
    converter.Convert(outputFile, options);
}
```

Αυτή η ρύθμιση αρχικοποιεί τη διαδικασία μετατροπής καθορίζοντας τις διαδρομές προέλευσης και εξόδου.
