### Φόρτωση και προετοιμασία μετατροπής αρχείου VSSX

#### Επισκόπηση
Το πρώτο βήμα είναι η φόρτωση του αρχείου VSSX πηγής χρησιμοποιώντας το GroupDocs.Conversion. Αυτό προετοιμάζει το αρχείο σας για μετασχηματισμό.

**Βήμα 1: Ορισμός διαδρομών αρχείων**
Καθορίστε καταλόγους και ονόματα αρχείων για αρχεία εισόδου και εξόδου:

```csharp
string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY/";

// Ορίστε τη διαδρομή προς το αρχείο VSSX εισόδου και το πρότυπο εξόδου
string inputFilePath = Path.Combine(documentDirectory, "sample.vssx");
string outputFileTemplate = Path.Combine(outputDirectory, "converted-page-{0}.psd");
```

**Βήμα 2: Φόρτωση του αρχείου προέλευσης**
Χρησιμοποιήστε το `Converter` κλάση για να φορτώσετε το αρχείο VSSX πηγαίου κώδικα:

```csharp
using (Converter converter = new Converter(inputFilePath))
{
    // Η μετατροπή θα εξεταστεί στην επόμενη ενότητα χαρακτηριστικών.
}
```

Αυτό το βήμα διασφαλίζει ότι το αρχείο σας είναι έτοιμο για μετατροπή.

### Μετατροπή VSSX σε μορφή PSD

#### Επισκόπηση
Στη συνέχεια, μετατρέψτε το φορτωμένο αρχείο VSSX σε μορφή PSD χρησιμοποιώντας εξειδικευμένες επιλογές μετατροπής.

**Βήμα 1: Ορισμός ροής εξόδου**
Ορίστε μια συνάρτηση για να δημιουργήσετε μια ροή εξόδου για κάθε σελίδα που μετατρέπεται:

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext =>
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

Αυτή η λειτουργία διασφαλίζει ότι κάθε σελίδα αποθηκεύεται ως ξεχωριστό αρχείο PSD.

**Βήμα 2: Ορισμός επιλογών μετατροπής**
Διαμορφώστε τις ρυθμίσεις μετατροπής για την επιθυμητή μορφή εξόδου:

```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd };
```

Εδώ, `options` Καθορίζει ότι η μορφή προορισμού είναι PSD.

**Βήμα 3: Εκτέλεση μετατροπής**
Εκτελέστε τη μετατροπή χρησιμοποιώντας την καθορισμένη ροή και τις επιλογές:

```csharp
converter.Convert(getPageStream, options);
```

Αυτό το βήμα χειρίζεται την πραγματική μετατροπή του VSSX σε PSD.
