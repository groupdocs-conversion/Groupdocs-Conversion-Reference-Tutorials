## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Μετατροπή BMP σε HTML

Αυτή η λειτουργία επισημαίνει τον τρόπο μετατροπής ενός αρχείου BMP σε μορφή HTML χρησιμοποιώντας το GroupDocs.Conversion.

#### Βήμα 1: Ρύθμιση καταλόγων και διαδρομών αρχείων
Αρχικά, ορίστε διαδρομές για τα αρχεία BMP εισόδου και HTML εξόδου:

```csharp
using System.IO;

string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// Ορίστε τη διαδρομή προς το αρχείο BMP που θέλετε να μετατρέψετε
string bmpFilePath = Path.Combine(documentDirectory, "sample.bmp");

// Ορίστε τη διαδρομή του αρχείου HTML εξόδου
string htmlOutputFile = Path.Combine(outputDirectory, "bmp-converted-to.html");
```

#### Βήμα 2: Φόρτωση και μετατροπή του αρχείου
Φορτώστε το BMP σας χρησιμοποιώντας το GroupDocs.Conversion και ρυθμίστε τις επιλογές μετατροπής:

```csharp
using (var converter = new Converter(bmpFilePath))
{
    // Ορισμός επιλογών μετατροπής για μετατροπή σε μορφή ιστού (HTML)
    var options = new WebConvertOptions();
    
    // Εκτελέστε τη μετατροπή από BMP σε HTML και αποθηκεύστε το αρχείο εξόδου
    converter.Convert(htmlOutputFile, options);
}
```

**Εξήγηση:**
- **Μετατροπέας**: Χειρίζεται τη φόρτωση και τη διαχείριση του εγγράφου προέλευσης.
- **Επιλογές Μετατροπής Web**: Διαμορφώνει ρυθμίσεις για μορφές συμβατές με το web, όπως η HTML.

#### Συμβουλές αντιμετώπισης προβλημάτων:
- Βεβαιωθείτε ότι η διαδρομή BMP εισόδου σας είναι σωστή για να αποφύγετε `FileNotFoundException`.
- Επαληθεύστε τα δικαιώματα καταλόγου εξόδου για να αποτρέψετε σφάλματα εγγραφής.
