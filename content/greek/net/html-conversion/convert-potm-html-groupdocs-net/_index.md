## Οδηγός Εφαρμογής
### Μετατροπή POTM σε HTML
Ακολουθήστε αυτά τα βήματα για να μετατρέψετε ένα αρχείο .potm σε HTML:
#### Βήμα 1: Ορισμός διαδρομών για αρχεία εισόδου και εξόδου
Ορίστε καταλόγους για τα αρχεία POTM εισόδου και τα αρχεία HTML εξόδου.
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY"; // Ενημέρωση με τη διαδρομή σας
string outputDirectory = "YOUR_OUTPUT_DIRECTORY"; // Ενημέρωση με τη διαδρομή σας

class Constants
{
    public static string GetOutputDirectoryPath() => outputDirectory;
    
    public static string SAMPLE_POTM => Path.Combine(documentDirectory, "sample.potm");
}
```
#### Βήμα 2: Εκτελέστε τη μετατροπή
Μετατρέψτε το αρχείο POTM σε HTML χρησιμοποιώντας το GroupDocs.Conversion.
```csharp
using System.IO;
using GroupDocs.Conversion.Options.Convert;

string outputFile = Path.Combine(Constants.GetOutputDirectoryPath(), "potm-converted-to.html");

// Αρχικοποίηση μετατροπέα με διαδρομή αρχείου εισόδου
using (var converter = new Converter(Constants.SAMPLE_POTM))
{
    // Καθορισμός επιλογών μετατροπής για τη μορφή HTML
    var options = new WebConvertOptions();

    // Εκτελέστε τη μετατροπή και αποθηκεύστε στο καθορισμένο αρχείο εξόδου
    converter.Convert(outputFile, options);
}
```
#### Επεξήγηση Βασικών Εννοιών
- **Επιλογές WebConvert:** Διαμορφώνει ρυθμίσεις ειδικά για μορφές φιλικές προς το web όπως η HTML.
- **μετατροπέας.Μετατροπή():** Ξεκινά τη διαδικασία μετατροπής με τις παρεχόμενες παραμέτρους.
