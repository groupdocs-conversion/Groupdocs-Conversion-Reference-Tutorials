### Βασική Αρχικοποίηση και Ρύθμιση

Δείτε πώς μπορείτε να αρχικοποιήσετε το GroupDocs.Conversion στο έργο σας C#:

```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main(string[] args)
    {
        // Αρχικοποιήστε το αντικείμενο Converter με μια διαδρομή προς το αρχείο VSD σας
        using (Converter converter = new Converter(@"YOUR_DOCUMENT_DIRECTORY\yourfile.vsd"))
        {
            // Προετοιμασία επιλογών μετατροπής για μορφή HTML
            var options = new MarkupConvertOptions();
            
            // Μετατρέψτε και αποθηκεύστε το αποτέλεσμα ως αρχείο HTML
            converter.Convert(@"YOUR_OUTPUT_DIRECTORY\output.html", options);
        }
    }
}
```


```csharp

```
