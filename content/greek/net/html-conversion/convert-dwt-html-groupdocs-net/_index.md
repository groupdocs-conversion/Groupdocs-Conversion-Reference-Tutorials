### Μετατροπή DWT σε HTML

**Επισκόπηση**Αυτή η λειτουργία μετατρέπει ένα αρχείο DWT σε μορφή HTML, καθιστώντας το προσβάσιμο σε πλατφόρμες ιστού.

#### Φόρτωση και ρύθμιση παραμέτρων επιλογών μετατροπής
```csharp
using System;
using System.IO;
using GroupDocs.Conversion.Options.Convert;

string documentPath = @"YOUR_DOCUMENT_DIRECTORY\sample.dwt";
string outputFolder = @"YOUR_OUTPUT_DIRECTORY\";
string outputFile = Path.Combine(outputFolder, "dwt-converted-to.html");

// Φορτώστε το αρχείο DWT πηγής χρησιμοποιώντας την κλάση Converter.
using (var converter = new Converter(documentPath))
{
    // Ρυθμίστε τις παραμέτρους των επιλογών μετατροπής για τη μορφή HTML.
    var options = new WebConvertOptions();
    
    // Βεβαιωθείτε ότι υπάρχει ο κατάλογος εξόδου
    Directory.CreateDirectory(outputFolder);
    
    // Εκτελέστε τη μετατροπή και αποθηκεύστε την έξοδο στην καθορισμένη διαδρομή.
    converter.Convert(outputFile, options);
}
```

**Εξήγηση**: 
- **Επιλογές Μετατροπής Web**: Διαμορφώνει ρυθμίσεις ειδικά για μετατροπή HTML.
- **Directory.CreateDirectory()**: Διασφαλίζει ότι ο φάκελος εξόδου είναι διαθέσιμος, αποτρέποντας σφάλματα χρόνου εκτέλεσης.
