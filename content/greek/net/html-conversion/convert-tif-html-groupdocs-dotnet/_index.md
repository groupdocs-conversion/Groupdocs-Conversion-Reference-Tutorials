## Οδηγός Εφαρμογής

### Μετατροπή TIFF σε HTML

Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε εικόνες TIFF υψηλής ποιότητας σε προσβάσιμες μορφές HTML για προβολή στο web. Ακολουθήστε τα παρακάτω βήματα για να εφαρμόσετε τη μετατροπή:

#### Βήμα 1: Προετοιμάστε το περιβάλλον σας
Βεβαιωθείτε ότι το αρχείο TIFF πηγής και ο κατάλογος εξόδου είναι έτοιμα. Δείτε πώς μπορείτε να τα ρυθμίσετε στον κώδικά σας:
```csharp
string sourceTiffPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.tif");
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "tiff-converted-to.html");

// Βεβαιωθείτε ότι ο κατάλογος εξόδου υπάρχει
Directory.CreateDirectory(outputFolder);
```
#### Βήμα 2: Φόρτωση και μετατροπή του αρχείου TIFF
Φορτώστε το αρχείο TIFF στον μετατροπέα και καθορίστε HTML ως μορφή προορισμού:
```csharp
using (var converter = new GroupDocs.Conversion.Converter(sourceTiffPath))
{
    // Ρύθμιση επιλογών μετατροπής για μορφή HTML
    var options = new WebConvertOptions();
    
    // Εκτελέστε τη μετατροπή και αποθηκεύστε το στο αρχείο εξόδου
    converter.Convert(outputFile, options);
}
```
**Επεξήγηση παραμέτρων κώδικα:**
- `sourceTiffPath`: Διαδρομή προς το αρχείο TIFF σας.
- `outputFolder` & `outputFile`Κατάλογοι για την αποθήκευση αρχείων HTML που έχουν μετατραπεί.
- `WebConvertOptions()`: Διαμορφώνει τις ρυθμίσεις μετατροπής για μορφές ιστού.

#### Συμβουλές αντιμετώπισης προβλημάτων:
- Βεβαιωθείτε ότι η διαδρομή του αρχείου TIFF προέλευσης είναι σωστή.
- Ελέγξτε αν οι κατάλογοι εξόδου έχουν δικαιώματα εγγραφής.
