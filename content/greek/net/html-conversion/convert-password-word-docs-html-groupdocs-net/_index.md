## Οδηγός Εφαρμογής

### Λειτουργία 1: Μετατροπή εγγράφου που προστατεύεται με κωδικό πρόσβασης σε HTML

#### Επισκόπηση

Η μετατροπή ενός εγγράφου που προστατεύεται με κωδικό πρόσβασης σε μορφή HTML επιτρέπει ευρύτερη προσβασιμότητα και ευκολότερη ενσωμάτωση με εφαρμογές ιστού. Δείτε πώς μπορείτε να το πετύχετε αυτό χρησιμοποιώντας το GroupDocs.Conversion.

#### Οδηγίες βήμα προς βήμα

**Βήμα 1**: Ορίστε τους καταλόγους και τις διαδρομές αρχείων σας.

```csharp
string dataDir = Path.Combine(Directory.GetCurrentDirectory(), "YOUR_DOCUMENT_DIRECTORY");
string outputDir = Path.Combine(Directory.GetCurrentDirectory(), "YOUR_OUTPUT_DIRECTORY");

string inputFilePath = Path.Combine(dataDir, "SAMPLE_DOCX_WITH_PASSWORD.docx");
string outputFile = Path.Combine(outputDir, "converted.html");
```

**Βήμα 2**Δημιουργήστε μια συνάρτηση για την ανάκτηση επιλογών φόρτωσης με λεπτομέρειες προστασίας με κωδικό πρόσβασης.

```csharp
Func<LoadContext, LoadOptions> getLoadOptions = loadContext => new WordProcessingLoadOptions
{
    Password = "12345" // Καθορίστε εδώ τον κωδικό πρόσβασης του εγγράφου
};
```

**Βήμα 3**: Αρχικοποίηση του μετατροπέα και ρύθμιση επιλογών μετατροπής.

```csharp
using (Converter converter = new Converter(inputFilePath, getLoadOptions))
{
    WebConvertOptions options = new WebConvertOptions
    {
        PageNumber = 2, 
        FixedLayout = true,
        PagesCount = 1,
        FixedLayoutShowBorders = false
    };
    
    // Εκτελέστε τη μετατροπή
    converter.Convert(outputFile, options);
}
```

Σε αυτήν την ενότητα, `PageNumber`, `PagesCount`και άλλες παράμετροι σάς επιτρέπουν να ρυθμίσετε με ακρίβεια ποια μέρη του εγγράφου σας θα μετατραπούν.

### Λειτουργία 2: Καθορισμός σελίδων για μετατροπή

#### Επισκόπηση

Μερικές φορές, απαιτείται η μετατροπή μόνο συγκεκριμένων σελίδων. Το GroupDocs.Conversion απλοποιεί τον καθορισμό των ακριβών σελίδων που χρειάζεστε σε μορφή HTML.

#### Οδηγίες βήμα προς βήμα

**Βήμα 1**Ορίστε διαδρομές και αρχικοποιήστε τον μετατροπέα όπως φαίνεται προηγουμένως, αλλά χωρίς προστασία με κωδικό πρόσβασης.

```csharp
string inputFilePath = Path.Combine(dataDir, "SAMPLE_DOCX.docx");
string outputFile = Path.Combine(outputDir, "pages_converted.html");

using (Converter converter = new Converter(inputFilePath))
{
    WebConvertOptions options = new WebConvertOptions
    {
        PageNumber = 2,
        PagesCount = 1,
        FixedLayout = true,
        FixedLayoutShowBorders = false
    };
    
    // Μετατροπή συγκεκριμένων σελίδων σε μορφή HTML
    converter.Convert(outputFile, options);
}
```
