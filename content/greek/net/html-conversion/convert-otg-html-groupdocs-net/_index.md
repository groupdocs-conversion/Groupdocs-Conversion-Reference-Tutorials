### Βασική Αρχικοποίηση και Ρύθμιση

Δείτε πώς μπορείτε να ρυθμίσετε το περιβάλλον σας για μετατροπή:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

// Ορίστε διαδρομές για καταλόγους εισόδου και εξόδου
string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY\";

// Αρχικοποίηση της διαδρομής του αρχείου OTG προέλευσης
string sourceOtgFilePath = Path.Combine(documentDirectory, "sample.otg");
// Ορίστε τη διαδρομή του αρχείου HTML εξόδου
string outputHtmlFilePath = Path.Combine(outputDirectory, "otg-converted-to.html");

// Φόρτωση και προετοιμασία για μετατροπή
using (var converter = new Converter(sourceOtgFilePath))
{
    // Δημιουργήστε επιλογές μετατροπής για μορφή HTML
    var htmlConversionOptions = new WebConvertOptions();
    
    // Εκτελέστε τη μετατροπή από OTG σε HTML
    converter.Convert(outputHtmlFilePath, htmlConversionOptions);
}
```

## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Φόρτωση και μετατροπή αρχείου OTG σε HTML

Αυτή η λειτουργία σάς επιτρέπει να φορτώσετε ένα αρχείο OpenDocument Graphic Template (OTG) και να το μετατρέψετε σε μορφή HTML χρησιμοποιώντας το GroupDocs.Conversion.

#### Βήμα 1: Ορισμός διαδρομών εγγράφων

Προσδιορίστε τις διαδρομές εισόδου και εξόδου για τα έγγραφά σας, ώστε να διασφαλίσετε ότι τα αρχεία φορτώνονται και αποθηκεύονται σωστά:

```csharp
// Ορισμός διαδρομών εγγράφων
currentDirectory = @"YOUR_DOCUMENT_DIRECTORY";
externalDirectory = @"YOUR_OUTPUT_DIRECTORY\";

sourceOtgFilePath = Path.Combine(currentDirectory, "sample.otg");
outputHtmlFilePath = Path.Combine(externalDirectory, "otg-converted-to.html");
```

#### Βήμα 2: Φόρτωση του αρχείου OTG

Χρήση του GroupDocs.Conversion `Converter` τάξη, φορτώστε το αρχείο OTG σας:

```csharp
// Αρχικοποίηση μετατροπέα με τη διαδρομή αρχείου OTG πηγής
groupDocsConverter = new Converter(sourceOtgFilePath);
{
    // Προετοιμασία για μετατροπή χρησιμοποιώντας επιλογές HTML
    htmlConversionOptions = new WebConvertOptions();
    
    // Μετατροπή και αποθήκευση ως HTML
    groupDocsConverter.Convert(outputHtmlFilePath, htmlConversionOptions);
}
```

**Παράμετροι & Σκοποί Μεθόδου:**
- `Converter`: Φορτώνει το αρχείο προέλευσης για μετατροπή.
- `WebConvertOptions()`: Καθορίζει ότι η μορφή εξόδου θα πρέπει να είναι HTML.

#### Συμβουλές αντιμετώπισης προβλημάτων

Εάν αντιμετωπίσετε προβλήματα:
- Βεβαιωθείτε ότι όλες οι διαδρομές είναι σωστά ρυθμισμένες και προσβάσιμες.
- Επαληθεύστε ότι το GroupDocs.Conversion έχει εγκατασταθεί σωστά και έχει λάβει άδεια χρήσης.
