### Χαρακτηριστικό: Μετατροπή WEBP σε HTML

#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς να μετατρέψετε απρόσκοπτα ένα αρχείο WEBP σε έγγραφο HTML. Αυτό είναι ιδιαίτερα χρήσιμο για προγραμματιστές ιστοσελίδων που χρειάζονται να εμφανίζουν εικόνες μέσα σε ιστοσελίδες δυναμικά.

##### Βήμα 1: Φόρτωση του αρχείου WEBP προέλευσης
Φορτώστε το αρχείο WEBP πηγής χρησιμοποιώντας το `Converter` κλάση που παρέχεται από το GroupDocs.Conversion. Βεβαιωθείτε ότι έχετε καθορίσει τη σωστή διαδρομή προς την εικόνα σας.

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
using (var converter = new Converter(webpFilePath))
{
    // Συνέχεια με τη ρύθμιση των επιλογών μετατροπής
}
```

##### Βήμα 2: Δημιουργία επιλογών μετατροπής για μορφή HTML
Ρυθμίστε τις απαραίτητες επιλογές μετατροπής χρησιμοποιώντας `WebConvertOptions`Αυτή η κλάση σάς επιτρέπει να διαμορφώσετε διάφορες παραμέτρους προσαρμοσμένες για έξοδο HTML.

```csharp
var options = new WebConvertOptions();
```

##### Βήμα 3: Μετατροπή και αποθήκευση του αρχείου HTML
Εκτελέστε τη μετατροπή καλώντας το `Convert` μέθοδος στο δικό σας `Converter` παράδειγμα. Καθορίστε τόσο τη διαδρομή του αρχείου εξόδου όσο και τις επιλογές που διαμορφώθηκαν νωρίτερα.

```csharp
string outputFile = Path.Combine(outputFolder, "webp-converted-to.html");
converter.Convert(outputFile, options);
```
