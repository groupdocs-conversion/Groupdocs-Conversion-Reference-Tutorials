# Μετατροπή OneNote σε HTML με το GroupDocs.Conversion για .NET

## Εισαγωγή

Χρειάζεται να μοιραστείτε ένα αρχείο Microsoft OneNote, αλλά ο παραλήπτης σας δεν έχει πρόσβαση; Μετατρέψτε εύκολα `.one` αρχεία σε μορφή HTML χρησιμοποιώντας το GroupDocs.Conversion για .NET. Αυτή η μορφή είναι καθολικά ορατή σε προγράμματα περιήγησης ιστού, κάνοντας την κοινή χρήση απλή.

Σε αυτό το σεμινάριο, θα σας καθοδηγήσουμε στη μετατροπή εγγράφων OneNote σε HTML με το GroupDocs.Conversion for .NET. Στο τέλος, θα κατανοήσετε πώς να μετατρέψετε `.one` αρχεία σε HTML χρησιμοποιώντας C#. Δείτε τι θα μάθετε:

- Ρύθμιση του περιβάλλοντός σας με το GroupDocs.Conversion για .NET
- Βήμα προς βήμα μετατροπή ενός αρχείου OneNote σε HTML
- Βασικές επιλογές διαμόρφωσης και παράμετροι απόδοσης

Ας ξεκινήσουμε καλύπτοντας τις προαπαιτούμενες προϋποθέσεις.

## Προαπαιτούμενα

Πριν βουτήξετε, βεβαιωθείτε ότι έχετε τα εξής:

- **Βιβλιοθήκη GroupDocs.Conversion**Απαιτείται έκδοση 25.3.0 ή νεότερη.
- **Ρύθμιση περιβάλλοντος**Ένα περιβάλλον .NET (κατά προτίμηση .NET Core ή .NET Framework) εγκατεστημένο στον υπολογιστή σας.
- **Απαιτήσεις Γνώσεων**Βασική κατανόηση της C# και εξοικείωση με τη διαχείριση πακέτων NuGet.

### Ρύθμιση του GroupDocs.Conversion για .NET

Εγκαταστήστε τη βιβλιοθήκη GroupDocs.Conversion χρησιμοποιώντας είτε την Κονσόλα Διαχείρισης Πακέτων είτε το .NET CLI:

**Χρησιμοποιώντας την Κονσόλα Διαχείρισης Πακέτων NuGet:**

```bash
Install-Package GroupDocs.Conversion -Version 25.3.0
```

**Χρησιμοποιώντας το .NET CLI:**

```bash
dotnet add package GroupDocs.Conversion --version 25.3.0
```

Μετά την εγκατάσταση, αποκτήστε μια άδεια χρήσης εάν σκοπεύετε να τη χρησιμοποιήσετε πέραν της δοκιμαστικής περιόδου, λαμβάνοντας μια δωρεάν δοκιμαστική ή προσωρινή άδεια χρήσης από το GroupDocs.

Συμπεριλάβετε τον απαραίτητο χώρο ονομάτων στο έργο σας:

```csharp
using GroupDocs.Conversion;
```

## Οδηγός Εφαρμογής

### Φόρτωση του αρχείου προέλευσης OneNote

Αρχικά, φορτώστε το `.one` αρχείο χρησιμοποιώντας C#:

#### Βήμα 1: Ορισμός διαδρομών εγγράφων

Αντικαθιστώ `"YOUR_DOCUMENT_DIRECTORY"` και `"YOUR_OUTPUT_DIRECTORY"` με τις πραγματικές διαδρομές καταλόγου σας.

```csharp
string yourDocumentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string sampleOneFilePath = Path.Combine(yourDocumentDirectory, "sample.one");
```

#### Βήμα 2: Αρχικοποίηση του μετατροπέα

Φορτώστε το `.one` αρχείο χρησιμοποιώντας το GroupDocs.Conversion:

```csharp
using (var converter = new Converter(sampleOneFilePath))
{
    // Η λογική μετατροπής θα εφαρμοστεί εδώ.
}
```

### Μετατροπή του OneNote σε HTML

Αφού φορτώσετε το αρχείο OneNote, προχωρήστε στη μετατροπή του σε HTML.

#### Βήμα 3: Ρύθμιση παραμέτρων του WebConvertOptions

Καθορίστε τις επιλογές μετατροπής για την έξοδο HTML:

```csharp
var options = new WebConvertOptions();
```

#### Βήμα 4: Ορισμός διαδρομής εξόδου και μετατροπή

Βεβαιωθείτε ότι ο κατάλογος εξόδου υπάρχει και, στη συνέχεια, αποθηκεύστε το αρχείο που έχει μετατραπεί:

```csharp
string yourOutputDirectory = "YOUR_OUTPUT_DIRECTORY";
if (!Directory.Exists(yourOutputDirectory))
{
    Directory.CreateDirectory(yourOutputDirectory);
}
string outputFile = Path.Combine(yourOutputDirectory, "one-converted-to.html");

// Εκτέλεση μετατροπής
converter.Convert(outputFile, options);
```
