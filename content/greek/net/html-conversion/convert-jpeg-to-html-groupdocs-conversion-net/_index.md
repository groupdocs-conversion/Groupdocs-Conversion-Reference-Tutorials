#### Βασική Αρχικοποίηση και Ρύθμιση
Δείτε πώς μπορείτε να αρχικοποιήσετε και να ρυθμίσετε το GroupDocs.Conversion στο έργο σας C#:

```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main(string[] args)
    {
        // Αρχικοποιήστε τον μετατροπέα με μια διαδρομή αρχείου JPEG
        using (var converter = new Converter("input.jpg"))
        {
            // Μετατροπή σε HTML και αποθήκευση εξόδου
            var options = new MarkupConvertOptions();
            converter.Convert("output.html", options);
        }
        
        Console.WriteLine("Conversion completed successfully!");
    }
}
```

Σε αυτό το απόσπασμα κώδικα, αρχικοποιούμε το `Converter` κλάση με μια διαδρομή προς το αρχείο JPEG σας. Η μετατροπή εκτελείται στη συνέχεια χρησιμοποιώντας `MarkupConvertOptions`και το αποτέλεσμα αποθηκεύεται ως αρχείο HTML.
