## Οδηγός Εφαρμογής

### Μετατροπή VSTM σε HTML

Το κύριο χαρακτηριστικό αυτού του οδηγού είναι η μετατροπή ενός αρχείου VSTM σε HTML. Δείτε πώς μπορείτε να το πετύχετε αυτό:

#### Βήμα 1: Ορισμός διαδρομών προέλευσης και εξόδου
Ξεκινήστε καθορίζοντας την πηγή σας `.vstm` βρίσκεται το αρχείο και πού θα αποθηκευτεί το HTML που έχει μετατραπεί.
```csharp
string documentPath = Path.Combine(@"YOUR_DOCUMENT_DIRECTORY", "sample.vstm");
string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "vstm-converted-to.html");
```

#### Βήμα 2: Αρχικοποίηση μετατροπέα
Δημιουργήστε μια παρουσία του `Converter` κλάση με τη διαδρομή αρχείου VSTM. Αυτό το βήμα είναι απαραίτητο για τη φόρτωση και την προετοιμασία του αρχείου για μετατροπή.
```csharp
using (var converter = new Converter(documentPath))
{
    // Η λογική μετατροπής θα ακολουθηθεί εδώ.
}
```

#### Βήμα 3: Ρύθμιση επιλογών μετατροπής HTML
Χρησιμοποιήστε το `WebConvertOptions` κλάση για να καθορίσετε ότι η μορφή εξόδου σας θα πρέπει να είναι HTML. Αυτή η επιλογή διασφαλίζει τον σωστό χειρισμό μορφών συμβατών με το web.
```csharp
var options = new WebConvertOptions();
```

#### Βήμα 4: Εκτελέστε τη μετατροπή
Επικαλέστε το `Convert` μέθοδο, περνώντας τη διαδρομή για το αρχείο εξόδου και τις επιλογές μετατροπής. Αυτό θα αποθηκεύσει το HTML που έχει μετατραπεί στην καθορισμένη τοποθεσία σας.
```csharp
converter.Convert(outputFile, options);
```
