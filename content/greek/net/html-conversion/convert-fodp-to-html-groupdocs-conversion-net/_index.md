## Οδηγός Εφαρμογής

Τώρα, ας αναλύσουμε τη διαδικασία μετατροπής σε λογικά βήματα:

### Χαρακτηριστικό: Μετατροπή FODP σε HTML

#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς να μετατρέψετε ένα αρχείο .fodp σε μορφή HTML χρησιμοποιώντας το GroupDocs.Conversion για .NET.

##### Βήμα 1: Φόρτωση του εγγράφου

```csharp
class Program
{
    static void Main()
    {
        string inputPath = "path/to/your/fodpfile.fodp";
        Converter converter = new Converter(inputPath);
        
        Console.WriteLine("Document loaded successfully.");
    }
}
```

Ο `Converter` Η κλάση χειρίζεται τη φόρτωση των εγγράφων. Η διαδρομή εισόδου καθορίζει πού βρίσκεται το αρχείο προέλευσης.

##### Βήμα 2: Ορισμός επιλογών μετατροπής

```csharp
class Program
{
    static void Main()
    {
        string inputPath = "path/to/your/fodpfile.fodp";
        Converter converter = new Converter(inputPath);
        
        var options = new MarkupConvertOptions();
        
        Console.WriteLine("Conversion options set for HTML format.");
    }
}
```

Εδώ, `MarkupConvertOptions` ρυθμίζει τη μετατροπή σε έξοδο HTML προορισμού.

##### Βήμα 3: Εκτελέστε τη μετατροπή

```csharp
class Program
{
    static void Main()
    {
        string inputPath = "path/to/your/fodpfile.fodp";
        Converter converter = new Converter(inputPath);
        var options = new MarkupConvertOptions();
        
        string outputPath = Path.Combine("output/directory", "output.html");
        converter.Convert(outputPath, options);
        
        Console.WriteLine($"Document converted and saved to {outputPath}.");
    }
}
```

Ο `Convert` η μέθοδος εκτελεί τη διαδικασία μετατροπής. Βεβαιωθείτε ότι το `outputPath` έχει ρυθμιστεί σωστά στο σημείο όπου θέλετε να αποθηκευτεί το αρχείο που έχει μετατραπεί.
