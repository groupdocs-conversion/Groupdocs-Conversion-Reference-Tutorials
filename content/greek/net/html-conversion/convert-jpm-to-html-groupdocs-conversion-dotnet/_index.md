## Ρύθμιση του GroupDocs.Conversion για .NET

Για να ξεκινήσετε να χρησιμοποιείτε το GroupDocs.Conversion, θα πρέπει να το εγκαταστήσετε μέσω του NuGet. Δείτε πώς:

### **Κονσόλα διαχείρισης πακέτων NuGet**
```bash
Install-Package GroupDocs.Conversion -Version 25.3.0
```

### **.NET CLI**
```bash
dotnet add package GroupDocs.Conversion --version 25.3.0
```

### Βήματα Απόκτησης Άδειας Χρήσης:
- Για μια δωρεάν δοκιμαστική περίοδο, κατεβάστε την τελευταία έκδοση από [Επίσημη ιστοσελίδα του GroupDocs](https://releases.groupdocs.com/conversion/net/).
- Για να αποκτήσετε μια προσωρινή άδεια χρήσης για πλήρη πρόσβαση σε λειτουργίες χωρίς περιορισμούς αξιολόγησης, επισκεφθείτε την ιστοσελίδα [Σελίδα Προσωρινής Άδειας Χρήσης](https://purchase.groupdocs.com/temporary-license/).
- Σκεφτείτε το ενδεχόμενο αγοράς εάν το έργο σας απαιτεί μακροχρόνια χρήση με επαγγελματική υποστήριξη.

### Βασική Αρχικοποίηση και Ρύθμιση
Δείτε πώς μπορείτε να αρχικοποιήσετε το GroupDocs.Conversion στην εφαρμογή C# που χρησιμοποιείτε:

```csharp
using GroupDocs.Conversion;
```

Ξεκινήστε δημιουργώντας ένα `Converter` αντικείμενο για εργασίες μετατροπής αρχείων. Εδώ θα καθορίσετε τη διαδρομή προς το έγγραφο JPM:

```csharp
string documentPath = "path/to/your/sample.jpm";
var converter = new Converter(documentPath);
```

Με αυτήν τη ρύθμιση, είστε έτοιμοι να εφαρμόσετε λειτουργίες μετατροπής αρχείων.
