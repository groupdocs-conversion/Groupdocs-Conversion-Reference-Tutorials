## Οδηγός Εφαρμογής

### Δυνατότητα μετατροπής ODG σε HTML

Αυτή η λειτουργία επιτρέπει τη μετατροπή αρχείων σχεδίασης OpenDocument σε μορφή HTML, διευκολύνοντας την εύκολη ενσωμάτωση στο web.

#### Βήμα 1: Αρχικοποίηση του μετατροπέα

Δημιουργήστε ένα `Converter` αντικείμενο με το αρχείο ODG πηγής σας:

```csharp
using GroupDocs.Conversion;
// ...

Converter converter = new Converter("source-file.odg");
```

**Γιατί;** Αυτό το βήμα καθορίζει το περιβάλλον μετατροπής καθορίζοντας το έγγραφο εισόδου.

#### Βήμα 2: Ορισμός επιλογών μετατροπής

Ορίστε επιλογές μετατροπής HTML χρησιμοποιώντας `HtmlConvertOptions`:

```csharp
using GroupDocs.Conversion.Options.Convert;

HtmlConvertOptions options = new HtmlConvertOptions();
options.FixedLayout = true; // Διατηρεί τη διάταξη όσο το δυνατόν περισσότερο
```

**Γιατί;** Αυτές οι επιλογές επιτρέπουν την προσαρμογή της μετατροπής ODG σε HTML.

#### Βήμα 3: Εκτέλεση μετατροπής

Εκτελέστε τη μετατροπή και αποθηκεύστε την έξοδο:

```csharp
string outputPath = "output-file.html";
converter.Convert(outputPath, options);
Console.WriteLine("Conversion completed.");
```

**Γιατί;** Αυτό το βήμα εκτελεί την πραγματική διαδικασία μετατροπής και αποθηκεύει το αποτέλεσμα στην καθορισμένη διαδρομή σας.
