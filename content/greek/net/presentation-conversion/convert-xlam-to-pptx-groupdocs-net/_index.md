## Οδηγός Εφαρμογής

Αυτή η ενότητα σας καθοδηγεί στη μετατροπή ενός αρχείου XLAM σε μορφή PPTX, εστιάζοντας σε κάθε βήμα της διαδικασίας.

### Φόρτωση και μετατροπή του εγγράφου

#### Επισκόπηση
Η μετατροπή εγγράφων περιλαμβάνει τη φόρτωση του αρχείου προέλευσης και τον καθορισμό επιλογών μετατροπής για να το μετατρέψετε στην επιθυμητή μορφή.

#### Βήμα προς βήμα εφαρμογή

**1. Ορισμός διαδρομών αρχείων**
Ξεκινήστε ορίζοντας διαδρομές για τα αρχεία εισόδου και εξόδου:

```csharp
string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY/";

// Εισαγωγή αρχείου XLAM
string inputFile = Path.Combine(documentDirectory, "sample.xlam");
// Έξοδος αρχείου PPTX
string outputFile = Path.Combine(outputDirectory, "xlam-converted-to.pptx");
```

**2. Αρχικοποίηση του μετατροπέα**
Τοποθετήστε το XLAM σας χρησιμοποιώντας το `Converter` κλάση και καθορίστε επιλογές για μετατροπή στο PowerPoint.

```csharp
using (var converter = new Converter(inputFile))
{
    var options = new PresentationConvertOptions(); // Ρύθμιση παραμέτρων επιλογών μετατροπής για PPTX
    converter.Convert(outputFile, options); // Εκτέλεση μετατροπής και αποθήκευση εξόδου
}
```

**Επεξήγηση παραμέτρων:**
- `inputFile`: Διαδρομή προς το αρχείο XLAM.
- `outputFile`: Διαδρομή προορισμού για το αρχείο PPTX που έχει μετατραπεί.
- `PresentationConvertOptions()`: Ορίζει τις προεπιλεγμένες επιλογές για τη μετατροπή του PowerPoint.
