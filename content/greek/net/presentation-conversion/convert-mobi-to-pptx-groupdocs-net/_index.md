### Επισκόπηση λειτουργιών: Μετατροπή MOBI σε PPTX

Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε ένα ηλεκτρονικό βιβλίο σε μορφή MOBI σε ένα αρχείο PowerPoint (PPTX) έτοιμο για παρουσίαση.

#### Βήμα 1: Ορισμός διαδρομών εισόδου και εξόδου

```csharp

string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
```

- **Σκοπός**Αυτές οι διαδρομές καθορίζουν πού βρίσκεται το αρχείο MOBI πηγής σας και πού θα αποθηκευτεί το PPTX που έχει μετατραπεί.

#### Βήμα 2: Φόρτωση του αρχείου MOBI προέλευσης

```csharp
using (var converter = new Converter(Path.Combine(documentDirectory, "sample.mobi")))
{
    // Η διαδικασία μετατροπής ξεκινάει εδώ
}
```

- **Γιατί αυτό έχει σημασία**: Φορτώνοντας το αρχείο προέλευσης στο `Converter` τάξη, την προετοιμάζετε για μετατροπή.

#### Βήμα 3: Ρύθμιση επιλογών μετατροπής

```csharp
var options = new PresentationConvertOptions();
```

- **Εξήγηση**: Το `PresentationConvertOptions` ρυθμίζει τον μετατροπέα για την έξοδο ενός αρχείου PowerPoint.

#### Βήμα 4: Εκτέλεση μετατροπής και αποθήκευση εξόδου

```csharp
string outputFile = Path.Combine(outputDirectory, "mobi-converted-to.pptx");
converter.Convert(outputFile, options);
```

- **Λεπτομέρειες μετατροπής**: Αυτό το βήμα εκτελεί τη διαδικασία μετατροπής και αποθηκεύει το αρχείο PPTX που προκύπτει στον καθορισμένο κατάλογο.

**Συμβουλές αντιμετώπισης προβλημάτων:**
- Βεβαιωθείτε ότι οι διαδρομές έχουν οριστεί σωστά για να αποφύγετε `FileNotFoundException`.
- Επαληθεύστε ότι έχετε δικαιώματα εγγραφής για τον κατάλογο εξόδου.
