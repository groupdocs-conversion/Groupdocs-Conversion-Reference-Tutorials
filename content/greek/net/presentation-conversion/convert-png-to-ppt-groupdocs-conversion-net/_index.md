## Οδηγός Εφαρμογής

Τώρα, ας εξερευνήσουμε πώς να μετατρέψετε μια εικόνα PNG σε μορφή PPT βήμα προς βήμα.

### Χαρακτηριστικό: Μετατροπή PNG σε PPT

**Επισκόπηση**Αυτή η λειτουργία επιτρέπει τη μετατροπή αρχείων PNG σε παρουσιάσεις PowerPoint, διατηρώντας τα οπτικά σας στοιχεία σε δυναμική μορφή.

#### Βήμα 1: Ορισμός διαδρομών εισόδου και εξόδου

Καθορίστε διαδρομές για το αρχείο PNG εισόδου και τον επιθυμητό κατάλογο εξόδου:

```csharp
string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.png"); // Αντικατάσταση με την πραγματική διαδρομή
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", ".\" );
string outputFile = Path.Combine(outputFolder, "png-converted-to.ppt");
```

**Εξήγηση**: Το `inputFilePath` θα πρέπει να δείχνει στο αρχείο PNG για μετατροπή. Προσαρμόστε τις διαδρομές όπως απαιτείται.

#### Βήμα 2: Φόρτωση και μετατροπή του αρχείου PNG

Φορτώστε το PNG πηγής χρησιμοποιώντας το GroupDocs.Conversion's `Converter` κλάσης και, στη συνέχεια, διαμορφώστε τις επιλογές για τη μορφή PowerPoint:

```csharp
using (var converter = new Converter(inputFilePath))
{
    PresentationConvertOptions options = new PresentationConvertOptions 
    {
        Format = PresentationFileType.Ppt // Μορφή PPT προορισμού
    };
    
    // Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο εξόδου
    converter.Convert(outputFile, options);
}
```

**Εξήγηση**: Το `Converter` η κλάση φορτώνει το PNG σας. Ορισμός του `Format` να `PresentationFileType.Ppt` Καθορίζει μια παρουσίαση PowerPoint ως τη μορφή στόχου.
