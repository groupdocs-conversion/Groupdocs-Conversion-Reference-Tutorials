### Βασική Αρχικοποίηση και Ρύθμιση
Αρχικοποιήστε το GroupDocs.Conversion στην εφαρμογή .NET σας με τα ακόλουθα βήματα:

```csharp
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Load;

// Αρχικοποιήστε τον μετατροπέα με ένα αρχείο PST προέλευσης.
string sourceFilePath = "path/to/your/sample.pst";
var loadOptions = new LoadOptions();

if (Path.GetExtension(sourceFilePath).Equals(".ost", StringComparison.OrdinalIgnoreCase)) {
    loadOptions = new PersonalStorageLoadOptions();
}

using (var converter = new GroupDocs.Conversion.Converter(sourceFilePath, loadOptions)) {
    // Η λογική μετατροπής θα εφαρμοστεί εδώ.
}
```

## Οδηγός Εφαρμογής

### Φόρτωση αρχείων PST
#### Επισκόπηση
Η φόρτωση ενός αρχείου PST είναι το αρχικό βήμα στη διαδικασία μετατροπής. Αυτή η ενότητα σας καθοδηγεί στην προετοιμασία των αρχείων σας για μετατροπή.

#### Βήμα προς βήμα εφαρμογή
**1. Προσδιορίστε τον τύπο αρχείου**
   - Προσδιορίστε εάν το αρχείο προέλευσης είναι OST ή PST και ορίστε τις επιλογές φόρτωσης ανάλογα.

```csharp
string sourceFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.pst");
var converterOptions = new LoadOptions();

if (Path.GetExtension(sourceFilePath).Equals(".ost", StringComparison.OrdinalIgnoreCase)) {
    // Χρησιμοποιήστε συγκεκριμένες διαμορφώσεις για αρχεία OST.
    converterOptions = new PersonalStorageLoadOptions();
}
```

**2. Αρχικοποίηση μετατροπέα**
   - Δημιουργήστε ένα `Converter` παράδειγμα χρησιμοποιώντας τη διαδρομή του αρχείου προέλευσης και τις επιλογές φόρτωσης.

```csharp
using (var converter = new GroupDocs.Conversion.Converter(sourceFilePath, converterOptions)) {
    // Έτοιμο για εκτέλεση μετατροπής.
}
```

### Μετατροπή PST σε PPTX
#### Επισκόπηση
Μετά τη φόρτωση, μετατρέψτε τα αρχεία PST σε ευπαρουσίαστη μορφή PPTX.

#### Βήμα προς βήμα εφαρμογή
**1. Ορίστε επιλογές μετατροπής**
   - Ορίστε τον κατάλογο εξόδου και το μοτίβο ονομασίας αρχείων.

```csharp
string outputDirectory = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFilePattern = Path.Combine(outputDirectory, "pst-converted-{0}-to.pptx");
var options = new PresentationConvertOptions();
int counter = 1;
```

**2. Εκτελέστε μετατροπή**
   - Χρησιμοποιήστε το `Converter` παράδειγμα για να μετατρέψετε και να αποθηκεύσετε το αρχείο.

```csharp
using (var converter = new GroupDocs.Conversion.Converter(sourceFilePath)) {
    converter.Convert(
        savePath => new FileStream(string.Format(outputFilePattern, counter++), FileMode.Create),
        options
    );
}
```
