## Οδηγός Εφαρμογής

Αφού ορίσετε το περιβάλλον σας, ακολουθήστε αυτά τα βήματα για να μετατρέψετε ένα αρχείο JPEG σε παρουσίαση PowerPoint.

### Βήμα 1: Φόρτωση του αρχείου προέλευσης

Καθορίστε τη διαδρομή της εικόνας JPEG. Αντικαταστήστε `@YOUR_DOCUMENT_DIRECTORY` με τον πραγματικό κατάλογο όπου βρίσκεται το αρχείο σας:

```csharp
string inputFilePath = "C:\\\\Your\\\\Document\\\\Directory\\\\sample.jpeg";
```

### Βήμα 2: Ορισμός διαμόρφωσης εξόδου

Ρυθμίστε τον φάκελο εξόδου και ορίστε το όνομα του αρχείου PowerPoint που προκύπτει:

```csharp
string outputFolder = "C:\\\\Your\\\\Output\\\\Directory";
string outputFile = System.IO.Path.Combine(outputFolder, "jpeg-converted-to.ppt");
```

### Βήμα 3: Εκτελέστε τη μετατροπή

Χρησιμοποιήστε το GroupDocs.Conversion για να μετατρέψετε το JPEG σας σε μορφή PPT:

```csharp
using (var converter = new Converter(inputFilePath))
{
    var options = new PresentationConvertOptions { Format = PresentationFileType.Ppt };
    converter.Convert(outputFile, options);
}
```

#### Εξήγηση
- **Κλάση μετατροπέα**: Χειρίζεται τη φόρτωση και τη μετατροπή αρχείων.
- **PresentationConvertOptions**: Καθορίζει ότι η μορφή εξόδου είναι PPT.
- **Μέθοδος μετατροπής**: Εκτελεί τη διαδικασία μετατροπής.
