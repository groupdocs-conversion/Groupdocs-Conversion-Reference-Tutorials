## Οδηγός Εφαρμογής

### Μετατροπή EML σε PPT

Τώρα, ας επικεντρωθούμε στην βασική λειτουργικότητα: τη μετατροπή ενός αρχείου EML σε παρουσίαση PowerPoint.

#### Βήμα 1: Φόρτωση του αρχείου EML

Αρχικά, χρησιμοποιήστε το GroupDocs.Conversion's `Converter` κλάση για να φορτώσετε το αρχείο EML πηγής σας. Αυτό ξεκινά τη διαδικασία μετατροπής:

```csharp
using (var converter = new Converter(inputFile))
{
    // Η ρύθμιση της μετατροπής ολοκληρώθηκε.
}
```

#### Βήμα 2: Καθορισμός επιλογών μετατροπής

Στη συνέχεια, ορίστε τη μορφή στην οποία θέλετε να μετατρέψετε χρησιμοποιώντας `PresentationConvertOptions`:

```csharp
PresentationConvertOptions options = new PresentationConvertOptions { Format = PresentationFileType.Ppt };
```

Αυτό καθορίζει ότι το αρχείο EML σας θα μετατραπεί σε αρχείο PPT.

#### Βήμα 3: Εκτέλεση μετατροπής και αποθήκευση

Τέλος, εκτελέστε τη μετατροπή και αποθηκεύστε την έξοδο στην επιθυμητή θέση:

```csharp
string outputFile = "YOUR_OUTPUT_DIRECTORY/eml-converted-to.ppt";
converter.Convert(outputFile, options);
```
