## Οδηγός Εφαρμογής

### Χαρακτηριστικό 1: Μετατροπή PPSX σε DOC

Μάθετε πώς να μετατρέψετε ένα αρχείο PowerPoint Open XML Slide Show (.ppsx) σε ένα έγγραφο του Microsoft Word (.doc).

#### Βήμα 1: Ορίστε τις διαδρομές σας
Ορίστε διαδρομές για τα αρχεία εισόδου και εξόδου.
```csharp
string documentPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.ppsx");
string outputFolder = GetOutputDirectoryPath();
string outputFile = Path.Combine(outputFolder, "ppsx-converted-to.doc");
```

#### Βήμα 2: Φόρτωση και μετατροπή του αρχείου PPSX
Χρησιμοποιήστε το GroupDocs.Conversion για να φορτώσετε το αρχείο PPSX και να καθορίσετε επιλογές μετατροπής για μορφές επεξεργασίας κειμένου.
```csharp
using (var converter = new Converter(documentPath))
{
    // Καθορίστε επιλογές μετατροπής για τη μορφή DOC.
    var options = new WordProcessingConvertOptions
    {
        Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc
    };
    
    // Μετατρέψτε και αποθηκεύστε το αρχείο PPSX ως αρχείο σε μορφή DOC.
    converter.Convert(outputFile, options);
}
```

### Χαρακτηριστικό 2: Ορισμός διαδρομής καταλόγου εξόδου
Διαχειριστείτε δυναμικά τις διαδρομές εξόδου ορίζοντας μια συνάρτηση για τον χειρισμό της δημιουργίας καταλόγου.
```csharp
string GetOutputDirectoryPath()
{
    string baseOutputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
    
    // Δημιουργήστε τον κατάλογο εάν δεν υπάρχει.
    if (!Directory.Exists(baseOutputFolder))
    {
        Directory.CreateDirectory(baseOutputFolder);
    }
    
    return baseOutputFolder;
}
```
