### Βασική Αρχικοποίηση και Ρύθμιση
Δείτε πώς μπορείτε να αρχικοποιήσετε το GroupDocs.Conversion στην εφαρμογή C# που χρησιμοποιείτε:

```csharp
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main(string[] args)
    {
        // Ορίστε τις διαδρομές για το αρχείο TIFF εισόδου και το αρχείο PPTX εξόδου
        string inputFile = @"path_to_your_tiff_file.tif";
        string outputFile = @"output_path\presentation.pptx";

        using (Converter converter = new Converter(inputFile))
        {
            var options = new PresentationConvertOptions();
            // Μετατροπή και αποθήκευση της παρουσίασης που έχει μετατραπεί
            converter.Convert(outputFile, options);
        }
    }
}
```
