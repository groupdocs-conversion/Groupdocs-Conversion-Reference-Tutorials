## Οδηγός Εφαρμογής

### Μετατροπή POTM σε PPTX
Ακολουθήστε αυτά τα βήματα για να μετατρέψετε ένα αρχείο POTM σε μορφή PPTX:

#### Βήμα 1: Ορισμός διαδρομών αρχείων και φόρτωση αρχείου προέλευσης
Καθορίστε διαδρομές για το αρχείο προέλευσης .potm και την τοποθεσία εξόδου .pptx:
```csharp
using System;
using System.IO;
using GroupDocs.Conversion;

string documentPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.potm");
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "potm-converted-to.pptx");
```
#### Βήμα 2: Διαμόρφωση επιλογών μετατροπής
Ορίστε τις επιλογές μετατροπής για να καθορίσετε τη μορφή PPTX:
```csharp
using GroupDocs.Conversion.Options.Convert;

var converter = new Converter(documentPath);
var options = new PresentationConvertOptions();
```
#### Βήμα 3: Εκτελέστε τη μετατροπή
Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο .pptx που προκύπτει:
```csharp
converter.Convert(outputFile, options);

Console.WriteLine("Conversion completed successfully!");
```
