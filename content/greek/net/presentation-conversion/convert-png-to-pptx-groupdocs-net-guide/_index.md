### Χαρακτηριστικό: Μετατροπή PNG σε PPTX

Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε ένα αρχείο PNG σε μορφή PPTX χωρίς κόπο. 

#### Βήμα 1: Ρύθμιση καταλόγων

Αρχικά, βεβαιωθείτε ότι οι κατάλογοι εισόδου και εξόδου έχουν ρυθμιστεί σωστά:
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// Βεβαιωθείτε ότι ο κατάλογος εξόδου υπάρχει ή δημιουργήστε τον εάν δεν υπάρχει
if (!Directory.Exists(outputDirectory))
{
    Directory.CreateDirectory(outputDirectory);
}
```

#### Βήμα 2: Φόρτωση και μετατροπή του αρχείου

Δείτε πώς μπορείτε να φορτώσετε ένα αρχείο PNG και να το μετατρέψετε σε PPTX:
```csharp
string sourceFile = Path.Combine(documentDirectory, "sample.png");
string outputFile = Path.Combine(outputDirectory, "png-converted-to.pptx");

// Φορτώστε το αρχείο PNG πηγής χρησιμοποιώντας το GroupDocs.Conversion
using (var converter = new Converter(sourceFile))
{
    var options = new PresentationConvertOptions(); // Ορισμός επιλογών μετατροπής για τη μορφή PPTX
    
    // Εκτελέστε τη μετατροπή και αποθηκεύστε το αποτέλεσμα σε ένα καθορισμένο αρχείο εξόδου
    converter.Convert(outputFile, options);
}
```

**Εξήγηση:**
- **`Converter`:** Αρχικοποιείται με το αρχείο PNG πηγής σας.
- **`PresentationConvertOptions`:** Καθορίζει ότι η μορφή προορισμού είναι PPTX.
- **`converter.Convert()`:** Εκτελεί τη μετατροπή και αποθηκεύει το αποτέλεσμα.

#### Συμβουλές αντιμετώπισης προβλημάτων

- Βεβαιωθείτε ότι η διαδρομή εισόδου του αρχείου PNG είναι σωστή για να αποφύγετε `FileNotFoundException`.
- Επαληθεύστε τα δικαιώματα εγγραφής για τον κατάλογο εξόδου για να αποτρέψετε `UnauthorizedAccessException`.
