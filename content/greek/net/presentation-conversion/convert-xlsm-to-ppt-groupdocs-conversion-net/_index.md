## Οδηγός Εφαρμογής

### Φόρτωση αρχείου προέλευσης

**Επισκόπηση:** Ξεκινήστε φορτώνοντας ένα αρχείο XLSM για μετατροπή. Αυτό το βήμα είναι κρίσιμο, καθώς προετοιμάζει το αρχείο για μετατροπή σε μορφή PPT.

#### Βήμα 1: Αρχικοποίηση μετατροπέα
```csharp
using System;
using GroupDocs.Conversion;

string documentPath = "YOUR_DOCUMENT_DIRECTORY/sample.xlsm";

// Φορτώστε το αρχείο XLSM πηγής χρησιμοποιώντας την κλάση Converter
class Program
{
    static void Main(string[] args)
    {
        using (var converter = new Converter(documentPath))
        {
            Console.WriteLine("XLSM file loaded successfully.");
        }
    }
}
```
- **Επεξήγηση παραμέτρων:** `documentPath` είναι μια μεταβλητή συμβολοσειράς που περιέχει τη διαδρομή προς το αρχείο XLSM σας. Το `Converter` Το αντικείμενο χειρίζεται λειτουργίες φόρτωσης και μετατροπής.

### Ρύθμιση επιλογών μετατροπής

**Επισκόπηση:** Διαμορφώστε τις ρυθμίσεις για τη μετατροπή ενός αρχείου XLSM σε μορφή PowerPoint καθορίζοντας επιλογές προορισμού.

#### Βήμα 2: Ορισμός επιλογών μετατροπής παρουσίασης
```csharp
using GroupDocs.Conversion.Options.Convert;

// Δημιουργήστε μια παρουσία του PresentationConvertOptions που καθορίζει τη μορφή PPT
class Program
{
    static void Main(string[] args)
    {
        PresentationConvertOptions options = new PresentationConvertOptions 
        { 
            Format = GroupDocs.Conversion.FileTypes.PresentationFileType.Ppt 
        };
    }
}
```
- **Διαμόρφωση κλειδιού:** Ο `Format` ιδιοκτησία εντός `PresentationConvertOptions` έχει οριστεί σε `Ppt`, που υποδεικνύει τον στόχο μετατροπής.

### Εκτέλεση μετατροπής και αποθήκευσης εξόδου

**Επισκόπηση:** Εκτελέστε τη διαδικασία μετατροπής, μετατρέποντας το αρχείο XLSM σε μια παρουσίαση PPT και αποθηκεύστε το σε έναν καθορισμένο κατάλογο.

#### Βήμα 3: Μετατροπή και αποθήκευση
```csharp
using System.IO;
using GroupDocs.Conversion;

class Program
{
    static void Main(string[] args)
    {
        string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
        string outputFile = Path.Combine(outputDirectory, "xlsm-converted-to.ppt");
        
        // Φόρτωση του αρχείου XLSM για μετατροπή
class Program
{
    static void Main(string[] args)
    {
        using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY/sample.xlsm"))
        {
            // Εφαρμόστε τις επιλογές μετατροπής και αποθηκεύστε το αρχείο PPT που προκύπτει
            converter.Convert(outputFile, options);
            
            Console.WriteLine($"Conversion completed. File saved at {outputFile}");
        }
    }
}
```
- **Συμβουλές αντιμετώπισης προβλημάτων:** Βεβαιωθείτε ότι οι διαδρομές έχουν οριστεί σωστά για να αποφύγετε `FileNotFoundException`Επαληθεύστε τα δικαιώματα εγγραφής για τον κατάλογο εξόδου.
