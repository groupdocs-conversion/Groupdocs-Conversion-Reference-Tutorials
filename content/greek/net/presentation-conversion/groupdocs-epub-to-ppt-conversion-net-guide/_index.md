## Ρύθμιση του GroupDocs.Conversion για .NET

Μόλις εγκατασταθούν οι απαραίτητες εξαρτήσεις, ας αρχικοποιήσουμε και ας ρυθμίσουμε τη βιβλιοθήκη GroupDocs.Conversion στο έργο .NET σας.

### Βασική Αρχικοποίηση και Ρύθμιση
Δείτε πώς μπορείτε να ξεκινήσετε να χρησιμοποιείτε C#:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

namespace EpubToPptConversion
{
    class Program
    {
        static void Main(string[] args)
        {
            // Ορίστε τη διαδρομή για το αρχείο EPUB και τον κατάλογο εξόδου.
            string inputFilePath = @"YOUR_DOCUMENT_DIRECTORY\\sample.epub";
            string outputFolder = Path.Combine(@"YOUR_OUTPUT_DIRECTORY");

            // Αρχικοποιήστε το αντικείμενο Converter με το αρχείο εισόδου.
            using (Converter converter = new Converter(inputFilePath))
            {
                // Ορίστε επιλογές μετατροπής για τη μορφή παρουσίασης PowerPoint.
                var convertOptions = new PresentationConvertOptions();

                // Μετατρέψτε και αποθηκεύστε το EPUB ως αρχείο PPT.
                string outputFilePath = Path.Combine(outputFolder, "output.pptx");
                converter.Convert(outputFilePath, convertOptions);
            }

            Console.WriteLine("Conversion completed successfully.");
        }
    }
}
```

Σε αυτήν τη ρύθμιση:
- Ορίζουμε διαδρομές για αρχεία εισόδου (EPUB) και εξόδου (PPT).
- Αρχικοποίηση ενός `Converter` αντικείμενο με το αρχείο EPUB.
- Ρύθμιση επιλογών μετατροπής για τη μορφή PowerPoint χρησιμοποιώντας `PresentationConvertOptions`.
- Μετατρέψτε και αποθηκεύστε το αρχείο σε μορφή PPTX.
