### Βασική Αρχικοποίηση και Ρύθμιση

Δείτε πώς μπορείτε να αρχικοποιήσετε το GroupDocs.Conversion στο έργο .NET σας:
```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

// Ορίστε τη διαδρομή του αρχείου προέλευσης PowerPoint και τον κατάλογο εξόδου
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

if (!Directory.Exists(outputDirectory))
{
    Directory.CreateDirectory(outputDirectory);
}

string sourceFilePath = Path.Combine(documentDirectory, "sample.ppt");
string outputFile = Path.Combine(outputDirectory, "ppt-converted-to.docx");

// Αρχικοποίηση της κλάσης Converter
using (var converter = new Converter(sourceFilePath))
{
    // Καθορισμός επιλογών μετατροπής για τη μορφή επεξεργασίας Word
    var options = new WordProcessingConvertOptions();
    
    // Μετατρέψτε και αποθηκεύστε το αρχείο DOCX εξόδου
    converter.Convert(outputFile, options);
}
```

**4. Εκτέλεση της μετατροπής**
