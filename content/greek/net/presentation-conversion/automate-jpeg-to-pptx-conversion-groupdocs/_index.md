## Οδηγός Εφαρμογής

### Μετατροπή JPEG σε PPTX

**Επισκόπηση**

Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε απρόσκοπτα μια εικόνα JPEG σε παρουσίαση PowerPoint, ιδανική για την ενσωμάτωση οπτικών στοιχείων στις παρουσιάσεις σας.

#### Βήμα 1: Ορισμός διαδρομών αρχείων

Ξεκινήστε καθορίζοντας πού θα βρίσκονται τα αρχεία εισόδου και εξόδου:

```csharp
string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY";

string sourceFilePath = Path.Combine(documentDirectory, "sample.jpeg");
string targetFilePath = Path.Combine(outputDirectory, "jpeg-converted-to.pptx");
```

Εδώ, αντικαταστήστε `"YOUR_DOCUMENT_DIRECTORY"` και `"YOUR_OUTPUT_DIRECTORY"` με τους πραγματικούς καταλόγους σας. Βεβαιωθείτε ότι `sample.jpeg` υπάρχει στον καθορισμένο κατάλογο εγγράφων.

#### Βήμα 2: Αρχικοποίηση του μετατροπέα

Στη συνέχεια, αρχικοποιήστε το `Converter` αντικείμενο:

```csharp
using (var converter = new GroupDocs.Conversion.Converter(sourceFilePath))
{
    // Η λογική μετατροπής θα προστεθεί εδώ.
}
```

Αυτό το μπλοκ κώδικα δημιουργεί ένα πλαίσιο για τη διαδικασία μετατροπής, ενσωματώνοντάς την σε ένα `using` δήλωση για να διασφαλιστεί η ορθή διάθεση των πόρων.

#### Βήμα 3: Ρύθμιση επιλογών μετατροπής

Ορίστε πώς θέλετε να μετατραπεί το αρχείο JPEG σας:

```csharp
var options = new PresentationConvertOptions();
```

`PresentationConvertOptions()` χρησιμοποιείται εδώ για να καθορίσει ότι η μορφή εξόδου θα πρέπει να είναι PPTX. Μπορείτε να εξερευνήσετε άλλες διαμορφώσεις εμβαθύνοντας στην τεκμηρίωση του GroupDocs.

#### Βήμα 4: Εκτελέστε τη μετατροπή

Εκτελέστε τη μετατροπή και αποθηκεύστε το αποτέλεσμα:

```csharp
converter.Convert(targetFilePath, options);
```

Αυτή η γραμμή μετατρέπει την εικόνα JPEG σε αρχείο PPTX και την αποθηκεύει στον καθορισμένο κατάλογο εξόδου.
