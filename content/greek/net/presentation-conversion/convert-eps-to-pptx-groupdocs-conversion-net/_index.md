## Οδηγός Εφαρμογής

### Φόρτωση αρχείου EPS

Αυτή η λειτουργία δείχνει πώς να φορτώσετε ένα αρχείο EPS χρησιμοποιώντας το GroupDocs.Conversion για .NET. Η σωστή διαχείριση πόρων είναι ζωτικής σημασίας εδώ, καθώς πρέπει να απορρίψουμε τους πόρους μετά τη χρήση.

**Βήμα 1:** Ορίστε τη διαδρομή
```csharp
c string documentPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.eps");
```

#### Αρχικοποίηση μετατροπέα

Δημιουργήστε ένα `Converter` αντιταχθείτε με τη διαδρομή του αρχείου EPS σας και βεβαιωθείτε ότι έχετε την κατάλληλη απόρριψη:

```csharp
using var converter = new Converter(documentPath);
converter.Dispose();
```

### Μετατροπή EPS σε μορφή PPTX

Αυτή η λειτουργία θα μετατρέψει το φορτωμένο αρχείο EPS σε μια παρουσίαση PowerPoint χρησιμοποιώντας το GroupDocs.Conversion.

**Βήμα 1:** Ορισμός διαδρομών για αρχεία εισόδου και εξόδου
```csharp
c string outputPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "eps-converted-to.pptx");
```

#### Αρχικοποίηση αντικειμένου μετατροπέα

Φορτώστε το αρχείο EPS αρχικοποιώντας ένα `Converter` αντικείμενο:

```csharp
using var converter = new Converter(documentPath);
```

**Βήμα 2:** Ορισμός επιλογών μετατροπής

Καθορίστε τις επιλογές μετατροπής για τη μορφή PowerPoint χρησιμοποιώντας `PresentationConvertOptions`.

```csharp
var options = new PresentationConvertOptions();
```

**Βήμα 3:** Εκτελέστε τη μετατροπή

Εκτελέστε τη μετατροπή και αποθηκεύστε το αποτέλεσμα στην καθορισμένη διαδρομή εξόδου:

```csharp
converter.Convert(outputPath, options);
```
