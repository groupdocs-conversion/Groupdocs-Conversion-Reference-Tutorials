## Οδηγός Εφαρμογής
Ας μετατρέψουμε ένα αρχείο XPS σε παρουσίαση PowerPoint.

### Βήμα 1: Ορισμός διαδρομών και επιλογών μετατροπής
Ορίστε διαδρομές για το αρχείο XPS εισόδου και το αρχείο PPT εξόδου:
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY"; // Αντικατάσταση με την πραγματική διαδρομή
string outputDirectory = "YOUR_OUTPUT_DIRECTORY"; // Αντικατάσταση με την πραγματική διαδρομή

// Ορίστε διαδρομές αρχείων προέλευσης και προορισμού
string sourceXpsFilePath = Path.Combine(documentDirectory, "sample.xps");
string convertedPptFilePath = Path.Combine(outputDirectory, "xps-converted-to.ppt");

using (var converter = new Converter(sourceXpsFilePath))
{
    PresentationConvertOptions options = new PresentationConvertOptions { Format = PresentationFileType.Ppt };

    // Μετατρέψτε και αποθηκεύστε το αρχείο εξόδου
    converter.Convert(convertedPptFilePath, options);
}
```
#### Εξήγηση:
- **Μονοπάτια**: Σετ `documentDirectory` και `outputDirectory` στους συγκεκριμένους καταλόγους σας.
- **PresentationConvertOptions**: Καθορίζει τη μετατροπή σε μορφή PowerPoint (PPT).
