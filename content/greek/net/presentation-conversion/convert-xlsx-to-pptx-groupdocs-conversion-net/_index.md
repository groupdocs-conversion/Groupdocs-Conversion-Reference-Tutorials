## Οδηγός Εφαρμογής
### Μετατροπή XLSX σε PPTX
Αυτή η λειτουργία σάς επιτρέπει να μετατρέπετε δεδομένα υπολογιστικών φύλλων σε μορφή παρουσίασης απρόσκοπτα.

#### Βήμα 1: Ορισμός διαδρομών και φόρτωση του αρχείου
Καθορίστε διαδρομές για το έγγραφο εισόδου και τον κατάλογο εξόδου. Στη συνέχεια, φορτώστε το αρχείο XLSX πηγής χρησιμοποιώντας το GroupDocs.Conversion.
```csharp
using System;
using System.IO;
using GroupDocs.Conversion;

string documentPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.xlsx");
```

#### Βήμα 2: Διαμόρφωση επιλογών μετατροπής
Ρυθμίστε τις επιλογές μετατροπής για τη μορφή PPTX. Αυτές οι επιλογές υπαγορεύουν τον τρόπο με τον οποίο θα μετατραπεί και θα αποθηκευτεί το αρχείο σας.
```csharp
using GroupDocs.Conversion.Options.Convert;
var options = new PresentationConvertOptions();
```

#### Βήμα 3: Εκτελέστε τη μετατροπή
Χρησιμοποιήστε το `Convert` μέθοδος για να μετατρέψετε το αρχείο XLSX σε αρχείο PPTX, αποθηκεύοντάς το στην καθορισμένη διαδρομή εξόδου.
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "xlsx-converted-to.pptx");

using (var converter = new Converter(documentPath))
{
    // Μετατρέψτε και αποθηκεύστε το αρχείο σε μορφή PPTX
    converter.Convert(outputFile, options);
}
```
