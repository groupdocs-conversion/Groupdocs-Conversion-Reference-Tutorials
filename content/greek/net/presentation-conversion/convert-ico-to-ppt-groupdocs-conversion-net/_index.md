## Οδηγός Εφαρμογής

### Φόρτωση και μετατροπή αρχείων ICO

#### Επισκόπηση
Φορτώστε ένα αρχείο ICO και μετατρέψτε το σε μορφή παρουσίασης PowerPoint, επιτρέποντας την εύκολη κοινή χρήση σε εφαρμογές που βασίζονται σε διαφάνειες.

**Βήμα 1: Ορισμός διαδρομών**
Ρυθμίστε τις διαδρομές εισόδου και εξόδου:

```csharp
string documentPath = @"YOUR_DOCUMENT_DIRECTORY\\\\sample.ico";
string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "ico-converted-to.ppt");
```

**Βήμα 2: Αρχικοποίηση του μετατροπέα**
Αρχικοποιήστε το GroupDocs.Conversion με τη διαδρομή αρχείου ICO:

```csharp
using (var converter = new GroupDocs.Conversion.Converter(documentPath))
{
    // Η λογική μετατροπής θα εφαρμοστεί εδώ.
}
```

**Βήμα 3: Ορισμός επιλογών μετατροπής**
Ρύθμιση παραμέτρων επιλογών μετατροπής για τη μορφή PowerPoint:

```csharp
PresentationConvertOptions options = new PresentationConvertOptions { Format = GroupDocs.Conversion.FileTypes.PresentationFileType.Ppt };
```
- **Επεξήγηση παραμέτρων**: 
  - `Format` καθορίζει τον τύπο αρχείου προορισμού (PPT σε αυτήν την περίπτωση).

**Βήμα 4: Εκτέλεση μετατροπής**
Εκτελέστε τη μετατροπή και αποθηκεύστε την έξοδο:

```csharp
converter.Convert(outputFile, options);
Console.WriteLine("Conversion completed successfully!");
```
