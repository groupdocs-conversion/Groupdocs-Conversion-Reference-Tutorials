## Οδηγός Εφαρμογής

Με την εγκατάσταση έτοιμη, ας μετατρέψουμε τα αρχεία POTX σε μορφή DOC.

### Μετατροπή POTX σε DOC

#### Επισκόπηση
Αυτή η λειτουργία επιτρέπει την εύκολη μετατροπή προτύπων Open XML του Microsoft PowerPoint σε μορφές εγγράφων Word. Ακολουθήστε τα παρακάτω βήματα:

#### Βήμα προς βήμα εφαρμογή

**1. Ορισμός καταλόγου εξόδου**
Ορίστε τον κατάλογο εξόδου όπου θα αποθηκευτεί το αρχείο που έχει μετατραπεί:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
```

**2. Φόρτωση και μετατροπή του αρχείου**
Χρησιμοποιήστε το GroupDocs.Conversion για να φορτώσετε και να μετατρέψετε το αρχείο POTX.

```csharp
using (var converter = new GroupDocs.Conversion.Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_POTX"))
{
    WordProcessingConvertOptions options = new WordProcessingConvertOptions 
    {
        Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc
    };

    string outputFile = System.IO.Path.Combine(outputFolder, "potx-converted-to.doc");
    converter.Convert(outputFile, options);
}
```

**Εξήγηση:**
- **μετατροπέας:** Αρχικοποιεί τη διαδικασία μετατροπής.
- **Επιλογές Μετατροπής Επεξεργασίας Κειμένου:** Καθορίζει επιλογές για μετατροπές επεξεργασίας Word.
- **Σχήμα και διάταξις βιβλίου:** Ορίζει το DOC ως τη μορφή στόχου.

#### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι οι διαδρομές αρχείων είναι σωστές για να αποφύγετε `FileNotFoundException`.
- Επαληθεύστε τα δικαιώματα εγγραφής για τον κατάλογο εξόδου.
