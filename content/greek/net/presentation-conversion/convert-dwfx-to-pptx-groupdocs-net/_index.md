### Μετατροπή DWFX σε PPTX
Μετατρέψτε ένα αρχείο XPS (.dwfx) σε παρουσίαση PowerPoint Open XML (.pptx).

#### Βήμα 1: Ορισμός διαδρομών
Ρυθμίστε τον κατάλογο εξόδου και τις διαδρομές αρχείων εισόδου:
```csharp
using System;
using System.IO;

string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY"); // Ορίστε τη διαδρομή του καταλόγου εξόδου
string inputFile = "YOUR_DOCUMENT_DIRECTORY\\sample.dwfx";  // Καθορίστε τη διαδρομή εισόδου του αρχείου DWFX
string outputFile = Path.Combine(outputFolder, "dwfx-converted-to.pptx"); // Ορίστε το όνομα του αρχείου PPTX εξόδου

// Βεβαιωθείτε ότι ο κατάλογος εξόδου υπάρχει
if (!Directory.Exists(outputFolder))
{
    Directory.CreateDirectory(outputFolder);
}
```
#### Βήμα 2: Αρχικοποίηση του μετατροπέα και μετατροπή αρχείου
Αρχικοποίηση του `Converter` αντικειμενοποιήστε το αρχείο DWFX, ορίστε επιλογές μετατροπής για τη μορφή PowerPoint και εκτελέστε τη μετατροπή.
```csharp
using GroupDocs.Conversion.Options.Convert;

// Φορτώστε το αρχείο DWFX πηγής και μετατρέψτε το σε PPTX
using (var converter = new Converter(inputFile))
{
    var options = new PresentationConvertOptions();  // Δημιουργήστε επιλογές μετατροπής για τη μορφή PowerPoint
    
    // Μετατρέψτε και αποθηκεύστε το αρχείο PPTX εξόδου
    converter.Convert(outputFile, options);
}
```
**Παράμετροι & Σκοπός μεθόδου:**
- `inputFile`: Διαδρομή προς το αρχείο DWFX σας.
- `options`: Καθορίζει ότι θέλουμε μια παρουσίαση PowerPoint ως έξοδο.
- `converter.Convert()`: Εκτελεί τη διαδικασία μετατροπής.

### Βοηθός διαμόρφωσης διαδρομής
Μια συνάρτηση βοηθητικού προγράμματος απλοποιεί τη λήψη μιας διαδρομής καταλόγου εξόδου, διασφαλίζοντας ότι δημιουργείται εάν δεν υπάρχει:
```csharp
using System;
using System.IO;

// Συνάρτηση για λήψη διαδρομής καταλόγου εξόδου με προεπιλεγμένη ρύθμιση
string GetOutputDirectoryPath()
{
    string outputPath = "YOUR_OUTPUT_DIRECTORY"; // Προεπιλεγμένη διαδρομή εξόδου
    
    if (!Directory.Exists(outputPath))
    {
        Directory.CreateDirectory(outputPath);  // Δημιουργήστε τον κατάλογο εάν δεν υπάρχει
    }
    
    return outputPath;  // Επιστρέψτε τη διαδρομή του καταλόγου εξόδου
}
```
