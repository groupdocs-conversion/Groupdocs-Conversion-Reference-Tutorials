## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Φόρτωση και μετατροπή VSDX σε PPTX

Αυτή η λειτουργία δείχνει πώς μπορείτε να φορτώσετε ένα αρχείο VSDX και να το μετατρέψετε σε μορφή PPTX, διασφαλίζοντας τη συμβατότητα για παρουσιάσεις.

#### Βήμα 1: Ορισμός διαδρομών αρχείων

Ξεκινήστε καθορίζοντας τους καταλόγους όπου βρίσκονται τα αρχεία προέλευσης και πού θα πρέπει να αποθηκευτεί η έξοδος:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// Δημιουργήστε διαδρομές αρχείων χρησιμοποιώντας σύμβολα κράτησης θέσης
string sourceFilePath = Path.Combine(documentDirectory, "sample.vsdx");
string outputFile = Path.Combine(outputDirectory, "vsdx-converted-to.pptx");
```

#### Βήμα 2: Φόρτωση και μετατροπή του αρχείου

Τώρα, φορτώστε το αρχείο VSDX και μετατρέψτε το σε PPTX:

```csharp
using (var converter = new Converter(sourceFilePath))
{
    var options = new PresentationConvertOptions(); // Ρύθμιση παραμέτρων μετατροπής για PPTX

    // Εκτελέστε τη μετατροπή και αποθηκεύστε την έξοδο
    converter.Convert(outputFile, options);
}
```
**Εξήγηση**: Το `Converter` Η κλάση φορτώνει το αρχείο VSDX σας. `PresentationConvertOptions` καθορίζει ότι θέλουμε να μετατρέψουμε σε μορφή PowerPoint. Τέλος, καλώντας `converter.Convert()` εκτελεί την πραγματική μετατροπή.
