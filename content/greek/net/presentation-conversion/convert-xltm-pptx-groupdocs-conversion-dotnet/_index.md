## Οδηγός Εφαρμογής

### Μετατροπή XLTM σε PPTX

Αυτή η ενότητα δείχνει πώς να μετατρέψετε ένα αρχείο XLTM σε μορφή PPTX χρησιμοποιώντας το GroupDocs.Conversion.

#### Βήμα προς βήμα εφαρμογή

##### Φόρτωση του αρχείου XLTM πηγής

Αρχικά, ορίστε τις διαδρομές για τους καταλόγους εισόδου και εξόδου:

```csharp
string sourceFilePath = "YOUR_DOCUMENT_DIRECTORY\\sample.xltm"; // Αντικαταστήστε με τη διαδρομή αρχείου XLTM
string outputFolder = "YOUR_OUTPUT_DIRECTORY"; // Διαδρομή για την αποθήκευση του PPTX που έχει μετατραπεί
```

Αρχικοποίηση του `Converter` αντικείμενο χρησιμοποιώντας τη διαδρομή του αρχείου προέλευσης:

```csharp
using (var converter = new Converter(sourceFilePath))
{
    string outputFile = Path.Combine(outputFolder, "xltm-converted-to.pptx");
    
    var options = new PresentationConvertOptions();
    converter.Convert(outputFile, options);
}
```

**Εξήγηση:**
- Ο `Converter` αρχικοποιείται με τη διαδρομή προς το αρχείο XLTM σας.
- `PresentationConvertOptions()` Καθορίζει ότι η μορφή εξόδου θα πρέπει να είναι PPTX.
- Ο `converter.Convert()` Η μέθοδος χειρίζεται τη μετατροπή και αποθηκεύει το αποτέλεσμα στον καθορισμένο κατάλογο.

##### Ρύθμιση καταλόγου εξόδου

Βεβαιωθείτε ότι υπάρχει ο κατάλογος όπου θέλετε να αποθηκεύσετε τα αρχεία που έχουν μετατραπεί:

```csharp
string outputDirectory = "YOUR_OUTPUT_DIRECTORY"; // Αντικαταστήστε με την επιθυμητή διαδρομή καταλόγου εξόδου

if (!Directory.Exists(outputDirectory))
{
    Directory.CreateDirectory(outputDirectory);
}
```

**Εξήγηση:**
- Αυτό το τμήμα κώδικα ελέγχει αν ο κατάλογος εξόδου υπάρχει.
- Αν δεν το κάνει, `Directory.CreateDirectory()` καλείται να δημιουργήσει τον απαραίτητο φάκελο.
