### Μετατροπή RTF σε PPT

Αυτή η ενότητα περιγράφει τον τρόπο μετατροπής ενός αρχείου RTF σε παρουσίαση PowerPoint χρησιμοποιώντας το GroupDocs.Conversion.

#### Βήμα 1: Ορισμός διαδρομών αρχείων και αρχικοποίηση μετατροπέα

Πριν από τη μετατροπή, καθορίστε τον κατάλογο εγγράφων και τον φάκελο εξόδου:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.rtf");
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "rtf-converted-to.ppt");

// Φόρτωση του αρχείου RTF πηγής
using (var converter = new Converter(inputFilePath))
{
    Console.WriteLine("RTF file loaded.");
}
```

#### Βήμα 2: Διαμόρφωση επιλογών μετατροπής

Ρύθμιση επιλογών μετατροπής για το PowerPoint:

```csharp
PresentationConvertOptions options = new PresentationConvertOptions
{
    Format = PresentationFileType.Ppt // Καθορίστε την επιθυμητή μορφή
};
```

#### Βήμα 3: Εκτελέστε τη μετατροπή

Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο εξόδου:

```csharp
converter.Convert(outputFile, options);
Console.WriteLine("Conversion completed successfully.");
```
