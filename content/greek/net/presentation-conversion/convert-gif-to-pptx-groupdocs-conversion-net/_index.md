## Οδηγός Εφαρμογής

### Μετατροπή κινούμενου GIF σε λειτουργία PPTX

Αυτή η λειτουργία επιτρέπει τη μετατροπή ενός αρχείου GIF σε μορφή παρουσίασης PowerPoint. Ας αναλύσουμε τα βήματα που απαιτούνται.

#### Βήμα 1: Ορισμός διαδρομών αρχείων και ρύθμιση μετατροπέα

Αρχικά, ορίστε τους καταλόγους εισόδου και εξόδου:

```csharp
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

string outputFolder = "YOUR_OUTPUT_DIRECTORY"; // Ορίστε τη διαδρομή του καταλόγου εξόδου
string inputFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.gif"); // Διαδρομή προς το αρχείο GIF εισόδου
string outputFile = Path.Combine(outputFolder, "gif-converted-to.pptx"); // Ορίστε το όνομα του αρχείου PPTX εξόδου

// Φόρτωση του αρχείου GIF πηγής
using (var converter = new Converter(inputFile))
{
    var options = new PresentationConvertOptions(); // Ορισμός επιλογών μετατροπής για τη μορφή PowerPoint
}
```

#### Βήμα 2: Εκτέλεση μετατροπής και αποθήκευση εξόδου

Μέσα στο μπλοκ using, εκτελέστε τη μετατροπή:

```csharp
converter.Convert(outputFile, options); // Μετατρέψτε και αποθηκεύστε το αρχείο PPTX
Console.WriteLine("Conversion Successful!");
```
