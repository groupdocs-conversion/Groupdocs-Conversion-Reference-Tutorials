## Οδηγός Εφαρμογής
### Φόρτωση και μετατροπή αρχείου TXT σε PPTX
Η κύρια λειτουργία εδώ είναι η μετατροπή ενός αρχείου TXT σε παρουσίαση PowerPoint (.pptx) χρησιμοποιώντας το GroupDocs.Conversion. Δείτε πώς:

#### Βήμα 1: Ορισμός διαδρομών αρχείων
Καθορίστε τις διαδρομές για το αρχείο TXT εισόδου και το αρχείο PPTX εξόδου.

```csharp
using System;
using System.IO;

string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.txt");
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "txt-converted-to.pptx");
```

#### Βήμα 2: Αρχικοποίηση μετατροπέα
Δημιουργήστε μια παρουσία του `Converter` κλάση και φορτώστε το αρχείο TXT. Αυτό το βήμα ρυθμίζει τη διαδικασία μετατροπής.

```csharp
using (var converter = new GroupDocs.Conversion.Converter(inputFilePath))
{
    var options = new PresentationConvertOptions(); // Ορίστε επιλογές μετατροπής για τη μορφή PPTX.
    
    // Μετατρέψτε και αποθηκεύστε την έξοδο στον καθορισμένο κατάλογο
    converter.Convert(outputFile, options);
}
```

#### Βασικές επιλογές διαμόρφωσης
- **PresentationConvertOptions**: Προσαρμόστε τον τρόπο με τον οποίο τα δεδομένα TXT μετατρέπονται σε διαφάνειες PowerPoint. Προσαρμόστε ρυθμίσεις όπως η διάταξη και η μορφοποίηση, όπως απαιτείται.
