## Οδηγός Εφαρμογής

Αφού όλα είναι έτοιμα, ας αναλύσουμε τη διαδικασία μετατροπής:

### Φόρτωση αρχείου XLS

Αρχικά, φορτώστε το αρχείο XLS πηγής καθορίζοντας τη διαδρομή του και δημιουργώντας ένα αντικείμενο μετατροπέα.

**Βήμα 1:** Ορισμός διαδρομής καταλόγου εγγράφων
```csharp
string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.xls");
```

**Βήμα 2:** Φόρτωση του αρχείου προέλευσης
```csharp
using (var converter = new Converter(inputFilePath))
{
    // Έτοιμο για εργασίες μετατροπής.
}
```
Εδώ, `Converter` αρχικοποιείται με τη διαδρομή του αρχείου σας.

### Ρύθμιση παραμέτρων επιλογών μετατροπής

Ορίστε τις επιλογές που απαιτούνται για τη μετατροπή από XLS σε PPT, συμπεριλαμβανομένης της μορφής προορισμού και των ρυθμίσεων.

**Βήμα 1:** Ορισμός μορφής εξόδου
```csharp
using GroupDocs.Conversion.Options.Convert;

PresentationConvertOptions options = new PresentationConvertOptions 
{
    Format = GroupDocs.Conversion.FileTypes.PresentationFileType.Ppt
};
```
Ο `options` Το αντικείμενο καθορίζει την έξοδο ως αρχείο PowerPoint.

### Μετατροπή XLS σε PPT και αποθήκευση εξόδου

Εκτελέστε τη μετατροπή του φορτωμένου αρχείου XLS σε μορφή PPT και αποθηκεύστε το.

**Βήμα 1:** Ορισμός διαδρομών για καταλόγους
```csharp
string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.xls");
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputDirectory, "xls-converted-to.ppt");
```

**Βήμα 2:** Μετατροπή και Αποθήκευση
```csharp
using (var converter = new Converter(inputFilePath))
{
    PresentationConvertOptions options = new PresentationConvertOptions 
    { 
        Format = GroupDocs.Conversion.FileTypes.PresentationFileType.Ppt 
    };

    // Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο
    converter.Convert(outputFile, options);
}
```
Το PPT που έχει μετατραπεί αποθηκεύεται στον καθορισμένο κατάλογο.
