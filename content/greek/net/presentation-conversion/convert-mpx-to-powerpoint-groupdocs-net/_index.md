## Οδηγός Εφαρμογής

Ας αναλύσουμε τη διαδικασία μετατροπής ενός αρχείου MPX σε παρουσίαση PowerPoint.

### Φόρτωση και μετατροπή αρχείου MPX σε PPT

**Επισκόπηση:** Αυτή η λειτουργία σάς επιτρέπει να φορτώνετε τα αρχεία MPX και να τα μετατρέπετε σε παρουσιάσεις PowerPoint χρησιμοποιώντας το GroupDocs.Conversion for .NET.

#### Βήμα 1: Ορισμός διαδρομών
Ξεκινήστε καθορίζοντας τους καταλόγους για τα αρχεία εισόδου και εξόδου:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
string inputFile = Path.Combine(documentDirectory, "sample.mpx");
string outputFile = Path.Combine(outputDirectory, "mpx-converted-to.ppt");
```

#### Βήμα 2: Διαμόρφωση επιλογών μετατροπής
Ρυθμίστε τις επιλογές μετατροπής για παρουσιάσεις PowerPoint:

```csharp
using GroupDocs.Conversion.Options.Convert;

// Ορίστε επιλογές μετατροπής παρουσίασης.
PresentationConvertOptions options = new PresentationConvertOptions
{
    Format = GroupDocs.Conversion.FileTypes.PresentationFileType.Ppt
};
```

#### Βήμα 3: Εκτέλεση μετατροπής

Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο εξόδου:

```csharp
// Μετατρέψτε MPX σε μορφή PPT χρησιμοποιώντας καθορισμένες επιλογές.
using (var converter = new Converter(inputFile))
{
    converter.Convert(outputFile, options);
}
```
