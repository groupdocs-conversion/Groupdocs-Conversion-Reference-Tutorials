## Βήμα 2: Φορτώστε το αρχείο προέλευσης GIF
```csharp
using (var converter = new GroupDocs.Conversion.Converter(Constants.SAMPLE_GIF))
{
    var options = new PdfConvertOptions();
    // Αποθηκεύστε το μετατρεπόμενο αρχείο PDF
    converter.Convert(outputFile, options);
}
```
 Στιγμιότυπο το`Converter` κλάση παρέχοντας τη διαδρομή προς το αρχείο προέλευσης GIF. Στη συνέχεια, ορίστε επιλογές μετατροπής, όπως π.χ`PdfConvertOptions` σε αυτήν την περίπτωση. Τέλος, ξεκινήστε τη διαδικασία μετατροπής καλώντας το`Convert` μέθοδο και περνώντας τη διαδρομή του αρχείου εξόδου μαζί με τις επιλογές.
