## Οδηγός Εφαρμογής

### Φόρτωση και μετατροπή PLT σε XLS

#### Επισκόπηση
Σε αυτήν την ενότητα, θα φορτώσουμε ένα αρχείο PLT και θα το μετατρέψουμε σε μορφή Excel χρησιμοποιώντας το GroupDocs.Conversion.

#### Βήμα 1: Φόρτωση του αρχείου προέλευσης
```csharp
string sourceFile = Path.Combine(DocumentDirectory, "sample.plt");
```
*Αυτό το βήμα καθορίζει τη διαδρομή προς το αρχείο PLT.*

#### Βήμα 2: Ορισμός επιλογών μετατροπής
```csharp
SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Xls };
```
*Εδώ, ορίζουμε τις ρυθμίσεις μετατροπής, καθορίζοντας ότι η μορφή εξόδου θα πρέπει να είναι XLS.*

#### Βήμα 3: Εκτελέστε τη μετατροπή
```csharp
string outputFile = Path.Combine(OutputDirectory, "plt-converted-to.xls");
converter.Convert(outputFile, options);
```
*Αυτό το βήμα εκτελεί τη μετατροπή και αποθηκεύει το αποτέλεσμα στον καθορισμένο κατάλογο.*
