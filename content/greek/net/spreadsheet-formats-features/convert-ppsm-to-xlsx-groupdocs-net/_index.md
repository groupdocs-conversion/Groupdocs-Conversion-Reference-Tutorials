## Οδηγός Εφαρμογής

### Μετατροπή PPSM σε XLSX

Αυτή η λειτουργία σάς βοηθά να μετατρέψετε μια παρουσίαση PowerPoint (.ppsm) σε ένα υπολογιστικό φύλλο Excel Open XML (.xlsx).

#### Επισκόπηση Διαδικασίας Μετατροπής
Θα χρησιμοποιήσουμε τη βιβλιοθήκη GroupDocs.Conversion για να φορτώσουμε, να μετατρέψουμε και να αποθηκεύσουμε τα αρχεία σας αποτελεσματικά.

##### Βήμα 1: Ορισμός διαδρομών για καταλόγους εισόδου και εξόδου
Ορίστε διαδρομές για το πού βρίσκεται το αρχείο PPSM σας και πού θέλετε την έξοδο XLSX:

```csharp
string documentPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "your-file.ppsm");
string outputFolder = "YOUR_OUTPUT_DIRECTORY"; 
string outputFile = Path.Combine(outputFolder, "ppsm-converted-to.xlsx");
```

##### Βήμα 2: Φόρτωση του εγγράφου PPSM
Δημιουργήστε μια παρουσία του `Converter` κλάση για να φορτώσετε το έγγραφό σας:

```csharp
using (var converter = new GroupDocs.Conversion.Converter(documentPath))
{
    // Συνεχίστε με τη ρύθμιση και την εκτέλεση της μετατροπής.
}
```

##### Βήμα 3: Αρχικοποίηση επιλογών μετατροπής για μορφή XLSX
Ορίστε τις απαραίτητες επιλογές για τη μετατροπή σε υπολογιστικό φύλλο Excel:

```csharp
var options = new SpreadsheetConvertOptions();
```

##### Βήμα 4: Μετατροπή και αποθήκευση του αρχείου εξόδου
Εκτελέστε τη διαδικασία μετατροπής και αποθηκεύστε το αρχείο σας στην επιθυμητή μορφή:

```csharp
converter.Convert(outputFile, options);
```
