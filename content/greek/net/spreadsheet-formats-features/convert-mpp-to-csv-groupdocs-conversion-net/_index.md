### Μετατροπή αρχείου MPP σε CSV

Η μετατροπή ενός αρχείου MPP σε CSV περιλαμβάνει τη φόρτωση του εγγράφου προέλευσης, τον καθορισμό των επιλογών μετατροπής και την αποθήκευση του αποτελέσματος. Δείτε πώς:

#### Βήμα 1: Ορισμός διαδρομών εγγράφων

Αρχικά, ρυθμίστε τις διαδρομές εισόδου και εξόδου για εύκολη διαχείριση:

```csharp
string inputDocumentPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.mpp");
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputDirectory, "mpp-converted-to.csv");
```

#### Βήμα 2: Φόρτωση του αρχείου MPP προέλευσης

Χρησιμοποιήστε το GroupDocs για να φορτώσετε το αρχείο MPP σε ένα `Converter` αντικείμενο:

```csharp
using (var converter = new Converter(inputDocumentPath))
{
    // Η λογική μετατροπής θα εφαρμοστεί εδώ.
}
```

#### Βήμα 3: Ορισμός επιλογών μετατροπής CSV

Ορίστε τις ρυθμίσεις μετατροπής για την έξοδο ενός αρχείου CSV:

```csharp
SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Csv };
```

#### Βήμα 4: Εκτελέστε και αποθηκεύστε τη μετατροπή

Τέλος, εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο CSV:

```csharp
csvConverter.Convert(outputFile, options);
```
