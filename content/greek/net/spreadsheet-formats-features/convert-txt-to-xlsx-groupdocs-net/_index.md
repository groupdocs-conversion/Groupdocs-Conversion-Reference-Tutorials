## Οδηγός Εφαρμογής

### Μετατροπή TXT σε XLSX

#### Επισκόπηση

Αυτή η ενότητα σας καθοδηγεί στη μετατροπή ενός `.txt` αρχείο σε μορφή υπολογιστικού φύλλου Excel Open XML (`xlsx`). Η διαδικασία είναι απλοποιημένη και απαιτεί ελάχιστη ρύθμιση.

#### Βήμα 1: Φόρτωση του αρχείου προέλευσης

Φορτώστε το αρχείο TXT πηγής σας χρησιμοποιώντας το `Converter` τάξη:
```csharp
using (var converter = new GroupDocs.Conversion.Converter(inputFilePath))
{
    // Περαιτέρω βήματα θα εφαρμοστούν εδώ.
}
```
**Γιατί;**: Η φόρτωση του αρχείου αρχικοποιεί τη διαδικασία μετατροπής και την προετοιμάζει για μετασχηματισμό.

#### Βήμα 2: Διαμόρφωση επιλογών μετατροπής

Στήνω `SpreadsheetConvertOptions` για να καθορίσετε τη μορφή προορισμού:
```csharp
var options = new SpreadsheetConvertOptions();
```
**Τι;**Αυτό το αντικείμενο ορίζει τις ρυθμίσεις για το αρχείο εξόδου, καθορίζοντας ότι θέλουμε ένα αρχείο XLSX.

#### Βήμα 3: Εκτελέστε τη μετατροπή

Εκτελέστε τη μετατροπή και αποθηκεύστε το αποτέλεσμα:
```csharp
converter.Convert(outputFile, options);
```
**Γιατί;**: Το `Convert` Η μέθοδος εφαρμόζει τις καθορισμένες επιλογές για να δημιουργήσει την επιθυμητή μορφή Excel από τα δεδομένα κειμένου σας.
