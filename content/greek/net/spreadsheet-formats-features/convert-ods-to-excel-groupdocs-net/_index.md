## Οδηγός Εφαρμογής
Ας αναλύσουμε τη διαδικασία μετατροπής σε διαχειρίσιμα βήματα.

### Φόρτωση και μετατροπή αρχείων
#### Επισκόπηση
Ξεκινάμε φορτώνοντας το αρχείο .ods του πηγαίου κώδικα και ρυθμίζοντας τις επιλογές μετατροπής για να στοχεύσουμε στη μορφή XLS.

#### Βήμα 1: Ορισμός διαδρομών αρχείων
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

string inputFilePath = Path.Combine(documentDirectory, "sample.ods");
string outputFile = Path.Combine(outputDirectory, "ods-converted-to.xls");
```

#### Βήμα 2: Ρύθμιση επιλογών μετατροπής
```csharp
using (var converter = new Converter(inputFilePath))
{
    // Καθορίστε τη μορφή μετατροπής σε XLS
    SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Xls };
    
    // Εκτελέστε τη μετατροπή και αποθηκεύστε την έξοδο
    converter.Convert(outputFile, options);
}
```
**Εξήγηση**: Εδώ, `SpreadsheetConvertOptions` χρησιμοποιείται για να ορίσει τον τύπο αρχείου προορισμού μας ως XLS. Η μέθοδος `converter.Convert()` χειρίζεται τον πραγματικό μετασχηματισμό αρχείου.

### Συμβουλές αντιμετώπισης προβλημάτων
- **Συνηθισμένο πρόβλημα**Βεβαιωθείτε ότι η διαδρομή του αρχείου εισόδου είναι σωστή και προσβάσιμη.
- **Διάλυμα**: Επικύρωση διαδρομών χρησιμοποιώντας `File.Exists(inputFilePath)` πριν από τη μετατροπή.
