#### Βασική Αρχικοποίηση και Ρύθμιση με C#
Δείτε πώς μπορείτε να αρχικοποιήσετε το GroupDocs.Conversion στην εφαρμογή σας:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

namespace DocumentConversionToSpreadsheet
{
    internal static class ConvertDocumentToSpreadsheet
    {
        public static void Run()
        {
            // Ορίστε τη διαδρομή του καταλόγου εξόδου χρησιμοποιώντας ένα σύμβολο κράτησης θέσης.
            string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");

            // Συνδυάστε τον φάκελο εξόδου και το όνομα αρχείου για να δημιουργήσετε την πλήρη διαδρομή για το αρχείο που έχει μετατραπεί.
            string outputFile = Path.Combine(outputFolder, "converted.xlsx");

            // Αρχικοποιήστε το αντικείμενο Converter με τη διαδρομή του εγγράφου προέλευσης χρησιμοποιώντας ένα σύμβολο κράτησης θέσης.
            using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY"))
            {
                // Δημιουργήστε μια παρουσία του SpreadsheetConvertOptions για να καθορίσετε επιλογές μετατροπής.
                SpreadsheetConvertOptions options = new SpreadsheetConvertOptions();

                // Εκτελέστε τη μετατροπή από το έγγραφο εισόδου στο καθορισμένο αρχείο εξόδου με επιλογές.
                converter.Convert(outputFile, options);
            }
        }
    }
}
```
