### Βασική Αρχικοποίηση και Ρύθμιση

Δείτε πώς μπορείτε να αρχικοποιήσετε το έργο σας με το GroupDocs.Conversion σε C#:

```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

namespace AiToXlsxConverter
{
    class Program
    {
        static void Main(string[] args)
        {
            string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY";
            string outputDirectory = @"YOUR_OUTPUT_DIRECTORY/";

            // Αρχικοποιήστε τον μετατροπέα με τη διαδρομή αρχείου AI
            using (var converter = new Converter(documentDirectory + "\sample.ai"))
            {
                var options = new SpreadsheetConvertOptions();
                string outputFile = System.IO.Path.Combine(outputDirectory, "ai-converted-to.xlsx");
                
                // Μετατροπή και αποθήκευση του αρχείου XLSX
                converter.Convert(outputFile, options);
            }
        }
    }
}
```

Αυτό το απόσπασμα δείχνει τη φόρτωση ενός αρχείου AI και τη μετατροπή του σε μορφή XLSX χρησιμοποιώντας `SpreadsheetConvertOptions`.
