### Ορισμός επιλογών μετατροπής για τη μορφή Excel

**Ρύθμιση παραμέτρων εξόδου**

Στη συνέχεια, καθορίστε ότι θέλετε να μετατρέψετε το αρχείο σε μορφή XLSX ορίζοντας τις επιλογές μετατροπής:

```csharp

```

Αυτό το απόσπασμα κώδικα δημιουργεί ένα `SpreadsheetConvertOptions` παράδειγμα, που σας επιτρέπει να ορίσετε παραμέτρους για μετατροπές Excel.

### Μετατροπή και αποθήκευση του αρχείου ως XLSX

**Εκτέλεση της μετατροπής**

Αφού έχετε ρυθμίσει όλα τα στοιχεία, εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο εξόδου:

```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", ".");

converter.Convert(outputFile, options);
```

Αυτός ο κώδικας θα αποθηκεύσει το αρχείο που έχει μετατραπεί σε `"YOUR_OUTPUT_DIRECTORY"` με το όνομα `vss-converted-to.xlsx`.
