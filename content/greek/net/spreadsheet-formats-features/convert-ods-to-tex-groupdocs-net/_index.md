## Οδηγός Εφαρμογής

### Μετατροπή ODS σε μορφή TEX

**Επισκόπηση:** Αυτή η ενότητα σας καθοδηγεί στην απρόσκοπτη μετατροπή ενός αρχείου ODS σε μορφή TEX.

#### Βήμα 1: Ρύθμιση παραμέτρων καταλόγων και διαδρομών αρχείων
Ορίστε διαδρομές για τα αρχεία προέλευσης και εξόδου:

```csharp
using System.IO;

string sourceFilePath = "YOUR_DOCUMENT_DIRECTORY\sample.ods";
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "ods-converted-to.tex");
```

#### Βήμα 2: Φόρτωση του αρχείου ODS προέλευσης
Φορτώστε το αρχείο ODS χρησιμοποιώντας το GroupDocs.Conversion:

```csharp
using (var converter = new Converter(sourceFilePath))
{
    // Ο κώδικας μετατροπής θα τοποθετηθεί εδώ
}
```

#### Βήμα 3: Ορισμός επιλογών μετατροπής
Ρυθμίστε τις παραμέτρους επιλογών για να καθορίσετε τη μορφή TEX:

```csharp
using GroupDocs.Conversion.Options.Convert;

var options = new PageDescriptionLanguageConvertOptions 
{ 
    Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Tex 
};
```

#### Βήμα 4: Εκτελέστε τη μετατροπή
Εκτελέστε τη μετατροπή και αποθηκεύστε την ως αρχείο TEX:

```csharp
converter.Convert(outputFile, options);
Console.WriteLine("Conversion completed successfully.");
```
