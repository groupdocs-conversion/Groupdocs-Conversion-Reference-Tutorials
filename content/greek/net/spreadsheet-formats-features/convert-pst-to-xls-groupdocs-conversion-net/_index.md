### Μετατροπή PST σε XLS

Αυτή η λειτουργία μετατρέπει ένα αρχείο Προσωπικού Πίνακα Αποθήκευσης (.pst) σε ένα Υπολογιστικό Φύλλο Excel (.xls).

#### Επισκόπηση
Η διαδικασία περιλαμβάνει την αρχικοποίηση του μετατροπέα με το αρχείο PST πηγής και τη διαμόρφωσή του για έξοδο σε μορφή XLS.

**Βήματα Υλοποίησης:**

##### Βήμα 1: Αρχικοποίηση μετατροπέα
Ρυθμίστε τον μετατροπέα χρησιμοποιώντας τη διαδρομή προς το αρχείο PST:

```csharp
string sourceFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.pst");
using (var converter = new GroupDocs.Conversion.Converter(sourceFilePath))
{
    // Θα ακολουθήσουν περαιτέρω βήματα διαμόρφωσης.
}
```

##### Βήμα 2: Διαμόρφωση επιλογών μετατροπής
Ορίστε επιλογές μετατροπής για να καθορίσετε τη μορφή εξόδου ως XLS:

```csharp
SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Xls };
```

##### Βήμα 3: Εκτέλεση μετατροπής
Εκτελέστε τη μετατροπή και αποθηκεύστε το αποτέλεσμα:

```csharp
string outputFileTemplate = Path.Combine("YOUR_OUTPUT_DIRECTORY", "pst-converted-{0}-to.xls");
converter.Convert(
    (SaveContext saveContext) => new FileStream(string.Format(outputFileTemplate, counter++), FileMode.Create),
    options
);
```

**Εξήγηση:**
- **Φόρτωση περιβάλλοντος:** Καθορίζει τον ειδικό χειρισμό για τα αρχεία OST.
- **Επιλογές μετατροπής:** Καθορίζει τη μορφή προορισμού ως XLS.
