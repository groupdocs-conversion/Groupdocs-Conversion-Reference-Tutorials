### Βασική Αρχικοποίηση

Μόλις εγκατασταθεί, αρχικοποιήστε το GroupDocs.Conversion στο έργο C# σας με αυτήν τη ρύθμιση:
```csharp

using GroupDocs.Conversion;

class Program
{
    static void Main()
    {
        // Ρυθμίστε το αρχείο ODT πηγής και τον κατάλογο εξόδου
        string sourceFilePath = @"YOUR_DOCUMENT_DIRECTORY\sample.odt";
        string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
        string outputFile = Path.Combine(outputFolder, "odt-converted-to.xlsx");

        // Αρχικοποίηση του GroupDocs.Conversion για .NET
        using (var converter = new Converter(sourceFilePath))
        {
            var options = new SpreadsheetConvertOptions();
            converter.Convert(outputFile, options);
        }
    }
}
```
