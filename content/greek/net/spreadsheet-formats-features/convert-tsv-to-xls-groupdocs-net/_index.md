## Οδηγός Εφαρμογής

Ας αναλύσουμε τη διαδικασία μετατροπής σε σαφή βήματα.

### Φόρτωση και προετοιμασία του αρχείου TSV

Καθορίστε διαδρομές για αρχεία εισόδου και εξόδου. Βεβαιωθείτε ότι είναι έτοιμο ένα έγκυρο αρχείο TSV:

```csharp
string dataDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
string inputFile = Path.Combine(dataDirectory, "sample.tsv");
string outputFile = Path.Combine(outputDirectory, "tsv-converted-to.xls");
```

### Μετατροπή TSV σε XLS

Ρυθμίστε τις επιλογές μετατροπής για τη μορφή-στόχο:

```csharp
using (var converter = new Converter(inputFile))
{
    // Ορίστε ρυθμίσεις μετατροπής για το Excel.
    var options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Xls };

    // Εκτελέστε τη διαδικασία μετατροπής.
    converter.Convert(outputFile, options);
}
```

Σε αυτόν τον κώδικα:
- `SpreadsheetConvertOptions` καθορίζει τη μορφή και τυχόν πρόσθετες παραμέτρους.
- Ο `Format` η ιδιότητα έχει οριστεί σε `Xls`, υποδεικνύοντας την επιθυμητή μορφή εξόδου.
