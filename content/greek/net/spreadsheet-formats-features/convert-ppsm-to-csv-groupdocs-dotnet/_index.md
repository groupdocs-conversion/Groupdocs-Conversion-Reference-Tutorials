```csharp

{
    // Το αρχείο PPSM είναι πλέον έτοιμο για μετατροπή.
}
```

### Καθορισμός επιλογών μετατροπής

Ορίστε τις επιλογές μετατροπής για να μετατρέψετε την παρουσίασή σας σε μορφή CSV:
```csharp
var options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Csv };
// Αυτές οι επιλογές καθορίζουν ότι η έξοδος θα είναι σε μορφή CSV.
```

### Ορισμός διαδρομής εξόδου και εκτέλεση μετατροπής

Καθορίστε τη διαδρομή εξόδου για το αρχείο που μετατρέψατε και εκτελέστε τη διαδικασία μετατροπής:
```csharp
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputDirectory, "ppsm-converted-to.csv");

converter.Convert(outputFile, options);
// Αυτό εγγράφει το μετατρεπόμενο CSV στην καθορισμένη τοποθεσία.
```
