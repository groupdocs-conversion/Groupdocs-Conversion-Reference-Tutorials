## Οδηγός Εφαρμογής

Θα αναλύσουμε τη διαδικασία μετατροπής σε εύκολα βήματα.

### Μετατροπή XLAM σε XLS

#### Επισκόπηση

Αυτή η λειτουργία μετατρέπει ένα αρχείο πρόσθετου του Excel (XLAM) σε ένα τυπικό βιβλίο εργασίας του Excel (XLS), διασφαλίζοντας τη συμβατότητα με διάφορες εκδόσεις του Microsoft Office.

#### Βήμα 1: Ρύθμιση διαδρομών αρχείων

Ρυθμίστε τις παραμέτρους των εγγράφων και των καταλόγων εξόδου χρησιμοποιώντας placeholders:

```csharp
string sourcePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "source.xlam");
string outputPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "xlam-converted-to.xls");
```

#### Βήμα 2: Φόρτωση του αρχείου XLAM πηγής

Χρησιμοποιήστε το GroupDocs.Conversion για να φορτώσετε το αρχείο XLAM, καθορίζοντας τη διαδρομή:

```csharp
using (var converter = new Converter(sourcePath))
{
    // Η λογική μετατροπής θα ακολουθηθεί εδώ.
}
```

#### Βήμα 3: Ορισμός επιλογών μετατροπής

Καθορίστε τις ρυθμίσεις μετατροπής για τη μορφή XLS χρησιμοποιώντας `SpreadsheetConvertOptions`:

```csharp
SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Xls };
```

Αυτή η ρύθμιση παραμέτρων διασφαλίζει ότι το αρχείο εξόδου έχει την επιθυμητή μορφή XLS.

#### Βήμα 4: Μετατροπή και αποθήκευση εξόδου

Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο XLS που προκύπτει:

```csharp
csv converter.Convert(outputPath, options);
```

### Ρύθμιση παραμέτρων διαδρομής αρχείου

#### Επισκόπηση

Η σωστή ρύθμιση των διαδρομών των αρχείων σας είναι ζωτικής σημασίας για επιτυχημένες μετατροπές. Αυτή η λειτουργία διασφαλίζει ότι έχετε ορίσει σωστά τους καταλόγους εισόδου και εξόδου.

#### Βήμα 1: Ορισμός καταλόγων

Καθορίστε τη δομή καταλόγου για τα αρχεία προέλευσης και προορισμού:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
```

#### Βήμα 2: Συνδυασμός διαδρομών

Δημιουργήστε πλήρεις διαδρομές αρχείων χρησιμοποιώντας `Path.Combine`:

```csharp
string sourceFilePath = Path.Combine(documentDirectory, "source.xlam");
string targetFilePath = Path.Combine(outputDirectory, "xlam-converted-to.xls");
```
