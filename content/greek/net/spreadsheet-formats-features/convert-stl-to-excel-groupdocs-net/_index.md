### Βασική Αρχικοποίηση

Μετά την εγκατάσταση, ρυθμίστε τη διαδικασία μετατροπής με μια απλή ρύθμιση:
```csharp
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

// Καθορίστε τη διαδρομή του αρχείου STL εισόδου σας
string inputFile = @"path\to\your\file.stl";
string outputFolder = @"YOUR_OUTPUT_DIRECTORY";

// Δημιουργήστε μια παρουσία της κλάσης Converter και φορτώστε το αρχείο STL
using (Converter converter = new Converter(inputFile))
{
    // Ρύθμιση επιλογών μετατροπής για μορφή Excel
    SpreadsheetConvertOptions convertOptions = new SpreadsheetConvertOptions();

    // Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο XLSX στον κατάλογο εξόδου
    converter.Convert(Path.Combine(outputFolder, "output.xlsx"), convertOptions);
}
```
