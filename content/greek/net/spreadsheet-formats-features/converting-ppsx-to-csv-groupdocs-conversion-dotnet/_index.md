## Οδηγός Εφαρμογής
Θα αναλύσουμε την υλοποίηση σε σαφείς ενότητες για ευκολία κατανόησης.
### Χαρακτηριστικό: Φόρτωση και μετατροπή αρχείου PPSX
Αυτή η ενότητα παρουσιάζει τη φόρτωση ενός αρχείου PPSX και τη μετατροπή του σε CSV χρησιμοποιώντας το GroupDocs.Conversion.
#### Βήμα 1: Αρχικοποίηση του μετατροπέα με ένα αρχείο PPSX
```csharp
using System.IO;
using GroupDocs.Conversion;

const string DocumentDirectory = @"YOUR_DOCUMENT_DIRECTORY";
const string OutputDirectory = @"YOUR_OUTPUT_DIRECTORY/";

string outputFolder = Path.Combine(OutputDirectory);
string outputFile = Path.Combine(outputFolder, "ppsx-converted-to.csv");

// Αρχικοποιήστε τον μετατροπέα με ένα αρχείο PPSX
using (var converter = new Converter(Path.Combine(DocumentDirectory, "sample.ppsx")))
{
    // Θα ακολουθήσουν επιπλέον βήματα εδώ
}
```
**Εξήγηση:** Ξεκινάμε ρυθμίζοντας τους καταλόγους εγγράφων και εξόδου. Στη συνέχεια, αρχικοποιούμε το `Converter` κλάση χρησιμοποιώντας τη διαδρομή προς το αρχείο PPSX μας.
#### Βήμα 2: Ορίστε τις Επιλογές Μετατροπής σε Μορφή CSV
```csharp
using GroupDocs.Conversion.Options.Convert;

// Ορισμός επιλογών μετατροπής για μορφή CSV
class SpreadsheetConvertOptions : ConvertOptions
{
    public SpreadsheetFileType Format { get; set; }
}

var options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Csv };
```
**Εξήγηση:** Εδώ, καθορίζουμε ότι η έξοδος μας θα πρέπει να είναι σε μορφή CSV.
#### Βήμα 3: Εκτελέστε τη μετατροπή και αποθηκεύστε την έξοδο
```csharp
// Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο εξόδου
class Program
{
    static void Main()
    {
        // Υποθέτοντας ότι ο μετατροπέας έχει αρχικοποιηθεί όπως φαίνεται προηγουμένως
        converter.Convert(outputFile, options);
    }
}
```
**Εξήγηση:** Τέλος, εκτελούμε την πραγματική μετατροπή και αποθηκεύουμε το αποτέλεσμα σε μια καθορισμένη τοποθεσία. `Convert` Η μέθοδος χειρίζεται αυτήν τη διαδικασία.
