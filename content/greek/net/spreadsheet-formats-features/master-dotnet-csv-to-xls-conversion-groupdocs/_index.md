## Οδηγός Εφαρμογής

Ας αναλύσουμε τη διαδικασία μετατροπής ενός αρχείου CSV σε μορφή XLS χρησιμοποιώντας το GroupDocs.Conversion for .NET σε διαχειρίσιμα βήματα.

### Φόρτωση και μετατροπή CSV σε XLS

Αυτή η λειτουργία σάς επιτρέπει να φορτώσετε ένα αρχείο CSV και να το μετατρέψετε σε μορφή XLS αναγνώσιμη από το Excel. Δείτε πώς:

#### Βήμα 1: Ορισμός καταλόγου εξόδου

Αρχικά, ρυθμίστε τον κατάλογο εξόδου όπου θα αποθηκευτούν τα αρχεία που έχουν μετατραπεί. Αντικαταστήστε `"YOUR_DOCUMENT_DIRECTORY"` με την επιθυμητή σας διαδρομή.

```csharp
string outputFolder = Path.Combine(@"YOUR_DOCUMENT_DIRECTORY");
```

#### Βήμα 2: Φόρτωση αρχείου CSV

Φορτώστε το αρχείο CSV στο εργαλείο μετατροπής:

```csharp
using (Converter converter = new Converter("input.csv"))
{
    // Ο υπόλοιπος κώδικας θα μπει εδώ
}
```

Αυτό αρχικοποιεί ένα `Converter` αντικείμενο που χειρίζεται το αρχείο εισόδου.

#### Βήμα 3: Ορισμός επιλογών μετατροπής

Στη συνέχεια, διαμορφώστε τις επιλογές μετατροπής για τη μορφή XLS χρησιμοποιώντας τις ενσωματωμένες λειτουργίες του GroupDocs.Conversion:

```csharp
var convertOptions = new SpreadsheetConvertOptions();
```

Αυτές οι επιλογές καθορίζουν ότι θέλετε να μετατρέψετε το αρχείο CSV σε μορφή υπολογιστικού φύλλου Excel.

#### Βήμα 4: Εκτέλεση μετατροπής

Τέλος, εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο εξόδου:

```csharp
converter.Convert(() => new FileStream(Path.Combine(outputFolder, "output.xls"), FileMode.Create), convertOptions);
```

Αυτό το βήμα εκτελεί την πραγματική μετατροπή και εγγράφει το αρχείο XLS που προκύπτει στον καθορισμένο κατάλογο.
