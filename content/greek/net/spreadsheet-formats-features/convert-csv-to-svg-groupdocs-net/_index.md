## Οδηγός Εφαρμογής

### Φόρτωση αρχείου CSV

**Επισκόπηση:**
Η φόρτωση του αρχείου CSV πηγής είναι το πρώτο βήμα για την προετοιμασία του για μετατροπή. Αυτή η διαδικασία περιλαμβάνει τον καθορισμό της διαδρομής και τη χρήση της ισχυρής λειτουργικότητας του GroupDocs.Conversion.

#### Βήμα 1: Ορισμός καταλόγου εγγράφων
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
```
Ορίστε τον κατάλογο όπου βρίσκεται το αρχείο CSV σας.

#### Βήμα 2: Φόρτωση του αρχείου CSV προέλευσης
```csharp
using (var converter = new Converter(Path.Combine(documentDirectory, "sample.csv")))
{
    // Το αρχείο CSV έχει πλέον φορτωθεί και είναι έτοιμο για μετατροπή.
}
```
**Εξήγηση:** Αυτό το απόσπασμα αρχικοποιεί ένα `Converter` αντικείμενο με το αρχείο CSV σας, καθιστώντας το διαθέσιμο για επόμενες λειτουργίες.

### Ρύθμιση παραμέτρων επιλογών μετατροπής για SVG

**Επισκόπηση:**
Η ρύθμιση των σωστών επιλογών διασφαλίζει ότι η μορφή εξόδου πληροί τις απαιτήσεις σας. Εδώ, θα ορίσουμε επιλογές για τη μετατροπή του αρχείου σε μορφή SVG.

#### Βήμα 3: Ρύθμιση επιλογών μετατροπής
```csharp
using GroupDocs.Conversion.Options.Convert;

PageDescriptionLanguageConvertOptions options = new PageDescriptionLanguageConvertOptions 
{
    Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Svg
};
```
**Εξήγηση:** Αυτή η διαμόρφωση καθορίζει ότι το αρχείο εξόδου θα πρέπει να είναι σε μορφή SVG, επιτρέποντας την ακριβή μετατροπή.

### Αποθήκευση ενός μετατρεπόμενου αρχείου ως SVG

**Επισκόπηση:**
Αφού διαμορφώσετε τις επιλογές σας, θα χρειαστεί να αποθηκεύσετε το αρχείο που μετατράπηκε. Αυτό το βήμα ολοκληρώνει τη διαδικασία και αποθηκεύει το νέο σας αρχείο SVG.

#### Βήμα 4: Ορισμός διαδρομής εξόδου
```csharp
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputDirectory, "csv-converted-to.svg");
```

#### Βήμα 5: Αποθήκευση του μετατρεπόμενου αρχείου
```csharp
// Αποθηκεύστε το αρχείο που μετατράπηκε ως SVG
converter.Convert(outputFile, options);
```
**Εξήγηση:** Αυτή η γραμμή εκτελεί τη μετατροπή και γράφει την έξοδο στην καθορισμένη διαδρομή σας σε μορφή SVG.
