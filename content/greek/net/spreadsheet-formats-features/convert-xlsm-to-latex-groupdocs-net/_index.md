## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Φόρτωση και μετατροπή αρχείου XLSM
Αυτή η λειτουργία δείχνει πώς να φορτώσετε ένα αρχείο υπολογιστικού φύλλου με δυνατότητα μακροεντολών του Microsoft Excel (.xlsm) και να το μετατρέψετε σε έγγραφο πηγαίου κώδικα LaTeX (.tex).

#### Βήμα 1: Ορισμός διαδρομών αρχείων
Αρχικά, καθορίστε τις διαδρομές εισόδου και εξόδου:
```csharp
string documentPath = "YOUR_DOCUMENT_DIRECTORY\\\\sample.xlsm";  // Εισαγωγή διαδρομής αρχείου XLSM
string outputFolder = "YOUR_OUTPUT_DIRECTORY";  // Κατάλογος εξόδου
string outputFile = System.IO.Path.Combine(outputFolder, "xlsm-converted-to.tex");  // Διαδρομή αρχείου TEX εξόδου
```
#### Βήμα 2: Φόρτωση του αρχείου προέλευσης
Χρησιμοποιήστε το GroupDocs.Conversion για να φορτώσετε το αρχείο XLSM. Το `Converter` Η κλάση χειρίζεται αποτελεσματικά αυτήν τη λειτουργία:
```csharp
using (var converter = new Converter(documentPath))
{
    // Η λογική μετατροπής θα εφαρμοστεί εδώ.
}
```
Ο `using` Η δήλωση διασφαλίζει την σωστή διάθεση των πόρων, κάτι που είναι κρίσιμο για τη διαχείριση μνήμης σε εφαρμογές .NET.
#### Βήμα 3: Ορισμός επιλογών μετατροπής
Καθορίστε τη μορφή εξόδου χρησιμοποιώντας `PageDescriptionLanguageConvertOptions`, στοχεύοντας LaTeX (.tex):
```csharp
var options = new PageDescriptionLanguageConvertOptions { Format = PageDescriptionLanguageFileType.Tex };
```
Ο `options` Το αντικείμενο επιτρέπει τη διαμόρφωση παραμέτρων όπως το μέγεθος σελίδας και τα περιθώρια.
#### Βήμα 4: Εκτελέστε τη μετατροπή
Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο εξόδου:
```csharp
converter.Convert(outputFile, options);
Console.WriteLine("Conversion completed successfully.");
```
Αυτό το βήμα μετατρέπει τα δεδομένα XLSM σας σε ένα έγγραφο LaTeX έτοιμο για στοιχειοθεσία.
