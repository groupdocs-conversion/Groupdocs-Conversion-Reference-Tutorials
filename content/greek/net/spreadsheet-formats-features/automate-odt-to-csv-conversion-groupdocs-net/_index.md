## Οδηγός Εφαρμογής

### Μετατροπή ODT σε CSV

**Επισκόπηση**
Σε αυτήν την ενότητα, θα δούμε πώς να μετατρέψετε ένα αρχείο .odt σε μορφή .csv χρησιμοποιώντας το GroupDocs.Conversion.

#### Βήμα 1: Ορισμός καταλόγου εξόδου και διαδρομής αρχείου
Ξεκινήστε καθορίζοντας πού θα πρέπει να αποθηκευτούν τα αρχεία που έχετε μετατρέψει:

```csharp
string outputFolder = Path.Combine(Directory.GetCurrentDirectory(), "Converted");
```
**Εξήγηση:** Αυτή η γραμμή ορίζει τον φάκελο προορισμού για το αρχείο CSV. Βεβαιωθείτε ότι `outputFolder` έχει οριστεί σωστά σε έναν εγγράψιμο κατάλογο.

#### Βήμα 2: Φόρτωση και μετατροπή εγγράφου
Εδώ, φορτώνουμε το αρχείο ODT και το μετατρέπουμε σε CSV:

```csharp
using (Converter converter = new Converter("path/to/your/document.odt"))
{
    var options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Csv };
    converter.Convert(Path.Combine(outputFolder, "output.csv"), options);
}
```
**Εξήγηση:** 
- `new Converter("path/to/your/document.odt")`: Φορτώνει το αρχείο ODT.
- `SpreadsheetConvertOptions`: Διαμορφώνει τις ρυθμίσεις μετατροπής σε μορφή CSV.
- `converter.Convert(...)`: Εκτελεί τη μετατροπή και αποθηκεύει το αποτέλεσμα.
