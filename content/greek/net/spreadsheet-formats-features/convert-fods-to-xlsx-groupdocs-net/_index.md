## Οδηγός Εφαρμογής

### Μετατροπή FODS σε XLSX

#### Επισκόπηση

Αυτή η λειτουργία μετατρέπει αρχεία με οριοθέτηση σταθερού πλάτους (FODS) σε μορφή Excel, XLSX—ιδανική για ανάλυση δεδομένων και δημιουργία αναφορών.

#### Βήμα προς βήμα εφαρμογή

**1. Ορισμός διαδρομών εξόδου**
Ρυθμίστε τον κατάλογο εξόδου και τις διαδρομές αρχείων:
```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "fods-converted-to.xlsx");
```

**2. Φορτώστε το αρχείο FODS πηγής**
Χρησιμοποιήστε το `Converter` κλάση για να φορτώσετε το αρχείο FODS:
```csharp
using (var converter = new GroupDocs.Conversion.Converter(Path.Combine("YOUR_DOCUMENT_DIRECTORY", "SAMPLE_FODS")))
{
    // Η λογική μετατροπής θα εφαρμοστεί εδώ.
}
```

**3. Ρύθμιση επιλογών μετατροπής**
Διαμόρφωση επιλογών για μετατροπή σε μορφή XLSX:
```csharp
var options = new SpreadsheetConvertOptions();
```

**4. Εκτελέστε τη μετατροπή**
Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο εξόδου:
```csharp
converter.Convert(outputFile, options);
```
