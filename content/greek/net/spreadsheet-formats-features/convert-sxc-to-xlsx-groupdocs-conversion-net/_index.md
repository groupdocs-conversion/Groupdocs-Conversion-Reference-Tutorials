### Δυνατότητα: Φόρτωση και μετατροπή αρχείου SXC σε XLSX

Αυτή η λειτουργία σάς επιτρέπει να φορτώσετε ένα υπάρχον αρχείο SXC και να το μετατρέψετε στην πιο ευρέως χρησιμοποιούμενη μορφή XLSX.

#### Βήμα 1: Ορισμός καταλόγου εξόδου και διαδρομής αρχείου

Ξεκινήστε ρυθμίζοντας τον κατάλογο εξόδου σας:

```csharp
string outputFolder = Path.Combine(Directory.GetCurrentDirectory(), "ConvertedFiles");
string outputFile = Path.Combine(outputFolder, "sxc-converted-to.xlsx");

// Βεβαιωθείτε ότι ο κατάλογος εξόδου υπάρχει
Directory.CreateDirectory(outputFolder);
```

#### Βήμα 2: Φόρτωση και μετατροπή του αρχείου SXC

Στη συνέχεια, φορτώστε το αρχείο SXC και αρχικοποιήστε τις επιλογές μετατροπής:

```csharp
using (var converter = new GroupDocs.Conversion.Converter("YOUR_DOCUMENT_DIRECTORY/sample.sxc"))
{
    var options = new SpreadsheetConvertOptions();
    // Εκτελέστε τη μετατροπή
    converter.Convert(outputFile, options);
}
```

**Εξήγηση:**
- `GroupDocs.Conversion.Converter`Φορτώνει το αρχείο SXC από την καθορισμένη διαδρομή.
- `SpreadsheetConvertOptions`Ορίζει τη μορφή στόχου μετατροπής σε XLSX.
