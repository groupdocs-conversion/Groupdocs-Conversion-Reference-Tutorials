## Οδηγός Εφαρμογής

### Επισκόπηση λειτουργιών: Μετατροπή JLS σε XLSX

Αυτή η λειτουργία επιδεικνύει τη μετατροπή ενός αρχείου JLS σε μορφή υπολογιστικού φύλλου Excel Open XML (.xlsx). Θα σας καθοδηγήσουμε σε κάθε βήμα, διασφαλίζοντας σαφήνεια και ακρίβεια.

#### Βήμα 1: Ορισμός καταλόγων αρχείων
Πριν από την εκτέλεση οποιωνδήποτε μετατροπών, καθορίστε πού βρίσκονται τα αρχεία σας:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY"; // Κατάλογος πηγαίου κώδικα για το αρχείο JLS.
string outputDirectory = "YOUR_OUTPUT_DIRECTORY"; // Κατάλογος προορισμού για το αρχείο XLSX.
```

#### Βήμα 2: Βεβαιωθείτε ότι υπάρχει κατάλογος εξόδου
Βεβαιωθείτε ότι ο κατάλογος εξόδου είναι διαθέσιμος ή δημιουργήστε τον, εάν είναι απαραίτητο:

```csharp
if (!Directory.Exists(outputDirectory))
{
    Directory.CreateDirectory(outputDirectory);
}
```

#### Βήμα 3: Ρύθμιση διαδρομών αρχείων
Δημιουργήστε πλήρεις διαδρομές για το αρχείο JLS πηγής και το αρχείο XLSX προορισμού:

```csharp
string inputFilePath = Path.Combine(documentDirectory, "sample.jls"); // Αντικαταστήστε το με το πραγματικό σας όνομα αρχείου.
string outputFilePath = Path.Combine(outputDirectory, "jls-converted-to.xlsx");
```

#### Βήμα 4: Αρχικοποίηση του μετατροπέα και εκτέλεση μετατροπής
Χρησιμοποιήστε το `Converter` κλάση για να ξεκινήσει η διαδικασία μετατροπής:

```csharp
using (var converter = new Converter(inputFilePath))
{
    var options = new SpreadsheetConvertOptions(); // Ορίστε επιλογές για τη μορφή Excel.
    
    // Εκτελέστε τη μετατροπή από JLS σε XLSX και αποθηκεύστε το αποτέλεσμα.
    converter.Convert(outputFilePath, options);
}
```
