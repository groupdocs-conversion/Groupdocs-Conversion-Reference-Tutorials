##### 1. Φορτώστε το αρχείο DOC πηγής

```csharp

    // Οι επιλογές μετατροπής θα οριστούν εδώ
```

Ρυθμίστε τις επιλογές μετατροπής σας για να καθορίσετε ότι η μορφή εξόδου θα πρέπει να είναι CSV:


**Γιατί:** Αυτό το βήμα εφαρμόζει τις καθορισμένες ρυθμίσεις μετατροπής για να μετασχηματίσει και να αποθηκεύσει την έξοδο σε μορφή CSV.
