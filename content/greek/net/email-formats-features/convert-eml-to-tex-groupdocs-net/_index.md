## Οδηγός Εφαρμογής

### Χαρακτηριστικό 1: Φόρτωση και μετατροπή αρχείου EML σε μορφή TEX
#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να φορτώσετε ένα αρχείο EML και να το μετατρέψετε σε μορφή TEX χρησιμοποιώντας το `GroupDocs.Conversion` βιβλιοθήκη, βελτιστοποιώντας και απλοποιώντας τη διαδικασία.

**Βήμα 1: Φόρτωση του αρχείου EML προέλευσης**
Δημιουργήστε ένα `Converter` αντικείμενο με τη διαδρομή του αρχείου EML πηγής σας:
```csharp
using System;
using GroupDocs.Conversion;

string sourceFilePath = "YOUR_DOCUMENT_DIRECTORY\sample.eml";
```

**Βήμα 2: Διαμόρφωση επιλογών μετατροπής**
Καθορίστε επιλογές μετατροπής για τη μορφή TEX χρησιμοποιώντας `PageDescriptionLanguageConvertOptions`.
```csharp
using GroupDocs.Conversion.Options.Convert;

PageDescriptionLanguageConvertOptions options = new PageDescriptionLanguageConvertOptions { Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Tex };
```

**Βήμα 3: Εκτελέστε τη μετατροπή**
Χρησιμοποιήστε το `Converter` αντικείμενο για να μετατρέψετε και να αποθηκεύσετε το αρχείο σας σε μορφή TEX:
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY\";
string outputFile = System.IO.Path.Combine(outputFolder, "eml-converted-to.tex");

using (var converter = new Converter(sourceFilePath))
{
    converter.Convert(outputFile, options);
}
```
**Συμβουλή αντιμετώπισης προβλημάτων:** Βεβαιωθείτε ότι η διαδρομή του αρχείου EML προέλευσης και ο κατάλογος εξόδου έχουν οριστεί σωστά για να αποτρέψετε τυχόν `FileNotFoundException`.

### Λειτουργία 2: Ρύθμιση παραμέτρων διαδρομής καταλόγου εξόδου
#### Επισκόπηση
Αυτή η λειτουργία δείχνει τη δημιουργία ενός καταλόγου εξόδου για την αποθήκευση των μετατρεπόμενων αρχείων, διασφαλίζοντας την οργάνωση και την εύκολη πρόσβαση μετά τη μετατροπή.

**Βήμα 1: Ορισμός βασικού καταλόγου**
Ορίστε έναν βασικό κατάλογο, συνήθως τον φάκελο MyDocuments:
```csharp
string baseDirectory = System.Environment.GetFolderPath(System.Environment.SpecialFolder.MyDocuments);
```

**Βήμα 2: Δημιουργία καταλόγου εξόδου εάν δεν υπάρχει**
Επαληθεύστε ή δημιουργήστε έναν κατάλογο εξόδου για την αποθήκευση των αρχείων που έχετε μετατρέψει:
```csharp
string outputDirectory = System.IO.Path.Combine(baseDirectory, "ConvertedFiles");

if (!System.IO.Directory.Exists(outputDirectory))
{
    System.IO.Directory.CreateDirectory(outputDirectory);
}
```
