## Οδηγός Εφαρμογής

Θα αναλύσουμε τη διαδικασία μετατροπής σε διαχειρίσιμα βήματα ανά χαρακτηριστικό.

### Φόρτωση αρχείου MBOX
**Επισκόπηση:**
Η φόρτωση ενός αρχείου MBOX είναι το πρώτο βήμα, προετοιμάζοντας το περιβάλλον μας για μετατροπή.

#### Βήμα 1: Ορίστε τη διαδρομή του αρχείου προέλευσης
```csharp
string sourceFilePath = "YOUR_DOCUMENT_DIRECTORY/sample.mbox"; // Αντικαταστήστε με τη διαδρομή σας προς το αρχείο MBOX
```

#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών φόρτωσης
Δημιουργήστε επιλογές φόρτωσης ειδικά για αρχεία MBOX:
```csharp
var loadOptions = new LoadOptions();
if (loadOptions.SourceFormat == EmailFileType.Mbox)
{
    var mboxLoadOptions = new MboxLoadOptions();
    // Ο μετατροπέας θα χρησιμοποιήσει αυτές τις επιλογές για τη φόρτωση του αρχείου.
}
```

### Ρύθμιση παραμέτρων επεξεργασίας κειμένου Επιλογές μετατροπής
**Επισκόπηση:**
Ρυθμίστε επιλογές μετατροπής για να μετατρέψετε το έγγραφό σας σε μορφή TXT.

#### Βήμα 1: Ορισμός επιλογών μετατροπής
```csharp
var convertOptions = new WordProcessingConvertOptions { Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt };
```
Αυτές οι επιλογές καθορίζουν ότι η έξοδος θα πρέπει να είναι σε μορφή απλού κειμένου (TXT), ευέλικτη για διάφορες εφαρμογές.

### Αποθήκευση μετατρεπόμενου αρχείου ως TXT
**Επισκόπηση:**
Το τελικό βήμα περιλαμβάνει την αποθήκευση του αρχείου που έχετε μετατρέψει σε μια καθορισμένη τοποθεσία.

#### Βήμα 1: Ρύθμιση διαδρομής εξόδου
```csharp
string outputFilePath = "YOUR_OUTPUT_DIRECTORY/mbox-converted-{0}-to.txt"; // Αντικαταστήστε με την επιθυμητή διαδρομή
```

#### Βήμα 2: Εκτελέστε τη μετατροπή
Χρησιμοποιήστε ένα `FileStream` για εξοικονόμηση:
```csharp
int counter = 1;
var saveOptions = new SaveOptions();
using (var converter = new Converter(sourceFilePath, () => new MboxLoadOptions()))
{
    converter.Convert(
        (saveContext) => new FileStream(string.Format(outputFilePath, counter++), FileMode.Create),
        convertOptions
    );
}
```
Αυτό το απόσπασμα δείχνει πώς να χειριστείτε τη διαδικασία μετατροπής και να αποθηκεύσετε κάθε τμήμα εγγράφου που προκύπτει σε ένα αρχείο διαδοχικά.
