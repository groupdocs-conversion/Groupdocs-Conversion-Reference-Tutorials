## Οδηγός Εφαρμογής

Ας αναλύσουμε τη διαδικασία μετατροπής βήμα προς βήμα για να διασφαλίσουμε σαφήνεια και ακρίβεια.

### Βήμα 1: Ορισμός διαδρομών αρχείων

Ορίστε διαδρομές για το αρχείο EML εισόδου και τον κατάλογο SVG εξόδου. Αυτό καθορίζει από πού θα γίνει η ανάγνωση και η εγγραφή της μετατροπής.
```csharp
string YOUR_DOCUMENT_DIRECTORY = "YOUR_DOCUMENT_DIRECTORY"; // Κατάλογος εγγράφου πηγής
string YOUR_OUTPUT_DIRECTORY = "YOUR_OUTPUT_DIRECTORY"; // Κατάλογος εξόδου

// Διαδρομές εισόδου και εξόδου
string inputFilePath = Path.Combine(YOUR_DOCUMENT_DIRECTORY, "sample.eml");
string outputFolder = YOUR_OUTPUT_DIRECTORY;
string outputFile = Path.Combine(outputFolder, "eml-converted-to.svg");
```

### Βήμα 2: Φόρτωση και μετατροπή του αρχείου EML

Φορτώστε το αρχείο EML στον μετατροπέα. Αρχικοποιήστε το `Converter` αντικείμενο με τη διαδρομή αρχείου εισόδου μας και, στη συνέχεια, καθορίστε τις επιλογές μετατροπής για τη μορφή SVG.
```csharp
using (var converter = new GroupDocs.Conversion.Converter(inputFilePath))
{
    // Ρύθμιση επιλογών μετατροπής σε SVG
    PageDescriptionLanguageConvertOptions options = new PageDescriptionLanguageConvertOptions 
    {
        Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Svg
    };

    // Εκτελέστε τη μετατροπή
    converter.Convert(outputFile, options);
}
```
**Βασικά σημεία:**
- Ο `Converter` Το αντικείμενο διαχειρίζεται τη φόρτωση και τη μετατροπή αρχείων.
- `PageDescriptionLanguageConvertOptions` Καθορίζει τις ρυθμίσεις μορφής SVG.
