## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Μετατροπή EML σε TXT
Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε ένα αρχείο EML σε μια πιο διαχειρίσιμη μορφή TXT χρησιμοποιώντας το GroupDocs.Conversion.

#### Βήμα 1: Ορισμός διαδρομών αρχείων
Ξεκινήστε ορίζοντας τις διαδρομές για τους καταλόγους εισόδου και εξόδου. Αυτό βοηθά την εφαρμογή σας να γνωρίζει από πού να διαβάσει και πού να γράψει.
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY"; // Καθορίστε τον κατάλογο αρχείων EML σας
string outputDirectory = "YOUR_OUTPUT_DIRECTORY"; // Καθορίστε τον κατάλογο εξόδου TXT

// Ορίστε ονόματα αρχείων
class Program
{
    static void Main()
    {
        string inputFile = Path.Combine(documentDirectory, "sample.eml");
        string outputFile = Path.Combine(outputDirectory, "eml-converted-to.txt");
    }
}
```
#### Βήμα 2: Φόρτωση του αρχείου EML προέλευσης
Χρησιμοποιήστε το GroupDocs.Conversion για να φορτώσετε το αρχείο EML πηγής.
```csharp
using (var converter = new Converter(inputFile))
{
    // Οι επιλογές μετατροπής θα οριστούν στο επόμενο βήμα.
}
```
*Γιατί να χρησιμοποιήσετε `using`Διασφαλίζει ότι οι πόροι διατίθενται σωστά μόλις ολοκληρωθεί η μετατροπή.*

#### Βήμα 3: Ορισμός επιλογών μετατροπής
Ρυθμίστε τις παραμέτρους της μετατροπής σας ώστε να εξάγεται ως αρχείο TXT χρησιμοποιώντας συγκεκριμένες ρυθμίσεις μορφής.
```csharp
var options = new WordProcessingConvertOptions { Format = FileType.Txt };
```
*Ο `WordProcessingConvertOptions` Η κλάση παρέχει διάφορες παραμέτρους διαμόρφωσης, επιτρέποντάς σας να καθορίσετε την επιθυμητή μορφή εξόδου.*

#### Βήμα 4: Μετατροπή και αποθήκευση
Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο TXT με τις διαμορφωμένες ρυθμίσεις.
```csharp
converter.Convert(outputFile, options);
```
### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι οι διαδρομές έχουν καθοριστεί σωστά για να αποφύγετε `FileNotFoundException`.
- Επαληθεύστε ότι η έκδοση του GroupDocs.Conversion ταιριάζει με τις εξαρτήσεις του έργου σας.
