### Φόρτωση και μετατροπή αρχείου POTM

#### Επισκόπηση
Θα φορτώσουμε ένα αρχείο POTM και θα το μετατρέψουμε σε μορφή DOC χρησιμοποιώντας το GroupDocs.Conversion. Αυτή η διαδικασία περιλαμβάνει τον καθορισμό επιλογών μετατροπής και την εκτέλεση της μεθόδου μετατροπής.

#### Βήμα προς βήμα εφαρμογή

**1. Ορισμός διαδρομών εγγράφων**
Ξεκινήστε ορίζοντας διαδρομές για το αρχείο POTM εισόδου και το αρχείο DOC εξόδου:

```csharp
const string DocumentDirectory = "YOUR_DOCUMENT_DIRECTORY";
const string OutputDirectory = "YOUR_OUTPUT_DIRECTORY/";
```

**2. Αρχικοποίηση μετατροπέα**
Φορτώστε το αρχείο POTM σε ένα αντικείμενο GroupDocs.Conversion:

```csharp
using (var converter = new GroupDocs.Conversion.Converter(Path.Combine(DocumentDirectory, "sample.potm")))
{
    // Η λογική μετατροπής θα εφαρμοστεί εδώ.
}
```

**3. Ορισμός επιλογών μετατροπής**
Καθορίστε επιλογές για τη μετατροπή του εγγράφου σε μορφή DOC:

```csharp
var options = new WordProcessingConvertOptions
{
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc
};
```

**4. Εκτελέστε μετατροπή**
Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο εξόδου:

```csharp
string outputFile = Path.Combine(OutputDirectory, "potm-converted-to.doc");
converter.Convert(outputFile, options);
```
