## Οδηγός Εφαρμογής
### Χαρακτηριστικό 1: Μετατροπή JPG σε DOCX
Αυτή η λειτουργία σάς επιτρέπει να μετατρέπετε αρχεία JPG σε μορφή DOCX απρόσκοπτα.
#### Φόρτωση του αρχείου JPG πηγής
Αρχικά, δημιουργήστε ένα `Converter` αντικείμενο με τη διαδρομή του αρχείου προέλευσης. Αυτό ξεκινά τη διαδικασία μετατροπής:
```csharp
using System.IO;
using GroupDocs.Conversion;

string documentPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.jpg");
```
#### Ρύθμιση παραμέτρων επιλογών μετατροπής για μορφή DOCX
Στη συνέχεια, ρυθμίστε τις επιλογές μετατροπής για την έξοδο ενός εγγράφου Word:
```csharp
using GroupDocs.Conversion.Options.Convert;
var options = new WordProcessingConvertOptions();
```
#### Μετατροπή και αποθήκευση του αρχείου ως DOCX
Τέλος, εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο εξόδου:
```csharp
string outputPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "jpg-converted-to.docx");

using (var converter = new Converter(documentPath))
{
    converter.Convert(outputPath, options);
}
```
### Χαρακτηριστικό 2: Ρύθμιση παραμέτρων διαδρομής καταλόγου
Η σωστή διαμόρφωση καταλόγου είναι ζωτικής σημασίας για τη διαχείριση των αρχείων εισόδου και εξόδου.
#### Ορισμός placeholders
Ορίστε placeholders για αποτελεσματική διαχείριση διαδρομών αρχείων:
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
```
#### Συνδυασμός στοιχείων διαδρομής
Δημιουργήστε μια μέθοδο για να συνδυάσετε δυναμικά τα στοιχεία της διαδρομής:
```csharp
public static string GetFullPath(string fileName)
{
    return Path.Combine(outputDirectory, fileName);
}
```
