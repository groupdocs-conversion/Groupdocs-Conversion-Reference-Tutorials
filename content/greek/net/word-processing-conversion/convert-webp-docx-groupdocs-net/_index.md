## Οδηγός Εφαρμογής

### Μετατροπή WebP σε DOCX

Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε εικόνες σε μορφή WebP σε επεξεργάσιμα έγγραφα Word. Ακολουθήστε τα παρακάτω βήματα για να την εφαρμόσετε:

#### Βήμα 1: Ρύθμιση διαδρομών εξόδου
Ορίστε πού θα αποθηκευτούν τα αρχεία που έχετε μετατρέψει.

```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", "output");
string outputFile = Path.Combine(outputFolder, "webp-converted-to.docx");
```

**Εξήγηση:** `Path.Combine` διασφαλίζει ότι οι διαδρομές έχουν σωστή μορφοποίηση σε διαφορετικά λειτουργικά συστήματα.

#### Βήμα 2: Φόρτωση και μετατροπή του αρχείου WebP

```csharp
using (var converter = new Converter(Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.webp")))
{
    var options = new WordProcessingConvertOptions();
    converter.Convert(outputFile, options);
}
```

**Εξήγηση:** 
- **Μετατροπέας:** Αρχικοποιείται με μια διαδρομή αρχείου πηγαίου κώδικα.
- **Επιλογές Μετατροπής Επεξεργασίας Κειμένου:** Ρυθμίζει τις παραμέτρους της μετατροπής σε μορφή DOCX.
- **μετατροπέας.Μετατροπή:** Εκτελεί τη διαδικασία μετατροπής.

#### Συμβουλές αντιμετώπισης προβλημάτων

- Βεβαιωθείτε ότι το αρχείο προέλευσης WebP υπάρχει στον καθορισμένο κατάλογο.
- Ελέγξτε για τυχόν προβλήματα δικαιωμάτων στον κατάλογο εξόδου.
- Επιβεβαιώστε ότι το GroupDocs.Conversion έχει εγκατασταθεί σωστά και έχει γίνει αναφορά σε αυτό.
