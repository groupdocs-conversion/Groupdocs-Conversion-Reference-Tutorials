## Οδηγός Εφαρμογής
Τώρα, ας αναλύσουμε τη διαδικασία μετατροπής ενός αρχείου PS σε έγγραφο DOCX.

### Δυνατότητα: Μετατροπή αρχείων PS σε έγγραφα Word
Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε αρχεία PostScript σε έγγραφα του Microsoft Word χρησιμοποιώντας το GroupDocs.Conversion for .NET. Ακολουθήστε τα παρακάτω βήματα:

#### Βήμα 1: Ορισμός διαδρομών αρχείων
Ξεκινήστε καθορίζοντας διαδρομές για τα αρχεία εισόδου και εξόδου. Αντικαταστήστε τα placeholders με πραγματικές διαδρομές καταλόγων στο περιβάλλον σας.
```csharp
string inputPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.ps");
string outputPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "ps-converted-to.docx");
```

#### Βήμα 2: Φόρτωση του αρχείου PS προέλευσης
Αρχικοποίηση του `Converter` κλάση με τη διαδρομή προς το αρχείο PS σας.
```csharp
using (var converter = new GroupDocs.Conversion.Converter(inputPath))
{
    // Προχωρήστε στα βήματα μετατροπής.
}
```

#### Βήμα 3: Καθορισμός επιλογών μετατροπής
Ορίστε επιλογές για τη μετατροπή του εγγράφου σε μορφή Word. Εδώ, χρησιμοποιούμε `WordProcessingConvertOptions`.
```csharp
var options = new WordProcessingConvertOptions();
```

#### Βήμα 4: Εκτελέστε τη μετατροπή
Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο DOCX εξόδου.
```csharp
converter.Convert(outputPath, options);
Console.WriteLine("Conversion completed successfully.");
```
