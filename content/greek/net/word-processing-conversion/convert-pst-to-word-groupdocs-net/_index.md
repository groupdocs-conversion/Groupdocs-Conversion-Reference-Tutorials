## Οδηγός Εφαρμογής

Τώρα, ας εμβαθύνουμε στα κύρια χαρακτηριστικά της διαδικασίας μετατροπής μας.

### Φόρτωση αρχείου PST

#### Επισκόπηση
Η φόρτωση ενός αρχείου PST είναι ζωτικής σημασίας, καθώς ρυθμίζει τα δεδομένα για μετατροπή. Θα χρησιμοποιήσουμε συγκεκριμένες επιλογές για να χειριστούμε αυτήν τη διαδικασία αποτελεσματικά.

```csharp
using System;
using GroupDocs.Conversion.FileTypes;
using GroupDocs.Conversion.Options.Load;

string sourceFilePath = "YOUR_DOCUMENT_DIRECTORY\\sample.pst";

// Δημιουργήστε το LoadContext με συνθήκες για αρχεία PST ή OST.
LoadContext loadContext = new LoadContext(sourceFilePath, (sourceFormat) =>
{
    return sourceFormat == EmailFileType.Ost ? new PersonalStorageLoadOptions() : null;
});

// Αρχικοποιήστε τον Μετατροπέα με το φορτωμένο αρχείο και το περιβάλλον.
GroupDocs.Conversion.Converter converter = new GroupDocs.Conversion.Converter(sourceFilePath, loadContext);
```

**Εξήγηση:**
- `LoadContext`: Ρυθμίζει τον τρόπο φόρτωσης των αρχείων. Καθορίζουμε τον χειρισμό των αρχείων OST για να διασφαλίσουμε τη συμβατότητα.
- `EmailFileType.Ost`Ελέγχει αν η μορφή πηγής είναι OST και εφαρμόζει συγκεκριμένες επιλογές φόρτωσης ανάλογα.

### Μετατροπή σε μορφή επεξεργασίας κειμένου (DOC)

#### Επισκόπηση
Αυτή η λειτουργία μετατρέπει το φορτωμένο αρχείο PST σε μορφή DOC, κατάλληλη για διάφορες εφαρμογές επεξεργασίας κειμένου.

```csharp
using System.IO;
using GroupDocs.Conversion.Options.Convert;

string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "pst-converted-{0}-to.doc");

WordProcessingConvertOptions options = new WordProcessingConvertOptions { Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc };
int counter = 1;

// Αποθηκεύστε το αρχείο DOC που έχει μετατραπεί χρησιμοποιώντας ένα FileStream για να διαχειριστείτε πολλές σελίδες/αρχεία.
converter.Convert((saveContext) =>
{
    return new FileStream(string.Format(outputFileTemplate, counter++), FileMode.Create);
}, options);
```

**Εξήγηση:**
- `WordProcessingConvertOptions`Καθορίζει τις ρυθμίσεις μετατροπής για έγγραφα Word (DOC).
- `FileStream`: Χρησιμοποιείται για τη διαχείριση της δημιουργίας αρχείου εξόδου, διασφαλίζοντας ότι κάθε σελίδα ή στοιχείο αποθηκεύεται ξεχωριστά.
