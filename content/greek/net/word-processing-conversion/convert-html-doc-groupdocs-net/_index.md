### Χαρακτηριστικό: Μετατροπή HTML σε DOC

Αυτή η λειτουργία μετατρέπει ένα αρχείο HTML σε έγγραφο του Word χρησιμοποιώντας τη βιβλιοθήκη GroupDocs.Conversion. Παρακάτω παρατίθενται τα βήματα για την επίτευξη αυτής της μετατροπής.

#### Βήμα 1: Βεβαιωθείτε ότι υπάρχει ο κατάλογος εξόδου

Ξεκινήστε δημιουργώντας έναν κατάλογο εξόδου, εάν δεν υπάρχει:

```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", "output");
if (!Directory.Exists(outputFolder))
{
    Directory.CreateDirectory(outputFolder);
}
```

- **Γιατί:** Αυτό διασφαλίζει ότι έχετε ένα καθορισμένο μέρος για να αποθηκεύσετε τα αρχεία DOC που έχουν μετατραπεί.

#### Βήμα 2: Φόρτωση του αρχείου HTML πηγής

Φορτώστε το αρχείο HTML πηγής χρησιμοποιώντας το `Converter` τάξη:

```csharp
string inputHtmlFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.html");
using (var converter = new Converter(inputHtmlFilePath))
{
    // Η λογική μετατροπής θα εφαρμοστεί εδώ.
}
```

- **Γιατί:** Ο `Converter` Το αντικείμενο χειρίζεται τη φόρτωση και τη μετατροπή εγγράφων.

#### Βήμα 3: Ορισμός επιλογών μετατροπής

Ρυθμίστε τις επιλογές μετατροπής για τη μορφή DOC:

```csharp
WordProcessingConvertOptions options = new WordProcessingConvertOptions
{
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc
};
```

- **Γιατί:** Αυτό ορίζει τις απαραίτητες παραμέτρους για τη μετατροπή του εγγράφου σας σε αρχείο .doc.

#### Βήμα 4: Εκτελέστε τη μετατροπή

Τέλος, εκτελέστε τη μετατροπή και αποθηκεύστε την έξοδο:

```csharp
string outputFile = Path.Combine(outputFolder, "html-converted-to.doc");
converter.Convert(outputFile, options);
```

- **Γιατί:** Αυτό το βήμα εκτελεί την πραγματική διαδικασία μετατροπής και αποθηκεύει το αρχείο DOC που προκύπτει στην καθορισμένη τοποθεσία σας.
