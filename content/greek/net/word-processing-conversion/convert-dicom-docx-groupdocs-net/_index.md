**Επισκόπηση**Η μετατροπή του φορτωμένου αρχείου DICOM σε μορφή DOCX επιτρέπει την εύκολη επεξεργασία.

#### Βήμα 1: Ορισμός διαδρομών προέλευσης και εξόδου
Ορίστε διαδρομές για τους καταλόγους προέλευσης και εξόδου:
```csharp

```

#### Βήμα 2: Φόρτωση και μετατροπή
Φορτώστε το αρχείο DCM και μετατρέψτε το σε μορφή DOCX:
```csharp

{
    var options = new WordProcessingConvertOptions();
    converter.Convert(outputFile, options);
}
```
