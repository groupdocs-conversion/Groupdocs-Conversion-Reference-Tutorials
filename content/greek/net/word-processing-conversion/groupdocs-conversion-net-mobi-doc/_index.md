### Χαρακτηριστικό: Μετατροπή Mobi σε Docx

#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς μπορείτε να μετατρέψετε ένα αρχείο MOBI σε μορφή DOCX χρησιμοποιώντας το GroupDocs.Conversion for .NET. Είναι ιδιαίτερα χρήσιμο σε περιπτώσεις όπου η συμβατότητα και η μορφοποίηση εγγράφων είναι κρίσιμες.

#### Βήμα 1: Ορισμός διαδρομών αρχείων
Ξεκινήστε ορίζοντας τις διαδρομές για το αρχείο εισόδου MOBI και το αρχείο εξόδου DOCX:

```csharp
string sourceMobiFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.mobi");
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "mobi-converted-to.docx");
```

#### Βήμα 2: Φόρτωση του αρχείου MOBI
Χρησιμοποιήστε το GroupDocs.Conversion για να φορτώσετε το αρχείο Mobi:

```csharp
using (var converter = new Converter(sourceMobiFilePath))
{
    Console.WriteLine("MOBI file loaded successfully.");
}
```

#### Βήμα 3: Ορισμός επιλογών μετατροπής
Ρυθμίστε τις παραμέτρους μετατροπής για την επεξεργασία κειμένου:

```csharp
var options = new WordProcessingConvertOptions { Format = WordProcessingFileType.Docx };
```

#### Βήμα 4: Εκτέλεση μετατροπής και αποθήκευση εξόδου
Εκτελέστε τη διαδικασία μετατροπής και αποθηκεύστε το έγγραφό σας σε μορφή DOCX:

```csharp
converter.Convert(outputFile, options);
Console.WriteLine("Conversion completed successfully.");
```
