## Ρύθμιση του GroupDocs.Conversion για .NET
Για να ξεκινήσετε τη μετατροπή των εγγράφων LaTeX, εγκαταστήστε πρώτα τη βιβλιοθήκη GroupDocs.Conversion. Δείτε πώς:
### Κονσόλα διαχείρισης πακέτων NuGet
```bash
Install-Package GroupDocs.Conversion -Version 25.3.0
```
### .NET CLI
```bash
dotnet add package GroupDocs.Conversion --version 25.3.0
```
#### Απόκτηση Άδειας:
- **Δωρεάν δοκιμή**Ξεκινήστε με μια δωρεάν δοκιμαστική περίοδο για να δοκιμάσετε τις δυνατότητες της βιβλιοθήκης.
- **Προσωρινή Άδεια**Αποκτήστε προσωρινή άδεια για εκτεταμένες δοκιμές.
- **Αγορά**: Σκεφτείτε το ενδεχόμενο αγοράς μιας πλήρους άδειας χρήσης για συνεχή χρήση.
Μόλις εγκατασταθεί, αρχικοποιήστε και ρυθμίστε το GroupDocs.Conversion στο έργο σας:
```csharp
using System.IO;
using GroupDocs.Conversion.Options.Convert;
using GroupDocs.Conversion;
// Ορίστε τις διαδρομές για τα αρχεία εισόδου και εξόδου
string outputFolder = "YOUR_OUTPUT_DIRECTORY"; // Ορίστε την επιθυμητή διαδρομή καταλόγου εξόδου
string inputFile = "YOUR_DOCUMENT_DIRECTORY/sample.tex"; // Διαδρομή προς το αρχείο .tex εισόδου
string outputFile = Path.Combine(outputFolder, "tex-converted-to.doc"); // Διαδρομή αρχείου εξόδου
// Φορτώστε το αρχείο προέλευσης TEX χρησιμοποιώντας το GroupDocs.Conversion
using (var converter = new Converter(inputFile)) 
{
    WordProcessingConvertOptions options = new WordProcessingConvertOptions { Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc };
    // Μετατρέψτε και αποθηκεύστε το αρχείο DOC στην καθορισμένη διαδρομή εξόδου
    converter.Convert(outputFile, options);
}
```
