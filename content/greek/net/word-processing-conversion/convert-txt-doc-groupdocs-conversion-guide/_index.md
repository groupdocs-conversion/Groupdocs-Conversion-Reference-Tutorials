## Οδηγός Εφαρμογής

### Δυνατότητα μετατροπής TXT σε DOC

Ας αναλύσουμε την υλοποίηση σε λογικά βήματα:

#### Βήμα 1: Φόρτωση του αρχείου προέλευσης

Καθορίστε τη διαδρομή προς την πηγή σας `.txt` αρχείο και αρχικοποιήστε το `Converter` αντικείμενο:

```csharp
string sampleTxtPath = "YOUR_DOCUMENT_DIRECTORY\sample.txt";
var converter = new Converter(sampleTxtPath);
```

**Γιατί;** Φόρτωση του αρχείου κειμένου σας σε ένα `Converter` Το αντικείμενο είναι απαραίτητο για την πρόσβαση στο περιεχόμενό του εντός του πλαισίου του GroupDocs.Conversion.

#### Βήμα 2: Ορισμός επιλογών μετατροπής

Ορίστε τις επιλογές μετατροπής για τη μετατροπή από μορφή TXT σε μορφή DOC:

```csharp
// Ορισμός επιλογών μετατροπής Word
class WordProcessingConvertOptions { public WordProcessingFileType Format; }
var wordOptions = new WordProcessingConvertOptions { Format = WordProcessingFileType.Doc };
```

**Παράμετροι:** `WordProcessingConvertOptions` καθορίζει τον τύπο αρχείου εξόδου, σε αυτήν την περίπτωση, `.doc`.

#### Βήμα 3: Εκτέλεση μετατροπής

Εκτελέστε τη μετατροπή και αποθηκεύστε το τελικό έγγραφο:

```csharp
string outputFile = "YOUR_OUTPUT_DIRECTORY\txt-converted-to.doc";
// Μετατρέψτε και αποθηκεύστε το αρχείο DOC
converter.Convert(outputFile, wordOptions);
```

**Γιατί;** Ο `Convert` Η μέθοδος επεξεργάζεται το αρχείο εισόδου σύμφωνα με τις καθορισμένες επιλογές και εξάγει ένα `.doc` αρχείο στον καθορισμένο κατάλογό σας.
