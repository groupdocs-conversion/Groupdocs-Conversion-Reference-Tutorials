### Μετατροπή XLTM σε DOC

Αυτή η ενότητα σας καθοδηγεί στη μετατροπή ενός αρχείου XLTm σε έγγραφο του Word χρησιμοποιώντας το GroupDocs.Conversion.

#### Επισκόπηση
Η λειτουργία επιτρέπει την απρόσκοπτη μετατροπή προτύπων Excel με δυνατότητα μακροεντολών (XLTM) σε έγγραφα του Microsoft Word (DOC), διευκολύνοντας την ενσωμάτωση δεδομένων σε όλες τις πλατφόρμες.

#### Βήμα προς βήμα εφαρμογή

**Ορισμός καταλόγου εξόδου και διαδρομής αρχείου**
Βεβαιωθείτε ότι έχετε έναν κατάλογο εξόδου για το αρχείο DOC που έχει μετατραπεί:
```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
if (!Directory.Exists(outputFolder))
{
    Directory.CreateDirectory(outputFolder);
}
string outputFile = Path.Combine(outputFolder, "xltm-converted-to.doc");
```

**Φόρτωση και μετατροπή του αρχείου προέλευσης**
Φορτώστε το αρχείο XLTm χρησιμοποιώντας το `Converter` κλάση και ρύθμιση επιλογών μετατροπής για DOC:
```csharp
using (var converter = new Converter(sourceXltmPath))
{
    var options = new WordProcessingConvertOptions { Format = WordProcessingFileType.Doc };
    converter.Convert(outputFile, options);
}
```

**Εξήγηση:**
- **Κλάση μετατροπέα:** Χειρίζεται τη φόρτωση και τη διαχείριση αρχείων εγγράφων.
- **Επιλογές Μετατροπής Επεξεργασίας Κειμένου:** Ρυθμίζει τις παραμέτρους για τη μετατροπή εγγράφων σε μορφές Word όπως DOC.

#### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι το αρχείο προέλευσης XLTm υπάρχει για να αποφύγετε σφάλματα κατά τη φόρτωση.
- Επαληθεύστε ότι τα δικαιώματα καταλόγου εξόδου επιτρέπουν την εγγραφή νέων αρχείων.
