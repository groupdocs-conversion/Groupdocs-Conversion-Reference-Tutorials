## Οδηγός Εφαρμογής
### Φόρτωση αρχείου GIF πηγής
#### Επισκόπηση
Η φόρτωση του αρχείου GIF πηγής είναι ζωτικής σημασίας για την προετοιμασία του για μετατροπή. Αυτή η λειτουργία διασφαλίζει ότι το GIF σας έχει αρχικοποιηθεί σωστά και είναι έτοιμο για μετατροπή.
**Φόρτωση του αρχείου GIF**
```csharp
using System;
using GroupDocs.Conversion;

namespace FeatureLoadGifFile
{
    internal static class LoadSourceGif
    {
        public static void Run()
        {
            // Ορίστε τη διαδρομή για το αρχείο GIF εισόδου
            string gifFilePath = "YOUR_DOCUMENT_DIRECTORY\\\\sample.gif";

            // Αρχικοποιήστε τον μετατροπέα με τη διαδρομή του αρχείου GIF πηγής
            using (var converter = new Converter(gifFilePath))
            {
                // Το αρχείο GIF έχει πλέον φορτωθεί και είναι έτοιμο για μετατροπή
            }
        }
    }
}
```
*Εξήγηση*: Αυτός ο κώδικας δημιουργεί ένα `Converter` αντικείμενο καθορίζοντας τη διαδρομή του GIF σας, διασφαλίζοντας ότι είναι έτοιμο για επεξεργασία.

### Μετατροπή GIF σε DOCX
#### Επισκόπηση
Αφού φορτώσετε το αρχείο προέλευσης και ρυθμίσετε τις επιλογές, εκτελέστε την πραγματική μετατροπή από μορφή GIF σε μορφή DOCX.
**Εκτέλεση μετατροπής**
```csharp
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

namespace FeatureConvertGifToDocx
{
    internal static class ConvertToDocx
    {
        public static void Run()
        {
            // Ορίστε τον κατάλογο εξόδου και τη διαδρομή αρχείου για το αρχείο DOCX που έχει μετατραπεί
            string outputFolder = Path.Combine(Directory.GetCurrentDirectory(), "YOUR_OUTPUT_DIRECTORY");
            string outputFile = Path.Combine(outputFolder, "gif-converted-to.docx");

            // Φόρτωση του αρχείου GIF πηγής
            using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY\\\\sample.gif"))
            {
                var options = new WordProcessingConvertOptions();

                // Μετατρέψτε και αποθηκεύστε το αρχείο DOCX στην καθορισμένη διαδρομή εξόδου
                converter.Convert(outputFile, options);
            }
        }
    }
}
```
*Εξήγηση*Αυτός ο κώδικας συνδυάζει όλα τα προηγούμενα βήματα φορτώνοντας το GIF, ορίζοντας παραμέτρους μετατροπής και εκτελώντας τη μετατροπή. Το DOCX που προκύπτει αποθηκεύεται στον καθορισμένο κατάλογο.
