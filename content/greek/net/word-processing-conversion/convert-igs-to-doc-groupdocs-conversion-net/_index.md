## Οδηγός Εφαρμογής

Έχοντας έτοιμο το περιβάλλον σας, ας εφαρμόσουμε τη μετατροπή από IGS σε DOC.

### Φόρτωση του αρχείου προέλευσης

Φορτώστε το αρχείο IGS πηγής χρησιμοποιώντας το `Converter` κλάση. Αυτό το βήμα προετοιμάζει το αρχείο για μετατροπή.

**Βήμα 1: Φόρτωση του αρχείου IGS**
```csharp
using System.IO;
using GroupDocs.Conversion;

// Ορίστε τις διαδρομές εισόδου και εξόδου σας
string inputFile = "YOUR_DOCUMENT_DIRECTORY/sample.igs";
string outputFolder = "YOUR_OUTPUT_DIRECTORY";

using (var converter = new Converter(inputFile))
{
    Console.WriteLine("Source file loaded.");
}
```

### Καθορισμός επιλογών μετατροπής

Στη συνέχεια, καθορίστε τις επιλογές μετατροπής για να βεβαιωθείτε ότι η έξοδος είναι σε μορφή DOC.

**Βήμα 2: Ορισμός επιλογών μετατροπής**
```csharp
using GroupDocs.Conversion.Options.Convert;

WordProcessingConvertOptions options = new WordProcessingConvertOptions
{
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc
};
```

### Εκτέλεση της μετατροπής

Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο DOC.

**Βήμα 3: Μετατροπή και αποθήκευση**
```csharp
string outputFile = Path.Combine(outputFolder, "igs-converted-to.doc");

// Εκτελέστε τη μετατροπή και αποθηκεύστε την έξοδο
converter.Convert(outputFile, options);

Console.WriteLine("Conversion completed. Output saved to: " + outputFile);
```

- **Συνηθισμένο πρόβλημα:** Σφάλματα "Δεν βρέθηκε αρχείο"—βεβαιωθείτε ότι οι διαδρομές των αρχείων σας είναι σωστές.
