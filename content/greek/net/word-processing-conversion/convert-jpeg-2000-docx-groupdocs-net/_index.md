## Οδηγός Εφαρμογής

### Μετατροπή JPEG 2000 σε μορφή DOCX

#### Επισκόπηση
Μάθετε πώς να μετατρέπετε απρόσκοπτα μια εικόνα JPEG 2000 (JP2) σε έγγραφο Word σε μορφή DOCX, ιδανικό για επεξεργασία ή ενσωμάτωση οπτικών δεδομένων με περιεχόμενο κειμένου.

##### Βήμα 1: Ρύθμιση διαδρομών αρχείων
Ορίστε τις διαδρομές για το αρχείο JP2 και την έξοδο DOCX:
```csharp
string sourceFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.jp2");
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "jp2-converted-to.docx");
```

##### Βήμα 2: Φόρτωση του αρχείου προέλευσης
Αρχικοποίηση του `Converter` αντικείμενο:
```csharp
using (var converter = new Converter(sourceFilePath))
{
    // Προχωρήστε στη ρύθμιση των επιλογών μετατροπής.
}
```

##### Βήμα 3: Διαμόρφωση επιλογών μετατροπής
Ρύθμιση επιλογών μετατροπής επεξεργασίας κειμένου για μορφή DOCX:
```csharp
var options = new WordProcessingConvertOptions();
```

##### Βήμα 4: Εκτελέστε τη μετατροπή
Εκτελέστε και αποθηκεύστε τη μετατροπή:
```csharp
converter.Convert(outputFile, options);
```
