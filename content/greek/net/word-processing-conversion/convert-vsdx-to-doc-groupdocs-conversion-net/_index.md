## Οδηγός Εφαρμογής
### Μετατροπή VSDX σε DOC
Αυτή η ενότητα θα σας καθοδηγήσει στη μετατροπή ενός αρχείου Visio (.vsdx) σε ένα έγγραφο του Word (.doc).

#### Βήμα 1: Ορισμός διαδρομών για αρχεία και καταλόγους
Ξεκινήστε ορίζοντας τις διαδρομές για το έγγραφο προέλευσης και τον κατάλογο εξόδου.
```csharp
string documentPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "SAMPLE_VSDX");
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "vsdx-converted-to.doc");

// Βεβαιωθείτε ότι ο κατάλογος εξόδου υπάρχει.
Directory.CreateDirectory(outputFolder);
```

#### Βήμα 2: Φόρτωση και μετατροπή του αρχείου VSDX
Στη συνέχεια, φορτώστε το αρχείο προέλευσης χρησιμοποιώντας το GroupDocs.Conversion και διαμορφώστε το για μετατροπή.
```csharp
using (var converter = new Converter(documentPath))
{
    // Ρύθμιση παραμέτρων επιλογών μετατροπής για τη μορφή επεξεργασίας κειμένου (DOC).
    var options = new WordProcessingConvertOptions { Format = FileType.Doc };

    // Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο DOC.
    converter.Convert(outputFile, options);
}
```
**Εξήγηση:**
- `Converter`Φορτώνει το αρχείο VSDX.
- `WordProcessingConvertOptions`: Καθορίζει ότι η μορφή εξόδου θα πρέπει να είναι DOC.
- `converter.Convert()`: Εκτελεί τη διαδικασία μετατροπής.

### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι όλες οι διαδρομές έχουν καθοριστεί σωστά για να αποφύγετε σφάλματα "δεν βρέθηκε αρχείο".
