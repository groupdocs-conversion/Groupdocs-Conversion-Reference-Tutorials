### Επισκόπηση λειτουργιών: Μετατροπή MPX σε DOC

Αυτή η λειτουργία αξιοποιεί τις ισχυρές δυνατότητες του GroupDocs.Conversion για τη μετατροπή αρχείων δεδομένων έργου από μορφή MPX σε επεξεργάσιμα έγγραφα Word. Ας δούμε αναλυτικά κάθε βήμα υλοποίησης:

#### Βήμα 1: Ορισμός καταλόγων εισόδου και εξόδου

Ξεκινήστε καθορίζοντας πού βρίσκεται το αρχείο MPX πηγής σας και πού θέλετε να αποθηκευτεί το αρχείο DOC που έχει μετατραπεί.

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY"; // Αντικατάσταση με την πραγματική διαδρομή καταλόγου
string outputFileDirectory = "YOUR_OUTPUT_DIRECTORY"; // Αντικατάσταση με την πραγματική διαδρομή καταλόγου

string sourceMpxFilePath = Path.Combine(documentDirectory, "sample.mpx"); 
string outputFile = Path.Combine(outputFileDirectory, "mpx-converted-to.doc");
```

#### Βήμα 2: Αρχικοποίηση του μετατροπέα

Δημιουργήστε μια παρουσία του `Converter` τάξη χρησιμοποιώντας το αρχείο MPX σας.

```csharp
using (var converter = new GroupDocs.Conversion.Converter(sourceMpxFilePath))
{
    // Η λογική μετατροπής θα προστεθεί εδώ
}
```

#### Βήμα 3: Ορισμός επιλογών μετατροπής για τη μορφή DOC

Ορίστε τις επιλογές μετατροπής για να καθορίσετε τη μορφή προορισμού ως DOC.

```csharp
WordProcessingConvertOptions options = new WordProcessingConvertOptions 
{ 
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc 
};
```

#### Βήμα 4: Εκτελέστε τη μετατροπή

Χρησιμοποιήστε το `Convert` τη μέθοδο της παρουσίας του μετατροπέα, μεταβιβάζοντας τη διαδρομή του αρχείου εξόδου και τις επιλογές μετατροπής.

```csharp
converter.Convert(outputFile, options);
```
