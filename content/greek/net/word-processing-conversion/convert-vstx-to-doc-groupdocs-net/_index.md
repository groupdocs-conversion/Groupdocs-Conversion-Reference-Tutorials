## Οδηγός Εφαρμογής

### Μετατροπή αρχείου VSTX σε μορφή DOC

Ο κύριος στόχος είναι η μετατροπή ενός αρχείου VSTX σε μορφή DOC χρησιμοποιώντας το GroupDocs.Conversion για .NET. Ακολουθήστε τα παρακάτω βήματα:

#### Βήμα 1: Ορισμός διαδρομών και φόρτωση αρχείων

Καθορίστε τις διαδρομές προέλευσης και εξόδου για τα αρχεία σας:

```csharp
string sourceVstxPath = "YOUR_DOCUMENT_DIRECTORY\\sample.vstx";
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "vstx-converted-to.doc");
```

#### Βήμα 2: Αρχικοποίηση μετατροπέα

Χρησιμοποιήστε το `Converter` κλάση για να φορτώσετε το αρχείο VSTX σας:

```csharp
using (var converter = new Converter(sourceVstxPath))
{
    // Η λογική μετατροπής θα προστεθεί εδώ.
}
```

Αυτό το βήμα διασφαλίζει ότι το αρχείο σας είναι έτοιμο για μετατροπή.

#### Βήμα 3: Ορισμός επιλογών μετατροπής

Ορίστε τις επιλογές που απαιτούνται για τη μετατροπή σε μορφή DOC:

```csharp
WordProcessingConvertOptions options = new WordProcessingConvertOptions { Format = WordProcessingFileType.Doc };
```

Ο `WordProcessingConvertOptions` Η κλάση σάς επιτρέπει να καθορίσετε διάφορες ρυθμίσεις, όπως τη μορφή του αρχείου προορισμού.

#### Βήμα 4: Εκτέλεση μετατροπής

Εκτελέστε τη μετατροπή και αποθηκεύστε την έξοδο:

```csharp
converter.Convert(outputFile, options);
```

Αυτή η μέθοδος μετατρέπει το αρχείο VSTX σε έγγραφο DOC, αποθηκεύοντάς το στην καθορισμένη τοποθεσία.
