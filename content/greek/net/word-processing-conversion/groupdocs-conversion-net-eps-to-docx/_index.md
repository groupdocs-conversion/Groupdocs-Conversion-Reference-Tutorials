## Οδηγός Εφαρμογής
### Φόρτωση και μετατροπή αρχείου EPS σε DOCX
Αυτή η ενότητα σας καθοδηγεί στη φόρτωση ενός αρχείου EPS και στη μετατροπή του σε μορφή DOCX χρησιμοποιώντας το GroupDocs.Conversion.

#### Βήμα 1: Ορισμός διαδρομών αρχείων
Καθορίστε τις διαδρομές για το αρχείο EPS προέλευσης και τον κατάλογο εξόδου:
```csharp
string sourceFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.eps");
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "eps-converted-to.docx");
```

#### Βήμα 2: Φόρτωση του αρχείου προέλευσης
Φορτώστε το αρχείο EPS χρησιμοποιώντας το GroupDocs.Conversion:
```csharp
using (var converter = new Converter(sourceFilePath))
{
    // Η λογική μετατροπής θα προστεθεί εδώ
}
```
*Γιατί αυτό το βήμα;*
Η φόρτωση του αρχείου προέλευσης αρχικοποιεί τη διαδικασία μετατροπής, επιτρέποντάς σας να εφαρμόσετε περαιτέρω διαμορφώσεις.

#### Βήμα 3: Ορισμός επιλογών μετατροπής
Ορίστε τη μορφή εξόδου χρησιμοποιώντας το WordProcessingConvertOptions:
```csharp
var options = new WordProcessingConvertOptions();
```
*Σε τι χρησιμεύουν αυτές οι παράμετροι;*
Καθορίζουν ότι το έγγραφο-στόχος θα πρέπει να είναι σε μορφή επεξεργασίας κειμένου (DOCX).

#### Βήμα 4: Εκτελέστε τη μετατροπή
Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο DOCX:
```csharp
converter.Convert(outputFile, options);
```
