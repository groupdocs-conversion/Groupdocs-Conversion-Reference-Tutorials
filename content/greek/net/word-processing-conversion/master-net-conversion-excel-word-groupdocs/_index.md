## Οδηγός Εφαρμογής

Αυτή η ενότητα θα αναλύσει τη διαδικασία σε διαχειρίσιμα βήματα, αναλύοντας κάθε χαρακτηριστικό.

### Φόρτωση αρχείου πηγής
#### Επισκόπηση
Το πρώτο βήμα είναι η φόρτωση του αρχείου XLSX για μετατροπή. Αυτό προετοιμάζει το έγγραφό σας για επόμενες εργασίες επεξεργασίας.

#### Βήματα
**1. Ορισμός καταλόγου εγγράφων**
Ξεκινήστε καθορίζοντας πού βρίσκονται τα αρχεία προέλευσης:

```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY\/";
```

**2. Φόρτωση του αρχείου**
Χρησιμοποιήστε το `Converter` κλάση για να φορτώσετε το αρχείο XLSX. Αυτό το βήμα διασφαλίζει ότι το έγγραφο είναι έτοιμο για μετατροπή.

```csharp
using (var converter = new Converter(dataDir + "sample.xlsx"))
{
    // Το αρχείο έχει πλέον φορτωθεί και είναι διαθέσιμο για μετατροπή.
}
```

### Μετατροπή XLSX σε DOCX
#### Επισκόπηση
Μετά τη φόρτωση του αρχείου προέλευσης, η μετατροπή του σε έγγραφο Word (μορφή DOCX) περιλαμβάνει τον ορισμό συγκεκριμένων επιλογών.

#### Βήματα
**1. Ορισμός καταλόγου εξόδου**
Ορίστε πού θα αποθηκευτούν τα αρχεία που έχουν μετατραπεί:

```csharp
string outputDir = @"YOUR_OUTPUT_DIRECTORY\/";
```

**2. Ορισμός διαδρομής αρχείου εξόδου**
Συνδυάστε τον κατάλογο και το όνομα αρχείου για λόγους σαφήνειας:

```csharp
string outputFile = Path.Combine(outputDir, "xlsx-converted-to.docx");
```

**3. Ορισμός επιλογών μετατροπής**
Καθορίστε την επιθυμητή μορφή χρησιμοποιώντας `WordProcessingConvertOptions`.

```csharp
using (var converter = new Converter(dataDir + "sample.xlsx"))
{
    var options = new WordProcessingConvertOptions();
    
    // Μετατροπή και αποθήκευση σε DOCX
    converter.Convert(outputFile, options);
}
```

#### Επεξήγηση παραμέτρων
- **`Converter(dataDir + "sample.xlsx")`**: Αρχικοποιεί τη διαδικασία μετατροπής με το αρχείο προέλευσης.
- **`WordProcessingConvertOptions()`**Διαμορφώνει ρυθμίσεις που αφορούν συγκεκριμένα τις μετατροπές εγγράφων του Word.
