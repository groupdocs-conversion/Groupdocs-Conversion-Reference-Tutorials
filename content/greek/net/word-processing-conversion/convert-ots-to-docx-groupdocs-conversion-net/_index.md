## Οδηγός Εφαρμογής

#### Επισκόπηση
Αυτή η ενότητα περιγράφει τη διαδικασία φόρτωσης ενός αρχείου προτύπου υπολογιστικού φύλλου OpenDocument (.ots) και τη μετατροπή του σε έγγραφο Open XML του Microsoft Word (.docx).

Δημιουργήστε μια παρουσία του `Converter` κλάση για τον χειρισμό λειτουργιών μετατροπής.
```csharp
// Αρχικοποιήστε τον μετατροπέα με τη διαδρομή αρχείου OTS πηγής

{
    // Η λογική μετατροπής ισχύει εδώ

```

#### Βήμα 2: Ρύθμιση επιλογών μετατροπής εγγράφων Word
Ορίστε επιλογές ειδικά για τη μορφή DOCX.
```csharp
// Ρύθμιση των επιλογών μετατροπής εγγράφων Word
var convertOptions = new DocxConvertOptions();
```

#### Βήμα 3: Εκτελέστε τη μετατροπή
Εκτελέστε τη μετατροπή καλώντας το `Convert` μέθοδος με τις απαραίτητες παραμέτρους.
```csharp
// Μετατρέψτε το OTS σε DOCX και αποθηκεύστε το αρχείο εξόδου
converter.Convert("outputFilePath.docx", convertOptions);
```

### Επεξήγηση Παραμέτρων και Μεθόδων
- **Μετατροπέας**: Διαχειρίζεται τη φόρτωση και τη μετατροπή εγγράφων. Ο κατασκευαστής απαιτεί ένα όρισμα διαδρομής αρχείου.
- **DocxConvertOptions**Καθορίζει ρυθμίσεις για τη μετατροπή DOCX, όπως το μέγεθος σελίδας και τα περιθώρια.
- **Μέθοδος μετατροπής**: Εκτελεί την πραγματική μετατροπή αρχείου, απαιτώντας μια διαδρομή αρχείου εξόδου και επιλογές μετατροπής.

#### Βασικές επιλογές διαμόρφωσης
Προσαρμόζω `DocxConvertOptions` για να προσαρμόσετε αποτελεσματικά τις ρυθμίσεις του εγγράφου σας.
