**Τι θα μάθετε:**
- Πώς να ρυθμίσετε και να χρησιμοποιήσετε το GroupDocs.Conversion για .NET.
- Βήματα για τη φόρτωση ενός αρχείου JPEG 2000 πηγής.

**Επισκόπηση:** Αυτή η λειτουργία επιδεικνύει τη φόρτωση ενός αρχείου εικόνας JPEG 2000 χρησιμοποιώντας το GroupDocs.Conversion.

### Ρύθμιση παραμέτρων επιλογών μετατροπής για μορφή DOC
**Επισκόπηση:** Ρυθμίστε τις απαραίτητες επιλογές για τη μετατροπή αρχείων σε μορφή εγγράφου Microsoft Word.

#### Βήμα 1: Ρύθμιση επιλογών μετατροπής
Δημιουργήστε μια παρουσία του `WordProcessingConvertOptions` και καθορίστε τη μορφή προορισμού.
```csharp
using GroupDocs.Conversion.Options.Convert;

WordProcessingConvertOptions options = new WordProcessingConvertOptions 
{
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc
};
```

### Μετατροπή και αποθήκευση εγγράφου σε μορφή DOC
**Επισκόπηση:** Μετατρέψτε το φορτωμένο αρχείο JPF σε έγγραφο DOC χρησιμοποιώντας τις διαμορφωμένες επιλογές.

#### Βήμα 1: Ορισμός καταλόγου εξόδου και διαδρομής αρχείου
```csharp
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputDirectory, "jpf-converted-to.doc");
```

#### Βήμα 2: Εκτέλεση μετατροπής
Επικαλέστε το `Convert` μέθοδος στο δικό σας `converter` για να αποθηκεύσετε το αρχείο DOC που έχει μετατραπεί.
```csharp
using (var converter = new Converter(documentDirectory + "/sample.jpf"))
{
    // Μετατρέψτε και αποθηκεύστε το αρχείο DOC χρησιμοποιώντας καθορισμένες επιλογές
    converter.Convert(outputFile, options);
}
```
