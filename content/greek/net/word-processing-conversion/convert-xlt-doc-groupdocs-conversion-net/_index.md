### Μετατροπή XLT σε μορφή DOC

**Επισκόπηση:**
Η μετατροπή ενός αρχείου XLT σε μορφή DOC επιτρέπει στους χρήστες να αξιοποιήσουν τις λειτουργίες εμπλουτισμένου κειμένου του Word, διευκολύνοντας την επεξεργασία και την κοινή χρήση εγγράφων.

#### Βήμα προς βήμα εφαρμογή:

**1. Ορίστε τη διαδρομή εξόδου και τις επιλογές μετατροπής:**

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

namespace FeatureConvertXltToDocFormat
{
    public static class ConvertXltToDoc
    {
        public static void Execute()
        {
            // Ορίστε τη διαδρομή του καταλόγου εξόδου
            string outputFolder = "YOUR_OUTPUT_DIRECTORY";
            string outputFile = Path.Combine(outputFolder, "xlt-converted-to.doc");

            // Αντικαταστήστε με την πραγματική διαδρομή καταλόγου εγγράφου για το αρχείο XLT πηγής
            string sampleXltPath = "YOUR_DOCUMENT_DIRECTORY\\sample.xlt";

            // Αρχικοποίηση του αντικειμένου Converter με μια διαδρομή αρχείου XLT
            using (var converter = new GroupDocs.Conversion.Converter(sampleXltPath))
            {
                // Ρύθμιση επιλογών μετατροπής για τη μορφή DOC της επεξεργασίας κειμένου
                WordProcessingConvertOptions options = new WordProcessingConvertOptions 
                { 
                    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc 
                };

                // Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο DOC εξόδου
                converter.Convert(outputFile, options);

                Console.WriteLine("Conversion to DOC completed successfully.");
            }
        }
    }
}
```

**Εξήγηση:** Εδώ, στήνουμε `WordProcessingConvertOptions` για να καθορίσετε τη μορφή προορισμού ως DOC. Στη συνέχεια, η διαδικασία μετατροπής εκτελείται με αυτές τις ρυθμίσεις.
