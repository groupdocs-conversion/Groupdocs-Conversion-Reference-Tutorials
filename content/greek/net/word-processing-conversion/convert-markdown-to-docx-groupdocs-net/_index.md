### Βασική Αρχικοποίηση

Μόλις εγκατασταθεί, μπορείτε να αρχικοποιήσετε και να ρυθμίσετε το GroupDocs.Conversion με μερικές γραμμές κώδικα C#:

```csharp
using GroupDocs.Conversion;
using System.IO;

string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string inputFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.md");
string outputFile = Path.Combine(outputFolder, "md-converted-to.docx");

// Αρχικοποιήστε τον μετατροπέα με το αρχείο markdown
using (var converter = new Converter(inputFile))
{
    // Ορίστε επιλογές μετατροπής για DOCX
    var options = new WordProcessingConvertOptions();
    
    // Εκτελέστε τη μετατροπή και αποθηκεύστε το αποτέλεσμα
    converter.Convert(outputFile, options);
}
```

## Οδηγός Εφαρμογής

### Μετατροπή Markdown σε DOCX

Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε αρχεία markdown σε μορφή DOCX χρησιμοποιώντας το GroupDocs.Conversion. Δείτε πώς λειτουργεί:

#### Βήμα 1: Προετοιμασία διαδρομών αρχείων
Ρυθμίστε τους καταλόγους εισόδου και εξόδου για εύκολη διαχείριση διαδρομών.

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// Κατασκευάστε τις πλήρεις διαδρομές αρχείων
string inputFile = Path.Combine(documentDirectory, "sample.md");
string outputFile = Path.Combine(outputDirectory, "md-converted-to.docx");
```

#### Βήμα 2: Φόρτωση και μετατροπή
Φορτώστε το αρχείο markdown και προετοιμάστε το για μετατροπή χρησιμοποιώντας συγκεκριμένες επιλογές.

```csharp
using (var converter = new Converter(inputFile))
{
    var options = new WordProcessingConvertOptions();
    converter.Convert(outputFile, options);
}
```

- **Παράμετροι:** `inputFile` είναι η διαδρομή προς το αρχείο markdown πηγής. `outputFile` καθορίζει πού θα αποθηκευτεί το DOCX που έχει μετατραπεί.
- **Σκοπός της μεθόδου:** Ο `Converter` Η κλάση χειρίζεται τη διαδικασία μετατροπής από markdown σε μορφή DOCX.

### Διαχείριση διαδρομών αρχείων
Η συνεπής και σαφής διαχείριση των διαδρομών αρχείων διασφαλίζει την ομαλή λειτουργία σε οποιαδήποτε εφαρμογή.

#### Κατασκευή μονοπατιών
Χρησιμοποιήστε το `System.IO.Path.Combine()` Μέθοδος για τη δημιουργία πλήρων διαδρομών συνδυάζοντας ονόματα καταλόγων με ονόματα αρχείων.

