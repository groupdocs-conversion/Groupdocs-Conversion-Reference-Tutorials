## Οδηγός Εφαρμογής

### Φόρτωση και μετατροπή αρχείου EPUB σε μορφή DOC

#### Επισκόπηση

Η μετατροπή ενός αρχείου EPUB σε μορφή εγγράφου του Microsoft Word (.doc) επιτρέπει την εύκολη επεξεργασία και κοινή χρήση περιεχομένου κειμένου. Δείτε πώς:

##### 1. Ορισμός καταλόγων

Καθορίστε πού θα βρίσκονται τα αρχεία EPUB εισόδου και τα αρχεία DOC εξόδου.

```csharp
string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY";

// Βεβαιωθείτε ότι ο κατάλογος εξόδου υπάρχει ή δημιουργήστε τον.
if (!Directory.Exists(outputDirectory))
{
    Directory.CreateDirectory(outputDirectory);
}
```

##### 2. Καθορίστε διαδρομές αρχείων

Προσδιορίστε το αρχείο EPUB προέλευσης και την επιθυμητή διαδρομή εξόδου DOC.

```csharp
string inputFile = Path.Combine(documentDirectory, "sample.epub");
string outputFile = Path.Combine(outputDirectory, "epub-converted-to.doc");
```

##### 3. Αρχικοποίηση μετατροπέα

Ρυθμίστε τη διαδικασία μετατροπής χρησιμοποιώντας το GroupDocs.Conversion.

```csharp
using (var converter = new Converter(inputFile))
{
    var options = new WordProcessingConvertOptions { Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc };
    converter.Convert(outputFile, options);
}
```

**Εξήγηση**: 
- `Converter`: Αρχικοποιεί με το αρχείο εισόδου.
- `WordProcessingConvertOptions`: Καθορίζει τις ρυθμίσεις μετατροπής.
- `Format`Ορίζει την έξοδο ως DOC.
