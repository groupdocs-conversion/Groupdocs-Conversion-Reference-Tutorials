#### Βήμα 2: Μετατροπή σε μορφή DOC

Καθορίστε τις επιλογές μετατροπής για ένα έγγραφο του Word:

```csharp
// Ρύθμιση επιλογών μετατροπής για τη μορφή εγγράφου Word
var convertOptions = new WordProcessingConvertOptions();

// Εκτελέστε τη μετατροπή

{
    var resultPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "converted.doc");
    converter.Convert(resultPath, convertOptions);
    Console.WriteLine("Conversion to DOC completed successfully.");
}
```
