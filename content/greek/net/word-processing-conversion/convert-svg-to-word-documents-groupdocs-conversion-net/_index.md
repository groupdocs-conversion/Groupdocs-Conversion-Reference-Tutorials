## Οδηγός Εφαρμογής

### Μετατροπή SVG σε έγγραφο του Word
Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε τα αρχεία SVG σας σε μια πιο καθολικά προσβάσιμη μορφή εγγράφου Word. Η βιβλιοθήκη GroupDocs.Conversion χειρίζεται αυτήν την εργασία αποτελεσματικά με ελάχιστο κώδικα.

#### Βήμα 1: Ορισμός διαδρομών αρχείων και φόρτωση SVG πηγαίου κώδικα
Αρχικά, καθορίστε τις διαδρομές για τους καταλόγους εισόδου και εξόδου:

```csharp
using System.IO;

// Ορίστε διαδρομές αρχείων χρησιμοποιώντας σύμβολα κράτησης θέσης
string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.svg");
string outputFolder = Constants.GetOutputDirectoryPath(); // Ορίστε συνεπή διαδρομή όπως "YOUR_OUTPUT_DIRECTORY"
string outputFile = Path.Combine(outputFolder, "svg-converted-to.doc");

// Φόρτωση του αρχείου SVG πηγής
using (var converter = new Converter(inputFilePath))
{
    // Οι επιλογές και η διαδικασία μετατροπής θα οριστούν εδώ...
}
```

**Εξήγηση:**
- Ο `inputFilePath` μεταβλητά σημεία στο αρχείο SVG σας.
- `outputFile` είναι το σημείο όπου θα αποθηκευτεί το αρχείο DOC που έχει μετατραπεί.

#### Βήμα 2: Διαμόρφωση επιλογών μετατροπής
Στη συνέχεια, ρυθμίστε τις επιλογές μετατροπής για τη μετατροπή ενός SVG σε έγγραφο του Word:

```csharp
// Δημιουργία WordProcessingConvertOptions για μορφή .doc
var options = new WordProcessingConvertOptions { Format = WordProcessingFileType.Doc };

// Εκτέλεση μετατροπής και αποθήκευση αρχείου εξόδου
converter.Convert(outputFile, options);
```

**Εξήγηση:**
- `WordProcessingConvertOptions` Καθορίζει τη μορφή DOC προορισμού.
- Ο `Format` η ιδιότητα έχει οριστεί σε `Doc` για συμβατότητα με το Microsoft Word.
