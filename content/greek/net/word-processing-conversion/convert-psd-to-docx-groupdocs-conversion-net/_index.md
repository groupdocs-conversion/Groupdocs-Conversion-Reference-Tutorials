## Οδηγός Εφαρμογής

### Βήμα 1: Φόρτωση αρχείου PSD προέλευσης

Ξεκινήστε φορτώνοντας το αρχείο PSD πηγής στην εφαρμογή .NET για μετατροπή.

**Ορισμός διαδρομής εγγράφου:**
```csharp
string psdFilePath = "YOUR_DOCUMENT_DIRECTORY/sample.psd";
```

**Φόρτωση του αρχείου PSD:**
Χρησιμοποιήστε το GroupDocs.Conversion για να φορτώσετε το αρχείο, προετοιμάζοντάς το για επόμενες λειτουργίες:
```csharp

{
    // Έτοιμο για εκτέλεση μετατροπής.
}
```

### Βήμα 2: Μετατροπή PSD σε μορφή DOCX

Αφού φορτώσετε το αρχείο PSD, προχωρήστε στη μετατροπή του σε μορφή επεξεργασίας κειμένου.

**Ορισμός διαδρομών εισόδου και εξόδου:**
Ορίστε διαδρομές για το αρχείο PSD εισόδου και το αρχείο DOCX εξόδου:
```csharp
string inputFilePath = "YOUR_DOCUMENT_DIRECTORY/sample.psd";
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "psd-converted-to.docx");
```

**Καθορίστε Επιλογές Μετατροπής:**
Υποδείξτε τη μετατροπή σε μορφή επεξεργασίας κειμένου (DOCX):
```csharp
var options = new WordProcessingConvertOptions();
```

**Εκτελέστε τη μετατροπή:**
Αποθηκεύστε το αρχείο DOCX που έχει μετατραπεί χρησιμοποιώντας τις καθορισμένες επιλογές:
```csharp
using (var converter = new Converter(inputFilePath))
{
    converter.Convert(outputFile, options);
}
```
