### Μετατροπή JPEG σε DOCX
Αυτή η λειτουργία μετατρέπει μια στατική εικόνα JPEG σε ένα επεξεργάσιμο έγγραφο του Word χρησιμοποιώντας το GroupDocs.Conversion για .NET.

#### Βήμα 1: Ορισμός διαδρομών αρχείων
Καθορίστε τους καταλόγους εισόδου και εξόδου:
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputFileDir = "YOUR_OUTPUT_DIRECTORY";
string sampleJpegPath = Path.Combine(documentDirectory, "sample.jpeg");
string outputFilePath = Path.Combine(outputFileDir, "jpeg-converted-to.docx");
```
Αντικαθιστώ `"YOUR_DOCUMENT_DIRECTORY"` και `"YOUR_OUTPUT_DIRECTORY"` με τις πραγματικές σου διαδρομές.

#### Βήμα 2: Φόρτωση του αρχείου JPEG
Χρησιμοποιήστε το `Converter` κλάση για να φορτώσετε το αρχείο πηγαίου κώδικα:
```csharp
using (var converter = new Converter(sampleJpegPath))
{
    // Η λογική μετατροπής θα εφαρμοστεί εδώ.
}
```
Αυτό το βήμα ξεκινά τη διαδικασία μετατροπής φορτώνοντας την εικόνα JPEG στη μνήμη.

#### Βήμα 3: Ορισμός επιλογών μετατροπής DOCX
Ρυθμίστε τον τρόπο μετατροπής του JPEG σε έγγραφο του Word:
```csharp
var options = new WordProcessingConvertOptions();
```
Ο `WordProcessingConvertOptions` Η κλάση καθορίζει ότι η μορφή-στόχος μας είναι DOCX. Προσαρμόστε αυτές τις ρυθμίσεις για προηγμένες μετατροπές.

#### Βήμα 4: Εκτελέστε τη μετατροπή
Τέλος, μετατρέψτε και αποθηκεύστε το αρχείο σας:
```csharp
converter.Convert(outputFilePath, options);
```
Αυτή η μέθοδος εκτελεί τη μετατροπή και γράφει την έξοδο σε `outputFilePath`.
