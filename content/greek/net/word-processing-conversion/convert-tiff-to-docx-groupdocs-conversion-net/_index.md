### Μετατροπή TIFF σε DOCX

**Επισκόπηση**Τώρα που φορτώθηκε το αρχείο TIFF, ας το μετατρέψουμε σε μορφή DOCX.

#### Βήμα 2: Ρύθμιση επιλογών μετατροπής

```csharp
// Ορίστε τον φάκελο εξόδου και το όνομα αρχείου για το μετατρεπόμενο έγγραφο
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "tiff-converted-to.docx");

// Αρχικοποιήστε τον μετατροπέα με το αρχείο TIFF πηγής
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_TIFF"))
{
    // Ρύθμιση επιλογών μετατροπής για τη μορφή επεξεργασίας κειμένου
    var options = new WordProcessingConvertOptions();

    // Μετατρέψτε και αποθηκεύστε την έξοδο ως αρχείο DOCX
    converter.Convert(outputFile, options);
}
```

**Εξήγηση**: Με ρύθμιση `WordProcessingConvertOptions`, καθορίζετε ότι η μορφή προορισμού είναι DOCX. Το `converter.Convert` Στη συνέχεια, η μέθοδος επεξεργάζεται το αρχείο και το αποθηκεύει στον καθορισμένο κατάλογο.
