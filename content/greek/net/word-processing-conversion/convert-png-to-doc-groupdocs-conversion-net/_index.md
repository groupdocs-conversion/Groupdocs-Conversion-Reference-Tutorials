#### Βασική Αρχικοποίηση και Ρύθμιση

Για να ξεκινήσετε με το GroupDocs.Conversion στο έργο σας:
1. **Αναφορά στη Βιβλιοθήκη**Βεβαιωθείτε ότι αναφέρεται στο έργο σας.
2. **Αρχικοποίηση του μετατροπέα**: Δημιουργήστε μια παρουσία του `Converter` κλάση για τη διαχείριση μετατροπών αρχείων.

Ακολουθεί ένα βασικό παράδειγμα ρύθμισης:
```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main()
    {
        // Ρύθμιση διαδρομών για αρχεία προέλευσης και εξόδου
        const string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
        const string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

        // Ορίστε τη διαδρομή για το αρχείο PNG και το αρχείο DOC που προκύπτει
        string pngFilePath = Path.Combine(documentDirectory, "sample.png");
        string docOutputPath = Path.Combine(outputDirectory, "png-converted-to.doc");

        // Αρχικοποιήστε την κλάση Converter με το αρχείο PNG πηγής
        using (var converter = new Converter(pngFilePath))
        {
            var convertOptions = new WordProcessingConvertOptions
            {
                Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc
            };

            // Μετατρέψτε και αποθηκεύστε το αρχείο DOC εξόδου
            converter.Convert(docOutputPath, convertOptions);
        }
    }
}
```
