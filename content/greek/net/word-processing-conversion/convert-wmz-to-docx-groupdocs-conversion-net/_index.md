## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Μετατροπή WMZ σε DOCX

Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε απρόσκοπτα ένα αρχείο WMZ σε μορφή DOCX χρησιμοποιώντας το GroupDocs.Conversion για .NET.

#### Βήμα 1: Ορισμός καταλόγων και φόρτωση αρχείου πηγαίου κώδικα
Ξεκινήστε ορίζοντας τους καταλόγους εγγράφων σας και φορτώνοντας το αρχείο WMZ πηγής:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY/";

// Φόρτωση του αρχείου WMZ πηγής
group (var converter = new Converter(Path.Combine(documentDirectory, "sample.wmz")))
{
    Console.WriteLine("WMZ file loaded successfully.");
}
```

#### Βήμα 2: Ρύθμιση επιλογών μετατροπής
Ρυθμίστε τις παραμέτρους των επιλογών μετατροπής για μορφές επεξεργασίας κειμένου:

```csharp
var options = new WordProcessingConvertOptions();

// Ορισμός διαδρομής εξόδου για DOCX
string outputFile = Path.Combine(outputDirectory, "wmz-converted-to.docx");
```

#### Βήμα 3: Εκτελέστε τη μετατροπή
Εκτελέστε τη μετατροπή από WMZ σε DOCX και αντιμετωπίστε τυχόν πιθανά προβλήματα:

```csharp
try
{
    converter.Convert(outputFile, options);
    Console.WriteLine("Conversion successful. Output saved at: " + outputFile);
}
catch (Exception ex)
{
    Console.WriteLine("Error during conversion: " + ex.Message);
}
```
