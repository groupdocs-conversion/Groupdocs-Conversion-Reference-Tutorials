Αυτό διευκολύνει τους επαγγελματίες υγείας και τους ερευνητές να μοιράζονται τα ευρήματά τους.

- Φορτώστε ένα αρχείο DICOM χρησιμοποιώντας το GroupDocs.Conversion.
- Μετατρέψτε ένα αρχείο DICOM σε μορφή DOC χωρίς κόπο.

## Οδηγός Εφαρμογής

Αυτός ο οδηγός σας καθοδηγεί στη φόρτωση και μετατροπή αρχείων DICOM σε μορφή DOC.

### Χαρακτηριστικό 1: Φόρτωση αρχείου DCM

#### Επισκόπηση
Η φόρτωση ενός αρχείου DICOM είναι το πρώτο βήμα πριν από οποιαδήποτε μετατροπή. Το GroupDocs.Conversion το απλοποιεί χρησιμοποιώντας το `Converter` τάξη.

#### Βήμα προς βήμα εφαρμογή

**Βήμα 1:** Ορισμός της διαδρομής και αρχικοποίηση του μετατροπέα

```csharp
using System;
using GroupDocs.Conversion;

string documentPath = "YOUR_DOCUMENT_DIRECTORY\\sample.dcm"; // Αντικατάσταση με την πραγματική διαδρομή
// Φόρτωση του αρχείου DCM προέλευσης
using (var converter = new Converter(documentPath))
{
    Console.WriteLine("DICOM file loaded successfully.");
}
```

**Εξήγηση:**
- **Διαδρομήεγγράφου**Καθορίστε τον κατάλογο και το όνομα αρχείου του αρχείου DICOM.
- Ο `Converter` Το αντικείμενο χειρίζεται τη φόρτωση και παρέχει μεθόδους για μετατροπή.

### Χαρακτηριστικό 2: Μετατροπή DCM σε DOC

#### Επισκόπηση
Μόλις φορτώσετε ένα αρχείο DICOM, η μετατροπή του σε μορφή εγγράφου Word είναι απρόσκοπτη με το GroupDocs.Conversion.

#### Βήμα προς βήμα εφαρμογή

**Βήμα 1:** Καθορισμός καταλόγου και αρχείου εξόδου

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

string outputDirectory = "YOUR_OUTPUT_DIRECTORY"; // Αντικατάσταση με την πραγματική διαδρομή
string outputFile = Path.Combine(outputDirectory, "dcm-converted-to.doc");
```

**Βήμα 2:** Ορισμός επιλογών μετατροπής και εκτέλεση μετατροπής

```csharp
// Φόρτωση του αρχείου DCM προέλευσης
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY\\sample.dcm")) // Αντικατάσταση με την πραγματική διαδρομή
{
    WordProcessingConvertOptions options = new WordProcessingConvertOptions 
    {
        Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc // Ορισμός μορφής σε DOC
    };
    
    // Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο DOC εξόδου
    converter.Convert(outputFile, options);
    Console.WriteLine("Conversion to DOC completed successfully.");
}
```

**Εξήγηση:**
- **Επεξεργασία ΚειμένουConvertOptions**: Διαμορφώνει τις ρυθμίσεις μετατροπής.
- **Σχήμα και διάταξις βιβλίου**: Καθορίζει ότι η έξοδος θα πρέπει να είναι σε μορφή DOC.
