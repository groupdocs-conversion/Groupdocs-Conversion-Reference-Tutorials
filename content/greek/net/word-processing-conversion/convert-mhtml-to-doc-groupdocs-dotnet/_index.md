## Οδηγός Εφαρμογής
### Δυνατότητα μετατροπής MHTML σε DOC
#### Επισκόπηση
Η κύρια λειτουργία που θα εφαρμόσουμε είναι η μετατροπή ενός αρχείου MHTML σε έγγραφο του Microsoft Word (DOC). Αυτή η ενότητα περιγράφει λεπτομερώς κάθε βήμα που απαιτείται για αυτόν τον μετασχηματισμό χρησιμοποιώντας το GroupDocs.Conversion.

#### Βήμα προς βήμα εφαρμογή
##### Φόρτωση του αρχείου MHTML πηγής
Για να ξεκινήσετε, φορτώστε το αρχείο MHTML πηγής. Βεβαιωθείτε ότι η διαδρομή προς το αρχείο έχει καθοριστεί σωστά.

```csharp
using System.IO;
using GroupDocs.Conversion;

// Ορίστε διαδρομές για καταλόγους εισόδου και εξόδου
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// Αρχικοποιήστε τον μετατροπέα με το αρχείο MHTML πηγής
using (var converter = new Converter(Path.Combine(documentDirectory, "sample.mhtml")))
{
    Console.WriteLine("MHTML file loaded.");
}
```

##### Ορισμός επιλογών μετατροπής για τη μορφή DOC
Στη συνέχεια, καθορίστε τις επιλογές μετατροπής. Εδώ το ορίζουμε για μετατροπή σε μορφές επεξεργασίας κειμένου, συγκεκριμένα `.doc`.

```csharp
using GroupDocs.Conversion.Options.Convert;

// Ορίστε επιλογές μετατροπής για τη μορφή DOC
WordProcessingConvertOptions options = new WordProcessingConvertOptions 
{
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc
};
```

##### Μετατροπή και αποθήκευση του αρχείου MHTML ως αρχείου DOC
Τέλος, εκτελέστε τη μετατροπή και αποθηκεύστε την έξοδο στον επιθυμητό κατάλογο.

```csharp
// Ορίστε τη διαδρομή αρχείου εξόδου για το μετατρεπόμενο έγγραφο
string outputFile = Path.Combine(outputDirectory, "mhtml-converted-to.doc");

// Εκτελέστε τη μετατροπή
converter.Convert(outputFile, options);

Console.WriteLine($"MHTML successfully converted to DOC: {outputFile}");
```
