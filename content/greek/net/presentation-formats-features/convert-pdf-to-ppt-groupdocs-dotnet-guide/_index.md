### Μετατροπή PDF σε PPT

**Επισκόπηση:** Αυτή η λειτουργία δείχνει πώς να μετατρέψετε ένα φορτωμένο αρχείο PDF σε μορφή PowerPoint (PPT).

#### Βήμα 1: Φόρτωση του αρχείου PDF προέλευσης
```csharp
using System.IO;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main(string[] args)
    {
        string outputFolder = "YOUR_OUTPUT_DIRECTORY"; // Ορίστε τη διαδρομή του καταλόγου εξόδου
        string outputFile = Path.Combine(outputFolder, "pdf-converted-to.ppt"); // Ορίστε τη διαδρομή του αρχείου εξόδου

        using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY\\sample.pdf")) // Αντικαταστήστε με την πραγματική διαδρομή PDF
        {
            // Βήμα 2: Ορισμός επιλογών μετατροπής για τη μορφή παρουσίασης PowerPoint
            var options = new PresentationConvertOptions { Format = GroupDocs.Conversion.FileTypes.PresentationFileType.Ppt };

            // Βήμα 3: Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο PPT στην καθορισμένη τοποθεσία
            converter.Convert(outputFile, options);
        }
    }
}
```
- **Παράμετροι:** `outputFolder` και `outputFile` ορίστε πού θα αποθηκευτεί το αρχείο που έχετε μετατρέψει.
- **Επιστρεφόμενες τιμές:** Καμία. Η μετατροπή ολοκληρώθηκε επιτόπου.

**Συμβουλές αντιμετώπισης προβλημάτων:**
- Βεβαιωθείτε ότι η διαδρομή του αρχείου PDF είναι σωστή για να αποτρέψετε σφάλματα "δεν βρέθηκε αρχείο".
- Επαληθεύστε ότι ο κατάλογος εξόδου υπάρχει πριν αποθηκεύσετε τα αρχεία εκεί.
