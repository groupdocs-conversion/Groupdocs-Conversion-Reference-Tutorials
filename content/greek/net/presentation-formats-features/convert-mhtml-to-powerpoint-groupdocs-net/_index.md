#### Βήμα 2: Αρχικοποίηση του μετατροπέα

### Μετατροπή MHTML σε PPT
**Επισκόπηση**Αυτή η ενότητα σάς καθοδηγεί στη μετατροπή ενός αρχείου MHTML σε παρουσίαση PowerPoint χρησιμοποιώντας το GroupDocs.Conversion.

#### Βήμα 1: Ορισμός διαδρομής εξόδου
Δημιουργήστε ή επαληθεύστε τον κατάλογο εξόδου όπου θα αποθηκευτεί το αρχείο που έχει μετατραπεί.
```csharp
string outputFolder = \@"YOUR_OUTPUT_DIRECTORY";
if (!Directory.Exists(outputFolder))
{
    Directory.CreateDirectory(outputFolder);
}

string outputFile = Path.Combine(outputFolder, "mhtml-converted-to.ppt");
```

#### Βήμα 2: Διαμόρφωση επιλογών μετατροπής
Ρυθμίστε τις επιλογές για τη μετατροπή σε μορφή PowerPoint.
```csharp
using GroupDocs.Conversion.Options.Convert;
using GroupDocs.Conversion.FileTypes;

PresentationConvertOptions options = new PresentationConvertOptions
{
    Format = PresentationFileType.Ppt // Η μορφή στόχου ορίστηκε σε PPT
};
```

#### Βήμα 3: Εκτελέστε τη μετατροπή
Εκτελέστε τη διαδικασία μετατροπής χρησιμοποιώντας τις καθορισμένες επιλογές.
```csharp
using (var converter = new Converter(\@"YOUR_DOCUMENT_DIRECTORY\\sample.mhtml"))
{
    converter.Convert(outputFile, options);
}
```
**Εξήγηση**: Το `Convert` Η μέθοδος εφαρμόζει τις καθορισμένες ρυθμίσεις και εκτελεί τον μετασχηματισμό αρχείου.
