## Οδηγός Εφαρμογής
### Φόρτωση αρχείου VSDM
**Επισκόπηση:** Αυτή η λειτουργία δείχνει πώς να φορτώσετε ένα αρχείο VSDM χρησιμοποιώντας το GroupDocs.Conversion. Η φόρτωση του αρχείου είναι το πρώτο σας βήμα πριν από την εφαρμογή οποιασδήποτε λογικής μετατροπής.

#### Βήμα 1: Αρχικοποίηση αντικειμένου μετατροπέα
```csharp

using GroupDocs.Conversion;

string sourceFilePath = "YOUR_DOCUMENT_DIRECTORY/sample.vsdm";

// Φορτώστε το αρχείο VSDM πηγής.

    // Η λογική μετατροπής μπορεί να εφαρμοστεί εδώ
}
```
- **Γιατί:** Ο `Converter` Η κλάση αρχικοποιείται με τη διαδρομή αρχείου, προετοιμάζοντάς την για τα επόμενα βήματα μετατροπής.

### Μετατροπή VSDM σε PPTX
**Επισκόπηση:** Αυτή η λειτουργία επιδεικνύει τη μετατροπή ενός φορτωμένου αρχείου VSDM σε μορφή PowerPoint. 

#### Βήμα 1: Ρύθμιση επιλογών μετατροπής
```csharp
using System.IO;
using GroupDocs.Conversion.Options.Convert;

string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "vsdm-converted-to.ppt");

// Φορτώστε το αρχείο VSDM πηγής και μετατρέψτε το σε PPT.
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY/sample.vsdm"))
{
    // Ορίστε επιλογές μετατροπής για τη μορφή PowerPoint.
    PresentationConvertOptions options = new PresentationConvertOptions { Format = PresentationFileType.Ppt };

    // Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο εξόδου.
    converter.Convert(outputFile, options);
}
```
- **Παράμετροι:** `PresentationConvertOptions` καθορίζει ότι μετατρέπετε σε μορφή PPT.
- **Επιστρεφόμενες τιμές:** Η μέθοδος αποθηκεύει το αρχείο που έχει μετατραπεί στην καθορισμένη τοποθεσία.
