## Οδηγός Εφαρμογής

### Μετατροπή STL σε PPT: Μια βήμα προς βήμα προσέγγιση

Αυτή η ενότητα θα σας καθοδηγήσει στη μετατροπή ενός αρχείου STL σε παρουσίαση PowerPoint.

#### Βήμα 1: Ορισμός διαδρομών αρχείων

Καθορίστε τους καταλόγους εγγράφων και εξόδου:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY"; // Αντικαταστήστε με την πραγματική σας διαδρομή
string outputDirectory = "YOUR_OUTPUT_DIRECTORY"; // Αντικαταστήστε με την επιθυμητή διαδρομή εξόδου
```

#### Βήμα 2: Φόρτωση του αρχείου STL

Χρήση `GroupDocs.Conversion` για να φορτώσετε το αρχείο προέλευσης:

```csharp
var stlFilePath = Path.Combine(documentDirectory, "sample.stl");
using (var converter = new GroupDocs.Conversion.Converter(stlFilePath))
{
    // Η λογική μετατροπής θα εφαρμοστεί εδώ.
}
```

#### Βήμα 3: Ρύθμιση παραμέτρων επιλογών PowerPoint

Ρύθμιση επιλογών μετατροπής για τη μορφή PowerPoint:

```csharp
PresentationConvertOptions options = new PresentationConvertOptions 
{ 
    Format = GroupDocs.Conversion.FileTypes.PresentationFileType.Ppt 
};
```

#### Βήμα 4: Εκτελέστε τη μετατροπή

Μετατρέψτε και αποθηκεύστε το αρχείο σας:

```csharp
string outputFile = Path.Combine(outputDirectory, "stl-converted-to.ppt");
converter.Convert(outputFile, options);
```
