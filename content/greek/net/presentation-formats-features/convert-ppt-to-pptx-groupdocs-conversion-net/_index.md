## Οδηγός Εφαρμογής

### Μετατροπή PPT σε PPTX

#### Επισκόπηση

Αυτή η λειτουργία θα σας καθοδηγήσει στη μετατροπή του PowerPoint `.ppt` αρχεία στο `.pptx` μορφή χρησιμοποιώντας C# και GroupDocs.Conversion.

**Βήμα 1: Ορισμός διαδρομών προέλευσης και εξόδου**

Ξεκινήστε ρυθμίζοντας τους καταλόγους σας για τις διαδρομές εισόδου και εξόδου. Βεβαιωθείτε ότι έχουν οριστεί σωστά για να αποφύγετε σφάλματα διαδρομής αρχείου:

```csharp
private static string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
private static string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
```

**Βήμα 2: Φόρτωση του αρχείου PPT**

Φορτώστε το `.ppt` αρχείο χρησιμοποιώντας το `Converter` κλάση, η οποία αρχικοποιεί τη διαδικασία μετατροπής.

```csharp
string inputFile = Path.Combine(documentDirectory, "sample.ppt");
using (var converter = new Converter(inputFile))
{
    // Ακολουθεί η λογική μετατροπής
}
```

**Βήμα 3: Ορισμός επιλογών μετατροπής**

Διαμορφώστε τις ρυθμίσεις μετατροπής σας με `PresentationConvertOptions`Αυτό το βήμα σάς επιτρέπει να προσαρμόσετε τον τρόπο μετατροπής του αρχείου.

```csharp
var options = new PresentationConvertOptions();
```

**Βήμα 4: Εκτελέστε τη μετατροπή**

Τέλος, μετατρέψτε και αποθηκεύστε το `.ppt` αρχείο ως `.pptx`.

```csharp
string outputFile = Path.Combine(outputDirectory, "ppt-converted-to.pptx");
converter.Convert(outputFile, options);
```
