## Οδηγός Εφαρμογής

### Φόρτωση αρχείου PPS

Η φόρτωση του αρχείου PPS πηγής είναι το πρώτο βήμα σε κάθε διαδικασία μετατροπής. Διασφαλίζει ότι έχετε πρόσβαση στα δεδομένα αρχείου που απαιτούνται για τον μετασχηματισμό.

#### Βήμα 1: Φόρτωση του αρχείου PPS προέλευσης
Για να φορτώσετε ένα αρχείο PPS, χρησιμοποιήστε το ακόλουθο απόσπασμα κώδικα:

```csharp
using System;
using GroupDocs.Conversion;

// Ορίστε τη διαδρομή του αρχείου PPS πηγής σας
string ppsFilePath = "@YOUR_DOCUMENT_DIRECTORY/sample.pps";

// Αρχικοποιήστε τον μετατροπέα με το καθορισμένο αρχείο PPS
using (var converter = new Converter(ppsFilePath))
{
    // Το αρχείο έχει πλέον φορτωθεί και είναι έτοιμο για μετατροπή.
}
```

### Μετατροπή PPS σε DOCX

Αφού φορτωθεί με επιτυχία το αρχείο PPS, μπορείτε να προχωρήσετε στη μετατροπή του σε έγγραφο του Word.

#### Βήμα 2: Ορισμός επιλογών μετατροπής
Ρυθμίστε την επιθυμητή μορφή εξόδου χρησιμοποιώντας `WordProcessingConvertOptions`.

```csharp
using System;
using GroupDocs.Conversion.Options.Convert;

// Καθορίστε επιλογές μετατροπής για τη μορφή DOCX
var options = new WordProcessingConvertOptions();
```

#### Βήμα 3: Εκτελέστε τη μετατροπή
Τέλος, εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο σας στην επιθυμητή θέση.

```csharp
using System.IO;
using GroupDocs.Conversion;

string outputFolder = "@YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "pps-converted-to.docx");

// Φόρτωση του αρχείου PPS προέλευσης
using (var converter = new Converter("@YOUR_DOCUMENT_DIRECTORY/sample.pps"))
{
    // Μετατρέψτε και αποθηκεύστε το αρχείο DOCX εξόδου
    converter.Convert(outputFile, options);
}
```
