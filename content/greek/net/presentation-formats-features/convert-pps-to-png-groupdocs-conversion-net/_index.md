## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Μετατροπή PPS σε PNG
Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε παρουσιάσεις PowerPoint (.pps) σε μορφή φορητών γραφικών δικτύου (.png).

#### Βήμα 1: Ορισμός διαδρομών αρχείων
Ξεκινήστε ορίζοντας τη διαδρομή του αρχείου εισόδου και τον κατάλογο εξόδου:

```csharp
string documentPath = "YOUR_DOCUMENT_DIRECTORY/Sample.pps";
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
```

Αντικαταστήστε τα placeholders με τις πραγματικές σας διαδρομές.

#### Βήμα 2: Ρύθμιση προτύπου εξόδου
Ορίστε ένα πρότυπο για την ονομασία του αρχείου PNG που έχει μετατραπεί σε κάθε σελίδα:

```csharp
string outputFileTemplate = System.IO.Path.Combine(outputFolder, "converted-page-{0}.png");
```

#### Βήμα 3: Ορισμός συνάρτησης ροής σελίδας
Δημιουργήστε μια συνάρτηση για να χειριστεί τη ροή εξόδου για κάθε σελίδα που έχει μετατραπεί. Αυτό διασφαλίζει ότι κάθε διαφάνεια αποθηκεύεται ως μεμονωμένο αρχείο PNG.

```csharp
Func<SavePageContext, System.IO.Stream> getPageStream = savePageContext =>
    new System.IO.FileStream(System.String.Format(outputFileTemplate, savePageContext.Page), System.IO.FileMode.Create);
```

#### Βήμα 4: Διαμόρφωση επιλογών μετατροπής
Ρυθμίστε τις επιλογές μετατροπής για τη μορφή εικόνας:

```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Png };
```

#### Βήμα 5: Εκτέλεση μετατροπής
Εκτελέστε την πραγματική μετατροπή καλώντας το `Convert` μέθοδος:

```csharp
using (Converter converter = new Converter(documentPath))
{
    // Μετατροπή PPS σε μορφή PNG
    converter.Convert(getPageStream, options);
}
```

**Συμβουλή αντιμετώπισης προβλημάτων:** Βεβαιωθείτε ότι οι διαδρομές σας είναι σωστές και ότι έχετε δικαιώματα εγγραφής για τον κατάλογο εξόδου.
