## Οδηγός Εφαρμογής

### Δυνατότητα: Μετατροπή OneNote σε PowerPoint

Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε αρχεία Microsoft OneNote (.one) σε μορφή PowerPoint Presentation Open XML (.pptx).

#### Βήμα 1: Διαμόρφωση διαδρομών εγγράφων

Ορίστε τις διαδρομές για τους καταλόγους εισόδου και εξόδου:
```csharp
string YOUR_DOCUMENT_DIRECTORY = "YOUR_DOCUMENT_DIRECTORY";
string YOUR_OUTPUT_DIRECTORY = "YOUR_OUTPUT_DIRECTORY";
// Βεβαιωθείτε ότι αυτές οι διαδρομές έχουν οριστεί σωστά για το πού βρίσκονται τα αρχεία σας.
```

#### Βήμα 2: Φόρτωση του αρχείου Source ONE

Φορτώστε το αρχείο .one χρησιμοποιώντας την κλάση Converter από το GroupDocs.Conversion:
```csharp
using (var converter = new Converter(YOUR_DOCUMENT_DIRECTORY + "/sample.one"))
{
    // Το αντικείμενο «μετατροπέας» είναι πλέον έτοιμο να χειριστεί εργασίες μετατροπής.
}
```

#### Βήμα 3: Ορισμός επιλογών μετατροπής

Ρυθμίστε τις επιλογές για τη μετατροπή σε μορφή PowerPoint χρησιμοποιώντας το PresentationConvertOptions:
```csharp
var options = new PresentationConvertOptions();
// Αυτές οι επιλογές υπαγορεύουν τον τρόπο μετατροπής του αρχείου.
```

#### Βήμα 4: Εκτελέστε τη μετατροπή

Εκτελέστε τη μετατροπή και αποθηκεύστε το αποτέλεσμα ως αρχείο .pptx:
```csharp
string outputFile = YOUR_OUTPUT_DIRECTORY + "/one-converted-to.pptx";
converter.Convert(outputFile, options);
// Το αρχείο αποθηκεύεται τώρα στον καθορισμένο κατάλογο εξόδου.
```

### Χαρακτηριστικό: Ρύθμιση παραμέτρων διαδρομής αρχείου

Βεβαιωθείτε ότι οι κατάλογοί σας έχουν ρυθμιστεί σωστά για είσοδο και έξοδο.

#### Βήμα 1: Ρύθμιση καταλόγου εξόδου

Λάβετε την πλήρη διαδρομή προς το σημείο όπου θα αποθηκεύσετε τα αρχεία που έχετε μετατρέψει:
```csharp
string outputFolder = Path.Combine(Directory.GetCurrentDirectory(), YOUR_OUTPUT_DIRECTORY);
if (!Directory.Exists(outputFolder))
{
    Directory.CreateDirectory(outputFolder); // Δημιουργήστε έναν κατάλογο εάν δεν υπάρχει.
}
```

#### Βήμα 2: Ορισμός διαδρομής αρχείου εξόδου

Ορίστε τη διαδρομή για το πού θέλετε να αποθηκεύσετε το αρχείο που έχει μετατραπεί:
```csharp
string outputFile = Path.Combine(outputFolder, "one-converted-to.pptx");
// Βεβαιωθείτε ότι αυτή η διαδρομή έχει οριστεί σωστά μέσα στον φάκελο εξόδου σας.
```
