## Οδηγός Εφαρμογής
Ας αναλύσουμε την υλοποίηση σε εύκολα βήματα.

### Μετατροπή αρχείου LaTeX (.tex) σε PowerPoint (PPTX)
Αυτή η λειτουργία δείχνει πώς να μετατρέψετε ένα αρχείο LaTeX (.tex) σε μορφή παρουσίασης PowerPoint Open XML (.pptx).

#### Βήμα 1: Ορισμός διαδρομών εισόδου και εξόδου
Ξεκινήστε ρυθμίζοντας τους καταλόγους εισόδου και εξόδου. Αυτό θα σας βοηθήσει να διαχειριστείτε από πού θα διαβάζονται και θα αποθηκεύονται τα αρχεία σας.

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY"; // Αντικατάσταση με την πραγματική διαδρομή
string outputDirectory = "YOUR_OUTPUT_DIRECTORY"; // Αντικατάσταση με την πραγματική διαδρομή

// Καθορίστε τη διαδρομή εισόδου του αρχείου TEX και τη διαδρομή εξόδου του αρχείου PPTX.
string inputFile = Path.Combine(documentDirectory, "sample.tex");
string outputFile = Path.Combine(outputDirectory, "tex-converted-to.pptx");
```

#### Βήμα 2: Αρχικοποίηση μετατροπέα
Δημιουργήστε μια παρουσία του `Converter` κλάση με το αρχείο εισόδου σας.

```csharp
using (var converter = new GroupDocs.Conversion.Converter(inputFile))
{
    // Η διαδικασία μετατροπής θα οριστεί εδώ.
}
```

#### Βήμα 3: Ορισμός επιλογών μετατροπής
Ρυθμίστε τις επιλογές μετατροπής για τη μορφή PowerPoint χρησιμοποιώντας `PresentationConvertOptions`.

```csharp
// Καθορίστε επιλογές μετατροπής για τη μορφή PowerPoint.
var options = new PresentationConvertOptions();
```

#### Βήμα 4: Εκτέλεση μετατροπής
Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο εξόδου.

```csharp
// Μετατρέψτε το TEX σε PPTX χρησιμοποιώντας καθορισμένες επιλογές.
converter.Convert(outputFile, options);
```
