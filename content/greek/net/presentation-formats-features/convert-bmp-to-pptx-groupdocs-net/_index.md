## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Μετατροπή BMP σε PPTX

Αυτή η λειτουργία εστιάζει στη μετατροπή αρχείων BMP σε παρουσιάσεις PowerPoint χρησιμοποιώντας τις ισχυρές δυνατότητες του GroupDocs.Conversion.

#### Φόρτωση και μετατροπή του αρχείου

1. **Ορισμός διαδρομών εξόδου**
   
   Ξεκινήστε ορίζοντας τις διαδρομές αρχείων σας τόσο για την είσοδο (BMP) όσο και για την έξοδο (PPTX):
   ```csharp
   string outputFolder = "YOUR_OUTPUT_DIRECTORY/"; // Ορίστε πού θα αποθηκευτεί το PPTX που έχει μετατραπεί.
   string sourceBmpPath = "YOUR_DOCUMENT_DIRECTORY/sample.bmp"; // Διαδρομή προς το αρχείο εικόνας BMP.
   ```

2. **Αρχικοποίηση του μετατροπέα**
   
   Χρησιμοποιήστε το GroupDocs.Conversion για να φορτώσετε και να μετατρέψετε το αρχείο BMP:
   ```csharp
   using (var converter = new Converter(sourceBmpPath))
   {
       var options = new PresentationConvertOptions(); // Επιλογές για μετατροπή σε μορφή PPTX.
       string outputFile = System.IO.Path.Combine(outputFolder, "bmp-converted-to.pptx");
       converter.Convert(outputFile, options); // Εκτελέστε τη μετατροπή και αποθηκεύστε την ως PPTX.
   }
   ```

#### Επεξήγηση παραμέτρων

- **`Converter`:** Αυτή η κλάση φορτώνει το αρχείο BMP για επεξεργασία. Βεβαιωθείτε ότι οι διαδρομές έχουν οριστεί σωστά.
- **`PresentationConvertOptions`:** Ρυθμίζει τον τρόπο με τον οποίο το αρχείο BMP θα μετατραπεί σε παρουσίαση PowerPoint.
