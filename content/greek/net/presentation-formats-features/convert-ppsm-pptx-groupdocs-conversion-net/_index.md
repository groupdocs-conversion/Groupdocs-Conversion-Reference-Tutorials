## Οδηγός Εφαρμογής

### Μετατροπή PPSM σε PPTX

#### Επισκόπηση
Σε αυτήν την ενότητα, θα μετατρέψουμε ένα αρχείο .ppsm σε .pptx φορτώνοντας το αρχείο προέλευσης και καθορίζοντας τις επιλογές μετατροπής.

#### Βήμα 1: Ορισμός καταλόγου εξόδου και διαδρομής αρχείου
Ρυθμίστε τον κατάλογο εξόδου και καθορίστε πού θέλετε να αποθηκευτεί το αρχείο που έχει μετατραπεί:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY\\\\";
string outputFile = System.IO.Path.Combine(outputFolder, "ppsm-converted-to.pptx");
```

Αυτό το απόσπασμα κώδικα διασφαλίζει ότι το αρχείο που έχει μετατραπεί αποθηκεύεται σε μια καθορισμένη τοποθεσία.

#### Βήμα 2: Φόρτωση του αρχείου PPSM προέλευσης
Χρησιμοποιήστε το GroupDocs.Conversion για να φορτώσετε το αρχείο .ppsm:

```csharp
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY\\\\sample.ppsm"))
{
    // Η λογική μετατροπής θα προστεθεί εδώ
}
```

Η φόρτωση του αρχείου το προετοιμάζει για μετατροπή.

#### Βήμα 3: Δημιουργία επιλογών μετατροπής και μετατροπή
Καθορίστε τις επιλογές μετατροπής μορφής PowerPoint και εκτελέστε τη μετατροπή:

```csharp
// Δημιουργήστε επιλογές μετατροπής για τη μορφή PowerPoint
var options = new PresentationConvertOptions();

// Μετατρέψτε και αποθηκεύστε το αρχείο PPTX
converter.Convert(outputFile, options);
```

Ο `PresentationConvertOptions` Η κλάση επιτρέπει την προσαρμογή της μορφής εξόδου.
