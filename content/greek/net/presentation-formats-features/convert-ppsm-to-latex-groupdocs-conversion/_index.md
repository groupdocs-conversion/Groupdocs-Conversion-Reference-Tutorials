## Οδηγός Εφαρμογής

Αφού όλα έχουν ρυθμιστεί, ας προχωρήσουμε στην εφαρμογή της διαδικασίας μετατροπής βήμα προς βήμα.

### Φόρτωση και μετατροπή PPSM σε TEX

#### Επισκόπηση
Σε αυτήν την ενότητα, θα δείξουμε πώς να φορτώσετε ένα αρχείο PPSM και να το μετατρέψετε σε μορφή εγγράφου LaTeX (.tex).

##### Βήμα 1: Ορισμός διαδρομών προέλευσης και εξόδου
```csharp
string sourceFilePath = @"YOUR_DOCUMENT_DIRECTORY\sample.ppsm";
string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "ppsm-converted-to.tex");
```

##### Βήμα 2: Ορισμός επιλογών μετατροπής για τη μορφή TEX
Ορίστε επιλογές μετατροπής για να καθορίσετε τη μορφή προορισμού ως LaTeX (TEX).
```csharp
PageDescriptionLanguageConvertOptions options = new PageDescriptionLanguageConvertOptions { 
    Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Tex 
};
```

##### Βήμα 3: Εκτελέστε τη διαδικασία μετατροπής
Τώρα, εκτελέστε τη μετατροπή χρησιμοποιώντας τις καθορισμένες επιλογές και αποθηκεύστε το αποτέλεσμα.
```csharp
using (var converter = new Converter(sourceFilePath))
{
    // Μετατροπή PPSM σε μορφή TEX.
    converter.Convert(outputFile, options);
}
```

#### Συμβουλές αντιμετώπισης προβλημάτων:
- Βεβαιωθείτε ότι όλες οι διαδρομές είναι σωστά ρυθμισμένες και προσβάσιμες.
- Επαληθεύστε ότι έχετε τα κατάλληλα δικαιώματα για ανάγνωση από τον κατάλογο προέλευσης και εγγραφή στον κατάλογο εξόδου.
