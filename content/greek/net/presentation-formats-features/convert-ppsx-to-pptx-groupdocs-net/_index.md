## Οδηγός Εφαρμογής

### Μετατροπή PPSX σε PPTX

#### Επισκόπηση
Αυτή η λειτουργία μετατρέπει ένα αρχείο PowerPoint Open XML Slide Show (.ppsx) σε μια παρουσίαση PowerPoint Open XML (.pptx), εξασφαλίζοντας συμβατότητα με πιο τυπικές εφαρμογές.

#### Βήμα 1: Ορισμός καταλόγων εισόδου και εξόδου
Ξεκινήστε ρυθμίζοντας τους καταλόγους εισόδου και εξόδου:
```csharp
string documentDirectory = \@"YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = Path.Combine(Directory.GetCurrentDirectory(), "YOUR_OUTPUT_DIRECTORY");

if (!Directory.Exists(outputDirectory))
{
    Directory.CreateDirectory(outputDirectory);
}
```
#### Βήμα 2: Φόρτωση και μετατροπή του αρχείου
Φορτώστε το αρχείο PPSX πηγής σας και καθορίστε τις επιλογές μετατροπής:
```csharp
using (var converter = new Converter(Path.Combine(documentDirectory, "sample.ppsx")))
{
    var options = new PresentationConvertOptions();
    
    string outputFile = Path.Combine(outputDirectory, "ppsx-converted-to.pptx");
    converter.Convert(outputFile, options);
}
```
**Εξήγηση**: Το `Converter` Η κλάση διαχειρίζεται τη φόρτωση και τη μετατροπή αρχείων. `PresentationConvertOptions` καθορίζει τη μορφή προορισμού ως PPTX.

#### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι η διαδρομή του αρχείου προέλευσης είναι σωστή.
- Επαληθεύστε τα δικαιώματα καταλόγου για την ανάγνωση/εγγραφή αρχείων.
