## Οδηγός Εφαρμογής

### Μετατροπή εικόνας WebP σε μορφή PowerPoint

Αυτή η ενότητα σάς καθοδηγεί στη μετατροπή ενός αρχείου εικόνας WebP σε μορφή PPT χρησιμοποιώντας το GroupDocs.Conversion for .NET. Ακολουθήστε τα παρακάτω βήματα:

**Βήμα 1: Ορισμός διαδρομών**
Ξεκινήστε καθορίζοντας διαδρομές για τους καταλόγους εισόδου και εξόδου:
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY/";
```

**Βήμα 2: Φόρτωση του αρχείου WebP προέλευσης**
Χρησιμοποιήστε το `Converter` κλάση για να φορτώσετε το αρχείο WebP, το οποίο χρησιμεύει ως σημείο εισόδου για τη μετατροπή:
```csharp
using (var converter = new Converter(Path.Combine(documentDirectory, "sample.webp")))
{
    // Η λογική μετατροπής θα προστεθεί εδώ
}
```

**Βήμα 3: Ορισμός επιλογών μετατροπής**
Ρυθμίστε τις παραμέτρους επιλογών που αφορούν συγκεκριμένα τη μορφή PowerPoint:
```csharp
PresentationConvertOptions options = new PresentationConvertOptions
{
    Format = GroupDocs.Conversion.FileTypes.PresentationFileType.Ppt
};
```

**Βήμα 4: Αποθήκευση του αρχείου PPT που έχει μετατραπεί**
Τέλος, αποθηκεύστε το αρχείο που μετατρέψατε στην επιθυμητή θέση:
```csharp
string outputFile = Path.Combine(outputDirectory, "webp-converted-to.ppt");
converter.Convert(outputFile, options);
```
