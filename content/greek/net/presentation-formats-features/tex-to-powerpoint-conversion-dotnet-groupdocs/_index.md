## Οδηγός Εφαρμογής
Ας αναλύσουμε τη διαδικασία μετατροπής σε σαφή, εφαρμόσιμα βήματα.
### Βήμα 1: Ορισμός καταλόγου εξόδου και διαδρομών αρχείων
Ξεκινήστε ρυθμίζοντας τον κατάλογο εξόδου και τη διαδρομή αρχείου:
```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY"); // Προσαρμόστε αυτήν τη διαδρομή
string outputFile = Path.Combine(outputFolder, "tex-converted-to.ppt");
```
Αντικαθιστώ `"YOUR_OUTPUT_DIRECTORY"` με την επιθυμητή τοποθεσία για την αποθήκευση των μετατρεπόμενων παρουσιάσεων.
### Βήμα 2: Φόρτωση του αρχείου LaTeX πηγής
Στη συνέχεια, φορτώστε το έγγραφο TEX χρησιμοποιώντας το `Converter` τάξη. Βεβαιωθείτε ότι παρέχετε τη σωστή διαδρομή:
```csharp
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY/sample.tex")) // Προσαρμόστε αυτήν τη διαδρομή
{
    Console.WriteLine("LaTeX file loaded successfully.");
}
```
Αντικαθιστώ `"YOUR_DOCUMENT_DIRECTORY"` με τον κατάλογο που περιέχει τα αρχεία TEX σας.
### Βήμα 3: Διαμόρφωση επιλογών μετατροπής
Ρύθμιση επιλογών μετατροπής για τη μορφή PowerPoint:
```csharp
PresentationConvertOptions options = new PresentationConvertOptions
{
    Format = PresentationFileType.Ppt // Η μορφή στόχου ορίστηκε σε PPT
};
```
Αυτή η ρύθμιση παραμέτρων καθορίζει ότι η έξοδος θα πρέπει να είναι σε μορφή PPT.
### Βήμα 4: Εκτελέστε τη μετατροπή και αποθηκεύστε την έξοδο
Τέλος, εκτελέστε τη μετατροπή και αποθηκεύστε το αποτέλεσμα:
```csharp
converter.Convert(outputFile, options);
Console.WriteLine("Conversion completed successfully.");
```
