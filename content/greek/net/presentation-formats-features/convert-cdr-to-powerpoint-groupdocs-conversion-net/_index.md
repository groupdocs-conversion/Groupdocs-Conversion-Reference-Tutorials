## Οδηγός Εφαρμογής
### Μετατροπή CDR σε PPT
Αυτή η λειτουργία επιτρέπει την αποτελεσματική μετατροπή αρχείων CDR σε παρουσιάσεις PowerPoint. Δείτε πώς:

#### Βήμα 1: Ρύθμιση καταλόγου εξόδου
Ορίστε πού θα αποθηκευτεί το αρχείο που μετατρέψατε:
```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "cdr-converted-to.ppt");
```
`outputFolder` είναι ο καθορισμένος κατάλογος για την αποθήκευση αρχείων και `outputFile` καθορίζει το όνομα του αρχείου PPT που προκύπτει.

#### Βήμα 2: Φόρτωση και μετατροπή αρχείου CDR
Φορτώστε το αρχείο CDR πηγής και μετατρέψτε το σε PowerPoint:
```csharp
using (var converter = new GroupDocs.Conversion.Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_CDR"))
{
    // Αρχικοποίηση επιλογών μετατροπής για το PowerPoint.
    PresentationConvertOptions options = new PresentationConvertOptions { Format = FileTypes.PresentationFileType.Ppt };
    
    // Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο PPT εξόδου.
    converter.Convert(outputFile, options);
}
```
**Εξήγηση**: 
Ο `Converter` Η κλάση φορτώνει το αρχείο CDR σας. Βεβαιωθείτε ότι η διαδρομή προς το "SAMPLE_CDR" έχει οριστεί σωστά.
`PresentationConvertOptions` καθορίζει τη μορφή PowerPoint (`Ppt`). Προσαρμόστε για άλλες εκδόσεις όπως PPTX, εάν χρειάζεται.
Τελικά, `converter.Convert(outputFile, options)` εκτελεί και αποθηκεύει τη μετατροπή.
