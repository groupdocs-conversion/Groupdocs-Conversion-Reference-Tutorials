### Μετατροπή OTP σε μορφή PPT

**Επισκόπηση:**
Αυτή η ενότητα παρουσιάζει τον τρόπο μετατροπής ενός αρχείου OTP σε παρουσίαση PowerPoint χρησιμοποιώντας το GroupDocs.Conversion.

#### Βήμα 1: Καθορίστε τον κατάλογο εξόδου και το όνομα αρχείου

Ορίστε πού θα αποθηκευτεί το αρχείο που μετατρέψατε:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = System.IO.Path.Combine(outputFolder, "otp-converted-to.ppt");
```

#### Βήμα 2: Διαμόρφωση επιλογών μετατροπής

Ορίστε την επιθυμητή μορφή για μετατροπή χρησιμοποιώντας `PresentationConvertOptions`:

```csharp
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY/sample.otp"))
{
    PresentationConvertOptions options = new PresentationConvertOptions { Format = GroupDocs.Conversion.FileTypes.PresentationFileType.Ppt };
    
    // Εκτελέστε τη μετατροπή και αποθηκεύστε την στην καθορισμένη διαδρομή αρχείου εξόδου
    converter.Convert(outputFile, options);
}
```

**Παράμετροι & Μέθοδοι:**
- `sourceFilePath`: Διαδρομή προς το αρχείο OTP εισαγωγής σας.
- `outputFolder`/`outputFile`: Κατάλογοι για την αποθήκευση αρχείων που έχουν μετατραπεί.
- `PresentationConvertOptions`: Καθορίζει ρυθμίσεις μετατροπής για συγκεκριμένη μορφή.
