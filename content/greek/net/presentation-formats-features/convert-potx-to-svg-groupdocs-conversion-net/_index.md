## Οδηγός Εφαρμογής

Θα δούμε πώς να μετατρέπετε αρχεία POTX σε μορφή SVG με σαφή βήματα. Ας ξεκινήσουμε!

### Φόρτωση του αρχείου προέλευσης

Αρχικά, προσδιορίστε τον κατάλογο εγγράφων σας όπου βρίσκεται το `sample.potx` βρίσκεται το αρχείο.

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
```

### Ορισμός επιλογών μετατροπής για SVG

Δημιουργήστε μια παρουσία του μετατροπέα και ρυθμίστε τις επιλογές μετατροπής ειδικά για τη μορφή SVG.

```csharp
using (var converter = new Converter(Path.Combine(documentDirectory, "sample.potx")))
{
    var options = new PageDescriptionLanguageConvertOptions { Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Svg };
```

### Ορισμός εξόδου και μετατροπή

Καθορίστε πού θέλετε να αποθηκευτεί το αρχείο SVG που έχει μετατραπεί:

```csharp
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputDirectory, "potx-converted-to.svg");

// Μετατροπή και αποθήκευση του αρχείου SVG
converter.Convert(outputFile, options);
}
```
