### Μετατροπή CSV σε παρουσίαση PowerPoint

Αυτή η ενότητα παρουσιάζει τον μετασχηματισμό δεδομένων από μορφή CSV σε διαφάνειες PPT χρησιμοποιώντας το GroupDocs.Conversion.

#### Διαδικασία μετατροπής βήμα προς βήμα

**1. Ορισμός διαδρομών αρχείων**

Καθορίστε καταλόγους για το αρχείο CSV πηγής και το PPT εξόδου:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputFolder = "YOUR_OUTPUT_DIRECTORY";

string sourceFilePath = Path.Combine(documentDirectory, "sample.csv");
string outputFile = Path.Combine(outputFolder, "csv-converted-to.ppt");
```

**2. Φορτώστε το αρχείο CSV πηγής**

Φορτώστε το αρχείο CSV χρησιμοποιώντας το GroupDocs.Conversion `Converter` τάξη:

```csharp
using (var converter = new Converter(sourceFilePath))
{
    // Προχωρήστε στη ρύθμιση των επιλογών μετατροπής
}
```

**3. Ορίστε επιλογές μετατροπής για το PowerPoint**

Διαμορφώστε τις ρυθμίσεις για να μετατρέψετε το αρχείο σας σε μορφή παρουσίασης PowerPoint:

```csharp
PresentationConvertOptions options = new PresentationConvertOptions
{
    Format = PresentationFileType.Ppt // Καθορίστε PPT ως μορφή εξόδου
};
```

**4. Μετατροπή και αποθήκευση του αρχείου**

Εκτελέστε τη μετατροπή και αποθηκεύστε την έξοδο:

```csharp
converter.Convert(outputFile, options);
Console.WriteLine("Conversion completed successfully.");
```
