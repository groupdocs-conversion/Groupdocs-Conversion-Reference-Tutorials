### Μετατροπή MPT σε λειτουργία TEX

#### Επισκόπηση

Η μετατροπή αρχείων MPT σε μορφή TEX μπορεί να είναι κρίσιμη για την ενσωμάτωση δεδομένων έργου σε έγγραφα LaTeX. Δείτε πώς μπορείτε να το πετύχετε αυτό με μια απλή προσέγγιση χρησιμοποιώντας το GroupDocs.Conversion.

#### Βήμα προς βήμα εφαρμογή

**1. Φορτώστε το αρχείο MPT προέλευσης**

Ξεκινήστε φορτώνοντας το αρχείο προέλευσης. Αυτό γίνεται χρησιμοποιώντας το `Converter` τάξη:

```csharp
string inputFilePath = "YOUR_DOCUMENT_DIRECTORY\\sample.mpt";
using (var converter = new Converter(inputFilePath))
{
    // Λογική μετατροπής εδώ
}
```

**2. Διαμόρφωση επιλογών μετατροπής για τη μορφή TEX**

Στη συνέχεια, ρυθμίστε τις επιλογές μετατροπής για να καθορίσετε ότι θέλουμε μια έξοδο TEX:

```csharp
PageDescriptionLanguageConvertOptions options = 
    new PageDescriptionLanguageConvertOptions { Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Tex };
```
*Εξήγηση:* Εδώ, `PageDescriptionLanguageConvertOptions` έχει ρυθμιστεί να μετατρέπει αρχεία σε μορφή LaTeX.

**3. Εκτελέστε τη μετατροπή**

Τέλος, εκτελέστε τη μετατροπή και αποθηκεύστε την έξοδο:

```csharp
string outputFolder = @"YOUR_OUTPUT_DIRECTORY\";
string outputFile = Path.Combine(outputFolder, "mpt-converted-to.tex");

converter.Convert(outputFile, options);
```
*Εξήγηση:* Ο `Convert` Η μέθοδος παίρνει τη διαδρομή του αρχείου εξόδου και τις διαμορφωμένες επιλογές για να εκτελέσει τη μετατροπή.
