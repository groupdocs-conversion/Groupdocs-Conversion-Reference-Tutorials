## Οδηγός Εφαρμογής

Τώρα που έχουμε ρυθμίσει τα πάντα, ας προχωρήσουμε στην εφαρμογή της λειτουργίας μετατροπής DOT σε PPTX.

### Φόρτωση και μετατροπή αρχείου DOT σε PPTX

Αυτή η ενότητα καλύπτει τη φόρτωση ενός αρχείου προτύπου εγγράφου Microsoft Word (.dot) και τη μετατροπή του σε μορφή παρουσίασης PowerPoint Open XML (.pptx).

#### Βήμα 1: Ορισμός διαδρομών προέλευσης και εξόδου

```csharp
string sourceDotFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.dot");
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "dot-converted-to.pptx");
```

Αντικαθιστώ `"YOUR_DOCUMENT_DIRECTORY"` με τον κατάλογο όπου βρίσκεται το αρχείο DOT σας και `"YOUR_OUTPUT_DIRECTORY"` με το πού θέλετε να αποθηκεύσετε το PPTX που έχει μετατραπεί.

#### Βήμα 2: Φόρτωση του αρχείου DOT προέλευσης

```csharp
using (var converter = new Converter(sourceDotFilePath))
{
    var options = new PresentationConvertOptions();
    
    // Μετατροπή και αποθήκευση ως PPTX
    converter.Convert(outputFile, options);
}
```

- **Μετατροπέας**: Αρχικοποιείται με τη διαδρομή του αρχείου προέλευσης.
- **PresentationConvertOptions**: Καθορίζει τις ρυθμίσεις μετατροπής για τη μορφή PowerPoint.

#### Επεξήγηση παραμέτρων

- `sourceDotFilePath`Η πλήρης διαδρομή προς το αρχείο DOT σας.
- `outputFile`: Πού θέλετε να αποθηκεύσετε το αρχείο PPTX που έχει μετατραπεί.
- `options`Επιλογές μετατροπής προσαρμοσμένες για παρουσίαση PowerPoint.
