## Οδηγός Εφαρμογής
### Χαρακτηριστικό: Μετατροπή HTML σε PPTX
#### Επισκόπηση
Σε αυτήν την ενότητα, θα μετατρέψουμε ένα αρχείο HTML σε μια παρουσίαση PowerPoint. Αυτή η λειτουργία απλοποιεί την κοινή χρήση περιεχομένου ιστού σε επαγγελματικές μορφές διαφανειών.
#### Βήμα 1: Φόρτωση του αρχείου HTML πηγής
Ξεκινήστε φορτώνοντας το αρχείο HTML χρησιμοποιώντας το `Converter` τάξη:
```csharp
string documentPath = Path.Combine(@"YOUR_DOCUMENT_DIRECTORY", "sample.html");
```
Αυτό το απόσπασμα κώδικα καθορίζει τη διαδρομή προς το αρχείο HTML πηγής. Αντικαταστήστε `"YOUR_DOCUMENT_DIRECTORY"` με τον πραγματικό κατάλογο που περιέχει το HTML σας.
#### Βήμα 2: Ορισμός επιλογών μετατροπής
Στη συνέχεια, διαμορφώστε τις επιλογές μετατροπής για το PowerPoint:
```csharp
var options = new PresentationConvertOptions();
```
Ο `PresentationConvertOptions` Η κλάση σάς επιτρέπει να καθορίσετε διάφορες ρυθμίσεις για τη μετατροπή εγγράφων σε μορφή PPTX. Στο πιο βασικό της επίπεδο, λέει στο GroupDocs.Conversion να εξάγει ένα αρχείο PowerPoint.
#### Βήμα 3: Μετατροπή και αποθήκευση της εξόδου
Τέλος, εκτελέστε τη μετατροπή:
```csharp
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputDirectory, "html-converted-to.pptx");

using (var converter = new Converter(documentPath))
{
    converter.Convert(outputFile, options);
}
```
Αυτό το απόσπασμα μετατρέπει το αρχείο HTML σας σε PPTX και το αποθηκεύει στον καθορισμένο κατάλογο. Βεβαιωθείτε ότι `"YOUR_OUTPUT_DIRECTORY"` έχει οριστεί στο σημείο όπου θέλετε να αποθηκεύονται τα αρχεία που έχετε μετατρέψει.
