#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς να φορτώσετε ένα αρχείο ICO χρησιμοποιώντας το GroupDocs.Conversion, το οποίο είναι απαραίτητο πριν από την πραγματοποίηση της μετατροπής.

#### Βήματα Υλοποίησης:
**Βήμα 1:** Ορίστε τη διαδρομή για το αρχείο ICO πηγής σας.
```csharp
using System;
using System.IO;

public static void LoadIcoFile() {
    string documentDirectory = "YOUR_DOCUMENT_DIRECTORY"; // Αντικατάσταση με τον πραγματικό κατάλογο
    string icoFilePath = Path.Combine(documentDirectory, "sample.ico");

    Console.WriteLine($"Loading ICO file from: {icoFilePath}");
}
```
**Βήμα 2:** Αρχικοποιήστε το αντικείμενο μετατροπέα.
```csharp

        Console.WriteLine("Converter initialized with ICO file.");

```
### Μετατροπή ICO σε PPTX

#### Επισκόπηση
Αυτή η λειτουργία εξηγεί πώς να μετατρέψετε ένα αρχείο ICO σε μορφή παρουσίασης PowerPoint, συγκεκριμένα PPTX.

#### Βήματα Υλοποίησης:
**Βήμα 1:** Ορίστε τον κατάλογο εξόδου και τη διαδρομή αρχείου.
```csharp
public static void ConvertIcoToPptx() {
    string outputDirectory = "YOUR_OUTPUT_DIRECTORY"; // Αντικατάσταση με τον πραγματικό κατάλογο
    string outputFile = Path.Combine(outputDirectory, "ico-converted-to.pptx");

    Directory.CreateDirectory(outputDirectory); // Βεβαιωθείτε ότι ο κατάλογος υπάρχει

    Console.WriteLine($"Output will be saved to: {outputFile}");
}
```
**Βήμα 2:** Ρυθμίστε τις επιλογές μετατροπής.
```csharp
using GroupDocs.Conversion.Options.Convert;

public static void ConvertIcoToPptx() {
    // Υποθέστε ότι οι διαδρομές έχουν ήδη οριστεί όπως στα προηγούμενα βήματα
    var options = new PresentationConvertOptions();
    Console.WriteLine("Conversion options for PPTX format set.");
}
```
**Βήμα 3:** Εκτελέστε τη μετατροπή.
```csharp
using GroupDocs.Conversion;

public static void ConvertIcoToPptx() {
    // Υποθέστε ότι οι διαδρομές και οι επιλογές έχουν ήδη οριστεί όπως στα προηγούμενα βήματα
    using (var converter = new Converter(icoFilePath)) {
        converter.Convert(outputFile, options);
        Console.WriteLine("ICO file successfully converted to PPTX.");
    }
}
```
