### Μετατροπή ODP σε DOC
Αυτή η λειτουργία επιτρέπει τη μετατροπή αρχείων παρουσίασης OpenDocument σε έγγραφα του Microsoft Word. Δείτε πώς:

#### 1. Φορτώστε το αρχείο ODP πηγής
Καθορίστε τη διαδρομή προς το αρχείο ODP προέλευσης και προετοιμάστε τον κατάλογο εξόδου:
```csharp
string documentPath = Path.Combine(@"YOUR_DOCUMENT_DIRECTORY", "sample.odp");
string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
```
Ο `documentPath` η μεταβλητή θα πρέπει να δείχνει στο αρχείο ODP σας, ενώ `outputFolder` είναι το σημείο όπου θέλετε να αποθηκευτούν τα αρχεία DOC που έχουν μετατραπεί.

#### 2. Καθορίστε Επιλογές Μετατροπής
Ορίστε επιλογές μετατροπής για μορφές επεξεργασίας κειμένου όπως το DOC:
```csharp
using (var converter = new Converter(documentPath))
{
    WordProcessingConvertOptions options = new WordProcessingConvertOptions { Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc };
```
Ο `WordProcessingConvertOptions` Η κλάση σάς επιτρέπει να καθορίσετε τη μορφή εξόδου, η οποία ορίζεται εδώ σε DOC.

#### 3. Εκτελέστε τη μετατροπή
Εκτελέστε τη μετατροπή και αποθηκεύστε το αποτέλεσμα:
```csharp
    // Μετατρέψτε και αποθηκεύστε το αρχείο DOC χρησιμοποιώντας καθορισμένες επιλογές
    converter.Convert(Path.Combine(outputFolder, "odp-converted-to.doc"), options);
}
```
Αυτό το μπλοκ κώδικα χειρίζεται την πραγματική διαδικασία μετατροπής, αποθηκεύοντας την έξοδο στην καθορισμένη διαδρομή σας.
