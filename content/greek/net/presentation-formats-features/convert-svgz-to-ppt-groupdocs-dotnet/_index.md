## Οδηγός Εφαρμογής
Ας αναλύσουμε τη διαδικασία μετατροπής ενός αρχείου SVGZ σε παρουσίαση PowerPoint.

### Βήμα 1: Φόρτωση και αρχικοποίηση του μετατροπέα
Αρχικά, αρχικοποιούμε το `Converter` αντικείμενο με τη διαδρομή προς το αρχείο SVGZ μας. Αυτό το βήμα θέτει τις βάσεις για την εργασία μετατροπής μας φορτώνοντας το συμπιεσμένο αρχείο SVG.

```csharp
string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.svgz");
using (var converter = new GroupDocs.Conversion.Converter(inputFilePath))
{
    // Περαιτέρω βήματα θα προστεθούν εδώ
}
```

### Βήμα 2: Ρύθμιση επιλογών μετατροπής
Στη συνέχεια, ορίζουμε τις επιλογές μετατροπής. Σε αυτήν την περίπτωση, καθορίζουμε ότι θέλουμε να μετατρέψουμε το αρχείο SVGZ σε μια παρουσίαση PowerPoint (μορφή .ppt).

```csharp
PresentationConvertOptions options = new PresentationConvertOptions {
    Format = GroupDocs.Conversion.FileTypes.PresentationFileType.Ppt
};
```

### Βήμα 3: Εκτέλεση της μετατροπής
Τέλος, εκτελούμε τη μετατροπή και αποθηκεύουμε το αρχείο PPT εξόδου. Αυτό το βήμα είναι κρίσιμο καθώς μετατρέπει το SVGZ μας σε παρουσίαση PowerPoint.

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "svgz-converted-to.ppt");
converter.Convert(outputFile, options);
```
