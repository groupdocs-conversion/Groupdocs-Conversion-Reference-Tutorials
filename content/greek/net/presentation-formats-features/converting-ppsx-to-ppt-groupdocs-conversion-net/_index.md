## Οδηγός Εφαρμογής

### Μετατροπή PPSX σε PPT

Αφού ρυθμίσετε το GroupDocs.Conversion, ας προχωρήσουμε στη διαδικασία μετατροπής.

#### Επισκόπηση

Αυτή η λειτουργία μετατρέπει τη μορφή PowerPoint Open XML Slide Show (.ppsx) σε παραδοσιακή παρουσίαση PowerPoint (.ppt).

#### Μετατροπή βήμα προς βήμα

##### 1. Αρχικοποίηση μετατροπέα
Ξεκινήστε αρχικοποιώντας το `Converter` αντικείμενο με τη διαδρομή του αρχείου PPSX σας:

```csharp
using (var converter = new Converter("input.ppsx"))
{
    // Τα βήματα μετατροπής θα μεταφερθούν εδώ
}
```

##### 2. Ορισμός επιλογών μετατροπής
Ορίστε επιλογές μετατροπής για τη μορφή παρουσίασης PowerPoint χρησιμοποιώντας `PresentationConvertOptions`.

```csharp
var convertOptions = new PresentationConvertOptions();
```
Ο `PresentationConvertOptions` Η κλάση σάς επιτρέπει να προσαρμόσετε τις ρυθμίσεις του αρχείου εξόδου, όπως το μέγεθος και τη μορφή της διαφάνειας.

##### 3. Εκτέλεση μετατροπής
Εκτελέστε τη μετατροπή καλώντας το `Convert()` μέθοδος:

```csharp
converter.Convert("output.ppt", convertOptions);
```

Αυτή η γραμμή μετατρέπει το αρχείο PPSX σας σε αρχείο PPT με συγκεκριμένες επιλογές.
