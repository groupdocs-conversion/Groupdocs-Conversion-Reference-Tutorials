## Οδηγός Εφαρμογής

### Μετατροπή αρχείου PS σε μορφή PPTX

Η μετατροπή ενός αρχείου PostScript (PS) σε μορφή παρουσίασης PowerPoint Open XML (.pptx) περιλαμβάνει τα ακόλουθα βήματα:

#### Βήμα 1: Ορισμός διαδρομών
Καθορίστε διαδρομές για το αρχείο PS προέλευσης και τον κατάλογο εξόδου.
```csharp
string sourceFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.ps");
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "ps-converted-to.pptx");
```
#### Βήμα 2: Φόρτωση και μετατροπή του αρχείου προέλευσης
Χρησιμοποιήστε το `Converter` κλάση για να φορτώσετε το αρχείο PS και να ορίσετε επιλογές μετατροπής.
```csharp
using (var converter = new Converter(sourceFilePath))
{
    var options = new PresentationConvertOptions(); // Ρύθμιση παραμέτρων για μορφή PPTX
    converter.Convert(outputFile, options); // Εκτελέστε τη μετατροπή
}
```
**Επεξήγηση παραμέτρων:**
- `sourceFilePath`: Διαδρομή προς το αρχείο PS εισόδου σας.
- `outputFile`: Διαδρομή προορισμού για το αρχείο PPTX που έχει μετατραπεί.
- `PresentationConvertOptions()`: Καθορίζει τη μετατροπή σε μορφή PowerPoint.

#### Συμβουλές αντιμετώπισης προβλημάτων:
- Βεβαιωθείτε ότι οι διαδρομές αρχείων είναι σωστές και προσβάσιμες.
- Επαληθεύστε την σωστή εγκατάσταση και αναφορά του GroupDocs.Conversion στο έργο σας.
- Ελέγξτε για εξαιρέσεις κατά τη μετατροπή, όπως μη υποστηριζόμενες μορφές ή προβλήματα δικαιωμάτων.
