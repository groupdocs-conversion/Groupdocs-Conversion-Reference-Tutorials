## Οδηγός Εφαρμογής

### Μετατροπή VSTX σε PPTX

**Επισκόπηση:**
Αυτή η λειτουργία επιδεικνύει τη μετατροπή αρχείων Visio (VSTX) σε παρουσιάσεις PowerPoint (PPTX) χρησιμοποιώντας το GroupDocs.Conversion για .NET.

#### Βήμα 1: Ορισμός καταλόγου εξόδου και διαδρομής αρχείου

Ρυθμίστε τον κατάλογο εξόδου και καθορίστε πού θα αποθηκευτεί το αρχείο που έχει μετατραπεί:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "vstx-converted-to.pptx");
```

#### Βήμα 2: Φόρτωση του αρχείου VSTX προέλευσης

Φορτώστε το αρχείο VSTX πηγής για μετατροπή:

```csharp
string sampleVstxPath = "YOUR_DOCUMENT_DIRECTORY/samples/sample.vstx"; // Διαδρομή προς το αρχείο VSTX εισόδου
```

#### Βήμα 3: Μετατροπή και αποθήκευση του αρχείου PPTX

Χρησιμοποιήστε το `Converter` τάξη και `PresentationConvertOptions` για να εκτελέσετε τη μετατροπή:

```csharp
using (Converter converter = new Converter(sampleVstxPath))
{
    PresentationConvertOptions options = new PresentationConvertOptions();
    // Μετατρέψτε και αποθηκεύστε το αρχείο PPTX.
    converter.Convert(outputFile, options);
}
```

**Παράμετροι & Σκοπός μεθόδου:**
- `sampleVstxPath`: Διαδρομή προς το αρχείο VSTX πηγής σας.
- `options`: Διαμορφώνει τις ρυθμίσεις μετατροπής για τη μορφή παρουσίασης.
