## Εισαγωγή χώρων ονομάτων
Πριν προχωρήσουμε στη μετατροπή, ας εισάγουμε τους απαραίτητους χώρους ονομάτων στο έργο μας.
```csharp
using System;
using System.IO;
using GroupDocs.Conversion.Options.Convert;
```
## Βήμα 1: Ορισμός φακέλου εξόδου και ονόματος αρχείου
Ξεκινήστε καθορίζοντας τον φάκελο στον οποίο θέλετε να αποθηκευτεί το αρχείο PDF που έχει μετατραπεί και το όνομα του αρχείου εξόδου.
```csharp
string outputFolder = "Your Document Directory";
string outputFile = Path.Combine(outputFolder, "mpx-converted-to.pdf");
```
## Βήμα 2: Φορτώστε το αρχείο προέλευσης MPX
Στη συνέχεια, φορτώστε το αρχείο MPX προέλευσης χρησιμοποιώντας το GroupDocs.Conversion.
```csharp
using (var converter = new GroupDocs.Conversion.Converter(Constants.SAMPLE_MPX))
{
    // Ο κωδικός μετατροπής θα πάει εδώ
}
```
## Βήμα 3: Ορίστε τις επιλογές μετατροπής
Καθορίστε τις επιλογές μετατροπής, προσδιορίζοντας τη μορφή εξόδου ως PDF.
```csharp
var options = new PdfConvertOptions();
```
## Βήμα 4: Εκτελέστε τη Μετατροπή
Εκτελέστε τη διαδικασία μετατροπής, μετατρέποντας το αρχείο MPX σε μορφή PDF.
```csharp
converter.Convert(outputFile, options);
```

Ενημερώστε τον χρήστη ότι η διαδικασία μετατροπής ολοκληρώθηκε με επιτυχία και δώστε τη θέση του αρχείου που μετατράπηκε.

