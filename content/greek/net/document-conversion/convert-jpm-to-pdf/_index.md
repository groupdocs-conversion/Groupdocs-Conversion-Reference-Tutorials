Καθορίστε το φάκελο στον οποίο θέλετε να αποθηκευτεί το αρχείο PDF που έχει μετατραπεί.
```csharp
string outputFolder = "Your Document Directory";
```
## Βήμα 2: Ορισμός διαδρομής αρχείου εξόδου
Συνδυάστε τη διαδρομή του φακέλου εξόδου με το επιθυμητό όνομα για το αρχείο PDF που μετατράπηκε.
```csharp
string outputFile = Path.Combine(outputFolder, "jpm-converted-to.pdf");
```
## Βήμα 3: Φορτώστε το αρχείο προέλευσης JPM
Εκκινήστε το αντικείμενο GroupDocs.Conversion Converter με τη διαδρομή προς το αρχείο προέλευσης JPM.

Προαιρετικά, μπορείτε να καθορίσετε επιλογές μετατροπής με βάση τις απαιτήσεις σας. Σε αυτήν την περίπτωση, πραγματοποιούμε μετατροπή σε PDF, επομένως θα χρησιμοποιήσουμε το PdfConvertOptions.
```csharp
var options = new PdfConvertOptions();
```
## Βήμα 5: Εκτελέστε μετατροπή
Εκτελέστε τη διαδικασία μετατροπής καλώντας τη μέθοδο Convert του αντικειμένου Converter και περνώντας τη διαδρομή του αρχείου εξόδου μαζί με τις επιλογές μετατροπής.
```csharp
converter.Convert(outputFile, options);
```
