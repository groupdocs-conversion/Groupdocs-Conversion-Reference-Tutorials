## Οδηγός Εφαρμογής

### Μετατροπή XPS σε TXT

Αυτή η ενότητα παρέχει μια λεπτομερή προσέγγιση για τη μετατροπή ενός αρχείου XPS σε μορφή κειμένου.

#### Επισκόπηση
Η μετατροπή αρχείων XPS σε TXT επιτρέπει ευκολότερο χειρισμό και ενσωμάτωση με άλλα συστήματα, καθώς το απλό κείμενο είναι προσβάσιμο σε όλους.

#### Βήμα προς βήμα εφαρμογή

**1. Ορισμός διαδρομών αρχείων**
Ρυθμίστε τους καταλόγους εισόδου και εξόδου:

```csharp
string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY\\";
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY\\";

// Εισαγωγή διαδρομής αρχείου XPS
string inputFile = Path.Combine(documentDirectory, "sample.xps");

// Διαδρομή αρχείου TXT εξόδου
string outputFile = Path.Combine(outputDirectory, "xps-converted-to.txt");
```

**2. Φορτώστε το αρχείο προέλευσης XPS**
Χρησιμοποιήστε το GroupDocs.Conversion για να φορτώσετε το αρχείο προέλευσης:

```csharp
using (var converter = new Converter(inputFile))
{
    Console.WriteLine("XPS file loaded successfully.");
}
```

**3. Ορισμός επιλογών μετατροπής**
Διαμόρφωση ρυθμίσεων μετατροπής για έξοδο TXT:

```csharp
var options = new WordProcessingConvertOptions { Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt };
```
Εδώ, `WordProcessingConvertOptions` καθορίζει τη μορφή κειμένου.

**4. Εκτελέστε τη μετατροπή**
Εκτελέστε τη διαδικασία μετατροπής και αποθηκεύστε το αποτέλεσμα:

```csharp
converter.Convert(outputFile, options);
Console.WriteLine("Conversion completed successfully.");
```
