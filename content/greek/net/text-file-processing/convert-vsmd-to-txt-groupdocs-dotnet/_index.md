## Οδηγός Εφαρμογής

### Μετατροπή VSDM σε TXT

Αυτή η ενότητα εστιάζει στη μετατροπή σχεδίων με δυνατότητα μακροεντολών του Visio σε απλό κείμενο.

#### Βήμα 1: Ρύθμιση καταλόγων
Ορίστε τους καταλόγους εισόδου και εξόδου. Αντικαταστήστε τα placeholders με πραγματικές διαδρομές στο περιβάλλον σας.

```csharp
string YOUR_DOCUMENT_DIRECTORY = "/path/to/your/documents";
string YOUR_OUTPUT_DIRECTORY = "/path/to/output/directory/";
```

#### Βήμα 2: Φόρτωση του αρχείου VSDM προέλευσης
Δημιουργήστε ένα `Converter` αντικείμενο για να φορτώσετε το αρχείο πηγής σας χρησιμοποιώντας την καθορισμένη διαδρομή. Αυτό το βήμα ξεκινά τη διαδικασία μετατροπής.

```csharp
using (var converter = new Converter(Path.Combine(YOUR_DOCUMENT_DIRECTORY, "yourfile.vsdm")))
{
    // Επιπλέον βήματα μετατροπής θα προστεθούν εδώ.
}
```

#### Βήμα 3: Διαμόρφωση επιλογών μετατροπής
Ορίστε τις απαραίτητες επιλογές για τη μετατροπή σε μορφή TXT. Καθορίστε ότι ο τύπος αρχείου προορισμού είναι TXT.

```csharp
WordProcessingConvertOptions options = new WordProcessingConvertOptions 
{
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt
};
```

#### Βήμα 4: Εκτελέστε τη μετατροπή
Εκτελέστε τη μετατροπή και αποθηκεύστε το αποτέλεσμα ως αρχείο TXT. Το αρχείο που προκύπτει θα αποθηκευτεί στον καθορισμένο κατάλογο εξόδου.

```csharp
string outputFile = Path.Combine(YOUR_OUTPUT_DIRECTORY, "vsdm-converted-to.txt");
converter.Convert(outputFile, options);
```

### Συμβουλές αντιμετώπισης προβλημάτων

- Βεβαιωθείτε ότι όλες οι διαδρομές έχουν ρυθμιστεί σωστά για να αποφύγετε `FileNotFoundException`.
