## Οδηγός Εφαρμογής
Ακολουθήστε αυτά τα βήματα για να μετατρέψετε ένα αρχείο PDF σε μορφή TXT.

### 1. Ορισμός καταλόγων για αρχεία εισόδου και εξόδου
Καθορίστε τους καταλόγους για τα αρχεία PDF πηγής και TXT εξόδου:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// Βεβαιωθείτε ότι υπάρχουν κατάλογοι ή δημιουργήστε τους
if (!Directory.Exists(documentDirectory)) Directory.CreateDirectory(documentDirectory);
if (!Directory.Exists(outputDirectory)) Directory.CreateDirectory(outputDirectory);
```

### 2. Φόρτωση και προετοιμασία του αρχείου PDF προέλευσης
Φορτώστε το αρχείο PDF πηγής χρησιμοποιώντας το `Converter` τάξη:

```csharp
string sourceFilePath = Path.Combine(documentDirectory, "sample.pdf");
string outputFile = Path.Combine(outputDirectory, "pdf-converted-to.txt");

using (var converter = new GroupDocs.Conversion.Converter(sourceFilePath))
{
    // Η μετατροπή θα γίνει στο επόμενο βήμα
}
```

### 3. Ορίστε τις επιλογές μετατροπής και εκτελέστε τη μετατροπή
Ρυθμίστε τις επιλογές μετατροπής για τη μορφή TXT και εκτελέστε τη μετατροπή:

```csharp
// Χρησιμοποιήστε το WordProcessingConvertOptions για μετατροπή σε μορφή κειμένου
WordProcessingConvertOptions options = new WordProcessingConvertOptions 
{ 
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt 
};

// Εκτέλεση μετατροπής και αποθήκευση στη διαδρομή αρχείου εξόδου
converter.Convert(outputFile, options);
```
