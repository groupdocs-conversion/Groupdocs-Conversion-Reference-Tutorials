## Οδηγός Εφαρμογής
### Μετατροπή αρχείου AI σε μορφή TXT
Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε αρχεία Adobe Illustrator σε μορφή απλού κειμένου για ευκολότερο χειρισμό ή ανάλυση δεδομένων.

#### Βήμα 1: Ορισμός καταλόγου εξόδου και ορισμός διαδρομής εξόδου
Ξεκινήστε καθορίζοντας τον κατάλογο εξόδου όπου θα αποθηκευτεί το αρχείο που μετατράπηκε. Αντικατάσταση `YOUR_OUTPUT_DIRECTORY` με μια πραγματική διαδρομή στο σύστημά σας.

```csharp
string outputFolder = @"C:\OutputDirectory\";
string outputFile = System.IO.Path.Combine(outputFolder, "ai-converted-to.txt");
```

#### Βήμα 2: Φόρτωση του αρχείου AI πηγής
Φορτώστε το αρχείο AI πηγής χρησιμοποιώντας το `GroupDocs.Conversion.Converter` κλάση. Αντικατάσταση `YOUR_DOCUMENT_DIRECTORY` με τη διαδρομή προς το αρχείο AI σας.

```csharp
using (var converter = new GroupDocs.Conversion.Converter(@"C:\DocumentDirectory\sample.ai"))
{
    // Θα ακολουθήσει η λογική μετατροπής.
}
```

#### Βήμα 3: Ορισμός επιλογών μετατροπής
Ορίστε τις επιλογές μετατροπής για να καθορίσετε ότι θέλετε μια μορφή εξόδου TXT. Αυτό είναι κρίσιμο για να καθορίσετε πώς θα πρέπει να μετατραπεί το αρχείο σας.

```csharp
var options = new GroupDocs.Conversion.Options.Convert.WordProcessingConvertOptions 
{ 
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt 
};
```

#### Βήμα 4: Εκτελέστε τη μετατροπή
Τέλος, εκτελέστε τη διαδικασία μετατροπής και αποθηκεύστε το αποτέλεσμα ως αρχείο κειμένου χρησιμοποιώντας τις καθορισμένες ρυθμίσεις.

```csharp
converter.Convert(outputFile, options);
```
