## Οδηγός Εφαρμογής

### Φόρτωση αρχείου MPP

#### Επισκόπηση
Η φόρτωση ενός αρχείου MPP είναι το πρώτο βήμα για τη μετατροπή του. Αυτή η λειτουργία σάς επιτρέπει να ανοίξετε και να προετοιμάσετε το αρχείο για περαιτέρω επεξεργασία.

##### Βήμα 1: Αρχικοποίηση του μετατροπέα
```csharp
using System;
using GroupDocs.Conversion;

string sourceFilePath = "YOUR_DOCUMENT_DIRECTORY\\sample.mpp"; // Βεβαιωθείτε ότι αυτή η διαδρομή είναι σωστή

// Η χρήση της δήλωσης διασφαλίζει την ορθή διάθεση των πόρων.
using (var converter = new Converter(sourceFilePath))
{
    // Σε αυτό το σημείο, το αρχείο MPP σας έχει φορτωθεί και είναι έτοιμο για μετατροπή.
}
```

**Εξήγηση**: Αυτό το απόσπασμα κώδικα αρχικοποιεί το `Converter` αντικείμενο με το αρχείο MPP προέλευσης. Το `using` Η δήλωση διασφαλίζει ότι οι πόροι απορρίπτονται σωστά μετά τη χρήση.

### Μετατροπή MPP σε TXT

#### Επισκόπηση
Μόλις φορτωθεί το αρχείο MPP, μπορείτε να το μετατρέψετε σε μορφή TXT. Αυτή η λειτουργία απλοποιεί την εξαγωγή δεδομένων έργου για επεξεργασία ή κοινή χρήση με βάση το κείμενο.

##### Βήμα 2: Ορισμός επιλογών μετατροπής
```csharp
using System;
using System.IO;
using GroupDocs.Conversion.Options.Convert;

string outputFolder = "YOUR_OUTPUT_DIRECTORY"; // Ορίστε τη διαδρομή του καταλόγου εξόδου σας
string outputFile = Path.Combine(outputFolder, "mpp-converted-to.txt");

// Επαναρχικοποιήστε τον Μετατροπέα με τη διαδρομή αρχείου MPP.
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY\\sample.mpp"))
{
    WordProcessingConvertOptions options = new WordProcessingConvertOptions { Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt };

    // Μετατρέψτε και αποθηκεύστε το αρχείο MPP σε μορφή TXT
    converter.Convert(outputFile, options);
}
```

**Εξήγηση**: Το `WordProcessingConvertOptions` Η κλάση καθορίζει ότι θέλουμε να μετατρέψουμε το αρχείο μας σε μορφή κειμένου. Στη συνέχεια, καλούμε την `Convert` μέθοδος επεξεργασίας και αποθήκευσης της εξόδου.

#### Συμβουλές αντιμετώπισης προβλημάτων
