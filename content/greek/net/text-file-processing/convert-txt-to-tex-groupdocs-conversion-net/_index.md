### Ρύθμιση παραμέτρων επιλογών μετατροπής

#### Επισκόπηση
Καθορίστε την επιθυμητή μορφή εξόδου: LaTeX (.tex).

#### Απόσπασμα κώδικα

```csharp
// Ρυθμίστε τις παραμέτρους των επιλογών μετατροπής για να καθορίσετε τη μορφή εξόδου ως LaTeX (.tex).
var options = new PageDescriptionLanguageConvertOptions
{
    Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Tex
};

// Εκτελέστε τη μετατροπή και αποθηκεύστε το αποτέλεσμα στον καθορισμένο κατάλογο εξόδου.
converter.Convert(outputFile, options);
```

**Εξήγηση**: Εδώ, `PageDescriptionLanguageConvertOptions` ορίζεται για να ορίσει `.tex` ως τη μορφή-στόχο μας, διασφαλίζοντας τη συμμόρφωση με τα πρότυπα LaTeX.
