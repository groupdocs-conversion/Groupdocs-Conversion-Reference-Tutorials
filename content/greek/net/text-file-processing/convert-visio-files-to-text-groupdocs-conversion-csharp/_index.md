#### Ορισμός επιλογών μετατροπής
Ορίστε επιλογές μετατροπής για τη μορφή TXT:

```csharp
// Ορίστε επιλογές μετατροπής για τη μορφή TXT.
WordProcessingConvertOptions options = new WordProcessingConvertOptions 
{
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt
};
```

#### Εκτέλεση της μετατροπής
Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο εξόδου:

```csharp
// Εκτελέστε τη μετατροπή και αποθηκεύστε το αποτέλεσμα ως αρχείο TXT.
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", ".");
string outputFile = Path.Combine(outputFolder, "vsd-converted-to.txt");
converter.Convert(outputFile, options);
```

- `WordProcessingConvertOptions`: Καθορίζει τη μετατροπή σε μορφή TXT.
- `converter.Convert(...)`: Εκτελεί τη διαδικασία μετατροπής.
