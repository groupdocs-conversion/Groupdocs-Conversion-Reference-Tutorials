### Φόρτωση και μετατροπή αρχείων

#### Επισκόπηση

Το GroupDocs.Conversion επιτρέπει την εύκολη φόρτωση και μετατροπή αρχείων DJVU σε κείμενο, χρησιμοποιώντας ισχυρές επιλογές μετατροπής.

##### Βήμα 1: Ορισμός διαδρομών αρχείων

Αρχικά, καθορίστε τους καταλόγους εγγράφων και εξόδου:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// Διαδρομή προς το αρχείο DJVU προέλευσης
class string djvuFilePath = Path.Combine(documentDirectory, "sample.djvu");
```

##### Βήμα 2: Ρύθμιση επιλογών μετατροπής

Επιλέξτε επιλογές μετατροπής προσαρμοσμένες για επεξεργασία κειμένου:

```csharp
var convertOptions = new WordProcessingConvertOptions
{
    Format = FileTypes.WordProcessingFileType.Txt
};
```

##### Βήμα 3: Εκτελέστε τη μετατροπή

Εκτελέστε τη μετατροπή και αποθηκεύστε την έξοδο:

```csharp
string txtOutputFile = Path.Combine(outputDirectory, "djvu-converted-to.txt");

using (var converter = new Converter(djvuFilePath))
{
    converter.Convert(txtOutputFile, convertOptions);
}
```

**Εξήγηση:** 
- **Κλάση μετατροπέα:** Αρχικοποιείται με το αρχείο DJVU σας.
- **Μέθοδος μετατροπής:** Μετατρέπει το αρχείο χρησιμοποιώντας καθορισμένες επιλογές και το αποθηκεύει.

##### Συμβουλές αντιμετώπισης προβλημάτων

- Βεβαιωθείτε ότι οι διαδρομές έχουν οριστεί σωστά για να αποφύγετε `FileNotFoundException`.
- Ελέγξτε τη συμβατότητα των εκδόσεων του GroupDocs.Conversion.
