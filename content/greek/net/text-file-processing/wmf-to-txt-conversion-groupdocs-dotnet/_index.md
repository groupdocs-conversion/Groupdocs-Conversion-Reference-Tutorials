## Οδηγός Εφαρμογής

Τώρα που έχετε ρυθμίσει τα πάντα, ας επικεντρωθούμε στην εφαρμογή της μετατροπής WMF σε TXT.

### Βήμα 1: Ορισμός διαδρομών αρχείων

Αρχικά, καθορίστε πού θα βρίσκονται τα αρχεία προέλευσης και προορισμού:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

string inputFile = Path.Combine(documentDirectory, "sample.wmf");
string outputFile = Path.Combine(outputDirectory, "wmf-converted-to.txt");
```

### Βήμα 2: Διαμόρφωση επιλογών μετατροπής

Στη συνέχεια, ορίστε τις παραμέτρους μετατροπής για να καθορίσετε ότι η μορφή-στόχος σας είναι TXT:

```csharp
using GroupDocs.Conversion.Options.Convert;

var options = new WordProcessingConvertOptions { Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt };
```

### Βήμα 3: Εκτελέστε τη μετατροπή

Τέλος, εκτελέστε τη διαδικασία μετατροπής και αποθηκεύστε το αρχείο εξόδου:

```csharp
using (var converter = new Converter(inputFile))
{
    converter.Convert(outputFile, options);
}
Console.WriteLine("Conversion completed successfully!");
```
