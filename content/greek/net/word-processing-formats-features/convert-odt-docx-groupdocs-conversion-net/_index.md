### Βασική Αρχικοποίηση
Δείτε πώς μπορείτε να αρχικοποιήσετε και να ρυθμίσετε το GroupDocs.Conversion στο έργο σας C#:

```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

namespace OdtToDocxConversion
{
    class Program
    {
        static void Main(string[] args)
        {
            string sourceOdtPath = "YOUR_DOCUMENT_DIRECTORY/sample.odt";
            string outputFile = "YOUR_OUTPUT_DIRECTORY/odt-converted-to.docx";

            using (var converter = new Converter(sourceOdtPath))
            {
                var options = new WordProcessingConvertOptions();
                converter.Convert(outputFile, options);
            }
        }
    }
}
```
Αυτό το απόσπασμα κώδικα δείχνει τη φόρτωση ενός αρχείου ODT και τη μετατροπή του σε μορφή DOCX.
