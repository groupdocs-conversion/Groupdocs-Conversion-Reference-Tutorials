### Βασική Αρχικοποίηση και Ρύθμιση
Μόλις εγκατασταθεί, αρχικοποιήστε το GroupDocs.Conversion στο έργο σας:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main(string[] args)
    {
        string outputFolder = Path.Combine(@"YOUR_OUTPUT_DIRECTORY");
        string outputFile = Path.Combine(outputFolder, "fods-converted-to.doc");

        using (var converter = new Converter(@"YOUR_DOCUMENT_DIRECTORY\sample.fods"))
        {
            WordProcessingConvertOptions options = new WordProcessingConvertOptions { Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc };
            converter.Convert(outputFile, options);
        }
    }
}
```

Σε αυτόν τον κώδικα:
- Καθορίστε τον κατάλογο εξόδου και το όνομα αρχείου.
- Αρχικοποίηση ενός `Converter` αντικείμενο με τη διαδρομή αρχείου FODS σας.
- Ορίστε επιλογές μετατροπής για τη μορφή DOC χρησιμοποιώντας `WordProcessingConvertOptions`.
- Εκτελέστε τη μετατροπή.

## Οδηγός Εφαρμογής
Τώρα, ας εξερευνήσουμε κάθε χαρακτηριστικό της εφαρμογής μας.

### Μετατροπή FODS σε DOC

#### Φόρτωση του αρχείου FODS πηγής
Φορτώστε το αρχείο προέλευσης FODS αντικαθιστώντας `'YOUR_DOCUMENT_DIRECTORY\sample.fods'` με την πραγματική διαδρομή εγγράφου:

```csharp
using (var converter = new Converter(@"YOUR_DOCUMENT_DIRECTORY\sample.fods"))
```

Αυτή η γραμμή αρχικοποιεί ένα `Converter` αντικείμενο, προετοιμάζοντας το αρχείο σας για μετατροπή.

#### Ορισμός επιλογών μετατροπής
Καθορίστε ότι θέλετε η έξοδος να είναι σε μορφή DOC χρησιμοποιώντας `WordProcessingConvertOptions`:

```csharp
WordProcessingConvertOptions options = new WordProcessingConvertOptions { Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc };
```

Αυτή η διαμόρφωση ορίζει τη μορφή-στόχο και μπορεί να προσαρμοστεί περαιτέρω.

#### Εκτελέστε τη μετατροπή
Τέλος, καλέστε τον `Convert` μέθοδος για την επεξεργασία του αρχείου σας και την αποθήκευσή του στην επιθυμητή τοποθεσία:

```csharp
converter.Convert(outputFile, options);
```
