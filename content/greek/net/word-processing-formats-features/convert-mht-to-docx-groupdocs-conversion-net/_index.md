## Οδηγός Εφαρμογής

Ας αναλύσουμε τη διαδικασία μετατροπής από MHT σε DOCX σε διαχειρίσιμα βήματα:

### Φόρτωση και προετοιμασία του αρχείου προέλευσης

Αρχικά, φορτώστε το αρχείο MHT πηγής σας ορίζοντας μια διαδρομή προς τον κατάλογο εγγράφων σας:

```csharp
string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY\sample.mht");
```

### Μετατροπή MHT σε DOCX

Στη συνέχεια, εκτελέστε τη μετατροπή χρησιμοποιώντας τις ενσωματωμένες λειτουργίες του GroupDocs.Conversion.

**Αρχικοποίηση μετατροπέα**

```csharp
using (var converter = new GroupDocs.Conversion.Converter(inputFilePath))
{
    // Ορίστε επιλογές για μετατροπή σε μορφή DOCX
    var convertOptions = converter.GetPossibleConversions()["docx"].ConvertOptions;

    // Εκτελέστε τη μετατροπή
    string outputFile = Path.Combine("YOUR_OUTPUT_DIRECTORY", "output.docx");
    converter.Convert(outputFile, convertOptions);
}
```

**Εξήγηση**: 
- `GetPossibleConversions()` ανακτά όλες τις πιθανές μορφές προορισμού. Εδώ καθορίζουμε το DOCX ως την επιθυμητή μορφή.
- Ο `converter.Convert()` Η συνάρτηση χειρίζεται τον πραγματικό μετασχηματισμό αρχείου.
