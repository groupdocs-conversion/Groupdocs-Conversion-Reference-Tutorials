## Οδηγός Εφαρμογής

### Φόρτωση αρχείου VSTM πηγής

#### Επισκόπηση
Η φόρτωση ενός αρχείου VSTM είναι το πρώτο βήμα στη διαδικασία μετατροπής. Αυτό διασφαλίζει ότι η εφαρμογή σας μπορεί να έχει πρόσβαση και να χειριστεί το αρχείο χρησιμοποιώντας το GroupDocs.Conversion.

#### Οδηγίες βήμα προς βήμα

**Εξήγηση:**
- Ο `Converter` Η κλάση αρχικοποιείται με τη διαδρομή του αρχείου VSTM σας, προετοιμάζοντάς το για μετατροπή.

### Μετατροπή VSTM σε μορφή DOC

#### Επισκόπηση
Η μετατροπή ενός αρχείου VSTM σε μορφή DOC είναι απλή χρησιμοποιώντας το GroupDocs.Conversion. Αυτό σας επιτρέπει να χρησιμοποιήσετε το μετατρεπόμενο έγγραφο σε εφαρμογές που υποστηρίζονται ευρέως, όπως το Microsoft Word.

#### Οδηγίες βήμα προς βήμα
**Ορισμός καταλόγου εξόδου και διαδρομής αρχείου**

```csharp
using System;
using System.IO;
using GroupDocs.Conversion.Options.Convert;

// Ορίστε διαδρομές για τον κατάλογο και το αρχείο εξόδου
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
if (!Directory.Exists(outputFolder))
{
    Directory.CreateDirectory(outputFolder);
}

string outputFile = Path.Combine(outputFolder, "vstm-converted-to.doc");
```
**Φόρτωση αρχείου VSTM πηγής και μετατροπή**

```csharp
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY/source.vstm"))
{
    // Ορισμός επιλογών μετατροπής για επεξεργασία Word σε μορφή DOC
    var options = new WordProcessingConvertOptions { Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc };

    // Εκτελέστε τη μετατροπή και αποθηκεύστε το στο αρχείο εξόδου
    converter.Convert(outputFile, options);
}
```
**Εξήγηση:**
- `WordProcessingConvertOptions`: Καθορίζει τη μετατροπή σε μορφή DOC.
- `converter.Convert()`: Εκτελεί τη διαδικασία μετατροπής.
