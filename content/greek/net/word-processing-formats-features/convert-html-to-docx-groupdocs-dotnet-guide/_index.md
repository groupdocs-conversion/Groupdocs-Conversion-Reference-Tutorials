## Ρύθμιση του GroupDocs.Conversion για .NET

Για να ξεκινήσετε, εγκαταστήστε το GroupDocs.Conversion. Δείτε πώς:

**Κονσόλα διαχείρισης πακέτων NuGet**
```bash
Install-Package GroupDocs.Conversion -Version 25.3.0
```

**.NET CLI**
```bash
dotnet add package GroupDocs.Conversion --version 25.3.0
```

## Οδηγός Εφαρμογής

Τώρα που είστε έτοιμοι, ας δούμε τη διαδικασία μετατροπής.

### Δυνατότητα μετατροπής HTML σε DOCX
Αυτή η λειτουργία επιτρέπει την απρόσκοπτη μετατροπή αρχείων HTML σε έγγραφα Word. Δείτε πώς:

#### Βήμα 1: Ορισμός καταλόγου εξόδου και διαδρομής αρχείου
Καθορίστε πού θα αποθηκευτεί το αρχείο που μετατρέψατε:
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY/";
string outputFile = Path.Combine(outputFolder, "html-converted-to.docx");
```

#### Βήμα 2: Φόρτωση του αρχείου HTML πηγής
Χρησιμοποιήστε ένα `Converter` αντικείμενο για να φορτώσει το έγγραφο HTML σας, ξεκινώντας τη διαδικασία μετατροπής:
```csharp
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY\\sample.html"))
{
    // Η λογική μετατροπής θα εφαρμοστεί εδώ
}
```

#### Βήμα 3: Αρχικοποίηση επιλογών μετατροπής για μορφή επεξεργασίας κειμένου
Ορίστε τις συγκεκριμένες επιλογές που απαιτούνται για τη μετατροπή DOCX:
```csharp
var options = new WordProcessingConvertOptions();
```

#### Βήμα 4: Εκτελέστε τη μετατροπή
Εκτελέστε τη μετατροπή και αποθηκεύστε το έγγραφό σας στην καθορισμένη διαδρομή:
```csharp
converter.Convert(outputFile, options);
```
