## Οδηγός Εφαρμογής

Ας αναλύσουμε την υλοποίηση σε λογικά βήματα.

### Επισκόπηση λειτουργιών: Μετατροπή PPSM σε DOCX
Αυτή η λειτουργία σάς επιτρέπει να μετατρέπετε απρόσκοπτα αρχεία PowerPoint Slide Show (.ppsm) σε έγγραφα Word (.docx), καθιστώντας την κοινή χρήση εγγράφων πιο ευέλικτη και προσβάσιμη.

#### Βήμα 1: Ορισμός καταλόγου εξόδου
Ορίστε έναν κατάλογο εξόδου όπου θα αποθηκευτούν τα αρχεία που έχετε μετατρέψει. Αντικαταστήστε `YOUR_OUTPUT_DIRECTORY` με την επιθυμητή διαδρομή:
```csharp
string outputFolder = Path.Combine(@"YOUR_OUTPUT_DIRECTORY");
```
**Γιατί αυτό έχει σημασία**Ο ορισμός μιας σαφούς θέσης εξόδου βοηθά στην αποτελεσματική διαχείριση και ανάκτηση των μετατρεπόμενων εγγράφων.

#### Βήμα 2: Ρύθμιση του μετατροπέα
Δημιουργήστε μια παρουσία του `Converter` κλάση, που περνάει στη διαδρομή του αρχείου PPSM πηγής:
```csharp
using (var converter = new Converter("source.ppsm"))
{
    // Η λογική μετατροπής ισχύει εδώ
}
```
**Εξήγηση**Αυτό το αντικείμενο διαχειρίζεται τη διαδικασία μετατροπής, χειριζόμενο εσωτερικά όλες τις απαραίτητες λειτουργίες.

#### Βήμα 3: Ρύθμιση παραμέτρων επιλογών DOCX
Ορίστε την επιθυμητή μορφή εξόδου χρησιμοποιώντας `WordProcessingConvertOptions`:
```csharp
var options = new WordProcessingConvertOptions();
```
**Διαμόρφωση κλειδιού**: Αυτό ορίζει παραμέτρους που αφορούν συγκεκριμένα τη μορφοποίηση και τη συμπεριφορά μετατροπής εγγράφων του Word.

#### Βήμα 4: Εκτελέστε τη μετατροπή
Εκτελέστε τη μετατροπή καλώντας το `Convert` μέθοδος, περνώντας τις επιλογές σας και τη διαδρομή εξόδου:
```csharp
converter.Convert(Path.Combine(outputFolder, "output.docx"), options);
```
**Αποτέλεσμα**Το αρχείο PPSM σας έχει πλέον μετατραπεί σε μορφή DOCX, έτοιμο για χρήση ή διανομή.
