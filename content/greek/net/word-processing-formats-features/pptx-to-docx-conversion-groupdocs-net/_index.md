## Οδηγός Εφαρμογής

### Μετατροπή PPTX σε DOCX βήμα προς βήμα

Αυτή η λειτουργία επιτρέπει την απρόσκοπτη μετατροπή από παρουσιάσεις PowerPoint σε έγγραφα Word.

#### Βήμα 1: Προετοιμασία των αρχείων σας

Βεβαιωθείτε ότι οι διαδρομές εισόδου και εξόδου είναι σωστές:

```csharp
string inputFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "presentation.pptx");
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
```

#### Βήμα 2: Διαμόρφωση επιλογών μετατροπής

Καθορίστε τη μορφή προορισμού (DOCX):

```csharp
var convertOptions = new WordProcessingConvertOptions();
```

#### Βήμα 3: Εκτελέστε τη μετατροπή

Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο DOCX:

```csharp
string outputFile = Path.Combine(outputFolder, "output.docx");
class ConverterExample {
    public void ConvertPptxToDocx() {
        var converter = new Converter(inputFile);
        converter.Convert(() => File.Create(outputFile), convertOptions);
    }
}
```

**Εξήγηση**: Το `Convert` Η μέθοδος χρησιμοποιεί μια συνάρτηση λάμδα για τη δημιουργία αρχείων, διασφαλίζοντας ότι η διαδρομή εξόδου είναι έγκυρη. Οι επιλογές μετατροπής ορίζουν τη δομή DOCX.

#### Συμβουλές αντιμετώπισης προβλημάτων
- **Σφάλματα διαδρομής αρχείου**: Ελέγξτε ξανά τις διαδρομές εισόδου και εξόδου.
- **Λείπουν οι εξαρτήσεις**Βεβαιωθείτε ότι όλα τα απαραίτητα πακέτα έχουν εγκατασταθεί μέσω του NuGet.
