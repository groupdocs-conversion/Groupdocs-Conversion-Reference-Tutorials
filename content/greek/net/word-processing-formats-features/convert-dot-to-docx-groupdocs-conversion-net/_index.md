## Οδηγός Εφαρμογής

### Μετατροπή αρχείου .dot σε .docx

Αυτή η ενότητα θα σας καθοδηγήσει στη μετατροπή ενός προτύπου εγγράφου του Microsoft Word (.dot) σε ένα έγγραφο Open XML (.docx).

#### Βήμα 1: Ορισμός διαδρομών προέλευσης και εξόδου

Αρχικά, καθορίστε τις διαδρομές για την πηγή σας `.dot` αρχείο και ο προορισμός `.docx` αρχείο:

```csharp
string sampleDotPath = "YOUR_DOCUMENT_DIRECTORY\\sample.dot";
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "dot-converted-to.docx");
```

#### Βήμα 2: Φόρτωση του αρχείου DOT προέλευσης

Φορτώστε το αρχείο προέλευσης χρησιμοποιώντας το GroupDocs.Conversion:

```csharp
using (var converter = new Converter(sampleDotPath))
{
    // Η λογική μετατροπής θα εφαρμοστεί εδώ.
}
```

#### Βήμα 3: Ρύθμιση επιλογών μετατροπής

Ορίστε τις επιλογές μετατροπής για μορφές επεξεργασίας Word:

```csharp
var options = new WordProcessingConvertOptions();
```

#### Βήμα 4: Εκτελέστε τη μετατροπή

Τώρα, εκτελέστε τη μετατροπή και αποθηκεύστε την έξοδο ως `.docx` αρχείο:

```csharp
converter.Convert(outputFile, options);
```

### Επεξήγηση των Στοιχείων Κώδικα

- **Αντικείμενο Μετατροπέα**: Φορτώνει το έγγραφο προέλευσης.
- **Επεξεργασία ΚειμένουConvertOptions**: Καθορίζει ότι μετατρέπετε σε μορφή επεξεργασίας Word.
- **Μέθοδος μετατροπής**: Εκτελεί τη μετατροπή και αποθηκεύει το αποτέλεσμα.
