Αρχικά, ορίστε διαδρομές για τους καταλόγους εισόδου και εξόδου:
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
```

Φορτώστε το αρχείο VSTX από τον καθορισμένο κατάλογο. Αυτό το βήμα διασφαλίζει ότι το αρχείο προέλευσης είναι έτοιμο για μετατροπή:

**Αρχικοποίηση μετατροπέα**
```csharp
// Φόρτωση του αρχείου VSTX πηγής
using (var converter = new Converter(Path.Combine(documentDirectory, "sample.vstx")))
{
    // Η λογική μετατροπής θα εφαρμοστεί εδώ.
}
```

### Καθορισμός επιλογών μετατροπής
Στη συνέχεια, καθορίστε τις επιλογές για τη μετατροπή σε μορφή DOCX. Αυτό το βήμα ρυθμίζει τον τρόπο με τον οποίο θα πρέπει να μορφοποιηθεί το έγγραφο εξόδου σας:

**Ορισμός επιλογών μετατροπής**
```csharp
// Καθορισμός επιλογών μετατροπής για Επεξεργασία Κειμένου (DOCX)
var options = new WordProcessingConvertOptions();
```

### Εκτέλεση της μετατροπής
Τέλος, εκτελέστε την πραγματική μετατροπή και αποθηκεύστε το αρχείο DOCX που προκύπτει στην επιθυμητή τοποθεσία. Εδώ βλέπετε τη μαγεία να συμβαίνει:

**Μετατροπή και Αποθήκευση**
```csharp
// Ορίστε τη διαδρομή εξόδου για το αρχείο DOCX που έχει μετατραπεί
string outputFile = Path.Combine(outputDirectory, "vstx-converted-to.docx");

// Εκτέλεση μετατροπής

```
