### Μετατροπή POTX σε DOCX

#### Επισκόπηση

Η μετατροπή ενός αρχείου POTX σε μορφή DOCX σάς επιτρέπει να μετατρέψετε πρότυπα παρουσιάσεων σε επεξεργάσιμα έγγραφα Word, παρέχοντας μεγαλύτερη ευελιξία για την τροποποίηση περιεχομένου.

#### Βήματα Υλοποίησης

**Βήμα 1: Ορισμός διαδρομών αρχείων**

Αρχικά, ρυθμίστε τις διαδρομές των εγγράφων σας χρησιμοποιώντας δυναμικές μεθόδους για να προσαρμόσετε διαφορετικά περιβάλλοντα:

```csharp
using System;
using System.IO;

class FilePathConfigurator
{
    string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
    string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

    public string GetFilePath(string fileName, bool isOutput = false)
    {
        string directory = isOutput ? outputDirectory : documentDirectory;
        return Path.Combine(directory, fileName);
    }
}
```

**Βήμα 2: Εκτελέστε τη μετατροπή**

Τώρα, ας μετατρέψουμε το αρχείο POTX σε DOCX χρησιμοποιώντας το GroupDocs.Conversion:

```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class ConverterDemo
{
    static void Main()
    {
        FilePathConfigurator filePathConfig = new FilePathConfigurator();
        string documentPath = filePathConfig.GetFilePath("sample.potx");
        string outputFile = filePathConfig.GetFilePath("potx-converted-to.docx", true);

        using (var converter = new Converter(documentPath))
        {
            var options = new WordProcessingConvertOptions(); // Επιλογές για μετατροπή DOCX
            converter.Convert(outputFile, options); // Εκτελέστε τη μετατροπή

            Console.WriteLine($"Conversion successful. File saved to: {outputFile}");
        }
    }
}
```

Αυτό το απόσπασμα κώδικα φορτώνει το αρχείο POTX και το μετατρέπει σε μορφή DOCX χρησιμοποιώντας συγκεκριμένες επιλογές μετατροπής.
