### Βασική Αρχικοποίηση

Δείτε πώς μπορείτε να αρχικοποιήσετε και να ρυθμίσετε το GroupDocs.Conversion στο έργο σας C#:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

// Ορίστε διαδρομές για το αρχείο εισόδου EMF και το αρχείο εξόδου DOCX
string inputFilePath = "YOUR_DOCUMENT_DIRECTORY/sample.emf";
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", "ConvertedFiles");
string outputFile = Path.Combine(outputFolder, "emf-converted-to.docx");

// Βεβαιωθείτε ότι ο κατάλογος εξόδου υπάρχει
Directory.CreateDirectory(outputFolder);

try
{
    // Αρχικοποιήστε τον μετατροπέα με τη διαδρομή αρχείου EMF πηγής
    using (var converter = new Converter(inputFilePath))
    {
        // Ορισμός επιλογών μετατροπής για τη μορφή επεξεργασίας κειμένου
        var options = new WordProcessingConvertOptions();
        
        // Μετατρέψτε και αποθηκεύστε το αρχείο DOCX στον καθορισμένο κατάλογο εξόδου
        converter.Convert(outputFile, options);
    }
}
catch (Exception ex)
{
    Console.WriteLine("An error occurred: " + ex.Message);
}
```

Αυτό το απόσπασμα κώδικα δείχνει πώς να φορτώσετε ένα αρχείο EMF, να διαμορφώσετε τις ρυθμίσεις μετατροπής για μορφές επεξεργασίας Word και να εκτελέσετε τη μετατροπή.
