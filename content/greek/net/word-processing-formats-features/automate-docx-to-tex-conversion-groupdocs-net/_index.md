## Οδηγός Εφαρμογής
### Δυνατότητα μετατροπής DOCX σε TEX
Αυτοματοποιήστε τη μετατροπή από DOCX σε TEX, εξοικονομώντας χρόνο και προσπάθεια. Δείτε πώς:

#### Βήμα 1: Ορισμός καταλόγου εξόδου και διαδρομής αρχείου
Αρχικά, καθορίστε πού θα αποθηκευτεί το αρχείο που έχει μετατραπεί:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = System.IO.Path.Combine(outputFolder, "docx-converted-to.tex");
```
Εδώ, `outputFolder` είναι το σημείο όπου θέλετε να αποθηκεύσετε τα αρχεία TEX σας. Το `outputFile` συνδυάζει αυτήν τη διαδρομή με ένα καθορισμένο όνομα αρχείου.

#### Βήμα 2: Φόρτωση του αρχείου DOCX πηγής
Στη συνέχεια, φορτώστε το έγγραφο Word πηγής σας:

```csharp
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_DOCX.docx"))
{
    // Η διαδικασία μετατροπής θα ξεκινήσει εδώ
}
```
Ο `Converter` Το αντικείμενο ακολουθεί μια διαδρομή προς το αρχείο DOCX. Βεβαιωθείτε ότι αυτή η διαδρομή είναι σωστή για να αποφύγετε σφάλματα.

#### Βήμα 3: Καθορισμός επιλογών μετατροπής
Ορίστε τις επιλογές μετατροπής για τη μορφή TEX:

```csharp
PageDescriptionLanguageConvertOptions options = new PageDescriptionLanguageConvertOptions 
{
    Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Tex
};
```
Ο `options` Το αντικείμενο ρυθμίζει τον μετατροπέα ώστε να εξάγει ένα αρχείο TEX. Προσαρμόστε τις παραμέτρους όπως απαιτείται για συγκεκριμένες απαιτήσεις.

#### Βήμα 4: Εκτελέστε τη μετατροπή
Εκτελέστε τη μετατροπή:

```csharp
converter.Convert(outputFile, options);
Console.WriteLine("Conversion completed successfully.");
```
Αυτή η μέθοδος γράφει το μετατρεπόμενο περιεχόμενο στο καθορισμένο `outputFile`.
