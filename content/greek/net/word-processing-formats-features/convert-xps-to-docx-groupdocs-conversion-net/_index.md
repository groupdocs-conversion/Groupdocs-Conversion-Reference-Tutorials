## Οδηγός Εφαρμογής

Αυτή η ενότητα περιγράφει τη μετατροπή ενός εγγράφου XPS σε μορφή DOCX.

### Φόρτωση και μετατροπή XPS σε DOCX

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να φορτώσετε ένα υπάρχον αρχείο XPS και να το μετατρέψετε σε έγγραφο Word (DOCX) χρησιμοποιώντας το GroupDocs.Conversion.

##### Βήμα 1: Ορισμός καταλόγων εισόδου και εξόδου
Αρχικά, καθορίστε τους καταλόγους για το αρχείο προέλευσης XPS και το αρχείο εξόδου DOCX:
```csharp
string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY\";
string outputFileDirectory = @"YOUR_OUTPUT_DIRECTORY\";
```

##### Βήμα 2: Φόρτωση του αρχείου προέλευσης XPS
Χρησιμοποιώντας το GroupDocs.Conversion, φορτώστε το έγγραφο XPS που θέλετε να μετατρέψετε:
```csharp
using (var converter = new Converter(documentDirectory + "sample.xps"))
{
    Console.WriteLine("XPS file loaded.");
}
```

##### Βήμα 3: Ορισμός επιλογών μετατροπής για τη μορφή DOCX
Καθορίστε ότι θέλετε να μετατρέψετε το έγγραφό σας σε μορφή επεξεργασίας κειμένου (DOCX):
```csharp
var options = new WordProcessingConvertOptions();
Console.WriteLine("Conversion options set.");
```

##### Βήμα 4: Εκτελέστε τη μετατροπή και αποθηκεύστε το αποτέλεσμα
Εκτελέστε τη διαδικασία μετατροπής και αποθηκεύστε το αποτέλεσμα ως αρχείο DOCX:
```csharp
string outputFile = Path.Combine(outputFileDirectory, "xps-converted-to.docx");
converter.Convert(outputFile, options);
Console.WriteLine("Conversion completed successfully.");
```
