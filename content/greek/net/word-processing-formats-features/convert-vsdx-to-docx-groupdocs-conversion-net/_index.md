### Βασική Αρχικοποίηση και Ρύθμιση
Αρχικοποιήστε το GroupDocs.Conversion στο έργο σας C#:
```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

// Ορίστε διαδρομές για αρχεία VSD εισόδου και αρχεία DOCX εξόδου
string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY";

if (!Directory.Exists(outputDirectory))
{
    Directory.CreateDirectory(outputDirectory);
}

string inputFile = Path.Combine(documentDirectory, "sample.vsd");
string outputFile = Path.Combine(outputDirectory, "vsd-converted-to.docx");

using (var converter = new Converter(inputFile))
{
    var options = new WordProcessingConvertOptions();
    converter.Convert(outputFile, options);
}
```

## Οδηγός Εφαρμογής
### Μετατροπή αρχείου VSD σε μορφή DOCX
#### Επισκόπηση
Αυτή η λειτουργία επιτρέπει την εύκολη μετατροπή ενός αρχείου VSD σε μορφή DOCX χρησιμοποιώντας το GroupDocs.Conversion.

#### Βήματα για τη μετατροπή
**1. Ορισμός Διαδρομών**
Βεβαιωθείτε ότι οι κατάλογοι εισόδου και εξόδου έχουν ρυθμιστεί σωστά:
```csharp
string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY";

if (!Directory.Exists(outputDirectory))
{
    Directory.CreateDirectory(outputDirectory);
}
```
**2. Φορτώστε το αρχείο VSD**
Αρχικοποίηση ενός `Converter` αντικείμενο με τη διαδρομή του αρχείου προέλευσης.
```csharp
string inputFile = Path.Combine(documentDirectory, "sample.vsd");
using (var converter = new Converter(inputFile))
{
    // Η λογική της μετατροπής ακολουθεί...
}
```
**3. Διαμόρφωση επιλογών μετατροπής**
Ορίστε επιλογές για τη διαδικασία μετατροπής:
```csharp
var options = new WordProcessingConvertOptions();
// Πρόσθετες ρυθμίσεις μπορούν να οριστούν εδώ
```
**4. Εκτελέστε τη μετατροπή**
Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο DOCX εξόδου:
```csharp
string outputFile = Path.Combine(outputDirectory, "vsd-converted-to.docx");
converter.Convert(outputFile, options);
```
