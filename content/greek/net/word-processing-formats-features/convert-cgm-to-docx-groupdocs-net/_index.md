## Οδηγός Εφαρμογής
Σε αυτήν την ενότητα, θα σας καθοδηγήσουμε στη μετατροπή ενός αρχείου CGM σε DOCX χρησιμοποιώντας το GroupDocs.Conversion.

### Μετατροπή CGM σε DOCX
**Επισκόπηση:**
Μετατρέψτε αρχεία CGM σε έγγραφα DOCX απρόσκοπτα. Αυτό είναι χρήσιμο για την ενσωμάτωση γραφικών σε περιεχόμενο επεξεργασίας κειμένου.

#### Βήμα προς βήμα εφαρμογή:
**Βήμα 1: Ρύθμιση παραμέτρων διαδρομών αρχείων**
Ορίστε τις διαδρομές εισόδου και εξόδου χρησιμοποιώντας σύμβολα κράτησης θέσης ή πραγματικούς καταλόγους:
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
// Κατασκευάστε την πλήρη διαδρομή για το αρχείο προέλευσης
string inputFile = Path.Combine(documentDirectory, "sample.cgm");
// Κατασκευάστε την πλήρη διαδρομή για το αρχείο προορισμού
string outputFile = Path.Combine(outputDirectory, "cgm-converted-to.docx");
```
**Βήμα 2: Αρχικοποίηση και μετατροπή**
Αρχικοποιήστε το αντικείμενο μετατροπέα με το αρχείο CGM εισόδου σας. Ορίστε τις επιλογές μετατροπής για τη μορφή προορισμού DOCX:
```csharp
using (var converter = new Converter(inputFile))
{
    // Ορισμός επιλογών μετατροπής για τη μορφή επεξεργασίας κειμένου
    var options = new GroupDocs.Conversion.Options.Convert.WordProcessingConvertOptions();
    
    // Μετατρέψτε και αποθηκεύστε το αρχείο DOCX στην καθορισμένη διαδρομή εξόδου
    converter.Convert(outputFile, options);
}
```
**Εξήγηση:**
- `Converter`: Αρχικοποιείται με το αρχείο CGM πηγής σας.
- `WordProcessingConvertOptions`: Καθορίζει τη μετατροπή σε μορφή συμβατή με το Word (DOCX).
- `converter.Convert()`: Εκτελεί τη μετατροπή και αποθηκεύει το αποτέλεσμα.
