## Οδηγός Εφαρμογής
### Φόρτωση και μετατροπή αρχείου DOCM σε DOCX

**Επισκόπηση:** Αυτή η λειτουργία σάς επιτρέπει να φορτώσετε ένα αρχείο DOCM και να το μετατρέψετε σε μορφή DOCX χρησιμοποιώντας το GroupDocs.Conversion. Επιτρέπει την εργασία με ευέλικτα έγγραφα Word στις εφαρμογές σας.

**Βήματα για τη μετατροπή**

**Βήμα 1: Ορισμός διαδρομών**
Καθορίστε διαδρομές για το αρχείο DOCM προέλευσης και το αρχείο DOCX εξόδου:

```csharp
string sourceDocmPath = Path.Combine(@"YOUR_DOCUMENT_DIRECTORY", "sample.docm");
string outputFile = Path.Combine(@"YOUR_OUTPUT_DIRECTORY", "docm-converted-to.docx");
```

**Βήμα 2: Φόρτωση του αρχείου προέλευσης**
Δημιουργήστε μια παρουσία του `Converter` κλάση με τη διαδρομή του αρχείου DOCM σας:

```csharp
using (var converter = new Converter(sourceDocmPath))
{
    // Η λογική μετατροπής ισχύει εδώ
}
```
Ο `Converter` Το αντικείμενο χειρίζεται τη φόρτωση εγγράφων και παρέχει μεθόδους για εργασίες μετατροπής.

**Βήμα 3: Καθορισμός επιλογών μετατροπής**
Ορίστε τη μορφή στόχου χρησιμοποιώντας `WordProcessingConvertOptions`:

```csharp
var options = new WordProcessingConvertOptions();
```
Αυτό καθορίζει ότι μετατρέπετε σε αρχείο DOCX, μέρος των μορφών επεξεργασίας κειμένου του Office της Microsoft.

**Βήμα 4: Εκτελέστε τη μετατροπή**
Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο DOCX εξόδου:

```csharp
converter.Convert(outputFile, options);
```
Εδώ, `converter.Convert` Μετατρέπει το DOCM σε DOCX χρησιμοποιώντας καθορισμένες επιλογές.
