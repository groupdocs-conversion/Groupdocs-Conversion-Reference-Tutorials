## Οδηγός Εφαρμογής

### Μετατροπή VSD σε DOC

#### Επισκόπηση
Αυτή η δυνατότητα μετατρέπει αρχεία σχεδίασης του Visio (.vsd) σε έγγραφα του Word (.doc), διευκολύνοντας την επεξεργασία και τη διανομή σύνθετων διαγραμμάτων.

**Βήμα 1: Ορισμός διαδρομών**
Ορίστε διαδρομές για το αρχείο VSD εισόδου και τον κατάλογο DOC εξόδου χρησιμοποιώντας placeholders:

```csharp
string inputVsdPath = "YOUR_DOCUMENT_DIRECTORY/sample.vsd"; // Αντικατάσταση με την πραγματική διαδρομή
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "vsd-converted-to.doc");
```

**Βήμα 2: Φόρτωση του αρχείου VSD**
Φορτώστε το αρχείο προέλευσης χρησιμοποιώντας το API GroupDocs:

```csharp
using (var converter = new Converter(inputVsdPath))
{
    // Η μετατροπή γίνεται εδώ
}
```
*Γιατί;*: Η φόρτωση του αρχείου αρχικοποιεί τη διαδικασία μετατροπής, προετοιμάζοντάς την για μετασχηματισμό.

**Βήμα 3: Διαμόρφωση επιλογών μετατροπής**
Ρύθμιση επιλογών για τη μορφή WordProcessing (.doc):

```csharp
var options = new WordProcessingConvertOptions { Format = WordProcessingFileType.Doc };
```
*Γιατί;*Ο καθορισμός της μορφής εξόδου διασφαλίζει τη συμβατότητα με τις εφαρμογές του Microsoft Word.

**Βήμα 4: Εκτελέστε τη μετατροπή**
Εκτελέστε τη μετατροπή και αποθηκεύστε το αποτέλεσμα:

```csharp
converter.Convert(outputFile, options);
```
*Γιατί;*Αυτό το βήμα μετατρέπει το αρχείο VSD σε αρχείο DOC για επεξεργασία σε οποιοδήποτε πρόγραμμα επεξεργασίας κειμένου.
