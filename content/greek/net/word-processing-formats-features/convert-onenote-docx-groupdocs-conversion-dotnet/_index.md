### Βήμα 1: Ορισμός διαδρομών αρχείων

Αρχικά, καθορίστε πού βρίσκεται το αρχείο εισόδου σας και πού θέλετε να αποθηκευτεί το έγγραφο που έχει μετατραπεί:

```csharp
string sourceFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "Sample.one");
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", "ConvertedFiles");

if (!Directory.Exists(outputFolder))
{
    Directory.CreateDirectory(outputFolder);
}

string outputFile = Path.Combine(outputFolder, "one-converted-to.docx");
```

Δημιουργήστε μια παρουσία του `Converter` τάξη χρησιμοποιώντας το αρχείο OneNote:

```csharp

{
    // Περαιτέρω βήματα θα εκτελεστούν εντός αυτού του μπλοκ.

```

Καθορίστε ότι θέλετε να μετατρέψετε σε μορφή επεξεργασίας κειμένου:
