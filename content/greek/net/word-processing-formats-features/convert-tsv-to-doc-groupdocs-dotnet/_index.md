## Οδηγός Εφαρμογής

### Επισκόπηση λειτουργιών: Μετατροπή TSV σε DOC

Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε αρχεία τιμών διαχωρισμένων με στηλοθέτες στην ευρέως χρησιμοποιούμενη μορφή εγγράφου Word, βελτιώνοντας την προσβασιμότητα και την παρουσίαση των δεδομένων.

#### Βήμα 1: Ορισμός διαδρομών εισόδου και εξόδου

Καθορίστε πού βρίσκεται το αρχείο TSV προέλευσης και πού θα πρέπει να αποθηκευτεί το αρχείο DOC εξόδου:
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

string sourceTsvFilePath = Path.Combine(documentDirectory, "sample.tsv");
string outputDocFilePath = Path.Combine(outputDirectory, "tsv-converted-to.doc");
```

#### Βήμα 2: Αρχικοποίηση της κλάσης μετατροπέα

Φορτώστε το αρχείο TSV χρησιμοποιώντας το `Converter` κλάση από το GroupDocs.Conversion:
```csharp
using (var converter = new Converter(sourceTsvFilePath))
{
    // Η διαδικασία μετατροπής θα ξεκινήσει εδώ.
}
```

#### Βήμα 3: Ορισμός επιλογών μετατροπής

Καθορίστε ότι θέλετε να μετατρέψετε το αρχείο σε μορφή DOC με αυτές τις ρυθμίσεις:
```csharp
var convertOptions = new WordProcessingConvertOptions 
{
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc
};
```

#### Βήμα 4: Εκτέλεση μετατροπής

Τέλος, εκτελέστε τη μετατροπή και αποθηκεύστε την έξοδο:
```csharp
converter.Convert(outputDocFilePath, convertOptions);
```
