## Οδηγός Εφαρμογής
Σε αυτήν την ενότητα, θα αναλύσουμε τη διαδικασία μετατροπής ενός αρχείου RTF σε TEX χρησιμοποιώντας το GroupDocs.Conversion για .NET.

### Φόρτωση και μετατροπή αρχείου RTF
#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να φορτώσετε ένα έγγραφο RTF πηγής και να το μετατρέψετε σε μορφή TEX αποτελεσματικά.

#### Βήματα:
**1. Ορισμός διαδρομών εξόδου**
Αρχικά, ορίστε τις διαδρομές του καταλόγου εξόδου:
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string inputFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.rtf");
string outputFile = Path.Combine(outputFolder, "rtf-converted-to.tex");
```
**2. Ρύθμιση επιλογών μετατροπής**
Ρυθμίστε τις παραμέτρους των επιλογών μετατροπής για να καθορίσετε τη μορφή προορισμού ως TEX:
```csharp
using (var converter = new Converter(inputFile))
{
    PageDescriptionLanguageConvertOptions options = new PageDescriptionLanguageConvertOptions 
    {
        Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Tex
    };
}
```
**3. Εκτελέστε μετατροπή**
Εκτελέστε τη μετατροπή και αποθηκεύστε την έξοδο:
```csharp
converter.Convert(outputFile, options);
Console.WriteLine("Conversion to TEX completed successfully.");
```
Αυτή η απλή διαδικασία φορτώνει το αρχείο RTF σας και το μετατρέπει σε έγγραφο TEX, διασφαλίζοντας ότι αποθηκεύεται στον καθορισμένο κατάλογο.
