Δείτε πώς μπορείτε να αρχικοποιήσετε και να ρυθμίσετε το GroupDocs.Conversion στο έργο σας C#:
```csharp
using GroupDocs.Conversion;

// Αρχικοποιήστε την κλάση Converter με τη διαδρομή αρχείου EPUB πηγής
class DocumentConverter
{
    public void ConvertEpubToDocx(string epubFilePath, string outputFolder)
    {
        // Βεβαιωθείτε ότι υπάρχει ο κατάλογος εξόδου

        var outputFile = Path.Combine(outputFolder, "epub-converted-to.docx");

        using (var converter = new Converter(epubFilePath))
        {
            var options = new WordProcessingConvertOptions();
            converter.Convert(outputFile, options);
        }
    }
}
```

Ξεκινήστε καθορίζοντας πού θα βρίσκονται τα αρχεία προέλευσης και εξόδου:
```csharp

string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
```

Φορτώστε το αρχείο EPUB χρησιμοποιώντας το `Converter` τάξη:

#### Βήμα 3: Ορισμός επιλογών μετατροπής

Καθορίστε ότι θέλετε να μετατρέψετε σε μορφή επεξεργασίας κειμένου (DOCX):
