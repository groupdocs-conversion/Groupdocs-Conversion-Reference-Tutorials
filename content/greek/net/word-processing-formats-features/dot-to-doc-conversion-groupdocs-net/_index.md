## Οδηγός Εφαρμογής
Ας δούμε πώς γίνεται η μετατροπή ενός αρχείου DOT σε μορφή DOC χρησιμοποιώντας το GroupDocs.Conversion. Θα το αναλύσουμε σε διαχειρίσιμα βήματα, καθένα από τα οποία εστιάζει σε συγκεκριμένες λειτουργίες και διαμορφώσεις.

Καθορίστε πού βρίσκεται το αρχείο DOT προέλευσης και πού θέλετε να αποθηκευτεί το αρχείο DOC που έχει μετατραπεί.
```csharp
string inputFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.dot");

```
### Βήμα 3: Φόρτωση και μετατροπή του αρχείου DOT
Τώρα, θα φορτώσουμε το αρχείο DOT πηγής και θα το μετατρέψουμε σε μορφή DOC χρησιμοποιώντας το GroupDocs.Conversion.
```csharp
using (var converter = new Converter(inputFile))
{
    // Ρύθμιση παραμέτρων επιλογών μετατροπής για μορφές επεξεργασίας Word
    var options = new WordProcessingConvertOptions
    {
        Format = WordProcessingFileType.Doc // Ορισμός μορφής στόχου σε DOC
    };

    // Μετατροπή και αποθήκευση του αρχείου ως εγγράφου DOC
    converter.Convert(outputFile, options);
}
```
**Βασικά σημεία:**
- `Converter` Η κλάση φορτώνει το αρχείο DOT σας.
- `WordProcessingConvertOptions` καθορίζει ρυθμίσεις μετατροπής όπως η μορφή εξόδου.
- `converter.Convert()` εκτελεί την πραγματική μετατροπή.
