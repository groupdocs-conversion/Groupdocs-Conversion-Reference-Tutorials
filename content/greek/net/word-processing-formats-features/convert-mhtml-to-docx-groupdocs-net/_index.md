Η μετατροπή ενός αρχείου MHTML σε μορφή DOCX περιλαμβάνει τη φόρτωση του πηγαίου κώδικα, τη διαμόρφωση των επιλογών και την εκτέλεση του μετασχηματισμού. Ακολουθήστε τα παρακάτω βήματα:

**Βήμα 1**: Ορισμός καταλόγου εξόδου
Ρυθμίστε τον κατάλογο εξόδου για την αποθήκευση των αρχείων που έχουν μετατραπεί.

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
```

Προσδιορίστε και φορτώστε το αρχείο MHTML που θέλετε να μετατρέψετε. Αντικαταστήστε `YOUR_DOCUMENT_DIRECTORY` με τη διαδρομή του φακέλου εγγράφων σας.

```csharp
var sampleMhtmlPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY\sample.mhtml");
using (Converter converter = new Converter(sampleMhtmlPath))
{
    // Τα βήματα μετατροπής θα οριστούν εδώ
}
```

**Βήμα 3**: Ρύθμιση παραμέτρων επιλογών μετατροπής DOCX
Καθορίστε τη μορφή εξόδου και πρόσθετες ρυθμίσεις για τη μετατροπή.

```csharp

converter.Convert(outputFile, convertOptions);
```

- **Μετατροπέας**: Αντιπροσωπεύει την παρουσία του μετατροπέα αρχείων.
- **DocxConvertOptions**Επιτρέπει την προσαρμογή των ρυθμίσεων εξόδου DOCX.
