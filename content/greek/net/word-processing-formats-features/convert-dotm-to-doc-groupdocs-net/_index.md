## Οδηγός Εφαρμογής

Τώρα που καλύψαμε τη ρύθμιση, ας προχωρήσουμε στην υλοποίηση της διαδικασίας μετατροπής.

### Φόρτωση και μετατροπή αρχείων DOTM

Η βασική λειτουργικότητα περιλαμβάνει τη φόρτωση ενός αρχείου .dotm και τη μετατροπή του σε μορφή DOC. Ας αναλύσουμε κάθε βήμα:

#### Βήμα 1: Ορισμός διαδρομών καταλόγου
Καθορίστε τους καταλόγους εισόδου και εξόδου. Αντικαταστήστε τα placeholders με πραγματικές διαδρομές.
```csharp
const string YOUR_DOCUMENT_DIRECTORY = "YOUR_DOCUMENT_DIRECTORY";
const string YOUR_OUTPUT_DIRECTORY = "YOUR_OUTPUT_DIRECTORY";

// Φορτώστε το αρχείο προέλευσης DOTM που βρίσκεται στο YOUR_DOCUMENT_DIRECTORY.
string dotmFilePath = Path.Combine(YOUR_DOCUMENT_DIRECTORY, "sample.dotm");
```

#### Βήμα 2: Προετοιμασία διαδρομής αρχείου εξόδου
Ορίστε πού θέλετε να αποθηκεύσετε το αρχείο DOC που έχετε μετατρέψει.
```csharp
// Προετοιμάστε τη διαδρομή εξόδου για το αρχείο DOC που έχει μετατραπεί.
string outputFile = Path.Combine(YOUR_OUTPUT_DIRECTORY, "dotm-converted-to.doc");
```

#### Βήμα 3: Μετατροπή χρησιμοποιώντας το GroupDocs.Conversion
Δείτε πώς μπορείτε να εκτελέσετε τη μετατροπή χρησιμοποιώντας τη βιβλιοθήκη:
```csharp
using (var converter = new GroupDocs.Conversion.Converter(dotmFilePath))
{
    // Ρυθμίστε τις παραμέτρους των επιλογών μετατροπής καθορίζοντας τη μορφή προορισμού ως DOC.
    WordProcessingConvertOptions options = new WordProcessingConvertOptions { Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc };

    // Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο DOC εξόδου στην καθορισμένη διαδρομή.
    converter.Convert(outputFile, options);
}
```

- **Παράμετροι & Σκοπός μεθόδου:** Ο `Converter` η κλάση φορτώνει το έγγραφό σας, ενώ `WordProcessingConvertOptions` καθορίζει τη μορφή στόχου. Το `Convert` Η μέθοδος εκτελεί τη διαδικασία και αποθηκεύει το αποτέλεσμα.
