## Οδηγός Εφαρμογής

### Φόρτωση και μετατροπή αρχείου IFC σε DOCX

Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε ένα αρχείο Industry Foundation Classes (IFC) σε ένα έγγραφο Microsoft Word Open XML (.docx).

#### Βήμα 1: Ρύθμιση του περιβάλλοντος σας

Βεβαιωθείτε ότι ο κατάλογος του έργου σας έχει ρυθμιστεί σωστά:
```csharp
string outputFolder = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "output");
string outputFile = Path.Combine(outputFolder, "ifc-converted-to.docx");
```
Αντικαθιστώ `"YOUR_DOCUMENT_DIRECTORY"` με τη διαδρομή όπου βρίσκεται το αρχείο IFC σας.

#### Βήμα 2: Αρχικοποίηση του μετατροπέα

Φορτώστε το αρχείο IFC χρησιμοποιώντας το `Converter` τάξη:
```csharp
using (var converter = new Converter(Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.ifc")))
{
    // Πρόσθετος κώδικας θα ακολουθήσει στα επόμενα βήματα.
}
```
Αντικαθιστώ `"sample.ifc"` με το πραγματικό όνομα αρχείου IFC σας.

#### Βήμα 3: Διαμόρφωση επιλογών μετατροπής

Ρύθμιση επιλογών για μετατροπή σε μορφή DOCX:
```csharp
var options = new WordProcessingConvertOptions();
```
Αυτό το αντικείμενο καθορίζει ότι στοχεύετε σε μια μετατροπή εγγράφου του Word.

#### Βήμα 4: Εκτελέστε τη μετατροπή

Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο εξόδου:
```csharp
converter.Convert(outputFile, options);
```

### Επεξήγηση Κώδικα

- **`Converter` Τάξη:** Διαχειρίζεται τις διαδικασίες φόρτωσης και μετατροπής.
- **`WordProcessingConvertOptions`:** Ρυθμίζει τη μορφή εξόδου ως DOCX.
- **`converter.Convert()`:** Εκτελεί μετατροπή με βάση καθορισμένες παραμέτρους.

### Συμβουλές αντιμετώπισης προβλημάτων

- Βεβαιωθείτε ότι η διαδρομή του αρχείου IFC είναι σωστή για να αποφύγετε `FileNotFoundException`.
- Ελέγξτε για επαρκή δικαιώματα στον κατάλογο όπου αποθηκεύετε τα αρχεία που έχουν μετατραπεί.
