### Μετατροπή RTF σε DOC

Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε ένα αρχείο RTF σε μορφή DOC χρησιμοποιώντας το GroupDocs.Conversion για .NET. 

#### Βήμα 1: Ρύθμιση του έργου σας και φόρτωση του αρχείου προέλευσης

Βεβαιωθείτε ότι το έργο σας έχει ρυθμιστεί με τις απαραίτητες εξαρτήσεις και φορτώστε το αρχείο RTF πηγής:

```csharp
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY\\sample.rtf"))
{
    // Οι επιλογές μετατροπής θα οριστούν στο επόμενο βήμα.
}
```

#### Βήμα 2: Ορισμός επιλογών μετατροπής

Στη συνέχεια, καθορίστε τις ρυθμίσεις μετατροπής χρησιμοποιώντας `WordProcessingConvertOptions` για να στοχεύσετε τη μορφή DOC:

```csharp
WordProcessingConvertOptions options = new WordProcessingConvertOptions 
{ 
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Doc 
};
```

#### Βήμα 3: Εκτελέστε τη μετατροπή

Τέλος, εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο εξόδου:

```csharp
string outputFile = Path.Combine(outputFolder, "rtf-converted-to.doc");
converter.Convert(outputFile, options);
```

### Ορισμός διαδρομής καταλόγου εξόδου

Η σωστή διαχείριση των διαδρομών εξόδου είναι ζωτικής σημασίας για την οργάνωση των αρχείων που έχουν μετατραπεί. Αυτή η μέθοδος διασφαλίζει ότι ο καθορισμένος κατάλογος υπάρχει πριν από την αποθήκευση των εγγράφων που έχουν μετατραπεί:

```csharp
string GetOutputDirectoryPath()
{
    string outputFolder = "YOUR_OUTPUT_DIRECTORY";
    if (!Directory.Exists(outputFolder))
        Directory.CreateDirectory(outputFolder);
    return outputFolder;
}
```
