## Οδηγός Εφαρμογής

Ας δούμε τη διαδικασία μετατροπής του αρχείου CSV σε HTML χρησιμοποιώντας το GroupDocs.Conversion για .NET.

### Αρχικοποίηση και Ρύθμιση

Ξεκινήστε αρχικοποιώντας τη βιβλιοθήκη μετατροπών. Ακολουθεί μια απλή ρύθμιση σε C#:

```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main(string[] args)
    {
        string sourceCsvPath = @"YOUR_DOCUMENT_DIRECTORY\sample.csv";
        string outputPath = Path.Combine(@"YOUR_OUTPUT_DIRECTORY", "output.html");

        // Αρχικοποιήστε τον μετατροπέα με τη διαδρομή του αρχείου CSV
        using (Converter converter = new Converter(sourceCsvPath))
        {
            var options = new MarkupConvertOptions(); // Επιλογές για μετατροπή HTML

            // Μετατρέψτε και αποθηκεύστε την έξοδο στην καθορισμένη διαδρομή
            converter.Convert(outputPath, options);
        }
    }
}
```

**Εξήγηση:**
- **Μετατροπέας:** Αυτή η κλάση χειρίζεται τη μετατροπή αρχείων.
- **Επιλογές MarkupConvert:** Ρυθμίζει τις παραμέτρους ειδικά για τη μετατροπή αρχείων σε μορφή HTML.

### Βασικές επιλογές διαμόρφωσης
