**Εγκατάσταση**

Έχετε δύο κύριες μεθόδους για να εγκαταστήσετε το GroupDocs.Conversion: χρησιμοποιώντας το **Κονσόλα διαχείρισης πακέτων NuGet** ή το **.NET CLI**.

**Βασική Αρχικοποίηση και Ρύθμιση**

Μόλις εγκατασταθεί, αρχικοποιήστε το GroupDocs.Conversion στην εφαρμογή C# που χρησιμοποιείτε. Παρακάτω είναι ένα παράδειγμα ρύθμισης μιας απλής διαδικασίας μετατροπής:
```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main(string[] args)
    {
        // Αρχικοποιήστε το αντικείμενο Converter με τη διαδρομή αρχείου VTX εισόδου
        using (Converter converter = new Converter("your-input-file.vtx"))
        {
            var options = new CsvConvertOptions();
            
            // Μετατρέψτε και αποθηκεύστε το αρχείο CSV εξόδου
            converter.Convert("output-file.csv", options);
        }
    }
}
```
