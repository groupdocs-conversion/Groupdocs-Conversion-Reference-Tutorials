**Επισκόπηση:** Αυτό το βήμα περιλαμβάνει τη φόρτωση του αρχείου EML που θέλετε να μετατρέψετε χρησιμοποιώντας το GroupDocs.Conversion για .NET.

Δημιουργήστε ένα `Converter` αντικείμενο με την καθορισμένη διαδρομή. Αυτό θα χειριστεί τη φόρτωση του αρχείου:

```csharp

{
    // Το αρχείο είναι πλέον έτοιμο για λειτουργίες μετατροπής.
```

### Μετατροπή σε μορφή CSV

**Επισκόπηση:** Εδώ, θα μετατρέψετε το φορτωμένο αρχείο EML σε μορφή CSV.

#### Βήμα 1: Ορισμός διαδρομής εξόδου και επιλογών

Ορίστε πού θα αποθηκευτεί το αρχείο που έχει μετατραπεί και ορίστε τις επιλογές μετατροπής:

```csharp
using System.IO;
using GroupDocs.Conversion.Options.Convert;

string outputDirectory = @"YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputDirectory, "eml-converted-to.csv");

// Καθορίστε επιλογές μετατροπής για τη μορφή CSV
SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = GroupDocs.Conversion.FileTypes.SpreadsheetFileType.Csv };
```

#### Βήμα 2: Εκτελέστε τη μετατροπή

Εκτελέστε τη μετατροπή χρησιμοποιώντας το `Converter` αντικείμενο:

```csharp
converter.Convert(outputFile, options);
```

**Συμβουλή αντιμετώπισης προβλημάτων:** Βεβαιωθείτε ότι η διαδρομή του καταλόγου εξόδου είναι σωστή και εγγράψιμη.
