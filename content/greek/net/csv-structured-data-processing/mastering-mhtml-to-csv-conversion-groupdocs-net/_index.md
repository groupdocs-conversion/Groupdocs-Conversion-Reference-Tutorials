### Μετατροπή MHTML σε CSV

#### Επισκόπηση

Η μετατροπή ενός αρχείου MHTML σε μορφή CSV περιλαμβάνει τον ορισμό συγκεκριμένων επιλογών μετατροπής και την εκτέλεση του μετασχηματισμού.

#### Βήματα για την εφαρμογή

**1. Ορισμός καταλόγου εξόδου και διαδρομής αρχείου**

Προσδιορίστε πού θα πρέπει να αποθηκευτεί το αρχείο CSV που έχει μετατραπεί:
```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "mhtml-converted-to.csv");
```

**2. Φόρτωση αρχείου MHTML πηγής και μετατροπή**

Προετοιμάστε το έγγραφο MHTML για μετατροπή:
```csharp
using (var converter = new Converter(Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.mhtml")))
{
    // Ορισμός επιλογών μετατροπής σε μορφή CSV
    SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Csv };

    // Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο εξόδου
    converter.Convert(outputFile, options);
}
```

#### Επεξήγηση παραμέτρων

- `SpreadsheetConvertOptions`: Διαμορφώνει τις ρυθμίσεις για τη μετατροπή αρχείων σε μορφές υπολογιστικών φύλλων.
- `Format`: Καθορίζει τη μορφή προορισμού (CSV σε αυτήν την περίπτωση).
