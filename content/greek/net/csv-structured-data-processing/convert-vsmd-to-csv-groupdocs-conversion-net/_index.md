### Βασική Αρχικοποίηση και Ρύθμιση

Δείτε πώς μπορείτε να αρχικοποιήσετε το GroupDocs.Conversion στην εφαρμογή C# που χρησιμοποιείτε:

```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

namespace VSDMtoCSVConverter
{
class Program
{
    static void Main(string[] args)
    {
        // Ορισμός άδειας χρήσης, εάν είναι διαθέσιμη
        // Αδεια χρήσης = νέα άδεια χρήσης();
        // lic.SetLicense("διαδρομή/προς/license.lic");

        string inputFilePath = @"YOUR_DOCUMENT_DIRECTORY\\example.vsdm";
        string outputDirectory = @"YOUR_OUTPUT_DIRECTORY";

        using (var converter = new Converter(inputFilePath))
        {
            var options = new CsvConvertOptions();
            
            converter.Convert(Path.Combine(outputDirectory, "output.csv"), options);
        }
    }
}
}
```

Αυτό το απόσπασμα κώδικα αρχικοποιεί τη διαδικασία μετατροπής και ρυθμίζει τις διαδρομές εισόδου και εξόδου.
