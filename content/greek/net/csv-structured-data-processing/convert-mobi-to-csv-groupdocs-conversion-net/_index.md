## Οδηγός Εφαρμογής

### Μετατροπή MOBI σε CSV
Αυτή η λειτουργία σάς επιτρέπει να εξάγετε δεδομένα κειμένου από αρχεία MOBI και να τα αποθηκεύετε ως CSV για περαιτέρω επεξεργασία ή ανάλυση.

#### Επισκόπηση
Η μετατροπή MOBI σε CSV είναι χρήσιμη όταν πρόκειται για μεγάλους όγκους ηλεκτρονικών βιβλίων που απαιτούν εξαγωγή και χειρισμό περιεχομένου.

#### Βήματα Υλοποίησης

**Βήμα 1: Φόρτωση του αρχείου MOBI**
Φορτώστε το αρχείο MOBI χρησιμοποιώντας το `Converter` τάξη.
```csharp
using (Converter converter = new Converter(mobiFilePath))
{
    // Προχωρήστε στις επιλογές μετατροπής
}
```

**Βήμα 2: Ορισμός επιλογών μετατροπής**
Ορίστε ρυθμίσεις για μετατροπή σε μορφή CSV.
```csharp
var convertOptions = new CsvConvertOptions();
// Προσαρμόστε ρυθμίσεις όπως οριοθέτη και κωδικοποίηση όπως απαιτείται
```
- **Σκοπός**Αυτή η μέθοδος ορίζει τον τρόπο μορφοποίησης του κειμένου από το αρχείο MOBI στο CSV εξόδου.

**Βήμα 3: Εκτέλεση μετατροπής**
Εκτελέστε μετατροπή καθορίζοντας τη διαδρομή εξόδου για το αρχείο CSV.
```csharp
string csvOutputPath = @"C:\Path\To\Your\OutputFile.csv";
converter.Convert(csvOutputPath, convertOptions);
```
- **Σκοπός**: Αυτό το βήμα ολοκληρώνει τη μετατροπή και αποθηκεύει το αρχείο CSV εξόδου.

#### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι τα αρχεία MOBI δεν είναι κατεστραμμένα ή κλειδωμένα από άλλες διεργασίες.
- Επαληθεύστε τις διαδρομές αρχείων για να αποφύγετε σφάλματα που σχετίζονται με τις διαδρομές.
- Βελτιστοποιήστε τη χρήση μνήμης εάν προκύψουν προβλήματα απόδοσης.
