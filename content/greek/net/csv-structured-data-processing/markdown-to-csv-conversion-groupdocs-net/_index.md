## Οδηγός Εφαρμογής
Τώρα, ας αναλύσουμε τη διαδικασία μετατροπής βήμα προς βήμα.
### Φόρτωση Markdown και μετατροπή σε CSV
#### Βήμα 1: Ορισμός καταλόγων
Καθορίστε πού βρίσκεται το αρχείο markdown και πού θέλετε να αποθηκευτεί το CSV που έχει μετατραπεί:
```csharp
string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY/";
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY/";
```
#### Βήμα 2: Φόρτωση του αρχείου Markdown
Χρησιμοποιήστε το GroupDocs.Conversion για να φορτώσετε το `.md` αρχείο. Αυτό το βήμα ξεκινά τη διαδικασία μετατροπής.
```csharp
using (var converter = new GroupDocs.Conversion.Converter(Path.Combine(documentDirectory, "sample.md")))
{
    // Η λογική μετατροπής θα προστεθεί εδώ.
}
```
#### Βήμα 3: Ρύθμιση παραμέτρων επιλογών CSV
Ορίστε επιλογές για να καθορίσετε ότι θέλετε η έξοδος να είναι σε μορφή CSV:
```csharp
var options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Csv };
```
#### Βήμα 4: Εκτέλεση μετατροπής και αποθήκευση εξόδου
Εκτελέστε τη μετατροπή καλώντας `Convert`καθορίζοντας τόσο τη διαδρομή του αρχείου εξόδου όσο και τις επιλογές.
```csharp
string outputFile = Path.Combine(outputDirectory, "md-converted-to.csv");
converter.Convert(outputFile, options);
```
