## Οδηγός Εφαρμογής
Αυτή η ενότητα σας καθοδηγεί στη φόρτωση ενός αρχείου MBOX και στη μετατροπή του σε μορφή CSV. Θα το αναλύσουμε ανά λειτουργία για λόγους σαφήνειας.

### Χαρακτηριστικό 1: Φόρτωση αρχείου MBOX
#### Επισκόπηση
Η φόρτωση ενός αρχείου MBOX είναι το πρώτο βήμα στη διαδικασία μετατροπής μας. Το GroupDocs.Conversion παρέχει έναν απλό τρόπο για να το χειριστείτε αυτό χρησιμοποιώντας `MboxLoadOptions`.

#### Βήματα Υλοποίησης
**Βήμα 1**Ορίστε τη διαδρομή του αρχείου MBOX και τις επιλογές φόρτωσης:
```csharp
string mboxFilePath = @"YOUR_DOCUMENT_DIRECTORY\\sample.mbox";
MboxLoadOptions loadOptions = new MboxLoadOptions();
```

**Βήμα 2**Δημιουργήστε μια παρουσία Converter για να χειριστείτε τη διαδικασία μετατροπής:
```csharp
var converter = new Converter(mboxFilePath, (LoadContext context) => {
    return context.SourceFormat == EmailFileType.Mbox ? loadOptions : null;
});
```
*Γιατί αυτό το βήμα;* Αυτό διασφαλίζει ότι μόνο τα αρχεία MBOX υποβάλλονται σε επεξεργασία με τις καθορισμένες επιλογές.

### Λειτουργία 2: Μετατροπή MBOX σε CSV
#### Επισκόπηση
Μόλις φορτωθεί το αρχείο MBOX, μπορούμε να το μετατρέψουμε σε μορφή CSV. Η διαδικασία μετατροπής χρησιμοποιεί `SpreadsheetConvertOptions`.

#### Βήματα Υλοποίησης
**Βήμα 1**: Ρυθμίστε τον κατάλογο εξόδου και το πρότυπο:
```csharp
string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "mbox-converted-{0}-to.csv");
```

**Βήμα 2**Ορισμός επιλογών μετατροπής για τη μορφή CSV:
```csharp
SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Csv };
int counter = 1;
```

**Βήμα 3**Εκτελέστε τη μετατροπή και αποθηκεύστε κάθε αρχείο ως CSV:
```csharp
counter = 1; // Επαναφορά για λόγους σαφήνειας στην εξήγηση
converter.Convert(
    (SaveContext saveContext) => new FileStream(string.Format(outputFileTemplate, counter++), FileMode.Create),
    options
);
```
*Γιατί αυτό το βήμα;* Αυτό χειρίζεται την πραγματική μετατροπή και διασφαλίζει ότι κάθε email αποθηκεύεται σε ένα νέο αρχείο CSV.
