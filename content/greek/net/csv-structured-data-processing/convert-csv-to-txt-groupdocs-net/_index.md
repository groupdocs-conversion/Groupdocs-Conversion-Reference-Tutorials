## Οδηγός Εφαρμογής

Ακολουθήστε αυτά τα βήματα για να μετατρέψετε ένα αρχείο CSV σε μορφή TXT.

### Χαρακτηριστικό: Μετατροπή CSV σε TXT

Αυτή η λειτουργία επιτρέπει την απρόσκοπτη μετατροπή οποιουδήποτε αρχείου CSV σε απλό αρχείο κειμένου χρησιμοποιώντας το GroupDocs.Conversion.

#### Βήμα 1: Προετοιμασία των διαδρομών αρχείων σας

Ορίστε τους καταλόγους για τα αρχεία CSV εισόδου και TXT εξόδου:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

string inputFile = Path.Combine(documentDirectory, "sample.csv");
string outputFile = Path.Combine(outputDirectory, "csv-converted-to.txt");

// Δημιουργήστε τον κατάλογο εξόδου εάν δεν υπάρχει
if (!Directory.Exists(outputDirectory))
{
    Directory.CreateDirectory(outputDirectory);
}
```

#### Βήμα 2: Φόρτωση και μετατροπή χρησιμοποιώντας το GroupDocs.Conversion

Φορτώστε το αρχείο CSV και ρυθμίστε τις επιλογές μετατροπής για τη μορφή TXT:

```csharp
using (var converter = new Converter(inputFile))
{
    // Ορισμός επιλογών μετατροπής για τη μορφή TXT
    var options = new WordProcessingConvertOptions { Format = FileTypes.WordProcessingFileType.Txt };
    
    // Μετατρέψτε το αρχείο και αποθηκεύστε το ως έγγραφο TXT
    converter.Convert(outputFile, options);
}
```

### Επεξήγηση Βασικών Βημάτων

- **Διαδρομές εισόδου και εξόδου:** Βεβαιωθείτε ότι οι διαδρομές σας είναι σωστές για να αποφύγετε σφάλματα.
- **Δημιουργία καταλόγου:** Ο κώδικας ελέγχει εάν ο κατάλογος εξόδου υπάρχει πριν τον δημιουργήσει για να αποτρέψει εξαιρέσεις.
- **Επιλογές μετατροπής:** `WordProcessingConvertOptions` έχει ρυθμιστεί για TXT, εξασφαλίζοντας μια ομαλή διαδικασία μετατροπής.
