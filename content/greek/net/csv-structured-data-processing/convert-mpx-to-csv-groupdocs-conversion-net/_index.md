Το GroupDocs προσφέρει διαφορετικές επιλογές αδειοδότησης, όπως:
- **Δωρεάν δοκιμή:** Αποκτήστε πρόσβαση σε βασικές λειτουργίες για να αξιολογήσετε τη βιβλιοθήκη.
- **Προσωρινή Άδεια:** Υποβάλετε αίτηση για προσωρινή άδεια χρήσης για την άρση των περιορισμών της δοκιμαστικής περιόδου.
- **Αγορά:** Αποκτήστε μόνιμη άδεια χρήσης για πλήρη πρόσβαση και υποστήριξη.

Για να αρχικοποιήσετε το GroupDocs.Conversion στο έργο σας, ακολουθήστε τα εξής βήματα:

```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main(string[] args)
    {
        // Αρχικοποίηση του χειριστή μετατροπών με μια προσωρινή ή αγορασμένη άδεια χρήσης
        Converter converter = new Converter("your-mpx-file.mpx");
        
        // Ορισμός επιλογών μετατροπής για μορφή CSV
        var convertOptions = new CsvConvertOptions();
        
        // Μετατρέψτε και αποθηκεύστε το αρχείο MPX σε CSV
        converter.Convert("output.csv", convertOptions);
    }
}
```

#### Ρύθμιση επιλογών μετατροπής


### 3. Εκτέλεση της Διαδικασίας Μετατροπής

Αφού φορτωθεί το αρχείο MPX και οριστούν οι επιλογές μετατροπής, προχωρήστε στην εκτέλεση της μετατροπής.

#### Μετατροπή και αποθήκευση εξόδου
