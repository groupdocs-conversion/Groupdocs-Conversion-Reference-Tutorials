## Οδηγός Εφαρμογής

Τώρα που το περιβάλλον σας έχει ρυθμιστεί, ας μετατρέψουμε τα αρχεία CSV σε JSON.

### Χαρακτηριστικό: Μετατροπή CSV σε JSON
Αυτή η λειτουργία επιτρέπει την αποτελεσματική μετατροπή δεδομένων CSV σε δομημένη μορφή JSON. Ακολουθήστε τα παρακάτω βήματα:

#### Βήμα 1: Ορισμός διαδρομών καταλόγου και ονομάτων αρχείων
Καθορίστε πού θα βρίσκονται τα αρχεία εισόδου και εξόδου σας για αποτελεσματική διαχείριση της διαδρομής αρχείων στον κώδικά σας.

```csharp
// Ορίστε τις διαδρομές καταλόγων για τα αρχεία εισόδου και εξόδου
cstring documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
cstring outputDirectory = "YOUR_OUTPUT_DIRECTORY/";

// Ορίστε τα ονόματα των αρχείων
cstring inputCsvFile = Path.Combine(documentDirectory, "sample.csv");
cstring outputFile = Path.Combine(outputDirectory, "converted.json");
```

#### Βήμα 2: Αρχικοποίηση επιλογών φόρτωσης CSV
Ρυθμίστε τις επιλογές φόρτωσης για να καθορίσετε το διαχωριστικό που χρησιμοποιείται στο CSV σας (σε αυτό το παράδειγμα, κόμμα).

```csharp
// Αρχικοποίηση επιλογών φόρτωσης CSV με ένα καθορισμένο διαχωριστικό
var loadOptions = new CsvLoadOptions
{
    Separator = ','
};
```

#### Βήμα 3: Δημιουργήστε μια παρουσία της κλάσης Converter
Χρησιμοποιώντας το αρχείο εισόδου και τις επιλογές φόρτωσης, δημιουργήστε ένα αντίγραφο του `Converter` κλάση για να ρυθμίσετε τη λογική μετατροπής σας.

```csharp
// Δημιουργήστε μια παρουσία της κλάσης Converter με ένα περιβάλλον φόρτωσης
using (Converter converter = new Converter(inputCsvFile, (LoadContext loadContext) => loadOptions))
{
    // Βήμα 4: Ορισμός επιλογών μετατροπής για τη μορφή JSON
    WebConvertOptions convertOptions = new WebConvertOptions
    {
        Format = WebFileType.Json
    };

    // Μετατρέψτε το CSV σε JSON και αποθηκεύστε το αρχείο εξόδου
    converter.Convert(outputFile, convertOptions);
}
```

### Επεξήγηση παραμέτρων κώδικα
- **`CsvLoadOptions`**: Ρυθμίζει τον τρόπο ανάγνωσης των δεδομένων CSV. Το διαχωριστικό ορίζει τις διαιρέσεις πεδίων.
- **`Converter` Τάξη**: Χειρίζεται κεντρικά τις λειτουργίες μετατροπής.
- **`WebConvertOptions`**: Υπαγορεύει τη μορφή εξόδου, JSON σε αυτήν την περίπτωση.

### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι οι διαδρομές αρχείων είναι σωστές και προσβάσιμες από την εφαρμογή σας.
- Επαληθεύστε την ακεραιότητα των δεδομένων CSV για να αποτρέψετε τυχόν λανθασμένες εξόδους JSON.
- Ελέγξτε για τυχόν εξαιρέσεις κατά την εκτέλεση για να διαγνώσετε προβλήματα εγκατάστασης.
