## Βασική Αρχικοποίηση και Ρύθμιση
Δείτε πώς μπορείτε να αρχικοποιήσετε τον μετατροπέα με βασική ρύθμιση:

```csharp
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

string sourceFilePath = "YOUR_DOCUMENT_DIRECTORY/SAMPLE_EML";
string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "converted.pdf");

// Φόρτωση επιλογών για μετατροπή email
Func<LoadContext, LoadOptions> getLoadOptions = loadContext => new EmailLoadOptions { ConvertOwned = false };

using (Converter converter = new Converter(sourceFilePath, getLoadOptions))
{
    PdfConvertOptions options = new PdfConvertOptions();
    converter.Convert(outputFile, options);
}
```

## Οδηγός Εφαρμογής
Σε αυτήν την ενότητα, θα αναλύσουμε κάθε βήμα της μετατροπής ενός αρχείου email σε PDF.

### Φόρτωση αρχείου email με συγκεκριμένες επιλογές
**Επισκόπηση:**
Η ρύθμιση των επιλογών φόρτωσης σάς επιτρέπει να ελέγχετε τον τρόπο με τον οποίο η διαδικασία μετατροπής χειρίζεται τα αρχεία email σας. Εδώ καθορίζετε προτιμήσεις, όπως το αν θα μετατρέπονται οι ιδιόκτητες ιδιότητες.

```csharp
Func<LoadContext, LoadOptions> getLoadOptions = loadContext => new EmailLoadOptions
{
    ConvertOwned = false // Μην μετατρέπετε τις ιδιόκτητες ιδιοκτησίες από προεπιλογή
};
```
**Εξήγηση:**
- `ConvertOwned`Όταν οριστεί σε false, διασφαλίζει τη μετατροπή των τυπικών χαρακτηριστικών email χωρίς να μετασχηματίζει τυχόν ιδιόκτητες μορφές.

### Αρχικοποίηση μετατροπέα και ορισμός επιλογών μετατροπής
**Επισκόπηση:**
Η κύρια εργασία γίνεται εδώ. Αρχικοποιείτε το `Converter` κλάση με τη διαδρομή του αρχείου πηγαίου κώδικα και τις επιλογές φόρτωσης.

```csharp
using (Converter converter = new Converter(sourceFilePath, getLoadOptions))
{
    PdfConvertOptions options = new PdfConvertOptions();
    converter.Convert(outputFile, options);
}
```
**Εξήγηση:**
- **Παράμετροι**: Το `sourceFilePath` καθορίζει το αρχείο email που θα μετατραπεί και `getLoadOptions` παρέχει τις ρυθμίσεις μετατροπής.
- **Επιστρεφόμενη τιμή**Αυτή η λειτουργία επιστρέφει ένα αρχείο PDF που βρίσκεται στη διεύθυνση `outputFile`.

### Βασικές επιλογές διαμόρφωσης
Διαμόρφωση του `PdfConvertOptions` σας επιτρέπει να προσαρμόσετε την έξοδο. Μπορείτε να καθορίσετε το μέγεθος σελίδας, τα περιθώρια και άλλα με βάση τις απαιτήσεις σας.
