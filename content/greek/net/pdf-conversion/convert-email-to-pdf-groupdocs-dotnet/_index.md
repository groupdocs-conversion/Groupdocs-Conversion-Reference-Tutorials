## Οδηγός Εφαρμογής

Ας αναλύσουμε την υλοποίηση σε λογικά τμήματα, εστιάζοντας στη μετατροπή ενός email μαζί με τα συνημμένα του.

### Ρύθμιση παραμέτρων επιλογών φόρτωσης

Αρχικά, διαμορφώστε τις επιλογές φόρτωσης για να καθορίσετε τον τρόπο χειρισμού των εγγράφων email σας κατά τη μετατροπή. Αυτό περιλαμβάνει τον ορισμό ιδιοτήτων όπως `ConvertOwner` και `ConvertOwned`.

```csharp
Func<LoadContext, LoadOptions> getLoadOptions = loadContext => new EmailLoadOptions
{
    ConvertOwner = true,
    ConvertOwned = true,
    Depth = 2 // Περιλαμβάνει συνημμένα στη διαδικασία μετατροπής
};
```

### Αρχικοποίηση του μετατροπέα

Στη συνέχεια, αρχικοποιήστε το `Converter` κλάση με το έγγραφο email σας και τις επιλογές φόρτωσης που ορίστηκαν προηγουμένως.

```csharp
using (Converter converter = new Converter(inputFilePath, getLoadOptions))
{
    int index = 1; // Ευρετήριο για την ονομασία αρχείων εξόδου
    
    PdfConvertOptions options = new PdfConvertOptions(); // Ορισμός επιλογών μετατροπής σε PDF
    
    // Ορίστε μια συνάρτηση επανάκλησης για την αποθήκευση κάθε εγγράφου ή συνημμένου που έχει μετατραπεί
    converter.Convert((SaveContext saveContext) =>
    {
        string fileName = index == 1 ? "converted.pdf" : $"converted-attachment-{index - 1}.pdf";
        index++;
        string outputFile = Path.Combine(outputFolder, fileName); // Κατασκευή πλήρους διαδρομής εξόδου
        return new FileStream(outputFile, FileMode.Create); // Δημιουργία ροής αρχείων για κάθε έγγραφο που έχει μετατραπεί
    }, options);
}
```

#### Εξήγηση:
- **Επιλογές Φόρτωσης**: Ελέγχει τον τρόπο επεξεργασίας του email και των συνημμένων του.
- **Κλάση μετατροπέα**: Διαχειρίζεται τη διαδικασία μετατροπής από είσοδο σε PDF.
- **Επιλογές Μετατροπής Pdf**Καθορίζει ότι η μορφή εξόδου θα πρέπει να είναι PDF.
- **Επανάκληση SaveContext**: Χειρίζεται την ονομασία και την αποθήκευση αρχείων για κάθε έγγραφο ή συνημμένο που έχει μετατραπεί.

### Συμβουλές αντιμετώπισης προβλημάτων
Βεβαιωθείτε ότι όλες οι διαδρομές είναι `inputFilePath` και `outputFolder` έχουν ρυθμιστεί σωστά. Βεβαιωθείτε ότι η παράμετρος βάθους επαρκεί για να συμπεριλάβει όλα τα εξαρτήματα.
