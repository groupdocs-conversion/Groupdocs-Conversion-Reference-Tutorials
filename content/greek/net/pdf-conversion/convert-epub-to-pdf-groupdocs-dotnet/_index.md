## Οδηγός Εφαρμογής

Ας αναλύσουμε τη διαδικασία μετατροπής ενός EPUB σε PDF σε διαχειρίσιμα βήματα.

### Φόρτωση του αρχείου EPUB προέλευσης

Αρχικά, καθορίστε το αρχείο προέλευσης και τον κατάλογο εξόδου:

```csharp
string sourceFilePath = "YOUR_DOCUMENT_DIRECTORY/sample.epub";
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "epub-converted-to.pdf");

// Βεβαιωθείτε ότι ο κατάλογος εξόδου υπάρχει
Directory.CreateDirectory(outputFolder);
```

### Ρύθμιση παραμέτρων επιλογών μετατροπής PDF

Στη συνέχεια, ορίστε τις ρυθμίσεις μετατροπής σας:

```csharp
using (var converter = new GroupDocs.Conversion.Converter(sourceFilePath))
{
    var options = new PdfConvertOptions(); // Ορισμός και ρύθμιση παραμέτρων επιλογών μετατροπής PDF
    
    // Μετατρέψτε και αποθηκεύστε το αρχείο EPUB ως PDF
    converter.Convert(outputFile, options);
}
```
