## Οδηγός Εφαρμογής

### Φόρτωση και μετατροπή OTT σε PDF

#### Επισκόπηση
Αυτή η ενότητα καλύπτει τον τρόπο φόρτωσης ενός αρχείου Open Document Template (OTT) και τη μετατροπή του σε μορφή Portable Document Format (PDF).

**Βήμα 1: Ορισμός καταλόγου εξόδου**
Καθορίστε πού θα αποθηκευτούν τα αρχεία που έχετε μετατρέψει:

```csharp
using System.IO;

string outputFolder = "YOUR_OUTPUT_DIRECTORY";

// Βεβαιωθείτε ότι ο κατάλογος υπάρχει ή δημιουργήστε τον εάν δεν υπάρχει
if (!Directory.Exists(outputFolder))
{
    Directory.CreateDirectory(outputFolder);
}
```

**Βήμα 2: Αρχικοποίηση μετατροπέα και μετατροπή**
Χρησιμοποιήστε το GroupDocs.Conversion για να φορτώσετε το αρχείο OTT και να το μετατρέψετε:

```csharp
string outputFile = Path.Combine(outputFolder, "ott-converted-to.pdf");

using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_OTT"))
{
    var options = new PdfConvertOptions();
    converter.Convert(outputFile, options);
}
```

**Εξήγηση:**
- **Κλάση μετατροπέα**: Αρχικοποιείται με τη διαδρομή προς το αρχείο OTT σας.
- **Επιλογές Μετατροπής Pdf**: Διαμορφώνει ρυθμίσεις μετατροπής ειδικά για PDF. Προσαρμόστε τες όπως απαιτείται.

### Ρύθμιση παραμέτρων καταλόγου εξόδου
Η ρύθμιση ενός καταλόγου εξόδου διασφαλίζει ότι τα αρχεία που έχουν μετατραπεί αποθηκεύονται σε μια συγκεκριμένη τοποθεσία, αποφεύγοντας την ακαταστασία και τις αντικαταστάσεις.

**Βήμα 1: Έλεγχος και δημιουργία καταλόγου**
Βεβαιωθείτε ότι ο κατάλογος υπάρχει:

```csharp
if (!Directory.Exists(outputFolder))
{
    Directory.CreateDirectory(outputFolder);
}
```

### Χρήση επιλογών μετατροπής
Η προσαρμογή των επιλογών μετατροπής σάς επιτρέπει να προσαρμόσετε το PDF εξόδου σύμφωνα με συγκεκριμένες απαιτήσεις, όπως περιθώρια ή μεγέθη σελίδων.

**Βήμα 1: Προσαρμογή του PdfConvertOptions**
Τροποποιήστε τις ρυθμίσεις χρησιμοποιώντας `PdfConvertOptions`:

```csharp
var options = new PdfConvertOptions()
{
    MarginTop = 10,
    MarginBottom = 10,
    PageSize = PageSize.A4 // Ορίστε το επιθυμητό μέγεθος σελίδας
};
```
