## Οδηγός Εφαρμογής
### Χαρακτηριστικό: Μετατροπή DICOM σε PDF
Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε αρχεία DICOM σε μορφή PDF, διευκολύνοντας την κοινή χρήση και την προβολή τους.

#### Βήμα 1: Ορισμός διαδρομών για καταλόγους εισόδου και εξόδου
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY/";

// Διαδρομή προς το αρχείο DICOM πηγής
string dicomFilePath = System.IO.Path.Combine(documentDirectory, "your-dicom-file.dcm");

// Ορίστε τη διαδρομή εξόδου PDF
string pdfOutputFile = System.IO.Path.Combine(outputDirectory, "dicom-converted-to.pdf");
```
#### Βήμα 2: Φόρτωση και μετατροπή του αρχείου DICOM
Δείτε πώς μπορείτε να φορτώσετε ένα αρχείο DICOM και να το μετατρέψετε σε PDF:
```csharp
using (Converter converter = new Converter(dicomFilePath))
{
    // Αρχικοποίηση επιλογών για μετατροπή σε PDF
    PdfConvertOptions options = new PdfConvertOptions();
    
    // Εκτελέστε μετατροπή από DICOM σε PDF και αποθηκεύστε το αποτέλεσμα
    converter.Convert(pdfOutputFile, options);
}
```
**Εξήγηση**: 
- `Converter`: Φορτώνει το αρχείο DICOM σας.
- `PdfConvertOptions`: Καθορίζει ότι θέλετε ένα αποτέλεσμα PDF.
- `converter.Convert()`: Εκτελεί τη μετατροπή.

#### Συμβουλές αντιμετώπισης προβλημάτων:
- Βεβαιωθείτε ότι οι διαδρομές προς τους καταλόγους είναι σωστές και προσβάσιμες.
- Ελέγξτε για τυχόν διενέξεις εξαρτήσεων στις αναφορές έργου.
