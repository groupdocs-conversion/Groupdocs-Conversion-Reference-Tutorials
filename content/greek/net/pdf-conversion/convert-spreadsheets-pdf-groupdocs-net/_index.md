## Οδηγός Εφαρμογής

Ακολουθήστε αυτά τα βήματα για να μετατρέψετε ένα υπολογιστικό φύλλο σε PDF χρησιμοποιώντας τις επιλογές για προχωρημένους:

### Μετατροπή υπολογιστικού φύλλου σε PDF με επιλογές για προχωρημένους

#### Επισκόπηση

Μετατρέψτε ένα αρχείο Excel σε PDF ενώ εμφανίζετε γραμμές πλέγματος και διασφαλίζετε ότι κάθε φύλλο εμφανίζεται σε μία σελίδα στο έγγραφο εξόδου.

#### Βήμα 1: Ορισμός επιλογών φόρτωσης

Διαμόρφωση επιλογών φόρτωσης για προχωρημένες ρυθμίσεις:

```csharp
using GroupDocs.Conversion.Options.Load;

Func<LoadContext, LoadOptions> getLoadOptions = loadContext => new SpreadsheetLoadOptions
{
    ShowGridLines = true,
    OnePagePerSheet = true
};
```

**Εξήγηση**: `SpreadsheetLoadOptions` σας επιτρέπει να διαμορφώσετε τον τρόπο φόρτωσης του υπολογιστικού φύλλου. Ρύθμιση `ShowGridLines` να `true` περιλαμβάνει γραμμές πλέγματος στο PDF σας, και `OnePagePerSheet` διασφαλίζει ότι κάθε φύλλο χωράει σε μία μόνο σελίδα.

#### Βήμα 2: Μετατροπή χρησιμοποιώντας την κλάση μετατροπέα

Χρησιμοποιήστε το `Converter` κλάση για να εκτελέσετε τη μετατροπή:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion.Options.Convert;

string inputFilePath = "sample.xlsx";
string outputFolder = "output";
string outputFile = Path.Combine(outputFolder, "converted.pdf");

// Αρχικοποίηση μετατροπέα με επιλογές φόρτωσης
using (Converter converter = new Converter(inputFilePath, getLoadOptions))
{
    PdfConvertOptions options = new PdfConvertOptions(); // Ρύθμιση επιλογών μετατροπής PDF
    converter.Convert(outputFile, options); // Εκτελέστε τη μετατροπή
}
```

**Εξήγηση**: Το `Converter` η τάξη λαμβάνει υπόψη τη διαδρομή του αρχείου Excel και τις επιλογές φόρτωσης. Το `PdfConvertOptions` Η κλάση καθορίζει τυχόν πρόσθετες ρυθμίσεις για την έξοδο PDF.

#### Συμβουλές αντιμετώπισης προβλημάτων
