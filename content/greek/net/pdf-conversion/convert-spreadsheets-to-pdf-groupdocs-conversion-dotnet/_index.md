## Οδηγός Εφαρμογής

### Μετατροπή υπολογιστικού φύλλου σε PDF με επιλογές για προχωρημένους

**Επισκόπηση:** Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε ένα έγγραφο υπολογιστικού φύλλου (π.χ., XLSX) σε αρχείο PDF, χρησιμοποιώντας παράλληλα προηγμένες επιλογές φόρτωσης. Μπορείτε να παραλείψετε κενές γραμμές/στήλες και να διαμορφώσετε κάθε φύλλο ώστε να μετατρέπεται σε μία μόνο σελίδα PDF.

#### Βήμα προς βήμα εφαρμογή:
1. **Ορισμός καταλόγου εξόδου**
   Ρυθμίστε τη διαδρομή του καταλόγου εξόδου χρησιμοποιώντας placeholders για ευελιξία:
   
   ```csharp
   string outputFolder = "YOUR_OUTPUT_DIRECTORY";
   string outputFile = Path.Combine(outputFolder, "converted.pdf");
   ```

2. **Ρύθμιση παραμέτρων επιλογών φόρτωσης**
   Προσαρμόστε τις επιλογές φόρτωσης για να παραλείψετε κενές σειρές/στήλες και να μετατρέψετε κάθε φύλλο σε μία μόνο σελίδα PDF:
   
   ```csharp
   Func<LoadContext, LoadOptions> getLoadOptions = loadContext => new SpreadsheetLoadOptions
   {
       SkipEmptyRowsAndColumns = true,
       OnePagePerSheet = true
   };
   ```

3. **Αρχικοποίηση μετατροπέα**
   Ρυθμίστε τον μετατροπέα με το αρχείο εισόδου σας και διαμορφώστε τις επιλογές φόρτωσης:
   
   ```csharp
   string inputFile = "YOUR_DOCUMENT_DIRECTORY" + "/sample.xlsx";
   using (Converter converter = new Converter(inputFile, getLoadOptions))
   {
       PdfConvertOptions options = new PdfConvertOptions();
       converter.Convert(outputFile, options);
   }
   ```

### Διαμορφώσιμες επιλογές φόρτωσης για μετατροπή υπολογιστικών φύλλων

**Επισκόπηση:** Αυτή η λειτουργία σάς επιτρέπει να ορίσετε και να προσαρμόσετε τις επιλογές φόρτωσης πριν από τη μετατροπή. Είναι χρήσιμη σε σενάρια όπου η προεπεξεργασία του υπολογιστικού φύλλου μπορεί να βελτιώσει το τελικό αποτέλεσμα.

#### Βήμα προς βήμα εφαρμογή:
1. **Ορισμός προσαρμοσμένων επιλογών φόρτωσης**
   Δημιουργήστε μια μέθοδο για να διαμορφώσετε τις επιθυμητές ρυθμίσεις φόρτωσης:
   
   ```csharp
   public static Func<LoadContext, LoadOptions> GetLoadOptions()
   {
       return loadContext => new SpreadsheetLoadOptions
       {
           SkipEmptyRowsAndColumns = true,
           OnePagePerSheet = true
       };
   }
   ```

2. **Χρησιμοποιήστε προσαρμοσμένες επιλογές φόρτωσης**
   Χρησιμοποιήστε αυτές τις διαμορφώσιμες επιλογές στη διαδικασία μετατροπής σας:
   
   ```csharp
   Func<LoadContext, LoadOptions> loadOptions = GetLoadOptions();
   string inputFile = "YOUR_DOCUMENT_DIRECTORY" + "/sample.xlsx";
   using (Converter converter = new Converter(inputFile, loadOptions))
   {
       PdfConvertOptions pdfOptions = new PdfConvertOptions();
       converter.Convert(outputFile, pdfOptions);
   }
   ```

#### Συμβουλές αντιμετώπισης προβλημάτων:
- Βεβαιωθείτε ότι η διαδρομή του αρχείου εισόδου είναι σωστή και προσβάσιμη.
- Βεβαιωθείτε ότι έχουν οριστεί όλα τα απαραίτητα δικαιώματα για την ανάγνωση/εγγραφή αρχείων.
- Ελέγξτε εάν η έκδοση του πακέτου GroupDocs.Conversion ταιριάζει με τη ρύθμιση του έργου σας.
