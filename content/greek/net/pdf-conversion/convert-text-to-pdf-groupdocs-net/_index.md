## Οδηγός Εφαρμογής
Αυτή η ενότητα θα αναλύσει την υλοποίηση σε δύο κύρια χαρακτηριστικά: τη φόρτωση ενός εγγράφου κειμένου με προηγμένες επιλογές και τη μετατροπή του σε PDF.

### Φόρτωση εγγράφου κειμένου με επιλογές για προχωρημένους
**Επισκόπηση:** Αυτή η λειτουργία σάς επιτρέπει να προσαρμόσετε τον τρόπο φόρτωσης του εγγράφου κειμένου σας, όπως τον χειρισμό των κενών στην αρχή και την ανίχνευση αρίθμησης που έχει μορφοποιηθεί με κενά.

#### Βήμα 1: Ορισμός Επιλογών Φόρτωσης
Διαμορφώστε τις επιλογές φόρτωσης χρησιμοποιώντας το `Func<LoadContext, LoadOptions>` αντιπρόσωπος:
```csharp
using System;
using GroupDocs.Conversion.Options.Load;

// Ρύθμιση παραμέτρων για προχωρημένους επιλογές φόρτωσης
Func<LoadContext, LoadOptions> getLoadOptions = loadContext => new TxtLoadOptions
{
    LeadingSpacesOptions = TxtLeadingSpacesOptions.ConvertToIndent,
    DetectNumberingWithWhitespaces = true
};
```
- **Επιλογές LeadingSpaces:** Μετατρέπει τα αρχικά κενά σε εσοχές.
- **ΕντοπισμόςΑριθμήσεωνΜεΛευκάΧώρα:** Αναγνωρίζει λίστες που έχουν μορφοποιηθεί με κενά.

### Μετατροπή εγγράφου κειμένου σε PDF
**Επισκόπηση:** Αυτή η λειτουργία δείχνει πώς να μετατρέψετε το φορτωμένο έγγραφο κειμένου σε αρχείο PDF, χρησιμοποιώντας προσαρμόσιμες επιλογές μετατροπής.

#### Βήμα 2: Ρύθμιση επιλογών μετατροπής
Ορίστε τις ρυθμίσεις μετατροπής σας:
```csharp
using System.IO;
using GroupDocs.Conversion.Options.Convert;

string outputFolder = "your-output-directory";
string outputFile = Path.Combine(outputFolder, "converted.pdf");

// Μετατρέψτε το έγγραφο σε PDF
using (Converter converter = new Converter("your-document-path", getLoadOptions))
{
    PdfConvertOptions options = new PdfConvertOptions();
    
    // Εκτελέστε μετατροπή και αποθηκεύστε ως αρχείο PDF
    converter.Convert(outputFile, options);
}
```
- **Επιλογές PdfConvert:** Επιτρέπει την προσαρμογή της εξόδου PDF.
