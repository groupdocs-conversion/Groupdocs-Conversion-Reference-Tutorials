## Οδηγός Εφαρμογής

### Φόρτωση αρχείου BMP πηγής

Αυτή η ενότητα δείχνει πώς να φορτώσετε το αρχείο BMP πηγής σας, κάτι που είναι απαραίτητο πριν από τη μετατροπή του.

#### Βήμα 1: Καθορίστε τη διαδρομή αρχείου BMP
Αρχικά, ορίστε πού αποθηκεύεται το αρχείο BMP. Βεβαιωθείτε ότι έχετε ορίσει τη σωστή διαδρομή στην εφαρμογή σας:

```csharp
string bmpFilePath = "YOUR_DOCUMENT_DIRECTORY/sample.bmp";
```

#### Βήμα 2: Φόρτωση του αρχείου BMP

Φορτώστε το BMP σε μια παρουσία μετατροπέα για να ξεκινήσετε τη διαδικασία μετατροπής.

```csharp
using (var converter = new GroupDocs.Conversion.Converter(bmpFilePath))
{
    // Το αρχείο BMP έχει πλέον φορτωθεί και είναι έτοιμο για μετατροπή.
}
```
**Εξήγηση:** Εδώ, δημιουργούμε μια παρουσία του `Converter` χρησιμοποιώντας τη διαδρομή αρχείου για να φορτώσουμε την εικόνα μας. Αυτή η ρύθμιση μας επιτρέπει να εκτελέσουμε περαιτέρω λειτουργίες σε αυτό το έγγραφο.

### Μετατροπή BMP σε PDF

Μόλις φορτώσετε το αρχείο προέλευσης, ήρθε η ώρα να το μετατρέψετε σε μορφή PDF.

#### Βήμα 1: Ορισμός διαδρομής εξόδου
Ορίστε πού θα αποθηκευτεί το PDF που έχει μετατραπεί:

```csharp
string pdfOutputPath = "YOUR_OUTPUT_DIRECTORY/bmp-converted-to.pdf";
```

#### Βήμα 2: Ορισμός επιλογών μετατροπής
Διαμορφώστε τις ρυθμίσεις μετατροπής ειδικά για την έξοδο PDF:

```csharp
var options = new PdfConvertOptions();
```
**Εξήγηση:** `PdfConvertOptions` παρέχει διάφορες παραμέτρους προσαρμογής όπως περιθώρια, μέγεθος σελίδας και άλλα.

#### Βήμα 3: Εκτέλεση μετατροπής
Εκτελέστε τη λειτουργία μετατροπής από BMP σε PDF:

```csharp
using (var converter = new GroupDocs.Conversion.Converter(bmpFilePath))
{
    // Μετατρέψτε το φορτωμένο αρχείο BMP σε μορφή PDF
    converter.Convert(pdfOutputPath, options);
}
```
**Εξήγηση:** Ο `Convert` Η μέθοδος εκτελεί τον πραγματικό μετασχηματισμό, λαμβάνοντας ως παραμέτρους τόσο τη διαδρομή προορισμού όσο και τις ρυθμίσεις μετατροπής.
