### Μετατροπή αρχείου XLTX σε PDF

#### Επισκόπηση
Τώρα που το αρχείο σας έχει φορτωθεί και οι επιλογές μετατροπής έχουν οριστεί, εκτελέστε την πραγματική διαδικασία μετατροπής.

**Βήμα 3: Εκτέλεση μετατροπής**

```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;
using System.IO;

class Program
{
    static void Main(string[] args)
    {
        string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
        string outputFile = Path.Combine(outputFolder, "xltx-converted-to.pdf");

        using (var converter = new Converter(@"YOUR_DOCUMENT_DIRECTORY\sample.xltx"))
        {
            var pdfOptions = new PdfConvertOptions();
            converter.Convert(outputFile, pdfOptions);
            Console.WriteLine("Conversion completed successfully!");
        }
    }
}
```

**Εξήγηση:**
- Ο `converter.Convert` Η μέθοδος παίρνει τη διαδρομή του αρχείου εξόδου και τις επιλογές PDF για να εκτελέσει τη μετατροπή.
- Βεβαιωθείτε ότι `outputFolder` έχει ρυθμιστεί σωστά για το πού θέλετε να αποθηκευτεί το PDF που έχετε μετατρέψει.
