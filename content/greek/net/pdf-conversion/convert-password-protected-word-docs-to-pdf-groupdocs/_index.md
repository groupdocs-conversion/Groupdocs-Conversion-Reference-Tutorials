Αυτό το σεμινάριο θα σας καθοδηγήσει στη χρήση της βιβλιοθήκης GroupDocs.Conversion για να μετατρέψετε ασφαλή έγγραφα Word σε αναγνώσιμα PDF, διατηρώντας παράλληλα τον έλεγχο συγκεκριμένων σελίδων και ρυθμίσεων.

Μέχρι το τέλος αυτού του οδηγού, θα είστε εξοπλισμένοι με τις απαραίτητες γνώσεις για να μετατρέπετε έγγραφα χωρίς κόπο.

## Οδηγός Εφαρμογής

Αυτή η ενότητα καλύπτει τη μετατροπή εγγράφων που προστατεύονται με κωδικό πρόσβασης και τον καθορισμό σελίδων για μετατροπή.

### Λειτουργία 1: Μετατροπή εγγράφου που προστατεύεται με κωδικό πρόσβασης σε PDF

#### Επισκόπηση
Η μετατροπή ενός εγγράφου Word που προστατεύεται με κωδικό πρόσβασης σε PDF σάς επιτρέπει να μοιράζεστε αρχεία με ασφάλεια, διατηρώντας παράλληλα την ακεραιότητα του περιεχομένου. Αυτή η λειτουργία δείχνει πώς να ξεκλειδώσετε ένα προστατευμένο έγγραφο χρησιμοποιώντας το GroupDocs.Conversion και να το μετατρέψετε σε μορφή PDF με συγκεκριμένες ρυθμίσεις.

#### Βήμα προς βήμα εφαρμογή

##### 1. Ρύθμιση επιλογών φόρτωσης
Ορίστε επιλογές φόρτωσης, συμπεριλαμβανομένου του κωδικού πρόσβασης για την πρόσβαση στο έγγραφο:
```csharp
Func<LoadContext, LoadOptions> getLoadOptions = loadContext => new WordProcessingLoadOptions
{
    Password = "12345" // Αντικαταστήστε με τον πραγματικό κωδικό πρόσβασης του εγγράφου σας
};
```

##### 2. Αρχικοποίηση αντικειμένου μετατροπέα
Δημιουργήστε ένα `Converter` παράδειγμα για τη διαχείριση της διαδικασίας μετατροπής:
```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", ".");
string outputFile = Path.Combine(outputFolder, "converted.pdf");

using (Converter converter = new Converter(Path.Combine("YOUR_DOCUMENT_DIRECTORY", "SAMPLE_DOCX_WITH_PASSWORD"), getLoadOptions))
{
    // Θα ακολουθήσει η ρύθμιση των επιλογών μετατροπής
}
```

##### 3. Διαμόρφωση επιλογών μετατροπής PDF
Καθορίστε τις ρυθμίσεις για το αρχείο PDF εξόδου:
```csharp
PdfConvertOptions options = new PdfConvertOptions
{
    PageNumber = 2,         // Ξεκινήστε από τη σελίδα με αριθμό 2
    PagesCount = 1,          // Μετατροπή μόνο μίας σελίδας
    Rotate = Rotation.On180, // Περιστροφή σελίδας κατά 180 μοίρες
    Dpi = 300,               // Ορίστε το DPI σε 300 για έξοδο υψηλής ποιότητας
    PageWidth = 1024,        // Ορίστε το πλάτος των σελίδων PDF
    PageHeight = 768         // Ορίστε το ύψος των σελίδων PDF
};
```

##### 4. Εκτελέστε μετατροπή
Εκτελέστε τη μετατροπή χρησιμοποιώντας τις διαμορφωμένες επιλογές:
```csharp
converter.Convert(outputFile, options);
// Το αρχείο που έχει μετατραπεί αποθηκεύεται στον κατάλογο 'YOUR_OUTPUT_DIRECTORY'
```
