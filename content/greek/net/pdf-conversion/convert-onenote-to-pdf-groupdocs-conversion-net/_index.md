## Οδηγός Εφαρμογής

### Φόρτωση και μετατροπή αρχείου .one σε PDF

#### Επισκόπηση
Αυτή η ενότητα εστιάζει στον τρόπο φόρτωσης του αρχείου OneNote και μετατροπής του σε μορφή PDF χρησιμοποιώντας το GroupDocs.Conversion για .NET.

##### Βήμα 1: Ορισμός διαδρομών

Ξεκινήστε ορίζοντας τις διαδρομές για την πηγή σας `.one` αρχείο και το τελικό αποτέλεσμα PDF:

```csharp
using System;
using System.IO;

string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// Καθορίστε τη διαδρομή προς το αρχείο OneNote και το PDF που προκύπτει.
string oneFilePath = Path.Combine(documentDirectory, "sample.one");
string pdfOutputFile = Path.Combine(outputDirectory, "one-converted-to.pdf");
```

##### Βήμα 2: Φόρτωση του αρχείου Source ONE

Φορτώστε το `.one` αρχείο χρησιμοποιώντας το GroupDocs.Conversion:

```csharp
using (var converter = new Converter(oneFilePath))
{
    // Συνεχίστε για να καθορίσετε τις επιλογές μετατροπής.
}
```

##### Βήμα 3: Καθορίστε επιλογές μετατροπής για τη μορφή PDF

Ρυθμίστε τις επιλογές μετατροπής PDF:

```csharp
var pdfOptions = new PdfConvertOptions();
```

##### Βήμα 4: Μετατροπή και αποθήκευση του αρχείου .one ως έγγραφο PDF

Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο εξόδου:

```csharp
converter.Convert(pdfOutputFile, pdfOptions);
Console.WriteLine("Conversion completed successfully.");
```

- **Επιλογές Μετατροπής Pdf**: Προσαρμόστε τα εύρη σελίδων, την περιστροφή και άλλες ρυθμίσεις για να προσαρμόσετε το αποτέλεσμα.
