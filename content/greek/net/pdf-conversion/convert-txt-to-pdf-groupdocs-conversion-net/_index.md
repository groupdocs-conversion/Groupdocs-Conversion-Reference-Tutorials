### Μετατροπή TXT σε PDF

Μετασχηματισμός ενός `.txt` Η μετατροπή ενός εγγράφου σε PDF είναι απλή με τις ενσωματωμένες επιλογές του GroupDocs.Conversion.

#### Βήμα 2: Ορισμός παραμέτρων εξόδου

Ρυθμίστε τον κατάλογο εξόδου και ορίστε πού θα αποθηκευτεί το PDF που έχει μετατραπεί:

```csharp
using System.IO;
using GroupDocs.Conversion.Options.Convert;

string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "txt-converted-to.pdf");
```

#### Βήμα 3: Εκτέλεση μετατροπής

Χρησιμοποιήστε το `PdfConvertOptions` για να εκτελέσετε τη μετατροπή και να αποθηκεύσετε το έγγραφό σας:

```csharp
var options = new PdfConvertOptions();
converter.Convert(outputFile, options);
// Το PDF που μετατράπηκε αποθηκεύεται πλέον στον καθορισμένο κατάλογο εξόδου.
```

Αυτή η ενότητα επισημαίνει πώς το GroupDocs.Conversion απλοποιεί τη μετατροπή αρχείων κειμένου σε μια ευέλικτη μορφή.
