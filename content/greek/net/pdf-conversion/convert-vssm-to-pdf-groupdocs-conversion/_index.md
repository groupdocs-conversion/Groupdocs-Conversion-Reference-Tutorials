#### Βασική Αρχικοποίηση και Ρύθμιση σε C#

```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main()
    {
        // Αρχικοποιήστε τον μετατροπέα με τη διαδρομή αρχείου VSSM πηγής.
        using (Converter converter = new Converter("path/to/your/file.vssm"))
        {
            // Προετοιμάστε επιλογές μετατροπής για μορφή PDF.
            PdfConvertOptions options = new PdfConvertOptions();
            
            // Μετατρέψτε και αποθηκεύστε το αποτέλεσμα ως έγγραφο PDF.
            converter.Convert("output/path/output.pdf", options);
        }
    }
}
```

Σε αυτήν τη ρύθμιση, αρχικοποιούμε `Converter` με τη διαδρομή αρχείου VSSM μας. Το `PdfConvertOptions` Το αντικείμενο καθορίζει τις ρυθμίσεις για τη μετατροπή σε PDF.
