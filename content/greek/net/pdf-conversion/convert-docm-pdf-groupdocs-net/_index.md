## Ρύθμιση του GroupDocs.Conversion για .NET

Για να ξεκινήσετε τη χρήση **GroupDocs.Conversion** στο έργο σας, ακολουθήστε τα παρακάτω βήματα εγκατάστασης:

### Εγκατάσταση μέσω της κονσόλας NuGet Package Manager
Ανοίξτε την κονσόλα NuGet Package Manager στο Visual Studio και εκτελέστε:

```bash
Install-Package GroupDocs.Conversion -Version 25.3.0
```

### Εγκατάσταση μέσω .NET CLI
Αν προτιμάτε να χρησιμοποιήσετε τη γραμμή εντολών, εκτελέστε:

```bash
dotnet add package GroupDocs.Conversion --version 25.3.0
```
