### Φόρτωση και μετατροπή αρχείου DNG σε PDF
#### Επισκόπηση
Αυτή η ενότητα εξηγεί πώς να φορτώσετε ένα αρχείο DNG και να το μετατρέψετε σε PDF χρησιμοποιώντας το GroupDocs.Conversion.

#### Βήμα προς βήμα εφαρμογή
##### Ορισμός διαδρομών και αρχικοποίηση μετατροπέα
```csharp
// Ορίστε διαδρομές για καταλόγους εγγράφων και εξόδου\string sourceDirectory = "YOUR_DOCUMENT_DIRECTORY";\string outputDirectory = "YOUR_OUTPUT_DIRECTORY/";

// Πλήρης διαδρομή για την εισαγωγή DNG file\string dngFilePath = Path.Combine(sourceDirectory, "sample.dng");

// Πλήρης διαδρομή προς το αρχείο PDF εξόδου\string pdfOutputPath = Path.Combine(outputDirectory, "dng-converted-to.pdf");
```
Εδώ, ορίζουμε τις απαραίτητες διαδρομές και αρχικοποιούμε το περιβάλλον μετατροπής.

##### Αρχικοποίηση μετατροπέα με αρχείο DNG πηγής
```csharp
using (var converter = new Converter(dngFilePath))
{
    // Ορίστε επιλογές μετατροπής για μορφή PDF
    var convertOptions = new PdfConvertOptions();

    // Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο PDF εξόδου
    converter.Convert(pdfOutputPath, convertOptions);
}
```
- **Αρχικοποίηση μετατροπέα**: Το `Converter` Το αντικείμενο αρχικοποιείται με τη διαδρομή του αρχείου DNG πηγής.
- **Επιλογές μετατροπής**: Ορίζουμε `PdfConvertOptions`, καθορίζοντας τις ρυθμίσεις μορφής στόχου.
- **Εκτέλεση μετατροπής**: Το `Convert` Η μέθοδος εκτελεί τη μετατροπή και αποθηκεύει το PDF στην καθορισμένη διαδρομή εξόδου.
