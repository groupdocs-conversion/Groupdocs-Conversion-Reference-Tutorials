## Οδηγός Εφαρμογής

Ας αναλύσουμε τη διαδικασία μετατροπής σε διαχειρίσιμα βήματα.

### Βήμα 1: Ορισμός καταλόγου εξόδου και ονόματος αρχείου

Ξεκινήστε ορίζοντας πού θα αποθηκευτεί το PDF που έχετε μετατρέψει:
```csharp
string outputFolder = Path.Combine(@"YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "oxps-converted-to.pdf");
```

### Βήμα 2: Φόρτωση του αρχείου προέλευσης OXPS

Φορτώστε το αρχείο προέλευσης χρησιμοποιώντας το GroupDocs.Conversion `Converter` κλάση. Αυτό χειρίζεται την αρχική ρύθμιση και προετοιμάζει το έγγραφό σας για μετατροπή.
```csharp
using (var converter = new Converter(@"YOUR_DOCUMENT_DIRECTORY\sample.oxps"))
{
    Console.WriteLine("Source OXPS file loaded.");
}
```

### Βήμα 3: Ρύθμιση επιλογών μετατροπής

Ρύθμιση παραμέτρων επιλογών ειδικά για PDF χρησιμοποιώντας `PdfConvertOptions`Αυτό σας επιτρέπει να καθορίσετε ρυθμίσεις σχετικές με τη μορφή εξόδου.
```csharp
var options = new PdfConvertOptions();
Console.WriteLine("PDF conversion options set.");
```

### Βήμα 4: Μετατροπή και αποθήκευση του αρχείου

Τέλος, χρησιμοποιήστε το `Convert` μέθοδος για την εκτέλεση της μετατροπής και την αποθήκευση του αρχείου PDF:
```csharp
converter.Convert(outputFile, options);
Console.WriteLine("Conversion completed successfully!");
```
