## Οδηγός Εφαρμογής

Ας αναλύσουμε τη διαδικασία υλοποίησης σε λογικά βήματα για τη μετατροπή των αρχείων OTP σε PDF.

### Φόρτωση και διαμόρφωση του εγγράφου σας
Αρχικά, φορτώστε το αρχείο OTP χρησιμοποιώντας το `Converter` κλάση. Αυτό λειτουργεί ως βάση για όλες τις λειτουργίες μετατροπής:

```csharp
// Ορισμός διαδρομών
string sourceFilePath = "YOUR_DOCUMENT_DIRECTORY\\sample.otp";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// Αρχικοποίηση του μετατροπέα με τη διαδρομή αρχείου OTP
using (var converter = new Converter(sourceFilePath))
{
    // Ρύθμιση παραμέτρων επιλογών μετατροπής PDF
    var pdfOptions = new PdfConvertOptions();

    // Ρύθμιση διαδρομής αρχείου εξόδου
    string outputFile = Path.Combine(outputDirectory, "otp-converted-to.pdf");

    // Μετατροπή και αποθήκευση του εγγράφου ως PDF
    converter.Convert(outputFile, pdfOptions);
}
```

#### Επεξήγηση Βασικών Παραμέτρων
- **Κλάση μετατροπέα**: Αντιπροσωπεύει το αρχείο προέλευσης. Απαιτεί μια έγκυρη διαδρομή αρχείου OTP.
- **Επιλογές Μετατροπής Pdf**: Ρυθμίζει τον τρόπο εκτέλεσης της μετατροπής. Μπορείτε να προσαρμόσετε ρυθμίσεις όπως αριθμούς σελίδων και επίπεδα ζουμ.
- **μετατροπέας.Μετατροπή()**: Η μέθοδος που εκτελεί την πραγματική μετατροπή αρχείου.

### Αντιμετώπιση συνηθισμένων προβλημάτων
- **Σφάλμα "Δεν βρέθηκε αρχείο"**Βεβαιωθείτε ότι η διαδρομή του αρχείου προέλευσης είναι σωστή.
- **Η άδεια απορρίφθηκε**Ελέγξτε εάν η εφαρμογή σας έχει πρόσβαση εγγραφής στον κατάλογο εξόδου.
