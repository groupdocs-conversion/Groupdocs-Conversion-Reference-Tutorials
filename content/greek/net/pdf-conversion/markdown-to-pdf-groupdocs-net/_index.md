## Οδηγός Εφαρμογής
### Μετατροπή Markdown σε PDF
#### Επισκόπηση
Μετατρέψτε τα έγγραφά σας Markdown σε μορφοποιημένα PDF χωρίς κόπο.

#### Βήμα προς βήμα εφαρμογή
##### 1. Φορτώστε το έγγραφο Markdown

```csharp
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

string markdownFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.md");

// Αρχικοποίηση της παρουσίας της κλάσης Converter με τη διαδρομή αρχείου εισόδου
using (var converter = new Converter(markdownFilePath))
{
    Console.WriteLine("Markdown loaded successfully!");
}
```
##### 2. Ορίστε τις επιλογές μετατροπής PDF

```csharp
PdfConvertOptions options = new PdfConvertOptions();

// Διαμορφώστε πρόσθετες ρυθμίσεις όπως μέγεθος σελίδας, περιθώρια, εάν χρειάζεται
options.PageSize = PageSize.A4;
```
##### 3. Μετατροπή και αποθήκευση του PDF εξόδου

```csharp
string outputFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "output.pdf");

converter.Convert(outputFilePath, options);
Console.WriteLine("Conversion to PDF completed!");
```
#### Επεξήγηση παραμέτρων
- **Μετατροπέας**: Διαχειρίζεται τη διαδικασία μετατροπής.
- **Επιλογές Μετατροπής Pdf**: Προσαρμόζει το PDF που προκύπτει.
##### Βασικές επιλογές διαμόρφωσης
- Προσαρμόστε το μέγεθος σελίδας χρησιμοποιώντας `options.PageSize`.
- Προσθήκη περιθωρίων ή άλλων προτιμήσεων στυλ στο `PdfConvertOptions`.
##### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι οι διαδρομές αρχείων είναι σωστές και προσβάσιμες.
- Επαληθεύστε τα απαραίτητα δικαιώματα για την ανάγνωση/εγγραφή αρχείων.
