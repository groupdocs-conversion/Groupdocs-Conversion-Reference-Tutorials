### Μετατροπή DIB σε PDF

#### Επισκόπηση
Αυτή η λειτουργία μετατρέπει ένα αρχείο DIB σε μια καθολικά προσβάσιμη μορφή PDF. Θα καλύψουμε τη φόρτωση του αρχείου DIB και τη διαμόρφωση των ρυθμίσεων μετατροπής για βέλτιστα αποτελέσματα.

#### Βήμα 1: Φόρτωση του αρχείου DIB προέλευσης
Αρχικά, ρυθμίστε τον κατάλογο εξόδου και φορτώστε το αρχείο DIB πηγής:

```csharp

string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "dib-converted-to.pdf");

// Αντικαταστήστε με μια πραγματική διαδρομή προς το αρχείο DIB σας
var dibFilePath = "YOUR_DOCUMENT_DIRECTORY\\sample.dib";
```

#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών μετατροπής PDF
Στη συνέχεια, διαμορφώστε τις επιλογές μετατροπής που είναι προσαρμοσμένες για την έξοδο PDF:

```csharp

var options = new PdfConvertOptions();
// Μπορείτε να ορίσετε πρόσθετες επιλογές εδώ, όπως απαιτείται
```

#### Βήμα 3: Εκτελέστε τη μετατροπή και αποθηκεύστε την έξοδο
Τέλος, εκτελέστε τη διαδικασία μετατροπής και αποθηκεύστε το αρχείο PDF που προκύπτει:

```csharp
using (var converter = new Converter(dibFilePath))
{
    // Μετατροπή DIB σε PDF χρησιμοποιώντας τις καθορισμένες επιλογές
    converter.Convert(outputFile, options);
    Console.WriteLine("Conversion completed successfully.");
}
```
