## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Μετατροπή CMX σε PDF
Αυτή η λειτουργία εστιάζει στη μετατροπή ενός αρχείου εικόνας Corel Metafile Exchange (.cmx) σε μορφή φορητού εγγράφου (PDF).

#### Βήμα 1: Φόρτωση του αρχείου CMX πηγής
Φορτώστε το αρχείο προέλευσης χρησιμοποιώντας το GroupDocs.Conversion `Converter` κλάση, ρυθμίζοντας τη διαδικασία μετατροπής διαβάζοντας το αρχικό αρχείο CMX.

```csharp
using (var converter = new Converter(cmxFilePath))
{
    // Η ρύθμιση της μετατροπής θα ακολουθήσει εδώ.
}
```
#### Βήμα 2: Ρύθμιση επιλογών μετατροπής PDF
Στη συνέχεια, διαμορφώστε τις επιλογές για τη μετατροπή σε PDF χρησιμοποιώντας το `PdfConvertOptions` κλάση, η οποία επιτρέπει διάφορες προσαρμογές ρυθμίσεων.

```csharp
var options = new PdfConvertOptions();
```
#### Βήμα 3: Εκτελέστε τη μετατροπή και αποθηκεύστε την έξοδο
Χρησιμοποιήστε το `Convert` μέθοδος για την εκτέλεση της μετατροπής και την αποθήκευση του αποτελέσματος ως αρχείο PDF. Αυτό το βήμα χειρίζεται τον μετασχηματισμό μορφών δεδομένων.

```csharp
converter.Convert(pdfOutputFile, options);
```
