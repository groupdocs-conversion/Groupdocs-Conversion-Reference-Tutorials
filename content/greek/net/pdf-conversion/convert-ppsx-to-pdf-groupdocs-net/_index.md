### Μετατροπή PPSX σε PDF

#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς να μετατρέψετε μια παρουσίαση PowerPoint (PPSX) σε έγγραφο PDF. Η διαδικασία μετατροπής είναι απλή με τη βιβλιοθήκη GroupDocs.

**Βήματα για την εφαρμογή:**

##### Βήμα 1: Ορισμός διαδρομών και φόρτωση αρχείου
Αρχικά, καθορίστε τους καταλόγους για τα αρχεία προέλευσης και τον προορισμό εξόδου:
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputFileDirectory = "YOUR_OUTPUT_DIRECTORY";

// Καθορίστε διαδρομές αρχείων
string ppsxFilePath = Path.Combine(documentDirectory, "sample.ppsx");
string pdfOutputPath = Path.Combine(outputFileDirectory, "ppsx-converted-to.pdf");
```

##### Βήμα 2: Ρύθμιση επιλογών μετατροπής
Στη συνέχεια, ρυθμίστε τις επιλογές για τη μετατροπή του PPSX σας σε PDF:
```csharp
// Φορτώστε το αρχείο προέλευσης και προετοιμάστε τις ρυθμίσεις μετατροπής
using (var converter = new Converter(ppsxFilePath))
{
    var convertOptions = new PdfConvertOptions();
    
    // Μετατρέψτε και αποθηκεύστε το αποτέλεσμα ως PDF
    converter.Convert(pdfOutputPath, convertOptions);
}
```
**Εξήγηση**: Το `Converter` η κλάση φορτώνει το αρχείο PPSX σας, ενώ `PdfConvertOptions` ρυθμίζει τον τρόπο χειρισμού της μετατροπής.
