## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Φόρτωση αρχείου JPX
**Επισκόπηση**
Η φόρτωση ενός αρχείου .jpx είναι το πρώτο βήμα για την προετοιμασία του για μετατροπή. Αυτή η λειτουργία σάς επιτρέπει να ενσωματώνετε δεδομένα εικόνας απρόσκοπτα στη ροή εργασίας της εφαρμογής σας.

#### Βήμα 1: Ρύθμιση του καταλόγου εγγράφων σας
Βεβαιωθείτε ότι η διαδρομή του καταλόγου εγγράφων σας έχει οριστεί σωστά όπου αποθηκεύονται τα αρχεία JPX.
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
```

#### Βήμα 2: Φόρτωση του αρχείου προέλευσης
Δημιουργήστε μια παρουσία του `Converter` κλάση και ανοίξτε το αρχείο JPX πηγαίου κώδικα:
```csharp
using (var converter = new Converter(Path.Combine(documentDirectory, "sample.jpx")))
{
    // Το αρχείο έχει πλέον φορτωθεί και είναι έτοιμο για μετατροπή
}
```

### Χαρακτηριστικό: Μετατροπή JPX σε μορφή XLS
**Επισκόπηση**
Η μετατροπή ενός αρχείου .jpx σε μορφή .xls σάς επιτρέπει να χρησιμοποιείτε δεδομένα εικόνας σε εφαρμογές υπολογιστικών φύλλων όπως το Microsoft Excel.

#### Βήμα 1: Ορισμός διαδρομών εξόδου
Ορίστε διαδρομές για τον κατάλογο εξόδου σας και το όνομα του αρχείου που μετατράπηκε:
```csharp
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputDirectory, "jpx-converted-to.xls");
```

#### Βήμα 2: Αρχικοποίηση επιλογών μετατροπής
Καθορίστε ότι θέλετε να μετατρέψετε το αρχείο εικόνας σε μορφή XLS χρησιμοποιώντας `SpreadsheetConvertOptions`:
```csharp
var options = new SpreadsheetConvertOptions { Format = GroupDocs.Conversion.FileTypes.SpreadsheetFileType.Xls };
```

#### Βήμα 3: Εκτελέστε τη μετατροπή
Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο εξόδου στην επιθυμητή θέση:
```csharp
converter.Convert(outputFile, options);
```

- **Συνηθισμένο πρόβλημα:** Το αρχείο δεν βρέθηκε. Βεβαιωθείτε ότι η διαδρομή προς το αρχείο JPX πηγής είναι σωστή.
