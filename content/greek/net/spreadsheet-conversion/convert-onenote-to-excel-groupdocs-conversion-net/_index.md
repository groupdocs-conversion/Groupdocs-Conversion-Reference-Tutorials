## Ρύθμιση του GroupDocs.Conversion για .NET

Για να ξεκινήσετε να χρησιμοποιείτε το GroupDocs.Conversion, εγκαταστήστε τη βιβλιοθήκη στο έργο σας:

**Κονσόλα διαχείρισης πακέτων NuGet**
```plaintext
Install-Package GroupDocs.Conversion -Version 25.3.0
```

**.NET CLI**
```bash
dotnet add package GroupDocs.Conversion --version 25.3.0
```

### Βήματα απόκτησης άδειας χρήσης
Για να αξιοποιήσετε πλήρως το GroupDocs.Conversion, ενδέχεται να χρειαστείτε μια άδεια χρήσης:
- Ξεκινήστε με μια δωρεάν δοκιμή για να εξερευνήσετε τις λειτουργίες.
- Αίτημα προσωρινής άδειας για εκτεταμένη αξιολόγηση.
- Αγοράστε μια πλήρη άδεια χρήσης για χρήση παραγωγής.

Μόλις εγκατασταθεί και αδειοδοτηθεί, αρχικοποιήστε το έργο σας:
```csharp
using GroupDocs.Conversion;
// Παράδειγμα βασικής ρύθμισης
var converter = new Converter("sample.one");
```
Αυτό το απόσπασμα δείχνει τη φόρτωση ενός `.one` αρχείο χρησιμοποιώντας το `Converter` κλάση, απαραίτητη για κάθε διαδικασία μετατροπής.

## Οδηγός Εφαρμογής

Ας εμβαθύνουμε τώρα στη μετατροπή αρχείων OneNote σε μορφή Excel. Κάθε ενότητα αναλύει συγκεκριμένα βήματα στην υλοποίηση.

### Φόρτωση του αρχείου προέλευσης
Ξεκινήστε φορτώνοντας το αρχείο προέλευσης OneNote:
```csharp
string sourceFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.one");
```
Αυτή η γραμμή καθορίζει τη διαδρομή προς το `.one` αρχείο, το οποίο θα μετατρέψουμε σε μορφή Excel.

### Καθορισμός επιλογών μετατροπής
Στη συνέχεια, ορίστε επιλογές μετατροπής για τη μορφή στόχου (XLS):
```csharp
var options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Xls };
```
Ο `SpreadsheetConvertOptions` Η κλάση επιτρέπει τον ορισμό διαφόρων παραμέτρων. Εδώ, καθορίζουμε `.xls` ως την επιθυμητή μορφή εξόδου μας.

### Εκτέλεση της μετατροπής
Εκτελέστε τη μετατροπή και αποθηκεύστε το αποτέλεσμα:
```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", ".");
string outputFile = Path.Combine(outputFolder, "one-converted-to.xls");

using (var converter = new Converter(sourceFile))
{
    converter.Convert(outputFile, options);
}
```
Αυτό το απόσπασμα κώδικα χειρίζεται την πραγματική διαδικασία μετατροπής, αξιοποιώντας τις δυνατότητες του GroupDocs.Conversion για να μετατρέψει το αρχείο OneNote σε υπολογιστικό φύλλο Excel.
