## Οδηγός Εφαρμογής

### Μετατροπή POTM σε XLS

Αυτή η λειτουργία μετατρέπει αρχεία POTM σε υπολογιστικά φύλλα Excel (XLS) για ευκολότερο χειρισμό δεδομένων.

#### Βήμα 1: Φόρτωση του αρχείου POTM προέλευσης

Φορτώστε το έγγραφο POTM χρησιμοποιώντας το GroupDocs.Conversion:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;

string inputFilePath = @"YOUR_DOCUMENT_DIRECTORY\\sample.potm";
```

#### Βήμα 2: Διαμόρφωση επιλογών μετατροπής

Ορίστε επιλογές μετατροπής για να καθορίσετε τη μορφή προορισμού (XLS):

```csharp
using GroupDocs.Conversion.Options.Convert;

var options = new SpreadsheetConvertOptions
{
    Format = GroupDocs.Conversion.FileTypes.SpreadsheetFileType.Xls
};
```

#### Βήμα 3: Εκτελέστε τη μετατροπή

Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο XLS:

```csharp
string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "potm-converted-to.xls");

using (var converter = new Converter(inputFilePath))
{
    converter.Convert(outputFile, options);
}
```
