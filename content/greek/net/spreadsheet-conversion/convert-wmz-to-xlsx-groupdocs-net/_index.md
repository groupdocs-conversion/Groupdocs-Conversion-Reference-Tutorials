### Εκτελέστε τη μετατροπή
**Επισκόπηση**Εκτελέστε τη μετατροπή από WMZ σε XLSX και αποθηκεύστε το αρχείο εξόδου.

#### Βήμα 1: Ορισμός διαδρομής εξόδου
```csharp

string outputFile = Path.Combine(outputDirectory, "wmz-converted-to.xlsx");
```

#### Βήμα 2: Μετατροπή και αποθήκευση αρχείου
```csharp

{
    var options = new SpreadsheetConvertOptions();
    // Μετατροπή και αποθήκευση του αρχείου XLSX
    converter.Convert(outputFile, options);
}
```
