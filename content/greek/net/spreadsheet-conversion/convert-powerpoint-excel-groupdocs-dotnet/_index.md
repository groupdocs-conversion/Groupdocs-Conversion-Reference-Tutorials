### Μετατροπή PPS σε XLSX
#### Επισκόπηση
Αυτή η λειτουργία μετατρέπει μια παρουσίαση PowerPoint σε αρχείο Excel, επιτρέποντας πιο αποτελεσματικό χειρισμό και ανάλυση δεδομένων.

#### Βήμα 1: Ορισμός διαδρομών αρχείων
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// Ορίστε διαδρομές αρχείων για μετατροπή
string sourceFilePath = Path.Combine(documentDirectory, "sample.pps");
string outputFile = Path.Combine(outputDirectory, "converted-to.xlsx");
```
*Γιατί αυτό το βήμα;* Ο καθορισμός σαφών καταλόγων εισόδου και εξόδου διασφαλίζει ότι τα αρχεία σας είναι οργανωμένα και εύκολα προσβάσιμα.

#### Βήμα 2: Φόρτωση του αρχείου προέλευσης
```csharp
using (var converter = new Converter(sourceFilePath))
{
    // Αρχικοποιεί τη διαδικασία μετατροπής με το αρχείο PPS πηγής μας.
}
```

#### Βήμα 3: Ορισμός επιλογών μετατροπής
```csharp
var options = new SpreadsheetConvertOptions();
// Διαμορφώστε τις ρυθμίσεις μετατροπής για τη μορφή XLSX.
```
*Γιατί αυτό το βήμα;* Ο `SpreadsheetConvertOptions` Η κλάση καθορίζει ότι στοχεύουμε ένα υπολογιστικό φύλλο Excel ως έξοδο.

#### Βήμα 4: Εκτελέστε τη μετατροπή
```csharp
converter.Convert(outputFile, options);
// Μετατρέπει και αποθηκεύει το αρχείο PPS σε μορφή XLSX στην καθορισμένη διαδρομή.
```
