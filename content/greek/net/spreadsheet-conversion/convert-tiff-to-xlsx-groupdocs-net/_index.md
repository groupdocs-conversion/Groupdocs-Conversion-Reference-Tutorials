#### Βήμα 1: Ορισμός διαδρομών για αρχεία εισόδου και εξόδου
Καθορίστε τη θέση του αρχείου TIFF και πού θέλετε να αποθηκεύσετε το XLSX που έχει μετατραπεί:
```csharp

string inputFile = Path.Combine(documentDirectory, "sample.tiff");
string outputFile = Path.Combine(outputDirectory, "tiff-converted-to.xlsx");
```

#### Βήμα 2: Φόρτωση του αρχείου TIFF χρησιμοποιώντας το GroupDocs.Conversion
Χρησιμοποιήστε το `Converter` κλάση για να φορτώσετε το αρχείο TIFF σας:
```csharp
using (var converter = new Converter(inputFile))
{
    Console.WriteLine("TIFF file loaded successfully.");
}
```

Διαμορφώστε επιλογές προσαρμοσμένες για την έξοδο του Excel, οι οποίες βοηθούν στην προσαρμογή της διαδικασίας μετατροπής:

#### Βήμα 4: Μετατροπή και αποθήκευση του αρχείου XLSX

```csharp

Console.WriteLine($"TIFF to XLSX conversion complete. File saved at {outputFile}");
```
