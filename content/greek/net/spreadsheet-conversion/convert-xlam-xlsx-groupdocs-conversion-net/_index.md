#### Βήμα 2: Μετατροπή XLAM σε XLSX
Φορτώστε το αρχείο XLAM και μετατρέψτε το χρησιμοποιώντας `SpreadsheetConvertOptions`.

```csharp
public static void ConvertXLAMToXlsx()
{
    string outputFolder = "YOUR_OUTPUT_DIRECTORY";
    string outputFile = Path.Combine(outputFolder, "xlam-converted-to.xlsx");

    // Φόρτωση του αρχείου προέλευσης XLAM
    using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_XLAM"))
    {
        // Δημιουργήστε επιλογές μετατροπής για τη μορφή υπολογιστικού φύλλου
        var options = new SpreadsheetConvertOptions();
        
        // Μετατροπή και αποθήκευση ως XLSX
        converter.Convert(outputFile, options);
    }
}
```

- `SpreadsheetConvertOptions`: Προσαρμόστε τις ρυθμίσεις εξόδου, όπως τον τύπο μορφής.
