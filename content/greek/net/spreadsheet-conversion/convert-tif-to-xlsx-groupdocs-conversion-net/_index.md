### Βασική Αρχικοποίηση

Δείτε πώς μπορείτε να ξεκινήσετε με την αρχικοποίηση του μετατροπέα:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

// Ορίστε τη διαδρομή του καταλόγου εξόδου
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "tif-converted-to.xlsx");

// Φορτώστε το αρχείο TIFF πηγής. Αντικαταστήστε το 'YOUR_DOCUMENT_DIRECTORY\sample.tif' με την πραγματική διαδρομή του αρχείου TIFF.
using (var converter = new GroupDocs.Conversion.Converter(@"YOUR_DOCUMENT_DIRECTORY\sample.tif"))
{
    // Ρύθμιση επιλογών μετατροπής για μετατροπή σε μορφή XLSX
    var options = new SpreadsheetConvertOptions();

    // Εκτελέστε τη μετατροπή και αποθηκεύστε το αποτέλεσμα ως αρχείο XLSX στον καθορισμένο κατάλογο
    converter.Convert(outputFile, options);
}
```

Σε αυτό το απόσπασμα κώδικα:
- Ορίζουμε έναν φάκελο εξόδου και προετοιμάζουμε ένα όνομα αρχείου για το αρχείο Excel μας.
- Φορτώστε το αρχείο TIFF χρησιμοποιώντας `GroupDocs.Conversion.Converter`.
- Ορίστε επιλογές μετατροπής ειδικά για υπολογιστικά φύλλα με `SpreadsheetConvertOptions`.


### Εκτέλεση της μετατροπής
Τέλος, εκτελέστε τη μετατροπή περνώντας τη διαδρομή εξόδου και τις επιλογές μετατροπής:

```csharp

```

Αυτή η εντολή μετατρέπει το αρχείο TIFF σε μορφή XLSX και το αποθηκεύει στον καθορισμένο κατάλογο.
