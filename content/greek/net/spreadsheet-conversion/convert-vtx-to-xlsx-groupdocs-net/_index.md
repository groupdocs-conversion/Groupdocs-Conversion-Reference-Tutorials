### Επισκόπηση λειτουργιών μετατροπής VTX σε XLSX
Θα καλύψουμε τον τρόπο μετατροπής ενός αρχείου VTX σε μορφή XLSX. Αυτή η δυνατότητα είναι απαραίτητη για τη μετατροπή σύνθετων διαγραμμάτων σε επεξεργάσιμα υπολογιστικά φύλλα.


#### Βήμα 2: Φόρτωση και μετατροπή του αρχείου VTX
Στη συνέχεια, φορτώστε το αρχείο VTX και εκτελέστε τη μετατροπή:

```csharp
using (var converter = new Converter(Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.vtx")))
{
    // Καθορισμός επιλογών για μετατροπή σε μορφή XLSX
    var options = new SpreadsheetConvertOptions();
    
    // Μετατρέψτε το φορτωμένο έγγραφο και αποθηκεύστε το ως XLSX
    converter.Convert(outputFile, options);
}
```

**Επεξήγηση παραμέτρων:**
- `Converter`: Χειρίζεται τη μετατροπή αρχείων. Την αρχικοποιείτε με τη διαδρομή του αρχείου VTX πηγής.
- `SpreadsheetConvertOptions()`: Ορίζει επιλογές που αφορούν συγκεκριμένα τις μετατροπές υπολογιστικών φύλλων.
