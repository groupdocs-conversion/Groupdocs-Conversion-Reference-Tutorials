### Αρχικοποίηση και Ρύθμιση

Δείτε πώς μπορείτε να αρχικοποιήσετε το GroupDocs.Conversion στο έργο σας C#:

```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main()
    {
        // Ορισμός καταλόγων για αρχεία εισόδου και εξόδου
        string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
        string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

        // Φορτώστε το αρχείο AI πηγής
        using (var converter = new Converter(Path.Combine(documentDirectory, "sample.ai")))
        {
            // Ρύθμιση παραμέτρων επιλογών μετατροπής για τη μορφή XLS
            var options = new SpreadsheetConvertOptions { Format = GroupDocs.Conversion.FileTypes.SpreadsheetFileType.Xls };

            // Ορίστε τη διαδρομή του αρχείου εξόδου και εκτελέστε τη μετατροπή
            string outputFile = Path.Combine(outputDirectory, "ai-converted-to.xls");
            converter.Convert(outputFile, options);
        }
    }
}
```

## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Μετατροπή αρχείου AI σε μορφή XLS

Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε αρχεία Adobe Illustrator (.ai) σε μορφή δυαδικού αρχείου (.xls) του Excel, διευκολύνοντας τον χειρισμό και την ανάλυση γραφικών δεδομένων.

#### Βήμα 1: Φόρτωση του αρχείου AI πηγής

Ξεκινήστε φορτώνοντας το αρχείο προέλευσης χρησιμοποιώντας `GroupDocs.Conversion`:

```csharp
using (var converter = new Converter(Path.Combine(documentDirectory, "sample.ai")))
```

Εδώ, δημιουργούμε ένα παράδειγμα `Converter` αντικείμενο με τη διαδρομή προς το αρχείο AI σας. Αυτό το βήμα είναι κρίσιμο καθώς προετοιμάζει το αρχείο για μετατροπή.

#### Βήμα 2: Διαμόρφωση επιλογών μετατροπής

Στη συνέχεια, διαμορφώστε τις επιλογές μετατροπής για να καθορίσετε την επιθυμητή μορφή εξόδου:

```csharp
var options = new SpreadsheetConvertOptions { Format = GroupDocs.Conversion.FileTypes.SpreadsheetFileType.Xls };
```

Ο `SpreadsheetConvertOptions` Η κλάση σάς επιτρέπει να ορίσετε διάφορες παραμέτρους για τη διαδικασία μετατροπής. Εδώ, την ορίζουμε για να μετατρέψουμε το αρχείο μας σε μορφή XLS.

#### Βήμα 3: Ορισμός διαδρομής αρχείου εξόδου και μετατροπή

Τέλος, ορίστε πού θα αποθηκευτεί το αρχείο που μετατρέψατε και εκτελέστε τη μετατροπή:

```csharp
string outputFile = Path.Combine(outputDirectory, "ai-converted-to.xls");
converter.Convert(outputFile, options);
```

Αυτό το βήμα περιλαμβάνει τον καθορισμό μιας διαδρομής καταλόγου εξόδου και την κλήση `Convert` για να εκτελέσετε τον πραγματικό μετασχηματισμό.
