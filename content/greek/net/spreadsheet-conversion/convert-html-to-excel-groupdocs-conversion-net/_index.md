### Μετατροπή HTML σε XLS

Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε ένα αρχείο HTML σε ένα καλά δομημένο υπολογιστικό φύλλο Excel, ιδανικό για ανάλυση δεδομένων και αναφορά.

#### Βήμα 1: Φόρτωση του αρχείου HTML πηγής

Ξεκινήστε φορτώνοντας το έγγραφο HTML πηγής χρησιμοποιώντας το `Converter` κλάση. Βεβαιωθείτε ότι η διαδρομή προς το αρχείο HTML σας έχει καθοριστεί σωστά:

```csharp
string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.html");
```

#### Βήμα 2: Καθορισμός επιλογών μετατροπής

Ορίστε επιλογές μετατροπής για τη μορφή XLS, οι οποίες υποδεικνύουν στο GroupDocs.Conversion πώς θέλετε να μορφοποιηθεί το αρχείο εξόδου σας:

```csharp
using (var converter = new Converter(inputFilePath))
{
    SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Xls };
}
```

#### Βήμα 3: Εκτελέστε τη μετατροπή

Εκτελέστε τη διαδικασία μετατροπής και αποθηκεύστε το αρχείο Excel που προκύπτει στον καθορισμένο κατάλογο:

```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "html-converted-to.xls");

converter.Convert(outputFile, options);
```
