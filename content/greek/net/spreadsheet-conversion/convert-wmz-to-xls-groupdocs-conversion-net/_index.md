## Ρύθμιση του GroupDocs.Conversion για .NET

Μόλις εγκατασταθεί, ρυθμίστε τη βιβλιοθήκη στο έργο σας:

```csharp
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

// Ορισμός καταλόγων για αρχεία προέλευσης και εξόδου
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY"; // Αντικαταστήστε με τον πραγματικό σας κατάλογο
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// Φόρτωση του αρχείου WMZ από τον καθορισμένο κατάλογο
string documentPath = Path.Combine(documentDirectory, "sample.wmz");

using (var converter = new Converter(documentPath))
{
    // Ορισμός επιλογών μετατροπής για τη μορφή XLS
    var convertOptions = new SpreadsheetConvertOptions
    {
        Format = SpreadsheetFileType.Xls
    };

    // Ορίστε τη διαδρομή εξόδου όπου θα αποθηκευτεί το αρχείο που έχει μετατραπεί
    string outputFile = Path.Combine(outputDirectory, "wmz-converted-to.xls");

    // Εκτελέστε μετατροπή και αποθηκεύστε το αρχείο XLS στην καθορισμένη θέση
    converter.Convert(outputFile, convertOptions);
}
```

Αυτό το απόσπασμα κώδικα αρχικοποιεί ένα `Converter` αντικείμενο με τη διαδρομή αρχείου WMZ, ορίζει επιλογές μετατροπής για τη μορφή XLS και εκτελεί τη μετατροπή. Το αρχείο που έχει μετατραπεί αποθηκεύεται στον καθορισμένο κατάλογο εξόδου.
