### Δυνατότητα: Μετατροπή PDF σε XLS

**Επισκόπηση:** Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε ένα αρχείο PDF σε μορφή υπολογιστικού φύλλου Excel (.xls).

Ξεκινήστε φορτώνοντας το έγγραφο PDF που θέλετε να μετατρέψετε. Βεβαιωθείτε ότι η διαδρομή του αρχείου σας είναι σωστή.

**Γιατί;** Η φόρτωση του εγγράφου το αρχικοποιεί για μετατροπή, προετοιμάζοντας τις απαραίτητες δομές δεδομένων.

#### Βήμα 2: Ορισμός επιλογών μετατροπής
Στη συνέχεια, καθορίστε ότι θέλετε να μετατρέψετε το PDF σας σε μορφή XLS χρησιμοποιώντας `SpreadsheetConvertOptions`.

```csharp
var options = new SpreadsheetConvertOptions { Format = GroupDocs.Conversion.FileTypes.SpreadsheetFileType.Xls };
```
**Γιατί;** Αυτό υποδεικνύει στη μηχανή μετατροπής ποια μορφή εξόδου στοχεύουμε.

#### Βήμα 3: Εκτέλεση μετατροπής
Τέλος, εκτελέστε την πραγματική μετατροπή και αποθηκεύστε το αρχείο σας σε μια καθορισμένη τοποθεσία.

```csharp
string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "pdf-converted-to.xls");
converter.Convert(outputFile, options);
```
**Γιατί;** Αυτό το βήμα ενεργοποιεί τη διαδικασία μετατροπής και εγγράφει την έξοδο στο δίσκο.
