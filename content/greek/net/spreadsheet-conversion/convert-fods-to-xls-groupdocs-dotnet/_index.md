## Οδηγός Εφαρμογής

Ας αναλύσουμε τη διαδικασία μετατροπής σε σαφή βήματα.

### Φόρτωση του αρχείου FODS πηγής

Ξεκινήστε καθορίζοντας καταλόγους για τα αρχεία προέλευσης και εξόδου:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// Φόρτωση του αρχείου προέλευσης FODS
string sourceFilePath = Path.Combine(documentDirectory, "sample.fods");
```

### Ρύθμιση επιλογών μετατροπής

Διαμόρφωση επιλογών για μετατροπή σε μορφή XLS:

```csharp
using GroupDocs.Conversion.Options.Convert;

var converter = new Converter(sourceFilePath);

// Ρύθμιση επιλογών μετατροπής για τη μορφή XLS
SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Xls };
```

### Μετατροπή και αποθήκευση του αρχείου XLS

Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο εξόδου:

```csharp
string outputFile = Path.Combine(outputDirectory, "fods-converted-to.xls");

// Μετατροπή και αποθήκευση του αρχείου XLS
converter.Convert(outputFile, options);

Console.WriteLine("Conversion completed successfully!");
```
