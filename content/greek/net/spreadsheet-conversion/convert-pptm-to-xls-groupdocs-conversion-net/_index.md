### Φόρτωση αρχείου PPTM
Η φόρτωση ενός αρχείου PowerPoint είναι το πρώτο βήμα.

**Βήμα 1: Ορίστε τη διαδρομή αρχείου σας**

```csharp

```
**Βήμα 2: Φόρτωση του αρχείου PPTM**
Δημιουργήστε ένα αντικείμενο μετατροπέα χρησιμοποιώντας το GroupDocs.Conversion:

```csharp
    // Το αντικείμενο μετατροπής είναι έτοιμο για περαιτέρω επεξεργασία.
}
```
### Ρύθμιση παραμέτρων επιλογών μετατροπής
Στη συνέχεια, διαμορφώστε τις ρυθμίσεις για να μετατρέψετε το αρχείο σας σε μορφή XLS.

**Βήμα 1: Ορισμός επιλογών μετατροπής**
Στήνω `SpreadsheetConvertOptions` και καθορίστε τη μορφή εξόδου:
```csharp
using GroupDocs.Conversion.Options.Convert;

var options = new SpreadsheetConvertOptions { Format = FileTypes.SpreadsheetFileType.Xls };
```
### Μετατροπή και αποθήκευση ως XLS
Εκτελέστε τη διαδικασία μετατροπής και αποθηκεύστε το αρχείο σας σε μορφή XLS.

**Βήμα 1: Προετοιμασία ρυθμίσεων εξόδου**
Ορίστε τη θέση του αρχείου εξόδου:
```csharp
using System.IO;

string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "pptm-converted-to.xls");
```
**Βήμα 2: Εκτέλεση μετατροπής και αποθήκευση**
Μετατρέψτε και αποθηκεύστε το αρχείο PPTM ως XLS:
```csharp
using (var converter = new Converter(pptmFilePath))
{
    var options = new SpreadsheetConvertOptions { Format = FileTypes.SpreadsheetFileType.Xls };
    
    // Μετατρέψτε και αποθηκεύστε το αρχείο PPTM ως αρχείο XLS στον καθορισμένο κατάλογο εξόδου
    converter.Convert(outputFile, options);
}
// Η διαδικασία μετατροπής έχει πλέον ολοκληρωθεί.
```
