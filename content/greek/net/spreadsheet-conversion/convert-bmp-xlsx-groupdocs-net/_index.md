## Οδηγός Εφαρμογής
### Χαρακτηριστικό: Φόρτωση αρχείου BMP
Αυτή η λειτουργία προετοιμάζει το αρχείο BMP για μετατροπή. Δείτε πώς μπορείτε να φορτώσετε ένα αρχείο BMP χρησιμοποιώντας το GroupDocs.Conversion.
#### Βήμα 1: Ορισμός της διαδρομής και αρχικοποίηση του μετατροπέα
Αρχικά, ορίστε τη διαδρομή προς το αρχείο BMP:
```csharp
using System.IO;
using GroupDocs.Conversion;

string bmpFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.bmp");

// Αρχικοποιήστε τον μετατροπέα με τη διαδρομή αρχείου BMP
using (var converter = new Converter(bmpFilePath))
{
    // Το αρχείο BMP έχει πλέον φορτωθεί και είναι έτοιμο για μετατροπή.
}
```
**Εξήγηση:**
Εδώ, `Converter` παίρνει τη διαδρομή του αρχείου BMP σας. Αρχικοποιεί το αντικείμενο που θα χειριστεί τη διαδικασία μετατροπής σας.
### Χαρακτηριστικό: Μετατροπή BMP σε XLSX
Τώρα που φορτώθηκε το αρχείο BMP, ας το μετατρέψουμε σε μορφή XLSX.
#### Βήμα 2: Ορισμός διαδρομής εξόδου και επιλογών μετατροπής
Ορίστε πού θέλετε να αποθηκευτεί το αρχείο που έχει μετατραπεί και καθορίστε τις επιλογές μετατροπής:
```csharp
using System.IO;
using GroupDocs.Conversion.Options.Convert;

string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "bmp-converted-to.xlsx");

// Φόρτωση του αρχείου BMP προέλευσης (υποθέτοντας ότι έχει ήδη φορτωθεί)
using (var converter = new Converter(bmpFilePath))
{
    // Ορισμός επιλογών μετατροπής για τη μορφή XLSX
    var options = new SpreadsheetConvertOptions();

    // Μετατροπή και αποθήκευση του αρχείου BMP ως αρχείου XLSX
    converter.Convert(outputFile, options);
}
```
**Εξήγηση:**
`SpreadsheetConvertOptions()` καθορίζει τις ρυθμίσεις που απαιτούνται για τη μετατροπή σε Excel. Το `converter.Convert()` Η μέθοδος εκτελεί τη μετατροπή χρησιμοποιώντας αυτές τις ρυθμίσεις.
