## Οδηγός Εφαρμογής

### Μετατροπή αρχείου OTS σε μορφή XLS

Ακολουθήστε αυτά τα βήματα για να μετατρέψετε ένα αρχείο προτύπου υπολογιστικού φύλλου OpenDocument (OTS) σε μορφή δυαδικού αρχείου Excel (.xls).

#### Βήμα 1: Ορισμός καταλόγων

Ορίστε καταλόγους για τα αρχεία προέλευσης και εξόδου. Αντικαταστήστε τα placeholders με πραγματικές διαδρομές:

```csharp
private const string DocumentDirectory = "YOUR_DOCUMENT_DIRECTORY";
private const string OutputDirectory = "YOUR_OUTPUT_DIRECTORY";
```

#### Βήμα 2: Καθορισμός αρχείων προέλευσης και προορισμού

Ορίστε διαδρομές τόσο για το αρχείο OTS προέλευσης όσο και για το αρχείο εξόδου XLS προορισμού:

```csharp
string sourceOtsFilePath = Path.Combine(DocumentDirectory, "sample.ots");
string outputFile = Path.Combine(OutputDirectory, "ots-converted-to.xls");
```

#### Βήμα 3: Αρχικοποίηση μετατροπέα

Αρχικοποίηση του `Converter` κλάση με τη διαδρομή του αρχείου OTS σας:

```csharp
using (var converter = new GroupDocs.Conversion.Converter(sourceOtsFilePath))
{
    // Η λογική μετατροπής θα εφαρμοστεί εδώ.
}
```

#### Βήμα 4: Διαμόρφωση επιλογών μετατροπής

Ρυθμίστε τις επιλογές μετατροπής για να καθορίσετε τη μορφή XLS χρησιμοποιώντας `SpreadsheetConvertOptions`:

```csharp
var options = new SpreadsheetConvertOptions { Format = GroupDocs.Conversion.FileTypes.SpreadsheetFileType.Xls };
```

#### Βήμα 5: Εκτέλεση μετατροπής

Εκτελέστε τη μετατροπή και αποθηκεύστε το αποτέλεσμα στην καθορισμένη διαδρομή εξόδου, μετατρέποντας το αρχείο OTS σε έγγραφο XLS:

```csharp
converter.Convert(outputFile, options);
```
