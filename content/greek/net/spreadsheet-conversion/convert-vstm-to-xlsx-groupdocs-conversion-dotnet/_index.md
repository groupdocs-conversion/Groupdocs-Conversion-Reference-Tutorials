```csharp

```

#### Βήμα 2: Αρχικοποίηση αντικειμένου μετατροπέα
Φορτώστε το αρχείο VSTM χρησιμοποιώντας το `Converter` κλάση. Αυτό το βήμα προετοιμάζει το αρχείο σας για μετατροπή.

```csharp
using (var converter = new Converter(sourceFilePath))
{
    // Προχωρήστε στη διαμόρφωση των επιλογών μετατροπής.
```

Καθορίστε ότι θέλετε να μετατρέψετε σε μορφή XLSX χρησιμοποιώντας το `SpreadsheetConvertOptions` κλάση. Αυτό ορίζει τις απαραίτητες παραμέτρους για τη μετατροπή.


Εκτελέστε τη μετατροπή και αποθηκεύστε το αποτέλεσμα ως αρχείο XLSX. Βεβαιωθείτε ότι ο κατάλογος εξόδου σας είναι προσβάσιμος για να αποφύγετε τυχόν σφάλματα εγγραφής αρχείου.

```csharp

converter.Convert(outputFile, options);
```
