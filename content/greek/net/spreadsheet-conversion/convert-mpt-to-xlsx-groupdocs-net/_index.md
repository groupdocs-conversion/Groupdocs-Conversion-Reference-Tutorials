## Οδηγός Εφαρμογής

### Μετατροπή MPT σε XLSX

Ακολουθήστε αυτά τα βήματα για να μετατρέψετε ένα αρχείο MPT σε μορφή XLSX χρησιμοποιώντας το GroupDocs.Conversion for .NET.

#### Βήμα 1: Ορισμός διαδρομής εξόδου

Καθορίστε πού θα αποθηκευτούν τα αρχεία που έχετε μετατρέψει:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "mpt-converted-to.xlsx");
```

Εδώ, `outputFile` ορίζει την πλήρη διαδρομή και το όνομα του αρχείου XLSX που προκύπτει.

#### Βήμα 2: Φόρτωση αρχείου MPT

Αρχικοποιήστε το αντικείμενο μετατροπέα με το αρχείο MPT πηγής σας:

```csharp
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY\\sample.mpt"))
{
    // Ο κώδικας μετατροπής θα ακολουθήσει εδώ
}
```
Ο `Converter` Η κλάση χειρίζεται τη φόρτωση και την επεξεργασία αρχείων.

#### Βήμα 3: Ορισμός επιλογών μετατροπής

Καθορίστε επιλογές μετατροπής για να μετατρέψετε το MPT σε αρχείο XLSX:

```csharp
var options = new SpreadsheetConvertOptions();
```
Ο `SpreadsheetConvertOptions` Η κλάση διαμορφώνει τις ρυθμίσεις εξόδου ειδικά για υπολογιστικά φύλλα.

#### Βήμα 4: Εκτέλεση μετατροπής

Εκτελέστε τη μετατροπή και αποθηκεύστε το αποτέλεσμα:

```csharp
converter.Convert(outputFile, options);
```
Αυτή η μέθοδος εκτελεί τον πραγματικό μετασχηματισμό αρχείου με βάση τις καθορισμένες επιλογές σας.
