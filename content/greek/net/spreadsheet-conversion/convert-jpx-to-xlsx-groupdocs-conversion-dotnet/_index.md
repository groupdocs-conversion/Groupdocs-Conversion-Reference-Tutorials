#### Βασική Αρχικοποίηση και Ρύθμιση

Δείτε πώς μπορείτε να αρχικοποιήσετε και να ρυθμίσετε το GroupDocs.Conversion στο έργο σας C#:

```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main(string[] args)
    {
        string sourceFilePath = "sample.jpx"; // Καθορίστε τη διαδρομή προς το αρχείο JPX.
        string outputFolder = "output_directory"; // Ορίστε τη διαδρομή του καταλόγου εξόδου.
        string outputFile = System.IO.Path.Combine(outputFolder, "jpx-converted-to.xlsx"); // Προκύπτουσα τοποθεσία αρχείου XLSX.

        using (var converter = new Converter(sourceFilePath))
        {
            var options = new SpreadsheetConvertOptions(); // Αρχικοποίηση επιλογών μετατροπής για υπολογιστικά φύλλα.
            converter.Convert(outputFile, options); // Μετατρέψτε και αποθηκεύστε το αρχείο JPX ως XLSX.
        }
    }
}
```

Αυτό το απόσπασμα δείχνει τη φόρτωση ενός αρχείου JPX και τη μετατροπή του σε μορφή XLSX χρησιμοποιώντας το GroupDocs.Conversion.
