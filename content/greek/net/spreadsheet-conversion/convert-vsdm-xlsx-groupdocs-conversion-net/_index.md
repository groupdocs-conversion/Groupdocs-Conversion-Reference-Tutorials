## Οδηγός Εφαρμογής

### Φόρτωση και μετατροπή αρχείου VSDM σε XLSX

#### Επισκόπηση

Αυτή η ενότητα περιγράφει λεπτομερώς τον τρόπο φόρτωσης ενός αρχείου Visio Drawing Macro (VSDM) και τη μετατροπή του σε μορφή υπολογιστικού φύλλου Excel.

**1. Ορισμός καταλόγου εξόδου**

Καθορίστε πού θα αποθηκευτούν τα αρχεία που έχουν μετατραπεί:
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "vsdm-converted-to.xlsx");
```

*Εξήγηση:* Ο `outputFolder` Η μεταβλητή περιέχει τη διαδρομή καταλόγου για την αποθήκευση του αρχείου XLSX που έχει μετατραπεί. Βεβαιωθείτε ότι έχει οριστεί σωστά για να αποτρέψετε σφάλματα χρόνου εκτέλεσης.

**2. Φόρτωση αρχείου VSDM**

Φορτώστε το αρχείο προέλευσης του Visio χρησιμοποιώντας το GroupDocs.Conversion:
```csharp
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY/sample.vsdm"))
{
    // Η λογική μετατροπής θα ακολουθήσει εδώ
}
```

*Εξήγηση:* Ο `Converter` Η κλάση αρχικοποιείται με τη διαδρομή του αρχείου σας, προετοιμάζοντάς το για επεξεργασία.

**3. Ρύθμιση επιλογών μετατροπής**

Ρυθμίστε τις παραμέτρους των επιλογών μετατροπής για να καθορίσετε τη μορφή προορισμού:
```csharp
var options = new SpreadsheetConvertOptions();
```

*Εξήγηση:* Αυτό ρυθμίζει τις απαραίτητες παραμέτρους για τη μετατροπή αρχείων σε μορφή XLSX.

**4. Εκτελέστε τη μετατροπή**

Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο εξόδου:
```csharp
converter.Convert(outputFile, options);
```

*Εξήγηση:* Ο `Convert` Η μέθοδος επεξεργάζεται το αρχείο VSDM εισόδου και το αποθηκεύει ως αρχείο XLSX στην καθορισμένη θέση.
