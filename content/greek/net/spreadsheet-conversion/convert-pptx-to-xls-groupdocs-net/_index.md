## Οδηγός Εφαρμογής

Τώρα, ας δούμε πώς να μετατρέψετε ένα αρχείο PPTX σε μορφή XLS.

### Βήμα 1: Ορίστε τη διαδρομή καταλόγου εξόδου

Ξεκινήστε ρυθμίζοντας τον κατάλογο εξόδου όπου θα αποθηκευτούν τα αρχεία που έχουν μετατραπεί. Αντικαταστήστε `YOUR_OUTPUT_DIRECTORY` με την επιθυμητή σας διαδρομή.

```csharp
string outputFolder = @"YOUR_OUTPUT_DIRECTORY\";
```

### Βήμα 2: Ορίστε το όνομα του αρχείου εξόδου

Συνδυάστε τον κατάλογο εξόδου και το όνομα αρχείου για να δημιουργήσετε μια πλήρη διαδρομή για το νέο αρχείο XLS.

```csharp
string outputFile = Path.Combine(outputFolder, "pptx-converted-to.xls");
```

### Βήμα 3: Φόρτωση του αρχείου PPTX πηγής

Φορτώστε την παρουσίαση PowerPoint πηγής από την τοποθεσία της. Βεβαιωθείτε ότι `YOUR_DOCUMENT_DIRECTORY` έχει οριστεί στο σημείο όπου βρίσκεται το αρχείο .pptx σας.

```csharp
using (var converter = new Converter(@"YOUR_DOCUMENT_DIRECTORY\sample-pptx.pptx"))
{
    // Ο κώδικας μετατροπής θα τοποθετηθεί εδώ.
}
```

### Βήμα 4: Διαμόρφωση επιλογών μετατροπής

Ρυθμίστε τις επιλογές μετατροπής που στοχεύουν στη μορφή XLS. Αυτό καθορίζει τον τρόπο με τον οποίο θα πρέπει να δομηθεί η έξοδος.

```csharp
SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = GroupDocs.Conversion.FileTypes.SpreadsheetFileType.Xls };
```

### Βήμα 5: Εκτελέστε τη μετατροπή

Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο που προκύπτει στην καθορισμένη τοποθεσία.

```csharp
csvconverter.Convert(outputFile, options);
Console.WriteLine("Conversion completed successfully.");
```
