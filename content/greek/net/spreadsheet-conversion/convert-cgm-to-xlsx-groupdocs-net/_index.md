## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Μετατροπή CGM σε XLSX

Αυτή η λειτουργία επιτρέπει τη μετατροπή αρχείων CGM στην ευρέως χρησιμοποιούμενη μορφή XLSX, διευκολύνοντας τον χειρισμό και την ανάλυση δεδομένων γραφικών.

#### Βήμα 1: Ορισμός διαδρομών αρχείων

Αρχικά, ρυθμίστε τους καταλόγους εισόδου και εξόδου. Αντικαταστήστε `YOUR_DOCUMENT_DIRECTORY` και `YOUR_OUTPUT_DIRECTORY` με τις κατάλληλες διαδρομές στο σύστημά σας.

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// Διαδρομές για το αρχείο CGM πηγής και το αρχείο XLSX προορισμού
string inputFile = Path.Combine(documentDirectory, "sample.cgm");
string outputFile = Path.Combine(outputDirectory, "cgm-converted-to.xlsx");
```

#### Βήμα 2: Φόρτωση αρχείου προέλευσης

Φορτώστε το αρχείο CGM πηγής χρησιμοποιώντας το GroupDocs.Conversion:

```csharp
using (var converter = new Converter(inputFile))
{
    // Η λογική μετατροπής θα εφαρμοστεί εδώ.
}
```

#### Βήμα 3: Καθορισμός επιλογών μετατροπής

Ορίστε τις επιλογές μετατροπής για τη μορφή XLSX:

```csharp
var options = new SpreadsheetConvertOptions();
```

#### Βήμα 4: Εκτέλεση μετατροπής

Εκτελέστε την πραγματική μετατροπή από CGM σε XLSX:

```csharp
converter.Convert(outputFile, options);
```

Αυτή η διαδικασία θα αποθηκεύσει το αρχείο που μετατράπηκε στον καθορισμένο κατάλογο εξόδου.
