## Οδηγός Εφαρμογής

### Φόρτωση αρχείου παρουσίασης PowerPoint
Ας ξεκινήσουμε φορτώνοντας το αρχείο PPT που θέλετε να μετατρέψετε.

#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς να φορτώσετε μια παρουσίαση PowerPoint χρησιμοποιώντας το GroupDocs.Conversion για .NET, προετοιμάζοντάς την για μετατροπή.

##### Βήμα 1: Ρύθμιση του περιβάλλοντος σας
Ορίστε τη διαδρομή προς το αρχείο PPT εισόδου σας. Αντικαταστήστε `@YOUR_DOCUMENT_DIRECTORY` με τον πραγματικό κατάλογο ή χρησιμοποιήστε ένα σύμβολο κράτησης θέσης.

```csharp
string pptFilePath = "@YOUR_DOCUMENT_DIRECTORY/sample.ppt";
```

##### Βήμα 2: Αρχικοποίηση αντικειμένου μετατροπέα
Χρήση του GroupDocs.Conversion `Converter` κλάση για να φορτώσετε το αρχείο PPT.

```csharp
using (var converter = new GroupDocs.Conversion.Converter(pptFilePath))
{
    // Ο μετατροπέας είναι τώρα έτοιμος για περαιτέρω λειτουργίες όπως η μετατροπή.
}
```

### Μετατροπή PPT σε XLS
Αφού φορτώσουμε την παρουσίαση, ας την μετατρέψουμε σε μορφή υπολογιστικού φύλλου Excel.

#### Επισκόπηση
Αυτή η ενότητα δείχνει πώς μπορείτε να μετατρέψετε ένα αρχείο PPT σε μορφή XLS χρησιμοποιώντας το GroupDocs.Conversion.

##### Βήμα 1: Ορισμός καταλόγου εξόδου και διαδρομής αρχείου
Ρυθμίστε τον κατάλογο εξόδου και καθορίστε πού θα αποθηκευτεί το αρχείο που έχει μετατραπεί.

```csharp
string outputFolder = "@YOUR_OUTPUT_DIRECTORY";
string outputFile = System.IO.Path.Combine(outputFolder, "ppt-converted-to.xls");
```

##### Βήμα 2: Δημιουργία επιλογών μετατροπής
Καθορίστε ότι θέλετε να μετατρέψετε σε μορφή XLS χρησιμοποιώντας `SpreadsheetConvertOptions`.

```csharp
using (var converter = new GroupDocs.Conversion.Converter("@YOUR_DOCUMENT_DIRECTORY/sample.ppt"))
{
    SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = GroupDocs.Conversion.FileTypes.SpreadsheetFileType.Xls };
    
    // Εκτελέστε τη μετατροπή και αποθηκεύστε την στην καθορισμένη τοποθεσία.
    converter.Convert(outputFile, options);
}
```
