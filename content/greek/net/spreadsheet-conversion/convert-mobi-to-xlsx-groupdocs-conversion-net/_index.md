### Φόρτωση και μετατροπή αρχείων MOBI

#### Επισκόπηση
Θα ξεκινήσουμε φορτώνοντας ένα αρχείο MOBI και ρυθμίζοντας τις επιλογές μετατροπής για να το εξαγάγουμε ως έγγραφο XLSX. Αυτή η δυνατότητα σάς επιτρέπει να χειρίζεστε δεδομένα eBook με μεγαλύτερη ευελιξία σε εφαρμογές υπολογιστικών φύλλων όπως το Microsoft Excel.

#### Βήμα προς βήμα εφαρμογή

**1. Ορισμός διαδρομών αρχείων**

Αρχικά, καθορίστε τους καταλόγους για τα αρχεία MOBI εισόδου και τα αρχεία XLSX εξόδου:

```csharp
string documentDirectory = @"C:\Path\To\Your\Documents";
string outputDirectory = @"C:\Path\To\Your\Output";
```

**2. Φορτώστε το αρχείο MOBI πηγής**

Χρησιμοποιήστε το GroupDocs.Conversion για να φορτώσετε το αρχείο προέλευσης:

```csharp
using (var converter = new Converter(Path.Combine(documentDirectory, "sample.mobi")))
{
    // Συνέχεια με τη ρύθμιση της μετατροπής
}
```

Ο `Converter` Η κλάση είναι υπεύθυνη για τον χειρισμό της φόρτωσης αρχείων και την έναρξη μετατροπών.

**3. Ρύθμιση επιλογών μετατροπής**

Ορίστε τη μορφή-στόχο χρησιμοποιώντας τις επιλογές μετατροπής:

```csharp
var options = new SpreadsheetConvertOptions();
string outputFile = Path.Combine(outputDirectory, "mobi-converted-to.xlsx");
```

Εδώ, `SpreadsheetConvertOptions` καθορίζει ότι η έξοδος θα πρέπει να είναι σε μορφή υπολογιστικού φύλλου.

**4. Εκτελέστε τη μετατροπή**

Τέλος, εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο:

```csharp
converter.Convert(outputFile, options);
```

Αυτή η γραμμή μετατρέπει το φορτωμένο αρχείο MOBI σε έγγραφο XLSX και το αποθηκεύει στον καθορισμένο κατάλογο.
