### Ορισμός επιλογών μετατροπής για μορφή XLS

Στη συνέχεια, διαμορφώστε τις επιλογές μετατροπής για να καθορίσετε ότι θέλετε μια έξοδο XLS:

#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών μετατροπής υπολογιστικού φύλλου

```csharp
SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Xls };
```

**Εξήγηση:**
- `SpreadsheetConvertOptions`Αυτή η κλάση επιτρέπει τον καθορισμό διαφόρων ρυθμίσεων μορφής υπολογιστικού φύλλου.
- `Format`: Ορίζει τον τύπο αρχείου προορισμού για μετατροπή.

### Εκτέλεση μετατροπής και αποθήκευση εξόδου

Τέλος, εκτελέστε τη μετατροπή και αποθηκεύστε το αποτέλεσμα:

#### Βήμα 3: Μετατροπή και αποθήκευση του αρχείου

```csharp
string outputFile = "path/to/output/log-converted-to.xls";
converter.Convert(outputFile, options);
```

**Επεξήγηση παραμέτρων:**
- `outputFile`: Η διαδρομή όπου θα αποθηκευτεί το αρχείο που έχει μετατραπεί.
- `options`: Περιέχει ρυθμίσεις μετατροπής.
