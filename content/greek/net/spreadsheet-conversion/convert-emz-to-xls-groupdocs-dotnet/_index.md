## Οδηγός Εφαρμογής

### Μετατροπή EMZ σε XLS

Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε ένα αρχείο Enhanced Windows Metafile Compressed (.emz) σε μορφή δυαδικού αρχείου Microsoft Excel (.xls), διευκολύνοντας τον χειρισμό γραφικών δεδομένων μέσα στο Excel.

#### Βήμα 1: Ορισμός διαδρομών

Ρυθμίστε τις διαδρομές καταλόγου σας χρησιμοποιώντας placeholders για καλύτερη οργάνωση:
```csharp
string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY";

// Διαδρομές αρχείων εισόδου και εξόδου
class PathSetup
{
    public string EmzFilePath => Path.Combine(documentDirectory, "sample.emz");
    public string OutputPath => Path.Combine(outputDirectory, "emz-converted-to.xls");
}
```

#### Βήμα 2: Φόρτωση του αρχείου EMZ προέλευσης

Φορτώστε το αρχείο .emz πηγής χρησιμοποιώντας τη βιβλιοθήκη GroupDocs.Conversion:
```csharp
using (var converter = new GroupDocs.Conversion.Converter(emzFilePath))
{
    // Ρύθμιση παραμέτρων επιλογών μετατροπής
}
```
*Γιατί αυτό το βήμα;*: Η φόρτωση του αρχείου το προετοιμάζει για μετατροπή αρχικοποιώντας ένα `Converter` παράδειγμα.

#### Βήμα 3: Καθορισμός επιλογών μετατροπής

Ρυθμίστε τις παραμέτρους των επιλογών για να καθορίσετε ότι θέλετε μια έξοδο XLS:
```csharp
var options = new SpreadsheetConvertOptions { Format = GroupDocs.Conversion.FileTypes.SpreadsheetFileType.Xls };
```
*Γιατί αυτό το βήμα;*Αυτή η ρύθμιση υποδεικνύει στον μετατροπέα την επιθυμητή μορφή αρχείου, διασφαλίζοντας την ορθή μετατροπή.

#### Βήμα 4: Εκτελέστε τη μετατροπή

Εκτελέστε τη μετατροπή και αποθηκεύστε την έξοδο:
```csharp
converter.Convert(outputPath, options);
```
*Γιατί αυτό το βήμα;*: Εκτελεί τη διαδικασία μετατροπής και αποθηκεύει το αποτέλεσμα στην καθορισμένη διαδρομή.
