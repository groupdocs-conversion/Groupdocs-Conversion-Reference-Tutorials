## Οδηγός Εφαρμογής

Αφού εγκαταστήσουμε το GroupDocs.Conversion, ας αναλύσουμε την υλοποίηση στη φόρτωση ενός αρχείου PST και στη μετατροπή του σε XLSX.

### Φόρτωση αρχείου PST

#### Επισκόπηση
Η φόρτωση ενός αρχείου PST είναι το πρώτο βήμα της μετατροπής. Αυτή η διαδικασία ελέγχει εάν η παρεχόμενη διαδρομή οδηγεί σε ένα έγκυρο αρχείο PST και το προετοιμάζει για μετατροπή.

**Βήμα 1: Έλεγχος τύπου αρχείου**

```csharp
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Load;

string samplePstPath = @"C:\\path\\to\\your\\sample.pst";
LoadOptions loadOptions = null;

// Επαληθεύστε αν το αρχείο είναι τύπου PST.
if (new Converter(samplePstPath, () => new PersonalStorageLoadOptions()).FileType == FileType.Pst)
{
    loadOptions = new PersonalStorageLoadOptions();
}
```

**Εξήγηση**: Αυτό το απόσπασμα κώδικα ελέγχει τον τύπο αρχείου χρησιμοποιώντας `PersonalStorageLoadOptions`Εάν επιβεβαιωθεί ως PST, ορίζονται οι κατάλληλες επιλογές φόρτωσης.

### Μετατροπή PST σε XLSX

#### Επισκόπηση
Αφού φορτώσετε το αρχείο PST, προχωρήστε στη μετατροπή των περιεχομένων του σε μορφή XLSX καθορίζοντας παραμέτρους μετατροπής και εκτελώντας τη διαδικασία.

**Βήμα 2: Ορισμός επιλογών μετατροπής**

```csharp
using System.IO;
using GroupDocs.Conversion.Options.Convert;

string outputFolder = @"C:\\path\\to\\output";
string outputFileTemplate = Path.Combine(outputFolder, "pst-converted-{0}-to.xlsx");
int counter = 1;

var options = new SpreadsheetConvertOptions();
```

**Εξήγηση**: Αυτό ορίζει τον φάκελο εξόδου και το πρότυπο ονομασίας αρχείου. `SpreadsheetConvertOptions` καθορίζει τη μετατροπή σε υπολογιστικό φύλλο Excel.

**Βήμα 3: Εκτέλεση μετατροπής**

```csharp
using (var converter = new Converter(samplePstPath, loadOptions))
{
    // Εκτελέστε τη διαδικασία μετατροπής.
    converter.Convert(
        (SaveContext saveContext) => new FileStream(string.Format(outputFileTemplate, counter++), FileMode.Create),
        options
    );
}
```

**Εξήγηση**Αυτό το απόσπασμα αρχικοποιεί ένα `Converter` παράδειγμα με το αρχείο PST και τις επιλογές φόρτωσης. Εκτελεί τη μετατροπή χρησιμοποιώντας καθορισμένες επιλογές και αποθηκεύει το αποτέλεσμα ως αρχείο XLSX.
