## Οδηγός Εφαρμογής

### Φόρτωση αρχείου OST

Φορτώστε ένα αρχείο OST χρησιμοποιώντας ένα υπό όρους περιβάλλον για να διασφαλίσετε ότι θα γίνει επεξεργασία μόνο των αρχείων OST:

#### Βήμα 1: Ορίστε τη διαδρομή του αρχείου προέλευσης
Καθορίστε τη διαδρομή προς το αρχείο OST:
```csharp
string sourceFilePath = "YOUR_DOCUMENT_DIRECTORY\sample.ost"; // Αντικαταστήστε με τη διαδρομή του αρχείου OST
```

#### Βήμα 2: Φόρτωση του αρχείου OST χρησιμοποιώντας ένα υπό όρους περιβάλλον
Φορτώστε το OST χρησιμοποιώντας συγκεκριμένο περιβάλλον για αρχεία email:
```csharp
var converter = new Converter(sourceFilePath, (loadContext) => 
{
    if (loadContext.SourceFormat == EmailFileType.Ost)
    {
        return new PersonalStorageLoadOptions();
    }
    return null;
});
```

- **Γιατί:** Αυτό διασφαλίζει ότι υποβάλλονται σε επεξεργασία μόνο αρχεία OST, αποφεύγοντας σφάλματα χρόνου εκτέλεσης από μη υποστηριζόμενες μορφές.

### Μετατροπή OST σε DOCX

Μετατρέψτε το φορτωμένο αρχείο OST σε μορφή DOCX για ευκολότερη πρόσβαση και επεξεργασία:

#### Βήμα 1: Ορισμός διαδρομής αρχείου εξόδου
Ορίστε πού θα αποθηκευτεί το DOCX που έχει μετατραπεί:
```csharp
string outputFilePath = "YOUR_OUTPUT_DIRECTORY\ost-converted.docx"; // Αντικαταστήστε με την επιθυμητή διαδρομή εξόδου
```

#### Βήμα 2: Μετατροπή και αποθήκευση του εγγράφου
Εκτελέστε τη μετατροπή χρησιμοποιώντας τις επιλογές GroupDocs.Conversion:
```csharp
var options = new WordProcessingConvertOptions();
counter.Convert(new FileStream(outputFilePath, FileMode.Create), options);
```

- **Επεξήγηση παραμέτρων:** `WordProcessingConvertOptions` ορίζει τον τρόπο μορφοποίησης του DOCX εξόδου.

### Συμβουλές αντιμετώπισης προβλημάτων

- Βεβαιωθείτε ότι η διαδρομή OST πηγής σας είναι σωστή για να αποφύγετε σφάλματα "το αρχείο δεν βρέθηκε".
- Επαληθεύστε ότι το GroupDocs.Conversion έκδοση 25.3.0 ή νεότερη είναι εγκατεστημένο στο έργο σας.
