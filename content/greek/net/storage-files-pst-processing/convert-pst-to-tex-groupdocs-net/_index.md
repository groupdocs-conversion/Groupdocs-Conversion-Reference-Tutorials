## Οδηγός Εφαρμογής
Θα χωρίσουμε την υλοποίηση σε δύο κύρια χαρακτηριστικά: τη φόρτωση ενός αρχείου PST και τη μετατροπή του σε μορφή TEX.

### Χαρακτηριστικό 1: Φόρτωση αρχείου PST
Αυτή η λειτουργία δείχνει πώς να φορτώσετε ένα αρχείο PST χρησιμοποιώντας το GroupDocs.Conversion. Η φόρτωση αρχείων με συγκεκριμένες επιλογές μπορεί να είναι κρίσιμη για την αποτελεσματική διαχείριση μεγάλων συνόλων δεδομένων.

#### Βήμα προς βήμα εφαρμογή:
**3.1 Ορισμός της διαδρομής εγγράφου**
Ορίστε τη διαδρομή προς το αρχείο PST, διασφαλίζοντας ότι είναι προσβάσιμο από την εφαρμογή σας.
```csharp
string documentPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.pst");
```

**3.2 Αρχικοποίηση επιλογών φόρτωσης**
Χρήση `PersonalStorageLoadOptions` για φόρτωση επιλογών προσαρμοσμένων σε αρχεία PST.
```csharp
var loadOptions = new PersonalStorageLoadOptions();
```

**3.3 Δημιουργία αντικειμένου μετατροπέα**
Δημιουργήστε ένα αντικείμενο μετατροπέα με συγκεκριμένες συνθήκες περιβάλλοντος χρησιμοποιώντας εκφράσεις λάμδα.
```csharp
using (var converter = new GroupDocs.Conversion.Converter(documentPath, context => 
    context.SourceFormat == GroupDocs.Conversion.FileTypes.EmailFileType.Ost ? loadOptions : null))
{
    // Ο μετατροπέας είναι τώρα έτοιμος για περαιτέρω λειτουργίες όπως η μετατροπή.
}
```

**3.4 Επεξήγηση**
- **`documentPath`**: Κατευθύνει την εφαρμογή στη θέση του αρχείου PST σας.
- **Λειτουργία λάμδα**: Εξασφαλίζει τη σωστή `loadOptions` εφαρμόζονται με βάση τον τύπο αρχείου.

### Χαρακτηριστικό 2: Μετατροπή αρχείου PST σε μορφή TEX
Στη συνέχεια, θα μετατρέψουμε το φορτωμένο αρχείο PST σε μορφή TEX. Αυτό είναι ιδιαίτερα χρήσιμο για τη δημιουργία μορφοποιημένων εγγράφων από δεδομένα email που είναι αποθηκευμένα σε αρχεία PST.

#### Βήμα προς βήμα εφαρμογή:
**4.1 Ορισμός καταλόγου εξόδου**
Ορίστε πού θα αποθηκευτούν τα αρχεία που έχουν μετατραπεί.
```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", "output");
string outputFileTemplate = Path.Combine(outputFolder, "pst-converted-{0}-to.tex");
```

**4.2 Αρχικοποίηση Μετατροπέα και Επιλογών Μετατροπής**
Φορτώστε το αρχείο PST σε ένα αντικείμενο μετατροπέα και καθορίστε τις επιλογές μετατροπής για τη μορφή TEX.
```csharp
using (var converter = new GroupDocs.Conversion.Converter(Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.pst")))
{
    PageDescriptionLanguageConvertOptions options = new PageDescriptionLanguageConvertOptions { Format = PageDescriptionLanguageFileType.Tex };
    
    // Εκτελέστε τη μετατροπή
    converter.Convert(
        (SaveContext saveContext) => new FileStream(string.Format(outputFileTemplate, counter++), FileMode.Create),
        options
    );
}
```

**4.3 Επεξήγηση**
- **`outputFolder`**: Διατηρεί τη διαδρομή για την αποθήκευση των αρχείων που έχουν μετατραπεί.
- **Επιλογές μορφής TEX**: Ρυθμίζει τον τύπο και τη δομή του αρχείου εξόδου.
