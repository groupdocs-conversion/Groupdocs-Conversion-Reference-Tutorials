## Οδηγός Εφαρμογής

Ας χωρίσουμε την υλοποίηση σε λογικά μέρη.

### Λειτουργία 1: Φόρτωση αρχείου PST

Η σωστή φόρτωση των αρχείων PST είναι ζωτικής σημασίας για ακριβείς μετατροπές.

#### Επισκόπηση

Αυτή η λειτουργία σάς επιτρέπει να φορτώσετε τα αρχεία PST για μετατροπή.

#### Βήμα προς βήμα εφαρμογή

**Αρχικοποίηση και διαμόρφωση**

```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Load;

string sourcePstPath = @"YOUR_DOCUMENT_DIRECTORY\sample.pst";
var converter = new Converter(sourcePstPath, (LoadContext loadContext) =>
{
    // Βεβαιωθείτε ότι το αρχείο είναι τύπου PST πριν εφαρμόσετε συγκεκριμένες επιλογές.
    return loadContext.SourceFormat == EmailFileType.Pst ? new PersonalStorageLoadOptions() : null;
});
```

- **Παράμετροι**: `sourcePstPath` καθορίζει τη θέση του αρχείου PST προέλευσης.
- **Επιστρεφόμενες τιμές**: Το `PersonalStorageLoadOptions()` ρυθμίζει τη φόρτωση με βάση τις προδιαγραφές PST.

### Λειτουργία 2: Μετατροπή σε PDF

Η μετατροπή του φορτωμένου αρχείου PST σε ένα φιλικό προς το χρήστη έγγραφο PDF διευκολύνει την κοινή χρήση και την αρχειοθέτηση δεδομένων email.

#### Επισκόπηση

Αυτή η μετατροπή διευκολύνει την καλύτερη προσβασιμότητα και την κοινή χρήση δεδομένων.

#### Βήμα προς βήμα εφαρμογή

**Εκτελέστε τη μετατροπή**

```csharp
using System.IO;
using GroupDocs.Conversion.Options.Convert;

string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
string outputFilePattern = Path.Combine(outputFolder, "pst-converted-{0}-to.pdf");
var options = new PdfConvertOptions();
int counter = 1;

// Μετατρέψτε και αποθηκεύστε το αρχείο PST ως PDF.
converter.Convert(
    (SaveContext saveContext) => new FileStream(string.Format(outputFilePattern, counter++), FileMode.Create),
    options
);
```

- **Παράμετροι**: `outputFolder` είναι το σημείο όπου θα αποθηκευτούν τα αρχεία που έχουν μετατραπεί. Το `outputFilePattern` διασφαλίζει ότι κάθε αρχείο έχει ένα μοναδικό όνομα.
- **Επιστρεφόμενες τιμές**: Ένα έγγραφο PDF που δημιουργείται για κάθε διαδικασία μετατροπής.
