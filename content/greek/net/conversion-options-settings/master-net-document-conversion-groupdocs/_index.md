## Οδηγός Εφαρμογής

### Αντικατάσταση γραμματοσειράς

Η αντικατάσταση γραμματοσειράς είναι ζωτικής σημασίας όταν οι αρχικές γραμματοσειρές σας δεν είναι διαθέσιμες στο περιβάλλον προορισμού. Δείτε πώς μπορείτε να την εφαρμόσετε:

#### Ορισμός επιλογών φόρτωσης για αντικατάσταση γραμματοσειράς

Ξεκινήστε ορίζοντας επιλογές φόρτωσης που καθορίζουν προεπιλεγμένες γραμματοσειρές και αντικαταστάσεις:
```csharp
using System;
using System.Collections.Generic;
using GroupDocs.Conversion.Options.Load;

Func<LoadContext, LoadOptions> getLoadOptions = loadContext => new SpreadsheetLoadOptions
{
    DefaultFont = "Helvetica",
    FontSubstitutes = new List<FontSubstitute>
    {
        FontSubstitute.Create("Tahoma", "Arial"),
        FontSubstitute.Create("Times New Roman", "Arial")
    },
    OnePagePerSheet = true
};
```

**Εξήγηση**: 
- `DefaultFont`: Ορίζει την προεπιλεγμένη γραμματοσειρά για ολόκληρο το έγγραφο.
- `FontSubstitutes`: Καθορίζει ποιες γραμματοσειρές θα αντικατασταθούν με εναλλακτικές.
- `OnePagePerSheet`: Εξασφαλίζει ότι κάθε φύλλο υπολογιστικού φύλλου αποδίδεται σε μία μόνο σελίδα PDF.

#### Συμβουλές αντιμετώπισης προβλημάτων
- **Λείπουν γραμματοσειρές**Βεβαιωθείτε ότι οι αντικαταστάσεις σας καλύπτουν όλες τις πιθανές γραμματοσειρές που χρησιμοποιούνται στα έγγραφά σας.
- **Προβλήματα απόδοσης**Επαληθεύστε ότι οι γραμματοσειρές που έχουν αντικατασταθεί είναι εγκατεστημένες και υποστηρίζονται από το περιβάλλον σας.

### Μετατροπή σε PDF με επιλογές για προχωρημένους

Στη συνέχεια, ας διαμορφώσουμε τη διαδικασία μετατροπής για να δημιουργήσουμε ένα PDF με σωστή μορφοποίηση:
```csharp
using System.IO;
using GroupDocs.Conversion.Contracts;
using GroupDocs.Conversion.Options.Convert;

string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "converted.pdf");

using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_XLSX", getLoadOptions))
{
    PdfConvertOptions options = new PdfConvertOptions();
    converter.Convert(outputFile, options);
}
```

**Εξήγηση**: 
- `outputFolder` και `outputFile`: Ορίστε πού θα αποθηκευτεί το PDF που έχει μετατραπεί.
- `PdfConvertOptions()`Αρχικοποιεί τις επιλογές για τη μετατροπή σε PDF.

#### Βασικές επιλογές διαμόρφωσης
- **Προσαρμοσμένο μέγεθος σελίδας**: Χρήση `options.PageSize = PageSize.A4;` για να ορίσετε προσαρμοσμένα μεγέθη σελίδων.
- **Ρυθμίσεις ασφαλείας**: Εφαρμογή ρυθμίσεων ασφαλείας όπως κρυπτογράφηση χρησιμοποιώντας `options.EncryptionSettings`.
