## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Ρύθμιση καταλόγων για μετατροπή

Αυτή η λειτουργία διασφαλίζει ότι τα έγγραφα και οι κατάλογοι εξόδου σας είναι έτοιμοι για μετατροπές.

#### Δημιουργία διαδρομών καταλόγου

Ξεκινήστε ορίζοντας τις διαδρομές για τους φακέλους προέλευσης και προορισμού:

```csharp
using System.IO;

string documentDirectory = "YOUR_DOCUMENT_DIRECTORY"; // Διαδρομή όπου αποθηκεύονται τα αρχεία προέλευσης
string outputDirectory = "YOUR_OUTPUT_DIRECTORY"; // Κατάλογος για την αποθήκευση των αρχείων που έχουν μετατραπεί
```

#### Διασφάλιση ύπαρξης καταλόγων

Για να αποφύγετε σφάλματα χρόνου εκτέλεσης, βεβαιωθείτε ότι υπάρχουν αυτοί οι κατάλογοι ή δημιουργήστε τους εάν δεν υπάρχουν:

```csharp
if (!Directory.Exists(documentDirectory))
{
    Directory.CreateDirectory(documentDirectory);
}

if (!Directory.Exists(outputDirectory))
{
    Directory.CreateDirectory(outputDirectory);
}
```

**Εξήγηση:**
- `Directory.Exists`: Ελέγχει την ύπαρξη ενός καταλόγου.
- `Directory.CreateDirectory`: Δημιουργεί έναν νέο κατάλογο εάν δεν υπάρχει.

### Συμβουλές αντιμετώπισης προβλημάτων

- Βεβαιωθείτε ότι έχετε τα σωστά δικαιώματα διαδρομής για να αποφύγετε σφάλματα πρόσβασης.
- Επαληθεύστε τις μορφές διαδρομής που αφορούν συγκεκριμένα το περιβάλλον (π.χ., Windows έναντι Unix).
