## Οδηγός Εφαρμογής

### Μετατροπή εγγράφου από ροή
#### Επισκόπηση
Αυτή η λειτουργία επιδεικνύει τη μετατροπή ενός εγγράφου από ροή εισόδου σε αρχείο PDF. Είναι ιδιαίτερα χρήσιμη όταν πρόκειται για αρχεία που έχουν μεταφορτωθεί μέσω εφαρμογών ιστού.

#### Βήμα προς βήμα εφαρμογή
**1. Ρύθμιση του καταλόγου εξόδου**
Ορίστε πού θα αποθηκευτεί το PDF που έχετε μετατρέψει:
```csharp
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputDirectory, "converted.pdf");
```

**2. Ορίστε την πηγή ροής εισόδου**
Εδώ, χρησιμοποιούμε μια μέθοδο `GetFileStream` για να παρέχετε μια ροή από ένα αρχείο:
```csharp
Func<Stream> inputStream = GetFileStream;
```

**3. Αρχικοποίηση και ρύθμιση παραμέτρων του μετατροπέα**
Αρχικοποιήστε τον μετατροπέα με τη ροή εισόδου και διαμορφώστε τον για μετατροπή PDF:
```csharp
using (Converter converter = new Converter(inputStream))
{
    PdfConvertOptions options = new PdfConvertOptions();
    converter.Convert(outputFile, options);
}
```

**4. Υλοποίηση `GetFileStream` Μέθοδος**
Αυτή η μέθοδος διαβάζει ένα δείγμα αρχείου DOCX σε μια ροή:
```csharp
private static Stream GetFileStream() => 
    File.OpenRead("YOUR_DOCUMENT_DIRECTORY/SAMPLE_DOCX");
```

### Μετατροπή εγγράφου από το MemoryStream
#### Επισκόπηση
Μετατροπή εγγράφων από ένα `MemoryStream`, ιδανικό για σενάρια όπου τα αρχεία υποβάλλονται σε επεξεργασία στη μνήμη.

#### Βήμα προς βήμα εφαρμογή
**1. Ρύθμιση του καταλόγου εξόδου**
Όπως και με τη μετατροπή ροής, ορίστε τη διαδρομή εξόδου σας:
```csharp
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputDirectory, "converted.pdf");
```

**2. Ορίστε την πηγή ροής εισόδου**
Χρήση `GetMemoryStream` να παρέχει ένα `MemoryStream` που περιέχει δεδομένα αρχείου:
```csharp
Func<Stream> inputStream = GetMemoryStream;
```

**3. Αρχικοποίηση και ρύθμιση παραμέτρων του μετατροπέα**
Αρχικοποιήστε με τη ροή μνήμης και ορίστε τις επιλογές μετατροπής PDF:
```csharp
using (Converter converter = new Converter(inputStream))
{
    PdfConvertOptions options = new PdfConvertOptions();
    converter.Convert(outputFile, options);
}
```

**4. Υλοποίηση `GetMemoryStream` Μέθοδος**
Ανάγνωση ενός δείγματος αρχείου TXT σε ένα `MemoryStream`:
```csharp
private static Stream GetMemoryStream()
{
    MemoryStream memStream = new MemoryStream();

    using (FileStream fStream = File.Open("YOUR_DOCUMENT_DIRECTORY/SAMPLE_TXT", FileMode.Open))
    {
        fStream.CopyTo(memStream);
    }
    return memStream;
}
```
