### Φόρτωση και μετατροπή αρχείων WMZ

**Επισκόπηση:** Αυτή η λειτουργία σάς επιτρέπει να φορτώσετε ένα αρχείο WMZ και να το μετατρέψετε σε μορφή TEX, αξιοποιώντας τις αποτελεσματικές δυνατότητες μετατροπής του GroupDocs.

#### Βήμα 1: Ορισμός καταλόγου εξόδου και ονόματος αρχείου

```csharp
string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "wmz-converted-to.tex");
```

Αντικαθιστώ `YOUR_OUTPUT_DIRECTORY` με την πραγματική διαδρομή καταλόγου σας για να ορίσετε την τοποθεσία του αρχείου μετατροπής.

#### Βήμα 2: Φόρτωση και μετατροπή αρχείου WMZ

```csharp
using (var converter = new Converter(@"YOUR_DOCUMENT_DIRECTORY\sample.wmz"))
{
    PageDescriptionLanguageConvertOptions options = new PageDescriptionLanguageConvertOptions 
    {
        Format = PageDescriptionLanguageFileType.Tex
    };
    
    // Εκτελέστε τη μετατροπή
    converter.Convert(outputFile, options);
}
```

Φορτώστε το αρχείο WMZ από `YOUR_DOCUMENT_DIRECTORY` και καθορίστε το TEX ως τη μορφή στόχου σας χρησιμοποιώντας τις επιλογές μετατροπής. `converter.Convert()` Η μέθοδος χειρίζεται τον μετασχηματισμό.

**Επεξήγηση παραμέτρων:**
- **επιλογές.Μορφή:** Καθορίζει τη μορφή στόχου (TEX σε αυτήν την περίπτωση).
- **Αρχείο εξόδου:** Διαδρομή όπου θα αποθηκευτεί το αρχείο που έχει μετατραπεί.
