## Οδηγός Εφαρμογής

### Ανάκτηση πιθανών μετατροπών

#### Επισκόπηση
Ανακαλύψτε το εύρος των μορφών στις οποίες μπορείτε να μετατρέψετε ένα έγγραφο χρησιμοποιώντας το GroupDocs.Conversion.

#### Βήμα προς βήμα εφαρμογή
**Ανάκτηση επιλογών μετατροπής**
```csharp
using System;
using GroupDocs.Conversion.Contracts;

string inputFile = "YOUR_DOCUMENT_DIRECTORY/sample.docx";
var possibleConversions = Converter.GetPossibleConversions(inputFile);

foreach (var conversion in possibleConversions)
{
    Console.WriteLine(conversion.Format);
}
```
**Εξήγηση**Αυτό το απόσπασμα ανακτά όλες τις μορφές στις οποίες μπορεί να μετατραπεί ένα αρχείο DOCX, χρησιμοποιώντας το `GetPossibleConversions` μέθοδος που επιστρέφει έναν πίνακα επιλογών μετατροπής.

### Ρύθμιση παραμέτρων επιλογών φόρτωσης για έγγραφα Word

#### Επισκόπηση
Μάθετε πώς να χειρίζεστε με ασφάλεια έγγραφα που προστατεύονται με κωδικό πρόσβασης.

#### Βήμα προς βήμα εφαρμογή
**Ρύθμιση προστασίας με κωδικό πρόσβασης**
```csharp
using System;
using GroupDocs.Conversion.Options.Load;

string inputFile = "YOUR_DOCUMENT_DIRECTORY/sample.docx";
var loadOptions = (WordProcessingLoadOptions) Converter.GetPossibleConversions(inputFile).LoadOptions;
loadOptions.Password = "12345";

Console.WriteLine("Password set in load options: {0}", loadOptions.Password);
```
**Εξήγηση**: Εδώ, διαμορφώνουμε `WordProcessingLoadOptions` για να καθορίσετε έναν κωδικό πρόσβασης για προστατευμένα έγγραφα. Αυτό διασφαλίζει ότι μόνο εξουσιοδοτημένοι χρήστες έχουν πρόσβαση στο περιεχόμενο.

### Μετατροπή εγγράφου σε μορφή PDF

#### Επισκόπηση
Μετατρέψτε έγγραφα Word σε PDF εύκολα με προσαρμοσμένες ρυθμίσεις μετατροπής.

#### Βήμα προς βήμα εφαρμογή
**Μετατροπή σε PDF**
```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

string inputFile = "YOUR_DOCUMENT_DIRECTORY/sample.docx";
string outputFolder = "path/to/output/directory";
string outputFile = Path.Combine(outputFolder, "converted.pdf");

var loadOptions = new WordProcessingLoadOptions
{
    Password = "12345"
};

using (Converter converter = new Converter(inputFile, () => loadOptions))
{
    var convertOptions = new PdfConvertOptions();
    converter.Convert(outputFile, convertOptions);
}
```
**Εξήγηση**Αυτός ο κώδικας δείχνει τη μετατροπή ενός αρχείου DOCX σε PDF. Αρχικοποιεί το `Converter` κλάση με επιλογές εισόδου και φόρτωσης, και στη συνέχεια εκτελεί μετατροπή χρησιμοποιώντας `PdfConvertOptions`.
