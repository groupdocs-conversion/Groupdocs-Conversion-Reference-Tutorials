## Οδηγός Εφαρμογής

Ακολουθήστε αυτά τα βήματα για να μετατρέψετε ένα έγγραφο Word που προστατεύεται με κωδικό πρόσβασης σε PDF.

### Φόρτωση εγγράφου που προστατεύεται με κωδικό πρόσβασης

#### Βήμα 1: Ρύθμιση επιλογών φόρτωσης
```csharp
var loadOptions = new LoadOptions();
loadOptions.Password = "your-password"; // Αντικατάσταση με τον πραγματικό κωδικό πρόσβασης
```

#### Βήμα 2: Αρχικοποίηση μετατροπέα
```csharp
using (Converter converter = new Converter("sample.docx", () => loadOptions))
{
    // Η λογική μετατροπής ισχύει εδώ
}
```
*Σημείωμα*: Χρησιμοποιήστε την έκφραση λάμδα για να περάσετε `loadOptions` επειδή εφαρμόζει `IDisposable`.

### Μετατροπή σε PDF

#### Βήμα 3: Ρύθμιση παραμέτρων επιλογών αποθήκευσης
```csharp
var convertOptions = new PdfConvertOptions();
```

#### Βήμα 4: Εκτέλεση μετατροπής
```csharp
converter.Convert("output.pdf", convertOptions);
```
*Εξήγηση*: Το `Converter.Convert` Η μέθοδος μετατρέπει το φορτωμένο αρχείο Word σε PDF χρησιμοποιώντας καθορισμένες ρυθμίσεις μετατροπής.

### Συμβουλές αντιμετώπισης προβλημάτων
- **Μη έγκυρος κωδικός πρόσβασης**: Ελέγξτε ξανά τον κωδικό πρόσβασης για τη φόρτωση εγγράφων.
- **Μη υποστηριζόμενες μορφές**Βεβαιωθείτε ότι η μορφή του εγγράφου σας υποστηρίζεται από το GroupDocs.Conversion για .NET.
