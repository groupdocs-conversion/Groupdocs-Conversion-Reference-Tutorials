## Ρύθμιση του GroupDocs.Conversion για .NET

Μόλις εγκαταστήσετε το GroupDocs.Conversion, αρχικοποιήστε το στο έργο σας. Δείτε πώς:

### Βασική Αρχικοποίηση και Ρύθμιση με C#

Αρχικά, εισαγάγετε τους απαραίτητους χώρους ονομάτων:
```csharp
using System;
using System.IO;
using GroupDocs.Conversion.Options.Load;
using GroupDocs.Conversion.Contracts;
using GroupDocs.Conversion.FileTypes;
using GroupDocs.Conversion;
```

Ορίστε τις διαδρομές αρχείων και τις επιλογές φόρτωσης για έγγραφα που προστατεύονται με κωδικό πρόσβασης:
```csharp
string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "SAMPLE_DOCX_WITH_PASSWORD");
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "converted.ppt");

Func<LoadContext, LoadOptions> getLoadOptions = loadContext => new WordProcessingLoadOptions
{
    Password = "12345" // Αντικαταστήστε με τον κωδικό πρόσβασης του εγγράφου σας
};
```

## Οδηγός Εφαρμογής

### Μετατροπή εγγράφου που προστατεύεται με κωδικό πρόσβασης σε παρουσίαση

#### Επισκόπηση

Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε συγκεκριμένες σελίδες ενός αρχείου DOCX που προστατεύεται με κωδικό πρόσβασης σε παρουσιάσεις PPT, ιδανικές για την εξαγωγή βασικών πληροφοριών.

#### Βήμα προς βήμα εφαρμογή

**1. Αρχικοποίηση του μετατροπέα**
Δημιουργήστε μια παρουσία του `Converter` κλάση με τη διαδρομή αρχείου εισόδου και τις επιλογές φόρτωσης:
```csharp
using (Converter converter = new Converter(inputFilePath, getLoadOptions))
{
    // Ο κώδικας για τη μετατροπή μπαίνει εδώ
}
```

**2. Ορίστε Επιλογές Μετατροπής**
Ρυθμίστε τις επιλογές μετατροπής για να καθορίσετε ποια σελίδα θέλετε να μετατρέψετε και τη μορφή εξόδου:
```csharp
PresentationConvertOptions options = new PresentationConvertOptions
{
    PageNumber = 2,
    PagesCount = 1,
    Format = PresentationFileType.Ppt // Καθορίστε το PPT ως την επιθυμητή μορφή
};
```

**3. Εκτέλεση μετατροπής**
Εκτελέστε τη μετατροπή καλώντας `converter.Convert()` και αποθηκεύστε την έξοδο:
```csharp
converter.Convert(outputFile, options);
```
Αυτό μετατρέπει τη δεύτερη σελίδα του αρχείου DOCX σε μια παρουσίαση PowerPoint.

#### Συμβουλές αντιμετώπισης προβλημάτων
- **Μη έγκυρος κωδικός πρόσβασης**: Βεβαιωθείτε ότι έχετε εισάγει τον σωστό κωδικό πρόσβασης `WordProcessingLoadOptions`.
- **Αριθμός σελίδας εκτός εύρους**Επαληθεύστε ότι η καθορισμένη σελίδα υπάρχει μέσα στο έγγραφό σας.
- **Προβλήματα διαδρομής αρχείου**Ελέγξτε ξανά τις διαδρομές καταλόγου εισόδου και εξόδου για ορθότητα.
