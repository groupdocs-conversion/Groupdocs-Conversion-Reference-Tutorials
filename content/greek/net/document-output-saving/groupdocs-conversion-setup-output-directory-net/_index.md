### Ρύθμιση παραμέτρων διαδρομής καταλόγου εξόδου
**Επισκόπηση:** Αυτή η λειτουργία διαμορφώνει έναν καθορισμένο κατάλογο για την αποθήκευση των αρχείων που έχουν μετατραπεί, επιτρέποντας τον δυναμικό χειρισμό των θέσεων αποθήκευσης αρχείων.

#### Βήμα 1: Ορισμός της μεθόδου
Ορίστε μια μέθοδο που ανακτά ή ορίζει τη διαδρομή του καταλόγου εξόδου:

```csharp
using System;
using System.IO;

// Ορίστε μια μέθοδο για να λάβετε τη διαδρομή του καταλόγου εξόδου
class DirectorySetup {
    public static string GetOutputDirectoryPath() {
        // Ορίστε τη διαδρομή του βασικού φακέλου εξόδου
        string baseOutputFolder = "YOUR_OUTPUT_DIRECTORY";

        // Ελέγξτε αν ο κατάλογος υπάρχει, δημιουργήστε τον αν όχι
        if (!Directory.Exists(baseOutputFolder)) {
            Directory.CreateDirectory(baseOutputFolder);
        }

        return baseOutputFolder;
    }
}
```

**Εξήγηση:** Αυτή η μέθοδος διασφαλίζει ότι ο καθορισμένος φάκελος εξόδου είναι διαθέσιμος. Εάν δεν υπάρχει, η μέθοδος τον δημιουργεί για εσάς.

#### Βήμα 2: Χρήση σε Μετατροπή
Ενσωματώστε αυτήν τη διαδρομή καταλόγου κατά την εκτέλεση μιας μετατροπής:

```csharp
class ConversionExample {
    public static void ConvertFile() {
        Converter converter = new Converter("path/to/your/input/file");
        string outputPath = DirectorySetup.GetOutputDirectoryPath();
        converter.Convert("convertedFile.pdf", new PdfSaveOptions { OutputFilePath = Path.Combine(outputPath, "output.pdf") });
    }
}
```

**Εξήγηση:** Αυτό το απόσπασμα δείχνει πώς να εφαρμόσετε τη διαμόρφωση της διαδρομής εξόδου σας κατά τη διάρκεια μιας λειτουργίας μετατροπής αρχείου.
