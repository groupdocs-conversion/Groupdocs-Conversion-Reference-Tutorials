## Οδηγός Εφαρμογής

### Φόρτωση και μετατροπή αρχείου PLT σε PNG

**Επισκόπηση:**
Το βασικό χαρακτηριστικό αυτού του σεμιναρίου είναι η φόρτωση ενός αρχείου PLT και η μετατροπή του σε μορφή PNG. Αυτή η διαδικασία περιλαμβάνει τη ρύθμιση επιλογών μετατροπής ειδικά για τις μορφές εικόνας.

#### Βήμα 1: Ρύθμιση καταλόγου εξόδου και συνάρτησης ροής
Αρχικά, καθορίστε πού θα αποθηκευτούν τα αρχεία που έχουν μετατραπεί:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.png");

Func<SavePageContext, Stream> getPageStream = savePageContext =>
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

- **Εξήγηση**: `getPageStream` είναι μια συνάρτηση που επιστρέφει μια ροή για κάθε σελίδα που μετατρέπεται. Βοηθά στην αποθήκευση των αρχείων PNG εξόδου στον καθορισμένο κατάλογο.

#### Βήμα 2: Διαμόρφωση επιλογών μετατροπής

Ορίστε πώς θα μετατραπεί το αρχείο PLT:

```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Png };
```

- **Εξήγηση**: `options` καθορίζει ότι η μορφή μετατροπής είναι PNG. Αυτή η ρύθμιση παραμέτρων διασφαλίζει ότι τα αρχεία εξόδου έχουν την επιθυμητή μορφή εικόνας.

#### Βήμα 3: Εκτέλεση μετατροπής

Εκτελέστε τη μετατροπή χρησιμοποιώντας την παρουσία του μετατροπέα:

```csharp
using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_PLT"))
{
    converter.Convert(getPageStream, options);
}
```

- **Εξήγηση**: Το `Convert` Η μέθοδος λαμβάνει τη συνάρτηση ροής και τις επιλογές μετατροπής για να επεξεργαστεί και να αποθηκεύσει κάθε σελίδα του αρχείου PLT ως εικόνα PNG.

**Συμβουλές αντιμετώπισης προβλημάτων:**
- Βεβαιωθείτε ότι η διαδρομή του καταλόγου εξόδου έχει καθοριστεί σωστά.
- Επαληθεύστε ότι το αρχείο προέλευσης PLT υπάρχει στη δεδομένη διαδρομή.
