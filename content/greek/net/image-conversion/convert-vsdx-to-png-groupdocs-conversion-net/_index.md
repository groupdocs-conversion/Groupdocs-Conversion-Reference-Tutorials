**Βασική Αρχικοποίηση και Ρύθμιση**

Δείτε πώς μπορείτε να αρχικοποιήσετε το GroupDocs.Conversion στο έργο σας C#:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main()
    {
        // Διαδρομή προς τον κατάλογο εξόδου
        string outputFolder = "YOUR_OUTPUT_DIRECTORY";

        // Πρότυπο για την ονομασία αρχείων που έχουν μετατραπεί
        string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.png");

        // Συνάρτηση για λήψη ροής για κάθε σελίδα μετατροπής
        Func<SavePageContext, Stream> getPageStream = savePageContext =>
            new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);

        // Φορτώστε το αρχείο VSD πηγής (αντικαταστήστε το με την πραγματική σας διαδρομή)
        using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY\\\\sample.vsd"))
        {
            // Ορισμός επιλογών μετατροπής για τη μορφή PNG
            ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Png };

            // Εκτελέστε τη διαδικασία μετατροπής σε μορφή PNG
            converter.Convert(getPageStream, options);
        }
    }
}
```

Αυτό το απόσπασμα κώδικα δημιουργεί ένα βασικό περιβάλλον για τη μετατροπή αρχείων VSD σε εικόνες PNG.
