## Οδηγός Εφαρμογής

### Μετατροπή αρχείου OneNote σε SVG

Σε αυτήν την ενότητα, θα περιγράψουμε τα βήματα που απαιτούνται για τη μετατροπή ενός αρχείου Microsoft OneNote (.one) σε μορφή SVG χρησιμοποιώντας το GroupDocs.Conversion for .NET.

#### Επισκόπηση
Ο κύριος στόχος είναι η φόρτωση ενός εγγράφου OneNote και η μετατροπή του σε SVG. Αυτό περιλαμβάνει τη διαμόρφωση επιλογών μετατροπής ειδικά για την έξοδο SVG.

#### Βήμα προς βήμα εφαρμογή

##### Φόρτωση του αρχείου Source ONE
Αρχικά, καθορίστε τη διαδρομή του αρχείου προέλευσης OneNote:
```csharp
string documentPath = "YOUR_DOCUMENT_DIRECTORY/sample.one";
```

##### Ορισμός επιλογών μετατροπής για τη μορφή SVG
Διαμορφώστε τις ρυθμίσεις μετατροπής προσαρμοσμένες στην έξοδο SVG:
```csharp
var options = new PageDescriptionLanguageConvertOptions { 
    Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Svg 
};
```

Αυτό διαμορφώνει το `PageDescriptionLanguageConvertOptions` αντικείμενο, καθορίζοντας ότι η μορφή προορισμού είναι SVG.

##### Εκτελέστε τη μετατροπή και αποθηκεύστε το αποτέλεσμα
Εκτελέστε τη διαδικασία μετατροπής και αποθηκεύστε το αποτέλεσμα:
```csharp
string outputDirectory = "YOUR_OUTPUT_DIRECTORY/";
string outputFile = Path.Combine(outputDirectory, "one-converted-to.svg");

using (var converter = new GroupDocs.Conversion.Converter(documentPath))
{
    converter.Convert(outputFile, options);
}
```

Αυτός ο κώδικας χρησιμοποιεί το `Converter` αντικείμενο για να μετατρέψετε και να αποθηκεύσετε το αρχείο ως SVG.
