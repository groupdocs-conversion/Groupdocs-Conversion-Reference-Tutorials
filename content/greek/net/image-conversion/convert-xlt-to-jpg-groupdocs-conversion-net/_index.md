## Οδηγός Εφαρμογής

### Φόρτωση αρχείου XLT
**Επισκόπηση**Ξεκινήστε φορτώνοντας το αρχείο XLT σε ένα `Converter` αντικείμενο.

#### Βήμα προς βήμα εφαρμογή:
1. **Δημιουργία αντικειμένου μετατροπέα**
   ```csharp
   using (Converter converter = new Converter(xltFilePath))
   {
       // Ο μετατροπέας διατηρεί τώρα το αρχείο XLT για επεξεργασία.
   }
   ```
2. **Εξηγήστε τις παραμέτρους και τον σκοπό**: 
   - `xltFilePath`: Διαδρομή προς το αρχείο XLT πηγής σας.
   - `Converter` αντικείμενο: Διαχειρίζεται τις διαδικασίες φόρτωσης και μετατροπής.

### Ορισμός επιλογών μετατροπής
**Επισκόπηση**: Ορίστε τις παραμέτρους μετατροπής, ρυθμίζοντάς την συγκεκριμένα για μετατροπή σε μορφή JPG.

#### Βήμα προς βήμα εφαρμογή:
1. **Ορισμός επιλογών μετατροπής εικόνας**
   ```csharp
   using GroupDocs.Conversion.Options.Convert;

   // Καθορίστε τη μορφή στόχου ως JPG
   ImageConvertOptions jpgOptions = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Jpg };
   ```
2. **Διαμόρφωση κλειδιού**: 
   - `ImageConvertOptions`: Ρυθμίζει τις παραμέτρους μετατροπής.
   - `Format`: Ορίζει τον τύπο αρχείου εξόδου, σε αυτήν την περίπτωση, JPG.

### Μετατροπή XLT σε JPG
**Επισκόπηση**Εκτελέστε τη μετατροπή από XLT σε μια σειρά εικόνων JPG χρησιμοποιώντας καθορισμένες επιλογές και διαδρομές.

#### Βήμα προς βήμα εφαρμογή:
1. **Φόρτωση αρχείου XLT**
   ```csharp
   using (Converter converter = new GroupDocs.Conversion.Converter("YOUR_DOCUMENT_DIRECTORY/sample.xlt"))
   {
       // Η λογική μετατροπής θα προστεθεί εδώ.
   }
   ```
2. **Ορισμός επιλογών μετατροπής και εκτέλεση μετατροπής**
   ```csharp
   string outputFolder = "YOUR_OUTPUT_DIRECTORY";
   string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");

   Func<SavePageContext, Stream> getPageStream = savePageContext =>
       new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);

   converter.Convert(getPageStream, jpgOptions);
   ```
3. **Εξηγήστε τις παραμέτρους και τον σκοπό της μεθόδου**: 
   - `outputFolder`: Κατάλογος όπου θα αποθηκευτούν τα αρχεία JPG.
   - `getPageStream`Συνάρτηση για τη διαχείριση της ροής εξόδου κάθε σελίδας κατά τη μετατροπή.
