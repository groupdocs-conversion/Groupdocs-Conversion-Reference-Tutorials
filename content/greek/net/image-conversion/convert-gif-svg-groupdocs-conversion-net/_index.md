### Χαρακτηριστικό: Μετατροπή GIF σε SVG

#### Επισκόπηση

Η μετατροπή GIF σε μορφή SVG ωφελεί τις εφαρμογές ιστού, καθιστώντας τα γραφικά κλιμακώσιμα και μειώνοντας το μέγεθος του αρχείου. Αυτή η ενότητα θα σας καθοδηγήσει σε κάθε βήμα αυτής της διαδικασίας:

**Βήμα 1: Ορισμός καταλόγου εξόδου και ρύθμιση επιλογών μετατροπής**

Βεβαιωθείτε ότι ο κατάλογος εξόδου σας υπάρχει και, στη συνέχεια, ρυθμίστε τις επιλογές μετατροπής:

```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", "output");
Directory.CreateDirectory(outputFolder); // Βεβαιωθείτε ότι υπάρχει ο κατάλογος εξόδου

// Καθορίστε τη διαδρομή εισόδου του αρχείου GIF
string inputFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.gif");

// Ορισμός διαδρομής αρχείου SVG εξόδου
string outputFile = Path.Combine(outputFolder, "gif-converted-to.svg");
```

**Βήμα 2: Αρχικοποίηση του μετατροπέα και εκτέλεση μετατροπής**

Αρχικοποιήστε το αντικείμενο μετατροπέα με το αρχείο GIF πηγής σας:

```csharp
using (var converter = new Converter(inputFile))
{
    // Ορισμός επιλογών μετατροπής για τη μορφή SVG
    var options = new PageDescriptionLanguageConvertOptions { Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Svg };

    // Μετατρέψτε και αποθηκεύστε το αρχείο SVG εξόδου
    converter.Convert(outputFile, options);
}
```

**Επεξήγηση παραμέτρων:**
- `inputFile`: Διαδρομή προς το GIF πηγής σας.
- `outputFolder`: Κατάλογος όπου θα αποθηκευτεί το μετατρεπόμενο SVG.
- `options.Format`: Καθορίζει τη μετατροπή σε μορφή SVG.
