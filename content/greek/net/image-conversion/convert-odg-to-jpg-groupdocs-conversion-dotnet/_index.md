## Οδηγός Εφαρμογής

### Φόρτωση αρχείου ODG πηγής

Το πρώτο βήμα είναι να φορτώσετε το αρχείο ODG πηγής σας. Αυτή η λειτουργία σάς επιτρέπει να προετοιμάσετε το έγγραφό σας για μετατροπή:

#### Αρχικοποίηση αντικειμένου μετατροπέα

```csharp
using System;
using GroupDocs.Conversion;

string inputFilePath = @"YOUR_DOCUMENT_DIRECTORY/Sample.odg";
Converter converter = new Converter(inputFilePath);
```

**Εξήγηση:**
- **`inputFilePath`:** Διαδρομή προς το αρχείο ODG που θέλετε να μετατρέψετε.
- **`converter`:** Ένα παράδειγμα `GroupDocs.Conversion` που διευκολύνει τις εργασίες μετατροπής.

### Ορισμός επιλογών μετατροπής για μορφή JPG

Μόλις φορτωθεί το έγγραφό σας, ρυθμίστε το για μετατροπή σε μορφή JPG:

#### Ορισμός παραμέτρων εξόδου και επιλογών μετατροπής

```csharp
using System;
using System.IO;
using GroupDocs.Conversion.Options.Convert;

string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");

Func<SavePageContext, Stream> getPageStream = savePageContext =>
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);

ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Jpg };
```

**Εξήγηση:**
- **`outputFolder`:** Κατάλογος για την αποθήκευση των μετατρεπόμενων εικόνων.
- **`outputFileTemplate`:** Πρότυπο για την ονομασία αρχείων εξόδου. Χρησιμοποιεί placeholders όπως `{0}` για δυναμικές τιμές όπως αριθμοί σελίδων.
- **`getPageStream`:** Συνάρτηση που επιστρέφει ένα `FileStream` για κάθε σελίδα που αποθηκεύεται.
- **`options`:** Ρυθμίζει τη μορφή μετατροπής σε JPG.

#### Εκτέλεση μετατροπής

Χρησιμοποιήστε τις διαμορφωμένες επιλογές για να μετατρέψετε και να αποθηκεύσετε το αρχείο ODG:

```csharp
converter.Convert(getPageStream, options);
```
