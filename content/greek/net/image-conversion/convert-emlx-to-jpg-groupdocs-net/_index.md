## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα δούμε βήμα προς βήμα τη μετατροπή των αρχείων EMLX σε εικόνες JPG.

### Φόρτωση και προετοιμασία αρχείων

#### Επισκόπηση

Ξεκινήστε προετοιμάζοντας το αρχείο προέλευσης EMLX και ρυθμίζοντας έναν κατάλογο εξόδου για τα αρχεία που έχουν μετατραπεί. Βεβαιωθείτε ότι ο φάκελος προορισμού υπάρχει πριν προχωρήσετε στις μετατροπές, για να αποφύγετε σφάλματα κατά τη λειτουργία αποθήκευσης.

```csharp
using System;
using System.IO;

string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.emlx");
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");

// Βεβαιωθείτε ότι ο κατάλογος εξόδου υπάρχει
if (!Directory.Exists(outputFolder))
{
    Directory.CreateDirectory(outputFolder);
}

Console.WriteLine("Directories are set up.");
```

### Ρύθμιση επιλογών μετατροπής

#### Επισκόπηση

Διαμορφώστε τις ρυθμίσεις μετατροπής για να καθορίσετε ότι θέλετε τα αρχεία σας σε μορφή JPG:

```csharp
using GroupDocs.Conversion.Options.Convert;

// Ορισμός επιλογών μετατροπής για μορφή JPG
ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Jpg };

Console.WriteLine("Conversion options configured.");
```

### Εκτέλεση της μετατροπής

#### Επισκόπηση

Αφού έχετε ρυθμίσει όλα τα στοιχεία, εκτελέστε την πραγματική μετατροπή:

```csharp
using System;
using GroupDocs.Conversion;

// Αρχικοποίηση ενός FileStream για κάθε σελίδα εξόδου
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");
Func<SavePageContext, Stream> getPageStream = savePageContext => new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);

using (Converter converter = new Converter(inputFilePath))
{
    // Εκτελέστε τη μετατροπή
    converter.Convert(getPageStream, options);
}

Console.WriteLine("Conversion completed successfully.");
```

**Εξήγηση**: Το `getPageStream` Η συνάρτηση δημιουργεί δυναμικά διαδρομές αρχείων για κάθε σελίδα που έχει μετατραπεί. Αυτό διασφαλίζει ότι πολλές σελίδες σε ένα αρχείο EMLX υποβάλλονται σε σωστή επεξεργασία.
