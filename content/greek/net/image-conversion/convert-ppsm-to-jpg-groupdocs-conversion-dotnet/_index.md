## Οδηγός Εφαρμογής
### Φόρτωση αρχείου PPSM
Πριν από τη μετατροπή, φορτώστε το αρχείο PPSM πηγής. Αυτό το βήμα ρυθμίζει τη διαδικασία μετατροπής αρχικοποιώντας ένα `Converter` παράδειγμα.

**Βήμα 1: Αρχικοποίηση του μετατροπέα**
```csharp
using System;
using GroupDocs.Conversion;

string documentPath = "your_path_to_ppsm_file.ppsm";
// Φόρτωση του αρχείου PPSM πηγής
Converter converter = new Converter(documentPath);
converter.Dispose();
```
Ο `Converter` Το αντικείμενο είναι κεντρικό στη διαχείριση των διαδικασιών μετατροπής σας και η απόρριψή του διασφαλίζει την εκκαθάριση των πόρων.

### Μετατροπή PPSM σε JPG
Τώρα που έχετε φορτώσει το έγγραφό σας, ας προχωρήσουμε στη μετατροπή του σε μια σειρά εικόνων JPG—μία ανά διαφάνεια στην παρουσίαση.

**Βήμα 2: Ρύθμιση καταλόγου εξόδου**
Προετοιμάστε πού θα αποθηκευτούν τα αρχεία που έχετε μετατρέψει:
```csharp
string outputFolder = "your_output_directory";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");
```

**Βήμα 3: Ορισμός συνάρτησης ροής σελίδας**
Αυτή η λειτουργία διασφαλίζει ότι κάθε σελίδα του PPSM αποθηκεύεται ως ξεχωριστό αρχείο JPG.
```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

**Βήμα 4: Διαμόρφωση επιλογών μετατροπής**
Καθορίστε ότι μετατρέπετε σε μορφή JPG:
```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Jpg };
```

**Βήμα 5: Εκτέλεση μετατροπής**
Τέλος, εκτελέστε τη μετατροπή καλώντας το `Convert` μέθοδος:
```csharp
using (Converter converter = new Converter(documentPath))
{
    // Μετατροπή σε μορφή JPG
    converter.Convert(getPageStream, options);
}
```
Αυτή η διαδικασία επαναλαμβάνεται σε κάθε διαφάνεια στο αρχείο PPSM, μετατρέποντάς τες ξεχωριστά σε εικόνες JPG.
