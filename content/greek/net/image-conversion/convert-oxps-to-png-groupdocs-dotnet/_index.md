## Οδηγός Εφαρμογής

Αυτή η ενότητα δείχνει πώς να μετατρέψετε αρχεία OXPS σε PNG χρησιμοποιώντας τη βιβλιοθήκη GroupDocs.Conversion.

### Φόρτωση και μετατροπή αρχείου OXPS

#### Επισκόπηση
Μάθετε πώς να φορτώνετε ένα αρχείο OXPS και να εκτελείτε αποτελεσματικά μια μετατροπή σε μορφή PNG.

**1. Ρύθμιση διαδρομών**
Ορίστε διαδρομές για τους καταλόγους εισόδου και εξόδου:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string inputFile = "YOUR_DOCUMENT_DIRECTORY/SAMPLE_OXPS.oxps";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.png");
```

**2. Δημιουργία ροής για κάθε σελίδα**
Χρησιμοποιήστε μια συνάρτηση για να δημιουργήσετε δυναμικά ροές κατά τη μετατροπή:

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext =>
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

**3. Διαδικασία μετατροπής**
Φορτώστε το αρχείο OXPS και μετατρέψτε το χρησιμοποιώντας το GroupDocs.Conversion:

```csharp
using (Converter converter = new Converter(inputFile))
{
    ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Png };
    converter.Convert(getPageStream, options);
}
```

### Ορισμός επιλογών μετατροπής για μορφή PNG

#### Επισκόπηση
Διαμορφώστε τις ρυθμίσεις μετατροπής εικόνας ειδικά προσαρμοσμένες για τη μορφή PNG.

**1. Αρχικοποίηση επιλογών μετατροπής**
Ξεκινήστε δημιουργώντας μια παρουσία του `ImageConvertOptions`:

```csharp
ImageConvertOptions options = new ImageConvertOptions();
```

**2. Καθορισμός μορφής εξόδου**
Ορίστε την επιθυμητή μορφή εξόδου σε PNG:

```csharp
options.Format = ImageFileType.Png;
```
