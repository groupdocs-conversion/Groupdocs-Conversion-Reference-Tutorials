## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Μετατροπή αρχείων TEX σε μορφή PSD

Αυτή η λειτουργία δείχνει πώς να μετατρέψετε ένα αρχείο LaTeX (TEX) σε ένα έγγραφο Adobe Photoshop (PSD) χρησιμοποιώντας τη βιβλιοθήκη GroupDocs.Conversion.

#### Βήμα 1: Ορισμός καταλόγου εξόδου και προτύπου ονομασίας αρχείου
Αρχικά, καθορίστε πού θα αποθηκευτούν τα αρχεία που έχουν μετατραπεί και δημιουργήστε μια σύμβαση ονομασίας για αυτά:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");
```

#### Βήμα 2: Δημιουργήστε ένα FileStream για κάθε σελίδα που έχει μετατραπεί
Δημιουργήστε μια ροή αρχείων για να χειριστείτε την αποθήκευση κάθε σελίδας που έχει μετατραπεί. Αυτό το βήμα διασφαλίζει ότι τα έγγραφά σας αποθηκεύονται σωστά σε μορφή PSD.

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

#### Βήμα 3: Φόρτωση και μετατροπή του αρχείου TEX
Φορτώστε το αρχείο προέλευσης TEX και ρυθμίστε τις επιλογές μετατροπής για να το εξαγάγετε ως PSD:

```csharp
using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY\\\\sample.tex"))
{
    // Ορίστε επιλογές μετατροπής για τη μορφή PSD.
    ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd };
    
    // Εκτελέστε τη μετατροπή σε μορφή PSD.
    converter.Convert(getPageStream, options);
}
```

Αυτό το απόσπασμα κώδικα χειρίζεται τη βασική λειτουργικότητα της φόρτωσης ενός αρχείου TEX και της μετατροπής του σε έγγραφο PSD. Κάθε σελίδα του εγγράφου αποθηκεύεται ως μεμονωμένο αρχείο PSD στον καθορισμένο κατάλογο εξόδου.

### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι οι διαδρομές έχουν οριστεί σωστά για να αποφύγετε `FileNotFoundException`.
- Ελέγξτε για επαρκή δικαιώματα εγγραφής αρχείων στον καθορισμένο φάκελο εξόδου.
- Επιβεβαιώστε ότι η βιβλιοθήκη GroupDocs.Conversion αναφέρεται σωστά στο έργο σας.
