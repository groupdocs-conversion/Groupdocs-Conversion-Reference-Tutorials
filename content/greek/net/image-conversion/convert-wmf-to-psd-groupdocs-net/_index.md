## Οδηγός Εφαρμογής
### Μετατροπή WMF σε PSD
Αυτή η λειτουργία μετατρέπει τα αρχεία WMF σε επεξεργάσιμη μορφή PSD για το Adobe Photoshop.

#### Βήμα 1: Ρύθμιση παραμέτρων καταλόγου εξόδου και προτύπου αρχείου
Ορίστε τον κατάλογο εξόδου όπου θα αποθηκευτούν τα αρχεία που έχουν μετατραπεί:
```csharp
string outputFolder = \@"YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");
```
Αυτό το πρότυπο διασφαλίζει ότι κάθε σελίδα έχει μοναδικό όνομα.

#### Βήμα 2: Ορίστε μια συνάρτηση για τη λήψη ροών σελίδων
Δημιουργήστε μια συνάρτηση για τη διαχείριση ροών αρχείων:
```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```
Αυτή η συνάρτηση ανοίγει μια ροή για την εγγραφή αρχείων PSD.

#### Βήμα 3: Φόρτωση και μετατροπή WMF σε PSD
Χρησιμοποιήστε το `Converter` κλάση για να φορτώσετε το αρχείο WMF και να εκτελέσετε μετατροπή:
```csharp
using (Converter converter = new Converter(\@"YOUR_DOCUMENT_DIRECTORY/sample.wmf"))
{
    ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd };
    converter.Convert(getPageStream, options);
}
```
- **Παράμετροι**: `options` καθορίζει τη μορφή μετατροπής (PSD).

- **Συνήθη προβλήματα**Βεβαιωθείτε ότι η διαδρομή του αρχείου WMF είναι σωστή και προσβάσιμη.
