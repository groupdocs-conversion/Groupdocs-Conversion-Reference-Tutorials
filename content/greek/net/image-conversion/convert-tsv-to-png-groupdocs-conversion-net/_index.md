## Οδηγός Εφαρμογής
### Φόρτωση και μετατροπή αρχείου TSV σε PNG
Ας αναλύσουμε τα βήματα που απαιτούνται για τη μετατροπή ενός αρχείου TSV σε εικόνες PNG.
#### Επισκόπηση
Αυτή η ενότητα παρουσιάζει τη φόρτωση ενός αρχείου TSV χρησιμοποιώντας το GroupDocs.Conversion και τη μετατροπή κάθε σελίδας του σε ξεχωριστά αρχεία PNG.
#### Βήμα 1: Ρύθμιση καταλόγου εξόδου
Αρχικά, καθορίστε τον κατάλογο εξόδου όπου θα αποθηκευτούν οι εικόνες που έχουν μετατραπεί:
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY"; // Ορίστε αυτό στην επιθυμητή διαδρομή
```
#### Βήμα 2: Φόρτωση του αρχείου TSV
Χρησιμοποιήστε το GroupDocs.Conversion για να φορτώσετε το αρχείο TSV πηγής. Βεβαιωθείτε ότι έχετε δώσει τη σωστή διαδρομή αρχείου:
```csharp
string inputFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.tsv");
using (Converter converter = new Converter(inputFile))
{
    // Συνέχεια με τις επιλογές μετατροπής
}
```
#### Βήμα 3: Διαμόρφωση επιλογών μετατροπής
Ορίστε πώς κάθε σελίδα του TSV σας θα πρέπει να μετατρέπεται σε αρχείο PNG καθορίζοντας επιλογές μορφής εικόνας:
```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Png };
```
#### Βήμα 4: Μετατροπή και αποθήκευση κάθε σελίδας ως PNG
Δημιουργήστε μια συνάρτηση για τη διαχείριση της ροής εξόδου για κάθε σελίδα. Αυτό το βήμα περιλαμβάνει τη δημιουργία ενός μοναδικού προτύπου ονόματος αρχείου για κάθε σελίδα που έχει μετατραπεί:
```csharp
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.png");

Func<SavePageContext, Stream> getPageStream = savePageContext =>
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);

converter.Convert(getPageStream, options);
```
**Εξήγηση:**
- `getPageStream`Αυτή η συνάρτηση δημιουργεί μια ροή αρχείων για κάθε σελίδα που μετατρέπεται.
- Ο `outputFileTemplate` Χρησιμοποιείται για τη δημιουργία μοναδικών ονομάτων αρχείων για τις εικόνες PNG.
