## Οδηγός Εφαρμογής
### Χαρακτηριστικό: Φόρτωση αρχείου MBOX
#### Επισκόπηση
Η φόρτωση ενός αρχείου MBOX είναι το πρώτο βήμα στη διαδικασία μετατροπής μας. Αυτή η λειτουργία δείχνει πώς να φορτώσετε το αρχείο email σας χρησιμοποιώντας το GroupDocs.Conversion για .NET.

**Βήμα 1:** Αρχικοποίηση του αντικειμένου μετατροπέα
Αρχικά, δημιουργήστε ένα `Converter` αντικείμενο καθορίζοντας τη διαδρομή του αρχείου MBOX. Αυτό προετοιμάζει το αρχείο για επόμενες λειτουργίες μετατροπής.

```csharp
using System;
using GroupDocs.Conversion;

string mboxFilePath = "YOUR_DOCUMENT_DIRECTORY/sample.mbox"; // Αντικαταστήστε με την πραγματική διαδρομή αρχείου MBOX

// Δημιουργήστε ένα αντικείμενο Converter για να φορτώσετε το αρχείο MBOX πηγής
using (Converter converter = new Converter(mboxFilePath))
{
    // Το αρχείο MBOX έχει πλέον φορτωθεί και είναι έτοιμο για λειτουργίες μετατροπής.
}
```

**Εξήγηση:** Αυτό το απόσπασμα δημιουργεί ένα `Converter` παράδειγμα, το οποίο διαβάζει το αρχείο MBOX από την καθορισμένη διαδρομή. Σε αυτό το στάδιο, το αρχείο σας είναι έτοιμο για μετατροπή σε διαφορετικές μορφές.

### Χαρακτηριστικό: Μετατροπή MBOX σε μορφή PSD
#### Επισκόπηση
Τώρα που έχουμε φορτώσει το αρχείο MBOX, ας το μετατρέψουμε σε μορφή PSD—μια δημοφιλή μορφή γραφιστικής.

**Βήμα 2:** Ορισμός διαδρομής εξόδου και επιλογών μετατροπής
Καθορίστε πού θα αποθηκευτούν τα αρχεία που έχουν μετατραπεί και ρυθμίστε τις επιλογές μετατροπής για το PSD.

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

string outputFolder = "YOUR_OUTPUT_DIRECTORY"; // Καθορίστε τον κατάλογο όπου θα αποθηκευτούν τα αρχεία που έχουν μετατραπεί
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");

// Ορίστε μια συνάρτηση για να λάβετε ροή σελίδας για κάθε αποτέλεσμα μετατροπής
Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);

using (Converter converter = new Converter(mboxFilePath)) // Φόρτωση του αρχείου MBOX που φορτώθηκε προηγουμένως
{
    // Ορισμός επιλογών μετατροπής για μορφή PSD
    ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd };

    // Εκτελέστε μετατροπή από μορφή MBOX σε μορφή PSD
    converter.Convert(getPageStream, options);
}
```

**Εξήγηση:** Αυτό το απόσπασμα κώδικα ορίζει τον κατάλογο εξόδου και ορίζει τον τρόπο με τον οποίο θα αποθηκευτεί κάθε σελίδα του αρχείου που έχει μετατραπεί. `ImageConvertOptions` έχει ρυθμιστεί για μορφή PSD, διασφαλίζοντας ότι τα email σας μετατρέπονται σε γραφικά υψηλής ποιότητας.
