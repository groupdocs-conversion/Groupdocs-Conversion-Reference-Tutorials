## Οδηγός Εφαρμογής

### Μετατροπή WEBP σε PSD

Η μετατροπή ενός αρχείου WEBP σε μορφή PSD περιλαμβάνει διάφορα βήματα. Ας το χωρίσουμε σε διαχειρίσιμες ενότητες.

#### Βήμα 1: Ρύθμιση καταλόγου εξόδου

Αρχικά, ορίστε πού θέλετε να αποθηκεύσετε τα αρχεία που έχετε μετατρέψει:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");
```

Αυτός ο κώδικας ορίζει το πρότυπο καταλόγου και ονόματος αρχείου για την αποθήκευση των εξόδων PSD.

#### Βήμα 2: Ορισμός συνάρτησης ροής σελίδας

Στη συνέχεια, δημιουργήστε μια συνάρτηση για τη διαχείριση ροών σελίδων κατά τη μετατροπή:

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext =>
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

Αυτή η συνάρτηση λάμδα δημιουργεί ροές αρχείων για κάθε σελίδα που έχει μετατραπεί.

#### Βήμα 3: Διαμόρφωση επιλογών μετατροπής

Καθορίστε τις ρυθμίσεις μετατροπής για τη μορφή PSD:

```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd };
```

Ο `ImageConvertOptions` Το αντικείμενο είναι κρίσιμο, καθώς υπαγορεύει τον τύπο του αρχείου προορισμού και άλλες παραμέτρους.

#### Βήμα 4: Εκτέλεση μετατροπής

Τέλος, εκτελέστε τη μετατροπή χρησιμοποιώντας τις διαμορφωμένες ρυθμίσεις:

```csharp
using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_WEBP"))
{
    converter.Convert(getPageStream, options);
}
```

Αυτό το απόσπασμα κώδικα εκτελεί τη διαδικασία μετατροπής και αποθηκεύει κάθε σελίδα ως αρχείο PSD.
