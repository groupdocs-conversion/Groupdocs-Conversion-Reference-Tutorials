## Οδηγός Εφαρμογής

### Φόρτωση αρχείου J2C
**Επισκόπηση:** Αυτή η λειτουργία δείχνει τη φόρτωση ενός αρχείου εικόνας JPEG 2000 για μετατροπή.

#### Οδηγίες βήμα προς βήμα:
1. **Αρχικοποίηση του μετατροπέα**: Χρησιμοποιήστε το `Converter` κλάση για να φορτώσετε το αρχείο .j2c.
   ```csharp
   string sourceFilePath = "YOUR_DOCUMENT_DIRECTORY\sample.j2c"; // Αντικαταστήστε με την πραγματική διαδρομή αρχείου σας
   
   using (Converter converter = new Converter(sourceFilePath))
   {
       Console.WriteLine("J2C file loaded successfully.");
   }
   ```

### Ορισμός επιλογών μετατροπής για τη μορφή JPG
**Επισκόπηση:** Ρυθμίστε τις παραμέτρους των επιλογών μετατροπής για να μετατρέψετε ένα αρχείο σε μορφή JPG.

#### Οδηγίες βήμα προς βήμα:
1. **Δημιουργία ImageConvertOptions**Ορίστε τη μορφή εξόδου ως JPG.
   ```csharp
   using GroupDocs.Conversion.Options.Convert;

   ImageConvertOptions options = new ImageConvertOptions { Format = FileTypes.ImageFileType.Jpg };
   Console.WriteLine("Conversion options for JPG set successfully.");
   ```

### Μετατροπή J2C σε μορφή JPG
**Επισκόπηση:** Εκτελέστε τη μετατροπή από ένα φορτωμένο αρχείο J2C σε μορφή JPG.

#### Οδηγίες βήμα προς βήμα:
1. **Ορισμός ροής εξόδου**: Καθορίστε πού θα αποθηκευτούν οι σελίδες που έχουν μετατραπεί.
   ```csharp
   string outputFolder = "YOUR_OUTPUT_DIRECTORY"; // Αντικαταστήστε με την πραγματική διαδρομή καταλόγου εξόδου σας
   string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");

   Func<SavePageContext, Stream> getPageStream = savePageContext => 
       new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
   ```

2. **Εκτέλεση μετατροπής**: Χρησιμοποιήστε το `Converter` αντικείμενο για εκτέλεση μετατροπής χρησιμοποιώντας καθορισμένες επιλογές.
   ```csharp
   using (Converter converter = new Converter(sourceFilePath)) // Επαναχρησιμοποίηση της διαδρομής του αρχείου προέλευσης από το προηγούμενο βήμα
   {
       converter.Convert(getPageStream, options); // Μετατροπή και αποθήκευση ως JPG
       Console.WriteLine("Conversion to JPG completed successfully.");
   }
   ```

#### Συμβουλές αντιμετώπισης προβλημάτων:
- Βεβαιωθείτε ότι ο κατάλογος εξόδου υπάρχει πριν εκτελέσετε τη μετατροπή.
- Ελέγξτε αν το αρχείο .j2c δεν είναι κατεστραμμένο ή κλειδωμένο από κάποια άλλη διεργασία.
