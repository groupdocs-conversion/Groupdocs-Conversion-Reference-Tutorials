## Οδηγός Εφαρμογής

Ας αναλύσουμε την υλοποίηση σε βασικά χαρακτηριστικά, τα οποία εξηγούνται βήμα προς βήμα.

### Ορισμός καταλόγου εξόδου και προτύπου αρχείου

Αρχικά, ορίστε τον κατάλογο εξόδου όπου θα αποθηκευτούν τα αρχεία JPG:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");
```

Αυτή η ρύθμιση βοηθά στην αποτελεσματική οργάνωση των δημιουργούμενων εικόνων.

#### Δημιουργήστε μια συνάρτηση για τη δημιουργία ροών για την αποθήκευση σελίδων ως JPG

Για να αποθηκεύσετε κάθε σελίδα του αρχείου XLSB ως μεμονωμένη εικόνα JPG:

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

### Φόρτωση του αρχείου XLSB προέλευσης και ορισμός επιλογών μετατροπής

Φορτώστε το αρχείο XLSB και καθορίστε ότι θέλετε να το μετατρέψετε σε μορφή JPG:

```csharp
using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/sample.xlsb"))
{
    ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Jpg };

    // Εκτελέστε τη μετατροπή
    converter.Convert(getPageStream, options);
}
```

#### Επεξήγηση παραμέτρων

- `SavePageContext`: Παρέχει πληροφορίες για κάθε σελίδα που αποθηκεύεται.
- `ImageConvertOptions`: Ρυθμίζει τη μορφή εξόδου και άλλες ρυθμίσεις μετατροπής.
