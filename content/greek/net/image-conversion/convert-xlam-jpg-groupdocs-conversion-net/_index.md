## Οδηγός Εφαρμογής

Θα αναλύσουμε τη διαδικασία σε τρία κύρια χαρακτηριστικά: φόρτωση ενός αρχείου πηγής, ρύθμιση επιλογών μετατροπής για JPG και εκτέλεση της μετατροπής αρχείου.

### Λειτουργία 1: Φόρτωση αρχείου προέλευσης

Αυτή η λειτουργία δείχνει πώς να φορτώσετε ένα αρχείο XLAM χρησιμοποιώντας το GroupDocs.Conversion. Το `Converter` Η κλάση αρχικοποιείται με τη διαδρομή του αρχείου XLAM που θέλετε να μετατρέψετε.

**Βήμα προς βήμα εφαρμογή**

#### Φόρτωση του αρχείου XLAM πηγής
```csharp
using System;
using System.IO;
using GroupDocs.Conversion;

string inputFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.xlam");
using (Converter converter = new Converter(inputFile))
{
    // Η λογική μετατροπής θα υλοποιηθεί στα επόμενα βήματα.
}
```
*Ο `Converter` Η κλάση παίρνει τη διαδρομή του αρχείου πηγαίου κώδικα και το προετοιμάζει για μετατροπή. `using` Η δήλωση διασφαλίζει ότι οι πόροι απελευθερώνονται μόλις ολοκληρωθεί η λειτουργία.*

### Λειτουργία 2: Ρύθμιση επιλογών μετατροπής για μορφή JPG

Για να μετατρέψετε αρχεία σε μορφή JPEG, πρέπει να ορίσετε συγκεκριμένες επιλογές χρησιμοποιώντας `ImageConvertOptions`.

**Βήμα προς βήμα εφαρμογή**

#### Ορισμός επιλογών μετατροπής για JPG
```csharp
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

// Δημιουργήστε και ρυθμίστε τις παραμέτρους του ImageConvertOptions για JPG.
ImageConvertOptions options = new ImageConvertOptions {
    Format = ImageFileType.Jpg 
};

// Αυτές οι επιλογές καθορίζουν ότι η μορφή εξόδου θα πρέπει να είναι JPEG.
```
*Ο `ImageConvertOptions` Η κλάση σάς επιτρέπει να ορίσετε διάφορες ρυθμίσεις για τη μετατροπή εικόνας, όπως μορφή αρχείου, ανάλυση και ποιότητα.*

### Λειτουργία 3: Εκτέλεση μετατροπής αρχείων

Τώρα, ας εκτελέσουμε την πραγματική μετατροπή από XLAM σε JPG και ας αποθηκεύσουμε κάθε σελίδα ως ξεχωριστή εικόνα.

**Βήμα προς βήμα εφαρμογή**

#### Ορισμός καταλόγου εξόδου
```csharp
using System;
using System.IO;

string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", "output");
Directory.CreateDirectory(outputFolder); // Βεβαιωθείτε ότι ο κατάλογος υπάρχει.
```
*Δημιουργήστε έναν φάκελο εξόδου για την αποθήκευση των εικόνων που έχουν μετατραπεί. `Directory.CreateDirectory` Η μέθοδος διασφαλίζει ότι ο κατάλογος δημιουργείται εάν δεν υπάρχει ήδη.*

#### Διαδικασία μετατροπής
```csharp
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");

Func<SavePageContext, Stream> getPageStream = savePageContext =>
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);

// Εκτελέστε τη μετατροπή.
using (Converter converter = new Converter(inputFile))
{
    converter.Convert(getPageStream, options);
}
```
*Ο `Convert` Η μέθοδος δέχεται μια συνάρτηση ροής και επιλογές ως παραμέτρους. Επεξεργάζεται κάθε σελίδα του αρχείου XLAM και την αποθηκεύει ως ξεχωριστή εικόνα JPG.*
