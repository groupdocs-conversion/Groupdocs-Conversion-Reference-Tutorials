#### Ορισμός επιλογών μετατροπής για μορφή JPG
Ορίστε τις επιλογές μετατροπής για να καθορίσετε ότι θέλετε το αποτέλεσμα να είναι σε μορφή JPG:
```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Jpg };
```

#### Ορισμός συνάρτησης ροής εξόδου
Δημιουργήστε μια συνάρτηση για να χειριστείτε τη ροή εξόδου για κάθε σελίδα του αρχείου PLT σας. Αυτό διασφαλίζει ότι κάθε σελίδα που έχει μετατραπεί αποθηκεύεται ως ξεχωριστό αρχείο JPG:
```csharp

    new FileStream(Path.Combine(outputDirectory, $"converted-page-{savePageContext.Page}.jpg"), FileMode.Create);
```

#### Εκτελέστε τη μετατροπή
Τέλος, εκτελέστε τη μετατροπή καλώντας το `Convert` μέθοδος με τις καθορισμένες επιλογές σας:
```csharp

```
