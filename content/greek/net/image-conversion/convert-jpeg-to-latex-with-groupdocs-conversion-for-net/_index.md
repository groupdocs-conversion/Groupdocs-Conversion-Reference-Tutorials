## Ρύθμιση του GroupDocs.Conversion για .NET
Για να ξεκινήσουμε τη διαδικασία μετατροπής, πρέπει πρώτα να εγκαταστήσουμε τη βιβλιοθήκη GroupDocs.Conversion. Μπορείτε εύκολα να την προσθέσετε στο έργο σας χρησιμοποιώντας μία από τις ακόλουθες μεθόδους:

### .NET CLI

**Απόκτηση Άδειας:** Μπορείτε να ξεκινήσετε με μια δωρεάν δοκιμαστική έκδοση για να δοκιμάσετε τις δυνατότητες του GroupDocs.Conversion. Εάν χρειάζεστε περισσότερες, σκεφτείτε να αγοράσετε μια άδεια χρήσης ή να αποκτήσετε μια προσωρινή για εκτεταμένες δοκιμές.

Δείτε πώς μπορείτε να αρχικοποιήσετε και να ρυθμίσετε το GroupDocs.Conversion στο έργο σας C#:

```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

namespace ConversionFeature
{
    internal static class JpegToTexConverter
    {
        private const string DocumentDirectory = "YOUR_DOCUMENT_DIRECTORY";
        private const string OutputDirectory = "YOUR_OUTPUT_DIRECTORY";

        public static void ConvertJpegToTex()
        {
            string outputFile = Path.Combine(OutputDirectory, "jpeg-converted-to.tex");

            using (var converter = new Converter(Path.Combine(DocumentDirectory, "sample.jpeg")))
            {
                var options = new PageDescriptionLanguageConvertOptions { Format = PageDescriptionLanguageFileType.Tex };
                converter.Convert(outputFile, options);
            }
        }
    }
}
```

### Επισκόπηση της λειτουργίας μετατροπής JPEG σε TEX

Καθορίστε τους καταλόγους όπου βρίσκονται τα αρχεία JPEG πηγής σας και πού θέλετε να αποθηκεύσετε τα αρχεία TEX που έχουν μετατραπεί. Αντικατάσταση `YOUR_DOCUMENT_DIRECTORY` και `YOUR_OUTPUT_DIRECTORY` με τις πραγματικές σου διαδρομές.



#### Βήμα 3: Ορισμός επιλογών μετατροπής
Δημιουργήστε μια παρουσία του `PageDescriptionLanguageConvertOptions` και καθορίστε ότι θέλετε να μετατρέψετε σε μορφή TEX.

```csharp

```

#### Βήμα 4: Εκτελέστε τη μετατροπή
Επικαλέστε το `Convert` μέθοδο στο αντικείμενο μετατροπέα σας, μεταβιβάζοντας τη διαδρομή του αρχείου εξόδου και τις επιλογές μετατροπής. Αυτό εκτελεί την πραγματική διαδικασία μετατροπής.
