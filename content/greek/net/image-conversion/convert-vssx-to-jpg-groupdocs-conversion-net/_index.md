### Μετατροπή VSSX σε μορφή JPG

**Επισκόπηση:**
Αφού φορτώσετε το αρχείο VSSX, το επόμενο βήμα είναι η μετατροπή του σε μορφή εικόνας JPEG. Αυτή η ενότητα σας καθοδηγεί στη διαδικασία μετατροπής.

#### Βήμα 1: Ρύθμιση φακέλου εξόδου
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY"; // Ορίστε πού θα πρέπει να αποθηκεύονται τα αρχεία που έχουν μετατραπεί.
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");
```
- **Σκοπός**: Ορίζει τη θέση και τη σύμβαση ονομασίας για τα αρχεία JPEG εξόδου.

#### Βήμα 2: Προετοιμασία συνάρτησης ροής σελίδας
```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```
- **Εξήγηση**Αυτή η συνάρτηση καθορίζει τον τρόπο με τον οποίο κάθε σελίδα του αρχείου VSSX θα αποθηκευτεί ως εικόνα JPEG.

#### Βήμα 3: Ορισμός επιλογών μετατροπής
```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Jpg };
```
- **Σκοπός**: Διαμορφώνει τις ρυθμίσεις μετατροπής για έξοδο σε μορφή JPG.

#### Βήμα 4: Εκτελέστε τη μετατροπή
```csharp
using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/sample.vssx"))
{
    converter.Convert(getPageStream, options);
}
```
- **Εξήγηση**: Εκτελεί τη διαδικασία μετατροπής χρησιμοποιώντας την προηγουμένως καθορισμένη συνάρτηση ροής και τις επιλογές εικόνας.
