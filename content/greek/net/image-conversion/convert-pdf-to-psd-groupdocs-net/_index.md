## Οδηγός Εφαρμογής

### Μετατροπή εγγράφου σε PSD

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε κάθε σελίδα ενός εγγράφου PDF σε ένα μεμονωμένο αρχείο PSD χρησιμοποιώντας το GroupDocs.Conversion. Αυτό είναι ιδιαίτερα χρήσιμο όταν εργάζεστε με εικόνες υψηλής ανάλυσης σε λογισμικό γραφιστικής όπως το Adobe Photoshop.

#### Βήμα προς βήμα εφαρμογή

**1. Ορισμός καταλόγου εξόδου και προτύπου αρχείου**
Ρυθμίστε τον κατάλογο εξόδου για την αποθήκευση των αρχείων που έχουν μετατραπεί:

```csharp
string outputFolder = Path.Combine(outputDirectory, \u0024ConvertedFiles\u0024);
if (!Directory.Exists(outputFolder))
{
    Directory.CreateDirectory(outputFolder);
}

string outputFileTemplate = Path.Combine(outputFolder, \u0024converted-page-{0}.psd\u0024);
```

**2. Δημιουργήστε μια συνάρτηση για να λαμβάνετε τη ροή για την αποθήκευση κάθε σελίδας**
Χειριστείτε τη δημιουργία ροών για κάθε σελίδα που έχει μετατραπεί με αυτήν τη συνάρτηση:

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext =>
{
    return new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
};
```

**3. Ορίστε τις επιλογές μετατροπής και εκτελέστε τη μετατροπή**
Καθορίστε το PSD ως τη μορφή στόχου σας και εκτελέστε τη μετατροπή:

```csharp
ImageConvertOptions options = new ImageConvertOptions
{
    Format = ImageFileType.Psd // Καθορίστε τον τύπο αρχείου εξόδου.
};

converter.Convert(getPageStream, options);
```

### Συμβουλές αντιμετώπισης προβλημάτων
- **Συνηθισμένο πρόβλημα**: Βεβαιωθείτε ότι υπάρχουν όλοι οι κατάλογοι πριν γράψετε αρχεία για να αποτρέψετε `DirectoryNotFoundException`.
- **Χειρισμός σφαλμάτων**Χρησιμοποιήστε μπλοκ try-catch γύρω από τη λογική μετατροπής για να διαχειριστείτε τις εξαιρέσεις με ομαλό τρόπο.
