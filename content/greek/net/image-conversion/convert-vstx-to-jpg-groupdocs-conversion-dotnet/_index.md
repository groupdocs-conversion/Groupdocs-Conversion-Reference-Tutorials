## Βήματα μετατροπής VSTX σε JPG
Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε αρχεία Visio σε εικόνες JPEG, διευκολύνοντας την κοινή χρήση και την προβολή τους. Ας αναλύσουμε τα βήματα:

### Βήμα 1: Ορισμός διαδρομών εισόδου και εξόδου
Αρχικά, ορίστε τις διαδρομές αρχείων τόσο για την είσοδο όσο και για την έξοδο. Αυτό περιλαμβάνει τον καθορισμό της τοποθεσίας του αρχείου VSTX και της τοποθεσίας αποθήκευσης των αρχείων JPEG.
```csharp
string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.vstx");
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", "converted_images");
Directory.CreateDirectory(outputFolder);
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");
```

### Βήμα 2: Προετοιμασία της συνάρτησης μετατροπής
Θα χρειαστείτε μια συνάρτηση για να χειριστεί τη μετατροπή κάθε σελίδας σε αρχείο JPEG. Αυτή η συνάρτηση δημιουργεί ροές εξόδου για κάθε σελίδα.
```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

### Βήμα 3: Εκτελέστε τη μετατροπή
Αφού έχετε ρυθμίσει όλα τα στοιχεία, χρησιμοποιήστε το GroupDocs.Conversion για να μετατρέψετε το αρχείο VSTX σε μορφή JPEG. Δείτε πώς το κάνετε:
```csharp
using (Converter converter = new Converter(inputFilePath))
{
    ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Jpg };
    converter.Convert(getPageStream, options);
}
```

- **Επιλογές μετατροπής εικόνας:** Αυτή η κλάση σάς επιτρέπει να καθορίσετε ρυθμίσεις όπως η μορφή εικόνας. Σε αυτήν την περίπτωση, χρησιμοποιούμε `Jpg` ως τη μορφή-στόχο μας.
