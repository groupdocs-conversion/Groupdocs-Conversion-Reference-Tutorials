## Οδηγός Εφαρμογής

Ας αναλύσουμε την υλοποίηση σε ξεχωριστά χαρακτηριστικά.

### Φόρτωση αρχείου MOBI πηγής
**Επισκόπηση**Αυτό το βήμα περιλαμβάνει τη φόρτωση του αρχείου MOBI που θέλετε να μετατρέψετε. Προετοιμάζει το έγγραφό σας στη μνήμη για μετατροπή.

1. **Αρχικοποίηση αντικειμένου μετατροπέα**
   ```csharp
   string sourceFilePath = "YOUR_DOCUMENT_DIRECTORY\\sample.mobi"; // Αντικατάσταση με την πραγματική διαδρομή
   using (Converter converter = new Converter(sourceFilePath))
   {
       // Η λογική μετατροπής θα εφαρμοστεί εδώ.
   }
   ```
2. **Εξηγήστε τις παραμέτρους**: `sourceFilePath` είναι η διαδρομή προς το αρχείο MOBI σας, η οποία βοηθά στον εντοπισμό και τη φόρτωση του εγγράφου για μετατροπή.

### Ορισμός επιλογών μετατροπής για τη μορφή PSD
**Επισκόπηση**Η διαμόρφωση της μορφής εξόδου ως PSD περιλαμβάνει τον ορισμό συγκεκριμένων επιλογών που υπαγορεύουν τον τρόπο χειρισμού της μετατροπής.

1. **Ρύθμιση παραμέτρων ImageConvertOptions**
   ```csharp
   using GroupDocs.Conversion.Options.Convert;

   ImageConvertOptions options = new ImageConvertOptions
   {
       Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd // Η μορφή εξόδου έχει οριστεί σε PSD
   };
   ```
2. **Βασικές επιλογές διαμόρφωσης**: Το `Format` Η ιδιότητα καθορίζει τον τύπο αρχείου προορισμού για μετατροπή.

### Μετατροπή MOBI σε PSD
**Επισκόπηση**: Εδώ μετατρέπετε το φορτωμένο αρχείο MOBI σε έγγραφο PSD.

1. **Ρύθμιση διαδρομής εξόδου και ροής**
   ```csharp
   using System.IO;

   string outputFolder = "YOUR_OUTPUT_DIRECTORY"; // Αντικατάσταση με την επιθυμητή διαδρομή
   string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");

   Func<SavePageContext, Stream> getPageStream = savePageContext => 
       new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
   ```
2. **Εκτέλεση μετατροπής**
   ```csharp
   converter.Convert(getPageStream, options);

   // Διασφάλιση της σωστής διάθεσης των πόρων
   if (getPageStream != null)
   {
       getPageStream.Invoke(null); 
   }
   ```
3. **Εξήγηση**: `getPageStream` είναι μια συνάρτηση delegate που δημιουργεί μια ροή αρχείων για κάθε σελίδα που έχει μετατραπεί.
