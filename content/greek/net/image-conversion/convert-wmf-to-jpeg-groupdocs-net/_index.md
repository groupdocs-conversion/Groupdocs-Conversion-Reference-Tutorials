### Επισκόπηση χαρακτηριστικών: Μετατροπή WMF σε JPEG
Αυτή η λειτουργία μετατρέπει ένα μετααρχείο των Windows (.wmf) σε αρχείο εικόνας JPEG χρησιμοποιώντας το GroupDocs.Conversion. Αυτή η μετατροπή είναι ιδιαίτερα χρήσιμη σε σενάρια όπου τα διανυσματικά γραφικά πρέπει να μετατραπούν σε εικόνες raster για λόγους συμβατότητας ή κοινής χρήσης.

#### Βήμα 1: Ορισμός καταλόγου εξόδου και προτύπου αρχείου
Αρχικά, ορίστε τη διαδρομή και το πρότυπο του καταλόγου εξόδου για την ονομασία των αρχείων JPEG:
```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", "converted");
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");

// Δημιουργήστε μια συνάρτηση για τη δημιουργία FileStream για κάθε σελίδα.
Func<SavePageContext, Stream> getPageStream = savePageContext =>
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

#### Βήμα 2: Φόρτωση του αρχείου WMF προέλευσης
Φορτώστε το αρχείο WMF πηγής χρησιμοποιώντας το `Converter` τάξη:
```csharp
using (Converter converter = new Converter(Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.wmf")))
{
    // Στη συνέχεια, θα ορίσουμε τις επιλογές μετατροπής.
}
```

#### Βήμα 3: Ορισμός επιλογών μετατροπής για μορφή JPEG
Καθορίστε τη μορφή εξόδου και τυχόν πρόσθετες ρυθμίσεις που απαιτούνται:
```csharp
ImageConvertOptions options = new ImageConvertOptions 
{ 
    Format = GroupDocs.Conversion.FileTypes.ImageFileType.Jpg // Καθορίστε JPEG ως μορφή εξόδου.
};
```

#### Βήμα 4: Εκτελέστε τη μετατροπή
Εκτελέστε τη μετατροπή χρησιμοποιώντας τις καθορισμένες επιλογές και τη συνάρτηση ροής:
```csharp
converter.Convert(getPageStream, options);
```
