## Οδηγός Εφαρμογής
### Χαρακτηριστικό: Μετατροπή VSTM σε PSD
Αυτή η λειτουργία εστιάζει στη μετατροπή ενός προτύπου σχεδίασης με δυνατότητα μακροεντολών του Visio σε ένα έγγραφο του Adobe Photoshop.
#### Βήμα 1: Ορίστε τον κατάλογο εξόδου και το πρότυπο αρχείου
Ρυθμίστε τον κατάλογο εξόδου για την αποθήκευση των αρχείων που έχουν μετατραπεί. Καθορίστε ένα πρότυπο ονομασίας για κάθε αρχείο σελίδας:
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");
```
#### Βήμα 2: Δημιουργήστε ροές για κάθε σελίδα
Ορίστε μια συνάρτηση για τη δημιουργία μιας ροής για κάθε σελίδα που έχει μετατραπεί. Αυτό διασφαλίζει ότι κάθε αρχείο PSD δημιουργείται σωστά:
```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```
#### Βήμα 3: Φορτώστε το αρχείο VSTM προέλευσης και ορίστε τις επιλογές μετατροπής
Χρησιμοποιήστε το `Converter` κλάση για να φορτώσετε το αρχείο .vstm. Καθορίστε επιλογές μετατροπής για τη μορφή PSD:
```csharp
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY\\sample.vstm"))
{
    var options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd };
    
    // Μετατρέψτε το VSTM σε PSD.
    converter.Convert(getPageStream, options);
}
```
**Εξήγηση:**
- `ImageConvertOptions` Καθορίζει ότι η μορφή εξόδου θα πρέπει να είναι PSD.
- Ο `converter.Convert()` Η μέθοδος χειρίζεται τη διαδικασία μετατροπής χρησιμοποιώντας την καθορισμένη συνάρτηση ροής.
