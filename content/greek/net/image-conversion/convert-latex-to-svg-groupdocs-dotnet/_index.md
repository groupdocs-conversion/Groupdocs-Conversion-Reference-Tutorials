## Οδηγός Εφαρμογής

### Μετατροπή LaTeX σε SVG

Η μετατροπή LaTeX σε SVG σάς επιτρέπει να αξιοποιήσετε την επεκτασιμότητα των διανυσματικών γραφικών χωρίς να χάσετε την ποιότητα. Αυτή η λειτουργία είναι ιδιαίτερα χρήσιμη για ακαδημαϊκές δημοσιεύσεις και παρουσιάσεις όπου η ακρίβεια είναι το κλειδί.

#### Φόρτωση του αρχείου TEX πηγής
```csharp
using System.IO;
string YOUR_DOCUMENT_DIRECTORY = "YOUR_DOCUMENT_DIRECTORY";  // Ορίστε τη διαδρομή του καταλόγου εγγράφων σας
// Φόρτωση του αρχείου .tex πηγής
going (var converter = new Converter(Path.Combine(YOUR_DOCUMENT_DIRECTORY, "sample.tex")))
{
    // Η διαδικασία μετατροπής θα πραγματοποιηθεί στα ακόλουθα βήματα
}
```
**Εξήγηση**: Το `Converter` η κλάση αρχικοποιείται με την πλήρη διαδρομή της `.tex` αρχείο. Αυτό δημιουργεί το περιβάλλον για τις επόμενες λειτουργίες μετατροπής.

#### Καθορισμός επιλογών μετατροπής
```csharp
// Καθορισμός επιλογών μετατροπής σε μορφή SVG
var options = new PageDescriptionLanguageConvertOptions 
{
    Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Svg
};
```
**Εξήγηση**: Εδώ, ορίζουμε `PageDescriptionLanguageConvertOptions` και ορίστε τη μορφή στόχου ως SVG. Αυτή η διαμόρφωση διασφαλίζει ότι η έξοδος θα είναι σε μορφή διανυσματικών γραφικών.

#### Εκτέλεση μετατροπής
```csharp
// Ορίστε τη διαδρομή του αρχείου εξόδου για το μετατρεπόμενο SVG
string outputFile = Path.Combine("YOUR_OUTPUT_DIRECTORY", "tex-converted-to.svg");

// Εκτελέστε μετατροπή και αποθηκεύστε το αρχείο SVG που προκύπτει
converter.Convert(outputFile, options);
```
**Εξήγηση**: Το `Convert` Η μέθοδος λαμβάνει δύο παραμέτρους: τη διαδρομή του αρχείου προορισμού και τις επιλογές μετατροπής. Αυτό το βήμα εκτελεί στην πραγματικότητα τη μετατροπή από LaTeX σε SVG.

#### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι το δικό σας `.tex` τα αρχεία είναι σωστά μορφοποιημένα και χωρίς σφάλματα πριν επιχειρήσετε τη μετατροπή.
- Βεβαιωθείτε ότι έχουν εκχωρηθεί όλα τα απαραίτητα δικαιώματα για την ανάγνωση και την εγγραφή αρχείων στις διαδρομές καταλόγου σας.
