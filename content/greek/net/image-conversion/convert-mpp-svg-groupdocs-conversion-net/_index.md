##### 1. Ορίστε τη διαδρομή εγγράφου

string documentPath = "path/to/your/document.mpp";

Δημιουργήστε μια παρουσία του `Converter` κλάση με τη διαδρομή εγγράφου:
```csharp
using (var converter = new GroupDocs.Conversion.Converter(documentPath))
{
    // Το αντικείμενο μετατροπέα είναι πλέον έτοιμο για λειτουργίες μετατροπής.
}
```
*Γιατί αυτό το βήμα;*
Η αρχικοποίηση του μετατροπέα με το αρχείο MPP σας δημιουργεί το περιβάλλον για τις επόμενες ενέργειες μετατροπής.

### Μετατροπή MPP σε SVG
#### Επισκόπηση
Αυτή η λειτουργία σάς καθοδηγεί στη μετατροπή ενός αρχείου MPP σε μορφή SVG, βελτιώνοντας την οπτική αναπαράσταση και τη συμβατότητα σε όλες τις πλατφόρμες.

#### Βήματα για την εφαρμογή
##### 1. Ρύθμιση διαδρομής εξόδου
Ορίστε πού θα πρέπει να αποθηκευτεί το αρχείο SVG που έχετε μετατρέψει:
```csharp
string outputFolder = "path/to/output/directory";
string outputFile = System.IO.Path.Combine(outputFolder, "mpp-converted-to.svg");
```

##### 2. Φόρτωση αρχείου MPP προέλευσης
Βεβαιωθείτε ότι η διαδρομή του αρχείου MPP προέλευσης έχει οριστεί σωστά πριν ξεκινήσετε τη μετατροπή:
```csharp
string documentPath = "path/to/your/document.mpp";
using (var converter = new GroupDocs.Conversion.Converter(documentPath))
{
    // Θα ακολουθήσουν οι εργασίες μετατροπής.
}
```

##### 3. Ορίστε Επιλογές Μετατροπής
Ορίστε τις απαραίτητες επιλογές για τη μετατροπή σε μορφή SVG:
```csharp
PageDescriptionLanguageConvertOptions options = new PageDescriptionLanguageConvertOptions 
{ 
    Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Svg 
};
```
*Γιατί να επιλέξετε αυτές τις ρυθμίσεις;*
Ο `PageDescriptionLanguageConvertOptions` Η κλάση επιτρέπει τον καθορισμό λεπτομερών παραμέτρων μετατροπής, διασφαλίζοντας ότι το SVG εξόδου πληροί τις απαιτήσεις μορφοποίησης.

##### 4. Εκτελέστε μετατροπή
Εκτελέστε τη μετατροπή και αποθηκεύστε το αποτέλεσμα:
```csharp
converter.Convert(outputFile, options);
```
