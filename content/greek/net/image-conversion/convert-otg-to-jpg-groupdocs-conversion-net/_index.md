    // Περαιτέρω βήματα θα ακολουθήσουν εδώ

### Ρύθμιση επιλογών μετατροπής

Ορίστε τις ρυθμίσεις μετατροπής χρησιμοποιώντας `ImageConvertOptions`, καθορίζοντας τη μορφή εξόδου ως JPEG.

**Ρύθμιση παραμέτρων επιλογών μετατροπής:**
```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Jpg };
```

### Δημιουργία ροών εξόδου

Χειριστείτε κάθε σελίδα του εγγράφου OTG δημιουργώντας δυναμικά ροές, διασφαλίζοντας ότι κάθε σελίδα αποθηκεύεται ως μεμονωμένο αρχείο JPG.

**Ορισμός χειριστή ροής σελίδας:**
```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

### Εκτέλεση της μετατροπής

Εκτελέστε τη μετατροπή χρησιμοποιώντας το `Convert` μέθοδος.

**Εκτέλεση μετατροπής:**
```csharp
converter.Convert(getPageStream, options);
```
