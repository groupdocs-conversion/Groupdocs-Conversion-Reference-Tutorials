## Οδηγός Εφαρμογής

### Φόρτωση αρχείου VST πηγής
Η φόρτωση ενός προτύπου σχεδίασης του Visio είναι το πρώτο σας βήμα. Αυτή η λειτουργία δείχνει πώς να φορτώσετε ένα αρχείο VST πηγής χρησιμοποιώντας το GroupDocs.Conversion:

#### Βήμα 1: Ορισμός διαδρομής εγγράφου
Ορίστε τη διαδρομή όπου βρίσκεται το αρχείο VST.
```csharp
string documentPath = "path/to/your/sample.vst";
```

#### Βήμα 2: Αρχικοποίηση μετατροπέα
Δημιουργήστε μια παρουσία του `Converter` για να εργαστείτε με το αρχείο σας.
```csharp
using (Converter converter = new Converter(documentPath))
{
    // Το αρχείο VST πηγής έχει πλέον φορτωθεί και είναι έτοιμο για μετατροπή.
}
```
Αυτό το βήμα διασφαλίζει ότι το αρχείο VST είναι προσβάσιμο και προετοιμασμένο για περαιτέρω λειτουργίες.

### Ορισμός επιλογών μετατροπής για μορφή JPG
Για να μετατρέψετε το αρχείο σας σε JPG, διαμορφώστε συγκεκριμένες επιλογές:

#### Βήμα 1: Δημιουργία ImageConvertOptions
Ορίστε τις απαραίτητες παραμέτρους για να καθορίσετε τη μορφή εξόδου.
```csharp
using GroupDocs.Conversion.Options.Convert;

ImageConvertOptions options = new ImageConvertOptions 
{
    Format = GroupDocs.Conversion.FileTypes.ImageFileType.Jpg // Έξοδος ως JPG
};
```
Ο `ImageConvertOptions` Η κλάση σάς επιτρέπει να ορίσετε διάφορες ρυθμίσεις μετατροπής, όπως τη μορφή και την ποιότητα εξόδου.

### Μετατροπή VST σε JPG
Τώρα είναι η ώρα να εκτελέσετε την πραγματική μετατροπή από VST σε JPG:

#### Βήμα 1: Ορισμός φακέλου και προτύπου εξόδου
Προετοιμάστε πού θα αποθηκευτούν τα αρχεία που έχετε μετατρέψει.
```csharp
string outputFolder = "path/to/your/output";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");
Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```
Αυτό το βήμα ορίζει τον προορισμό εξόδου για τις εικόνες που έχετε μετατρέψει.

#### Βήμα 2: Εκτέλεση μετατροπής
Χρησιμοποιήστε τις επιλογές που ορίστηκαν προηγουμένως για να μετατρέψετε το αρχείο VST.
```csharp
using (Converter converter = new Converter(documentPath))
{
    // Μετατρέψτε και αποθηκεύστε κάθε σελίδα του VST ως ξεχωριστή εικόνα JPG
    converter.Convert(getPageStream, options);
}
```
Αυτό το βήμα επαναλαμβάνεται στις σελίδες του εγγράφου σας, μετατρέποντας την καθεμία σε μορφή JPG.
