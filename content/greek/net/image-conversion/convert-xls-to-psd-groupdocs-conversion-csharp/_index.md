## Οδηγός Εφαρμογής
Ας δούμε τη διαδικασία μετατροπής βήμα προς βήμα.
### Επισκόπηση Μετατροπών
Αυτή η ενότητα καλύπτει τη μετατροπή ενός αρχείου XLS σε μορφή PSD χρησιμοποιώντας το GroupDocs.Conversion. Θα χειριστούμε τις ροές αρχείων, θα διαμορφώσουμε τις ρυθμίσεις μετατροπής και θα εκτελέσουμε τη μετατροπή.
#### Βήμα 1: Ρύθμιση καταλόγου εξόδου και προτύπου αρχείου
Ορίστε πού θα αποθηκευτούν τα αρχεία που έχουν μετατραπεί:
```csharp
string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");
```
Αυτή η ρύθμιση διασφαλίζει ότι κάθε σελίδα του αρχείου XLS σας αποθηκεύεται ως ξεχωριστό PSD.
#### Βήμα 2: Δημιουργία συνάρτησης ροής συγκεκριμένης σελίδας
Δημιουργήστε ροές για κάθε σελίδα που έχει μετατραπεί χρησιμοποιώντας μια συνάρτηση λάμδα:
```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext =>
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```
Αυτή η συνάρτηση δημιουργεί μια ροή αρχείων για κάθε σελίδα στον κατάλογο εξόδου.
#### Βήμα 3: Φόρτωση και μετατροπή αρχείου XLS
Φορτώστε το έγγραφο προέλευσης και μετατρέψτε το χρησιμοποιώντας τις καθορισμένες επιλογές:
```csharp
using (Converter converter = new Converter(@"YOUR_DOCUMENT_DIRECTORY\sample.xls"))
{
    ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Psd };
    converter.Convert(getPageStream, options);
}
```
Εδώ, `ImageConvertOptions` καθορίζει τη μορφή προορισμού ως PSD.
