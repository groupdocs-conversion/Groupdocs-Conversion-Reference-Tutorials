### Μετατροπή αρχείου BMP σε μορφή TEX

Αυτή η λειτουργία χρησιμοποιεί το GroupDocs.Conversion για να μετατρέψει εικόνες BMP σε έγγραφα TEX απρόσκοπτα.

#### Φόρτωση του αρχείου BMP προέλευσης
Αρχικά, φορτώστε το αρχείο BMP χρησιμοποιώντας το `Converter` κλάση. Αυτό το βήμα αρχικοποιεί τη διαδικασία μετατροπής:

```csharp
using (var converter = new GroupDocs.Conversion.Converter(inputFile))
{
    // Η λογική μετατροπής ισχύει εδώ
}
```

#### Ορισμός επιλογών μετατροπής για τη μορφή TEX
Στη συνέχεια, ορίστε τη μορφή εξόδου ρυθμίζοντας `PageDescriptionLanguageConvertOptions`:

```csharp
PageDescriptionLanguageConvertOptions options = new PageDescriptionLanguageConvertOptions { Format = PageDescriptionLanguageFileType.Tex };
```

Ο `options.Format` Η παράμετρος καθορίζει ότι μετατρέπουμε σε TEX. Αυτό κατευθύνει τη διαδικασία μετατροπής.

#### Μετατροπή και Αποθήκευση
Τέλος, εκτελέστε τη μετατροπή και αποθηκεύστε την έξοδο:

```csharp
converter.Convert(outputFile, options);
```

Αυτή η μέθοδος μετατρέπει το αρχείο BMP σε μορφή TEX και το αποθηκεύει στην καθορισμένη τοποθεσία.
