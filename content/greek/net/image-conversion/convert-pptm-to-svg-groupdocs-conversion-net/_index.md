### Βασική Αρχικοποίηση
Δείτε πώς μπορείτε να αρχικοποιήσετε και να ρυθμίσετε το GroupDocs.Conversion στο έργο σας C#:
```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

namespace PptmToSvgConversion
{
class Program
{
    static void Main(string[] args)
    {
        // Διαδρομή προς το αρχείο προέλευσης PPTM και τον κατάλογο εξόδου
        string inputFilePath = "YOUR_DOCUMENT_DIRECTORY\\sample.pptm";
        string outputFolder = "YOUR_OUTPUT_DIRECTORY";
        string outputFile = System.IO.Path.Combine(outputFolder, "pptm-converted-to.svg");

        using (var converter = new Converter(inputFilePath))
        {
            // Καθορισμός επιλογών μετατροπής για τη μορφή SVG
            var options = new PageDescriptionLanguageConvertOptions { Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Svg };
            
            // Μετατρέψτε και αποθηκεύστε το αρχείο PPTM ως αρχείο SVG
            converter.Convert(outputFile, options);
        }
    }
}
```
Σε αυτό το απόσπασμα κώδικα:
- Αρχικοποιούμε ένα `Converter` αντικείμενο με τη διαδρομή προς το αρχείο PPTM μας.
- Ο `PageDescriptionLanguageConvertOptions` Η κλάση καθορίζει ότι θέλουμε να μετατρέψουμε σε μορφή SVG.

**1. Φορτώστε το αρχείο PPTM πηγής**

Στη συνέχεια, ορίστε τις επιλογές μετατροπής για να βεβαιωθείτε ότι η έξοδος είναι σε μορφή SVG:
