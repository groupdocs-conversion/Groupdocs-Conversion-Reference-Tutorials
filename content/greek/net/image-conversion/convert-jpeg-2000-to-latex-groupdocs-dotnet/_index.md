### Βασική Αρχικοποίηση και Ρύθμιση

Μόλις εγκατασταθεί, αρχικοποιήστε το GroupDocs.Conversion API στο έργο σας C# ως εξής:

```csharp
using System;
using GroupDocs.Conversion;

class Program
{
    static void Main(string[] args)
    {
        string sourceFilePath = "YOUR_DOCUMENT_DIRECTORY/sample.jp2";
        string outputFolder = "YOUR_OUTPUT_DIRECTORY";
        string outputFile = Path.Combine(outputFolder, "jp2-converted-to.tex");

        // Φόρτωση του αρχείου JP2 πηγής
        using (var converter = new GroupDocs.Conversion.Converter(sourceFilePath))
        {
            // Ορισμός επιλογών μετατροπής για τη μορφή TEX
            var options = new PageDescriptionLanguageConvertOptions { Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Tex };
            
            // Μετατροπή και αποθήκευση του αρχείου ως εγγράφου TEX
            converter.Convert(outputFile, options);
        }
    }
}
```

Αυτό το απόσπασμα δείχνει πώς να φορτώσετε ένα αρχείο JP2 και να το μετατρέψετε σε TEX χρησιμοποιώντας το GroupDocs.Conversion.
