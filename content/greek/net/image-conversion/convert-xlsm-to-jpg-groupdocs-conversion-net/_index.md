# Μετατροπή XLSM σε JPG με το GroupDocs.Conversion .NET
## Εισαγωγή
Θέλετε να μετατρέψετε απρόσκοπτα τις μακροεντολές του Excel (XLSM) σε οπτικά στιγμιότυπα με τη μορφή εικόνων; Η μετατροπή αρχείων XLSM σε JPG μπορεί να είναι ζωτικής σημασίας για την κοινή χρήση δεδομένων με χρήστες που δεν χρησιμοποιούν το Excel ή για την ενσωμάτωση υπολογιστικών φύλλων σε παρουσιάσεις και έγγραφα. Αυτό το σεμινάριο σας καθοδηγεί στη χρήση του GroupDocs.Conversion .NET, μιας ισχυρής βιβλιοθήκης που απλοποιεί αυτήν τη διαδικασία μετατροπής.

**Τι θα μάθετε:**
- Πώς να φορτώσετε ένα αρχείο XLSM χρησιμοποιώντας το GroupDocs.Conversion
- Ρύθμιση επιλογών μετατροπής JPG με το API
- Εκτέλεση της πραγματικής μετατροπής από XLSM σε JPG
- Πρακτικές εφαρμογές και ζητήματα απόδοσης

## Οδηγός Εφαρμογής
Θα αναλύσουμε τη διαδικασία μετατροπής σε βήματα χρησιμοποιώντας τις λειτουργίες του GroupDocs.Conversion.
### Φόρτωση του αρχείου XLSM πηγής
Αρχικά, φορτώστε το αρχείο XLSM:
#### Ορισμός καταλόγου εγγράφων
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
```
#### Αρχικοποίηση και φόρτωση του αρχείου XLSM
```csharp
using (Converter converter = new Converter(Path.Combine(documentDirectory, "sample.xlsm")))
{
    // Το αντικείμενο μετατροπέα είναι τώρα έτοιμο για μετατροπή.
}
```
Αυτό το απόσπασμα κώδικα αρχικοποιεί ένα `Converter` παράδειγμα καθορίζοντας τη διαδρομή του αρχείου XLSM.
### Ορισμός επιλογών μετατροπής για μορφή JPG
Στη συνέχεια, διαμορφώστε τη διαδικασία μετατροπής:
#### Ορισμός καταλόγου εξόδου
```csharp
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
```
#### Ρύθμιση παραμέτρων επιλογών μετατροπής εικόνας
```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Jpg };
```
Εδώ, `options` έχουν ρυθμιστεί για να μετατρέψουν το αρχείο XLSM σε εικόνες μορφής JPG.
### Μετατροπή αρχείου XLSM σε μορφή JPG
Εκτελέστε την πραγματική μετατροπή:
#### Ορισμός προτύπου ονόματος αρχείου εξόδου
```csharp
string outputFileTemplate = Path.Combine(outputDirectory, "converted-page-{0}.jpg");
```
#### Δημιουργία συνάρτησης ροής σελίδας
```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext =>
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```
Αυτή η συνάρτηση δημιουργεί μια ροή για κάθε σελίδα που μετατρέπεται.
#### Εκτέλεση μετατροπής
```csharp
using (Converter converter = new Converter(Path.Combine(documentDirectory, "sample.xlsm")))
{
    converter.Convert(getPageStream, options);
}
```
