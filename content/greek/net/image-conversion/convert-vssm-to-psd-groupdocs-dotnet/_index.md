## Οδηγός Εφαρμογής

### Φόρτωση και μετατροπή αρχείου VSSM σε μορφή PSD

Αυτή η λειτουργία σάς επιτρέπει να φορτώσετε ένα αρχείο με δυνατότητα μακροεντολών του Microsoft Visio (.vssm) και να το μετατρέψετε σε μορφή εγγράφου Adobe Photoshop (.psd).

#### Βήμα 1: Φόρτωση του αρχείου VSSM προέλευσης
Φορτώστε το αρχείο .vssm χρησιμοποιώντας το GroupDocs.Conversion `Converter` τάξη.

```csharp
using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_VSSM"))
{
    // Περαιτέρω βήματα μετατροπής θα ακολουθήσουν εδώ
}
```

#### Βήμα 2: Ορισμός επιλογών μετατροπής για τη μορφή PSD
Ορίστε τη μορφή εικόνας στην οποία θέλετε να μετατρέψετε το αρχείο σας χρησιμοποιώντας `ImageConvertOptions`.

```csharp
var options = new ImageConvertOptions { Format = ImageFileType.Psd };
```

**Εξήγηση:** Ο `Format` Η ιδιότητα καθορίζει ότι η έξοδος θα είναι σε μορφή PSD.

#### Βήμα 3: Ρύθμιση παραμέτρων ροής εξόδου
Δημιουργήστε μια συνάρτηση που καθορίζει τον τρόπο με τον οποίο αποθηκεύεται κάθε σελίδα σε μια ροή. Αυτό σας επιτρέπει να διαχειρίζεστε αποτελεσματικά την ονομασία και την αποθήκευση αρχείων.

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = System.IO.Path.Combine(outputFolder, "converted-page-{0}.psd");

Func<SavePageContext, Stream> getPageStream = savePageContext =>
{
    return new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
};
```

**Εξήγηση:** Αυτή η συνάρτηση λάμδα μορφοποιεί το όνομα αρχείου εξόδου και δημιουργεί μια νέα ροή αρχείων για κάθε σελίδα.

#### Βήμα 4: Εκτέλεση μετατροπής
Τέλος, εκτελέστε τη διαδικασία μετατροπής χρησιμοποιώντας το `Convert` μέθοδος.

```csharp
converter.Convert(getPageStream, options);
```

**Εξήγηση:** Ο `Convert` Η μέθοδος χρησιμοποιεί τις παρεχόμενες επιλογές και τον χειριστή ροής για να εκτελέσει τη μετατροπή αρχείου.
