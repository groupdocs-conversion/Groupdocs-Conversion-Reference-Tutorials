### Φόρτωση και μετατροπή XLSM σε PNG

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε τα αρχεία XLSM απευθείας σε μορφή PNG. Αυτό είναι χρήσιμο για την οπτική κοινή χρήση δεδομένων ή την ενσωμάτωσή τους σε αναφορές.

##### Βήμα 1: Φόρτωση του αρχείου XLSM
Αρχικά, βεβαιωθείτε ότι η διαδρομή του αρχείου σας έχει καθοριστεί σωστά:

```csharp
string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "yourfile.xlsm");
using (Converter converter = new Converter(inputFilePath))
{
    // Η λογική μετατροπής θα εφαρμοστεί εδώ.
}
```

##### Βήμα 2: Ρύθμιση επιλογών μετατροπής
Ορίστε τις επιλογές μετατροπής για να καθορίσετε ότι θέλετε μια έξοδο PNG:

```csharp
var convertOptions = new ImageConvertOptions
{
    Format = ImageFileType.Png // Ορίστε τη μορφή εξόδου ως PNG
};
```

##### Βήμα 3: Εκτελέστε τη μετατροπή
Εκτελέστε τη μετατροπή και αποθηκεύστε την έξοδο:

```csharp
string outputFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "output.png");
converter.Convert(outputFilePath, convertOptions);
```

### Παράμετροι και Σκοποί Μεθόδου
- `Converter`: Φορτώνει το αρχείο XLSM σας.
- `ImageConvertOptions`: Ρυθμίζει τη μορφή εξόδου.
- `Convert()`: Εκτελεί τη διαδικασία μετατροπής.

### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι όλες οι διαδρομές έχουν ρυθμιστεί σωστά για να αποφύγετε `FileNotFoundException`.
