## Οδηγός Εφαρμογής

### Βήμα 1: Προετοιμάστε το περιβάλλον σας

Βεβαιωθείτε ότι το έργο σας αναφέρεται στο GroupDocs.Conversion και προετοιμάστε τους απαραίτητους χώρους ονομάτων:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;
```

### Βήμα 2: Διαμόρφωση ρυθμίσεων εξόδου

Ορίστε τον φάκελο εξόδου για τα αρχεία PNG χρησιμοποιώντας ένα πρότυπο:

```csharp
string outputFolder = "path/to/output/folder";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.png");

Func<SavePageContext, Stream> getPageStream = savePageContext =>
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

### Βήμα 3: Ρύθμιση επιλογών μετατροπής

Καθορίστε ότι μετατρέπετε σε μορφή PNG:

```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Png };
```

### Βήμα 4: Εκτελέστε τη μετατροπή

Χρησιμοποιήστε το `Converter` αντικείμενο για να αποθηκεύσετε κάθε σελίδα ως ξεχωριστό αρχείο PNG:

```csharp
using (Converter converter = new Converter("path/to/your/sample.mpx"))
{
    converter.Convert(getPageStream, options);
}
```

**Συμβουλές αντιμετώπισης προβλημάτων**

- Βεβαιωθείτε ότι η διαδρομή του αρχείου MPX είναι σωστή.
- Επαληθεύστε τα δικαιώματα εγγραφής στον κατάλογο εξόδου.
