- Υλοποιήστε ισχυρή λογική μετατροπής σε C#.

## Οδηγός Εφαρμογής

Τώρα που όλα είναι έτοιμα, ας μετατρέψουμε τα αρχεία POTM σε JPG.

### Φόρτωση και μετατροπή ενός αρχείου POTM

#### Επισκόπηση

Ο στόχος εδώ είναι να φορτώσετε ένα αρχείο POTM χρησιμοποιώντας το GroupDocs.Conversion και να το μετατρέψετε σε εικόνα JPG. Αυτή η διαδικασία μπορεί να δημιουργήσει προεπισκοπήσεις ή να μοιραστεί πρότυπα PowerPoint με δυνατότητα μακροεντολών σε προσβάσιμη μορφή.

#### Βήμα προς βήμα εφαρμογή

##### 1. Ορισμός διαδρομής καταλόγου εξόδου

Ορίστε τη διαδρομή όπου θέλετε να αποθηκευτούν τα αρχεία που έχετε μετατρέψει:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");
```

Εξασφαλίζω `outputFolder` αντικαθίσταται με την πραγματική διαδρομή καταλόγου σας.

##### 2. Δημιουργήστε μια συνάρτηση για να λάβετε ροή σελίδας

Αυτή η συνάρτηση χειρίζεται την αποθήκευση κάθε σελίδας POTM ως μεμονωμένη εικόνα JPG:

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

##### 3. Φόρτωση και μετατροπή χρησιμοποιώντας το GroupDocs.Conversion

Χρησιμοποιήστε το `Converter` κλάση για να φορτώσετε το αρχείο POTM και να το μετατρέψετε:

```csharp
using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_POTM"))
{
    // Ορισμός επιλογών μετατροπής για μορφή JPG
    ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Jpg };

    // Εκτελέστε τη μετατροπή
    converter.Convert(getPageStream, options);
}
```

Αντικαθιστώ `"YOUR_DOCUMENT_DIRECTORY/SAMPLE_POTM"` με την πραγματική διαδρομή του αρχείου σας.

**Συμβουλές αντιμετώπισης προβλημάτων:**
- Βεβαιωθείτε ότι όλες οι διαδρομές είναι σωστές και προσβάσιμες.
- Επαληθεύστε ότι έχετε δικαιώματα εγγραφής για τον κατάλογο εξόδου.
