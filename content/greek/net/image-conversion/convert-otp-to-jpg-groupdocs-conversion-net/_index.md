### Λειτουργία 2: Ρύθμιση επιλογών μετατροπής για μορφή JPG

**Επισκόπηση:** Αυτή η λειτουργία ορίζει τις απαραίτητες επιλογές για τη μετατροπή αρχείων σε μορφή JPEG.

#### Βήμα 2: Ρύθμιση παραμέτρων του ImageConvertOptions

Καθορίστε ότι θέλετε να μετατρέψετε την έξοδο σε JPEG:

```csharp

ImageConvertOptions jpgOptions = new ImageConvertOptions { Format = ImageFileType.Jpg };
```

**Εξήγηση:** Ο `ImageConvertOptions` Η κλάση επιτρέπει τον καθορισμό ρυθμίσεων μετατροπής, συμπεριλαμβανομένης της επιθυμητής μορφής.

### Χαρακτηριστικό 3: Ορισμός Συνάρτησης Ροής Εξόδου

**Επισκόπηση:** Ορίστε μια συνάρτηση που παρέχει μια ροή εξόδου για κάθε αρχείο που έχει μετατραπεί.

#### Βήμα 3: Δημιουργήστε μια συνάρτηση ροής εξόδου

Χρησιμοποιήστε αυτήν τη συνάρτηση για να διαχειριστείτε πού και πώς αποθηκεύεται κάθε σελίδα:

```csharp
using System.IO;
Func<SavePageContext, Stream> getPageStream = savePageContext =>
{
    string outputFileTemplate = Path.Combine("YOUR_OUTPUT_DIRECTORY", "converted-page-{0}.jpg");
    return new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
};
```

**Εξήγηση:** Αυτή η συνάρτηση δημιουργεί μια διαδρομή αρχείου για κάθε σελίδα και την γράφει στον καθορισμένο κατάλογο.
