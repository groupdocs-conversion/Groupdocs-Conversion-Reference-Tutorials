## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Μετατροπή αρχείων από VSTM σε PNG

Αυτή η λειτουργία επιδεικνύει τη μετατροπή ενός αρχείου Visio Macro-Enabled Drawing Template (.vstm) σε μορφή Portable Network Graphic (.png) χρησιμοποιώντας το GroupDocs.Conversion for .NET.

#### Επισκόπηση
Σε αυτήν την ενότητα, θα περιγράψουμε τα βήματα που απαιτούνται για τη μετατροπή αρχείων VSTM σε PNG. Αυτή η διαδικασία περιλαμβάνει τη φόρτωση του αρχείου προέλευσης, τον ορισμό επιλογών μετατροπής και την εκτέλεση της μετατροπής.

##### Βήμα 1: Ορισμός καταλόγου εξόδου

Αρχικά, καθορίστε πού θέλετε να αποθηκευτούν τα αρχεία PNG που έχουν μετατραπεί:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
```

##### Βήμα 2: Ρύθμιση προτύπου αρχείου εξόδου

Στη συνέχεια, ορίστε ένα πρότυπο για την ονομασία των αρχείων εξόδου σας. Αυτό διασφαλίζει ότι κάθε σελίδα του αρχείου VSTM αποθηκεύεται ως ξεχωριστή εικόνα PNG:

```csharp
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.png");
Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

##### Βήμα 3: Φόρτωση του αρχείου VSTM προέλευσης

Φορτώστε το αρχείο VSTM χρησιμοποιώντας το GroupDocs.Conversion. Αντικαταστήστε `'YOUR_DOCUMENT_DIRECTORY/sample.vstm'` με τη διαδρομή προς το πραγματικό σας έγγραφο:

```csharp
using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/sample.vstm"))
{
    // Ο κώδικας μετατροπής θα τοποθετηθεί εδώ
}
```

##### Βήμα 4: Ορισμός επιλογών μετατροπής

Ορίστε τις επιλογές μετατροπής για τη μορφή PNG. Αυτό το βήμα ρυθμίζει τον τρόπο με τον οποίο κάθε σελίδα του αρχείου VSTM μετατρέπεται σε PNG:

```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Png };
```

##### Βήμα 5: Εκτελέστε τη μετατροπή

Τέλος, εκτελέστε τη διαδικασία μετατροπής χρησιμοποιώντας το `converter.Convert` μέθοδος. Αυτό θα δημιουργήσει αρχεία PNG με βάση τις διαμορφώσεις σας:

```csharp
converter.Convert(getPageStream, options);
```
