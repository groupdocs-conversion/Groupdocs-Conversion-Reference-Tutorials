## Οδηγός Εφαρμογής

### Μετατροπή PPTX σε JPG

#### Επισκόπηση
Αυτή η λειτουργία επιδεικνύει τη φόρτωση ενός αρχείου PowerPoint (PPTX) και τη μετατροπή κάθε διαφάνειας σε μορφή JPG. Αυτό είναι χρήσιμο για τη δημιουργία μικρογραφιών ή την ενσωμάτωση παρουσιάσεων σε εφαρμογές ιστού.

**Βήμα 1: Ορισμός διαδρομών**
Αρχικά, καθορίστε διαδρομές για το έγγραφο προέλευσης και τον κατάλογο εξόδου:
```csharp
string sourceFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.pptx");
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");
```

**Βήμα 2: Λειτουργία δημιουργίας ροής**
Δημιουργήστε μια συνάρτηση για να χειριστείτε τη ροή για κάθε σελίδα που έχει μετατραπεί:
```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext =>
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

**Βήμα 3: Φόρτωση και μετατροπή χρησιμοποιώντας το GroupDocs.Conversion**
Φορτώστε το αρχείο PPTX χρησιμοποιώντας το `Converter` κλάση και ρύθμιση επιλογών μετατροπής:
```csharp
using (Converter converter = new Converter(sourceFilePath))
{
    ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Jpg };
    converter.Convert(getPageStream, options);
}
```

**Εξήγηση:**
- **`Converter`:** Φορτώνει το αρχείο PPTX.
- **`ImageConvertOptions`:** Διαμορφώνει ρυθμίσεις μετατροπής όπως η μορφή εξόδου (JPG).
- **`getPageStream`:** Δημιουργεί μια ροή για κάθε διαφάνεια που μετατρέπεται σε JPG.

#### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι οι διαδρομές είναι σωστά καθορισμένες και προσβάσιμες.
- Βεβαιωθείτε ότι το GroupDocs.Conversion έχει εγκατασταθεί σωστά και αναφέρεται στο έργο σας.

### Ρύθμιση παραμέτρων επιλογών μετατροπής

#### Επισκόπηση
Η ρύθμιση παραμέτρων των επιλογών μετατροπής σάς επιτρέπει να καθορίσετε τη μορφή εξόδου και να προσαρμόσετε ρυθμίσεις όπως η ανάλυση ή η ποιότητα για τα αρχεία εικόνας. Αυτή η λειτουργία είναι ζωτικής σημασίας για την προσαρμογή των μετατροπών ώστε να ανταποκρίνονται σε συγκεκριμένες ανάγκες.

**Βήμα 1: Δημιουργία ImageConvertOptions**
Ορίστε τις παραμέτρους μετατροπής:
```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Jpg };
```

**Εξήγηση:**
- **`Format`:** Καθορίζει τη μορφή αρχείου εξόδου, σε αυτήν την περίπτωση, JPG.

Ορίζοντας αυτές τις επιλογές, μπορείτε να ελέγξετε τον τρόπο με τον οποίο οι διαφάνειες PPTX σας αποδίδονται ως εικόνες.
