## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Μετατροπή αρχείου EMLX σε μορφή PNG

Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε ένα αρχείο EMLX σε μια σειρά εικόνων PNG. Κάθε βήμα παρακάτω θα σας καθοδηγήσει στη διαδικασία.

#### Βήμα 1: Ορισμός προτύπου διαδρομής αρχείου εξόδου

Αρχικά, ρυθμίστε τον κατάλογο εξόδου και ορίστε πώς θα ονομάζεται η εικόνα PNG κάθε σελίδας:

```csharp
string outputFileTemplate = Path.Combine(outputDirectory, "converted-page-{0}.png");
```

#### Βήμα 2: Δημιουργήστε μια συνάρτηση για ροές σελίδων

Δημιουργήστε μια συνάρτηση που θα παρέχει μια ροή για κάθε σελίδα που έχει μετατραπεί. Αυτό διασφαλίζει ότι κάθε PNG αποθηκεύεται σωστά:

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

#### Βήμα 3: Αρχικοποίηση του μετατροπέα

Έχοντας έτοιμη τη διαδρομή αρχείου EMLX και τη ρύθμιση εξόδου, αρχικοποιήστε το `Converter` αντικείμενο:

```csharp
using (Converter converter = new Converter(Path.Combine(documentDirectory, "sample.emlx")))
{
    // Η διαδικασία μετατροπής θα εκτελεστεί εδώ
}
```

#### Βήμα 4: Ορισμός επιλογών μετατροπής για τη μορφή PNG

Καθορίστε ότι θέλετε να μετατρέψετε το έγγραφό σας σε μορφή PNG χρησιμοποιώντας `ImageConvertOptions`:

```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Png };
```

#### Βήμα 5: Εκτελέστε τη μετατροπή

Τέλος, εκτελέστε τη διαδικασία μετατροπής:

```csharp
converter.Convert(getPageStream, options);
```
