## Οδηγός Εφαρμογής

Τώρα, ας μετατρέψουμε ένα αρχείο Excel σε εικόνες PNG βήμα προς βήμα.

### Χαρακτηριστικό: Μετατροπή XLSX σε PNG
Αυτή η λειτουργία επιδεικνύει τη μετατροπή ενός ολόκληρου εγγράφου XLSX σε μια σειρά αρχείων PNG—ένα για κάθε σελίδα υπολογιστικού φύλλου.

#### Βήμα 1: Ορισμός διαδρομών και φόρτωση αρχείου
Καθορίστε την τοποθεσία του αρχείου προέλευσης Excel και πού θέλετε να αποθηκευτούν οι εικόνες εξόδου:
```csharp
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

// Ορίστε διαδρομές για καταλόγους εισόδου και εξόδου (αντικαταστήστε τις με τις πραγματικές διαδρομές καταλόγων σας)
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// Φόρτωση του αρχείου XLSX πηγής
using (Converter converter = new Converter(Path.Combine(documentDirectory, "sample.xlsx")))
{
    // Η λογική μετατροπής θα εφαρμοστεί εδώ.
}
```
#### Βήμα 2: Ορισμός επιλογών μετατροπής
Καθορίστε τη μετατροπή κάθε σελίδας Excel σε εικόνα PNG:
```csharp
// Ορίστε τις επιλογές μετατροπής για τη μορφή PNG
ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Png };
```
#### Βήμα 3: Ορισμός διαδρομής εξόδου και μετατροπή
Ορίστε ονόματα αρχείων εξόδου και, στη συνέχεια, εκτελέστε τη μετατροπή:
```csharp
// Ορίστε το πρότυπο διαδρομής αρχείου εξόδου για κάθε μετατροπή σελίδας
string outputFileTemplate = Path.Combine(outputDirectory, "converted-page-{0}.png");

// Δημιουργήστε μια συνάρτηση για τη δημιουργία ροών για την αποθήκευση σελίδων που έχουν μετατραπεί
Func<SavePageContext, Stream> getPageStream = savePageContext =>
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);

// Μετατρέψτε το έγγραφο XLSX σε εικόνες PNG χρησιμοποιώντας καθορισμένες επιλογές και γεννήτρια ροής εξόδου
converter.Convert(getPageStream, options);
```
