## Οδηγός Εφαρμογής

### Φόρτωση και μετατροπή αρχείου XLTX σε JPG
Αυτή η λειτουργία μετατρέπει ένα αρχείο XLTX σε μια σειρά εικόνων JPG, ιδανική για οπτική κοινή χρήση δεδομένων υπολογιστικών φύλλων.

**Βήμα 1: Ορισμός της διαδρομής καταλόγου εξόδου**
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
// Ορίστε πού θα αποθηκευτούν τα αρχεία που έχετε μετατρέψει.
```

**Βήμα 2: Ορίστε το πρότυπο για τα ονόματα αρχείων εξόδου**
```csharp
string outputFileTemplate = System.IO.Path.Combine(outputFolder, "converted-page-{0}.jpg");
// Αυτό το πρότυπο βοηθά στην ονομασία κάθε σελίδας του εγγράφου με έναν μοναδικό αριθμό.
```

**Βήμα 3: Δημιουργήστε μια ροή για κάθε σελίδα του αποτελέσματος μετατροπής**
```csharp
Func<SavePageContext, System.IO.Stream> getPageStream = savePageContext => new System.IO.FileStream(System.String.Format(outputFileTemplate, savePageContext.Page), System.IO.FileMode.Create);
// Αυτή η λειτουργία διασφαλίζει ότι κάθε σελίδα αποθηκεύεται ως ξεχωριστό αρχείο.
```

**Βήμα 4: Φορτώστε το αρχείο XLTX πηγής και ορίστε τις επιλογές μετατροπής**
```csharp
using (Converter converter = new Converter(filePath))
{
    // Ορίστε επιλογές μετατροπής για τη μορφή JPG
    ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Jpg };

    // Εκτελέστε τη μετατροπή από XLTX σε JPG
    converter.Convert(getPageStream, options);
}
```
