## Οδηγός Εφαρμογής

### Μετατροπή VSTX σε SVG

Μετατρέψτε αρχεία Visio σε κλιμακώσιμα διανυσματικά γραφικά, ιδανικά για εφαρμογές web ή οπτικές απαιτήσεις υψηλής ποιότητας.

#### Βήμα 1: Ορισμός διαδρομών για αρχεία εισόδου και εξόδου

Ορίστε καταλόγους για τα αρχεία προέλευσης (.vstx) και προορισμού (.svg):

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

string inputFile = Path.Combine(documentDirectory, "sample.vstx");
string outputFile = Path.Combine(outputDirectory, "vstx-converted-to.svg");
```

#### Βήμα 2: Φόρτωση του αρχείου VSTX προέλευσης

Χρησιμοποιήστε το GroupDocs.Conversion για να φορτώσετε το αρχείο Visio:

```csharp
using (var converter = new Converter(inputFile))
{
    // Συνεχίστε με τη μετατροπή στα επόμενα βήματα
}
```

#### Βήμα 3: Ρύθμιση επιλογών μετατροπής

Διαμόρφωση επιλογών για μετατροπή σε μορφή SVG:

```csharp
var options = new PageDescriptionLanguageConvertOptions { Format = PageDescriptionLanguageFileType.Svg };
```

#### Βήμα 4: Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο εξόδου

Εκτελέστε τη μετατροπή και αποθηκεύστε το αποτέλεσμα:

```csharp
converter.Convert(outputFile, options);
```

### Συμβουλές αντιμετώπισης προβλημάτων:

- Βεβαιωθείτε ότι έχετε τα απαραίτητα δικαιώματα για την ανάγνωση/εγγραφή αρχείων σε αυτούς τους καταλόγους.
