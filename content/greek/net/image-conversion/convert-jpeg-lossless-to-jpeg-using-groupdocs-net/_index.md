## Οδηγός Εφαρμογής

### Επισκόπηση λειτουργιών: Μετατροπή αρχείων JLS σε μορφή JPG
Αυτή η λειτουργία αξιοποιεί τη δύναμη του GroupDocs.Conversion for .NET για να μετατρέψει αρχεία JPEG Lossless σε μορφή JPEG, καθιστώντας τα πιο συμβατά με όλους και πιο φιλικά προς την αποθήκευση.

#### Βήμα 1: Ορισμός διαδρομών καταλόγου
Ορίστε διαδρομές καταλόγου για τα αρχεία εισόδου (JLS) και εξόδου (JPG):
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY/";
```
Αυτά τα placeholder θα σας βοηθήσουν να οργανώσετε πού θα βρείτε τα αρχεία προέλευσης και θα αποθηκεύσετε τις εικόνες που έχουν μετατραπεί.

#### Βήμα 2: Ορισμός διαδρομής αρχείου προέλευσης
Δημιουργήστε μια διαδρομή για το αρχείο JLS σας:
```csharp
string sourceFilePath = System.IO.Path.Combine(documentDirectory, "sample.jls");
```
Αυτό το βήμα καθορίζει την ακριβή τοποθεσία του αρχείου που θα μετατραπεί.

#### Βήμα 3: Ορισμός προτύπου ονομασίας αρχείου εξόδου
Ορίστε πώς θέλετε να ονομάζονται τα αρχεία JPG εξόδου σας:
```csharp
string outputFileTemplate = System.IO.Path.Combine(outputDirectory, "converted-page-{0}.jpg");
```
Ο `{0}` Το σύμβολο κράτησης θέσης επιτρέπει τη δυναμική αρίθμηση σελίδων κατά τη μετατροπή.

#### Βήμα 4: Δημιουργία FileStream για κάθε σελίδα
Δημιουργήστε μια ροή αρχείων για να χειριστείτε κάθε σελίδα που έχει μετατραπεί:
```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new System.IO.FileStream(string.Format(outputFileTemplate, savePageContext.Page), System.IO.FileMode.Create);
```
Αυτό διασφαλίζει ότι κάθε σελίδα εξόδου αποθηκεύεται ως μεμονωμένο αρχείο JPG.

#### Βήμα 5: Φόρτωση και μετατροπή του αρχείου JLS
Χρησιμοποιήστε το GroupDocs.Conversion για να φορτώσετε το αρχείο προέλευσης και να το μετατρέψετε:
```csharp
using (Converter converter = new Converter(sourceFilePath))
{
    ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Jpg };
    converter.Convert(getPageStream, options);
}
```
Αυτό το απόσπασμα δείχνει τη φόρτωση του αρχείου JLS, τον ορισμό επιλογών μετατροπής για τη μορφή JPG και την εκτέλεση της διαδικασίας μετατροπής.
