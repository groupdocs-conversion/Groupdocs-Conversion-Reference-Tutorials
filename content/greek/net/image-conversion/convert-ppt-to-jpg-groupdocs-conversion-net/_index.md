## Οδηγός Εφαρμογής

Αυτή η ενότητα χωρίζεται σε λογικές ενότητες ανά χαρακτηριστικό, παρέχοντας έναν οδηγό υλοποίησης βήμα προς βήμα.

### Φόρτωση αρχείου PPT πηγής

**Επισκόπηση**: Αυτό το μέρος δείχνει τη φόρτωση ενός αρχείου παρουσίασης PowerPoint για μετατροπή.

#### Βήμα 1: Αρχικοποίηση του αντικειμένου μετατροπέα
```csharp
using System;
using System.IO;
using GroupDocs.Conversion;

string documentPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.ppt");
Converter converter = new Converter(documentPath);
converter.Dispose();
```
**Εξήγηση**: Αρχικοποιούμε ένα `Converter` αντικείμενο με τη διαδρομή προς το αρχείο PPT σας. Αυτό το βήμα είναι κρίσιμο καθώς φορτώνει την παρουσίαση στη μνήμη για επεξεργασία.

### Ορισμός επιλογών μετατροπής για μορφή JPG

**Επισκόπηση**Εδώ, ορίζουμε και ορίζουμε επιλογές μετατροπής ειδικά προσαρμοσμένες για τη μετατροπή αρχείων σε μορφή JPG.

#### Βήμα 2: Ορισμός επιλογών μετατροπής εικόνας
```csharp
using System;
using GroupDocs.Conversion.Options.Convert;

ImageConvertOptions options = new ImageConvertOptions { Format = FileTypes.ImageFileType.Jpg };
Console.WriteLine("Conversion format set to: " + options.Format);
```
**Εξήγηση**: Το `ImageConvertOptions` Η κλάση σάς επιτρέπει να καθορίσετε τη μορφή εξόδου. Ορίζοντάς την σε `Jpg` διασφαλίζει ότι κάθε σελίδα της παρουσίασής σας μετατρέπεται σε εικόνα JPG.

### Μετατροπή PPT σε JPG

**Επισκόπηση**Αυτή η λειτουργία μετατρέπει κάθε διαφάνεια σε μια παρουσίαση PowerPoint σε ξεχωριστά αρχεία JPG.

#### Βήμα 3: Εκτέλεση μετατροπής
```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");

Func<SavePageContext, Stream> getPageStream = savePageContext => new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);

converter.Convert(getPageStream, options);
converter.Dispose();
```
**Εξήγηση**: Το `Convert` Η μέθοδος επαναλαμβάνεται σε κάθε διαφάνεια στην παρουσίαση και δημιουργεί ένα αρχείο JPG για αυτήν. Χρησιμοποιούμε μια συνάρτηση delegate για να ορίσουμε πώς πρέπει να αποθηκεύεται κάθε σελίδα.
