## Οδηγός Εφαρμογής
### Μετατροπή EMLX σε μορφή SVG
Αυτή η ενότητα θα σας καθοδηγήσει στη μετατροπή ενός αρχείου μηνύματος Apple Mail (.emlx) σε κλιμακούμενα διανυσματικά γραφικά (SVG).

#### Ορισμός διαδρομών για αρχεία εισόδου και εξόδου
Αρχικά, ρυθμίστε τους καταλόγους εισόδου και εξόδου:

```csharp
string inputDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// Ορίστε τις διαδρομές
string inputFile = Path.Combine(inputDirectory, "sample.emlx");
string outputFile = Path.Combine(outputDirectory, "emlx-converted-to.svg");
```

#### Φόρτωση και μετατροπή χρησιμοποιώντας το GroupDocs.Conversion
Φορτώστε το αρχείο EMLX και καθορίστε τις επιλογές μετατροπής για SVG:

```csharp
using (var converterInstance = new Converter(inputFile))
{
    // Καθορίστε τη μορφή εξόδου ως SVG
    var convertOptions = new PageDescriptionLanguageConvertOptions
    {
        Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Svg
    };
    
    // Μετατροπή και αποθήκευση αρχείου
    converterInstance.Convert(outputFile, convertOptions);
}
```

**Επεξήγηση παραμέτρων:**
- **Αρχείο εισόδου:** Διαδρομή προς το αρχείο προέλευσης EMLX.
- **Αρχείο εξόδου:** Διαδρομή προορισμού για την έξοδο SVG.
- **convertOptions.Μορφή:** Καθορίζει ότι ο στόχος μετατροπής είναι το SVG.
