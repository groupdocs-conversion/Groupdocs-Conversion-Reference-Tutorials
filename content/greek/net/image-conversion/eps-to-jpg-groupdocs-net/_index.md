## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Μετατροπή EPS σε JPG

Αυτή η λειτουργία σάς επιτρέπει να μετατρέπετε αρχεία EPS σε μορφή JPG απρόσκοπτα.

#### Βήμα 1: Προετοιμάστε το περιβάλλον σας
Βεβαιωθείτε ότι οι διαδρομές εγγράφων και οι κατάλογοι εξόδου έχουν ρυθμιστεί σωστά:

```csharp
string documentPath = @"YOUR_DOCUMENT_DIRECTORY\sample.eps";
string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
```

#### Βήμα 2: Ορισμός προτύπου ονομασίας εξόδου
Για να διαχειριστείτε τα ονόματα αρχείων για κάθε σελίδα που έχει μετατραπεί, δημιουργήστε ένα πρότυπο ονομασίας:

```csharp
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");
```

#### Βήμα 3: Δημιουργήστε μια συνάρτηση για τη δημιουργία ροών αρχείων
Αυτή η συνάρτηση δημιουργεί ροές για κάθε αποτέλεσμα μετατροπής σελίδας:

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

#### Βήμα 4: Διαμόρφωση επιλογών μετατροπής
Ρυθμίστε τις απαραίτητες επιλογές για τη μετατροπή αρχείων EPS σε μορφή JPG:

```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Jpg };
```

#### Βήμα 5: Εκτελέστε τη μετατροπή
Χρησιμοποιήστε το αντικείμενο Converter για να εκτελέσετε τη μετατροπή με τις καθορισμένες ρυθμίσεις σας:

```csharp
using (Converter converter = new Converter(documentPath))
{
    converter.Convert(getPageStream, options);
}
```
