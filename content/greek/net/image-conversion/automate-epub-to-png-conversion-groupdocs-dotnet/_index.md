## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα σας καθοδηγήσουμε στη διαδικασία μετατροπής ενός αρχείου EPUB σε εικόνες PNG χρησιμοποιώντας λογικά βήματα και λειτουργίες.

### Χαρακτηριστικό: Μετατροπή EPUB σε PNG από PDF

**Επισκόπηση**

Αυτή η λειτουργία σάς επιτρέπει να εξαγάγετε κάθε σελίδα από ένα αρχείο EPUB ως ξεχωριστή εικόνα PNG. 

#### Βήμα 1: Ορισμός διαδρομών προέλευσης και εξόδου

Ξεκινήστε ρυθμίζοντας τους καταλόγους προέλευσης και εξόδου:

```csharp
string sourceFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.epub");
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", "ConvertedPNGs");

// Βεβαιωθείτε ότι ο κατάλογος εξόδου υπάρχει
Directory.CreateDirectory(outputFolder);
```

#### Βήμα 2: Ρύθμιση παραμέτρων ονομασίας αρχείου εξόδου

Ορίστε ένα πρότυπο για την ονομασία των αρχείων PNG εξόδου σας:

```csharp
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.png");
```

#### Βήμα 3: Ρύθμιση λειτουργίας δημιουργίας ροής

Δημιουργήστε μια συνάρτηση για να χειριστείτε τη δημιουργία ροής κατά τη μετατροπή:

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

#### Βήμα 4: Διαμόρφωση επιλογών μετατροπής

Ορίστε επιλογές για μετατροπή PNG:

```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Png };
```

#### Βήμα 5: Εκτελέστε τη μετατροπή

Εκτελέστε τη διαδικασία μετατροπής για να δημιουργήσετε εικόνες PNG από το αρχείο EPUB σας:

```csharp
using (Converter converter = new Converter(sourceFilePath))
{
    converter.Convert(getPageStream, options);
}
```
