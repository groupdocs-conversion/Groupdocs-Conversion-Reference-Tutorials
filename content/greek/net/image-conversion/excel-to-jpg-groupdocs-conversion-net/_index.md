## Οδηγός Εφαρμογής

Ακολουθήστε αυτά τα βήματα για να μετατρέψετε ένα αρχείο Excel σε εικόνες JPEG χρησιμοποιώντας το GroupDocs.Conversion for .NET.

### Βήμα 1: Προετοιμασία της συνάρτησης ροής εξόδου

Δημιουργήστε μια συνάρτηση που θα χειρίζεται την εγγραφή κάθε σελίδας που έχει μετατραπεί ως ξεχωριστό αρχείο JPG:

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

**Εξήγηση:**
- `getPageStream` είναι ένας εκπρόσωπος που δημιουργεί μια ροή για κάθε σελίδα.
- Η διαδρομή εξόδου περιλαμβάνει `{0}` για να διακρίνετε μεταξύ σελίδων.

### Βήμα 2: Φόρτωση του αρχείου XLS

Φορτώστε το αρχείο Excel χρησιμοποιώντας την κλάση GroupDocs.Converter:

```csharp
using (Converter converter = new Converter(inputFilePath))
{
    // Συνεχίστε με τα βήματα μετατροπής
}
```

**Εξήγηση:**
- Ο `Converter` Το αντικείμενο χειρίζεται τη διαδικασία μετατροπής.
- Βεβαιωθείτε ότι `inputFilePath` είναι σωστό.

### Βήμα 3: Διαμόρφωση επιλογών μετατροπής

Ρύθμιση επιλογών για τη μετατροπή αρχείων σε μορφή JPG:

```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Jpg };
```

**Εξήγηση:**
- `ImageConvertOptions` Καθορίζει ρυθμίσεις για μετατροπές εικόνων.
- Ο `Format` Η ιδιότητα ορίζει την έξοδο σε JPEG.

### Βήμα 4: Εκτελέστε τη μετατροπή

Εκτελέστε τη διαδικασία μετατροπής:

```csharp
converter.Convert(getPageStream, options);
```

**Εξήγηση:**
- Αυτή η κλήση ξεκινά τη μετατροπή από XLS σε JPG χρησιμοποιώντας την καθορισμένη συνάρτηση ροής και τις επιλογές.
- Εξασφαλίζω `outputFileTemplate` έχει ρυθμιστεί σωστά για την αποθήκευση εικόνων.
