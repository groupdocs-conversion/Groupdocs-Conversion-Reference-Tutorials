#### Βασική Αρχικοποίηση και Ρύθμιση

Δείτε πώς μπορείτε να ρυθμίσετε το έργο σας για μετατροπή JPG σε TEX:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

namespace ConversionTutorial
{
    class Program
    {
        static void Main(string[] args)
        {
            string documentDirectory = "YOUR_DOCUMENT_DIRECTORY"; // Αντικαταστήστε με τη διαδρομή του καταλόγου προέλευσης
            string outputDirectory = "YOUR_OUTPUT_DIRECTORY"; // Αντικαταστήστε με τη διαδρομή του καταλόγου εξόδου σας

            using (var converter = new Converter(Path.Combine(documentDirectory, "sample.jpg")))
            {
                var options = new PageDescriptionLanguageConvertOptions { Format = PageDescriptionLanguageFileType.Tex };
                string outputFile = Path.Combine(outputDirectory, "converted-to.tex");
                converter.Convert(outputFile, options);
            }
        }
    }
}
```

Αυτή η λειτουργία σάς επιτρέπει να φορτώσετε ένα αρχείο JPG πηγής και να το μετατρέψετε σε μορφή TEX χρησιμοποιώντας το GroupDocs.Conversion. Είναι ισχυρό για την αυτοματοποίηση μετατροπών εγγράφων στις εφαρμογές σας.

Δημιουργήστε μια παρουσία του `Converter` κλάση, περνώντας τη διαδρομή προς το αρχείο JPG πηγής σας:

```csharp
using (var converter = new Converter(Path.Combine(documentDirectory, "sample.jpg")))
```

##### Βήμα 2: Ορισμός επιλογών μετατροπής
Ορίστε τις επιλογές μετατροπής σας, καθορίζοντας το TEX ως τη μορφή στόχου χρησιμοποιώντας `PageDescriptionLanguageConvertOptions`.

##### Βήμα 3: Εκτέλεση μετατροπής
Καλέστε το `Convert` τη μέθοδο με τη διαδρομή του αρχείου εξόδου και τις επιλογές μετατροπής. Αυτό το βήμα μετατρέπει το JPG σας σε έγγραφο TEX:
