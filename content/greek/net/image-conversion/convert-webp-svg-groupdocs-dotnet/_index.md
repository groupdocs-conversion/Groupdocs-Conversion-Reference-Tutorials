### Βασική Αρχικοποίηση και Ρύθμιση
Δείτε πώς μπορείτε να αρχικοποιήσετε το GroupDocs.Conversion στο έργο σας C#:
```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main()
    {
        // Αρχικοποίηση του μετατροπέα
        using (var converter = new Converter("input.webp"))
        {
            var options = new ImageConvertOptions { Format = FileType.Svg };
            converter.Convert("output.svg", options);
        }
    }
}
```
Αυτό το απόσπασμα κώδικα δείχνει τη ρύθμιση της διαδικασίας μετατροπής. `Converter` η κλάση αρχικοποιείται με ένα αρχείο WEBP και καθορίζουμε το SVG ως τη μορφή στόχου χρησιμοποιώντας `ImageConvertOptions`.
