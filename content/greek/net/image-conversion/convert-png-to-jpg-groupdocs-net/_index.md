## Οδηγός Εφαρμογής

### Μετατροπή PNG σε JPG Λειτουργία
Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε ένα αρχείο PNG σε μορφή JPG χρησιμοποιώντας το GroupDocs.Conversion. Δείτε πώς:

#### Βήμα 1: Ορισμός καταλόγου εξόδου και προτύπου ονομασίας αρχείου
Καθορίστε πού θα πρέπει να αποθηκευτούν τα αρχεία που έχετε μετατρέψει και τη σύμβαση ονοματοδοσίας τους.
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY"; 
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");
```
**Γιατί;** Αυτή η ρύθμιση διασφαλίζει ότι κάθε εικόνα που έχει μετατραπεί αποθηκεύεται σε έναν καθορισμένο κατάλογο με σαφή σύμβαση ονομασίας.

#### Βήμα 2: Δημιουργήστε μια συνάρτηση ροής για κάθε σελίδα
Ορίστε μια συνάρτηση για τον χειρισμό της δημιουργίας ροής αρχείων για κάθε σελίδα που αποθηκεύεται.
```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```
**Γιατί;** Αυτή η συνάρτηση δημιουργεί δυναμικά μια ροή αρχείων για κάθε σελίδα, επιτρέποντας την αποτελεσματική διαχείριση πολλαπλών σελίδων, εάν είναι απαραίτητο.

#### Βήμα 3: Φόρτωση του αρχείου PNG προέλευσης
Φορτώστε το αρχείο PNG πηγής χρησιμοποιώντας το αντικείμενο Converter. Αντικαταστήστε `"YOUR_DOCUMENT_DIRECTORY/SAMPLE_PNG.png"` με την πραγματική διαδρομή του αρχείου PNG σας.
```csharp
using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_PNG.png"))
{
    // Οι επιλογές μετατροπής θα οριστούν εδώ
}
```
**Γιατί;** Η φόρτωση του αρχείου προέλευσης είναι απαραίτητη για την έναρξη της διαδικασίας μετατροπής.

#### Βήμα 4: Ορισμός επιλογών μετατροπής
Ρυθμίστε τις παραμέτρους μετατροπής για να ορίσετε JPG ως μορφή εξόδου.
```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Jpg };
```
**Γιατί;** Αυτό διασφαλίζει ότι το αρχείο εξόδου έχει την επιθυμητή μορφή JPG.

#### Βήμα 5: Εκτελέστε τη μετατροπή
Εκτελέστε τη μετατροπή χρησιμοποιώντας το `Convert` μέθοδος.
```csharp
converter.Convert(getPageStream, options);
```
**Γιατί;** Αυτό το βήμα ενεργοποιεί την πραγματική διαδικασία μετατροπής, χρησιμοποιώντας όλες τις προηγουμένως ορισμένες διαμορφώσεις και λειτουργίες.
