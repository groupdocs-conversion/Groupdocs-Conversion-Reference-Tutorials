### Μετατροπή ODT σε PSD

**Επισκόπηση**Αυτό το τελευταίο βήμα δείχνει πώς να μετατρέψετε ένα αρχείο ODT σε μορφή PSD, ενώ αποθηκεύετε κάθε σελίδα ως ξεχωριστό αρχείο.

#### Βήμα 3: Εκτέλεση μετατροπής
Χρησιμοποιήστε το `Converter` αντικείμενο και καθορισμένες επιλογές για την εκτέλεση της μετατροπής, αποθηκεύοντας κάθε σελίδα σε έναν καθορισμένο κατάλογο εξόδου.
```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

namespace ConvertOdtToPsdExample {
    internal class Program {
        public static void Main() {
            string outputFolder = "YOUR_OUTPUT_DIRECTORY";
            string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");
            Func<SavePageContext, Stream> getPageStream = savePageContext => new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);

            string documentPath = "YOUR_DOCUMENT_DIRECTORY/sample.odt";
            using (Converter converter = new Converter(documentPath)) {
                ImageConvertOptions options = new ImageConvertOptions { Format = FileTypes.ImageFileType.Psd };
                converter.Convert(getPageStream, options);
            }
        }
    }
}
```

**Εξήγηση**: Το `getPageStream` Η συνάρτηση καθορίζει τον τρόπο με τον οποίο κάθε σελίδα που έχει μετατραπεί αποθηκεύεται ως αρχείο PSD. Χρησιμοποιώντας την `Converter` Ένα αντικείμενο με καθορισμένες επιλογές διασφαλίζει μια αποτελεσματική διαδικασία μετατροπής.
