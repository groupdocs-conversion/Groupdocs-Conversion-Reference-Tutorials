## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Μετατροπή ICO σε JPG

Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε ένα αρχείο ICO σε μορφή JPEG. Ας εξερευνήσουμε τα βήματα που εμπλέκονται.

#### Βήμα 1: Ορισμός διαδρομής εξόδου και προτύπου

Αρχικά, καθορίστε πού θα αποθηκευτούν τα αρχεία που έχετε μετατρέψει:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");
```

Αυτό το πρότυπο βοηθά στην συστηματική ονομασία των αρχείων εξόδου για κάθε σελίδα μετατροπής.

#### Βήμα 2: Δημιουργήστε μια συνάρτηση ροής

Πρέπει να ορίσουμε πώς αποθηκεύεται κάθε σελίδα που έχει μετατραπεί:

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext =>
{
    return new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
};
```

Αυτή η λειτουργία διασφαλίζει ότι κάθε αποτέλεσμα μετατροπής αποθηκεύεται ως ξεχωριστό αρχείο JPEG.

#### Βήμα 3: Ρύθμιση επιλογών μετατροπής

Διαμορφώστε τις ρυθμίσεις για την έξοδο JPG:

```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Jpg };
```

Αυτές οι επιλογές καθορίζουν τη μορφή και την ποιότητα των εικόνων εξόδου σας.

#### Βήμα 4: Εκτελέστε τη μετατροπή

Εκτελέστε τη διαδικασία μετατροπής με τις καθορισμένες διαμορφώσεις:

```csharp
using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY\sample.ico"))
{
    converter.Convert(getPageStream, options);
}
```

Αυτό το απόσπασμα κώδικα μετατρέπει το αρχείο ICO σας σε μορφή JPG χρησιμοποιώντας το GroupDocs.Conversion.
