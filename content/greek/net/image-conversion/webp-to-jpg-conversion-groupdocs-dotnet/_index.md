### Ορισμός επιλογών μετατροπής για μορφή JPG
Στη συνέχεια, διαμορφώστε τις ρυθμίσεις μετατροπής για να μετατρέψετε αρχεία WebP σε μορφή JPG.

#### Επισκόπηση
Αυτό το βήμα περιλαμβάνει τη ρύθμιση `ImageConvertOptions`, καθορίζοντας τη μορφή εικόνας-στόχου και άλλες ιδιότητες.

#### Βήματα Υλοποίησης
1. **Δημιουργία αντικειμένου ImageConvertOptions**

```csharp
using GroupDocs.Conversion.Options.Convert;

// Ορίστε επιλογές μετατροπής για τη μορφή JPG
ImageConvertOptions options = new ImageConvertOptions
{
    Format = GroupDocs.Conversion.FileTypes.ImageFileType.Jpg
};
```

### Μετατροπή και αποθήκευση εξόδου ως JPG
Τέλος, εκτελέστε τη διαδικασία μετατροπής και αποθηκεύστε τα αρχεία εξόδου.

#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς να εκτελέσετε την πραγματική μετατροπή αρχείου χρησιμοποιώντας τις επιλογές που ορίστηκαν προηγουμένως και πώς να χειριστείτε τη διαχείριση του καταλόγου εξόδου.

#### Βήματα Υλοποίησης
1. **Ορισμός καταλόγου και προτύπου εξόδου**

```csharp
string outputFolder = @"path\to\your\output";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");

Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```
2. **Μετατροπή WebP σε JPG**

```csharp
using (Converter converter = new Converter(inputFilePath))
{
    // Εκτελέστε μετατροπή με τις καθορισμένες επιλογές και τη συνάρτηση ροής εξόδου
    converter.Convert(getPageStream, options);
}
```
