## Οδηγός Εφαρμογής

Ας αναλύσουμε την υλοποίηση σε λογικά βήματα για λόγους σαφήνειας.

### Φόρτωση του αρχείου GIF σας
Αρχικά, καθορίστε τις διαδρομές για το αρχείο GIF εισόδου και τον κατάλογο εξόδου. Αντικαταστήστε τα placeholders με τις πραγματικές διαδρομές:

```csharp
string inputGifPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.gif");
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", "Output");
Directory.CreateDirectory(outputFolder);
string outputFile = Path.Combine(outputFolder, "gif-converted-to.tex");
```

### Ρύθμιση παραμέτρων επιλογών μετατροπής
Στη συνέχεια, ρυθμίστε τις επιλογές μετατροπής για να καθορίσετε τη μορφή-στόχο σας:

```csharp
using (var converter = new Converter(inputGifPath))
{
    var options = new PageDescriptionLanguageConvertOptions 
    {
        Format = FileType.Tex // Ορίστε τη μορφή εξόδου ως TEX
    };
```

### Εκτέλεση της μετατροπής
Τέλος, εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο TEX:

```csharp
converter.Convert(outputFile, options);
}
```

**Εξήγηση**: Το `Converter` Η κλάση χειρίζεται τη φόρτωση του αρχείου GIF σας. Καθορίζοντας `FileType.Tex`, δίνετε εντολή στο GroupDocs.Conversion να εξάγει σε μορφή TEX.
