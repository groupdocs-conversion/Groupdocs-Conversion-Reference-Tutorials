#### Βήμα προς βήμα εφαρμογή

### Ορισμός επιλογών μετατροπής σε μορφή JPG
#### Επισκόπηση
Στη συνέχεια, ορίσαμε επιλογές μετατροπής προσαρμοσμένες για τη μετατροπή εγγράφων σε εικόνες JPEG χρησιμοποιώντας `ImageConvertOptions`.

#### Βήμα προς βήμα εφαρμογή
1. **Ορισμός επιλογών μετατροπής εικόνας**: Καθορίστε τη μορφή εξόδου ως JPEG.
   
   ```csharp
   using GroupDocs.Conversion.Options.Convert;

   // Ορίστε τις επιλογές μετατροπής σε JPG
   ImageConvertOptions options = new ImageConvertOptions { Format = FileTypes.ImageFileType.Jpg };
   ```

2. **Εξηγήστε τη διαμόρφωση κλειδιού**: Το `Format` Η ιδιότητα ορίζει τον τύπο αρχείου προορισμού για μετατροπή, καθιστώντας την κρίσιμη για τον ορισμό των προδιαγραφών εξόδου.

### Μετατροπή MPT σε JPG και αποθήκευση ροής εξόδου
#### Επισκόπηση
Τέλος, εκτελέστε την πραγματική διαδικασία μετατροπής μετατρέποντας το φορτωμένο έγγραφο MPT σε εικόνες JPEG και αποθηκεύοντάς τες στον καθορισμένο κατάλογο.

#### Βήμα προς βήμα εφαρμογή
1. **Ορισμός διαδρομής και συνάρτησης εξόδου**: Χρησιμοποιήστε μια συνάρτηση για να δημιουργήσετε δυναμικά διαδρομές αρχείων εξόδου για κάθε σελίδα που έχει μετατραπεί.
   
   ```csharp
   string outputFolder = "YOUR_OUTPUT_DIRECTORY";
   string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");

   Func<SavePageContext, Stream> getPageStream = savePageContext =>
       new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
   ```

2. **Μετατροπή και Αποθήκευση**: Υλοποιήστε τη μετατροπή χρησιμοποιώντας το `Converter` αντικείμενο.
   
   ```csharp
   using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/sample.mpt"))
   {
       // Εκτελέστε μετατροπή σε μορφή JPG με καθορισμένες επιλογές και συνάρτηση ροής εξόδου
       converter.Convert(getPageStream, new ImageConvertOptions { Format = FileTypes.ImageFileType.Jpg });
   }
   ```

3. **Συμβουλές αντιμετώπισης προβλημάτων**Βεβαιωθείτε ότι οι διαδρομές αρχείων είναι σωστές και ελέγξτε για προβλήματα δικαιωμάτων κατά την εγγραφή αρχείων.
