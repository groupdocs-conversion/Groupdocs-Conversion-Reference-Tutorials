## Οδηγός Εφαρμογής

### Μετατροπή BMP σε TXT
Αυτή η ενότητα περιγράφει λεπτομερώς τη διαδικασία μετατροπής.

#### Βήμα 1: Φόρτωση του αρχείου BMP
Προετοιμάστε το αρχείο BMP πηγής σας για μετασχηματισμό:
```csharp
using (var converter = new Converter(sourceBmpPath))
{
    // Το BMP είναι τώρα έτοιμο για μετατροπή.
}
```

#### Βήμα 2: Ορισμός επιλογών μετατροπής
Καθορίστε τις ρυθμίσεις μετατροπής χρησιμοποιώντας `WordProcessingConvertOptions`:
```csharp
var options = new WordProcessingConvertOptions { Format = FileType.Txt };
```

#### Βήμα 3: Εκτελέστε τη μετατροπή
Εκτελέστε και αποθηκεύστε το αποτέλεσμα της μετατροπής:
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "bmp-converted-to.txt");
converter.Convert(outputFile, options);
```
**Συμβουλές αντιμετώπισης προβλημάτων:** Βεβαιωθείτε ότι οι διαδρομές είναι σωστές και προσβάσιμες. Χειριστείτε εξαιρέσεις κατά τη διάρκεια των λειτουργιών αρχείων για να αποτρέψετε σφάλματα χρόνου εκτέλεσης.

### Ορισμός διαδρομής καταλόγου εξόδου
Οργανώστε αποτελεσματικά τα αρχεία εξόδου:
#### Βήμα 1: Ορίστε τον βασικό κατάλογο
Ορίστε πού θα αποθηκευτούν τα αρχεία που έχουν μετατραπεί:
```csharp
string baseOutputDirectory = "YOUR_OUTPUT_DIRECTORY";
```

#### Βήμα 2: Δημιουργία και επαλήθευση διαδρομής εξόδου
Βεβαιωθείτε ότι ο κατάλογος υπάρχει ή δημιουργήστε τον, εάν είναι απαραίτητο:
```csharp
string GetOutputDirectoryPath()
{
    string outputPath = Path.Combine(baseOutputDirectory, "ConvertedFiles");
    
    // Έλεγχος ύπαρξης· δημιουργία εάν δεν υπάρχει.
    if (!Directory.Exists(outputPath))
        Directory.CreateDirectory(outputPath);
    
    return outputPath;
}
```
