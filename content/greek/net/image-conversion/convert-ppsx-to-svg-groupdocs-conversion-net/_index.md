### Μετατροπή αρχείου PPSX σε μορφή SVG

#### Επισκόπηση
Η μετατροπή ενός αρχείου .ppsx σε μορφή SVG περιλαμβάνει τη φόρτωση του αρχείου προέλευσης, τη διαμόρφωση των ρυθμίσεων μετατροπής και την αποθήκευση του αποτελέσματος. Αυτή η ενότητα σας καθοδηγεί σε κάθε βήμα με λεπτομερείς εξηγήσεις και αποσπάσματα κώδικα.

#### Βήμα 1: Ορισμός διαδρομών για καταλόγους εισόδου/εξόδου
Ξεκινήστε καθορίζοντας πού βρίσκονται τα αρχεία εισόδου σας και πού θέλετε να αποθηκευτούν τα αρχεία που έχουν μετατραπεί:

```csharp
string documentPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.ppsx");
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "ppsx-converted-to.svg");
```

#### Βήμα 2: Φόρτωση του αρχείου PPSX πηγής
Φορτώστε το αρχείο .ppsx χρησιμοποιώντας το GroupDocs.Conversion `Converter` τάξη:

```csharp
using (var converter = new Converter(documentPath))
{
    // Η λογική μετατροπής θα εφαρμοστεί εδώ.
}
```
Αυτό το βήμα διασφαλίζει ότι το αρχείο σας είναι έτοιμο για επεξεργασία.

#### Βήμα 3: Διαμόρφωση επιλογών μετατροπής
Ρυθμίστε τις επιλογές μετατροπής για να ορίσετε το SVG ως μορφή εξόδου:

```csharp
var options = new PageDescriptionLanguageConvertOptions { Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Svg };
```

Αυτές οι επιλογές υπαγορεύουν τον τρόπο με τον οποίο θα πρέπει να γίνει η διαδικασία μετατροπής.

#### Βήμα 4: Εκτελέστε τη μετατροπή και αποθηκεύστε
Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο SVG που προκύπτει:

Αυτή η εντολή μετατρέπει την παρουσίασή σας σε αρχείο SVG και την αποθηκεύει στην καθορισμένη θέση.
