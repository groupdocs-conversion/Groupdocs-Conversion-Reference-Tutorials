## Οδηγός Εφαρμογής

### Βήμα 1: Φόρτωση του αρχείου προέλευσης
Ξεκινήστε φορτώνοντας το αρχείο .jpc πηγαίου κώδικα που θέλετε να μετατρέψετε:

```csharp

// Ορίστε την πλήρη διαδρομή για το αρχείο SVG που έχει μετατραπεί.
string outputFile = Path.Combine(outputFolder, "jpc-converted-to.svg");

using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY/sample.jpc"))
{
    // Η λογική μετατροπής θα ακολουθηθεί εδώ.
}
```

### Βήμα 2: Ορισμός επιλογών μετατροπής
Στη συνέχεια, καθορίστε ότι θέλετε να μετατρέψετε το φορτωμένο αρχείο JPC σε μορφή SVG:

```csharp
// Ρυθμίστε τις επιλογές μετατροπής για τη μορφή SVG.
var options = new PageDescriptionLanguageConvertOptions
{
    Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Svg
};
```

### Βήμα 3: Εκτελέστε τη μετατροπή
Τέλος, εκτελέστε τη μετατροπή και αποθηκεύστε την έξοδο:

```csharp

converter.Convert(outputFile, options);
```

**Παράμετροι & Επιστρεφόμενες Τιμές**
- `outputFile`: Η διαδρομή προορισμού για το αρχείο που μετατράπηκε.
- `options`: Ορίζει τον τρόπο μετατροπής του αρχείου.
