### Λειτουργία 2: Ορισμός επιλογών μετατροπής σε μορφή JPG

#### Επισκόπηση

Ρυθμίστε τις απαραίτητες επιλογές για τη μετατροπή σε μορφή JPG:


**Εξήγηση**: Το `ImageConvertOptions` Η κλάση σάς επιτρέπει να καθορίσετε τη μορφή εικόνας-στόχου, σε αυτήν την περίπτωση, JPG.

### Λειτουργία 3: Εκτέλεση μετατροπής από DJVU σε JPG

#### Επισκόπηση
Εκτελέστε τη διαδικασία μετατροπής και αποθηκεύστε τις εικόνες εξόδου.

##### Βήμα 1: Ορισμός διαδρομής εξόδου και λογικής
Ρυθμίστε τη λογική για την αποθήκευση κάθε σελίδας που έχει μετατραπεί ως ξεχωριστό αρχείο JPEG:

```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");

Func<SavePageContext, Stream> getPageStream = savePageContext =>
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

##### Βήμα 2: Εκτέλεση μετατροπής
Χρησιμοποιήστε το `Converter` παράδειγμα για μετατροπή και αποθήκευση της εξόδου:

```csharp
using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/sample.djvu"))
{
    // Μετατρέψτε και αποθηκεύστε κάθε σελίδα ως αρχείο JPG
    converter.Convert(getPageStream, options);
}
```
**Εξήγηση**Αυτό το τμήμα κώδικα μετατρέπει κάθε σελίδα του εγγράφου DJVU σε ξεχωριστή εικόνα JPEG χρησιμοποιώντας την καθορισμένη λογική ροής.
