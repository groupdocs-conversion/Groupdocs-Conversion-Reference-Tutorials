## Οδηγός Εφαρμογής

### Φόρτωση και μετατροπή αρχείου TEX σε PNG
Αυτή η λειτουργία δείχνει πώς να φορτώσετε ένα αρχείο TEX και να το μετατρέψετε σε μορφή PNG χρησιμοποιώντας το GroupDocs.Conversion για .NET.

#### Ρύθμιση καταλόγου εξόδου και διαδρομής προτύπου αρχείου
Αρχικά, ορίστε πού θα αποθηκευτούν τα αρχεία που έχετε μετατρέψει:
```csharp
string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.png");

// Συνάρτηση για τη λήψη της ροής για κάθε σελίδα
Func<SavePageContext, Stream> getPageStream = savePageContext => 
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```
#### Φόρτωση του αρχείου TEX πηγής
Φορτώστε το αρχείο TEX χρησιμοποιώντας το `Converter` τάξη:
```csharp
using (Converter converter = new Converter(inputFile))
{
    // Οι επιλογές μετατροπής θα οριστούν στο επόμενο βήμα.
}
```
#### Ορισμός επιλογών μετατροπής για μορφή PNG
Διαμορφώστε τις ρυθμίσεις μετατροπής για να εξάγετε σε μορφή PNG:
```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Png };
```
#### Εκτελέστε τη μετατροπή
Εκτελέστε τη διαδικασία μετατροπής:
```csharp
converter.Convert(getPageStream, options);
```
