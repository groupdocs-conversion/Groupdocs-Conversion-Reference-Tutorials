## Οδηγός Εφαρμογής

### Φόρτωση αρχείου EPUB

Ξεκινήστε φορτώνοντας το αρχείο EPUB χρησιμοποιώντας το `Converter` τάξη:

```csharp
using System;
using GroupDocs.Conversion;

string documentPath = @"YOUR_DOCUMENT_DIRECTORY/sample.epub"; // Αντικαταστήστε με τη διαδρομή του αρχείου EPUB
using (Converter converter = new Converter(documentPath))
{
    // Το αρχείο EPUB πηγής έχει πλέον φορτωθεί στο αντικείμενο 'converter'.
}
```

### Ορισμός επιλογών μετατροπής για τη μορφή JPG

Ρύθμιση παραμέτρων `ImageConvertOptions` για να καθορίσετε την έξοδο ως JPEG:

```csharp
using System;
using GroupDocs.Conversion.Options.Convert;

ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Jpg };
// Το αντικείμενο 'options' καθορίζει ότι η μετατροπή θα πρέπει να έχει ως αποτέλεσμα εικόνες JPG.
```

### Εκτέλεση μετατροπής από EPUB σε JPG

Εκτελέστε τη μετατροπή περνώντας τις διαμορφωμένες επιλογές σας:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

string outputFolder = @"YOUR_OUTPUT_DIRECTORY"; // Αντικαταστήστε με την επιθυμητή διαδρομή καταλόγου εξόδου
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");

Func<SavePageContext, Stream> getPageStream = savePageContext => new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);

using (Converter converter = new Converter(documentPath))
{
    // Μετατροπή σε μορφή JPG χρησιμοποιώντας τις καθορισμένες 'επιλογές' και 'getPageStream'
    converter.Convert(getPageStream, options);
}
```
