## Οδηγός Εφαρμογής

Τώρα, ας δούμε βήμα προς βήμα τη μετατροπή των αρχείων ODG σε TXT.

### Φόρτωση και μετατροπή αρχείου ODG

#### Επισκόπηση
Αυτή η ενότητα εστιάζει στη φόρτωση ενός αρχείου ODG και στη μετατροπή του σε μορφή απλού κειμένου χρησιμοποιώντας το GroupDocs.Conversion για .NET.

##### Βήμα 1: Ορισμός διαδρομών
Ξεκινήστε καθορίζοντας τις διαδρομές για το αρχείο ODG πηγής και το αρχείο TXT εξόδου:

```csharp
string documentPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.odg");
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "odg-converted-to.txt");
```

##### Βήμα 2: Αρχικοποίηση μετατροπέα
Δημιουργήστε ένα νέο `Converter` αντικείμενο για τη διαχείριση της διαδικασίας μετατροπής:

```csharp
using (var converter = new GroupDocs.Conversion.Converter(documentPath))
{
    // Η λογική μετατροπής θα προστεθεί εδώ.
}
```

##### Βήμα 3: Καθορισμός επιλογών μετατροπής
Ορίστε τις επιλογές μορφής TXT για μετατροπή. Αυτό περιλαμβάνει τον καθορισμό ότι θέλετε να μετατρέψετε σε απλό κείμενο:

```csharp
WordProcessingConvertOptions options = new WordProcessingConvertOptions { Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt };
```

##### Βήμα 4: Εκτέλεση μετατροπής
Τέλος, εκτελέστε την πραγματική μετατροπή αρχείου και αποθηκεύστε το ως αρχείο TXT:

```csharp
converter.Convert(outputFile, options);
Console.WriteLine("Conversion complete!");
```
