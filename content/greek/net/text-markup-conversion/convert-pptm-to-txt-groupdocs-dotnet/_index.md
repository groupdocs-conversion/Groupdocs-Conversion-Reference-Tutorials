Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε αρχεία PowerPoint με μακροεντολές (.pptm) σε αρχεία απλού κειμένου. Αυτή η μετατροπή βοηθά στην εξαγωγή και επεξεργασία περιεχομένου σε εφαρμογές που βασίζονται σε κείμενο.

#### Βήμα προς βήμα εφαρμογή

**Βήμα 1: Ορισμός διαδρομής εξόδου**

Καθορίστε τον κατάλογο εξόδου και τη διαδρομή αρχείου:

```csharp

string outputFile = Path.Combine(outputFolder, "pptm-converted-to.txt");
```

**Γιατί;** Αυτό το βήμα διασφαλίζει ότι τα αρχεία κειμένου που έχετε μετατρέψει αποθηκεύονται σε μια καθορισμένη τοποθεσία.

**Βήμα 2: Φόρτωση του αρχείου PPTM**

Φορτώστε το αρχείο PowerPoint χρησιμοποιώντας το αντικείμενο GroupDocs.Converter:

```csharp
using (var converter = new Converter(Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.pptm")))
{
    // Ο κώδικας μετατροπής θα προστεθεί εδώ.
}
```

**Γιατί;** Αυτό το βήμα ξεκινά τη διαδικασία μετατροπής φορτώνοντας το έγγραφο προέλευσης.

**Βήμα 3: Ρύθμιση παραμέτρων επιλογών μορφής TXT**

Ρυθμίστε τις επιλογές μετατροπής για τη μορφή κειμένου:

```csharp
WordProcessingConvertOptions options = new WordProcessingConvertOptions
{
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt
};
```

**Γιατί;** Η ρύθμιση αυτών των επιλογών καθορίζει ότι η έξοδος θα πρέπει να είναι σε μορφή TXT.

**Βήμα 4: Εκτέλεση μετατροπής**

Εκτελέστε τη μετατροπή και αποθηκεύστε το αποτέλεσμα:

```csharp
converter.Convert(outputFile, options);
```

**Γιατί;** Αυτό το τελικό βήμα εκτελεί τη διαδικασία μετατροπής και αποθηκεύει το αρχείο σας ως έγγραφο κειμένου.
