## Οδηγός Εφαρμογής
### Χαρακτηριστικό: Φόρτωση αρχείου OTG πηγής
Αυτή η λειτουργία δείχνει πώς να φορτώσετε ένα αρχείο OpenDocument Graphic Template (OTG) χρησιμοποιώντας τη βιβλιοθήκη GroupDocs.Conversion.

#### Βήμα 1: Ορισμός διαδρομής εισόδου
Καθορίστε τη διαδρομή προς το αρχείο OTG πηγής, αντικαθιστώντας `"YOUR_DOCUMENT_DIRECTORY\sample.otg"` με την πραγματική διαδρομή αρχείου:
```csharp
string sourceFilePath = "YOUR_DOCUMENT_DIRECTORY\\sample.otg";
```

#### Βήμα 2: Φόρτωση του αρχείου OTG
Χρησιμοποιήστε το `Converter` κλάση για να φορτώσετε το αρχείο OTG για μετατροπή ή άλλες λειτουργίες:
```csharp
using (var converter = new Converter(sourceFilePath))
{
    // Το φορτωμένο έγγραφο είναι πλέον έτοιμο για περαιτέρω επεξεργασία.
}
```

### Χαρακτηριστικό: Μετατροπή OTG σε TXT
Αυτή η ενότητα σας καθοδηγεί στη μετατροπή ενός αρχείου OTG σε μορφή απλού κειμένου (.txt).

#### Βήμα 1: Ορισμός διαδρομής εξόδου
Καθορίστε τη διαδρομή όπου θα αποθηκευτεί το αρχείο TXT που έχει μετατραπεί:
```csharp
string outputFile = Path.Combine("YOUR_OUTPUT_DIRECTORY", "otg-converted-to.txt");
```

#### Βήμα 2: Διαμόρφωση επιλογών μετατροπής
Ρύθμιση επιλογών μετατροπής για τη μετατροπή αρχείων OTG σε μορφή TXT χρησιμοποιώντας `WordProcessingConvertOptions`:
```csharp
WordProcessingConvertOptions options = new WordProcessingConvertOptions
{
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt
};
```

#### Βήμα 3: Εκτελέστε τη μετατροπή
Εκτελέστε τη μετατροπή και αποθηκεύστε το αποτέλεσμα ως αρχείο TXT:
```csharp
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY\\sample.otg"))
{
    converter.Convert(outputFile, options);
}
```
