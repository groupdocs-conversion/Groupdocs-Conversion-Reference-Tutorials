## Οδηγός Εφαρμογής

### Μετατροπή CMX σε TEX

Ας αναλύσουμε τη διαδικασία μετατροπής σε λεπτομερή βήματα:

#### Βήμα 1: Προετοιμάστε το περιβάλλον σας
Βεβαιωθείτε ότι οι κατάλογοι εισόδου και εξόδου έχουν ρυθμιστεί σωστά.

```csharp
string outputFile = Path.Combine(outputFolder, "cmx-converted-to.tex");
```

#### Βήμα 2: Διαμόρφωση επιλογών μετατροπής
Ρυθμίστε τη διαμόρφωση για μετατροπή σε μορφή TEX χρησιμοποιώντας `PageDescriptionLanguageConvertOptions`.

```csharp
// Ρύθμιση παραμέτρων επιλογών μετατροπής
PageDescriptionLanguageConvertOptions options = new PageDescriptionLanguageConvertOptions
{
    Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Tex
};
```

#### Βήμα 3: Εκτελέστε τη μετατροπή
Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο εξόδου.

```csharp
// Εκτελέστε τη μετατροπή
converter.Convert(outputFile, options);
Console.WriteLine("Conversion completed successfully.");
```
