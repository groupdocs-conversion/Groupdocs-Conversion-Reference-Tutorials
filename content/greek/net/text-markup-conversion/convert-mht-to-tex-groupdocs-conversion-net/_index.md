### Βασική Αρχικοποίηση και Ρύθμιση
Αρχικοποιήστε το GroupDocs.Conversion API στο έργο σας C# ως εξής:
```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main()
    {
        string outputFolder = "YOUR_OUTPUT_DIRECTORY";
        string inputFile = "YOUR_DOCUMENT_DIRECTORY\sample.mht";

        // Αρχικοποιήστε το αντικείμενο Converter με ένα αρχείο MHT.
        using (var converter = new Converter(inputFile))
        {
            Console.WriteLine("Converter initialized successfully.");
        }
    }
}
```
Αντικαθιστώ `YOUR_OUTPUT_DIRECTORY` και `YOUR_DOCUMENT_DIRECTORY\sample.mht` με τις πραγματικές σου διαδρομές.

## Οδηγός Εφαρμογής
Δείτε πώς μπορείτε να μετατρέψετε αρχεία MHT σε μορφή TEX:

### Βήμα 1: Φόρτωση του αρχείου MHT προέλευσης
Φορτώστε το αρχείο MHT πηγής για μετατροπή:
```csharp
using (var converter = new Converter(@"YOUR_DOCUMENT_DIRECTORY\sample.mht"))
{
    Console.WriteLine("Source MHT file loaded.");
}
```

### Βήμα 2: Διαμόρφωση επιλογών μετατροπής
Διαμόρφωση επιλογών προσαρμοσμένων στη μορφή TEX:
```csharp
PageDescriptionLanguageConvertOptions options = new PageDescriptionLanguageConvertOptions
{
    Format = PageDescriptionLanguageFileType.Tex // Καθορίστε το TEX ως τη μορφή στόχου.
};
```

### Βήμα 3: Εκτελέστε τη μετατροπή
Εκτελέστε τη μετατροπή και αποθηκεύστε το αποτέλεσμα ως αρχείο .tex:
```csharp
string outputFile = Path.Combine(outputFolder, "mht-converted-to.tex");
converter.Convert(outputFile, options);
Console.WriteLine("Conversion completed successfully.");
```
Αυτό το βήμα μετατρέπει το έγγραφό σας στην επιθυμητή μορφή.
