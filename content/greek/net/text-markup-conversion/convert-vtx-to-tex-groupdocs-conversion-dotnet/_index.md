## Οδηγός Εφαρμογής

### Μετατροπή VTX σε TEX

Ακολουθήστε αυτά τα βήματα για να μετατρέψετε πρότυπα Visio σε έγγραφα LaTeX:

#### Βήμα 1: Ορισμός καταλόγου εξόδου και διαδρομής αρχείου
Καθορίστε πού θα αποθηκευτεί το αρχείο που μετατρέψατε:
```csharp
string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "vtx-converted-to.tex");
```

#### Βήμα 2: Φόρτωση του αρχείου VTX πηγής
Χρησιμοποιήστε το GroupDocs.Conversion για να φορτώσετε το αρχείο προέλευσης:
```csharp
using (var converter = new Converter(@"YOUR_DOCUMENT_DIRECTORY/sample.vtx"))
{
    // Η λογική μετατροπής θα εφαρμοστεί εδώ.
}
```
**Εξήγηση**: Το `Converter` Η κλάση προετοιμάζει το αρχείο VTX για μετατροπή.

#### Βήμα 3: Διαμόρφωση επιλογών μετατροπής
Ορίστε επιλογές για τη μετατροπή του εγγράφου σε μορφή TEX:
```csharp
var options = new PageDescriptionLanguageConvertOptions
{
    Format = FileType.Tex // Ο τύπος αρχείου προορισμού έχει οριστεί σε .tex
};
```
**Εξήγηση**: `PageDescriptionLanguageConvertOptions` σας επιτρέπει να καθορίσετε τη μορφή εξόδου και άλλες ρυθμίσεις.

#### Βήμα 4: Εκτελέστε τη μετατροπή
Εκτελέστε τη μετατροπή και αποθηκεύστε την έξοδο:
```csharp
converter.Convert(outputFile, options);
```
**Εξήγηση**: Το `Convert` Η μέθοδος επεξεργάζεται το αρχείο σύμφωνα με τις καθορισμένες επιλογές και το αποθηκεύει στην επιθυμητή θέση.
