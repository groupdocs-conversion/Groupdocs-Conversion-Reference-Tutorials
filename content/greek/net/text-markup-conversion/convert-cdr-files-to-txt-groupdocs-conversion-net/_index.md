### Φόρτωση και μετατροπή CDR σε TXT
Η μετατροπή αρχείων CorelDRAW (CDR) σε απλό κείμενο (TXT) επιτρέπει ευκολότερο χειρισμό ή εξαγωγή δεδομένων. Ακολουθήστε τα παρακάτω βήματα:

#### Βήμα 1: Ορίστε τον κατάλογο εξόδου
Προσδιορίστε πού θα αποθηκευτεί το αρχείο που μετατρέψατε:

```csharp
// Ορίστε τη διαδρομή του καταλόγου εξόδου
string outputFolder = "YOUR_OUTPUT_DIRECTORY/";
string outputFile = Path.Combine(outputFolder, "cdr-converted-to.txt");
```
*Γιατί;* Ο καθορισμός ενός καταλόγου εξόδου διασφαλίζει ότι το αρχείο σας αποθηκεύεται σωστά και ότι η πρόσβαση σε αυτό είναι εύκολη αργότερα.

#### Βήμα 2: Φόρτωση του αρχείου CDR
Φορτώστε το αρχείο CDR προέλευσης στο GroupDocs.Conversion. Αντικαταστήστε το `'YOUR_DOCUMENT_DIRECTORY\\sample.cdr'` με τη διαδρομή προς το πραγματικό αρχείο CDR σας:

```csharp
// Φόρτωση του αρχείου CDR πηγής
class Program
{
    static void Main(string[] args)
    {
        using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY\sample.cdr"))
        {
            // Η μετατροπή θα πραγματοποιηθεί σε αυτό το μπλοκ
        }
    }
}
```
*Γιατί;* Η σωστή φόρτωση των αρχείων είναι ζωτικής σημασίας για την επιτυχή μετατροπή και η χρήση ενός `using` Η δήλωση διασφαλίζει ότι οι πόροι διατίθενται σωστά.

#### Βήμα 3: Διαμόρφωση επιλογών μετατροπής
Ρυθμίστε τις απαραίτητες επιλογές για να μετατρέψετε το αρχείο CDR σε μορφή TXT:

```csharp
// Ρύθμιση παραμέτρων επιλογών μετατροπής για μορφή TXT
class Program
{
    static void Main(string[] args)
    {
        WordProcessingConvertOptions options = new WordProcessingConvertOptions 
        {
            Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt
        };
    }
}
```
*Γιατί;* Η ρύθμιση παραμέτρων επιλογών σάς επιτρέπει να καθορίσετε τη μορφή προορισμού και τυχόν πρόσθετες ρυθμίσεις που απαιτούνται κατά τη μετατροπή.

#### Βήμα 4: Εκτέλεση μετατροπής
Εκτελέστε τη διαδικασία μετατροπής και αποθηκεύστε το αρχείο σας:

```csharp
// Μετατρέψτε το αρχείο CDR σε TXT και αποθηκεύστε το
class Program
{
    static void Main(string[] args)
    {
        using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY\sample.cdr"))
        {
            WordProcessingConvertOptions options = new WordProcessingConvertOptions 
            {
                Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt
            };
            converter.Convert(outputFile, options);
        }
    }
}
```
*Γιατί;* Η εκτέλεση αυτού του βήματος ολοκληρώνει τη διαδικασία μετατροπής, δημιουργώντας ένα αρχείο TXT από το αρχικό σας έγγραφο CDR.
