## Οδηγός Εφαρμογής

### Μετατροπή αρχείων FODP σε μορφή TXT
Αυτή η λειτουργία δείχνει πώς να μετατρέψετε ένα αρχείο OpenDocument Flat XML Presentation (.fodp) σε μορφή απλού αρχείου κειμένου (.txt).

#### Βήμα 1: Ρύθμιση καταλόγου εξόδου και διαδρομής αρχείου
Ορίστε τον κατάλογο εξόδου για τα αρχεία κειμένου που έχετε μετατρέψει.

```csharp
using System;
using System.IO;

// Ορίστε τον κατάλογο εξόδου.
string outputFolder = @"YOUR_OUTPUT_DIRECTORY/";

// Δημιουργήστε μια συνδυασμένη διαδρομή για το αρχείο εξόδου.
string outputFile = Path.Combine(outputFolder, "fodp-converted-to.txt");
```

#### Βήμα 2: Φόρτωση και μετατροπή του αρχείου FODP
Φορτώστε την πηγή σας `.fodp` αρχείο χρησιμοποιώντας το GroupDocs.Conversion's `Converter` κλάση. Αντικαταστήστε το σύμβολο κράτησης θέσης με την πραγματική διαδρομή του εγγράφου σας.

```csharp
using (var converter = new Converter(@"YOUR_DOCUMENT_DIRECTORY/sample.fodp"))
{
    // Ορίστε τις επιλογές μετατροπής σε μορφή TXT.
    var options = new WordProcessingConvertOptions 
    {
        Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt
    };

    // Εκτελέστε τη μετατροπή και αποθηκεύστε την ως αρχείο .txt.
    converter.Convert(outputFile, options);
}
```
**Εξήγηση**: Το `Converter` το αντικείμενο φορτώνει το `.fodp` έγγραφο. Οι επιλογές μετατροπής έχουν οριστεί για να μετατρέψουν το αρχείο σε μορφή TXT χρησιμοποιώντας `WordProcessingConvertOptions`.
