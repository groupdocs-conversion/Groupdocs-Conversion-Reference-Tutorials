# Μετατροπή PSD σε TXT με το GroupDocs.Conversion για .NET: Οδηγός βήμα προς βήμα

Σε αυτό το σεμινάριο, θα μάθετε:
- Πώς να φορτώσετε ένα αρχείο PSD πηγής
- Ρύθμιση παραμέτρων επιλογών μετατροπής για έξοδο σε μορφή TXT
- Εκτέλεση της μετατροπής και αποθήκευση των αποτελεσμάτων σας

## Οδηγός Εφαρμογής

### Φόρτωση αρχείου πηγής

**Επισκόπηση:** Η φόρτωση ενός αρχείου PSD είναι απαραίτητη για την πρόσβαση και τον χειρισμό του εγγράφου προέλευσης.

#### Βήμα 1: Καθορίστε τη διαδρομή του αρχείου προέλευσης
```csharp
string sourceFilePath = @"YOUR_DOCUMENT_DIRECTORY\sample.psd";
```
*Εξήγηση:* Αντικαθιστώ `YOUR_DOCUMENT_DIRECTORY` με τη διαδρομή προς το αρχείο PSD σας, διασφαλίζοντας την ακριβή ανάκτηση της τοποθεσίας.

### Ρύθμιση παραμέτρων επιλογών μετατροπής

**Επισκόπηση:** Ο ορισμός των επιλογών μετατροπής είναι κρίσιμος για την προσαρμογή της μορφής εξόδου TXT.

#### Βήμα 2: Ρύθμιση επιλογών μετατροπής
```csharp
using GroupDocs.Conversion.Options.Convert;

WordProcessingConvertOptions options = new WordProcessingConvertOptions { Format = FileTypes.WordProcessingFileType.Txt };
```
*Εξήγηση:* Αυτό ορίζει ότι η μορφή εξόδου θα πρέπει να είναι TXT. `WordProcessingConvertOptions` Η κλάση χρησιμοποιείται για μετατροπές που σχετίζονται με κείμενο.

### Εκτέλεση μετατροπής και αποθήκευση εξόδου

**Επισκόπηση:** Μετατροπή από PSD σε TXT και αποθήκευση σε έναν καθορισμένο κατάλογο.

#### Βήμα 3: Ορισμός καταλόγου εξόδου
```csharp
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY";

if (!Directory.Exists(outputDirectory))
{
    Directory.CreateDirectory(outputDirectory);
}
```
*Εξήγηση:* Βεβαιωθείτε ότι η διαδρομή εξόδου υπάρχει ή δημιουργήστε την για να αποφύγετε σφάλματα κατά την αποθήκευση του αρχείου.

#### Βήμα 4: Εκτέλεση μετατροπής και αποθήκευση αρχείου
```csharp
string outputFile = Path.Combine(outputDirectory, "psd-converted-to.txt");

using (var converter = new Converter(@"YOUR_DOCUMENT_DIRECTORY\sample.psd"))
{
    WordProcessingConvertOptions options = new WordProcessingConvertOptions { Format = FileTypes.WordProcessingFileType.Txt };

    // Εκτέλεση μετατροπής και αποθήκευση εξόδου
    converter.Convert(outputFile, options);
}
```
*Εξήγηση:* Αρχικοποίηση του `Converter` με το αρχείο PSD σας, ορίστε τις επιλογές μετατροπής, εκτελέστε τη μετατροπή και αποθηκεύστε το ως "psd-converted-to.txt".
