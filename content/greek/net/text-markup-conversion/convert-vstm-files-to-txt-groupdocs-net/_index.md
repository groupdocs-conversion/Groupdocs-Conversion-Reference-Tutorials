### Χαρακτηριστικό: Μετατροπή VSTM σε TXT
Αυτή η λειτουργία δείχνει πώς να αξιοποιήσετε το GroupDocs.Conversion για τη μετατροπή ενός αρχείου VSTM σε μορφή απλού κειμένου.

#### Βήμα 1: Ορισμός καταλόγου εξόδου και διαδρομής αρχείου
Καθορίστε πού θα αποθηκευτούν τα αρχεία TXT που έχετε μετατρέψει:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY/";
string outputFile = Path.Combine(outputFolder, "vstm-converted-to.txt");
```

#### Βήμα 2: Φόρτωση του αρχείου VSTM προέλευσης
Αρχικοποίηση ενός `Converter` αντικείμενο για να φορτώσετε το αρχείο VSTM. Βεβαιωθείτε ότι η διαδρομή δείχνει στον πραγματικό κατάλογο εγγράφων σας:

```csharp
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY/sample.vstm"))
{
    // Η λογική μετατροπής θα εφαρμοστεί εδώ.
}
```

#### Βήμα 3: Ρύθμιση επιλογών μετατροπής
Ρυθμίστε τις παραμέτρους μετατροπής για τη μετατροπή ενός αρχείου επεξεργασίας κειμένου σε μορφή TXT:

```csharp
var options = new WordProcessingConvertOptions 
{
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt
};
```

#### Βήμα 4: Εκτελέστε τη μετατροπή και αποθηκεύστε την έξοδο
Εκτελέστε τη διαδικασία μετατροπής, αποθηκεύοντας το αποτέλεσμα ως αρχείο TXT:

```csharp
converter.Convert(outputFile, options);
```
