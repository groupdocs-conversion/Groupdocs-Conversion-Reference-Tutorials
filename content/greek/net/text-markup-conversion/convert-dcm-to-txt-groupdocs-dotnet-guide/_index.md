## Οδηγός Εφαρμογής

### Μετατροπή DCM σε TXT

Ακολουθήστε αυτά τα βήματα για να μετατρέψετε ένα αρχείο DICOM σε μορφή κειμένου:

#### Βήμα 1: Ορισμός διαδρομών αρχείων

Ορίστε διαδρομές τόσο για τα αρχεία εισόδου όσο και για τα αρχεία εξόδου. Αντικαταστήστε τα placeholders με τις πραγματικές διαδρομές καταλόγων στο σύστημά σας.

```csharp
string sourceFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.dcm");
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string outputFile = Path.Combine(outputFolder, "dcm-converted-to.txt");

if (!Directory.Exists(outputFolder))
{
    Directory.CreateDirectory(outputFolder);
}
```

#### Βήμα 2: Φόρτωση του αρχείου DCM

Χρήση του GroupDocs.Conversion `Converter` κλάση, φορτώστε το αρχείο πηγαίου κώδικα.

```csharp
using (var converter = new Converter(sourceFile))
{
    // Τα βήματα μετατροπής θα προστεθούν εδώ
}
```

#### Βήμα 3: Ορισμός επιλογών μετατροπής

Καθορίστε τις ρυθμίσεις μετατροπής για τη μορφή TXT χρησιμοποιώντας `WordProcessingConvertOptions`.

```csharp
WordProcessingConvertOptions options = new WordProcessingConvertOptions { Format = FileTypes.WordProcessingFileType.Txt };
```

#### Βήμα 4: Εκτελέστε τη μετατροπή

Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο εξόδου.

```csharp
converter.Convert(outputFile, options);
```
