## Οδηγός Εφαρμογής

### Μετατροπή DOC σε TXT
Αυτή η λειτουργία επικεντρώνεται στην απρόσκοπτη μετατροπή εγγράφων Word σε αρχεία απλού κειμένου.

#### Βήμα 1: Ορισμός διαδρομών καταλόγου
Ορίστε τις διαδρομές για τους καταλόγους εισόδου και εξόδου χρησιμοποιώντας placeholders:

```csharp
string documentDirectory = "@YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "@YOUR_OUTPUT_DIRECTORY";
```

#### Βήμα 2: Φόρτωση του αρχείου DOC προέλευσης
Φορτώστε το έγγραφο του Word με το `Converter` κλάση καθορίζοντας τη διαδρομή της.

```csharp
using (var converter = new Converter(Path.Combine(documentDirectory, "sample.doc")))
{
    // Η λογική μετατροπής θα εφαρμοστεί εδώ.
}
```
**Γιατί;** Αυτό το βήμα ξεκινά τη διαδικασία μετατροπής προετοιμάζοντας το αρχείο προέλευσης για μετασχηματισμό.

#### Βήμα 3: Ορισμός επιλογών μετατροπής
Ορίστε τις ρυθμίσεις μετατροπής για να καθορίσετε ότι μετατρέπετε από μορφή Word σε μορφή TXT:

```csharp
var options = new WordProcessingConvertOptions { Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt };
```
**Γιατί;** Αυτή η διαμόρφωση διασφαλίζει ότι το αρχείο εξόδου έχει την επιθυμητή μορφή κειμένου, διατηρώντας την απλότητα και την αναγνωσιμότητα.

#### Βήμα 4: Εκτελέστε τη μετατροπή
Εκτελέστε τη μετατροπή και αποθηκεύστε το νέο αρχείο TXT σε μια καθορισμένη τοποθεσία:

```csharp
string outputFile = Path.Combine(outputDirectory, "doc-converted-to.txt");
converter.Convert(outputFile, options);
```
**Γιατί;** Αυτό το τελευταίο βήμα ολοκληρώνει τη διαδικασία δημιουργώντας το αρχείο κειμένου που έχει μετατραπεί στον κατάλογο εξόδου σας.
