## Οδηγός Εφαρμογής

Ας αναλύσουμε τη διαδικασία μετατροπής ενός DIB σε έγγραφο LaTeX σε διαχειρίσιμα βήματα.

### Μετατροπή DIB σε TEX

Αυτή η λειτουργία μετατρέπει ένα αρχείο Bitmap ανεξάρτητο από συσκευή (DIB) σε ένα έγγραφο προέλευσης LaTeX (.tex).

#### Βήμα 1: Ορισμός καταλόγου εξόδου και ονόματος αρχείου

Ορίστε τον κατάλογο εξόδου σας και δημιουργήστε τη διαδρομή αρχείου για το έγγραφο που έχει μετατραπεί. Αντικαταστήστε τα placeholders με τις πραγματικές διαδρομές:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY"; // Καθορίστε τον κατάλογο εξόδου σας
string outputFile = Path.Combine(outputFolder, "dib-converted-to.tex");
```

#### Βήμα 2: Φόρτωση αρχείου DIB προέλευσης

Φορτώστε το αρχείο DIB πηγής χρησιμοποιώντας το `Converter` κλάση. Βεβαιωθείτε ότι έχετε αντικαταστήσει το σύμβολο κράτησης θέσης με τη σωστή διαδρομή:

```csharp
using (var converter = new Converter("@YOUR_DOCUMENT_DIRECTORY/SAMPLE_DIB"))
{
    // Η λογική μετατροπής θα τοποθετηθεί εδώ.
}
```

#### Βήμα 3: Ορισμός επιλογών μετατροπής

Ρυθμίστε τις παραμέτρους μετατροπής για να καθορίσετε ότι η μορφή εξόδου θα πρέπει να είναι LaTeX (.tex):

```csharp
PageDescriptionLanguageConvertOptions options = new PageDescriptionLanguageConvertOptions 
{ 
    Format = GroupDocs.Conversion.FileTypes.PageDescriptionLanguageFileType.Tex 
};
```

#### Βήμα 4: Εκτέλεση μετατροπής και αποθήκευση

Εκτελέστε τη διαδικασία μετατροπής και αποθηκεύστε το αρχείο .tex που προκύπτει:

```csharp
converter.Convert(outputFile, options);
```
