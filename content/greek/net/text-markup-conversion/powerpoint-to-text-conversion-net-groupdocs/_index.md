## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Φόρτωση και μετατροπή PPT σε TXT
Αυτή η λειτουργία παρουσιάζει πώς μπορείτε να φορτώσετε ένα αρχείο παρουσίασης PowerPoint (.ppt) και να το μετατρέψετε σε μορφή απλού κειμένου (.txt).
#### Βήμα 1: Ορισμός διαδρομών για αρχεία εισόδου και εξόδου
Ξεκινήστε καθορίζοντας πού θα αποθηκευτούν τα αρχεία PPT πηγής και TXT εξόδου.
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY"; // Ενημερώστε με τον κατάλογο PPT σας.
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";     // Ενημερώστε με τη διαδρομή του καταλόγου εξόδου σας.
// Ορίστε διαδρομές για το αρχείο εισόδου PowerPoint και το αρχείο κειμένου εξόδου.
string pptFilePath = Path.Combine(documentDirectory, "sample.ppt"); // Αντικαταστήστε το 'sample.ppt' με το πραγματικό όνομα αρχείου σας.
string txtOutputFile = Path.Combine(outputDirectory, "ppt-converted-to.txt");
```
#### Βήμα 2: Φόρτωση του αρχείου PPT προέλευσης
Χρησιμοποιήστε το GroupDocs.Conversion για να φορτώσετε το αρχείο PowerPoint.
```csharp
using (var converter = new Converter(pptFilePath))
{
    // Το αρχείο PPT έχει πλέον φορτωθεί και είναι έτοιμο για μετατροπή.
}
```
#### Βήμα 3: Καθορισμός επιλογών μετατροπής
Ρυθμίστε τις παραμέτρους για να μετατρέψετε το έγγραφο σε μορφή TXT.
```csharp
// Ορίστε επιλογές μετατροπής κειμένου χρησιμοποιώντας το WordProcessingConvertOptions.
var options = new WordProcessingConvertOptions { Format = FileTypes.WordProcessingFileType.Txt };
```
#### Βήμα 4: Μετατρέψτε το αρχείο PPT σε TXT
Εκτελέστε τη διαδικασία μετατροπής με τις καθορισμένες επιλογές σας.
```csharp
converter.Convert(txtOutputFile, options);
```
