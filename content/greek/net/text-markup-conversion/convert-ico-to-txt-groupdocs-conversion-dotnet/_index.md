## Οδηγός Εφαρμογής

### Φόρτωση αρχείου ICO
**Επισκόπηση:** 
Η φόρτωση ενός αρχείου ICO είναι το πρώτο σας βήμα πριν από τη μετατροπή.

#### Βήμα 1: Δημιουργία μιας παρουσίας μετατροπέα
```csharp
using System;
using GroupDocs.Conversion;

string documentPath = "YOUR_DOCUMENT_DIRECTORY/sample.ico"; // Αντικαταστήστε με τη διαδρομή αρχείου ICO
using (var converter = new Converter(documentPath))
{
    Console.WriteLine("ICO file loaded successfully!");
}
```
**Εξήγηση:** 
Ο `Converter` Η κλάση αρχικοποιείται με τη διαδρομή προς το αρχείο ICO σας. Αυτό την προετοιμάζει για επόμενες λειτουργίες.

### Μετατροπή ICO σε μορφή TXT
**Επισκόπηση:**
Μετατρέψτε το φορτωμένο αρχείο ICO σε μορφή κειμένου χρησιμοποιώντας συγκεκριμένες επιλογές μετατροπής.

#### Βήμα 2: Ορισμός επιλογών μετατροπής
```csharp
using System.IO;
using GroupDocs.Conversion.Options.Convert;

string outputFolder = "YOUR_OUTPUT_DIRECTORY"; // Αντικαταστήστε με την επιθυμητή διαδρομή καταλόγου εξόδου
string outputFile = Path.Combine(outputFolder, "ico-converted-to.txt");

WordProcessingConvertOptions options = new WordProcessingConvertOptions { Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt };
```
**Εξήγηση:** 
Εδώ, ορίζουμε τη μορφή μετατροπής ως TXT χρησιμοποιώντας `WordProcessingConvertOptions`.

#### Βήμα 3: Εκτέλεση μετατροπής
```csharp
using (var converter = new Converter("YOUR_DOCUMENT_DIRECTORY/sample.ico"))
{
    converter.Convert(outputFile, options);
    Console.WriteLine("Conversion to TXT completed successfully!");
}
```
**Εξήγηση:** 
Ο `Convert` Η μέθοδος λαμβάνει τη διαδρομή του αρχείου εξόδου και τις επιλογές μετατροπής για να εκτελέσει τη μετατροπή.
