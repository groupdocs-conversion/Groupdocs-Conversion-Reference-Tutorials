## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Μετατροπή VSTX σε TEX

Αυτή η ενότητα παρουσιάζει τη μετατροπή ενός αρχείου VSTX σε μορφή TEX χρησιμοποιώντας το GroupDocs.Conversion για .NET:

#### Βήμα 1: Ορισμός καταλόγου εξόδου και διαδρομής αρχείου
Καθορίστε πού θα αποθηκευτούν τα αρχεία TEX που έχετε μετατρέψει:
```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputFolder, "vstx-converted-to.tex");
```

#### Βήμα 2: Αρχικοποίηση του μετατροπέα με ένα αρχείο VSTX πηγής
Φορτώστε και αρχικοποιήστε τον μετατροπέα για το αρχείο VSTX πηγής σας:
```csharp
using (var converter = new GroupDocs.Conversion.Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_VSTX"))
{
    // Ρύθμιση παραμέτρων επιλογών μετατροπής για τη μορφή TEX
    PageDescriptionLanguageConvertOptions options = new PageDescriptionLanguageConvertOptions 
    {
        Format = PageDescriptionLanguageFileType.Tex
    };
    
    // Εκτελέστε τη μετατροπή από VSTX σε TEX
    converter.Convert(outputFile, options);
}
```

**Εξήγηση:**
- **Αρχικοποίηση μετατροπέα**: Το `GroupDocs.Conversion.Converter` Η κλάση φορτώνει το αρχείο πηγαίου κώδικα.
- **Επιλογές μετατροπής**: Καθορίστε `PageDescriptionLanguageFileType.Tex` για μετατροπή σε μορφή TEX.

### Χαρακτηριστικό: Φόρτωση αρχείου πηγής και ρύθμιση παραμέτρων επιλογών μετατροπής
Αυτή η ενότητα εξηγεί τη διαμόρφωση επιλογών μετατροπής για διαφορετικές περιπτώσεις χρήσης:

#### Βήμα 1: Ορισμός καταλόγου εξόδου και διαδρομής αρχείου εξόδου
Βεβαιωθείτε ότι η διαδρομή του αρχείου εξόδου έχει οριστεί σωστά, όπως φαίνεται παραπάνω.

#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών μετατροπής για τη μορφή TEX
