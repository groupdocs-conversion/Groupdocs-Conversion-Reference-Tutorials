#### Βήμα 1: Ορισμός διαδρομών αρχείων
Ξεκινήστε ορίζοντας τις διαδρομές εισόδου και εξόδου για τα αρχεία σας:

```csharp
using System.IO;

// Σταθερές για καταλόγους
cnst string inputFilePath = "YOUR_DOCUMENT_DIRECTORY\\\\sample.xlsm";
cnst string outputDirectoryPath = "YOUR_OUTPUT_DIRECTORY";

// Διαδρομή αρχείου εξόδου

```

#### Βήμα 2: Φόρτωση του αρχείου προέλευσης
Χρησιμοποιήστε το `Converter` κλάση για να φορτώσετε το αρχείο XLSM σας:

```csharp

{
    // Η λογική μετατροπής θα εφαρμοστεί εδώ.
}
```
**Εξήγηση**: Το `Converter` Το αντικείμενο χειρίζεται τη διαδικασία μετατροπής, διασφαλίζοντας ότι οι πόροι διαχειρίζονται και απορρίπτονται σωστά.

#### Βήμα 3: Ορισμός επιλογών μετατροπής
Καθορίστε τη μορφή στόχου χρησιμοποιώντας τις επιλογές μετατροπής:

```csharp
var options = new WordProcessingConvertOptions { Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt };
```
**Εξήγηση**: Το `WordProcessingConvertOptions` σας επιτρέπει να διαμορφώσετε ρυθμίσεις ειδικά για μετατροπές που βασίζονται σε κείμενο.

#### Βήμα 4: Εκτελέστε τη μετατροπή
Εκτελέστε τη μετατροπή και αποθηκεύστε την έξοδο:

```csharp

```
**Εξήγηση**Αυτή η μέθοδος εκτελεί τον πραγματικό μετασχηματισμό αρχείου, γράφοντας το αποτέλεσμα στην καθορισμένη θέση.
