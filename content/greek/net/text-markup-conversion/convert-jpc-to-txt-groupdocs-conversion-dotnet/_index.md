**Γιατί αυτό το βήμα;** Η φόρτωση του αρχείου αρχικοποιεί τα δεδομένα που απαιτούνται για τη μετατροπή.

### Βήμα 2: Ορισμός επιλογών μετατροπής

Διαμορφώστε τις επιλογές μετατροπής για να καθορίσετε τη μορφή-στόχο:

```csharp
WordProcessingConvertOptions options = new WordProcessingConvertOptions
{
    Format = WordProcessingFileType.Txt
};
```
**Γιατί αυτές οι επιλογές;** Σύνθεση `Format` ως `Txt` διασφαλίζει ότι το αρχείο εξόδου είναι σε μορφή απλού κειμένου, κάτι που είναι κρίσιμο για περαιτέρω επεξεργασία ή ανάλυση δεδομένων.

### Βήμα 3: Μετατροπή και αποθήκευση του αρχείου TXT

Μετατρέψτε το αρχείο JPC σε αρχείο TXT χρησιμοποιώντας τις καθορισμένες επιλογές:

```csharp
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY", "ConversionResults");
Directory.CreateDirectory(outputFolder);
string outputFile = Path.Combine(outputFolder, "jpc-converted-to.txt");

converter.Convert(outputFile, options);
```
**Γιατί Αυτός ο Κώδικας;** Αυτός ο κώδικας εκτελεί την πραγματική μετατροπή και αποθηκεύει το αρχείο TXT που προκύπτει στον καθορισμένο κατάλογο.
