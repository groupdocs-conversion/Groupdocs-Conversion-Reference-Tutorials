### Μετατροπή JPM σε TEX

**Επισκόπηση:**
Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε αρχεία εικόνας JPEG 2000 σε έγγραφα LaTeX. Θα χρησιμοποιήσουμε τις επιλογές μετατροπής που παρέχονται από το GroupDocs για να το πετύχουμε αυτό.

#### Βήμα προς βήμα εφαρμογή
1. **Φόρτωση του αρχείου προέλευσης**
   Ξεκινήστε φορτώνοντας το αρχείο .jpm χρησιμοποιώντας μια διαδρομή κράτησης θέσης για τον κατάλογο εγγράφων:
   ```csharp
   string inputFilePath = @"YOUR_DOCUMENT_DIRECTORY\\sample.jpm";
   ```
2. **Ορισμός επιλογών μετατροπής**
   Καθορίστε τη μορφή εξόδου ως TEX ορίζοντας τις κατάλληλες επιλογές μετατροπής:
   ```csharp
   var options = new PageDescriptionLanguageConvertOptions { Format = PageDescriptionLanguageFileType.Tex };
   ```
3. **Εκτελέστε τη μετατροπή**
   Εκτελέστε τη μετατροπή και αποθηκεύστε το αρχείο .tex που προκύπτει στην επιθυμητή θέση:
   ```csharp
   string outputFolder = @"YOUR_OUTPUT_DIRECTORY";
   string outputFile = Path.Combine(outputFolder, "jpm-converted-to.tex");

   using (var converter = new Converter(inputFilePath))
   {
       converter.Convert(outputFile, options);
   }
   ```
**Επεξήγηση Παραμέτρων και Μεθόδων:**
- `PageDescriptionLanguageConvertOptions`: Ρυθμίζει τις παραμέτρους της μετατροπής για την έξοδο ενός αρχείου TEX.
- `converter.Convert()`: Εκτελεί τη διαδικασία μετατροπής.
