## Οδηγός Εφαρμογής

### Φόρτωση και μετατροπή αρχείου MPT σε TXT

Ακολουθήστε αυτά τα βήματα για να φορτώσετε ένα αρχείο MPT και να το μετατρέψετε σε μορφή TXT.

#### Βήμα 1: Ορισμός διαδρομών καταλόγου

Ξεκινήστε ορίζοντας διαδρομές για το έγγραφο εισόδου και τον κατάλογο εξόδου:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// Ορίστε τη διαδρομή προς το αρχείο MPT προέλευσης και το αρχείο TXT εξόδου.
string mptFilePath = Path.Combine(documentDirectory, "sample.mpt");
string txtOutputFile = Path.Combine(outputDirectory, "mpt-converted-to.txt");
```

#### Βήμα 2: Φόρτωση του αρχείου MPT

Χρησιμοποιήστε το `Converter` κλάση από το GroupDocs.Conversion για να φορτώσετε το αρχείο MPT σας:

```csharp
using (var converter = new Converter(mptFilePath))
{
    // Η μετατροπή θα πραγματοποιηθεί εδώ.
}
```

#### Βήμα 3: Ορισμός επιλογών μετατροπής

Καθορίστε επιλογές μετατροπής για τη μετατροπή ενός MPT σε μορφή TXT:

```csharp
WordProcessingConvertOptions options = new WordProcessingConvertOptions 
{ 
    Format = GroupDocs.Conversion.FileTypes.WordProcessingFileType.Txt 
};
```

**Εξήγηση**: Το `WordProcessingConvertOptions` σας επιτρέπει να ορίσετε τον τύπο αρχείου προορισμού. Εδώ, το ορίζουμε σε `Txt`.

#### Βήμα 4: Εκτέλεση μετατροπής

Εκτελέστε τη μετατροπή και αποθηκεύστε την έξοδο:

```csharp
converter.Convert(txtOutputFile, options);
```
