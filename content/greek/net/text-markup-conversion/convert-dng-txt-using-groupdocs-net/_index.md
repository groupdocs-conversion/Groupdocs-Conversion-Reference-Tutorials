## Οδηγός Εφαρμογής
Ας εμβαθύνουμε στην βασική λειτουργικότητα: τη μετατροπή ενός αρχείου DNG σε μορφή TXT χρησιμοποιώντας το GroupDocs.Conversion.

### Φόρτωση και μετατροπή αρχείου DNG σε TXT
#### Επισκόπηση
Σε αυτήν την ενότητα, θα φορτώσουμε ένα αρχείο Digital Negative (DNG) και θα το μετατρέψουμε σε αρχείο απλού κειμένου. Αυτή η διαδικασία χρησιμοποιεί το ισχυρό API του GroupDocs.Conversion.

#### Βήμα 1: Ρύθμιση διαδρομών αρχείων
Ξεκινήστε ορίζοντας τις διαδρομές για το αρχείο DNG εισόδου και το αρχείο TXT εξόδου:
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY"; // Διαδρομή προς το αρχείο DNG
string outputDirectory = "YOUR_OUTPUT_DIRECTORY"; // Κατάλογος όπου θα αποθηκευτεί το TXT

// Προετοιμάστε την πλήρη διαδρομή για το αρχείο DNG πηγής
string sourceDngPath = Path.Combine(documentDirectory, "sample.dng");

// Προετοιμασία της διαδρομής του αρχείου εξόδου
string outputFile = Path.Combine(outputDirectory, "dng-converted-to.txt");
```
*Σημείωση: Αντικαταστήστε τα "YOUR_DOCUMENT_DIRECTORY" και "YOUR_OUTPUT_DIRECTORY" με τις πραγματικές διαδρομές στο σύστημά σας.*

#### Βήμα 2: Μετατροπή DNG σε TXT
Χρήση του GroupDocs.Conversion `Converter` κλάση για να φορτώσετε το αρχείο DNG και να καθορίσετε επιλογές μετατροπής για τη μορφή TXT:
```csharp
using (var converter = new Converter(sourceDngPath))
{
    // Ορισμός επιλογών μετατροπής για τη μορφή TXT
    var options = new WordProcessingConvertOptions { Format = FileTypes.WordProcessingFileType.Txt };
    
    // Εκτελέστε τη μετατροπή και αποθηκεύστε την στην καθορισμένη διαδρομή
    converter.Convert(outputFile, options);
}
```
*Εξήγηση: Το `Converter` το αντικείμενο φορτώνει το αρχείο DNG σας. Ορίζοντας `WordProcessingConvertOptions`, καθορίζετε ότι η έξοδος θα πρέπει να είναι σε μορφή TXT.*
