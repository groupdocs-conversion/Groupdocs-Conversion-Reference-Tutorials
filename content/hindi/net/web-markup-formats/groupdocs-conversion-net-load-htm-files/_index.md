### आप क्या सीखेंगे:
- .NET के लिए GroupDocs.Conversion सेट अप करना

```csharp
using GroupDocs.Conversion;

// अपने दस्तावेज़ निर्देशिका का पथ निर्धारित करें
string documentDirectory = "/path/to/your/documents";
```

## कार्यान्वयन मार्गदर्शिका

### स्रोत HTM फ़ाइल लोड करें

GroupDocs.Conversion का उपयोग करके रूपांतरण के लिए HTML दस्तावेज़ लोड और तैयार करना सीखें।

#### चरण 1: दस्तावेज़ निर्देशिका परिभाषित करें
सबसे पहले, वह निर्देशिका निर्दिष्ट करें जहां आपके दस्तावेज़ स्थित हैं:

```csharp
string documentDirectory = "/path/to/your/documents";
```

#### चरण 2: कनवर्टर ऑब्जेक्ट को आरंभ करें
एक बनाने के `Converter` फ़ाइल लोडिंग प्रक्रिया को प्रबंधित करने के लिए ऑब्जेक्ट:

```csharp
using (Converter converter = new Converter(Path.Combine(documentDirectory, "sample.htm")))
{
    // रूपांतरण कॉन्फ़िगरेशन और निष्पादन यहां होगा।
}
```

**पैरामीटर्स की व्याख्या:**
- `documentDirectory`: वह पथ जहाँ आपकी स्रोत फ़ाइलें स्थित हैं.
- `Path.Combine(...)`: पूर्ण पथ बनाने के लिए निर्देशिका पथ को फ़ाइल नाम के साथ संयोजित करता है।

#### चरण 3: रूपांतरण विकल्प कॉन्फ़िगर करें
अपनी आवश्यकताओं के अनुसार रूपांतरण सेटिंग्स कॉन्फ़िगर करें (जैसे, लक्ष्य प्रारूप):

```csharp
var options = new PdfConvertOptions(); // पीडीएफ में परिवर्तित करने का उदाहरण
```

**मुख्य कॉन्फ़िगरेशन विकल्प:**
- `PdfConvertOptions`: पीडीएफ रूपांतरण के लिए सेटिंग्स निर्दिष्ट करता है।
