## कार्यान्वयन मार्गदर्शिका

सब कुछ सेट हो जाने के बाद, आइए HTML से CSV रूपांतरण को क्रियान्वित करें।

### फ़ाइल लोड करना और परिवर्तित करना

1. **दस्तावेज़ पथ सेट करें**
   अपनी स्रोत फ़ाइलों और परिवर्तित आउटपुट के लिए निर्देशिकाएँ परिभाषित करें:
   ```csharp
   const string DocumentDirectory = "YOUR_DOCUMENT_DIRECTORY";
   const string OutputDirectory = "YOUR_OUTPUT_DIRECTORY";
   ```

2. **स्रोत HTM फ़ाइल लोड करें**
   उपयोग `Converter` अपनी HTML फ़ाइल लोड करने के लिए क्लास का उपयोग करें:
   ```csharp
   string inputFilePath = Path.Combine(DocumentDirectory, "sample.htm");

   using (var converter = new Converter(inputFilePath))
   {
       // रूपांतरण कोड यहां जाएगा.
   }
   ```

3. **रूपांतरण विकल्प परिभाषित करें**
   CSV प्रारूप के लिए रूपांतरण विकल्प सेट करें:
   ```csharp
   SpreadsheetConvertOptions options = new SpreadsheetConvertOptions { Format = SpreadsheetFileType.Csv };
   ```

4. **रूपांतरण करें**
   रूपांतरण निष्पादित करें और आउटपुट सहेजें:
   ```csharp
   string outputFilePath = Path.Combine(OutputDirectory, "converted.csv");
   converter.Convert(outputFilePath, options);
   ```

### समस्या निवारण युक्तियों
- सुनिश्चित करना `sample.htm` आपके दस्तावेज़ निर्देशिका में मौजूद है.
- पहुँच संबंधी समस्याओं को रोकने के लिए इनपुट और आउटपुट निर्देशिकाओं के लिए फ़ाइल अनुमतियों को सत्यापित करें।
