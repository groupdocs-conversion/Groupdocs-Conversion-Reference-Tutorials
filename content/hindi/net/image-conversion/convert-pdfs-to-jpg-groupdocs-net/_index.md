## कार्यान्वयन मार्गदर्शिका

अब, चलिए GroupDocs.Conversion का उपयोग करके PDF से JPG रूपांतरण सुविधा को लागू करते हैं।

### फ़ीचर: पीडीएफ से जेपीजी रूपांतरण

यह सुविधा आपको PDF फ़ाइल को अलग-अलग JPG इमेज में बदलने की सुविधा देती है। यह तब काम आता है जब आपको अपने दस्तावेज़ के हर पेज के उच्च-गुणवत्ता वाले स्नैपशॉट की ज़रूरत होती है।

#### चरण 1: आउटपुट निर्देशिका और टेम्पलेट परिभाषित करें

सबसे पहले, निर्दिष्ट करें कि परिवर्तित JPG फ़ाइलें कहाँ सहेजी जाएंगी और उनका नामकरण पैटर्न निर्धारित करें:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.jpg");
```

ये पंक्तियाँ आपकी छवियों को संग्रहीत करने के लिए एक फ़ोल्डर सेट करती हैं और फ़ाइल-नामकरण परंपरा स्थापित करती हैं।

#### चरण 2: पेज स्ट्रीम को संभालने के लिए एक फ़ंक्शन बनाएँ

इसके बाद, हमें एक फ़ंक्शन की आवश्यकता है जो प्रत्येक रूपांतरित पृष्ठ को JPG फ़ाइल के रूप में लिखे:

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext =>
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

यह अनाम फ़ंक्शन आउटपुट छवि को डिस्क पर लिखने के लिए एक स्ट्रीम बनाता है।

#### चरण 3: पीडीएफ लोड करें और कन्वर्ट करें

का उपयोग करके अपनी स्रोत पीडीएफ फाइल लोड करें `Converter` क्लास में, JPG के लिए रूपांतरण विकल्प सेट करें, और रूपांतरण करें:

```csharp
using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY\sample.pdf"))
{
    ImageConvertOptions options = new ImageConvertOptions { Format = GroupDocs.Conversion.FileTypes.ImageFileType.Jpg };
    converter.Convert(getPageStream, options);
}
```

यहाँ, `ImageConvertOptions` आउटपुट प्रारूप को JPG के रूप में निर्दिष्ट करता है। `Convert` विधि प्रदान की गई स्ट्रीम फ़ंक्शन का उपयोग करके प्रत्येक पृष्ठ को संसाधित करती है।

### विशेषता: फ़ाइल पथ कॉन्फ़िगरेशन

आपके अनुप्रयोग में स्वचालन और मापनीयता के लिए कुशल फ़ाइल पथ प्रबंधन महत्वपूर्ण है।

#### चरण 1: पथ फ़ंक्शन परिभाषित करें

आप गतिशील रूप से पथ निर्धारित करने के लिए उपयोगिता फ़ंक्शन बना सकते हैं:

```csharp
string GetOutputDirectoryPath()
{
    return Path.Combine(Directory.GetCurrentDirectory(), "YOUR_OUTPUT_DIRECTORY");
}

string GetSamplePdfPath()
{
    return Path.Combine(Directory.GetCurrentDirectory(), "YOUR_DOCUMENT_DIRECTORY", "sample.pdf");
}
```

ये विधियां स्वच्छ और संगठित कोडबेस बनाए रखने में मदद करती हैं।
