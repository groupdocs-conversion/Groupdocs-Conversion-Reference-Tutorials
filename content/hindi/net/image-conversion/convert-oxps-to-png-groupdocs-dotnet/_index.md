**आप क्या सीखेंगे:**

## कार्यान्वयन मार्गदर्शिका

यह अनुभाग दर्शाता है कि GroupDocs.Conversion लाइब्रेरी का उपयोग करके OXPS फ़ाइलों को PNG में कैसे परिवर्तित किया जाए।

### OXPS फ़ाइल लोड करना और परिवर्तित करना

#### अवलोकन
जानें कि OXPS फ़ाइल को कैसे लोड करें और कुशलतापूर्वक PNG प्रारूप में रूपांतरण कैसे करें।

**1. पथ स्थापित करना**
अपने इनपुट और आउटपुट निर्देशिकाओं के लिए पथ परिभाषित करें:

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string inputFile = "YOUR_DOCUMENT_DIRECTORY/SAMPLE_OXPS.oxps";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.png");
```

**2. प्रत्येक पेज के लिए एक स्ट्रीम बनाना**
रूपांतरण के दौरान गतिशील रूप से स्ट्रीम बनाने के लिए फ़ंक्शन का उपयोग करें:

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext =>
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

**3. रूपांतरण प्रक्रिया**
OXPS फ़ाइल लोड करें और GroupDocs.Conversion का उपयोग करके इसे परिवर्तित करें:

```csharp
using (Converter converter = new Converter(inputFile))
{
    ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Png };
    converter.Convert(getPageStream, options);
}
```

### PNG प्रारूप के लिए रूपांतरण विकल्प सेट करना

#### अवलोकन
PNG प्रारूप के लिए विशेष रूप से तैयार छवि रूपांतरण सेटिंग्स कॉन्फ़िगर करें।

**1. कन्वर्ट विकल्प आरंभ करना**
इसका एक उदाहरण बनाकर शुरू करें `ImageConvertOptions`:

```csharp
ImageConvertOptions options = new ImageConvertOptions();
```

**2. आउटपुट प्रारूप निर्दिष्ट करना**
इच्छित आउटपुट प्रारूप को PNG पर सेट करें:

```csharp
options.Format = ImageFileType.Png;
```

- **फ़ाइल पथ संबंधी समस्याएँ:** सुनिश्चित करें कि सभी फ़ाइल पथ सही ढंग से सेट हैं.
