# .NET के लिए GroupDocs.Conversion का उपयोग करके XLS को PNG में परिवर्तित करें: एक चरण-दर-चरण मार्गदर्शिका

## कार्यान्वयन मार्गदर्शिका

### XLS को PNG में लोड करें और परिवर्तित करें

#### अवलोकन:

यह अनुभाग एक एक्सेल फ़ाइल को लोड करने और प्रत्येक शीट को अलग-अलग PNG छवियों में परिवर्तित करने पर केंद्रित है।

#### चरण 1: पथ परिभाषित करें

सुनिश्चित करें कि आपका दस्तावेज़ पथ और आउटपुट फ़ोल्डर सही ढंग से सेट किया गया है। इनपुट फ़ाइल का पता लगाने और परिवर्तित छवियों को संग्रहीत करने के लिए यह महत्वपूर्ण है।

```csharp
string documentPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.xls");
string outputFolder = Path.Combine("YOUR_OUTPUT_DIRECTORY");

if (!Directory.Exists(outputFolder))
{
    Directory.CreateDirectory(outputFolder);
}
```

#### चरण 2: कनवर्टर आरंभ करें

एक बनाने के `Converter` XLS फ़ाइल को संभालने के लिए इंस्टेंस। यह ऑब्जेक्ट रूपांतरण प्रक्रिया का प्रबंधन करता है।

```csharp
using (Converter converter = new Converter(documentPath))
{
    // रूपांतरण तर्क यहाँ है
}
```

#### चरण 3: रूपांतरण विकल्प सेट करें

आउटपुट प्रारूप और अतिरिक्त सेटिंग्स को परिभाषित करें `ImageConvertOptions`.

```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Png };
```

#### चरण 4: आउटपुट टेम्पलेट परिभाषित करें

संगठित फ़ाइल भंडारण सुनिश्चित करने के लिए प्रत्येक परिवर्तित PNG पृष्ठ के लिए एक नामकरण टेम्प्लेट बनाएं।

```csharp
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.png");
```

#### चरण 5: पेज स्ट्रीम उत्पन्न करें

प्रत्येक PNG फ़ाइल के लिए आउटपुट स्ट्रीम जेनरेट करने के लिए एक फ़ंक्शन सेट करें। छवियों को डिस्क पर लिखने के लिए यह आवश्यक है।

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext =>
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

#### चरण 6: रूपांतरण करें

कॉल करके रूपांतरण प्रक्रिया निष्पादित करें `Convert`, अपने पेज स्ट्रीम फ़ंक्शन और विकल्पों को पास करना।

```csharp
converter.Convert(getPageStream, options);
```

### समस्या निवारण युक्तियों

- **सुनिश्चित करें कि फ़ाइल पथ सही हैं:** फ़ाइल नहीं मिली त्रुटि से बचने के लिए निर्देशिका पथ की दोबारा जांच करें।
- **लाइब्रेरी संस्करण सत्यापित करें:** सुनिश्चित करें कि आपके पास GroupDocs.Conversion का सही संस्करण स्थापित है।
- **अनुमतियाँ जांचें:** सुनिश्चित करें कि आपके एप्लिकेशन में आउटपुट निर्देशिका के लिए लेखन अनुमति है।
