## नामस्थान आयात करें
रूपांतरण के साथ आगे बढ़ने से पहले, आइए आवश्यक नामस्थानों को अपनी परियोजना में आयात करें।
```csharp
using System;
using System.IO;
using GroupDocs.Conversion.Options.Convert;
```
## चरण 1: आउटपुट फ़ोल्डर और फ़ाइल नाम निर्धारित करें
सबसे पहले उस फ़ोल्डर को निर्दिष्ट करें जहां आप परिवर्तित पीडीएफ फाइल को सहेजना चाहते हैं और आउटपुट फाइल का नाम भी निर्दिष्ट करें।
```csharp
string outputFolder = "Your Document Directory";
string outputFile = Path.Combine(outputFolder, "mpx-converted-to.pdf");
```
## चरण 2: स्रोत MPX फ़ाइल लोड करें
इसके बाद, GroupDocs.Conversion का उपयोग करके स्रोत MPX फ़ाइल लोड करें।
```csharp
using (var converter = new GroupDocs.Conversion.Converter(Constants.SAMPLE_MPX))
{
    // रूपांतरण कोड यहाँ जाएगा
}
```
## चरण 3: रूपांतरण विकल्प सेट करें
आउटपुट प्रारूप को PDF के रूप में निर्दिष्ट करते हुए रूपांतरण विकल्प परिभाषित करें।
```csharp
var options = new PdfConvertOptions();
```
## चरण 4: रूपांतरण करें
MPX फ़ाइल को PDF प्रारूप में परिवर्तित करके रूपांतरण प्रक्रिया को निष्पादित करें।
```csharp
converter.Convert(outputFile, options);
```

उपयोगकर्ता को सूचित करें कि रूपांतरण प्रक्रिया सफलतापूर्वक पूरी हो गई है और परिवर्तित फ़ाइल का स्थान प्रदान करें।

```csharp
Console.WriteLine("\nConversion to PDF completed successfully. \nCheck output in {0}", outputFolder);
```
