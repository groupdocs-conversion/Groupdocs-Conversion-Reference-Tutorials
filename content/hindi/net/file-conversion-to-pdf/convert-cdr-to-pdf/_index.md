## नामस्थान आयात करें
सुनिश्चित करें कि आपने GroupDocs.Conversion कार्यक्षमताओं का उपयोग करने के लिए अपने .NET प्रोजेक्ट में आवश्यक नामस्थान आयात किए हैं। यहां बताया गया है कि आप यह कैसे कर सकते हैं:

## चरण 1: आउटपुट फ़ोल्डर और फ़ाइल नाम निर्धारित करें
सबसे पहले, आउटपुट फ़ोल्डर को निर्दिष्ट करें जहां परिवर्तित पीडीएफ फाइल को वांछित फ़ाइल नाम के साथ सहेजा जाएगा।
```csharp
string outputFolder = "Your Document Directory";
string outputFile = Path.Combine(outputFolder, "cdr-converted-to.pdf");
```
प्रतिस्थापित करना सुनिश्चित करें `"Your Document Directory"` अपने इच्छित आउटपुट फ़ोल्डर के पथ के साथ.
## चरण 2: स्रोत CDR फ़ाइल लोड करें
इसके बाद, स्रोत CDR फ़ाइल लोड करें जिसे आप GroupDocs.Conversion का उपयोग करके PDF में कनवर्ट करना चाहते हैं।
```csharp
using (var converter = new GroupDocs.Conversion.Converter(Constants.SAMPLE_CDR))
{
    // रूपांतरण तर्क यहाँ जाएगा
}
```
प्रतिस्थापित करें `Constants.SAMPLE_CDR` अपनी वास्तविक CDR फ़ाइल के पथ के साथ.
## चरण 3: रूपांतरण विकल्प निर्दिष्ट करें
रूपांतरण विकल्प परिभाषित करें, जैसे आउटपुट प्रारूप (पीडीएफ) और कोई अतिरिक्त सेटिंग निर्दिष्ट करना।
```csharp
var options = new PdfConvertOptions();
```
आप अपनी आवश्यकताओं के अनुसार रूपांतरण विकल्पों को अनुकूलित कर सकते हैं।
## चरण 4: रूपांतरण करें
निर्दिष्ट विकल्पों के साथ रूपांतरण प्रक्रिया निष्पादित करें.
```csharp
converter.Convert(outputFile, options);
```
यह कमांड CDR फ़ाइल को PDF में परिवर्तित करेगा और उसे दिए गए फ़ाइल नाम के साथ निर्दिष्ट आउटपुट फ़ोल्डर में सहेज देगा।
## चरण 5: रूपांतरण पूरा होने की पुष्टि करें
अंत में, रूपांतरण प्रक्रिया के सफल समापन की पुष्टि करने वाला एक संदेश प्रदर्शित करें।
```csharp

```
यह संदेश आपको उस स्थान के बारे में सूचित करेगा जहां परिवर्तित पीडीएफ फाइल सहेजी गई है।
