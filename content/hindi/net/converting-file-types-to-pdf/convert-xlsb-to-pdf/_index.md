# XLSB को PDF में बदलें

## परिचय

## नामस्थान आयात करें
अपने C# प्रोजेक्ट में, .NET के लिए GroupDocs.Conversion की कार्यक्षमताओं तक पहुँचने के लिए आवश्यक नामस्थान आयात करें:
```csharp
using System;
using System.IO;
using GroupDocs.Conversion.Options.Convert;
```

## चरण 1: आउटपुट फ़ोल्डर और फ़ाइल नाम सेट करें
आउटपुट फ़ोल्डर पथ को परिभाषित करें जहां परिवर्तित PDF फ़ाइल सहेजी जाएगी, और आउटपुट PDF फ़ाइल का नाम निर्दिष्ट करें।
```csharp
string outputFolder = "Your Document Directory";
string outputFile = Path.Combine(outputFolder, "xlsb-converted-to.pdf");
```
## चरण 2: स्रोत XLSB फ़ाइल लोड करें
स्रोत XLSB फ़ाइल का पथ प्रदान करके कनवर्टर को आरंभ करें।
```csharp
using (var converter = new GroupDocs.Conversion.Converter(Constants.SAMPLE_XLSB))
{
    // रूपांतरण तर्क यहाँ है
}
```
## चरण 3: रूपांतरण विकल्प निर्दिष्ट करें
रूपांतरण विकल्प सेट करें। इस मामले में, हम उपयोग करेंगे `PdfConvertOptions` चूँकि हम पीडीएफ प्रारूप में परिवर्तित कर रहे हैं।
```csharp
var options = new PdfConvertOptions();
```
## चरण 4: रूपांतरण करें
आह्वान करें `Convert` कनवर्टर ऑब्जेक्ट पर विधि, आउटपुट फ़ाइल पथ और रूपांतरण विकल्पों को पैरामीटर के रूप में पास करना।
```csharp
converter.Convert(outputFile, options);
```
## चरण 5: पूर्णता संदेश प्रदर्शित करें
एक बार रूपांतरण पूरा हो जाने पर, उपयोगकर्ता को सफल रूपांतरण के बारे में सूचित करें और वह पथ प्रदान करें जहां परिवर्तित पीडीएफ फाइल सहेजी गई है।
```csharp
Console.WriteLine("\nConversion to PDF completed successfully.\nCheck output in {0}", outputFolder);
```
