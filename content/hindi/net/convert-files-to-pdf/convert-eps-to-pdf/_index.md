## परिचय
इस ट्यूटोरियल में, हम दिखाएंगे कि .NET के लिए GroupDocs.Conversion का उपयोग करके EPS (एनकैप्सुलेटेड पोस्टस्क्रिप्ट) फ़ाइलों को पीडीएफ में कैसे परिवर्तित किया जाए।
## आवश्यक शर्तें
रूपांतरण के साथ आगे बढ़ने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित हैं:
1.  .NET के लिए GroupDocs.Conversion: सुनिश्चित करें कि आपने .NET के लिए GroupDocs.Conversion स्थापित किया है। आप इसे यहां से डाउनलोड कर सकते हैं[यहाँ](https://releases.groupdocs.com/conversion/net/).
2. स्रोत ईपीएस फ़ाइल: वह ईपीएस फ़ाइल तैयार करें जिसे आप पीडीएफ में बदलना चाहते हैं।

## नामस्थान आयात करें
रूपांतरण प्रक्रिया शुरू करने से पहले, आवश्यक नामस्थान आयात करें:
```csharp
using System;
using System.IO;
using GroupDocs.Conversion.Options.Convert;
```
## चरण 1: आउटपुट फ़ोल्डर और फ़ाइल को परिभाषित करें
```csharp
string outputFolder = "Your Document Directory";
string outputFile = Path.Combine(outputFolder, "eps-converted-to.pdf");
```
 प्रतिस्थापित करना सुनिश्चित करें`"Your Document Directory"` आपके इच्छित आउटपुट फ़ोल्डर के पथ के साथ।
## चरण 2: स्रोत ईपीएस फ़ाइल लोड करें और पीडीएफ में कनवर्ट करें
```csharp
// स्रोत ईपीएस फ़ाइल लोड करें
using (var converter = new GroupDocs.Conversion.Converter("Path to Your EPS File"))
{
    var options = new PdfConvertOptions();
    // परिवर्तित पीडीएफ फाइल को सहेजें
    converter.Convert(outputFile, options);
}
```
 प्रतिस्थापित करें`"Path to Your EPS File"` आपकी ईपीएस फ़ाइल के वास्तविक पथ के साथ।
## चरण 3: रूपांतरण समापन संदेश प्रदर्शित करें
```csharp
Console.WriteLine("\nConversion to pdf completed successfully. \nCheck output in {0}", outputFolder);
```
यह लाइन उस पथ के साथ एक सफलता संदेश आउटपुट करेगी जहां परिवर्तित पीडीएफ फाइल सहेजी गई है।
