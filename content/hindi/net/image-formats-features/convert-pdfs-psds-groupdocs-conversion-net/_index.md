## कार्यान्वयन मार्गदर्शिका

### फ़ीचर: पीडीएफ से PSD रूपांतरण

PDF फ़ाइल को PSD फ़ॉर्मेट में बदलना उन ग्राफ़िक डिज़ाइनरों के लिए बहुत उपयोगी है जिन्हें संपादन योग्य लेयर्स की ज़रूरत होती है। आइए इस प्रक्रिया को विस्तार से समझते हैं:

#### चरण 1: आउटपुट फ़ोल्डर और फ़ाइल पथ परिभाषित करें

इनपुट और आउटपुट फ़ाइलों के लिए निर्देशिकाएँ सेट करें। आवश्यकतानुसार पथ समायोजित करें।

```csharp
string outputFolder = "YOUR_OUTPUT_DIRECTORY";
string inputFile = "YOUR_DOCUMENT_DIRECTORY\\sample.pdf";
string outputFileTemplate = Path.Combine(outputFolder, "converted-page-{0}.psd");
```

#### चरण 2: स्ट्रीम फ़ंक्शन बनाएँ

हम प्रत्येक परिवर्तित पृष्ठ के लिए स्ट्रीम बनाने के लिए एक फ़ंक्शन का उपयोग करेंगे। यह सुनिश्चित करता है कि प्रत्येक PSD फ़ाइल का नाम सही ढंग से रखा गया है।

```csharp
Func<SavePageContext, Stream> getPageStream = savePageContext =>
    new FileStream(string.Format(outputFileTemplate, savePageContext.Page), FileMode.Create);
```

#### चरण 3: रूपांतरण विकल्प सेट करें

यह निर्दिष्ट करने के लिए कि हम PSD प्रारूप में रूपांतरण कर रहे हैं, रूपांतरण विकल्प परिभाषित करें।

```csharp
ImageConvertOptions options = new ImageConvertOptions { Format = ImageFileType.Psd };
```

#### चरण 4: रूपांतरण करें

का उपयोग करके रूपांतरण निष्पादित करें `Converter` ऑब्जेक्ट और आपकी निर्धारित सेटिंग्स।

```csharp
using (Converter converter = new Converter(inputFile))
{
    // पीडीएफ के प्रत्येक पृष्ठ को PSD प्रारूप में परिवर्तित करें
    converter.Convert(getPageStream, options);
}
```

### समस्या निवारण युक्तियों

- सुनिश्चित करें कि सभी फ़ाइल पथ सही हैं.
- सत्यापित करें कि आपके पास आउटपुट निर्देशिका के लिए लेखन अनुमति है।
