**आप क्या सीखेंगे:**
- .NET के लिए GroupDocs.Conversion की स्थापना और उपयोग करना
- MHT फ़ाइलों को TEX प्रारूप में परिवर्तित करने के लिए चरण-दर-चरण निर्देश
- मुख्य कॉन्फ़िगरेशन विकल्प और समस्या निवारण युक्तियाँ

आइए सबसे पहले यह सुनिश्चित करें कि आपके पास आवश्यक पूर्वापेक्षाएँ मौजूद हैं।

### बुनियादी आरंभीकरण और सेटअप
अपने C# प्रोजेक्ट में GroupDocs.Conversion API को इस तरह प्रारंभ करें:
```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;

class Program
{
    static void Main()
    {
        string outputFolder = "YOUR_OUTPUT_DIRECTORY";
        string inputFile = "YOUR_DOCUMENT_DIRECTORY\sample.mht";

        // कनवर्टर ऑब्जेक्ट को MHT फ़ाइल के साथ आरंभ करें।

        {
            Console.WriteLine("Converter initialized successfully.");
        }
    }
}
```

## कार्यान्वयन मार्गदर्शिका
MHT फ़ाइलों को TEX प्रारूप में परिवर्तित करने का तरीका यहां दिया गया है:

### चरण 1: स्रोत MHT फ़ाइल लोड करें
रूपांतरण के लिए स्रोत MHT फ़ाइल लोड करें:
```csharp
using (var converter = new Converter(@"YOUR_DOCUMENT_DIRECTORY\sample.mht"))
{
    Console.WriteLine("Source MHT file loaded.");
}
```

### चरण 2: रूपांतरण विकल्प कॉन्फ़िगर करें
TEX प्रारूप के अनुरूप विकल्प कॉन्फ़िगर करें:
```csharp
PageDescriptionLanguageConvertOptions options = new PageDescriptionLanguageConvertOptions
{
    Format = PageDescriptionLanguageFileType.Tex // लक्ष्य प्रारूप के रूप में TEX निर्दिष्ट करें.
};
```

### चरण 3: रूपांतरण करें
रूपांतरण निष्पादित करें और आउटपुट को .tex फ़ाइल के रूप में सहेजें:
```csharp
string outputFile = Path.Combine(outputFolder, "mht-converted-to.tex");
converter.Convert(outputFile, options);
Console.WriteLine("Conversion completed successfully.");
```
यह चरण आपके दस्तावेज़ को वांछित प्रारूप में बदल देता है।
