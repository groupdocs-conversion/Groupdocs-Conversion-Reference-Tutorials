## .NET के लिए GroupDocs.Conversion सेट करना
अब जब आपके पास सब कुछ तैयार है, तो आइए दस्तावेज़ रूपांतरण की स्थापना और कार्यान्वयन पर ध्यान दें।

### FTP से दस्तावेज़ डाउनलोड करना
#### अवलोकन
यह अनुभाग दर्शाता है कि C# का उपयोग करके FTP सर्वर से दस्तावेज़ कैसे लाया जाए।
##### FTP अनुरोध बनाएँ
एक बनाकर शुरू करें `FtpWebRequest` फ़ाइल डाउनलोड करने के लिए:
```csharp
private static FtpWebRequest CreateRequest(Uri uri)
{
    // URI के साथ FTP अनुरोध आरंभ करें.
    FtpWebRequest request = (FtpWebRequest)WebRequest.Create(uri);
    
    // FTP से फ़ाइल डाउनलोड करने के लिए विधि सेट करें।
    request.Method = WebRequestMethods.Ftp.DownloadFile;
    
    return request;
}
```
यह विधि एक FTP वेब अनुरोध सेट करती है जो फ़ाइल डाउनलोड करने को निर्दिष्ट करती है।

##### दस्तावेज़ स्ट्रीम प्राप्त करें
इसके बाद, दस्तावेज़ को स्ट्रीम के रूप में पुनः प्राप्त करें:
```csharp
private static Stream GetFileFromFtp(string filePath)
{
    Uri uri = new Uri(filePath); // FTP पथ के लिए URI ऑब्जेक्ट बनाएँ.
    FtpWebRequest request = CreateRequest(uri); // FTP वेब अनुरोध सेटअप करें.

    using (WebResponse response = request.GetResponse()) // प्रतिक्रिया स्ट्रीम भेजें और प्राप्त करें.
        return GetFileStream(response); // मेमोरीस्ट्रीम में परिवर्तित करें.
}
```
यह फ़ंक्शन एक FTP सर्वर से एक दस्तावेज़ प्राप्त करता है, और उसे एक में परिवर्तित करता है `MemoryStream` आगे और भी परिवर्तन के लिए।

##### स्ट्रीम निकालें
HTTP/FTP प्रतिक्रिया को पठनीय स्ट्रीम में परिवर्तित करें:
```csharp
private static Stream GetFileStream(WebResponse response)
{
    MemoryStream fileStream = new MemoryStream(); // मेमोरी स्ट्रीम आरंभ करें.
    
    using (Stream responseStream = response.GetResponseStream()) // डेटा स्ट्रीम तक पहुँचें.
        responseStream.CopyTo(fileStream); // डेटा को मेमोरी स्ट्रीम में कॉपी करें.

    fileStream.Position = 0; // पढ़ने के लिए स्थिति रीसेट करें.
    return fileStream; // भरी हुई स्ट्रीम लौटाएँ.
}
```
यह विधि सुनिश्चित करती है कि आपके पास `MemoryStream` आपके दस्तावेज़ का डेटा युक्त, रूपांतरण के लिए तैयार।

### पीडीएफ में परिवर्तित करना
#### अवलोकन
दस्तावेज़ डाउनलोड होने के बाद, हम इसे GroupDocs.Conversion का उपयोग करके PDF प्रारूप में परिवर्तित कर देंगे।
##### कनवर्टर आरंभ करें और दस्तावेज़ परिवर्तित करें
रूपांतरण प्रक्रिया को सेट अप करने का तरीका यहां दिया गया है:
```csharp
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputDirectory, "converted.pdf");
string ftpPath = "ftp://लोकलहोस्ट/सैंपल.doc";

using (Converter converter = new Converter(() => GetFileFromFtp(ftpPath)))
{
    // पीडीएफ रूपांतरण विकल्प सेट करें.
    PdfConvertOptions options = new PdfConvertOptions();
    
    // दस्तावेज़ को PDF फ़ाइल के रूप में परिवर्तित करें और सहेजें।
    converter.Convert(outputFile, options);
}
```
यह स्निपेट आरंभ करता है `Converter` यह एक FTP दस्तावेज़ स्ट्रीम के साथ काम करता है और निर्दिष्ट विकल्पों का उपयोग करके इसे PDF में परिवर्तित करता है।
