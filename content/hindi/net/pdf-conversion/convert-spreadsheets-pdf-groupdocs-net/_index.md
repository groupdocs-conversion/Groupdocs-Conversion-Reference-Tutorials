# .NET में GroupDocs.Conversion का उपयोग करके स्प्रेडशीट को PDF में सहजता से परिवर्तित करें

## परिचय

क्या आप अपने स्प्रेडशीट दस्तावेज़ों को उनके स्वरूपण और विवरण को बनाए रखते हुए पॉलिश पीडीएफ फाइलों में बदलना चाहते हैं? कई व्यवसायों को महत्वपूर्ण लेआउट खोए बिना या प्रति शीट कई पृष्ठों की आवश्यकता के बिना एक्सेल स्प्रेडशीट (.xlsx) को पीडीएफ प्रारूप में बदलने की चुनौती का सामना करना पड़ता है। यह ट्यूटोरियल आपको .NET के लिए GroupDocs.Conversion का उपयोग करने के माध्यम से मार्गदर्शन करेगा, ग्रिड लाइनों को प्रदर्शित करने और यह सुनिश्चित करने जैसे उन्नत विकल्पों के साथ सहज रूपांतरण सक्षम करेगा कि प्रत्येक शीट आपके अंतिम पीडीएफ में एक ही पृष्ठ पर फिट हो।

### आप क्या सीखेंगे:
- .NET के लिए GroupDocs.Conversion की स्थापना और उपयोग करना
- स्वरूपण को संरक्षित रखते हुए एक्सेल फ़ाइलों को पीडीएफ में परिवर्तित करना
- ग्रिड लाइन्स और एक-पृष्ठ-प्रति-शीट विकल्प प्रदर्शित करने जैसी उन्नत रूपांतरण सुविधाओं का लाभ उठाना

आइए इस शक्तिशाली समाधान में उतरने से पहले आवश्यक पूर्वापेक्षाओं पर नजर डालें।

## कार्यान्वयन मार्गदर्शिका

उन्नत विकल्पों का उपयोग करके स्प्रेडशीट को PDF में बदलने के लिए इन चरणों का पालन करें:

### उन्नत विकल्पों के साथ स्प्रेडशीट को पीडीएफ में बदलें

#### अवलोकन

ग्रिड लाइन प्रदर्शित करते हुए एक्सेल फ़ाइल को पीडीएफ में परिवर्तित करें और सुनिश्चित करें कि आउटपुट दस्तावेज़ में प्रत्येक शीट एक पृष्ठ पर दिखाई दे।

#### चरण 1: लोड विकल्प परिभाषित करें

उन्नत सेटिंग्स के लिए लोड विकल्प कॉन्फ़िगर करें:

```csharp
using GroupDocs.Conversion.Options.Load;

Func<LoadContext, LoadOptions> getLoadOptions = loadContext => new SpreadsheetLoadOptions
{
    ShowGridLines = true,
    OnePagePerSheet = true
};
```

**स्पष्टीकरण**: `SpreadsheetLoadOptions` आपको स्प्रेडशीट लोड करने का तरीका कॉन्फ़िगर करने देता है। सेटिंग `ShowGridLines` को `true` आपके पीडीएफ में ग्रिड लाइनें शामिल हैं, और `OnePagePerSheet` यह सुनिश्चित करता है कि प्रत्येक शीट एक ही पृष्ठ पर फिट हो।

#### चरण 2: कनवर्टर क्लास का उपयोग करके कन्वर्ट करें

उपयोग `Converter` रूपांतरण करने के लिए क्लास:

```csharp
using System;
using System.IO;
using GroupDocs.Conversion.Options.Convert;

string inputFilePath = "sample.xlsx";
string outputFolder = "output";
string outputFile = Path.Combine(outputFolder, "converted.pdf");

// लोड विकल्पों के साथ कनवर्टर आरंभ करें
using (Converter converter = new Converter(inputFilePath, getLoadOptions))
{
    PdfConvertOptions options = new PdfConvertOptions(); // PDF रूपांतरण विकल्प सेट अप करें
    converter.Convert(outputFile, options); // रूपांतरण करें
}
```

**स्पष्टीकरण**: द `Converter` क्लास आपके एक्सेल फ़ाइल पथ और लोड विकल्पों को लेता है। `PdfConvertOptions` क्लास पीडीएफ आउटपुट के लिए कोई अतिरिक्त सेटिंग्स निर्दिष्ट करता है।

#### समस्या निवारण युक्तियों
- सुनिश्चित करें कि आपका इनपुट फ़ाइल पथ सही है.
- जांचें कि GroupDocs.Conversion लाइब्रेरी संस्करण आपके प्रोजेक्ट के .NET फ्रेमवर्क संस्करण से मेल खाता है या नहीं।
- सुनिश्चित करें कि आपके पास आउटपुट निर्देशिका के लिए लेखन अनुमति है।
