## परिचय
ओटीजी (ओपन डॉक्यूमेंट ग्राफ़िक्स) फ़ाइलों को पीडीएफ प्रारूप में परिवर्तित करना एक महत्वपूर्ण कार्य हो सकता है, खासकर जब दस्तावेज़ प्रबंधन प्रणालियों से निपटना हो या विभिन्न प्लेटफार्मों पर फ़ाइलें साझा करना हो। इस ट्यूटोरियल में, हम आपको .NET के लिए GroupDocs.Conversion लाइब्रेरी का उपयोग करके OTG फ़ाइलों को पीडीएफ में परिवर्तित करने की प्रक्रिया के बारे में मार्गदर्शन करेंगे। आइए गोता लगाएँ!
## आवश्यक शर्तें
शुरू करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित आवश्यकताएँ हैं:
1. .NET के लिए GroupDocs.Conversion: सुनिश्चित करें कि आपने .NET के लिए GroupDocs.Conversion लाइब्रेरी स्थापित की है। आप इसे यहां से डाउनलोड कर सकते हैं[यहाँ](https://releases.groupdocs.com/conversion/net/).
2. ओटीजी फ़ाइल: जिस ओटीजी फ़ाइल को आप पीडीएफ में कनवर्ट करना चाहते हैं उसे अपनी दस्तावेज़ निर्देशिका में तैयार रखें।

## नामस्थान आयात करें
सबसे पहले, आपको अपने .NET प्रोजेक्ट में आवश्यक नेमस्पेस आयात करना होगा। 
```csharp
using System;
using System.IO;
using GroupDocs.Conversion.Options.Convert;
```
## चरण 1: आउटपुट निर्देशिका और फ़ाइल नाम सेट करें
```csharp
string outputFolder = "Your Document Directory";
string outputFile = Path.Combine(outputFolder, "otg-converted-to.pdf");
```
 इस चरण में, आप आउटपुट निर्देशिका को परिभाषित करते हैं जहां परिवर्तित पीडीएफ फ़ाइल सहेजी जाएगी। प्रतिस्थापित करें`"Your Document Directory"` आपकी इच्छित आउटपुट निर्देशिका के पथ के साथ।
## चरण 2: स्रोत ओटीजी फ़ाइल लोड करें और पीडीएफ में कनवर्ट करें
```csharp
using (var converter = new GroupDocs.Conversion.Converter(Constants.SAMPLE_OTG))
{
    var options = new PdfConvertOptions();
    // परिवर्तित पीडीएफ फाइल को सहेजें
    converter.Convert(outputFile, options);
}
```
 यहां, हम एक नया इंस्टेंट करते हैं`Converter` GroupDocs.Conversion लाइब्रेरी से ऑब्जेक्ट, स्रोत OTG फ़ाइल का पथ पार करते हुए। फिर, हम बनाते हैं`PdfConvertOptions` रूपांतरण विकल्प निर्दिष्ट करने के लिए. अंत में, हम कॉल करते हैं`Convert` ओटीजी फाइल को पीडीएफ फॉर्मेट में बदलने की विधि।
## चरण 3: रूपांतरण पूर्णता की जाँच करें
```csharp
Console.WriteLine("\nConversion to pdf completed successfully. \nCheck output in {0}", outputFolder);
```
यह चरण उपयोगकर्ता को सूचित करता है कि रूपांतरण प्रक्रिया सफलतापूर्वक पूरी हो गई है और वह पथ प्रदान करता है जहां परिवर्तित पीडीएफ फ़ाइल सहेजी गई है।
