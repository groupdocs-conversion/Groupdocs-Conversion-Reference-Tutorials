# OTG को PDF में बदलें

## परिचय
OTG (OpenDocument Graphics) फ़ाइलों को PDF प्रारूप में बदलना एक महत्वपूर्ण कार्य हो सकता है, खासकर जब दस्तावेज़ प्रबंधन प्रणालियों से निपटना हो या विभिन्न प्लेटफ़ॉर्म पर फ़ाइलें साझा करना हो। इस ट्यूटोरियल में, हम आपको .NET के लिए GroupDocs.Conversion लाइब्रेरी का उपयोग करके OTG फ़ाइलों को PDF में बदलने की प्रक्रिया के माध्यम से मार्गदर्शन करेंगे। चलो गोता लगाएँ!
## आवश्यक शर्तें
शुरू करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित पूर्वापेक्षाएँ हैं:
1. .NET के लिए GroupDocs.Conversion: सुनिश्चित करें कि आपने .NET के लिए GroupDocs.Conversion लाइब्रेरी स्थापित की है। आप इसे यहाँ से डाउनलोड कर सकते हैं [यहाँ](https://releases.groupdocs.com/conversion/net/).
2. OTG फ़ाइल: जिस OTG फ़ाइल को आप PDF में बदलना चाहते हैं उसे अपने दस्तावेज़ निर्देशिका में तैयार रखें।

## नामस्थान आयात करें
सबसे पहले, आपको अपने .NET प्रोजेक्ट में आवश्यक नेमस्पेस आयात करने होंगे। 
```csharp
using System;
using System.IO;
using GroupDocs.Conversion.Options.Convert;
```
## चरण 1: आउटपुट निर्देशिका और फ़ाइल नाम सेट करें
```csharp
string outputFolder = "Your Document Directory";
string outputFile = Path.Combine(outputFolder, "otg-converted-to.pdf");
```
इस चरण में, आप आउटपुट डायरेक्टरी को परिभाषित करते हैं जहाँ परिवर्तित पीडीएफ फाइल को सहेजा जाएगा। `"Your Document Directory"` अपनी इच्छित आउटपुट निर्देशिका के पथ के साथ.
## चरण 2: स्रोत OTG फ़ाइल लोड करें और PDF में कनवर्ट करें
```csharp
using (var converter = new GroupDocs.Conversion.Converter(Constants.SAMPLE_OTG))
{
    var options = new PdfConvertOptions();
    // परिवर्तित पीडीएफ फाइल सहेजें
    converter.Convert(outputFile, options);
}
```
यहाँ, हम एक नया उदाहरण प्रस्तुत करते हैं `Converter` GroupDocs.Conversion लाइब्रेरी से ऑब्जेक्ट, स्रोत OTG फ़ाइल का पथ पास करना। फिर, हम बनाते हैं `PdfConvertOptions` रूपांतरण विकल्प निर्दिष्ट करने के लिए। अंत में, हम कॉल करते हैं `Convert` ओटीजी फ़ाइल को पीडीएफ प्रारूप में परिवर्तित करने की विधि।
## चरण 3: रूपांतरण पूर्णता की जाँच करें
```csharp
Console.WriteLine("\nConversion to pdf completed successfully. \nCheck output in {0}", outputFolder);
```
यह चरण उपयोगकर्ता को सूचित करता है कि रूपांतरण प्रक्रिया सफलतापूर्वक पूरी हो गई है और वह पथ प्रदान करता है जहां परिवर्तित पीडीएफ फाइल को सहेजा गया है।
