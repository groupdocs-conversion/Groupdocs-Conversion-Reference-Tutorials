## नामस्थान आयात करें
रूपांतरण प्रक्रिया में गहराई से जाने से पहले, सुनिश्चित करें कि आप GroupDocs.Conversion कार्यात्मकताओं तक पहुँचने के लिए आवश्यक नामस्थान आयात करते हैं।

```csharp
using System;
using System.IO;
using GroupDocs.Conversion.Options.Convert;
```
अब जब हमने मंच तैयार कर लिया है, तो आइए .NET के लिए GroupDocs.Conversion का उपयोग करके PPSM (पावरपॉइंट स्लाइड शो) फ़ाइल को पीडीएफ में परिवर्तित करने की चरण-दर-चरण प्रक्रिया पर ध्यान दें।
## 1. आउटपुट फ़ोल्डर और फ़ाइल पथ को परिभाषित करें
```csharp
string outputFolder = "Your Document Directory";
string outputFile = Path.Combine(outputFolder, "ppsm-converted-to.pdf");
```
## 2. स्रोत पीपीएसएम फ़ाइल लोड करें
```csharp
using (var converter = new GroupDocs.Conversion.Converter(Constants.SAMPLE_PPSM))
{
    var options = new PdfConvertOptions();
```
## 3. रूपांतरण विकल्प कॉन्फ़िगर करें (वैकल्पिक)
अपनी आवश्यकताओं के आधार पर, आप विभिन्न रूपांतरण विकल्पों को कॉन्फ़िगर कर सकते हैं। इस उदाहरण में, हम बिना किसी विशिष्ट विकल्प के पीडीएफ में परिवर्तित कर रहे हैं।
## 4. रूपांतरण करें
```csharp
// परिवर्तित पीडीएफ फाइल को सहेजें
converter.Convert(outputFile, options);
```
## 5. रूपांतरण समापन को संभालें
```csharp
Console.WriteLine("\nConversion to pdf completed successfully. \nCheck output in {0}", outputFolder);
```
