## कार्यान्वयन मार्गदर्शिका

आइए, आपकी OTP फ़ाइलों को PDF में परिवर्तित करने के लिए कार्यान्वयन प्रक्रिया को तार्किक चरणों में विभाजित करें।

### अपना दस्तावेज़ लोड और कॉन्फ़िगर करें
सबसे पहले, का उपयोग करके अपनी OTP फ़ाइल लोड करें `Converter` क्लास। यह सभी रूपांतरण कार्यों के लिए आधार के रूप में कार्य करता है:

```csharp
// पथ परिभाषित करें
string sourceFilePath = "YOUR_DOCUMENT_DIRECTORY\\sample.otp";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

// OTP फ़ाइल पथ के साथ कनवर्टर आरंभ करें
using (var converter = new Converter(sourceFilePath))
{
    // PDF रूपांतरण विकल्प कॉन्फ़िगर करें
    var pdfOptions = new PdfConvertOptions();

    // आउटपुट फ़ाइल पथ सेटअप

    // दस्तावेज़ को PDF के रूप में परिवर्तित करें और सहेजें
    converter.Convert(outputFile, pdfOptions);
}
```

#### प्रमुख मापदंडों का स्पष्टीकरण
- **कनवर्टर वर्ग**: स्रोत फ़ाइल को दर्शाता है। इसके लिए एक वैध OTP फ़ाइल पथ की आवश्यकता होती है।
- **पीडीएफ कन्वर्ट विकल्प**: यह कॉन्फ़िगर करता है कि रूपांतरण कैसे निष्पादित किया जाना चाहिए। आप पृष्ठ संख्या और ज़ूम स्तर जैसी सेटिंग्स समायोजित कर सकते हैं।
- **कनवर्टर.कन्वर्ट()**: वह विधि जो वास्तविक फ़ाइल रूपांतरण करती है.
