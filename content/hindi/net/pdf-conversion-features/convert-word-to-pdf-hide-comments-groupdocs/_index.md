### बुनियादी आरंभीकरण और सेटअप

अपने C# प्रोजेक्ट में GroupDocs.Conversion प्रारंभ करें:
```csharp
using System;
using GroupDocs.Conversion;
using GroupDocs.Conversion.Options.Convert;
using GroupDocs.Conversion.Options.Load;

class Program
{
    static void Main()
    {
        // आउटपुट निर्देशिका और फ़ाइल पथ को परिभाषित करें.
        string outputFolder = "YOUR_OUTPUT_DIRECTORY";
        string outputFile = Path.Combine(outputFolder, "converted.pdf");

        // रूपांतरण के दौरान टिप्पणियाँ छिपाने के लिए लोड विकल्प प्रारंभ करें।
        Func<LoadContext, LoadOptions> getLoadOptions = loadContext => new WordProcessingLoadOptions
        {
            CommentDisplayMode = WordProcessingCommentDisplay.Hidden // टिप्पणियाँ छिपाने के लिए सेट करें
        };

        // GroupDocs.Conversion का उपयोग करके दस्तावेज़ को परिवर्तित करें।
        using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_DOCX_WITH_TRACKED_CHANGES", getLoadOptions))
        {
            PdfConvertOptions options = new PdfConvertOptions(); // पीडीएफ रूपांतरण विकल्प आरंभ करें
            converter.Convert(outputFile, options); // पीडीएफ में रूपांतरण करें
        }
    }
}
```

## कार्यान्वयन मार्गदर्शिका

### छुपी हुई टिप्पणियों के साथ वर्ड प्रोसेसिंग दस्तावेज़ को पीडीएफ में बदलें

किसी को परिवर्तित करने के लिए इन चरणों का पालन करें `.docx` टिप्पणियों को छिपाते हुए फ़ाइल को पीडीएफ में परिवर्तित करें।

#### चरण 1: लोड विकल्प परिभाषित करें
तय करना `CommentDisplayMode` को `Hidden`:
```csharp
Func<LoadContext, LoadOptions> getLoadOptions = loadContext => new WordProcessingLoadOptions
{
    CommentDisplayMode = WordProcessingCommentDisplay.Hidden // रूपांतरण के दौरान टिप्पणियाँ छिपाएँ
};
```

#### चरण 2: कनवर्टर आरंभ करें
एक बनाने के `Converter` उदाहरण:
```csharp
using (Converter converter = new Converter("YOUR_DOCUMENT_DIRECTORY/SAMPLE_DOCX_WITH_TRACKED_CHANGES", getLoadOptions))
{
    // रूपांतरण सेटअप के साथ आगे बढ़ें.
}
```

#### चरण 3: पीडीएफ रूपांतरण विकल्प सेट करें
प्रारंभ `PdfConvertOptions`:
```csharp
PdfConvertOptions options = new PdfConvertOptions();
```

#### चरण 4: रूपांतरण करें
रूपांतरण प्रक्रिया निष्पादित करें:
```csharp
converter.Convert(outputFile, options); // छुपी हुई टिप्पणियों के साथ PDF में कनवर्ट करें
```
